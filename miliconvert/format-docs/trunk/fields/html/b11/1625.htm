FIELD USE DESIGNATOR
EXPLANATION
002
PARTICIPATING UNIT NUMBER, LINK 11
( 2 N )
AN OCTAL IDENTIFIER FOR A UNIT COMMUNICATING DIRECTLY ON LINK 11
(TADIL A).
004
TRACK NUMBER
( 4 N )
AN OCTAL REFERENCE NUMBER ASSIGNED TO AIR TRACKS, SURFACE TRACKS,
SUBSURFACE TRACKS, BEARINGS, FIXES, AND/OR SPECIAL POINTS IN LINKS
11, 11B, AND 4. ALSO REFERRED TO AS A JTAO TRACK NUMBER.
006
TRACK NUMBER BLOCK LOWER LIMIT
( 4 N )
A JTAO OCTAL TRACK NUMBER USED AS THE LOWER LIMIT OF A TRACK BLOCK.
007
SPECIAL TRACK NUMBER
( 4 N )
AN OCTAL REFERENCE NUMBER USED TO IDENTIFY A PARTICULAR TYPE PLATFORM
(E.G., MPA, KRESTA) OR PLATFORM NAME (E.G., TROMP, MOUNT WHITNEY)
WHICH IS NOT INCLUDED IN ASSIGNED TRACK BLOCKS.
008
TRACK NUMBER BLOCK UPPER LIMIT
( 4 N )
A JTAO OCTAL TRACK NUMBER USED AS THE UPPER LIMIT OF A TRACK BLOCK.
013
SYSTEM REFERENCE NUMBER BLOCK LOWER LIMIT
( 5 N )
AN INTERIM JOINT TACTICAL INFORMATION DISTRIBUTION SYSTEM (JTIDS)
MESSAGE SPECIFICATION (IJMS) OCTAL TRACK NUMBER USED AS THE LOWER
LIMIT OF A SYSTEM REFERENCE NUMBER BLOCK.
014
SYSTEM REFERENCE NUMBER BLOCK UPPER LIMIT
( 5 N )
AN IJMS OCTAL TRACK NUMBER USED AS THE UPPER LIMIT OF A SYSTEM REFERENCE
NUMBER BLOCK.
015
REPORTING UNIT (RU) NUMBER
( 3 N )
AN OCTAL IDENTIFIER FOR A UNIT COMMUNICATING DIRECTLY ON LINK 11B
(TADIL B).
017
LINK 4 ADDRESS
( 5 N )
AN OCTAL NUMBER (01000-17776, OMIT 8, 9) APPLIED TO A LINK 4 INTERFACE
UNIT TO ASSOCIATE INFORMATION AND DIRECTIVES WITH INTERFACE UNITS
OR TRACKS FOR BOTH DIGITAL AND VOICE COMMUNICATIONS.
018
LINK ADDRESS BLOCK LOWER LIMIT
( 5 N )
AN OCTAL IDENTIFIER REPRESENTING THE LOWER LIMIT OF A LINK ADDRESS.
019
LINK ADDRESS BLOCK UPPER LIMIT
( 5 N )
AN OCTAL IDENTIFIER REPRESENTING THE UPPER LIMIT OF A LINK ADDRESS.
021
PRIMARY JTIDS UNIT (JU) ADDRESS
( 5 N )
THE OCTAL NUMBER REPRESENTING A PRIMARY JTIDS (JOINT TACTICAL
INFORMATION DISTRIBUTION SYSTEM) UNIT ADDRESS.
022
SECONDARY JTIDS UNIT (JU) ADDRESS
( 5 N )
AN OCTAL NUMBER REPRESENTING A SECONDARY JTIDS (JOINT TACTICAL
INFORMATION DISTRIBUTION SYSTEM) UNIT ADDRESS.
024
SUPPORTING UNIT (SU) NUMBER
( 4 N )
AN OCTAL IDENTIFIER OF A SUPPORTING UNIT IN A DATA LINK.
025
DISCRETE IDENTIFIER
( 4 N )
THE DISCRETE IDENTIFIER CODE IS A UNIQUE IDENTIFIER WHICH ASSISTS IN THE CORRELATION OF SHIPS AND AIRCRAFT REPORTED IN CHARACTER-ORIENTED MESSAGES WITH SHIPS AND AIRCRAFT REPORTED IN BIT-ORIENTED MESSAGES.
