:
AN ANGLE, IN MILS, MEASURED CLOCKWISE BETWEEN A REFERENCE DIRECTION
AND ANY OTHER LINE. THIS MEASUREMENT MAY INCLUDE NEGATIVE VALUES WHEN
BEING MEASURED IN OPPOSITE DIRECTIONS (COUNTERCLOCKWISE) FROM A
REFERENCE LINE SUCH AS THE HORIZON.
