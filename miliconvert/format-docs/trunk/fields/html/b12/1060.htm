FIELD FORMAT NAME

UTM FALSE 1-METRE NORTHING
FIELD FORMAT LENGTH
:
7-8
DEFINITION
:
A NORTHWARD (I.E., BOTTOM TO TOP) READING OF UNIVERSAL TRANSVERSE
MERCATOR (UTM) GRID VALUES MEASURED FROM A FALSE GRID ZONE ORIGIN (A
FIXED POINT TO THE SOUTH AND WEST OF A GRID ZONE ORIGIN) TO A
PRECISION OF ONE METRE.
RELATED DOCUMENTS
:
TM 5-243
STANAG 5620
REMARKS
:
ENTRIES MAY HAVE VALUES CONSISTING OF EITHER SEVEN CHARACTERS (FOR A
LOCATION IN THE SAME EARTH HEMISPHERE AS THAT ENTERED IN A TACTICAL
DATA SYSTEM (TDS) AS ITS REFERENCE OR BASE) OR OF EIGHT CHARACTERS
(WHEN A LOCATION IS IN A DIFFERENT EARTH HEMISPHERE FROM THE TDS
BASE).
