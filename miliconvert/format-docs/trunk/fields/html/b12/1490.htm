PROTOCOL REFERENCE
( 3-9 ANS )
ALLOWED ENTRIES ARE GIVEN BY "PROTOCOL ON PROCEDURES COVERING THE
REDUCTION OF CONVENTIONAL ARMAMENTS LIMITED BY THE TREATY ON CFE". EG.
III-2-B (REDUCTION PROCEDURE OF BATTLE TANKS FOR DESTRUCTION BY
SEVERING [REMOVAL OF TURRET]).
003
REMARK REFERENCE CODE
( 1-5 X )
A UNIQUE REFERENCE CODE USED TO LINK A QUESTION, ANSWER OR STATEMENT ON
AMBIGUITIES REGARDING FACTUAL INFORMATION TO THE FILE "FREE TEXT" IN
VERITY.
004
REFERENCE TO CALENDAR REPORTING PERIOD
( 1-15 X )
REFERENCE TO CALENDAR REPORTING PERIOD NOTIFICATION OR CERTIFICATION
PROCESS NOTIFICATION.
