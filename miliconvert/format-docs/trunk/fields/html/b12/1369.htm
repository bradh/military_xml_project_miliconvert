DEFINITION
:
A UNIQUE GROUP OF CHARACTERS ASSIGNED TO EACH SET FORMAT AS A KEY TO THE STRUCTURE AND INFORMATION CONTENT OF THAT FORMAT. NOTE 1: LINEAR AND FREE TEXT SET FORMAT IDENTIFIERS COMPRISE THREE TO EIGHT ALPHABETIC CHARACTERS. NOTE 2: COLUMNAR SET FORMAT IDENTIFIERS
COMPRISE ONE NUMERIC CHARACTER FOLLOWED BY TWO TO SEVEN ALPHABETIC CHARACTERS.
