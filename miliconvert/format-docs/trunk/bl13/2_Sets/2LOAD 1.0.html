DATA ENTRY

 

ENTER THE TWO DIGIT NUMBER (01 THROUGH 99) THAT TIES THE INFORMATION IN THIS DATA LINE TO RELATED INFORMATION IN SET LOADINFO, FOR EXAMPLE: "01". See table 1348/1 a range [01 through 99]

NUMBER OF CARGO ITEMS

 

 

 

 

2A

NUMBER OF CARGO ITEMS

 

ENTER THE QUANTITY OF CARGO ITEMS (1 THROUGH 999) REPORTED IN FIELD 3, FOR EXAMPLE: "4". See table 1044/149 a range [1 through 999]

3

3A

TYPE OF AIR CARGO

 

ENTER THE TYPE OF CARGO BEING AIRLIFTED, FOR EXAMPLE: "PALL". See table 1173/6 which contains a list of data items and associated data codes

DANGEROUS CARGO DESIGNATION

 

 

 

 

4A

DANGEROUS CARGO DESIGNATION

 

ENTER THE LITERAL DANGEROUS CARGO DESIGNATION AS CONTAINED IN ICAO UN OR NATIONAL REGULATIONS OR STANAG 3854, FOR EXAMPLE: "UA5716". See table 1221/2 which is an instructive entry

INDIVIDUAL CARGO WEIGHT

 

ENTER THE INDIVIDUAL CARGO WEIGHT.  ENTER A CONTEXT QUANTITY, DECIMAL POINT PERMITTED, 8-CHARACTER MAXIMUM (1 THROUGH 999999.9) AND THE UNIT OF WEIGHT MEASUREMENT, FOR EXAMPLE: "5585.5KG" See table 2185 which is a composite

AIR CARGO LINEAR DIMENSIONS

 

ENTER THE LINEAR DIMENSION OF THE WEIGHT.  ENTER A CONTEXT QUANTITY, DECIMAL POINT PERMITTED, 5-CHARACTER MAXIMUM (0 THROUGH 99999) OR (.0001 THROUGH 999.9), AN "X" CHARACTER, A CONTEXT QUANTITY, DECIMAL POINT PERMITTED, 5-CHARACTER MAXIMUM (0 THROUGH 99999) OR (.0001 THROUGH 999.9), AN "X" CHARACTER, A CONTEXT QUANTITY, DECIMAL POINT PERMITTED, 5-CHARACTER MAXIMUM (0 THROUGH 99999) OR (.0001 THROUGH 999.9) AND THE UNIT OF LINEAR MEASUREMENT, FOR EXAMPLE: "460X190X200CM". See table 2182 which is a composite
