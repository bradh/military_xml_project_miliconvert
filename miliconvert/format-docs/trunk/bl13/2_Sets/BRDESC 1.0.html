Concept/Explanation/Examples

1

BRIDGE COMPONENT

 

Identify the bridge component.  You must specify at least one instance of the Bridge Structure, Pier Type and Bridge Beam, and depending on the message, the Bridge Abutment.

 

 

1A

BRIDGE STRUCTURE

 

Describes the bridge structure, for example: "ST". See table 1705/2 which contains a list of data items and associated data codes

 

 

1B

PIER TYPE

PIER

Enter "PIER:" followed by the code that describes the bridge pier type, for example: "PIER:GR". See table 1705/3 which contains a list of data items and associated data codes

 

 

1C

BRIDGE BEAM

 

Enter "BE" to indicate that the bridge beams are being described. See table 1705/5 which contains a list of data items and associated data codes

 

 

1D

BRIDGE ABUTMENT

 

Enter "AB" to indicate the bridge abutments are being described. See table 1705/6 which contains a list of data items and associated data codes

2

CONSTRUCTION MATERIAL

 

 

 

 

2A

CONSTRUCTION MATERIAL

 

Describes the construction material of the bridge component identified in field 1, for example: "ML". See table 1592/5 which contains a list of data items and associated data codes
