1

FIXED WING IDENTIFICATION SAFETY RANGE, NAUTICAL MILES

 

 

 

 

1A

ISR FIXED WING, NAUTICAL MILES

 

ENTER THE FIXED WING IDENTIFICATION SAFETY RANGE (ISR) (0 THROUGH 99999) (0 TO 4 DECIMAL PLACES)  AND THE UNIT OF LINEAR MEASUREMENT NAUTICAL MILES, FOR EXAMPLE: "50NM". See table 2272 which is a composite

2

ROTARY  WING IDENTIFICATION SAFETY RANGE, NAUTICAL MILES

 

 

 

 

2A

ISR ROTARY WING, NAUTICAL MILES

 

ENTER THE ROTARY WING IDENTIFICATION SAFETY RANGE (ISR) (0 THROUGH 99999) (0 TO 4 DECIMAL PLACES)  AND THE UNIT OF LINEAR MEASUREMENT NAUTICAL MILES, FOR EXAMPLE: "30NM". See table 2272 which is a composite

3

THE LOCATION THAT THE ISR IS CENTRED UPON

 

THE LOCATION THAT THE ISR IS CENTRED UPON STATED AS EITHER AN "ATP-1" POSITION DESIGNATOR OR A UNIT/SHIP NAME.

 

 

3A

ATP-1 POSITION DESIGNATOR

ATP1

ENTER "ATP1:" FOLLOWED BY THE ATP1 POSITION DESIGNATOR, FOR EXAMPLE: "ATP1:ZZ". See table 1612/1 which contains a list of data items and associated data codes

 

 

3B

UNIT NAME

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OR DESIGNATION OF THE UNIT UPON WHICH THE ISR IS CENTRED, FOR EXAMPLE: "UNIT:INVINCIBLE". See table 1022/48 which is an instructive entry

4

THE MEAN LINE OF ADVANCE OF THE CENTRE OF THE ISR.

 

 

 

 

4A

MEAN LINE OF ADVANCE, DEGREES TRUE

 

ENTER THE DIRECTION OF MOVEMENT OF THE CENTER POINT OF THE SAFETY CIRCLE IN DEGREES (000 THROUGH 359), AND THE ANGULAR MEASUREMENT REFERENCE TRUE (T), FOR EXAMPLE: "135T". See table 2260 which is a composite

5

SPEED OF ADVANCE

 

THE MEAN SPEED OF ADVANCE EXPRESSED EITHER AS MEASURED IN NM OR USING THE "ADVANCE" CODE.

 

 

5A

SPEED OF ADVANCE, KNOTS

 

ENTER THE SPEED OF ADVANCE IN KNOTS, FOR EXAMPLE: "55KTS". See table 2265 which is a composite

 

 

5B

SPEED OF ADVANCE, APPROXIMATE

 

THE APPROXIMATE SPEED OF ADVANCE CODE, FOR EXAMPLE: "SLOW". See table 1379/2 which contains a list of data items and associated data codes
