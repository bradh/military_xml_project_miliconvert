MISSION NUMBER

 

 

 

 

1A

MISSION NUMBER

 

ENTER THE MISSION NUMBER ASSIGNED BY THE OPERATIONAL COMMANDER, FOR EXAMPLE: "CM28361". See table 1012/69 which is an instructive entry

2

NUMBER OF CONTACTS GAINED

 

 

 

 

2A

NUMBER OF CONTACTS GAINED

 

ENTER THE TOTAL NUMBER OF CONTACTS GAINED DURING THE MISSION, GIVEN AS "GAINED" PLUS ONE OR TWO NUMERALS, FOR EXAMPLE: "GAINED6". See table 2299 which is a composite

3

NUMBER OF CONTACTS PASSED

 

 

 

 

3A

NUMBER OF CONTACTS PASSED

 

ENTER THE CONTACTS RELAYED FROM THE OPCON FACILITY AND RECEIVED BY THE AIRCRAFT EITHER BY RADIO OR DURING THE PREFLIGHT BRIEFING, GIVEN AS "PASSED" PLUS ONE OR TWO NUMERALS, FOR EXAMPLE: "PASSED3". See table 2299 which is a composite

4

NUMBER OF CONTACTS INVESTIGATED

 

 

 

 

4A

NUMBER OF CONTACTS INVESTIGATED

 

ENTER THE NUMBER OF CONTACTS THAT WERE INVESTIGATED DURING THE MISSION, EXPRESSED AS "INVEST" PLUS ONE OR TWO NUMERALS, FOR EXAMPLE: "INVEST3". See table 2299 which is a composite

5

NUMBER OF CONTACTS COMPARED

 

 

 

 

5A

NUMBER OF CONTACTS COMPARED

 

ENTER THE NUMBER OF CONTACTS INVESTIGATED THAT WERE CORRELATED TO CONTACTS PASSED, EXPRESSED AS "COMP" PLUS ONE OR TWO NUMERALS, FOR EXAMPLE: "COMP3". See table 2299 which is a composite

6

NUMBER OF CONTACTS NOT COMPARED

 

 

 

 

6A

NUMBER OF CONTACTS NOT COMPARED

 

ENTER THE NUMBER OF CONTACTS INVESTIGATED THAT WERE NOT CORRELATED TO CONTACTS PASSED, EXPRESSED AS "NCOMP" PLUS ONE OR TWO NUMERALS, FOR EXAMPLE: "NCOMP2". See table 2299 which is a composite

7

NUMBER OF CONTACTS CONVERTED

 

 

 

 

7A

NUMBER OF CONTACTS CONVERTED

 

ENTER THE NUMBER OF CONTACTS COMPARED AND ALSO SUCCESSFULLY LOCALIZED, EXPRESSED AS "CONV", PLUS ONE OR TWO NUMERALS, FOR EXAMPLE: "CONV1". See table 2299 which is a composite

8

NUMBER OF CONTACTS ASSOCIATED

 

 

 

 

8A

NUMBER OF CONTACTS ASSOCIATED

 

ENTER THE NUMBER OF CONTACTS INITIATED BY THE INVESTIGATING UNIT AND SUBSEQUENTLY CORRELATED TO A KNOWN TARGET, EXPRESSED AS "ASSOC", PLUS ONE OR TWO NUMERALS, FOR EXAMPLE: "ASSOC2". See table 2299 which is a composite
