ICAO BASE CODE

 

SPECIFY THE LOCATION OF THE ASSIGNED UNIT AIRCRAFT.  ENTER THE FOUR CHARACTER ICAO LOCATION CODE, FOR EXAMPLE: "ENOL". See table 1125/2 which is an instructive entry

2

AEW AIRCRAFT

 

 

 

 

2A

AEW AIRCRAFT

 

SPECIFY THE TYPE OF AEW AIRCRAFT ASSIGNED.  ENTER THE AIRCRAFT TYPE CODE, FOR EXAMPLE: "USE3B". See table 1015/14 which contains a list of data items and associated data codes

3

TOTAL NUMBER OF AIRCRAFT

 

 

 

 

3A

TOTAL NUMBER OF AIRCRAFT

 

SPECIFY THE NUMBER OF THE TYPE OF AIRCRAFT ASSIGNED, FOR EXAMPLE: "5". See table 1044/212 a range [0 through 999]

4

NUMBER OF COMBAT READY AIRCRAFT

 

 

 

 

4A

NUMBER OF COMBAT READY AIRCRAFT

 

SPECIFY THE NUMBER OF THE AIRCRAFT THAT ARE COMBAT READY, FOR EXAMPLE: "4". See table 1044/215 a range [0 through 999]
