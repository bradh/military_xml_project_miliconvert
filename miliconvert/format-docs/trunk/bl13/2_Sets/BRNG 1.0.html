1

DATE-TIME

 

PROVIDES THE DAY-TIME OF BEARING IN ONE OF THE FOLLOWING WAYS:

 

 

1A

DAY-TIME OF BEARING

 

ENTER THE DAY-TIME OF BEARING, FOR EXAMPLE: "190830Z". See table 2000 which is a composite

2

BEARING ORIGIN OR REFERENCE POINT IDENTIFIER

 

THE GEOGRAPHICAL DESCRIPTION OF THE ORIGIN OF THE BEARING.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

REFERENCE POINT IDENTIFIER

ID

ENTER "ID:" FOLLOWED BY THE REFERENCE POINT IDENTIFIER, FOR EXAMPLE: "ID:ZK". See table 1012/91 which is an instructive entry

3

BEARING, DEGREES TRUE

 

 

 

 

3A

BEARING, DEGREES TRUE

 

ENTER THE BEARING IN DEGREES TRUE (T) FROM THE BEARING ORIGIN TO THE TARGET OF INTEREST BY USING THREE NUMERALS PLUS "T", FOR EXAMPLE: "235T". See table 2253 which is a composite

4

SECTOR HALF-WIDTH

 

SPECIFY THE SECTOR HALF WIDTH IN ONE OF THE FOLLOWING WAYS:

 

 

4A

SECTOR HALF-WIDTH DISTANCE, NAUTICAL MILES

 

ENTER THE SECTOR HALF-WIDTH DISTANCE IN NAUTICAL MILES, FOR EXAMPLE: "60NM". See table 2280 which is a composite

 

 

4B

SECTOR HALF-WIDTH (DEGREE)

 

ENTER THE ANGULAR WIDTH IN DEGREES, ONE TO THREE NUMERICS PLUS "DEG", FOR EXAMPLE: "95DEG". See table 2278 which is a composite

5

LINE OF BEARING (LOB) RANGE, NAUTICAL MILES

 

 

 

 

5A

LINE OF BEARING (LOB) RANGE, NAUTICAL MILES

 

ENTER THE MAXIMUM LENGTH OF THE LINE OF BEARING (LOB) RANGE IN NAUTICAL MILES, FOR EXAMPLE: "45NM". See table 2272 which is a composite
