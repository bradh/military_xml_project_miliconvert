UNIT DESIGNATOR

 

THE DESIGNATOR OF THE FRIENDLY UNIT.

 

 

1A

UNIT

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT DESIGNATION OF A MILITARY UNIT (1-30 ANBS).  FOR EXAMPLE, "UNIT:TASK FORCE DELTA" See table 1022/57 which is an instructive entry

 

 

1B

TASKED UNIT DESIGNATOR

DESIG

ENTER "DESIG:" FOLLOWED BY THE IDENTIFYING DESIGNATOR OF A SPECIFIC FRIENDLY NAVAL TASK ORGANIZATION (3-15 ANS).  FOR EXAMPLE,"DESIG:CTG122.2" See table 2143 which is a composite

UNIT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE FRIENDLY UNIT.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE GEOGRAPHIC LOCATION OF THE FRIENDLY FORCES, GEOREF IN CENTIMINUTES, FOR EXAMPLE: "GEO:AABB15233527". See table 2018 which is a composite

 

 

2H

PLACE NAME, EXTENDED

NAME

ENTER "NAME:" FOLLOWED BY THE NAME ASSIGNED TO THE LOCATION, FOR EXAMPLE: "NAME:GARGANTA DEL DIABLO". See table 1022/261 which is an instructive entry

 

 

2I

TRUE BEARING AND DISTANCE IN NM FROM REFERENCE POINT

TBRN

ENTER "TBRN:" FOLLOWED BY THE POSITION OF A FRIENDLY FORCE EXPRESSED AS A BEARING IN DEGREES TRUE AND DISTANCE IN NAUTICAL MILES FROM A NAMED REFERENCE POINT,  FOR EXAMPLE: "TBRN:330-PT ALFA-50". See table 2016 which is a composite

 

 

2J

PORT CODE

PORT

ENTER "PORT:" FOLLOWED BY THE CODE THAT IDENTIFIES A PORT, FOR EXAMPLE: "PORT:AAA". See table 1182/1 which is an instructive entry

 

 

2K

BEARING AND DISTANCE IN NM FROM REFERENCE POINT

BARN

ENTER "BARN:" FOLLOWED BY THE POSITION OF FRIENDLY FORCES IN TERMS OF BEARING AND DISTANCE FROM A REFERENCE POINT, FOR EXAMPLE: "BARN:330T-PT ALFA-50NM". See table 2307 which is a composite

 

 

2L

TRUE BEARING AND DISTANCE FROM REF POINT

TBR

ENTER "TBR:" FOLLOWED BY THE POSITION OF FRIENDLY FORCES IN TERMS OF TRUE BEARING AND DISTANCE, FOR EXAMPLE: "TBR:310T-PT BRAVO-50NM". See table 2047 which is a composite

 

 

2M

Q-ROUTE POSITION

QPOS

ENTER "QPOS:" FOLLOWED BY THE LOCATION OF THE FRIENDLY FORCES IN TERMS OF A POSITION ON A PRESCRIBED TRACK OVER THE SURFACE TO BE FOLLOWED FROM A SPECIFIC POINT OF ORIGIN TO A SPECIFIC DESTINATION, FOR EXAMPLE: "QPOS:123Q450PS300". See table 2203 which is a composite

 

 

2N

BEARING AND DISTANCE

BAR

ENTER "BAR:" FOLLOWED BY THE POSITION OF FRIENDLY FORCES GIVEN AS A BEARING AND RANGE FROM A DESIGNATED POINT, FOR EXAMPLE: "BAR:330-PT ALFA-50". See table 2111 which is a composite

 

 

2O

POSITION, TRUE BEARING AND RANGE IN NM

FBAR

ENTER "FBAR:" FOLLOWED BY THE POSITION OF FRIENDLY FORCES GIVEN AS A BEARING AND RANGE FROM A DESIGNATED POINT,  FOR EXAMPLE: "FBAR:330-PT ALFA-50". See table 2281 which is a composite

 

 

2P

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES VERIFIED

SUB

ENTER "SUB:", FOLLOWED BY THE LATITUDE AND LONGITUDE OF SUBMARINE FRIENDLY FORCES UP TO 1/10,000 OF A MINUTE, VERIFIED, FOR EXAMPLE: "SUB:2010.30N6-01040.20W7". See table 2577 which is a composite

3

TIME

 

THE TIME THE FRIENDLY UNIT WAS AT THE LOCATION REPORTED IN FIELD TWO.

 

 

3A

DAY-TIME OF POSITION

TIME

ENTER THE DAY OF A MONTH AND TIMEKEEPING IN HOURS AND MINUTES OF A CALENDAR DAY, USING THE 24-HOUR CLOCK SYSTEM AND AN ASSOCIATED TIME ZONE (7 AN).  FOR EXAMPLE,"TIME:141325Z" See table 2000 which is a composite

4

COURSE AND SPEED

 

THE COURSE AND SPEED OF THE FRIENDLY UNIT.

 

 

4A

FRIENDLY FORCE COURSE (TRUE) AND SPEED (KNOTS)

SPEED

THE DIRECTION OF TRAVEL MEASURED IN DEGREES TRUE AND THE MEASURED OR ESTIMATED SPEED/VELOCITY OF A FRIENDLY FORCE IN KNOTS (8-10 ANS).  FOR EXAMPLE,"SPEED:270T12KTS" See table 2037 which is a composite

 

 

4B

COURSE AND APPROXIMATE SPEED

RSPEED

THE DIRECTION IN DEGREES TRUE AND RELATIVE SPEED OF AN OBJECT (7-8 AN).  FOR EXAMPLE, "RSPEED:270TMED" See table 2270 which is a composite

5A

GEOGRAPHIC ENTITIES, NOTIONAL AND ACTUAL

 

ENTER THE COUNTRY OF THE WORLD TO WHICH  MILITARY EQUIPMENT, A UNIT, OR A FACILITY BELONGS (2 A).  FOR EXAMPLE, "US" See table 1026/16 which contains a list of data items and associated data codes

DISCRETE IDENTIFIER

 

 

 

 

6A

DISCRETE IDENTIFIER
