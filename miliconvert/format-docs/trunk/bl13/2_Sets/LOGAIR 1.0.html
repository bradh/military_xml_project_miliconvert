1

MNLC AIR INSTRUCTION TYPE

 

 

 

 

1A

MNLC AIR INSTRUCTION TYPE

 

THIS FIELD CONTAINS  ONE OF THE THREE TYPES OF AIR INSTRUCTIONS, FOR EXAMPLE "VOD". See table 1902/2 which contains a list of data items and associated data codes

2

NUMBER OF AIRCRAFT ALLOCATED

 

ENTER THE NUMBER OF AIRCRAFT.

 

 

2A

NUMBER OF AIRCRAFT ALLOCATED

 

ENTER THE NUMBER OF AIRCRAFT OF THE SPECIFIC TYPE REPORTED IN THE NEXT FIELD THAT ARE ALLOCATED, FOR EXAMPLE "15". See table 1044/154 a range [1 through 999]

3

AIRCRAFT CODE

 

 

 

 

3A

AIRCRAFT CODE

 

ENTER THE APPROPRIATE AIRCRAFT CODE, FOR EXAMPLE "C141". See table 1015/2 which contains a list of data items and associated data codes

4

BASE LOCATION OF AIRCRAFT

 

LOCATION OF THE AIRCRAFT THAT FORM PART OF THE LOGISTICS PLAN.

 

 

4A

BASE NAME

BASE

ENTER  "BASE:" FOLLOWED BY THE LITERAL BASE NAME, FOR EXAMPLE "BASE:LISBON". See table 1022/104 which is an instructive entry

 

 

4B

SHIP NAME

SHIP

ENTER  "SHIP:" FOLLOWED BY THE SHIP NAME, FOR EXAMPLE "SHIP:USS ENTERPRISE". See table 1022/49 which is an instructive entry

 

 

4C

ICAO BASE CODE

ICAO

ENTER  "ICAO:" FOLLOWED BY THE ICAO BASE CODE, FOR EXAMPLE "ICAO:DFFD". See table 1125/2 which is an instructive entry

5

FLIGHT MISSION DESCRIPTION

 

 

 

 

5A

FLIGHT MISSION DESCRIPTION

 

ENTER A LITERAL DESCRIPTION OF THE FLIGHT MISSION, FOR EXAMPLE "REFUELLING". See table 1363/69 which is an instructive entry
