TARGET CATEGORY DESIGNATOR

TARGET DESCRIPTION

 

ENTER A DESCRIPTION OF THE TARGET OF THE TARGET BY GIVING EITHER ON OF THE FOLLOWING:

 

 

3A

TARGET ELEMENT DESCRIPTION

EQUIPMENT TYPE

EQPT

ENTER "EQPT:" FOLLOWED BY THE EQUIPMENT TYPE, FOR EXAMPLE: "EQPT:FIGHTER AIRCRAFT". See table 1135/1 which is an instructive entry

NUMBER OF TARGET ELEMENTS

 

 

 

 

4A

NUMBER OF TARGET ELEMENTS

GEOGRAPHICAL ENTITY

 

 

 

 

5A

GEOGRAPHICAL ENTITY

 

IF USED, ENTER CODE FROM STANAG 1059 THAT INDICATES THE NATIONALITY OF THE TARGET, FOR EXAMPLE: "RS". See table 1026/1 which contains a list of data items and associated data codes

6

TARGET LOCATION

 

THE GEOGRAPHICAL POSITION OF THE TARGET.

 

 

6A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

6B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

6C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

6D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

6E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

6F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

BAR

NAME

ENTER "NAME:" FOLLOWED BY THE TARGET LOCATION NAME, FOR EXAMPLE: "NAME:REGENSBURG". See table 1022/103 which is an instructive entry

TARGET ACTIVITY DESCRIPTION

TARGET ACTIVITY DESCRIPTION

DAY-TIME

 

ENTER DAY-TIME IN ONE OF THE FOLLOWING WAYS:

 

 

8A

DAY-TIME OF POSITION

 

ENTER TIME INFORMATION IN THE FORM: DAY [01-31], HOUR [00-23], MINUTE [00-59], AND TIME ZONE INDICATOR [1A], FOR EXAMPLE: "201200Z". See table 2000 which is a composite

9

TARGET COURSE

 

IF USED, ENTER THE TARGET COURSE IN ONE OF THE FOLLOWING WAYS:

 

 

9A

TARGET COURSE IN DEGREES

 

ENTER DIRECTION INFORMATION IN DEGREES [000-359] FOLLOWED BY A ONE LETTER CODE INDICATING THE ANGULAR MEASUREMENT REFERENCE, FOR EXAMPLE: "180T". See table 2065 which is a composite

 

 

9B

DIRECTION OF MOVEMENT (COMPASS POINT)

 

ENTER THE DIRECTION OF MOVEMENT BY COMPASS POINT, FOR EXAMPLE: "SSW". See table 1161/1 which contains a list of data items and associated data codes

10

TARGET SPEED

 

SPECIFY THE TARGET SPEED IN ONE OF THE FOLLOWING WAYS:

 

 

10A

TARGET SPEED

TGTSPEED

ENTER "TGTSPEED:" FOLLOWED BY THE TARGET SPEED AND A  UNIT OF SPEED MEASUREMENT, FOR EXAMPLE: "TGTSPEED:500KTS". See table 2063 which is a composite

 

 

10B

SPEED QUALIFIER

SPDQUAL

ENTER "SPDQUAL:" FOLLOWED BY THE CODE FOR A SPEED QUALIFIER, FOR EXAMPLE: "SPDQUAL:STAT". See table 1672/1 which contains a list of data items and associated data codes
