RENDEZVOUS BREAKUP ACTIVITY TYPE

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

GRID

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

3

RENDEZVOUS/BREAKUP ALTITUDE

 

PROVIDES THE ALTITUDE OR FLIGHT LEVEL OF THE RENDEZVOUS/BREAK UP ACTIVITY.

 

 

3A

ALTITUDE, HUNDREDS OF FEET

ALT

ENTER "ALT:" FOLLOWED BY THE ALTITUDE IN HUNDREDS OF FEET THAT THE ACTIVITY IS TO TAKE PLACE, FOR EXAMPLE: "ALT:050". See table 1255/14 a range [000 through 999]

 

 

3B

FLIGHT LEVEL

 

ENTER THE FLIGHT LEVEL THAT THE ACTIVITY IS TO TAKE PLACE, FOR EXAMPLE: "FL230". See table 2129 which is a composite

4

RENDEZVOUS/BREAKUP TIME

 

PROVIDES THE TIME OF THE RENDEZVOUS/BREAKUP ACTIVITY.

 

 

4A

RELATIVE AIRCRAFT ARRIVAL TIME

REL

ENTER "REL:" FOLLOWED BY THE TIME OF ARRIVAL OF AN AIRCRAFT AT A SPECIFIC LOCATION, FOR EXAMPLE: "REL:LP0130". See table 2482 which is a composite

 

 

4B

DAY-TIME OF ARRIVAL

ARRV

ENTER "ARRV:" FOLLOWED BY THE DAY-TIME OF ARRIVAL OF AN AIRCRAFT AT A SPECIFIC LOCATION, FOR EXAMPLE: "ARRV:141325Z". See table 2000 which is a composite

5

RENDEZVOUS LEADER INDICATOR

 

 

 

 

5A

RENDEZVOUS LEADER INDICATOR

 

ENTER THE CODE TO INDICATE IF THE TASKED MISSION AIRCRAFT IS THE RENDEZVOUS LEADER, FOR EXAMPLE: "Y". See table 1359/21 which contains a list of data items and associated data codes

6

MISSION IDENTIFIER

 

PROVIDES THE MISSION IDENTIFIER OF THE AIRCRAFT WITH WHICH THE MISSION IS TO RENDEZVOUS, BREAKUP, SPREAD OR RE-JOIN WITH.

 

 

6A

CONTACT CALL SIGN

CS

ENTER "CS:" FOLLOWED BY THE CALL SIGN OF THE RENDEZVOUS LEADER, FOR EXAMPLE: "CS:JOLLY 20". See table 1129/31 which is an instructive entry

 

 

6B

RENDEZVOUS AIRCRAFT MISSION NUMBER

MN

ENTER "MN:" FOLLOWED BY THE NUMBER ASSIGNED TO THE RENDEZVOUS LEADER FOR EXAMPLE: "MN:SAR101". See table 1012/242 which is an instructive entry

LINK 16 ABBREVIATED CALL SIGN

AIRCRAFT NUMBER

 

 

 

 

11A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT PARTECIPATING IN THE RENDEZVOUS/BREAKUP ACTIVITY, FOR EXAMPLE: "32". See table 1044/322 a range [1 through 99]

12

AIRCRAFT TYPE

 

PROVIDES THE TYPE AND MODEL OF THE AIRCRAFT PARTICIPATING IN THE RENDEZVOUS/BREAKUP ACTIVITY.  CAN ONLY BE USED IN CONJUNCTION WITH AIRCRAFT NUMBER FOR EVERY OCCURRENCE IN THIS SET.

 

 

12A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE CODE USED TO IDENTIFY THE TYPE OF AIRCRAFT, FOR EXAMPLE: "ACTYP:C12A". See table 1015/20 which contains a list of data items and associated data codes

 

 

12B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE IDENTIFICATION OF OTHER AIRCRAFT, FOR EXAMPLE: "OTHAC:XX35". See table 1015/18 which is an instructive entry
