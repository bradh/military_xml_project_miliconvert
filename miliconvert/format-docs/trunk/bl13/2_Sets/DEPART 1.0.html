DEPARTURE STATUS

 

 

 

 

1A

DEPARTURE STATUS

 

ENTER THE STATUS OF THE DEPARTURE MOVEMENT, FOR EXAMPLE: "DEP". See table 1095/6 which contains a list of data items and associated data codes

2

CALL SIGN

 

 

 

 

2A

CALL SIGN

 

ENTER THE CALL SIGN FOR THE DEPARTURE, FOR EXAMPLE: "CHARLIE". See table 1129/1 which is an instructive entry

3

DTG OF DEPARTURE

 

 

 

 

3A

DTG OF DEPARTURE

 

ENTER THE DAY-TIME, MONTH AND YEAR OF THE DEPARTURE, FOR EXAMPLE: "231500ZOCT2003". See table 2033 which is a composite

4

 DTG OF DELAYED DEPARTURE

 

CAN ONLY BE USED IF THE DEPARTURE STATUS EQUALS "DEL" IN THIS SET.

 

 

4A

DTG OF DELAYED DEPARTURE

 

ENTER THE DAY-TIME, MONTH AND YEAR OF THE NEW DEPARTURE FOR A DELAYED FLIGHT, FOR EXAMPLE: "301500ZOCT2003". See table 2033 which is a composite

5

REASON FOR DELAY/CANCEL

 

CAN ONLY BE USED IF THE DEPARTURE STATUS EQUALS "DEL" OR "CXL" IN THIS SET.

 

 

5A

REASON FOR DELAY CANCEL

 

ENTER THE REASON WHY THE DEPARTURE WAS DELAYED OR CANCELLED, FOR EXAMPLE: "AIRCRAFT". See table 1318/3 which contains a list of data items and associated data codes

Notes:   

STRUCTURE LANGUAGE IS USED TO ENSURE THE DTG FOR DELAYED DEPARTURE CAN BE FILLED WHENEVER THE DEPARTURE STATUS IS "DELAYED". MOREOVER, THE REASON FOR DELAY OR CANCELLATION MUST NOT BE ENTERED IF THE MOVEMENT DEPARTED.
