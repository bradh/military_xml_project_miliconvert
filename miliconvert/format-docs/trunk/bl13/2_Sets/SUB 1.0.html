Concept/Explanation/Examples

1

CONTACT SERIAL NUMBER

 

 

 

 

1A

CONTACT SERIAL NUMBER

 

ENTER THE SEQUENTIAL NUMBER OF THE CONTACT TYPE WITHIN A SINGLE MESSAGE, FOR EXAMPLE: "1". See table 1012/67 a range [1 through 999]

2

UNIT IDENTIFIER

 

SPECIFIES THE UNIT IDENTIFIER.

 

 

2A

UNIT

 

ENTER THE SUBMARINE NAME, FOR EXAMPLE: "LE REDOUTABLE". See table 1022/57 which is an instructive entry

 

 

2B

TRADEMARK

TM

ENTER FIELD DESCRIPTOR "TM:" FOLLOWED BY THE TRADEMARK NAVAL OPERATIONS DESIGNATOR, FOR EXAMPLE: "TM:XYZ9-32". See table 1416/3 which is an instructive entry

3

LOCATING REPORT NUMBER

 

SPECIFIES THE LOCATING REPORT NUMBER.

 

 

3A

LOCATING REPORT NUMBER

 

ENTER THE LOCATING REPORT QUALIFIER FOLLOWED BY THE LOCATING REPORT SERIAL NUMBER, FOR EXAMPLE: "LR21". See table 2296 which is a composite

 

 

3B

NOT REPORTED IN FLIGHT

 

ENTER THE CODE "NRIF" SPECIFYING NOT REPORTED IN FLIGHT. See table 1449/1 which contains a list of data items and associated data codes

4

SUBMARINE CLASSIFICATION/CONFIDENCE LEVEL

 

 

 

 

4A

SUBMARINE CLASSIFICATION CONFIDENCE LEVEL

 

ENTER THE CODE SPECIFYING THE SUBMARINE CLASSIFICATION/CONFIDENCE LEVEL, FOR EXAMPLE: "PROB". See table 1123/5 which contains a list of data items and associated data codes

5

FLAG

 

 

 

 

5A

FLAG

 

ENTER THE COUNTRY CODE OF THE FLAG UNDER WHICH THE SUBMARINE OPERATES, FOR EXAMPLE: "FR". See table 1026/3 which contains a list of data items and associated data codes

6

SUBMARINE PROPULSION TYPE

 

 

 

 

6A

SUBMARINE PROPULSION TYPE

 

ENTER THE SUBMARINE PROPULSION TYPE, FOR EXAMPLE: "DSL". See table 1339/1 which contains a list of data items and associated data codes

7

CLASS OR TYPE

 

SPECIFIES THE STANDARD CLASS NAME OR TYPE OF SUBMARINE

 

 

7A

SUBMARINE TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE SUBMARINE TYPE, FOR EXAMPLE: "SSN". See table 1166/2 which contains a list of data items and associated data codes

 

 

7B

SUBMARINE CLASS NAME

 

ENTER THE SUBMARINE CLASS NAME, FOR EXAMPLE: "NOVEMBER". See table 1038/1 which contains a list of data items and associated data codes

8

CREDIBILITY

 

 

 

 

8A

CREDIBILITY

 

ENTER THE LEVEL OF CREDIBILITY OF STATED INFORMATION, FOR EXAMPLE: "HIGH". See table 1461/1 which contains a list of data items and associated data codes

9

CONTACT PHOTO STATUS

 

 

 

 

9A

CONTACT PHOTO STATUS

 

ENTER THE INDICATION OF WHETHER OR NOT PHOTOGRAPHS WERE TAKEN OF THE CONTACT, FOR EXAMPLE: "PHT". See table 1450/1 which contains a list of data items and associated data codes

10

SHIP CONTROL NUMBER

 

 

 

 

10A

SHIP CONTROL NUMBER

 

ENTER THE SHIP CONTROL NUMBER OF THE CONTACT, FOR EXAMPLE: "B10456". See table 2282 which is a composite

11

DESIGNATOR OR IDENTIFIER

 

SPECIFIES THE NATO DESIGNATOR OR DISCRETE IDENTIFIER.

 

 

11A

NATO SUBMARINE CONTACT DESIGNATOR

SUB

ENTER FIELD DESCRIPTOR "SUB:" FOLLOWED BY THE NATO SUBMARINE CONTACT DESIGNATOR, FOR EXAMPLE: "SUB:2458". See table 2152 which is a composite

 

 

11B

DISCRETE IDENTIFIER

 

ENTER THE DISCRETE IDENTIFIER OF THE CONTACT (A CODE USED IN MARITIME REPORTING OF TRACKS THAT ARE NOT IDENTIFIED), FOR EXAMPLE: "1234". See table 1625/25 which is an alphanumeric range
