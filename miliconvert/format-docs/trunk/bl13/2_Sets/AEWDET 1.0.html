1

AEW SORTIE NUMBER

 

 

 

 

1A

AEW SORTIE NUMBER

 

ENTER THE UNIQUE NUMBER ASSIGNED TO AN AEW SORTIE SATISFYING AN AEW REQUEST, FOR EXAMPLE:"03021234". See table 2463 which is a composite

2

AEW REQUEST IDENTIFIER

 

 

 

 

2A

AEW REQUEST IDENTIFIER

 

ENTER THE UNIQUE AEW REQUEST IDENTIFIER, WHICH IS SATISFIED BY THIS AEW SORTIE, FOR EXAMPLE: "06060071". See table 1012/17 a range [0 through 9999999999999]

3

MISSION NUMBER

 

 

 

 

3A

MISSION NUMBER

 

ENTER THE MISSION NUMBER FOR THE AEW MISSION PROVIDING THE MISSION REPORT TAKEN FROM THE ATO, FOR EXAMPLE: "ABCD1234". See table 1012/69 which is an instructive entry

4

UNIT IDENTIFICATION

 

SPECIFIES THE UNIT IDENTIFIER FOR THE UNIT WHICH COMPLETED THE AEW MISSION.

 

 

4A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UNIT UIC, FOR EXAMPLE: "UIC:USF2141". See table 2141 which is a composite

 

 

4B

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT DESIGNATOR, FOR EXAMPLE: "UNIT:552 AWACS". See table 1028/43 which is an instructive entry

5

AEW AIRCRAFT

 

 

 

 

5A

AEW AIRCRAFT

 

ENTER THE TYPE OF AEW AIRCRAFT THAT CONDUCTED THE MISSION, FOR EXAMPLE: "E3A". See table 1015/14 which contains a list of data items and associated data codes

6

AIRCRAFT CALL SIGN

 

 

 

 

6A

AIRCRAFT CALL SIGN

 

ENTER THE CALL SIGN FOR THE AIRCRAFT FLIGHT DECK, FOR EXAMPLE: "LOOKER 01". See table 1129/9 which is an instructive entry

7

MISSION CALL SIGN

 

 

 

 

7A

MISSION CALL SIGN

 

ENTER THE CALL SIGN FOR THE MISSION, FOR EXAMPLE: "SCREEN 01". See table 1129/21 which is an instructive entry

8

DEPARTURE LOCATION

 

PROVIDES THE DEPARTURE LOCATION FOR THE AEW AIRCRAFT.

 

 

8A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE DEPARTURE LOCATION CODE, FOR EXAMPLE: "UIC:GEF1234". See table 2141 which is a composite

 

 

8B

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE DEPARTURE LOCATION ICAO CODE, FOR EXAMPLE: "ICAO:KILA". See table 1125/2 which is an instructive entry

 

 

8C

LOCATION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE PLACE NAME OF THE DEPARTURE LOCATION, FOR EXAMPLE: "NAME:BRUSSELS". See table 1022/4 which is an instructive entry

9

DTG OF DEPARTURE

 

 

 

 

9A

DTG OF DEPARTURE

 

ENTER THE TIME THE AIRCRAFT DEPARTED, FOR EXAMPLE: "011200ZAUG2001". See table 2033 which is a composite

10

ARRIVAL LOCATION

 

PROVIDES THE ARRIVAL LOCATION FOR THE AIRCRAFT.

 

 

10A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE LANDING LOCATION CODE, FOR EXAMPLE: "UIC:GEF1235". See table 2141 which is a composite

 

 

10B

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE LANDING LOCATION ICAO CODE, FOR EXAMPLE: "ICAO:KILL". See table 1125/2 which is an instructive entry

 

 

10C

LOCATION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE LANDING LOCATION NAME, FOR EXAMPLE: "NAME:AWAY FIELD". See table 1022/4 which is an instructive entry

11

DTG OF ARRIVAL

 

 

 

 

11A

DTG OF ARRIVAL

 

ENTER THE TIME THE AIRCRAFT LANDED, FOR EXAMPLE: "011500ZAUG2001". See table 2033 which is a composite

12

AMSH 1707 CALL SIGN

 

 

 

 

12A

AMSH 1707 CALL SIGN

 

ENTER THE THREE CHARACTER CALL SIGN AS SPECIFIED BY ASMH 1707, FOR EXAMPLE: "ABC". See table 1129/6 which is an instructive entry

13

MODE 2 IFF SETTING

 

 

 

 

13A

IFF SIF CODE

 

ENTER THE MODE 2 IFF SETTING FOR THE MISSION AIRCRAFT, FOR EXAMPLE: "4343". See table 1303/1 which is an alphanumeric range
