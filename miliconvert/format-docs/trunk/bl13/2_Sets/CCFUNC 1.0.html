Concept/Explanation/Examples

1

COMMAND AND CONTROL FUNCTION

 

 

 

 

1A

COMMAND AND CONTROL FUNCTION

 

ENTER THE TYPE OF COMMAND AND CONTROL (C2) FUNCTION THAT HAS BEEN AFFECTED. FOR EXAMPLE: "SUR". See table 1874/3 which contains a list of data items and associated data codes

2

UNIT OPERATIONAL STATUS

 

 

 

 

2A

COMMAND AND CONTROL FUNCTION OPERATIONAL STATUS

 

ENTER THE OPERATIONAL STATUS OF THE C2 FUNCTION REPORTED IN FIELD 1. FOR EXAMPLE: "DESTR". See table 1236/35 which contains a list of data items and associated data codes

3

REASON FOR OPERATIONAL STATUS

 

ENTER THE REASON FOR THE OPERATIONAL STATUS OF THE AFFECTED FUNCTION, USING ONE OF THE FOLLOWING:

 

 

3A

MISCELLANEOUS REASON

MISC

ENTER "MISC:" FOLLOWED BY THE  MISCELLANEOUS REASON FOR THE STATUS REPORTED IN FIELD 2. FOR EXAMPLE: "MISC:BRK". See table 1212/2 which contains a list of data items and associated data codes

 

 

3B

WEATHER REASON

WEATH

ENTER "WEATH:" FOLLOWED BY THE  WEATHER REASON FOR THE STATUS REPORTED IN FIELD 2. FOR EXAMPLE: "WEATH:ICE". See table 1212/3 which is an instructive entry

 

 

3C

EMCON REASON

EMCON

ENTER "EMCON:" FOLLOWED BY THE  EMCOM REASON FOR THE STATUS REPORTED IN FIELD 2. FOR EXAMPLE: "EMCOM:S05". See table 1212/4 which is an instructive entry

 

 

3D

SPECIAL REASON RELATED TO MOVEMENT NOTIFICATION

MOVE

ENTER "MOVE:" FOLLOWED BY THE  MOVEMENT NOTIFICATION REASON FOR THE STATUS REPORTED IN FIELD 2. FOR EXAMPLE: "MOVE:TTN". See table 1212/5 which is an instructive entry

 

 

3E

SPECIAL REASON RELATED TO RELEASED STATUS

REL

ENTER "REL:" FOLLOWED BY THE  SPECIAL RELEASE REASON FOR THE STATUS REPORTED IN FIELD 2. FOR EXAMPLE: "REL:STD". See table 1212/6 which is an instructive entry

 

 

3F

HOSTILE ACTION

HOST

ENTER "HOST:" FOLLOWED BY THE HOSTILE ACTION REASON FOR THE STATUS REPORTED IN FIELD 2. FOR EXAMPLE: "HOST:NUC". See table 1212/7 which is an instructive entry

4

SUB-CATEGORY LIMITING FACTORS
