1

COUNTRY

 

 

 

 

1A

COUNTRY

 

ENTER A CODE REPRESENTING THE COUNTRY WHICH PRODUCES THE EQUIPMENT, FOR EXAMPLE: "GE". See table 1026/18 which contains a list of data items and associated data codes

2

START DATE-TIME (Z) OF PRODUCTION CYCLE, YYYYMMDDHHMMSS

 

 

 

 

2A

START DATE-TIME (Z) OF PRODUCTION CYCLE, YYYYMMDDHHMMSS

 

ENTER THE DATE [YYYYMMDDHHMMSS] OF WHEN PRODUCTION STARTED, OR IS ESTIMATED TO HAVE STARTED FOR A PIECE OF EQUIPMENT, FOR EXAMPLE: "19990712134657". See table 2417 which is a composite

3

END DATE-TIME (Z) OF PRODUCTION CYCLE, YYYYMMDDHHMMSS

 

 

 

 

3A

END DATE-TIME (Z) OF PRODUCTION CYCLE, YYYYMMDDHHMMSS

 

ENTER THE DATE [YYYYMMDDHHMMSS] WHEN PRODUCTION ENDED, OR IS ESTIMATED TO HAVE ENDED FOR A PIECE OF EQUIPMENT, FOR EXAMPLE: "20001102163832". See table 2417 which is a composite

4

ALSO PRODUCED AS

 

 

 

 

4A

ALSO PRODUCED AS

 

IF THE IDENTIFIED EQUIPMENT IS BEING PRODUCED BY ANOTHER MANUFACTURING ORGANIZATION, ENTER, IN FREE TEXT, THE NAME OF THE OTHER ORGANIZATION AND THE NAME UNDER WHICH IT PRODUCES THE EQUIPMENT IF THIS IS DIFFERENT FROM THAT OF THE BASIC EQUIPMENT, FOR EXAMPLE: "(LITERAL)". See table 1022/247 which is an instructive entry

5

DERIVED FROM

 

 

 

 

5A

DERIVED FROM

 

IF THE EQUIPMENT IS CLEARLY A DERIVATIVE OF ANOTHER IDENTIFIED EQUIPMENT, ENTER THE NAME [LITERAL] OF THE SOURCE EQUIPMENT, FOR EXAMPLE: "(LITERAL)". See table 1022/248 which is an instructive entry

6

PRODUCTION CYCLE

 

ALTERNATE FIELDS CONTAINING A FIGURE EXPRESSING THE PRODUCTION CYCLE OR A CODE INDICATING A LACK OF DATA:

 

 

6A

MANUFACTURING TIME

 

ENTER THE PERIOD OF TIME TAKEN TO MANUFACTURE ONE PIECE OF THE EQUIPMENT, FOR EXAMPLE: "23DAY". See table 2161 which is a composite

7

PRODUCTION QUANTITY

 

ALTERNATE FIELDS CONTAINING A FIGURE FOR THE PRODUCTION QUANTITY OR A CODE INDICATING A LACK OF DATA:

 

 

7A

PRODUCTION QUANTITY, 0-9999999

 

ENTER THE OVERALL NUMBERS PRODUCED BY SUBJECT COUNTRY, FOR EXAMPLE: "5000". GIVE FURTHER DETAILS IN REMARKS. See table 1044/332 a range [0 through 9999999]

8

REMARKS

 

 

 

 

8A

REMARKS

 

ENTER ANY OTHER DATA (LITERAL UP TO 68 CHARACTERS) WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS. See table 1357/28 which is an instructive entry
