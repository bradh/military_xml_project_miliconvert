Designator

Field Desc

Concept/Explanation/Examples

1

AIRSPACE TIME MODE

 

 

 

 

1A

AIRSPACE TIME MODE

 

ENTER THE QUALIFIER INDICATING THE TYPE OF TIME MODE FOR THE AIRSPACE, FOR EXAMPLE: "DISCRETE". See table 1220/19 which contains a list of data items and associated data codes

2

START TIME

 

 

 

 

2A

DTG OF START

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF THE START OF THE AIRSPACE, FOR EXAMPLE: "141325ZFEB2003". See table 2033 which is a composite

3

STOP TIME

 

PROVIDES THE STOP TIME FOR THE AIRSPACE USING EITHER A DATE-TIME GROUP OR A TIME QUALIFIER.

 

 

3A

DTG OF STOP

 

ENTER THE DAY-TIME, MONTH, AND YEAR THE AIRSPACE ENDS, FOR EXAMPLE: "151225ZFEB2003". See table 2033 which is a composite

 

 

3B

STOP TIME QUALIFIER

 

ENTER THE STOP TIME QUALIFIER, FOR EXAMPLE: "UFN". See table 1220/8 which contains a list of data items and associated data codes

4

INTERVAL FREQUENCY

 

 

 

 

4A

INTERVAL FREQUENCY

 

ENTER THE REPEATING CYCLE FOR THE AIRSPACE START, FOR EXAMPLE: "DAILY". See table 1346/3 which contains a list of data items and associated data codes

5

INTERVAL DURATION

 

PROVIDES THE DURATION OF THE AIRSPACE USING EITHER AN INTERVAL DURATION, A DATE-TIME GROUP OR THE TIME QUALIFIER UNTIL FURTHER NOTICE.

 

 

5A

INTERVAL DURATION

ID

ENTER "ID:" FOLLOWED BY THE DURATION OF THE AIRSPACE INTERVAL, FOR EXAMPLE: "ID:4WK". See table 2525 which is a composite

 

 

5B

TIME QUALIFIER UNTIL FURTHER NOTICE

 

ENTER TIME QUALIFIER, UNTIL FURTHER NOTICE, FOR EXAMPLE: "UFN". See table 1220/5 which contains a list of data items and associated data codes

 

 

5C

DTG OF COMPLETION

 

ENTER THE DAY-TIME, MONTH, AND YEAR THE AIRSPACE INTERVAL ENDS, FOR EXAMPLE: "251230ZMAR2003" See table 2033 which is a composite
