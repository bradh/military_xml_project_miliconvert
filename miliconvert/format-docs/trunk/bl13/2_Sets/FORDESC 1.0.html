FORD CAPABILITY

 

 

 

 

1A

FORD CAPABILITY

 

ENTER THE MAXIMUM SUITABLE TRAFFICABILITY OF THE  FORD, EXPRESSED AS THE TYPE OF TRAFFIC IT WILL SUPPORT, FOR EXAMPLE: "VEH". See table 1666/8 which contains a list of data items and associated data codes

2

LENGTH OF FORDING SITE

 

 

 

 

2A

LENGTH OF FORDING SITE

 

ENTER THE LENGTH OF THE FORDING SITE (THE DISTANCE BEING MEASURED FROM THE ENTRANCE POINT TO THE EXIT POINT), FOR EXAMPLE: "700M". See table 2454 which is a composite

3

WIDTH OF FORDING SITE

 

 

 

 

3A

WIDTH OF FORDING SITE

 

ENTER THE USABLE WIDTH OF THE FORDING SITE, FOR EXAMPLE: "50M". See table 2451 which is a composite

4

EARTH COMPOSITION TYPE

 

 

 

 

4A

EARTH COMPOSITION TYPE

 

ENTER THE CODE BEST DESCRIBING THE BOTTOM COMPOSITION AT THE FORDING SITE, FOR EXAMPLE: "CLAY". See table 1289/7 which contains a list of data items and associated data codes

5

DEPTH OF WATER AT FORDING SITE

 

 

 

 

5A

DEPTH OF WATER AT FORDING SITE

 

ENTER THE DEEPEST DEPTH OF WATER AT THE FORDING SITE IN EITHER FEET OR METRES, FOR EXAMPLE: "3M". See table 2264 which is a composite

6

WATER SPEED

 

 

 

 

6A

WATER SPEED

 

ENTER THE SPEED OF THE WATER AT THE DEEPEST POINT IN THE FORDING SITE IN EITHER FEET PER SECOND OR METRES PER SECOND, FOR EXAMPLE: "30FPS". See table 2559 which is a composite

7

DAY-TIME AND MONTH OF OBSERVATION

 

 

 

 

7A

DAY-TIME AND MONTH OF OBSERVATION

 

ENTER TO DATE, TIME, TIME ZONE AND MONTH THAT THE WATER DEPTH AND SPEED SPECIFIED IN PREVIOUS TWO FIELDS WAS OBSERVED, FOR EXAMPLE: "272215ZJUN". See table 2030 which is a composite
