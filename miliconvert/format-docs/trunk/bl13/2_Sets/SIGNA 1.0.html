Concept/Explanation/Examples

1

ACOUSTIC SOURCE

 

 

 

 

1A

ACOUSTIC SOURCE

 

ENTER THE CODE GIVING THE ORIGIN OF AN ACOUSTIC SIGNAL, AS SIGNALLED BY INDIVIDUAL COMMAND, FOR EXAMPLE: "UNKNOWN". See table 1468/1 which is an instructive entry

2

FUNDAMENTAL FREQUENCY

 

 

 

 

2A

FUNDAMENTAL FREQUENCY

 

ENTER THE FUNDAMENTAL FREQUENCY, GIVEN AS FIVE TO EIGHT NUMERALS PLUS "HZ", 3 DECIMAL PLACES REQUIRED.  THIS REPRESENTS THE ACOUSTIC FREQUENCY TO THE NEAREST .001 HZ,  FOR EXAMPLE: "1250.100HZ".{THE FUDS IN THIS FFIRN HAVE BEEN RENUMBERED  BECAUSE TWO FUDS WERE MODIFIED AND MOVED UNDER A DIFFERENT COMPOSITE.} See table 2283 which is a composite

3

HARMONIC

 

 

 

 

3A

HARMONIC

 

ENTER ONE TO THREE NUMERAL HARMONIC COMBINATIONS, SEPARATED BY HYPHENS, LISTED IN DESCENDING ORDER OF PREDOMINANCE, FOR EXAMPLE: ""1-2-3". See table 1466/1 which is an instructive entry

4

REVOLUTIONS PER MINUTE

 

ENTER THE NUMBER OF TIMES AN OBJECT REVOLVES PER  MINUTE.

 

 

4A

REVOLUTIONS PER MINUTE

 

ENTER THE NUMBER OF TIUMES AN OBJECT REVOLVES PER MINUTE, FOR EXAMPLE: "10".{THIS FFIRN IS TO BE A REPLACEMENT FOR THE CONTENT IN FIELD 4 OF THE SIGNA SET.} See table 1778/1 a range [0 through 9999.0] (0 to 1 decimal places)

5

TURNS PER KNOT (TPK)

 

 

 

 

5A

TURNS PER KNOT (TPK)

 

ENTER THE TURNS PER KNOT, DECIMAL ALLOWED AND FOLLOWED BY "TPK",  FOR EXAMPLE: "12.4TPK". See table 2287 which is a composite

6

DAY-TIME

 

 

 

 

6A

DAY-TIME

 

ENTER THE  DAY-TIME OF THE EVENT, FOR EXAMPLE:"091359Z". See table 2000 which is a composite

7

MONTH NAME, ABBREVIATED

 

 

 

 

7A

MONTH NAME, ABBREVIATED

 

ENTER THE NAME OF A SPECIFIC MONTH WHICH IS ONE OF THE TWELVE PARTS INTO WHICH A YEAR IS DIVIDED AS DEFINED BY THE GREGORIAN CALENDAR, FOR EXAMPLE: "FEB". See table 1004/1 which contains a list of data items and associated data codes

8

SENSOR TYPE

 

 

 

 

8A

SENSOR TYPE

 

ENTER THE CODE TO DESIGNATE THE TYPE OF SENSOR, FOR EXAMPLE: "SONACT". See table 1434/1 which contains a list of data items and associated data codes

9

DETECTION STATUS

 

 

 

 

9A

DETECTION STATUS

 

ENTER THE CODE FOR THE STATUS OF THE DETECTION OF AN ACOUSTIC SIGNATURE, FOR EXAMPLE: "GAIN".{THIS IS A NEW FFIRN/FUD} See table 1777/1 which contains a list of data items and associated data codes

Notes:   

SIGNA SET IS USED IN CONJUNCTION WITH SUB, NAVAL, MERCH, FISHCTC OR UNK SET. A SEPARATE SIGNA SET MUST BE USED TO REPORT EACH DIFFERENT SOURCE CONTAINED IN A SINGLE CONTACT. DOUBLETS MUST BE REPORTED, FOR EXAMPLE, WITH TWO SIGNA SETS.
