Designator

Field Desc

Concept/Explanation/Examples

1

PRIMARY/ALTERNATE TARGET DESIGNATOR

 

PROVIDES INFORMATION WHETHER THE TARGET IS PRIMARY OR ALTERNATE.

 

 

1A

PRIMARY ALTERNATE TARGET DESIGNATOR

 

ENTER A DESIGNATOR THAT CLASSIFIES THE TARGET AS PRIMARY OR ALTERNATE, FOR EXAMPLE: "P". See table 1076/13 which contains a list of data items and associated data codes

2

TIME ON TARGET

 

IF USED, PROVIDES THE TIME ON TARGET.

 

 

2A

DAY-TIME AND MONTH TASKED ON TARGET

TOT

ENTER "TOT:" FOLLOWED BY THE DAY-TIME AND MONTH TASKED TO BE ON THE TARGET, FOR EXAMPLE: "TOT:141325ZFEB". See table 2030 which is a composite

 

 

2B

DAY-TIME ON TARGET TO THE SECOND, ZULU

TOTS

ENTER "TOTS:" FOLLOWED BY THE DAY-TIME IN SECONDS TO BE ON THE  TARGET, FOR EXAMPLE: "TOTS:14132535Z". See table 2483 which is a composite

3

NET TIME ON TARGET

 

REQUIRED ONLY IF THE TIME ON TARGET IS NOT PROVIDED AT FIELD 2, PROVIDES THE EARLIEST TIME ON TARGET.

 

 

3A

NOT EARLIER THAN DAY-TIME AND MONTH

NET

ENTER "NET:" FOLLOWED BY NOT EARLIER THAN DAY-TIME AND MONTH TO BE ON THE TARGET, FOR EXAMPLE: "NET:141320ZFEB". See table 2030 which is a composite

4

NLT TIME OFF TARGET

 

REQUIRED ONLY IF THE TIME ON TARGET IS NOT PROVIDED AT FIELD 2, PROVIDES THE LATEST DAY-TIME OFF THE TARGET.

 

 

4A

NOT LATER THAN DAY-TIME

NLT

ENTER "NLT:" FOLLOWED BY THE LATEST DAY-TIME AT WHICH A MISSION SHOULD BE OFF THE TARGET, FOR EXAMPLE: "NLT:141325Z". See table 2000 which is a composite

 

 

4B

DAY-TIME AND MONTH OFF TARGET

TOF

ENTER "TOF:" FOLLOWED BY THE LATEST DAY-TIME AND MONTH OFF THE TARGET, FOR EXAMPLE: "TOF:141325ZFEB". See table 2030 which is a composite

5

TARGET/FACILITY NAME

 

PROVIDES THE NAME OF THE FACILITY TARGET.

 

 

5A

TARGET FACILITY NAME

 

ENTER THE NAME OF THE TARGET FACILITY, FOR EXAMPLE: "JACKRIDER". See table 1022/234 which is an instructive entry

6

TARGET POSITION

 

THE GEOGRAPHIC LOCATION WHERE THE TARGET IS SITUATED.

 

 

6A

TARGET POSITION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE TARGET POSITION NAME, FOR EXAMPLE: "NAME:VARSANE". See table 1022/81 which is an instructive entry

 

 

6B

GEOGRAPHIC LOCATION, LAT/LONG, CENTISECONDS

LATCS

ENTER "LATCS:" FOLLOWED BY THE COORDINATES IN LAT/LONG, CENTISECONDS OF THE POSITION WHERE THE TARGET IS SITUATED, FOR EXAMPLE "LATCS:45201550N131182030E". See table 2488 which is a composite

 

 

6C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

7

TARGET TYPE

 

PROVIDES THE TARGET TYPE, EITHER SHIP OR NON-SHIP.

 

 

7A

NON-SHIP TARGET TYPE

NONSHIP

ENTER "NONSHIP:" FOLLOWED BY THE CODE FOR THE TYPE OF NON-SHIP TARGET, FOR EXAMPLE: "NONSHIP:BRIDGE". See table 1650/6 which contains a list of data items and associated data codes

 

 

7B

VESSEL TYPE

SHIP

ENTER "SHIP:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE TYPE OF SHIP OR VESSEL TARGET, FOR EXAMPLE: "SHIP:CG". See table 1166/1 which contains a list of data items and associated data codes

8

DMPI IDENTIFIER

 

IF USED, PROVIDES THE IDENTIFIER OF THE DESIRED MEAN POINT OF IMPACT (DMPI) OF WEAPONS ON TARGET.

 

 

8A

TARGET DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER

 

ENTER THE DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER, FOR EXAMPLE: "OCTOGON CAMP 54". See table 1416/7 which is an instructive entry

9

DMPI DESCRIPTION

 

IF USED, PROVIDES A DESCRIPTION OF THE DESIRED MEAN POINT OF IMPACT (DMPI) OF WEAPONS ON TARGET.

 

 

9A

DESIRED MEAN POINT OF IMPACT DESCRIPTION

 

ENTER THE PLAIN LANGUAGE DESCRIPTION OF THE DESIRED MEAN POINT OF IMPACT (DMPI), FOR EXAMPLE: "CENTER SPAN". See table 1363/44 which is an instructive entry

10

DESIRED MEAN POINT OF IMPACT

 

THE GEOGRAPHICAL LOCATION OF THE DESIRED MEAN POINT OF IMPACT (DMPI) OF WEAPONS ON TARGET.

 

 

10A

GEOGRAPHIC LOCATION, LAT/LONG, CENTISECONDS

DMPIC

ENTER "DMPIC:" FOLLOWED BY THE DESIRED MEAN POINT OF IMPACT IN LAT/LONG CENTISECONDS, FOR EXAMPLE: "DMPIC:35102530N079012545W". See table 2488 which is a composite

 

 

10B

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

11

GEODETIC DATUM

 

IF USED, PROVIDES THE GEOGRAPHIC DATUM CONCERNING THE TARGET.

 

 

11A

GEODETIC DATUM

 

IF THE DEFAULT GEODETIC DATUM SPECIFIED IN THE BEGINNING OF THE MESSAGE DOES NOT APPLY TO THE DMPI, ENTER THE CORRECT GEODATUM, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

12

PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR

 

USED ONLY IF THE DESIRED MEAN POINT OF IMPACT (DMPI) GEOGRAPHICAL COORDINATES ARE PROVIDED AT SEQ.10, PROVIDES THE PROBABLE DMPI COORDINATE HORIZONTAL CIRCULAR ERROR.

 

 

12A

PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR

 

ENTER THE PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR, FOR EXAMPLE: "9999.99M". See table 2532 which is a composite

13

DMPI ELEVATION

 

USED ONLY IF THE DESIRED MEAN POINT OF IMPACT (DMPI) GEOGRAPHICAL COORDINATES ARE PROVIDED AT SEQ.10, IT PROVIDES THE DMPI ELEVATION.

 

 

13A

DESIRED MEAN POINT OF IMPACT ELEVATION IN FEET OR METRES

 

ENTER THE ELEVATION OF THE DMPI ABOVE OR BELOW THE SEA LEVEL MEASURED IN FEET OR METRES ABOVE OR BELOW MEAN SEA LEVEL OR ABOVE THE REFERNCE ELLIPSOID, FOR EXAMPLE: "235FTMSL". See table 2492 which is a composite

14

PROBABLE COORDINATE ELEVATION LINEAR ERROR

 

USED ONLY IF THE DESIRED MEAN POINT OF IMPACT (DMPI)

ELEVATION IS PROVIDED AT SEQ.13, PROVIDES THE PROBABLE DMPI COORDINATE ELEVATION LINEAR ERROR.

 

 

14A

PROBABLE COORDINATE ELEVATION LINEAR ERROR

 

ENTER THE PROBABLE COORDINATE ELEVATION LINEAR ERROR, FOR EXAMPLE: "20.55FT". See table 2532 which is a composite

15

COMPONENT TARGET IDENTIFIER

 

IF USED, IT PROVIDES THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE REQUESTING SERVICE.

 

 

15A

COMPONENT TARGET IDENTIFIER

CTGT

ENTER "CTGT:" FOLLOWED BY THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE REQUESTING SERVICE, FOR EXAMPLE: "CTGT:A194761". See table 1416/5 which is an instructive entry

 

 

15B

TARGET ENTITY ID REFERENCE NUMBER

EIRN

ENTER "EIRN:" FOLLOWED BY THE EIRN NUMBER TO IDENTIFY A TARGET ON A VARIABLE MESSAGE INTERFACE (VMF) INTERFACE, FOR EXAMPLE: "EIRN:167772104294967290" See table 2085 which is a composite

16

TARGET PRIORITY

 

IF USED, IT PROVIDES THE CODE FOR THE PRIORITY ASSIGNED TO THE TARGET.

 

 

16A

TARGET PRIORITY, LITERAL

 

ENTER THE CODE FOR THE PRIORITY ASSIGNED TO THE TARGET, FOR EXAMPLE: "1A". See table 1076/11 which is an instructive entry

17

TARGET OBJECTIVE

 

IF USED, IT PROVIDES A PLAIN LANGUAGE DESCRIPTION OF THE GOAL TO BE ACHIEVED BY THE MISSION HITTING THE TARGET.

 

 

17A

TARGET OBJECTIVE

 

ENTER THE TARGET OBJECTIVE, FOR EXAMPLE:  "DESTROY". See table 1456/3 which is an instructive entry

18

LINK 16 TRACK NUMBER REFERENCE

 

IF USED, IT PROVIDES THE UNIQUE LINK16 FIVE CHARACTERS IDENTIFIER ASSIGNED TO THE TARGET.

 

 

18A

LINK 16 TRACK NUMBER REFERENCE FOR PRE-PLANNED TARGET

 

ENTER THE UNIQUE LINK 16 IDENTIFIER ASSIGNED TO PRE- PLANNED TARGET, FOR EXAMPLE: "AA234". See table 1625/27 which is an alphanumeric range

19

ADDITIONAL TARGET IDENTIFICATION FEATURES

 

IF USED, IT PROVIDES ADDITIONAL AND ESSENTIAL IDENTIFYING INFORMATION ABOUT THE TARGET.

 

 

19A

ADDITIONAL TARGET IDENTIFICATION FEATURES

 

ENTER ADDITIONAL IDENTIFYING INFORMATION ABOUT THE GROUND TARGET, FOR EXAMPLE: "THIRD ARCH". See table 1357/23 which is an instructive entry
