FRIENDLY AIRCRAFT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE MISSION AIRCRAFT AT THE TIME OF OBSERVATION.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

DIRECTION OF MOVEMENT

11A

DIRECTION OF MOVEMENT IN DEGREES, WITH REFERENCE

DIR

ENTER "DIR:", FOLLOWED BY THE DIRECTION OF MOVEMENT OF ENTER "DIR:" FOLLOWED BY THE ELEMENT OBJECTS SIGHTED DURING MISSION OPERATIONS, EXPRESSED IN DEGREES WITH THE INDICATION WHETHER TRUE, GRID, MAGNETIC OR RELATIVE, FOR EXAMPLE: "DIR:140T". See table 2065 which is a composite
