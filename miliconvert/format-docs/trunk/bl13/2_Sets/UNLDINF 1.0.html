1

PORTION OF LOADED CARGO

 

 

 

 

1A

PORTION OF LOADED CARGO

 

ENTER A LITERAL TO IDENTIFY A SPECIFIC PORTION OF CARGO, FOR EXAMPLE: "ALL SUPPLY ELEMENTS." THIS IDENTIFIER IS NEEDED TO TRACE PORTIONS OF AN ENTITY TO BE TRANSPORTED AND ELEMENTS OF WHICH MIGHT BE LOADED AND/OR UNLOADED IN DIFFERENT LOCATIONS. See table 1203/4 which is an instructive entry

2

LOCATION NAME

 

 

 

 

2A

LOCATION NAME

 

ENTER A LITERAL GIVING THE PREFERRED UNLOADING LOCATION, FOR EXAMPLE: "FRANKFURT GUETERBHF". See table 1022/4 which is an instructive entry

3

GEOGRAPHIC LOCATION

 

THE GEOGRAPHIC LOCATION OF CARGO.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

3C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

3D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

3E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

3F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

4

CONVEYANCE INDICATOR

 

 

 

 

4A

CONVEYANCE INDICATOR

 

ENTER AN ALPHABETIC INDICATOR THAT IS CLEARLY LINKED TO THE CONVEYANCE THAT IS DESCRIBED IN THE FOLLOWING FIELD, FOR EXAMPLE: "A". See table 1895/1 which is an alphanumeric range

5

CONVEYANCE

 

 

 

 

5A

CONVEYANCE

 

ENTER A LITERAL TO IDENTIFY THE TYPE OF CONVEYANCE, FOR EXAMPLE: "SPECIAL TRAIN". See table 1269/6 which is an instructive entry

6

HOURS REQUIRED FOR UNLOADING

 

 

 

 

6A

HOURS REQUIRED FOR UNLOADING

 

ENTER THE TIME IN HOURS WHICH IS NEEDED FOR UNLOADING, FOR EXAMPLE: "24". See table 1047/29 a range [1 through 999]

7

EARLIEST ARRIVAL DTG

 

 

 

 

7A

EARLIEST ARRIVAL DTG

 

ENTER THE DTG WHEN THE TRANSPORT IS TO ARRIVE AT THE EARLIEST, FOR EXAMPLE: "091230ZAPR2001". See table 2033 which is a composite

8

LATEST ARRIVAL DTG

 

 

 

 

8A

LATEST ARRIVAL DTG

 

ENTER THE DTG WHEN THE TRANSPORT IS TO ARRIVE AT THE LATEST, FOR EXAMPLE: "092030ZAPR2001". See table 2033 which is a composite
