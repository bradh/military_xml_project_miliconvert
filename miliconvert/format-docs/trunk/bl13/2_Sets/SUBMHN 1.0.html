Concept/Explanation/Examples

1

HAVEN IDENTIFIER

 

IDENTIFY THE SUBMARINE HAVEN PHONETIC IDENTIFIERS (ALTERNATIVES A-C) OR AS DUAL TRACK IDENTIFIERS (ALT D AND E).

 

 

1A

PHONETIC ALPHABETIC OR NUMERIC IDENTIFIER

 

THE AREA DESIGNATOR GIVEN AS A PHONETIC LETTER OR A NUMBER, FOR EXAMPLE: "ALFA" OR "THREE". See table 1128/7 which contains a list of data items and associated data codes

 

 

1B

PHONETIC ALPHABETIC-NUMERIC IDENTIFIER

 

THE AREA DESIGNATOR GIVEN AS A PHONETIC ALPHABETIC-NUMERIC COMBINATION, FOR EXAMPLE: "ALFA ONE". See table 2406 which is a composite

 

 

1C

PHONETIC IDENTIFIER

 

THE AREA DESIGNATOR GIVEN AS A PHONETIC NUMERIC-ALPHABETIC COMBINATION, FOR EXAMPLE: "ONE ALFA". See table 2407 which is a composite

 

 

1D

DUAL SUBMARINE TRACK IDENTIFIERS

 

THAT PART OF THE SUBMARINE TRACK FOR WHICH THE MOVING HAVEN IS APPLICABLE, POINT TO POINT ROUTING IS TO BE USED FOR THIS ALTERNATIVE.  THE START AND END POINTS OF THE TRACK ON WHICH THE HAVEN IS CONSTRUCTED ARE TO BE LISTED AND SEPARATED BY A HYPHEN, FOR EXAMPLE: "OC-OD". See table 2198 which is a composite

2

SIZE OF SUBMARINE HAVEN

 

SPECIFY THE SIZE OF THE MOVING SUBMARINE HAVEN AS FOLLOWS:

 

 

2A

RECTANGULAR MOVING HAVEN DIMENSIONS, NM, VERIFIED

 

LENGTH AND WIDTH OF A RECTANGLE IN RELATION TO THE SUBMARINE'S POSITION AND INTENDED MOVEMENT (PIM) VERIFIED: THE FIRST NUMBER OF THE MOVING HAVEN IS LENGTH AHEAD OF THE SUBMARINES POSITION, THE SECOND NUMBER IS THE LENGTH ASTERN OF THE POSITION, AND THE THIRD NUMBER IS THE WIDTH EITHER SIDE OF THE SUBMARINE TRACK; NUMBERS ARE SEPARATED BY A LETTER "X" AND THE LAST NUMBER FOLLOWED BY "NM", FOR EXAMPLE: "20X30X5NM0". See table 2199 which is a composite

 

 

2B

NON-DISCRETE INDICATOR, AS REQUIRED

 

ENTER THE WORDS "AS REQD", WHEN THE MOVEMENT IS SPECIFIED ELSEWHERE (E.G OPORD WPP OR NOI), IN WHICH CASE THE SET SHOULD BE FOLLOWED BY AN AMPN SET SPECIFYING THE REFERENCE DOCUMENT OR MESSAGE.  "AS REQD" MAY ALSO BE USED FOR RELATIVELY SHORT TRANSITS IN CONFINED COASTAL WATERS, FOR EXAMPLE: "AS REQD". See table 1084/2 which contains a list of data items and associated data codes

Notes:   

1. THE TIME IN THE PRECEDING SUBTRK OR SUBTRKST SET INDICATES WHEN THE HAVEN IS EFFECTIVE.2. A SUBMHN SET CAN BE PRECEDED BY MORE THAN ONE SUBTRK SET, IN WHICH CASE THE SECOND AND SUBSEQUENT POSITIONS ARE IMMEDIATE (VIA) POSITIONS ALONG THE TRACK OF THE HAVEN, WHERE THE SIZE, SPEED OF ADVANCE AND MODE OF OPERATION DO NOT CHANGE.3. DO NOT USE MORE THAN ONE SUBMHN SET WITHIN THE SUBTRKST SEGMENT.
