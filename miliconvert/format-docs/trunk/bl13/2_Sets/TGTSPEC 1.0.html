Concept/Explanation/Examples

1

MAIN DIRECTION

 

IF USED, ENTER MAIN DIRECTION OF TARGET OR EQUIPMENT MOVEMENT IN ONE OF THE FOLLOWING WAYS:

 

 

1A

TARGET COURSE IN DEGREES

 

ENTER THE TARGET COURSE IN DEGREES [000-359] FOLLOWED BY A ONE LETTER CODE INDICATING THE ANGULAR MEASUREMENT REFERENCE, FOR EXAMPLE: "180T". See table 2065 which is a composite

 

 

1B

DIRECTION OF MOVEMENT (COMPASS POINT)

 

ENTER THE DIRECTION OF MOVEMENT (COMPASS POINT), FOR EXAMPLE: "SSW". See table 1161/1 which contains a list of data items and associated data codes

2

SPEED QUALIFIER

 

 

 

 

2A

SPEED QUALIFIER

 

ENTER DESCRIPTION WHICH QUALIFIES SPEED OF THE TARGET, FOR EXAMPLE: "SLOW". See table 1672/1 which contains a list of data items and associated data codes

3

DESCRIPTION OF MOVEMENT PATTERN

 

 

 

 

3A

DESCRIPTION OF MOVEMENT PATTERN

 

ENTER PATTERN TYPE OF MOVEMENT AS A LITERAL [1-20X], FOR EXAMPLE: "REGULAR". See table 1363/38 which is an instructive entry

4

DESCRIPTION OF CAMOUFLAGE EMPLOYED

 

 

 

 

4A

DESCRIPTION OF CAMOUFLAGE EMPLOYED

 

ENTER DESCRIPTION OF CAMOUFLAGE EMPLOYED BY TARGET, LITERAL ENTRY [1-20X], FOR EXAMPLE: "FOLIAGE". See table 1363/36 which is an instructive entry

5

DESCRIPTION OF DECEPTION EMPLOYED

 

 

 

 

5A

DESCRIPTION OF DECEPTION EMPLOYED

 

ENTER DESCRIPTION OF DECEPTION EMPLOYED BY TARGET, LITERAL ENTRY [1-20X], FOR EXAMPLE: "DISRUPTIVE". See table 1363/37 which is an instructive entry

6

TYPE OF CONCEALMENT

 

 

 

 

6A

TYPE OF CONCEALMENT

 

ENTER TYPE OF CONCEALMENT EMPLOYED, LITERAL ENTRY [1-30X], FOR EXAMPLE: "STRUCTURAL". See table 1668/3 which is an instructive entry

7

DEGREE OF TARGET MASKING

 

 

 

 

7A

DEGREE OF TARGET MASKING

 

ENTER DEGREE THAT TARGET IS MASKED, FOR EXAMPLE: "P". See table 1668/4 which contains a list of data items and associated data codes

8

KEYPOINT QUALITY INDICATOR

 

 

 

 

8A

KEYPOINT QUALITY INDICATOR

 

ENTER A "YES" OR "NO" TO INDICATE IF DAMAGE TO OR DESTRUCTION OF THE TARGET ELEMENT IN FIELD 1 OF SET RECERES WILL SEVERELY HAMPER OPERATIONS. See table 1027/54 which contains a list of data items and associated data codes

9

EQUIPMENT OR MATERIAL LOADED

 

IF USED, ENTER ONE OF THE FOLLOWING ALTERNATIVES FOR EQUIPMENT OR MATERIAL LOADED, REPEATING AS NECESSARY:

 

 

9A

QUANTITY EQUIPMENT LOADED

 

ENTER THE NUMBER OF ITEMS [1-9999], HYPHEN, AND EQUIPMENT TYPE [LITERAL ENTRY, 1-20X],  FOR EXAMPLE: "5-TANKS". See table 2413 which is a composite

 

 

9B

QUANTITY/UNIT - MATERIAL LOADED

MAT

ENTER "MAT:" FOLLOWED BY QUANTITY [1-999], QUANTITY UNIT OF MEASUREMENT [1-3A], HYPHEN, AND DESCRIPTION OF THE MATERIAL LOADED [LITERAL ENTRY, 1-30X], FOR EXAMPLE: "MAT:250TNE-COAL". See table 2414 which is a composite
