Concept/Explanation/Examples

1

TYPE OF ZONE AID

 

 

 

 

1A

TYPE OF ZONE AID

 

ENTER THE ZONE RECOGNITION MARKING AID TYPE, FOR EXAMPLE: "RECOGN". See table 1785/1 which contains a list of data items and associated data codes

2

ZONE AID DEVICE

 

 

 

 

2A

ZONE AID DEVICE

 

ENTER THE TYPE OF ZONE AID DEVICE THAT IS MARKING THE AREA, FOR EXAMPLE: "FLARE". See table 1613/3 which contains a list of data items and associated data codes

3

ZONE AID IDENTIFIER (COLOR/FREQUENCY)

 

ENTER THE ZONE AID DEVICE COLOR OR RADIO FREQUENCY IN THE FOLLOWING WAY:

 

 

3A

ZONE AID COLOR

 

THE COLOR OF THE ZONE AID DEVICE, FOR EXAMPLE: "ORANGE". See table 1246/59 which contains a list of data items and associated data codes

 

 

3B

ZONE AID PRIMARY FREQUENCY IN MEGAHERTZ

 

ENTER "MHZ:" AND THE RADIO FREQUENCY TO COVER HF, VHF AND UHF BANDS (2.0 TO 400MHZ), FOR EXAMPLE: "MHZ:197.5". See table 1068/9 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

4

LOCATION OF RECOGNITION DEVICE

 

THE GEOGRAPHIC LOCATION OF THE RECOGNITION DEVICE.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

MAGNETIC BEARING AND DISTANCE FROM ZONE FIXED POSITION

BAR

ENTER "BAR:" FOLLOWED BY THE MAGNETIC BEARING AND DISTANCE, FOR EXAMPLE: "BAR:045-100M". See table 2654 which is a composite

 

 

Structured Notation:

[^]F4 RP ([^]F1 EQ /RECOGN/)

 

 

Structured Language:      

FIELD 4 IN SET ^ (ZONEAID) IS REQUIRED IF FIELD 1 IN SET ^ (ZONEAID) EQUALS /RECOGN/, OTHERWISE IT IS PROHIBITED.
