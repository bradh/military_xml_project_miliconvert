1

NATO SUBMARINE DEPLOYMENT DESIGNATOR

 

THE NATO DESIGNATOR ASSIGNED TO A SUBMARINE DEPLOYMENT.

 

 

1A

NATO SUBMARINE DEPLOYMENT DESIGNATOR

 

ENTER THE NATO SUBMARINE DEPLOYMENT (DSUB) DESIGNATOR, FOR EXAMPLE: "012". See table 1214/1 a range [001 through 599]

 

 

1B

NATO SUBMARINE DEPLOYMENT DESIGNATOR AND 4-DIGIT YEAR

 

ENTER THE NATO SUBMARINE DEPLOYMENT (DSUB) DESIGNATOR AND YEAR, FOR EXAMPLE: "012-2002". See table 2040 which is a composite

 

 

1C

NATO EXERCISE SUBMARINE DEPLOYMENT DESIGNATOR

 

ENTER THE NATO EXERCISE SUBMARINE DEPLOYMENT (DSUB) DESIGNATOR, FOR EXAMPLE: "X012". See table 2081 which is a composite

 

 

1D

NATO EXERCISE SUB DEPLOYMENT DESIGNATOR AND 4-DIGIT YEAR

 

ENTER THE NATO EXERCISE SUBMARINE DEPLOYMENT (DSUB) DESIGNATOR AND YEAR, FOR EXAMPLE: "X012-2002". See table 2035 which is a composite

2

SUBMARINE TYPE

 

 

 

 

2A

SUBMARINE TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE SUBMARINE TYPE, FOR EXAMPLE: "SSG". See table 1166/2 which contains a list of data items and associated data codes

3

NATIONALITY

 

 

 

 

3A

GEOGRAPHICAL ENTITY

 

ENTER THE NATIONALITY OF THE DESIGNATED SUBMARINE, FOR EXAMPLE: "RS". See table 1026/1 which contains a list of data items and associated data codes

4

SUBMARINE CLASSIFICATION/CONFIDENCE LEVEL

 

 

 

 

4A

SUBMARINE CLASSIFICATION CONFIDENCE LEVEL

 

ENTER THE SUBMARINE CLASSIFICATION/CONFIDENCE LEVEL, FOR EXAMPLE: "POSS2". See table 1123/5 which contains a list of data items and associated data codes

5

SUBMARINE CLASS IDENTIFIER

 

THE IDENTIFICATION OF THE SUBMARINE CLASS OR, IF THE CLASS IS UNKNOWN, OTHER AMPLIFYING INFORMATION.

 

 

5A

SUBMARINE CLASS NAME

CLASS

ENTER FIELD DESCRIPTOR "CLASS:" FOLLOWED BY THE SUBMARINE CLASS NAME, FOR EXAMPLE: "CLASS:KILO". See table 1038/1 which contains a list of data items and associated data codes

 

 

5B

SUBMARINE CLASS DESIGNATOR

SCD

ENTER FIELD DESCRIPTOR "SCD:" FOLLOWED BY THE SUBMARINE CLASS DESIGNATOR, FOR EXAMPLE: "SCD:V". See table 1338/1 which is an instructive entry

 

 

5C

SUBMARINE PROPULSION TYPE

PROPUL

ENTER FIELD DESCRIPTOR "PROPUL:" FOLLOWED BY THE SUBMARINE PROPULSION TYPE, FOR EXAMPLE: "PROPUL:NUC2-3". See table 1339/1 which contains a list of data items and associated data codes

 

 

5D

UNKNOWN SUBMARINE CONTACT

 

ENTER AN UNKNOWN SUBMARINE CONTACT AS "UNKNOWN". See table 1139/3 which contains a list of data items and associated data codes

6

LOCATION

 

THE GEOGRAPHIC LOCATION OF THE DESIGNATED SUBMARINE.

 

 

6A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

6B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

6C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

6D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

6E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

6F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

6G

PLACE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "NAME:SHETLANDS". See table 1022/170 which is an instructive entry

7

TIME OF CONTACT

 

THE DATE AND/OR TIME AT WHICH THE DESIGNATED CONTACT WAS AT THE SPECIFIED POSITION.

 

 

7A

DAY-TIME OF CONTACT

 

ENTER THE DAY-TIME OF CONTACT, FOR EXAMPLE: "162215Z". See table 2000 which is a composite

 

 

7B

DAY-MONTH NAME

 

IF EXACT TIME OF CONTACT IS UNKNOWN, ENTER DAY-MONTH NAME, FOR EXAMPLE: "16SEP". See table 2067 which is a composite

 

 

7C

DTG

 

ENTER THE DTG OF CONTACT, FOR EXAMPLE "162215ZSEP2002". See table 2033 which is a composite

8

CONTACT MOVEMENT

 

THE OBSERVED OR PREDICTED MOVEMENT OF THE CONTACT.

 

 

8A

COURSE IN DEGREES TRUE AND SPEED IN KNOTS

 

ENTER THE COURSE IN DEGREES TRUE, FOLLOWED BY A HYPHEN, FOLLOWED BY THE SPEED IN KNOTS, FOR EXAMPLE: "265-13". See table 2025 which is a composite

 

 

8B

MOVEMENT ACTIVITY

 

ENTER THE CONTACT MOVEMENT ACTIVITY, FOR EXAMPLE: "DIW". See table 1095/4 which contains a list of data items and associated data codes

9

CONTACT VALIDITY

 

 

 

 

9A

CONTACT VALIDITY

 

ENTER THE CODE SPECIFYING THE VALIDITY OF THE CONTACT REPORT, FOR EXAMPLE: "CB". See table 2049 which is a composite

10E
