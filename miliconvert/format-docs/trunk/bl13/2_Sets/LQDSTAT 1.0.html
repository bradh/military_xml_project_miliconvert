Concept/Explanation/Examples

1

UNIT

 

 

 

 

1A

UNIT

 

ENTER, AS A LITE4RAL, THE NAME OF UNIT FOR WHICH LIQUID(S) STATUS IS BEING REPORTED, FOR EXAMPLE: "AJAX". See table 1022/57 which is an instructive entry

2

EFFECTIVE DAY-TIME AND MONTH

 

 

 

 

2A

EFFECTIVE DAY-TIME AND MONTH

 

ENTER THE DAY-TIME AND MONTH THAT THE REPORTED LIQUID STATUS IS EFFECTIVE, FOR EXAMPLE: "230925Z". See table 2030 which is a composite

3

TYPE OF LIQUID

 

GIVES THE TYPE OF LIQUID BEING REPORTED IN ONE OF THE TWO FOLLOWING WAYS:

 

 

3A

NATO STANDARD FUEL TYPE

 

ENTER THE TYPE OF FUEL, FOR EXAMPLE: "F54". See table 1172/1 which contains a list of data items and associated data codes

 

 

3B

WATER TYPE

 

ENTER, AS A LITERAL, THE TYPE OF WATER, FOR EXAMPLE: "FRESHWATER". See table 1513/1 which contains a list of data items and associated data codes

4

PERCENTAGE OF TOTAL CAPACITY HELD

 

 

 

 

4A

PERCENTAGE OF TOTAL CAPACITY HELD

 

ENTER THE PERCENTAGE OF TOTAL CAPACITY OF A REPORTED LIQUID REMAINING ON BOARD, FOR EXAMPLE: "80". See table 1064/6 a range [0 through 100]

5

PERCENTAGE CONSUMED DAILY

 

 

 

 

5A

PERCENTAGE CONSUMED DAILY

 

ENTER THE PERCENTAGE OF REPORTED LIQUID CONSUMED PER DAY, FOR EXAMPLE: "10". See table 1064/13 a range [0 through 99]
