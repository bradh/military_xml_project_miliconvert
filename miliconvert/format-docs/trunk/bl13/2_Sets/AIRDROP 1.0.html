No

Designator

Field Desc

Concept/Explanation/Examples

1

DROP ZONE ELEVATION

 

 

 

 

1A

DROP ZONE ELEVATION IN FEET

 

ENTER THE HIGHEST TERRAIN ELEVATION ON THE DROP ZONE IN FEET, FOR EXAMPLE: "125". See table 1799/2 a range [-1290 through 29028]

2

AIRDROP LOCATION

 

THE GEOGRAPHICAL LOCATION OF THE AIRDROP.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE AIRDROP LOCATION IN GEOREF, CENTIMINUTE, FOR EXAMPLE: "GEO:AABB15233527". See table 2018 which is a composite

3

GEODETIC DATUM

 

 

 

 

3A

GEODETIC DATUM

 

IF THE DEFAULT GEODETIC DATUM SPECIFIED IN THE BEGINNING OF THE MESSAGE DOES NOT APPLY TO THE LOCATION OF THE AIRDROP, ENTER THE CORRECT GEODATUM, FOR EXAMPLE: "NAS-D". See table 1195/1 which contains a list of data items and associated data codes

4

RUN IN HEADING IN DEGREES MAGNETIC

 

 

 

 

4A

RUN IN HEADING IN DEGREES MAGNETIC

 

ENTER THE MAGNETIC HEADING FOR THE AIRCRAFT WHEN APPROACHING THE DROP ZONE, FOR EXAMPLE: "235". See table 1804/4 a range [000 through 359]

5

DISTANCE BETWEEN LEADING EDGE AND IMPACT POINT

 

 

 

 

5A

DISTANCE BETWEEN LEADING EDGE AND IMPACT POINT IN METRES

 

ENTER THE DISTANCE FROM THE LEADING EDGE OF THE DROP ZONE TO THE POINT OF IMPACT, FOR EXAMPLE: "500". See table 1052/3 a range [0 through 99999]

6

DROP ZONE LENGTH

 

 

 

 

6A

DROP ZONE LENGTH IN METRES

 

ENTER THE LENGTH OF THE DROP ZONE, FOR EXAMPLE: "1500". See table 1657/9 a range [1 through 9999]

7

DROP ZONE WIDTH

 

CAN ONLY BE USED IN CONJUNCTION WITH DROP ZONE LENGTH SPECIFIED IN THIS SET.

 

 

7A

DROP ZONE WIDTH IN METRES

 

ENTER THE WIDTH OF THE DROP ZONE IN METRES, FOR EXAMPLE: "500". See table 1658/9 a range [1 through 9999]

8

MARKING METHOD

 

 

 

 

8A

METHOD OF MARKING FRIENDLY POSITIONS

 

ENTER THE METHOD FOR MARKING THE DROP ZONE, FOR EXAMPLE: "PANELS". See table 1613/2 which contains a list of data items and associated data codes

9

COLOUR

 

CAN ONLY BE USED IN CONJUNCTION WITH MARKING METHOD SPECIFIED IN THIS SET.

 

 

9A

COLOUR

 

ENTER THE COLOR OF THE MARKINGS, FOR EXAMPLE: "ORANGE". See table 1138/1 which contains a list of data items and associated data codes
