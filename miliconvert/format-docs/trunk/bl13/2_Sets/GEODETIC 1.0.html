INTELLIGENCE COORDINATES

 

THE GEOGRAPHIC LOCATION OF THE DMPI.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

UTM

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

UPS

MUPS

1F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

2

COORDINATES BASIS

 

 

 

 

2A

COORDINATES BASIS

 

ENTER A CODE TO INDICATE THE PARTICULAR INTELLIGENCE ENTITY TO WHICH THE COORDINATES THAT ARE GIVEN REFER, FOR EXAMPLE: "GO". IT MUST BE UNDERSTOOD THAT, EVEN THOUGH THE COORDINATES GIVEN MAY BE FOR A FACILITY ENTITY, THE BASIS FOR THE COORDINATES MAY ACTUALLY BE THE INSTALLATION.  See table 1385/2 which contains a list of data items and associated data codes

3

COORDINATES DATE-TIME (Z), YYYYMMDDHHMMSS

 

 

 

 

3A

COORDINATES DATE-TIME (Z), YYYYMMDDHHMMSS

 

ENTER A 14 DIGIT DATE (Z) OF THE STRUCTURE YYYYMMDDHHMMSS INDICATING THE DATE-TIME ON WHICH A SPECIFIC COORDINATE WAS REPORTED OR DEVELOPED, FOR EXAMPLE: "19990710163435". See table 2417 which is a composite

4

COORDINATES DATUM

 

 

 

 

4A

COORDINATES DATUM

 

ENTER THE DATUM THAT IS USED IN THE PRODUCTION OF THE BASE REFERENCE MAP OR THE COORDNATE ITSELF, FOR EXAMPLE: "WGA". See table 1195/2 which contains a list of data items and associated data codes

5

METHOD OF COORDINATES DERIVATION

 

 

 

 

5A

METHOD OF COORDINATES DERIVATION

 

ENTER A CODE TO IDENTIFY THE METHOD USED TO DETERMINE OR DERIVE THE COORDINATE, FOR EXAMPLE: "PM".  See table 1866/5 which contains a list of data items and associated data codes

6

COORDINATES DERIVATION ACCURACY

 

 

 

 

6A

COORDINATES DERIVATION ACCURACY

 

ENTER THE  PLUS OR MINUS ERROR (ABSOLUTE VALUE OF THE ERROR) ASSESSED AGAINST THE METHOD USED TO  DERIVE THE COORDINATE, FOR EXAMPLE: "10". See table 1811/4 a range [0 through 9999]

7

UNIT OF LINEAR MEASUREMENT

 

 

 

 

7A

UNIT OF LINEAR MEASUREMENT

 

ENTER THE UNIT OF MEASURE USED TO EXPRESS COORD DERIVATION ACCURACY, FOR EXAMPLE: "M". See table 1008/4 which contains a list of data items and associated data codes

8

COORDINATES RADIUS OF ACCURACY

 

 

 

 

8A

COORDINATES RADIUS OF ACCURACY

 

ENTER THE RADIUS OF THE CIRCLE THAT THE COORDINATES ARE CONTAINED WITHIN, FOR EXAMPLE: "370-M". See table 2170 which is a composite

9

COORDINATES RADIUS OF ACCURACY CONFIDENCE LEVEL

 

 

 

 

9A

COORDINATES RADIUS OF ACCURACY CONFIDENCE LEVEL

 

ENTER THE CONFIDENCE LEVEL FOR THE COORDINATES RADIUS OF ACCURACY EXPRESSED AS A PERCENTAGE, FOR EXAMPLE: "99".  See table 1123/10 a range [00 through 99]

10

GEODETIC INFORMATION PRODUCER IDENTIFIER

 

 

 

 

10A

GEODETIC INFORMATION PRODUCER IDENTIFIER

 

ENTER A CODE IDENTIFYING THE PRODUCER OF THE GEODETIC INFORMATION, FOR EXAMPLE: "CL". See table 1012/331 which is an alphanumeric range
