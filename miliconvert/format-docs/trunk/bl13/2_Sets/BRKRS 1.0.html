Concept/Explanation/Examples

1

SURF LOCATION

 

 

 

 

1A

SURF LOCATION

 

ENTER THE DIRECTION OF WAVEBREAK AS OBSERVED FROM SEAWARD, FOR EXAMPLE: "RF". See table 1118/2 which contains a list of data items and associated data codes

2

WAVE/BREAKER ANGLE

 

 

 

 

2A

WAVE BREAKER ANGLE

 

ENTER THE ACUTE ANGLE, INDEGREES, FORMED BETWEEN THE BREAKER LINES AND THE SHORELINE, FOR EXAMPLE: "30". See table 1043/21 a range [0 through 90]

3

SURF ZONE WIDTH (FEET, YARDS OR METRES)

 

 

 

 

3A

SURF ZONE WIDTH (FEET, YARDS OR METRES)

 

ENTER THE SURF ZONE WIDTH, THE DISTANCE FROM THE OUTERMOST BREAKER TO THE EXTREME UPRUSH OF WATER ON THE BEACH, MEASURED IN FEET, YARDS OR METRES, FOR EXAMPLE: "75M". See table 2451 which is a composite

4

NUMBER OF WAVES IN SURF ZONE

 

 

 

 

4A

NUMBER OF WAVES IN SURF ZONE

 

ENTER THE NUMBER OF WELL DEFINED BREAKERS IN THE SURF ZONE, FOR EXAMPLE: "7". See table 1044/63 a range [0 through 99]

5

SIGNIFICANT WAVE/BREAKER HEIGHT IN FEET OR METRES

 

 

 

 

5A

SIGNIFICANT WAVE/BREAKER HEIGHT IN FEET OR METRES

 

ENTER THE AVERAGE HEIGHT OF THE HIGHEST ONE THIRD OF ALL BREAKERS OBSERVED IN A 10 MINUTE PERIOD, TO THE NEAREST ONE HALF FOOT OR 1/10 METRE, FOR EXAMPLE: "4.5FT". See table 2478 which is a composite

6

MAXIMUM WAVE/BREAKER HEIGHT IN FEET OR METRES

 

 

 

 

6A

MAXIMUM WAVE/BREAKER HEIGHT IN FEET OR METRES

 

ENTER THE LARGEST BREAKER OBSERVED IN FEET OR METRES, FOR EXAMPLE: "15.5FT". See table 2478 which is a composite

7

WAVE PERIOD IN SECONDS AND HALF SECONDS

 

 

 

 

7A

WAVE PERIOD IN SECONDS AND HALF SECONDS

 

ENTER THE TIME PERIOD IN SECONDS BETWEEN BREAKERS TO THE NEAREST HALF SECOND, FOR EXAMPLE: "7.5". See table 2104 which is a composite

8

BREAKER TYPE

 

 

 

 

8A

BREAKER TYPE

 

ENTER THE TYPE OF WAVE BREAKING ACTION OBSERVED IN THE SURF ZONE, FOR EXAMPLE: "SURGE". See table 1776/1 which contains a list of data items and associated data codes

9

PERCENTAGE OF OBSERVED BREAKERS

 

 

 

 

9A

PERCENTAGE OF OBSERVED BREAKERS

 

ENTER THE PERCENTAGE OF BREAKERS IN THE SURF ZONE OF THE TYPE INDICATED ABOVE.  REPEATED WITH EACH BREAKER TYPE, FOR EXAMPLE: "30". See table 1064/47 a range [1 through 100]
