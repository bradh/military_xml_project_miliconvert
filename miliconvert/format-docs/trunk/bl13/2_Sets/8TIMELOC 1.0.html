REFERENCE NUMBER

 

ENTER THE REFERENCE NUMBER OF THE MISSION [LITERAL], FOR EXAMPLE: "AF234". THE REFERENCE NUMBER ENABLES ASSOCIATING OF RELATED DATA, E.G. CONTAINED IN SET 8CHAFF. See table 1348/2 which is an instructive entry

GEOGRAPHICAL ENTITY
