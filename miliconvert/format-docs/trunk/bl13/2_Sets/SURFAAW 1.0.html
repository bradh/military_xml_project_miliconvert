Designator

Field Desc

Concept/Explanation/Examples

1

BASE OR UNIT NAME

 

THE NAME OF THE BASE OR UNIT CHARGED WITH AAW DUTIES.

 

 

1A

BASE NAME

BASE

ENTER "BASE:" FOLLOWED BY THE LITERAL BASE NAME, FOR EXAMPLE: "BASE:LAJES". See table 1022/104 which is an instructive entry

 

 

1B

UNIT

 

ENTER THE LITERAL NAME OF THE UNIT OR SHIP, FOR EXAMPLE: "BIDDLE". See table 1022/57 which is an instructive entry

2

STATION

 

THE GEOGRAPHIC LOCATION OF THE STATION FROM WHICH THE AAW ASSIGNMENT IS TO BE CONDUCTED.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE GEOGRAPHIC LOCATION OF THE STATION AS GEOREF, CENTIMINUTE, FOR EXAMPLE:"GEO:KQAA24151224". See table 2018 which is a composite

 

 

2H

STATION ASSIGNMENT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE STATION ASSIGNMENT NAME, FOR EXAMPLE: "NAME:STATION3". See table 1022/124 which is an instructive entry

 

 

2I

BEARING AND DISTANCE IN NM FROM REFERENCE POINT

BARN

ENTER "BARN:" FOLLOWED BY THE GEOGRAPHIC LOCATION OF THE STATION AS AS BEARING AND DISTANCE FROM A REFERENCE POINT, FOR EXAMPLE: "BARN:270T-ZZ-50NM". See table 2307 which is a composite

3

ANTI-AIR WARFARE CALL SIGN

 

 

 

 

3A

ANTI-AIR WARFARE CALL SIGN

 

ENTER A ONE OR TWO LETTER AAW CALL SIGN FOR THE UNIT SPECIFIED IN FIELD 1, FOR EXAMPLE: "AD". See table 1129/8 which is an alphanumeric range
