1

ZIPPO GROUP

 

 

 

 

1A

ZIPPO GROUP

 

ENTER THE ZIPPO GROUP TO WHICH THE THREAT MISSILE SPECIFIED IN FIELD 2 IS ASSIGNED, FOR EXAMPLE: "2". See table 1256/1 which is an alphanumeric range

2

THREAT MISSILE

 

SPECIFIES THE THREAT MISSILE ASSIGNED TO THE PRECEDING ZIPPO GROUP AND FOR WHICH THE ZIPPO REACTIONS IN FIELD 3 ARE ALLOCATED.

 

 

2A

THREAT MISSILE, SPECIFIC

 

ENTER THE SPECIFIC THREAT MISSILE, FOR EXAMPLE: "HY 2A". See table 1407/37 which contains a list of data items and associated data codes

 

 

2B

THREAT MISSILE, GENERIC

GEN

ENTER "GEN:" FOLLOWED BY THE LITERAL NAME OR DESIGNATOR OF A THREAT MISSILE NOT LISTED IN THE 'THREAT MISSILE, SPECIFIC' ALTERNATIVE, FOR EXAMPLE: "GEN:NEW THREAT MISSILE 1". See table 1407/38 which is an instructive entry

3

ZIPPO REACTION CODE

 

 

 

 

3A

ZIPPO REACTION CODE

 

ENTER THE CODE FOR THE ZIPPO REACTION(S) TO BE TAKEN FOR THE SPECIFIED THREAT MISSILE, FOR EXAMPLE: "B". See table 1257/2 which is an alphanumeric range
