1

MORE THAN ONE COLUMN INVOLVED IN MOVEMENT

 

 

 

 

1A

MORE THAN ONE COLUMN INVOLVED IN MOVEMENT

 

ENTER "Y" OR "N" (YES OR NO) TO INDICATE WHETHER OR NOT THERE ARE MORE THAN ONE COLUMN INVOLVED IN THE MOVE. See table 1359/32 which contains a list of data items and associated data codes

2

OPERATION CODEWORD

 

 

 

 

2A

OPERATION CODEWORD

 

ENTER THE NAME (CODEWORD) OF THE OPERATION WITH WHICH THE MOVEMENT IS ASSOCIATED, FOR EXAMPLE: "STRONG ALLIANCE". See table 1020/1 which is an instructive entry

3

COLUMN/ROUTE DETAILS

 

 

 

 

3A

COLUMN ROUTE DETAILS

 

ENTER LITERAL COLUMN/ROUTE DETAILS FOR THE MOVEMENT. See table 1357/29 which is an instructive entry

4

POSITION IDENTIFYING NUMBER

 

 

 

 

4A

POSITION IDENTIFYING NUMBER

 

ENTER A NUMERIC FIGURE IDENTIFYING THE POSITION IN THE SEQUENCE OF COLUMNS IN WHICH THE COLUMN IS TO MOVE, FOR EXAMPLE: "08". See table 1012/300 a range [1 through 999]

5

TIME GAP

 

 

 

 

5A

TIME GAP

 

ENTER THE TIME GAP BETWEEN COLUMNS  IN HOURS AND MINUTES WHICH IS TO BE CONSIDERED BY THE RESPONSIBLE MOVEMENT AGENCY, FOR EXAMPLE: "0012". See table 2041 which is a composite

6

UNIT DESIGNATOR

 

 

 

 

6A

UNIT DESIGNATOR

 

MAKE A LITERAL ENTRY PROVIDING THE UNIT DESIGNATOR FOR THE UNIT TO MOVE, FOR EXAMPLE: "4077 MASH". See table 1028/43 which is an instructive entry

7

UNIT SIZE INDICATOR

 

 

 

 

7A

UNIT SIZE INDICATOR

 

ENTER A CODE TO IDENTIFY THE SIZE OF THE UNIT TO MOVE, FOR EXAMPLE: "AG". See table 1045/4 which contains a list of data items and associated data codes

8

HIGHER FORMATION NAME

 

 

 

 

8A

HIGHER FORMATION NAME

 

ENTER A LITERAL GIVING THE DESIGNATOR OF THE NEXT HIGHER HQ/ORGANISATION OF THE UNIT TO BE MOVED, FOR EXAMPLE: "3RD(US)CORPS". See table 1022/40 which is an instructive entry

9

CODE NAME

 

 

 

 

9A

CODE NAME

 

ENTER THE CODE NAME IDENTIFIER ALLOCATED TO THE MOVE, FOR EXAMPLE: "COMBINED EFFORT 1999". See table 1022/96 which is an instructive entry

10

ALERT STATUS

 

 

 

 

10A

ALERT STAGE

 

ENTER THE ALERT STAGE UPON DECLARATION OF WHICH THE MOVEMENT IS TO BE EXECUTED, FOR EXAMPLE: "ORANGE". See table 1798/1 which is an instructive entry

11

GROUPING

 

 

 

 

11A

GROUPING

 

ENTER THE GROUPING UPON DECLARATION OF WHICH THE MOVEMENT IS TO BE EXECUTED. See table 1864/5 which is an instructive entry

12

PRECAUTIONARY MEASURE

 

 

 

 

12A

PRECAUTIONARY MEASURE

 

MAKE A LITERAL ENTRY GIVING THE PRECUTIONARY MEASURE UPON DECLARATION OF WHICH THE MOVEMENT IS TO BE EXECUTED. See table 1864/6 which is an instructive entry

13

TIMINGS

 

ENTER EITHER THE EARLIEST STARTING TIME OR THE LATEST RELEASE TIME:

 

 

13A

EARLIEST STARTING TIME

ESTIME

ENTER "ESTIME:" FOLLOWED BY THE STARTING TIME AT WHICH THE UNIT INTENDS TO MOVE, FOR EXAMPLE: "EST:171645ZAPR2001". See table 2033 which is a composite

 

 

13B

LATEST RELEASE TIME

LRTIME

ENTER "LRTIME:" FOLLOWED BY THE TIME (DTG) BY WHICH THE RELEASE POINT IS TO BE PASSED, FOR EXAMPLE: "LRTIME:172030ZAPR2001". See table 2033 which is a composite

14

EXECUTION ON CALL

 

 

 

 

14A

EXECUTION ON CALL

 

ENTER "Y" OR "N" TO INDICATE WHETHER OR NOT THE MOVEMENT IS TO BE EXECUTED ON DECLARATION OF AN ALERT STAGE, A GROUPING OR A PRECAUTIONARY MEASURE. See table 1359/33 which contains a list of data items and associated data codes

15

MOVEMENT PRIORITY

 

 

 

 

15A

MOVEMENT PRIORITY

 

ENTER THE OVERALL PRIORITY ALLOCATED TO THE MOVE, FOR EXAMPLE: "ALPHA". See table 1076/15 which contains a list of data items and associated data codes

16

PRIORITY OF MOVE ON ROUTE

 

 

 

 

16A

PRIORITY OF MOVE ON ROUTE

 

ENTER THE ASSIGNED PRIORITY OF MOVE ON THE MARCH, FOR EXAMPLE: "BRAVO". See table 1076/16 which contains a list of data items and associated data codes

17

ARRIVAL SEQUENCE

 

 

 

 

17A

ARRIVAL SEQUENCE

 

ENTER ANY LITERAL INFORMATION CONCERNING PREDETERMINED ARRIVAL SEQUENCE. See table 1494/5 which is an instructive entry

18

NUMBER AND TYPE OF WHEELED VEHICLES

 

 

 

 

18A

NUMBER AND TYPE OF WHEELED VEHICLES

 

ENTER NUMBER AND TYPE OF WHEELED VEHICLES TO MOVE, FOR EXAMPLE: "154-TRUCK, 5TONS, 4X4". See table 2510 which is a composite

19

NUMBER AND TYPE OF TRACKED VEHICLES

 

 

 

 

19A

NUMBER AND TYPE OF TRACKED VEHICLES

 

ENTER NUMBER AND TYPE OF TRACKED VEHICLES, FOR EXAMPLE: "66-LITERAL FOR TYPE". See table 2510 which is a composite
