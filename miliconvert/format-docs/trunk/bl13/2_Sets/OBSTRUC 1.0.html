Designator

Field Desc

Concept/Explanation/Examples

1

OBSTRUCTION TYPE

 

 

 

 

1A

OBSTRUCTION TYPE

 

ENTER OBSTRUCTION TYPE, FOR EXAMPLE: "BRDEMO". See table 1270/1 which contains a list of data items and associated data codes

2

NUMBER OF OBSTRUCTIONS

 

 

 

 

2A

NUMBER OF OBSTRUCTIONS

 

ENTER NUMBER OF OBSTRUCTIONS [0-99], FOR EXAMPLE: "1". See table 1044/319 a range [0 through 99]

3

OBSTRUCTION LOCATION

 

MULTIPLE OBSTRUCTION LOCATION CAN BE SPECIFIED BY REPEATING FIELD NO 3 TO FIELD NO 7 AS A GROUP.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

3C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

3D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

3E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

3F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

4

APPROXIMATE HEIGHT ABOVE GROUND

 

IF USED, ENTER APPROXIMATE HEIGHT ABOVE GROUND IN ONE OF THE FOLLOWING WAYS:

 

 

4A

APPROXIMATE HEIGHT ABOVE GROUND IN METRES

 

ENTER THE APPROXIMATE HEIGHT IN METRES [0-999], FOR EXAMPLE: "20". See table 1651/7 a range [0 through 999]

 

 

4B

HEIGHT IN FEET

FT

ENTER "FT:" FOLLOWED BY THE HEIGHT IN FEET [0-999999999], FOR EXAMPLE: "FT:60". See table 1581/1 a range [0 through 999999999]

5

DEPTH BELOW SURFACE IN METRES

 

 

 

 

5A

DEPTH BELOW SURFACE IN METRES

 

ENTER DEPTH BELOW SURFACE IN METRES [1-9999], FOR EXAMPLE: "12" See table 1247/7 a range [1 through 9999]

6

WIDTH IN METRES, 1-99999

 

 

 

 

6A

WIDTH IN METRES, 1-99999

 

ENTER WIDTH OF OBSTRUCTION IN METRES [1-99999], FOR EXAMPLE: "20". See table 1658/6 a range [1 through 99999]

7

LENGTH IN METRES, 1-99999

 

 

 

 

7A

LENGTH IN METRES, 1-99999

 

ENTER LENGTH OF OBSTRUCTION IN METRES [1-99999], FOR EXAMPLE: "25". See table 1657/7 a range [1 through 99999]
