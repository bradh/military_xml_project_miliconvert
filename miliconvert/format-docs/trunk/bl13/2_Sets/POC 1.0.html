No

Designator

Field Desc

Concept/Explanation/Examples

1

POC NAME

 

PROVIDES THE NAME IDENTIFYING THE POINT OF CONTACT.

 

 

1A

CONTACT NAME

 

ENTER THE NAME OF THE PERSON TO BE CONTACTED, FOR EXAMPLE: "F. BURNS". See table 1022/7 which is an instructive entry

2

POC RANK OR POSITION

 

PROVIDES THE RANK OR POSITION OF THE POINT OF CONTACT.

 

 

2A

RANK OR POSITION

 

ENTER THE RANK OR POSITION OF THE PERSON TO BE CONTACTED, FOR EXAMPLE: "MAJ". See table 1046/2 which is an instructive entry

3

POC UNIT

 

PROVIDES THE IDENTIFICATION OF THE UNIT TO WHOM THE POINT OF CONTACT BELONGS.

 

 

3A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE IDENTIFICATION OF A MILITARY, PARA-MILITARY OR GOVERNMENT AGENCY UNIT, AS USED IN OFFICIAL COMMUNICATIONS WITHIN THE MILITARY ESTABLISHMENTS, FOR EXAMPLE: "UNIT:4077 MASH". See table 1028/43 which is an instructive entry

 

 

3B

CONTACT CALL SIGN

CALLSIGN

ENTER "CALLSIGN:" FOLLOWED BY THE CALL SIGN OF THE UNIT OR AGENCY, FOR EXAMPLE: "CALLSIGN:HAWKEYE". See table 1129/31 which is an instructive entry

 

 

3C

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 2 LETTER COUNTRY CODE), OF THE UNIT TO WHICH THE POC BELONGS, FOR EXAMPLE: "UIC2:GEF123456". See table 2243 which is a composite

 

 

3D

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 3 LETTER COUNTRY CODE), OF THE UNIT TO WHICH THE POC BELONGS, FOR EXAMPLE: "UIC3:DEUF123456". See table 2241 which is a composite

4

POC LOCATION

 

PROVIDES THE GEOGRAPHIC LOCATION OF THE POINT OF CONTACT.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME ASSIGNED TO THE LOCATION, FOR EXAMPLE: "LOC:GARGANTA DEL DIABLO". See table 1022/4 which is an instructive entry

 

 

4H

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LETTER BASE CODE, FOR EXAMPLE: "ICAO:ABCD". See table 1125/2 which is an instructive entry

 

 

4I

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES VERIFIED

SUB

ENTER "SUB:" FOLLOWED BY THE LATITUDE AND LONGITUDE UP TO 1/10,000 OF A MINUTE, VERIFIED, FOR EXAMPLE: "SUB:2010.30N6-01040.20W7". See table 2577 which is a composite

5

COMMUNICATION MEANS PARAMETERS

 

PROVIDES THE COMMUNICATION MEANS PARAMETERS FOR THE POINT OF CONTACT IDENTIFIED IN FIELD ONE.

 

 

5A

NON-SECURE TELEPHONE

TEL

ENTER "TEL:" FOLLOWED BY THE DESIGNATED NONSECURE TELEPHONE NUMBER, FOR EXAMPLE: "TEL:804-555-4142". See table 1361/3 which is an instructive entry

 

 

5B

SECURE TELEPHONE NUMBER

SECTEL

ENTER "SECTEL:" FOLLOWED BY THE DESIGNATED SECURE TELEPHONE NUMBER, FOR EXAMPLE: "SECTEL:555-4311". See table 1361/4 which is an instructive entry

 

 

5C

SECURE FACSIMILE (FAX) NUMBER

SECFAX

ENTER "SECFAX:" FOLLOWED BY THE DESIGNATED SECURE FAX NUMBER, FOR EXAMPLE: "SECFAX:0032-2-707-4444". See table 1361/8 which is an instructive entry

 

 

5D

NON-SECURE FACSIMILE (FAX) NUMBER

FAX

ENTER "FAX:" FOLLOWED BY THE DESIGNATED NONSECURE FAX NUMBER, FOR EXAMPLE: "FAX:0032-2-707-5834". See table 1361/7 which is an instructive entry

 

 

5E

RADIO FREQUENCY

FRQ

ENTER "FRQ:" FOLLOWED BY THE FREQUENCY, FOR EXAMPLE: "FRQ:35.25MHZ". See table 2064 which is a composite

 

 

5F

FREQUENCY DESIGNATOR

FRQDES

ENTER "FRQDES:" FOLLOWED BY THE FREQUENCY DESIGNATOR,  FOR EXAMPLE: "FRQDES:RED1". See table 1396/1 which is an instructive entry

 

 

5G

FREQUENCY MODE AND NUMERIC

FREQ

ENTER "FREQ:" FOLLOWED BY THE FREQUENCY VALUE FOR RADIO TRANSMISSION, AND AN INDICATOR OF THE FREQUENCY MODE, FOR EXAMPLE: "FREQ:U5025". See table 2498 which is a composite

 

 

5H

ELECTRONIC MAIL ADDRESS

EMAIL

ENTER "EMAIL:" FOLLOWED BY THE IDENTIFIER USED BY SENDERS AND RECIPIENTS OF ELECTRONIC MAIL(E-MAIL) OVER COMPUTER NETWORKS, FOR EXAMPLE: "EMAIL:EWINGP(AT)NYKNICKS.COM". See table 1183/7 which is an instructive entry

 

 

5I

SECURE ELECTRONIC MAIL ADDRESS

SEMAIL

ENTER "SEMAIL:" FOLLOWED BY THE IDENTIFIER USED BY SENDERS AND RECIPIENTS OF ELECTRONIC MAIL (E-MAIL) OVER COMPUTER NETWORKS, FOR EXAMPLE: "SEMAIL:SEWINGP(AT)SNEAKYNICKS.COM". See table 1183/9 which is an instructive entry
