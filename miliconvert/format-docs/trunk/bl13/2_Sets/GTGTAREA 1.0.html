No

Designator

Field Desc

Concept/Explanation/Examples

1

AREA NAME

 

IT PROVIDES A PLAIN LANGUAGE DESCRIPTION OF THE GROUND TARGET AREA NAME.

 

 

1A

AREA NAME

 

ENTER THE GROUND TARGET AREA NAME, FOR EXAMPLE: "RED TWO". See table 1022/69 which is an instructive entry

2

AREA TYPE

 

IT PROVIDES A PLAIN LANGUAGE DESCRIPTION OF THE GROUND TARGET AREA TYPE.

 

 

2A

AREA TYPE

 

ENTER THE GROUND TARGET AREA TYPE, FOR EXAMPLE: "SUPPLY". See table 1284/17 which is an instructive entry

3

TARGET ELEVATION

 

IT PROVIDES THE TARGET ELEVATION EXPRESSED IN FEET OR IN METRE, WITH ELEVATION QUALIFIER MSL (MEAN SEA LEVEL) OR HAE (HEIGHT ABOVE THE REFERENCE ELLIPSOID).

 

 

3A

TARGET ELEVATION

 

ENTER THE TARGET ELEVATION EXPRESSED IN FEET OR IN METRES ABOVE OR BELOW MEAN SEA LEVEL OR ABOVE THE REFERENCE ELLIPSOID, FOR EXAMPLE: "300FTMSL". See table 2492 which is a composite

4

AREA GEOMETRY

 

IF USED, IT PROVIDES THE MAIN GEOMETRIC FEATURE OF THE GROUND TARGET AREA, SUCH AS EITHER THE RADIUS OF THE CIRCLE AROUND THE LOCATION, OR THE DIMENSIONS AND ORIENTATION OF THE SEMI-MAJOR AXIS OF AN ELLIPTIC AREA, OR SECTOR MEASURES AND LIMITS.

 

 

4A

RADIUS, IN, FT, YD, KF, SM, NM, MM, CM, M, KM, HM

CIRC

ENTER "CIRC:"  FOLLOWED BY THE RADIUS OF THE CIRCULAR GROUND TARGET AREA, FOR EXAMPLE: "CIRC:50SM". See table 2490 which is a composite

 

 

4B

ELLIPTICAL AREA, DIMENSIONS AND MAJOR AXIS ORIENTATION

ELL

ENTER "ELL:"  FOLLOWED BY ELLIPTICAL AREA OF THE GROUND TARGET AREA, FOR EXAMPLE: "ELL:470NM-750NM-180.0". See table 2494 which is a composite

 

 

4C

SECTOR GEOMETRY WITH OUTER RANGE

SECTO

ENTER "SECTO:"  FOLLOWED BY THE SECTOR GEOMETRY OF THE GROUND TARGET AREA  USING AN OUTER RANGE, FOR EXAMPLE: "SECTO:180240T27NM". See table 2526 which is a composite

 

 

4D

SECTOR GEOMETRY WITH INNER AND OUTER RANGE

SECTIO

ENTER "SECTIO:"  FOLLOWED BY THE SECTOR GEOMETRY OF THE GROUND TARGET AREA USING AN INNER AND OUTER RANGE, FOR EXAMPLE: "SECTIO:180240T27-100NM". See table 2527 which is a composite

5

AREA COORDINATES OR ORIGIN

 

THE GEOGRAFIC LOCATION OF THE GROUND TARGET AREA.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

5C

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO INDICATOR, FOR EXAMPLE: "ICAO:KILA". See table 1125/1 which is an instructive entry
