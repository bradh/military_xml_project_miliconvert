1

MINIMUM CLEAR WIDTH (FEET, YARDS OR METRES)

 

 

 

 

1A

MINIMUM CLEAR WIDTH (FEET, YARDS OR METRES)

 

ENTER THE MINIMUM CLEAR DISTANCE BETWEEN THE INSIDE EDGES OF THE BRIDGE STRUCTURE FROM A HEIGHT OF 30 CENTIMETRES OR ONE FOOT ABOVE THE ROADWAY SURFACE AND UPWARDS, FOR EXAMPLE: "30M". See table 2451 which is a composite

2

ROADWAY WIDTH (FEET, YARDS OR METRES)

 

 

 

 

2A

ROADWAY WIDTH (FEET, YARDS OR METRES)

 

ENTER THE WIDTH OF THE TRAVELLED WAY MEASURED BETWEEN THE INSIDE EDGES OF THE CURBS IN FEET, YARDS, OR METRES, FOR EXAMPLE: "25M". See table 2451 which is a composite

3

GEOGRAPHIC FEATURE SPANNED

 

 

 

 

3A

GEOGRAPHIC FEATURE SPANNED

 

ENTER THE TYPE OF TERRAIN OVER WHICH THE BRIDGE SPANS, FOR EXAMPLES: "TW". See table 1787/1 which contains a list of data items and associated data codes

4

UNDER BRIDGE CLEARANCE

 

 

 

 

4A

UNDER BRIDGE CLEARANCE

 

ENTER THE MAXIMUM CLEAR DISTANCE BETWEEN THE UNDERSIDE OF THE BRIDGE AND THE SURFACE OF THE GROUND OR WATER, FOR EXAMPLE: "20M". See table 2452 which is a composite

5

DTG OF MEASUREMENT

 

 

 

 

5A

DTG OF MEASUREMENT

 

ENTER THE DTG OF THE MEASUREMENT OF THE UNDERBRIDGE CLEARANCE.   IF THE BRIDGE SPANS TIDAL WATER, I.E. FIELD THREE IS "TW", THIS FIELD IS MANDATORY, FOR EXAMPLE: "261515ZJUN2000". See table 2033 which is a composite

6

MINIMUM CLEAR HEIGHT/CLEARANCE

 

 

 

 

6A

MINIMUM CLEAR HEIGHT/CLEARANCE

 

ENTER THE MINIMUM DISTANCE BETWEEN THE TRAVELLED WAY AND ANY OBSTRUCTION ABOVE IT.  MEASUREMENTS ARE TAKEN AT THE LEFT SHOULDER, CENTRE OF ROADWAY, AND THE RIGHT SHOULDER.  IF ALL MEASUREMENTS ARE EQUAL, REPORT ONLY ONCE, FOR EXAMPLE: "15M". See table 2452 which is a composite
