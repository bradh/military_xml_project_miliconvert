1

TRANSPORTATION REQUIREMENT IDENTIFIER

 

PROVIDES THE IDENTIFICATION OF THE TRANSPORTATION REQUEST, USING EITHER A LITTERAL IDENTIFIER, OR THE REQUESTING UNIT LINE NUMBER OR UIC, OR THE PERSONNEL/CARGO INCREMENT NUMBER DUE TO TRANSPORT.

 

 

1A

TRANSPORTATION REQUIREMENT IDENTIFIER

ID

ENTER "ID:" FOLLOWED BY THE TRANSPORTATION REQUIREMENT IDENTIFIER, FOR EXAMPLE:"ID:CASTILE2". See table 1102/11 which is an instructive entry

 

 

1B

UNIT LINE NUMBER

ULN

ENTER "ULN: " FOLLOWED BY THE LINE NUMBER OF THE UNIT CONCERNED BY THE TRANSPORT REQUEST, FOR EXAMPLE: "ULN:BAQ4611". See table 1012/246 which is an instructive entry

 

 

1C

UNIT IDENTIFICATION CODE (UIC2)

 

ENTER THE UNIT IDENTIFICATION CODE (USING THE NATO 2 LETTER COUNTRY CODE), OF THE UNIT CONCERNED BY THE AIR TRANSPORTATION REQUEST, FOR EXAMPLE: "UKJ12345". See table 2243 which is a composite

 

 

1D

UNIT IDENTIFICATION CODE (UIC3)

 

ENTER THE UNIT IDENTIFICATION CODE (USING THE NATO 3 LETTER COUNTRY CODE), OF THE UNIT CONCERNED BY THE AIR TRANSPORTATION REQUEST, FOR EXAMPLE, "GBRF243539". See table 2241 which is a composite

 

 

1E

PERSONNEL INCREMENT NUMBER

PIN

ENTER "PIN:" FOLLOWED BY THE PERSONNEL INCREMENT NUMBER DUE TO TRANSPORT, FOR EXAMPLE: "PIN:0123456". See table 1012/251 which is an instructive entry

 

 

1F

CARGO INCREMENT NUMBER

CIN

ENTER "CIN:" FOLLOWED BY THE CARGO INCREMENT NUMBER DUE TO TRANSPORT, FOR EXAMPLE: "CIN:0123456". See table 1012/252 which is an instructive entry

2

AIRPORT OF EMBARKATION

 

THE AIRPORT OF EMBARKATION FOR THE REQUESTED TRANSPORTATION, EITHER USING THE ICAO LOCATION INDICATOR, OR THE GEOLOCATION OF THE AIRPORT.

 

 

2A

ICAO CODE - AIRPORT OF EMBARKATION

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR OF THE AIRPORT OF EMBARKATION, FOR EXAMPLE "ICAO:BEIR". See table 1125/12 which is an instructive entry

 

 

2B

SPECIFIED GEOLOCATION CODE

GEO

ENTER "GEO:" FOLLOWED BY THE SPECIFIC GEOLOCATION OF EMBARKMENT, FOR EXAMPLE: "GEO:K3TW" See table 1125/11 which is an instructive entry

3

DATE OF DEPARTURE

 

THE REQUESTED DATE OF DEPATURE OF THE TRANPORTATION, USING EITHER A DATE-TME GRUP, A DAY-TIME OR A JULIAN DATE.

 

 

3A

DTG OF DEPARTURE

DTG

ENTER "DTG:" FOLLOWED BY THE DTG OF DEPARTURE, FOR EXAMPLE:"DTG:142030ZNOV2005". See table 2033 which is a composite

 

 

3B

DAY-TIME OF DEPARTURE

DT

ENTER "DT:" FOLLOWED BY THE DAY-TIME OF DEPARTURE, FOR EXAMPLE: "DT:142030Z". See table 2000 which is a composite

 

 

3C

JULIAN DAY-TIME OF DEPARTURE

JDT

ENTER "JDT:" FOLLOWED BY THE JULIAN DAY-TIME OF DEPARTURE, FOR EXAMPLE: "JDT:3002030Z". See table 2484 which is a composite

4

 AIRPORT OF DEBARKATION

 

THE AIRPORT OF DEBARKATION FOR THE REQUESTED TRANSPORTATION, EITHER USING THE ICAO LOCATION INDICATOR, OR THE GEOLOCATION.

 

 

4A

ICAO CODE - AIRPORT OF DEBARKATION

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR OF THE AIRPORT OF DEBARKATION, FOR EXAMPLE: "ICAO:LIPZ". See table 1125/7 which is an instructive entry

 

 

4B

SPECIFIED GEOLOCATION CODE

GEO

ENTER "GEO:", FOLLOWED BY THE SPECIFIED GEOLOCATION CODE OF THE AIRPORT OF DEBARKATION, FOR EXAMPLE: "GEO:VRJT". See table 1125/11 which is an instructive entry

5

DATE OF ARRIVAL

 

THE REQUESTED DATE OF ARRIVSAL OF THE TRANPORTATION, USING EITHER A DATE-TIME GROUP, A DAY-TIME OR A JULIAN DATE.

 

 

5A

DTG OF SCHEDULED ARRIVAL

DTG

ENTER "DTG:" FOLLOWED BY THE DTG OF ARRIVAL, FOR EXAMPLE: "DTG:251000ZNOV2005". See table 2033 which is a composite

 

 

5B

DAY-TIME OF ARRIVAL

DT

ENTER "DT:" FOLLOWED BY THE DAY-TIME OF ARRIVAL, FOR EXAMPLE: "DT:251000Z". See table 2000 which is a composite

 

 

5C

JULIAN DAY-TIME OF ARRIVAL

JDT

ENTER "JDT:" FOLLOWED BY THE JULIAN DATE-TIME OF ARRIVAL, FOR EXAMPLE: "JDT:2961000Z". See table 2484 which is a composite
