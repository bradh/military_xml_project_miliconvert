CIVILIAN/MILITARY AIR NAVIGATION AID TYPE

 

 

 

 

1A

CIVILIAN MILITARY AIR NAVIGATION AID TYPE

 

ENTER THE CIVILIAN/MILITARY AIR NAVIGATION AID TYPE AVAILABLE AT THE BASE BEING REPORTED, FOR EXAMPLE: "TACAN". See table 1200/4 which contains a list of data items and associated data codes

2

NAVIGATION AID NAME

 

 

 

 

2A

NAVIGATION AID NAME

 

ENTER THE NAVIGATION AID NAME, FOR EXAMPLE: "COLA". See table 1022/37 which is an instructive entry

3

NAVIGATION AID ICAO CODE

 

 

 

 

3A

NAVIGATION AID ICAO CODE

 

ENTER THE ICAO CODE IDENTIFIER FOR THE NAVIGATIONAL AID, FOR EXAMPLE: "COL". See table 1246/28 which is an instructive entry

4

NAVIGATION AID FREQUENCY

 

THE FREQUENCY FOR THE NAVIGATION AID.

 

 

4A

FREQUENCY IN MEGAHERTZ

NAVFRQ

ENTER "NAVFRQ:" FOLLOWED BY THE FREQUENCY IN MEGAHERTZ, FOR EXAMPLE: "NAVFRQ:112.5" See table 1068/1 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

4B

FREQUENCY DESIGNATOR

NAVDSG

ENTER "NAVDSG:" FOLLOWED BY THE FREQUENCY DESIGNATOR, FOR EXAMPLE: "NAVDSG:CH77" See table 1396/1 which is an instructive entry

 

 

4C

FREQUENCY MODE AND NUMERIC

FREQ

ENTER "FREQ:" FOLLOWED BY THE FREQUENCY MODE AND NUMBER, FOR EXAMPLE: "FREQ:T25.5" See table 2498 which is a composite

5

NAVIGATION AID LOCATION

 

PROVIDES THE NAVIGATION AID LOCATION INFORMATION.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

5C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

5D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

5E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

5F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression
