Designator

Field Desc

Concept/Explanation/Examples

1

LASER/DATALINK CODE

 

PROVIDES A CODE THAT ENSURES CORRELATION BETWEEN THE TARGET DESIGNATION SYSTEM AND THE WEAPON DELIVERY SYSTEM  USING ONE OF THE FOLLOWING:

 

 

1A

LASER CORRELATION CODE

LC

ENTER "LC:" FOLLOWED BY THE CODE FOR THE LASER CORRELATION, FOR EXAMPLE: "LC:3290". See table 1012/254 a range [0000 through 9999]

 

 

1B

DATA LINK FREQUENCY CODE

DL

ENTER "DL:" FOLLOWED BY THE CODE FOR THE FREQUENCY OF THE DATALINK, FOR EXAMPLE:  "DL:A1". See table 1396/4 which is an instructive entry

2

GUIDANCE FREQUENCY

 

IF USED, PROVIDES THE GUIDANCE FREQUENCY FOR THE PRECISION GUIDED WEAPON USING ONE OF THE FOLLOWING:

 

 

2A

GUIDANCE FREQUENCY IN MEGAHERTZ

FREQ

ENTER "FREQ:" FOLLOWED BY THE FREQUENCY USED FOR IN-FLIGHT/ TERMINAL GUIDANCE OF PRECISION MUNITIONS, FOR EXAMPLE: "FREQ:123.456". See table 1068/11 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

2B

GUIDANCE FREQUENCY DESIGNATOR

DESIG

ENTER "DESIG:" FOLLOWED BY THE FREQUENCY DESIGNATOR USED FOR IN-FLIGHT/ TERMINAL GUIDANCE OF PRECISION MUNITIONS, FOR EXAMPLE: "DESIG:RED". See table 1396/7 which is an instructive entry

3

CONTACT CALL SIGN

 

 

 

 

3A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN OF THE PRECISION GUIDED MUNITION, FOR EXAMPLE: "BRAVO CHARLIE 35". See table 1129/31 which is an instructive entry

4

LINK 16 ABBREVIATED CALL SIGN

 

 

 

 

4A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE LINK 16 ABBREVIATED CALL SIGN IF THE PRECISION GUIDED MUNITION GROUND CONTACT IS LINK 16 EQUIPPED, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry

5

PRIMARY FREQUENCY

 

IF USED, PROVIDES THE PRIMARY FREQUENCY FOR CONTACTING THE PRECISION GUIDED MUNITION USING ONE OF THE FOLLOWING:

 

 

5A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY TO BE USED, FOR EXAMPLE: "PFREQ:342.5". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

5B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR TO BE USED, FOR EXAMPLE: "PDESIG:TAD01". See table 1396/2 which is an instructive entry

6

SECONDARY FREQUENCY

 

IF USED, PROVIDES THE SECONDARY FREQUENCY FOR CONTACTING THE PRECISION GUIDED MUNITION USING ONE OF THE FOLLOWING:

 

 

6A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE SECONDARY FREQUENCY TO BE USED, FOR EXAMPLE: "SFREQ:350.5". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

6B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY DESIGNATOR TO BE USED, FOR EXAMPLE: "SDESIG:TAD02". See table 1396/3 which is an instructive entry
