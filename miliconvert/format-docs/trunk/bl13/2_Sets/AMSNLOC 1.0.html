No

Designator

Field Desc

Concept/Explanation/Examples

1

MISSION PRIORITY

 

 

 

 

1A

MISSION PRIORITY, LITERAL

 

ENTER THE CODE FOR THE PRIORITY ASSIGNED TO THE MISSION, FOR EXAMPLE: "1A". See table 1076/12 which is an instructive entry

2

DAY-TIME AND MONTH OF START

 

 

 

 

2A

DAY-TIME AND MONTH OF START

 

ENTER THE DAY-TIME AND MONTH OF THE BEGINNING OF THE MISSION AT THE MISSION LOCATION, FOR EXAMPLE: "152000ZMAR". See table 2030 which is a composite

3

DAY-TIME AND MONTH OF STOP

 

 

 

 

3A

DAY-TIME AND MONTH OF STOP

 

ENTER THE DAY-TIME AND MONTH OF THE END OF THE MISSION AT THE MISSION LOCATION, FOR EXAMPLE: "152200ZMAR". See table 2030 which is a composite

4

LOCATION NAME

 

 

 

 

4A

MISSION PLACE NAME

 

ENTER A NAME THAT IDENTIFIES THE LOCATION AT WHICH AN AIR MISSION IS TO BE PERFORMED.  THIS CAN BE THE NAME OF A GENERAL TARGET AREA, ORBIT, CAP POINT, STATION, ETC.  FOR EXAMPLE: "MISTY ORBIT". See table 1022/80 which is an instructive entry

5

LOCATION ALTITUDE

 

THE ALTITUDE OF MISSION LOCATION.

 

 

5A

ALTITUDE, HUNDREDS OF FEET

ALT

ENTER "ALT:" FOLLOWED BY THE VERTICAL DISTANCE IN HUNDREDS OF FEET ABOVE MEAN SEA LEVEL (MSL) OF THE MISSION LOCATION, FOR EXAMPLE: "ALT:210". See table 1255/14 a range [000 through 999]

 

 

5B

ALTITUDE BAND

ALTBAND

ENTER "ALTBAND:" FOLLOWED BY THE ALTITUDE BAND (MINIMUM AND MAXIMUM VERTICAL DISTANCE) IN HUNDREDS OF FEET, FOR EXAMPLE: "ALTBAND:310-350". See table 2100 which is a composite

6

AREA GEOMETRY

 

THE GEOMETRIC DETAILS OF THE AREA OF MISSION LOCATION.

 

 

6A

RADIUS, IN, FT, YD, KF, SM, NM, MM, CM, M, KM, HM

CIRC

ENTER "CIRC:" FOLLOWED BY THE RADIUS OF THE CIRCULAR AREA, FOR EXAMPLE: "CIRC:50NM". See table 2490 which is a composite

 

 

6B

ELLIPTICAL AREA, DIMENSIONS AND MAJOR AXIS ORIENTATION

ELL

ENTER "ELL:" FOLLOWED BY THE MEASURE IN NAUTIC MILES OF THE MAJOR AND MINOR AXES, AND THE ORIENTATION IN DEGREES OF THE MAJOR AXIS OF THE ELLIPTICAL AREA OF THE MISSION LOCATION, FOR EXAMPLE: "ELL:470NM-750NM-180.0". See table 2494 which is a composite

 

 

6C

SECTOR GEOMETRY WITH OUTER RANGE

SECTO

ENTER "SECTO:" FOLLOWED BY SECTOR GEOMETRY OF THE MISSION AREA USING AN OUTER RANGE, FOR EXAMPLE: "SECTO:180240T27NM". See table 2526 which is a composite

 

 

6D

SECTOR GEOMETRY WITH INNER AND OUTER RANGE

SECTIO

ENTER "SECTIO:" FOLLOWED BY THE SECTOR GEOMETRY OF THE MISSION AREA USING AN INNER AND OUTER RANGE, FOR EXAMPLE: "SECTIO:058190T45-60NM". See table 2527 which is a composite

7

LOCATION

 

THE GEOGRAPHICAL LOCATION OF THE AERIAL MISSION.

 

 

7A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

7B

BEARING AND DISTANCE FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

7C

POSITION OR POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE POINT OR AREA, FOR EXAMPLE: "NAME:PT ALFA". See table 1022/18 which is an instructive entry
