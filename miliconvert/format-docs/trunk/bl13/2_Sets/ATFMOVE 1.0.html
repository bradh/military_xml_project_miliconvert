DTG OF ARRIVAL

 

 

 

 

1A

DTG OF ARRIVAL

 

ENTER THE DTG OF ARRIVAL OF THE AMPHIBIOUS TASK FORCE, FOR EXAMPLE: "120300ZJUN2003". See table 2033 which is a composite

2

AOA ARRIVAL POINT

 

THE GEOGRAPHIC LOCATION OF THE AMPHIBIOUS TASK FORCE POINT OF ARRIVAL WITHIN THE AOA.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

3

GEODETIC DATUM

 

 

 

 

3A

GEODETIC DATUM

 

ENTER THE ALTERNATIVE GEODETIC DATUM USED FOR THE AMPHIBIOUS TASKFORCE MOVEMENT, FOR EXAMPLE:"WGE". See table 1195/1 which contains a list of data items and associated data codes

4

AOA APPROACH FORMATION

 

 

 

 

4A

AOA APPROACH FORMATION

 

ENTER THE FORMATION USED BY THE AMPHIBIOUS TASK FORCE WHEN APPROACHING THE AOA, FOR EXAMPLE: "FORM D". (1-20X) See table 1022/331 which is an instructive entry

5

FORMATION GUIDE NAME

 

 

 

 

5A

FORMATION GUIDE NAME

 

ENTER THE NAME OR CALL SIGN OF THE FORMATION GUIDE, FOR EXAMPLE: "ARK ROYAL". See table 1022/145 which is an instructive entry

6

INTERVAL SPACING, YARDS

 

 

 

 

6A

INTERVAL SPACING, YARDS

 

ENTER THE INTERVAL SPACING IN YARDS TO BE USED WITHIN THE FORMATION, FOR EXAMPLE: "6000". See table 1218/3 a range [0 through 99999]

7

FORMATION SPEED IN KNOTS

 

 

 

 

7A

FORMATION SPEED IN KNOTS

 

ENTER THE FORMATION SPEED IN KNOTS, FOR EXAMPLE: "15". See table 1061/3 a range [0 through 99]

8

SHIP ENTRY SEQUENCE

 

 

 

 

8A

SHIP ENTRY SEQUENCE

 

ENTER THE NAME OF A SHIP IN THE TASK FORCE, REPEATING AS NECESSARY IN ORDER OF SHIP ENTRY, FOR EXAMPLE "OCEAN". See table 1022/339 which is an instructive entry
