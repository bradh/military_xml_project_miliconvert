POC LINK FUNCTION

CONTACT NAME

 

ENTER THE NAME OF THE PERSON TO BE CONTACTED, FOR EXAMPLE: "F.BURNS". See table 1022/7 which is an instructive entry

RANK OR POSITION

 

 

 

 

3A

RANK OR POSITION

 

ENTER THE RANK OR POSITION OF THE PERSON TO BE CONTACTED, FOR EXAMPLE: "MAJ". See table 1046/2 which is an instructive entry

4

UNIT DESIGNATION

 

PROVIDES THE UNIT IDENTIFIER OR CALL SIGN OF THE POINT OF CONTACT USING ONE OF THE FOLLOWING:

 

 

4A

SHIP NAME

SHIP

ENTER "SHIP:" FOLLOWED BY THE SHIP NAME, OMITTING ANY GOVERNMENT DESIGNATION, FOR EXAMPLE, "SHIP:STENNIS". See table 1022/49 which is an instructive entry

 

 

4B

BASE NAME

BASE

ENTER "BASE:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "BASE:BOULMARE". See table 1022/104 which is an instructive entry

 

 

4C

PLACE NAME

PLACE

ENTER "PLACE:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "PLACE:OLFCOUPEVILLE". See table 1022/170 which is an instructive entry

 

 

4D

UNIT

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT NAME AS A LITERAL, FOR EXAMPLE: "UNIT:4077 MASH". See table 1022/57 which is an instructive entry

 

 

4E

UNIT CALL SIGN

CS

ENTER "CS:" FOLLOWED BY THE CALL SIGN OF THE UNIT OR AGENCY, FOR EXAMPLE: "CS:HAWKEYE". See table 1129/33 which is an instructive entry

5

COMMUNICATIONS MEANS PARAMETERS

 

PROVIDES THE COMMUNICATIONS MEANS  FOR THE POINT OF CONTACT IDENTIFIED IN FIELD ONE  USING ONE OF THE FOLLOWING:

 

 

5A

NON-SECURE TELEPHONE

TEL

ENTER "TEL:" FOLLOWED BY THE DESIGNATED NONSECURE TELEPHONE NUMBER, FOR EXAMPLE: "TEL:804-555-4142". See table 1361/3 which is an instructive entry

 

 

5B

SECURE TELEPHONE NUMBER

SECTEL

ENTER "SECTEL:" FOLLOWED BY THE DESIGNATED SECURE TELEPHONE NUMBER, FOR EXAMPLE: "SECTEL:555-4311". See table 1361/4 which is an instructive entry

 

 

5C

RADIO FREQUENCY

FRQ

ENTER "FRQ:" FOLLOWED BY THE FREQUENCY, FOR EXAMPLE: "FRQ:35.25MHZ" See table 2064 which is a composite

 

 

5D

FREQUENCY DESIGNATOR

FRQDES

ENTER "FRQDES:" FOLLOWED BY THE FREQUENCY DESIGNATOR,  FOR EXAMPLE: "FRQDES:RED1". See table 1396/1 which is an instructive entry

 

 

5E

FREQUENCY MODE AND NUMERIC

FREQ

ENTER "FREQ:" FOLLOWED BY THE FREQUENCY VALUE FOR RADIO TRANSMISSION, AND AN INDICATOR OF THE FREQUENCY MODE, FOR EXAMPLE: "FREQ:U5025". See table 2498 which is a composite

 

 

5F

ELECTRONIC MAIL ADDRESS

EMAIL

ENTER "EMAIL:" FOLLOWED BY THE IDENTIFIER USED BY SENDERS AND RECIPIENTS OF ELECTRONIC MAIL(E-MAIL) OVER COMPUTER NETWORKS, FOR EXAMPLE: "EMAIL:EWINGP(AT)NYKNICKS.COM". See table 1183/7 which is an instructive entry

 

 

5G

SECURE ELECTRONIC MAIL ADDRESS

SMAIL

ENTER "SMAIL:" FOLLOWED BY THE IDENTIFIER USED BY SENDERS AND RECIPIENTS OF SECURE ELECTRONIC MAIL(E-MAIL) OVER COMPUTER NETWORKS, FOR EXAMPLE: "SMAIL:EWINGP(AT)NYKNICKS.SMIL.MIL". See table 1183/9 which is an instructive entry

 

 

5H

NON-SECURE FACSIMILE (FAX) NUMBER

FAX

ENTER "FAX:" FOLLOWED BY THE DESIGNATED NONSECURE FAX NUMBER, FOR EXAMPLE:"FAX:0032-2-707-5834". See table 1361/7 which is an instructive entry

 

 

5I

SECURE FACSIMILE (FAX) NUMBER

SECFAX

ENTER "SECFAX:" FOLLOWED BY THE DESIGNATED SECURE FAX NUMBER, FOR EXAMPLE:"SECFAX:0032-2-707-4444". See table 1361/8 which is an instructive entry
