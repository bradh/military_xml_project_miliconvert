Concept/Explanation/Examples

1

EMISSION CONTROL PLAN

 

PROVIDES THE EMISSION CONTROL PLAN IDENTIFICATION, EITHER BY PHONETIC ALPHABET IDENTIFIER OR BY NUMERIC IDENTIFIER.

 

 

1A

EMISSION CONTROL PLAN PHONETIC IDENTIFIER

PHID

ENTER "PHID:" FOLLOWED BY THE EMISSION CONTROL PLAN PHONETIC IDENTIFIER, FOR EXAMPLE: "PHID:BRAVO". See table 1128/11 which contains a list of data items and associated data codes

 

 

1B

EMISSION CONTROL PLAN IDENTIFIER

CPID

ENTER "CPID:" FOLLOWED BY THE EMISSION CONTROL PLAN IDENTIFIER, FOR EXAMPLE: "CPID:A33". See table 1128/8 which is an instructive entry
