RANGE LIMITATION INDICATOR

 

ONE DIGIT IS USED TO SPECIFY THE AMMUNITION TO WHICH THE MAXIMUM RANGE IN FIELD 2 APPLIES, FOR EXAMPLE: "1". See table 1117/2 which contains a list of data items and associated data codes

2

MAXIMUM RANGE, TENS OF METRES

 

 

 

 

2A

MAXIMUM RANGE, TENS OF METRES

 

ONE TO FOUR DIGITS ARE USED TO SPECIFY THE MAXIMUM RANGE IN TENS OF METRES FOR THE AMMUNITION TYPE SPECIFIED IN FIELD 1, FOR EXAMPLE: "2400". See table 1563/2 a range [0 through 9999]
