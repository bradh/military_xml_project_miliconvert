No

Designator

Field Desc

Concept/Explanation/Examples

1

UNIT DESIGNATOR

 

PROVIDES AN IDENTIFICATION OF THE FRIENDLY UNIT, USING ITS OFFICIAL DESIGNATOR.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT DESIGNATION OF A MILITARY UNIT, FOR EXAMPLE: "UNIT:TASK FORCE DELTA". See table 1028/43 which is an instructive entry

 

 

1B

TASKED UNIT DESIGNATOR

DESIG

ENTER "DESIG:" FOLLOWED BY THE IDENTIFYING DESIGNATOR OF A SPECIFIC FRIENDLY NAVAL TASK ORGANIZATION, FOR EXAMPLE: "DESIG:CTG122.2". See table 2143 which is a composite

2

GEODETIC DATUM

 

 

 

 

2A

GEODETIC DATUM

 

IF THE DEFAULT GEODETIC DATUM SPECIFIED IN THE BEGINNING OF THE MESSAGE DOES NOT APPLY TO THE LOCATION OF THE FRIENDLY FORCES, ENTER THE CORRECT GEODATUM, FOR EXAMPLE: "NAR-A". See table 1195/1 which contains a list of data items and associated data codes

3

POSITION DAY-TIME

 

 

 

 

3A

DAY-TIME OF POSITION

 

ENTER THE DAY OF A MONTH AND THE ZULU TIME, FOR EXAMPLE: "141325Z". See table 2000 which is a composite

4

COURSE AND SPEED

 

PROVIDES THE ESTIMATED COURSE IN DEGREES TRUE OF THE FRIENDLY UNIT, AS WELL AS ITS SPEED, EXPRESSED EITHER IN KNOTS OR AS A SUBJECTIVE DESCRIPTION.

 

 

4A

FRIENDLY FORCE COURSE (TRUE) AND SPEED (KNOTS)

SPEED

ENTER "SPEED:" FOLLOWED BY THE DIRECTION OF TRAVEL MEASURED IN DEGREES TRUE AND THE MEASURED OR ESTIMATED SPEED/VELOCITY OF A FRIENDLY FORCE IN KNOTS, FOR EXAMPLE: "SPEED:270T12KTS". See table 2037 which is a composite

 

 

4B

COURSE AND APPROXIMATE SPEED

RSPEED

ENTER "RSPEED:" FOLLOWED BY THE DIRECTION IN DEGREES TRUE AND RELATIVE SPEED OF THE FRIENDLY FORCE, FOR EXAMPLE: "RSPEED:270TMED". See table 2270 which is a composite

5

COUNTRY CODE

 

 

 

 

5A

GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE FOR THE COUNTRY OF THE FRIENDLY FORCES, FOR EXAMPLE: "US". See table 1026/1 which contains a list of data items and associated data codes

6

TRACK IDENTIFIER

 

 

 

 

6A

TRACK IDENTIFIER

 

ENTER THE FOUR DIGIT DISCRETE IDENTIFIER USED IN MARITIME REPORTING OF TRACKS WHICH ARE NOT IDENTIFIED AS "FRIEND" OR "UNKNOWN, ASSUMED FRIEND".  FOR CONSTRUCTION REFER TO DISCRETE IDENTIFIER CODES, FOR EXAMPLE: "1234". See table 1012/60 which is an alphanumeric range

7

FRIENDLY FORCE LOCATION

 

THE FRIENDLY FORCE GEOGRAPHIC LOCATION.

 

 

7A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

7B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

7C

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME OF THE LOCATION OF THE FRIENDLY FORCES, FOR EXAMPLE: "LOC:POGGIO RENATICO". See table 1022/4 which is an instructive entry

 

 

7D

BEARING AND DISTANCE FROM REFERENCE POINT

TBR

ENTER "TBR:" FOLLOWED BY THE BEARING DISTANCE FROM REF POINT, FOR EXAMPLE: "TBR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

7E

GEOGRAPHIC POSITION, GEOREF, MINUTE

GEOREFM

ENTER "GEOREFM:" FOLLOWED BY THE LOCATION OF THE ROUTE POINT USING GEOREF MINUTE, FOR EXAMPLE: "GEOREFM:AABB1535". See table 2017 which is a composite

 

 

7F

PORT CODE

PORT

ENTER "PORT:" FOLLOWED BY THE CODE THAT IDENTIFIES THE PORT CONTAINING THE FRIENDLY FORCE, FOR EXAMPLE: "PORT:AAA". See table 1182/1 which is an instructive entry
