OBJECTIVE OF MINEFIELD

OBJECTIVE OF MINEFIELD

 

ENTER OBJECTIVE OF MINEFIELD WHAT IS THE MINEFIELD'S PART IN THE OVERALL PLAN OR CAMPAIGN, FOR EXAMPLE: "BLK". See table 1160/4 which contains a list of data items and associated data codes

CLASS NAME OF THE PRIMARY SHIP TARGET

CLASS NAME OF THE PRIMARY SHIP TARGET

 

ENTER CLASS NAME [LITERAL] OF THE SHIP IDENTIFIED AS THE PRIMARY TARGET AGAINST WHICH THE MINE FIELD IS PLANNED, FOR EXAMPLE: "BATTLESHIP". See table 1022/209 which is an instructive entry

SPEED OF PRIMARY TARGET, KNOTS

LEVEL OF EXPECTED MCM

7

MMOE SIMPLE INITIAL THREAT PROBABILITY

 

 

 

 

7A

MMOE SIMPLE INITIAL THREAT PROBABILITY

 

IF USED, ENTER MMOE SIMPLE INITIAL THREAT PROBABILITY [.01-.99] WHAT IS THE PLANNED OR ESTIMATED PROBALILITY THAT THE FIRST TARGET SHIP WILL ENTER MINEFIELD, FOR EXAMPLE: ".50". See table 1064/29 a range [0.01 through 0.99] (2 to 2 decimal places)

8

AVERAGE NUMBER OF CASUALTIES

 

 

 

 

8A

AVERAGE NUMBER OF CASUALTIES

 

IF USED, ENTER THE AVERAGE NUMBER OF CASUALTIES WHAT MEANS THE PLANNED OR ESTIMATED AVERAGE NUMBER OF CASUALTIES IN A GIVEN NUMBER OF TRANSITS THAT WOULD OCCUR IN A LARGE NUMBER OF TRANSIT ATTEMPTS, FOR EXAMPLE: "12OF99". See table 2236 which is a composite

9

MMOE SUSTAINED THREAT PROBABILITY

 

 

 

 

9A

MMOE SUSTAINED THREAT PROBABILITY

 

IF USED, ENTER THE CONSTANT THREAT PROBABILITY [.00-.99] PLANNED OR ESTIMATED FOR THE NUMBER OF DAYS, FOR EXAMPLE: ".20". See table 1064/30 a range [0.01 through 0.99] (2 to 2 decimal places)

10

PLANNED SUSTAINED THREAT OF MINEFIELD IN DAYS

 

 

 

 

10A

PLANNED SUSTAINED THREAT OF MINEFIELD IN DAYS

 

IF USED, ENTER PLANNED SUSTAINED THREAT OF MINEFIELD IN DAYS [1-365], FOR EXAMPLE: "12". See table 1055/8 a range [1 through 365]

11

EXPECTED NUMBER OF TRANSITORS

 

 

 

 

11A

EXPECTED NUMBER OF TRANSITORS

 

IF USED, ENTER EXPECTED NUMBER OF TRANSITORS [1-9999], FOR EXAMPLE: "500". See table 1044/193 a range [1 through 9999]
