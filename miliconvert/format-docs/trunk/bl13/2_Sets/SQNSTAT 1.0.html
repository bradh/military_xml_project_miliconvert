1

UNIT IDENTIFICATION

 

THE CODE FOR THE UNIT TO BE IDENTIFIED.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE IDENTIFICATION FOR THE UNIT ON THE BASE BEING REPORTED, FOR EXAMPLE: "UNIT:CHARLIE". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (USING THE NATO 2 LETTER COUNTRY CODE), FOR THE SPECIFIC UNIT ON THE BASE BEING REPORTED, FOR EXAMPLE: "UIC2:GEF4146". See table 2243 which is a composite

 

 

1C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (USING THE NATO 3 LETTER COUNTRY CODE), FOR THE SPECIFIC UNIT ON THE BASE BEING REPORTED, FOR EXAMPLE: "UIC3:DEUF4146". See table 2241 which is a composite

2

AIRCRAFT TYPE, MODEL OR CODE

 

THE TYPE, MODEL OR CODE OF AIRCRAFT ASSIGNED TO THE UNIT.

 

 

2A

AIRCRAFT BY TYPE, MODEL OR CODE

AC

ENTER "AC:" FOLLOWED BY THE TYPE OF AIRCRAFT ASSIGNED TO THE UNIT, TAKEN FROM THE MASTER LIST, FOR EXAMPLE:"AC:F16". See table 1015/20 which contains a list of data items and associated data codes

 

 

2B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE OF AIRCRAFT ASSIGNED TO THE UNIT, NOT YET INSERTED IN THE MASTER LIST, FOR EXAMPLE:"OTHAC:TYPHOON". See table 1015/18 which is an instructive entry

3

AIRCRAFT ROLE

 

 

 

 

3A

AIRCRAFT ROLE, GENERAL

 

ENTER THE ROLE FOR THE AIRCRAFT ASSIGNED, FOR EXAMPLE: "DEF". See table 1010/1 which is an instructive entry

4

UNIT ASSIGNEMENT STATUS

 

 

 

 

4A

UNIT ASSIGNMENT STATUS

ASN

ENTER "ASN:" FOLLOWED BY THE ASSIGNMENT STATUS FOR THE UNIT, FOR EXAMPLE: "ASN:A" See table 1229/1 which contains a list of data items and associated data codes

5

UNIT ESTABLISHMENT (NUMBER OF AIRCRAFT)

 

 

 

 

5A

NUMBER OF AIRCRAFT, 1-99

UE

ENTER "UE:" FOLLOWED BY THE NUMBER OF AIRCRAFT ASSIGNED TO THE UNIT, FOR EXAMPLE: "UE:18" See table 1044/322 a range [1 through 99]

6

NUMBER OF AIRCRAFT REMAINING

 

 

 

 

6A

NUMBER OF AIRCRAFT AVAILABLE

AR

ENTER "AR:" FOLLOWED BY THE NUMBER OF AIRCRAFT REMAINING AT THE UNIT, FOR EXAMPLE: "AR:15". See table 1044/8 a range [0 through 999]

7

TOTAL NUMBER OF AIRCRAFT LOST

 

 

 

 

7A

NUMBER OF AIRCRAFT LOST OR DESTROYED

LOST

ENTER "LOST:" FOLLOWED BY THE NUMBER OF AIRCRAFT LOST BY THE UNIT, FOR EXAMPLE: "LOST:3". See table 1044/17 a range [0 through 999]

8

AIRCRAFT DEPLOYED OR RECEIVED

 

 

 

 

8A

NUMBER OF AIRCRAFT, QUALIFIED

 

ENTER THE NUMBER OF AIRCRAFT DEPLOYED OR RECEIVED, FOR EXAMPLE: "D15". See table 2147 which is a composite

9

DEPLOYED OR RECEIVED AIRCRAFT LOCATION

 

THE LOCATION FOR THE DEPLOYED AIRCRAFT OR THE ORIGIN OF THE RECEIVED AIRCRAFT.  CAN ONLY BE USED IN CONJUNCTION WITH AIRCRAFT DEPLOYED OR RECEIVED.

 

 

9A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

9B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

9C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

9D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

9E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

9F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

9G

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR OF THE DEPLOYED AIRCRAFT OR THE ICAO LOCATION INDICATOR OF THE RECEIVER AIRCRAFT, FOR EXAMPLE: "ICAO:KILA". See table 1125/1 which is an instructive entry

 

 

9H

BASE NAME

BASE

ENTER "BASE:" FOLLOWED BY THE NAME OF THE AIR BASE, FOR EXAMPLE: "NAME:RAMSTEIN AB". See table 1022/104 which is an instructive entry

10

AIR-TO-AIR REFUELING CAPABILITY

 

 

 

 

10A

AIR-TO-AIR REFUELLING CAPABILITY (RECEIVER AND TANKER)

AAR

ENTER "AAR:" FOLLOWED BY THE AIR-TO-AIR REFUELLING CAPABILITY OF AIRCRAFT ASSIGNED TO THE UNIT, FOR EXAMPLE: "AAR:B". THIS FIELD APPLIES TO BOTH RECEIVERS AND CAPABILITY TANKERS. See table 1579/3 which contains a list of data items and associated data codes

11

NUMBER OF COMBAT READY AIRCRAFT

 

 

 

 

11A

NUMBER OF COMBAT READY AIRCRAFT

CR

ENTER "CR:" FOLLOWED BY THE NUMBER OF COMBAT READY AIRCRAFT AT THE UNIT, FOR EXAMPLE: "CR:12". See table 1044/215 a range [0 through 999]

12

NON COMBAT READY AIRCRAFT

 

 

 

 

12A

NUMBER AND STATUS OF NON MISSION CAPABLE AIRCRAFT

READY

ENTER "READY:" FOLLOWED BY THE READINESS OF THE AIRCRAFT AT THE UNIT, FOR EXAMPLE: "READY:3-48H". IF IT EQUALS "VAR", THEN AN AMPN SET IS REQUIRED. See table 2462 which is a composite
