1

UNIT OR ORGANIZATION

 

PROVIDES THE UNIT OR THE ORGANIZATION TO WHICH A DUTY IS ASSIGNED.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT NAME, FOR EXAMPLE: "UNIT:POGGIO RENATICO". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UNIT IDENTIFICATION CODE, FOR EXAMPLE: "UIC:GEF1234". See table 2141 which is a composite

 

 

1C

SHIP NAME

SHIP

ENTER "SHIP:" FOLLOWED BY THE SHIP NAME, FOR EXAMPLE: "SHIP:USS KENNEDY". See table 1022/49 which is an instructive entry

 

 

1D

NAVAL ORGANIZATION DESIGNATOR

NAVORG

ENTER "NAVORG:" FOLLOWED BY THE NAVAL ORGANIZATION DESIGNATOR, FOR EXAMPLE: "NAVORG:TG500.1". See table 2143 which is a composite

 

 

1E

INTERFACE UNIT DESIGNATOR

IUD

ENTER "IUD:" FOLLOWED BY THE INTERFACE UNIT DESIGNATOR, FOR EXAMPLE: "IUD:CRC2222". See table 2367 which is a composite

2

TACTICAL DUTIES

 

PROVIDES THE TACTICAL DUTY OR DUTIES THAT HAVE BEEN  ASSIGNED TO THE UNIT OR ORGANIZATION IDENTIFIED IN FIELD 1.

 

 

2A

AIR COMMAND AND CONTROL DUTY ASSIGNED

CCDASN

ENTER "CCDASN:" FOLLOWED BY THE CODE FOR THE COMMAND AND CONTROL DUTY ASSIGNED, FOR EXAMPLE: "CCDASN:ASO". See table 1109/4 which contains a list of data items and associated data codes

 

 

2B

DUTY

DUTY

ENTER "DUTY:" FOLLOWED BY THE ASSIGNED DUTY, FOR EXAMPLE: "DUTY:10". See table 1109/1 which contains a list of data items and associated data codes

 

 

2C

LINK DUTY

LNK

ENTER "LNK:" FOLLOWED BY THE ASSIGNED LINK DUTY, FOR EXAMPLE: "LNK:87". See table 1109/2 which contains a list of data items and associated data codes

 

 

2D

MULTILINK COORDINATION DUTY

MLNK

ENTER "MLNK:" FOLLOWED BY THE ASSIGNED MULTILINK COORDINATION DUTY, FOR EXAMPLE: "MLNK:RTDC". See table 1109/3 which contains a list of data items and associated data codes

 

 

2E

DUTY, OTHER

OTH

ENTER "OTH:" FOLLOWED BY THE DUTY ASSIGNED WHOSE CODE IS NOT PRESENT IN THE EXISTING LISTS, FOR EXAMPLE: "OTH:ELECTRONIC WARFARE COMMANDER". See table 1109/5 which is an instructive entry
