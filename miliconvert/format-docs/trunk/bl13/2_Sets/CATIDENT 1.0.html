Concept/Explanation/Examples

1

INTELLIGENCE CATEGORY KEY

 

ENTER THE RESPECTIVE KEY TO IDENTIFY THE CATEGORY (PLACE, EQUIPMENT, UNIT, EVENT, BIOGRAPHIC) THAT THE RELATIONSHIP RECORD IS LINKED TO:

 

 

1A

PLACE KEY

 

ENTER THE UNIQUE IDENTIFIER FOR A GIVEN PLACE, FOR EXAMPLE: "0191-UK12-0001-GE001". See table 2441 which is a composite

 

 

1B

EQUIPMENT KEY

 

ENTER THE UNIQUE IDENTIFIER FOR A GIVEN EQUIPMENT, FOR EXAMPLE: "Q-CC-AU99-A0001". See table 2442 which is a composite

 

 

1C

UNIT KEY

 

ENTER THE UNIT KEY, FOR EXAMPLE: "U-GE12-A-BE-AB123". See table 2443 which is a composite

 

 

1D

EVENT KEY

 

ENTER THE EVENT KEY, FOR EXAMPLE: "E-2000-GE02-12345". See table 2444 which is a composite

 

 

1E

BIOGRAPHIC KEY

 

ENTER THE BIOGRAPHIC KEY, FOR EXAMPLE: "B-UKXX-12345". See table 2445 which is a composite

2

ENTITY NAME

 

ENTER THE NAME OF THE ENTITY THAT IS THE SUBJECT OF THE RELATIONSHIP RECORD IDENTIFIED IN THE KEY FIELD.  THE CORRESPONDING ALTERNATIVE ANALOG TO THE ALTERNATIVE USED IN FIELD 1 (A-A; B-B; C-C, D-D, E-E) MUST BE USED.

 

 

 

2A

PLACE NAME, EXTENDED

 

ENTER A LITERAL (UP TO 54 CHARACTERS) GIVING THE GENERALLY UNDERSTOOD NAME OF THE PLACE, FOR EXAMPLE: "DEVIL'S HORN". See table 1022/261 which is an instructive entry

 

 

2B

EQUIPMENT NOMENCLATURE

 

ENTER  A  LITERAL ("ANALYST'S SHORTHAND")  DESCRIBING THE EQUIPMENT, FOR EXAMPLE: "ENG,TURBOFAN,CHINESE". See table 1714/2 which is an instructive entry

 

 

2C

UNIT NAME, EXPANDED

 

ENTER A LITERAL GIVING THE FULL (NATIVE) UNIT NAME, FOR EXAMPLE: "KOMMANDO LUFTBEWEGLICHE KRAEFTE." See table 1022/253 which is an instructive entry

 

 

2D

EVENT NAME

 

ENTER A LITERAL TO SPECIFY THE NAME OF THE RECOGNIZED EVENT, "JOINT TRANSPORTATION EXERCISE". See table 1022/254 which is an instructive entry

 

 

2E

LAST NAME

 

ENTER THE LAST NAME OF THE INDIVIDUAL, FOR EXAMPLE: "MAIER". See table 1022/255 which is an instructive entry

3

GENDER, ABBREVIATED

 

THIS FIELD MUST BE USED, IF ALTERNATIVE "E" (BIOGRAPHIC KEY) IS USED IN FIELD 1.

 

 

3A

GENDER, ABBREVIATED

 

ENTER THE INDIVIDUAL'S SEX USING AN ABBREVIATED CODE, FOR EXAMPLE: "M". See table 1286/13 which contains a list of data items and associated data codes

4

EVENT TYPE

 

THIS FIELD MUST BE USED, IF ALTERNATIVE "A" (PLACE KEY) IS USED IN FIELD 1.

 

 

4A

EVENT TYPE

 

ENTER A CODE TO IDENTIFY THE EVENT TYPE, FOR EXAMPLE: "MILMOV". See table 1211/8 which contains a list of data items and associated data codes

5

COUNTRY

 

THIS FIELD MUST BE USED, IF ALTERNATIVE "A" (PLACE KEY) IS USED IN FIELD 1.

 

 

5A

COUNTRY

 

ENTER A CODE TO IDENTIFY THE COUNTRY IN WHICH THE PLACE IS LOCATED, FOR EXAMPLE: "GE". See table 1026/18 which contains a list of data items and associated data codes

6

FUNCTIONAL CODE

 

THIS FIELD MUST BE USED, IF ALTERNATIVE "A" (PLACE KEY) IS USED IN FIELD 1.

 

 

6A

FUNCTIONAL CODE

 

ENTER A FIVE DIGIT NUMERICAL CODE TAKE FROM AINTP-3, FOR EXAMPLE: "43304". See table 1496/3 a range [00000 through 99999]
