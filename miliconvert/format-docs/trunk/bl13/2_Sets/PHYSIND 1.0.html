1

PHOTO AVAILABLE

 

 

 

 

1A

PHOTO AVAILABLE

 

THE VALUE GIVEN IN THIS FIELD INDICATES WHETHER A PHOTO OF THE INDIVIDUAL IS AVAILABLE OR NOT, FOR EXAMPLE: "Y". See table 1359/26 which contains a list of data items and associated data codes

2

HEIGHT OF PERSON IN METRES

 

 

 

 

2A

HEIGHT OF PERSON IN METRES

 

INDICATES THE MEASUREMENT OF THE STATURE OF THE INDIVIDUAL IN METRES [0.55 TO 2.55], FOR EXAMPLE: "1.82". See table 1651/13 a range [0.00 through 2.99] (2 to 2 decimal places)

3

WEIGHT OF PERSON IN KILOGRAMS

 

 

 

 

3A

WEIGHT OF PERSON IN KILOGRAMS

 

INDICATES THE MEASUREMENT OF THE WEIGHT OF THE INDIVIDUAL IN KILOGRAMS [20 TO 200], FOR EXAMPLE: "76". See table 1194/4 a range [0 through 500]

4

COLOUR OF EYES

 

 

 

 

4A

COLOUR OF EYES

 

INDICATES THE PREDOMINANT COLOUR OF THE INDIVIDUAL'S EYES, FOR EXAMPLE: "BLUE". See table 1138/11 which contains a list of data items and associated data codes

5

COLOUR OF HAIR

 

 

 

 

5A

COLOUR OF HAIR

 

INDICATES THE PREDOMINANT COLOUR OF THE INDIVIDUAL'S HAIR, FOR EXAMPLE: "PINK". See table 1138/12 which contains a list of data items and associated data codes

6

COMMENTS

 

 

 

 

6A

COMMENTS

 

THIS FIELD ALLOWS THE RECORDING OF ANY OTHER DATA [LITERAL] WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS IN THIS SET, FOR EXAMPLE: "(REMARKS IN FREE TEXT UP TO 68 CHARACTERS)". See table 1357/1 which is an instructive entry
