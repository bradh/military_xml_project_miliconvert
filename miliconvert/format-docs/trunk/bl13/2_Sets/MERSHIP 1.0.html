No

Designator

Field Desc

Concept/Explanation/Examples

1

MERCHANT VESSEL TYPE

 

 

 

 

1A

MERCHANT VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE TYPE OF MERCHANT OR FISHING VESSEL, FOR EXAMPLE: "TME". See table 1166/10 which contains a list of data items and associated data codes

2

MERSHIP IDENTIFIER

 

SPECIFIES THE MERCHANT VESSEL'S NAME OR VESSEL CLASS. IF BOTH ARE KNOWN BUT NOT PREVIOUSLY REPORTED, ENTER VESSEL NAME IN THIS FIELD AND SPECIFY VESSEL CLASS IN A FOLLOW-ON AMPN SET.

 

 

2A

SHIP NAME

 

ENTER THE SHIP NAME, FOR EXAMPLE: "KAPITAN SMIRNOV". See table 1022/49 which is an instructive entry

 

 

2B

NAME OF SHIP CLASS

CLASS

ENTER FIELD DESCRIPTOR "CLASS:" FOLLOWED BY THE NAME OF THE SHIP CLASS, FOR EXAMPLE: "CLASS:ATLANTIK II". See table 1022/35 which is an instructive entry

3

NATIONALITY

 

 

 

 

3A

GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE FOR THE NATIONALITY OF THE VESSEL, FOR EXAMPLE: "RS". See table 1026/1 which contains a list of data items and associated data codes

4

POSITION

 

SPECIFIES THE GEOGRAPHIC POSITION OF THE VESSEL.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

PLACE NAME

NAME

ENTER FIELD DESCRIPTOR "NAME:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "NAME:CATALZEYTIN". See table 1022/170 which is an instructive entry

5

DATE-TIME OF CONTACT

 

SPECIFIES THE DATE AND/OR TIME OF CONTACT.

 

 

5A

DAY-TIME OF CONTACT

 

ENTER THE DAY-TIME OF CONTACT, FOR EXAMPLE: "162215Z". See table 2000 which is a composite

 

 

5B

DTG

 

ENTER THE DATE-TIME-GROUP OF CONTACT, FOR EXAMPLE "162215ZSEP2002". See table 2033 which is a composite

 

 

5C

DAY-MONTH NAME

 

IF CONTACT TIME IS UNKNOWN, ENTER DAY-MONTH NAME, FOR EXAMPLE "16SEP". See table 2067 which is a composite

6

CONTACT MOVEMENT

 

SPECIFIES THE MOVEMENT OF THE CONTACT.

 

 

6A

COURSE IN DEGREES TRUE AND SPEED IN KNOTS

 

ENTER THE CONTACT'S COURSE IN DEGREES TRUE, FOLLOWED BY A HYPHEN, FOLLOWED BY ITS SPEED IN KNOTS,  FOR EXAMPLE: "265-13". See table 2025 which is a composite

 

 

6B

MOVEMENT ACTIVITY

 

ENTER THE CONTACT'S MOVEMENT ACTIVITY, FOR EXAMPLE: "DIW". See table 1095/4 which contains a list of data items and associated data codes

7

CARGO INDICATOR

 

 

 

 

7A

CARGO INDICATOR

 

ENTER THE INDICATOR THAT DESCRIBES THE VESSEL'S CARGO, FOR EXAMPLE: "POL". See table 1173/4 which contains a list of data items and associated data codes

8

CONTACT VALIDITY

 

 

 

 

8A

CONTACT VALIDITY

 

ENTER THE CODE SPECIFYING THE VALIDITY OF THE CONTACT REPORT, FOR EXAMPLE: "CB". See table 2049 which is a composite
