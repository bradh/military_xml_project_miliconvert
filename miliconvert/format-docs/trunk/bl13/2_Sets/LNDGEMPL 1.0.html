LAUNCH UNIT NAME

 

 

 

 

1A

LAUNCH UNIT NAME

 

ENTER THE NAME OF THE SHIP LAUNCHING THE LANDING CRAFT TO BE LOADED, FOR EXAMPLE:  "PELELIU" See table 1022/333 which is an instructive entry

2

LANDING CRAFT TYPE

 

 

 

 

2A

LANDING CRAFT TYPE

 

ENTER THE TYPE OF LANDING CRAFT TO BE LOADED, FOR EXAMPLE: "LCVP". See table 1166/6 which contains a list of data items and associated data codes

3

WAVE ASSIGNMENT

 

 

 

 

3A

WAVE ASSIGNMENT

 

ENTER THE WAVE ASSIGNMENT NUMBER, FOR EXAMPLE: "2-10". See table 2460 which is a composite

4

LOADING SHIP NAME

 

 

 

 

4A

LOADING SHIP NAME

 

ENTER THE NAME OF THE LOADING SHIP, FOR EXAMPLE: "OCEAN". See table 1022/334 which is an instructive entry

5

LOADING TIME

 

SPECIFIES THE TIME AT WHICH THE LANDING CRAFT WILL ARRIVE AT THE LOADING SHIP.

 

 

5A

RELATIVE TIME IN HOURS AND MINUTES

 

ENTER THE LANDING CRAFT ARRIVAL TIME RELATIVE TO H-HOUR, FOR EXAMPLE: "H-0100". See table 2482 which is a composite

 

 

5B

DTG OF ARRIVAL

 

ENTER THE LANDING CRAFT DATE-TIME GROUP OF ARRIVAL AT THE LOADING SHIP, FOR EXAMPLE: "120600ZFEB03". See table 2033 which is a composite

6

LANDING CARGO TYPE

 

 

 

 

6A

LANDING CARGO TYPE

 

ENTER THE TYPE OF CARGO TO BE LOADED, FOR EXAMPLE: "AMMO". See table 1173/14 which contains a list of data items and associated data codes

7

SERIAL ASSIGNMENT NUMBER

 

 

 

 

7A

SERIAL ASSIGNMENT NUMBER

 

ENTER THE NUMBER ASSIGNED TO THE CARGO SERIAL SPECIFIED IN THE PRECEDING FIELD, FOR EXAMPLE: "0004". See table 1012/213 a range [0000 through 9999]
