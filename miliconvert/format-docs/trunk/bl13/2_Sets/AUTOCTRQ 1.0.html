Concept/Explanation/Examples

1

TRANSMIT FREQUENCY

 

PROVIDES THE TRANSMISSION FREQUENCY, USING A FREQUENCY IN MHZ OR A LINE NUMBER.

 

 

1A

TRANSMIT FREQUENCY

TFREQ

ENTER "TFREQ:" FOLLOWED BY THE TRANSMIT FREQUENCY, FOR EXAMPLE: "TFREQ:241.3MHZ". See table 2064 which is a composite

 

 

1B

TRANSMIT LINE NUMBER

TLN

ENTER "TLN:" FOLLOWED BY THE TRANSMIT LINE NUMBER, FOR EXAMPLE: "TLN:NP3020". See table 1012/117 which is an instructive entry

2

RECEIVE FREQUENCY

 

PROVIDES THE RECEIVE FREQUENCY, USING A FREQUENCY IN MHZ OR A LINE NUMBER.

 

 

2A

RECEIVE FREQUENCY

RFREQ

ENTER "RFREQ:" FOLLOWED BY THE RECEIVE FREQUENCY, FOR EXAMPLE: "RFREQ:241.3MHZ". See table 2064 which is a composite

 

 

2B

RECEIVE LINE NUMBER

RLN

ENTER "RLN:" FOLLOWED BY THE RECEIVE LINE NUMBER, FOR EXAMPLE: "RLN:NP3021". See table 1012/118 which is an instructive entry

3

MINIMUM FLIGHT LEVEL/MINIMUM ALTITUDE IN HUNDREDS OF FEET

 

PROVIDES THE  MINIMUM FLIGHT LEVEL OR THE MINIMUM ALTITUDE IN HUNDREDS OF FEET OF THE RADIO RELAY STATION.

 

 

3A

MINIMUM ALTITUDE, HUNDREDS OF FEET

ALT

ENTER "ALT:" FOLLOWED BY THE MINIMUM ALTITUDE IN HUNDREDS OF FEET, FOR EXAMPLE: "ALT:445". See table 1255/4 a range [000 through 999]

 

 

3B

MINIMUM FLIGHT LEVEL

 

ENTER THE MINIMUM FLIGHT LEVEL FOR THE STATION, FOR EXAMPLE: "FL044". See table 2129 which is a composite

4

STATION POSITION

 

THE GEOGRAPHIC POSITION OF THE AIRBORNE RELAY STATION.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

REFERENCE POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE REFERENCE POINT NAME, FOR EXAMPLE: "NAME:ALPHA BRAVO". See table 1022/2 which is an instructive entry

 

 

4C

BEARING AND DISTANCE FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite
