1

PLATFORM TYPE

 

SPECIFIES THE TYPE OF PLATFORM REPORTING THE MINE EXPLOSION.

 

 

1A

SURFACE VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE SURFACE VESSEL TYPE, FOR EXAMPLE: "MC". See table 1166/7 which contains a list of data items and associated data codes

 

 

1B

SUBMARINE TYPE

SUB

ENTER FIELD DESCRIPTOR "SUB:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE SUBMARINE TYPE, FOR EXAMPLE: "SUB:SSN". See table 1166/2 which contains a list of data items and associated data codes

 

 

1C

AIRCRAFT TYPE

AIR

ENTER FIELD DESCRIPTOR "AIR:" FOLLOWED BY THE AIRCRAFT TYPE, FOR EXAMPLE: "AIR:HELO". See table 1015/1 which is an instructive entry

2

PLATFORM IDENTIFIER

 

SPECIFIES THE IDENTITY OF THE PLATFORM OR UNIT REPORTING THE MINE EXPLOSION.

 

 

2A

SHIP NAME

 

ENTER THE SHIP NAME, FOR EXAMPLE: "BRECON". See table 1022/49 which is an instructive entry

 

 

2B

SHIP CALL SIGN

CALLSIGN

ENTER FIELD DESCRIPTOR "CALLSIGN:" FOLLOWED BY THE SHIP CALL SIGN, FOR EXAMPLE: "CALLSIGN:8TG". See table 1129/7 which is an instructive entry

 

 

2C

NAVAL TASK DESIGNATOR

NTD

ENTER FIELD DESCRIPTOR "NTD:" FOLLOWED BY THE NAVAL TASK DESIGNATOR, CONSISTING OF THE NAVAL LEVEL OF ECHELON AND THE NAVAL ORGANIZATION NUMERIC DESIGNATOR, FOR EXAMPLE: "NTD:CTF401". See table 2143 which is a composite

3

GEODETIC DATUM

 

 

 

 

3A

GEODETIC DATUM

 

ENTER THE GEODETIC DATUM USED FOR POSITION REFERENCING, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

4

POSITION

 

SPECIFIES THE GEOGRAPHIC POSITION OF THE REPORTING PLATFORM AT THE TIME OF THE MINE EXPLOSION.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER THE POSITION AS GEOREF LOCATION, CENTIMINUTE, FOR EXAMPLE: "GEO:BBAA33443344". See table 2018 which is a composite

 

 

4H

PLACE NAME

NAME

ENTER FIELD DESCRIPTOR "NAME:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "NAME:ZULU". See table 1022/170 which is an instructive entry

 

 

4I

TRUE BEARING AND DISTANCE FROM REF POINT

TBR

ENTER THE POSITION AS A BEARING AND DISTANCE FROM A NAMED REFERENCE POINT, FOR EXAMPLE: "TBR:191.5T-LIZARD POINT LIGHT-47.4NM". See table 2047 which is a composite

 

 

4J

Q-ROUTE POSITION

QPOS

ENTER FIELD DESCRIPTOR "QPOS:" FOLLOWED BY THE Q-ROUTE POSITION, FOR EXAMPLE: "QPOS:171Z87.4MS645". See table 2203 which is a composite

DAY-TIME
