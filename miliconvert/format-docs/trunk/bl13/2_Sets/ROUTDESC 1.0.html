WEATHER RELATED ROUTE CONDITIONS

 

 

 

 

1A

WEATHER RELATED ROUTE CONDITIONS

 

ENTER THE WEATHER RELATED CHARACTERISTICS OF THE ROUTE (AS A CODE), FOR EXAMPLE: "AWR". See table 1848/3 which contains a list of data items and associated data codes

2

WIDTH OF TRAVELED WAY (FEET, YARDS OR METRES)

 

 

 

 

2A

WIDTH OF TRAVELED WAY (FEET, YARDS OR METRES)

 

ENTER THE TRAVELED WAY WIDTH OF THE HARD SURFACE OF THE ROAD IN EITHER FEET, YARDS OR METRES, FOR EXAMPLE: "20M". See table 2451 which is a composite

3

WIDTH OF GRADING (FEET, YARDS OR METRES)

 

 

 

 

3A

WIDTH OF GRADING (FEET, YARDS OR METRES)

 

ENTER THE WIDTH OF THE TRAVELED WAY PLUS THE WIDTH OF THE HARD SHOULDERS EITHER IN FEET, YARDS OR METRES, FOR EXAMPLE: "30M". See table 2451 which is a composite

4

ROUTE CONCEALMENT POTENTIAL

 

 

 

 

4A

ROUTE CONCEALMENT POTENTIAL

 

ENTER THE AVAILABILITY OF CONCEALMENT FROM THE AIR FOR GROUND FORCES TO OPERATE ALONG THE ROUTE, FOR EXAMPLE: "LIMITED". See table 1668/5 which contains a list of data items and associated data codes

5

MAXIMUM LOAD SUPPORTED IN METRIC TONS

 

 

 

 

5A

MAXIMUM LOAD SUPPORTED IN METRIC TONS

 

ENTER THE MAXIMUM LOAD IN METRIC TONS THE ROUTE CAN SUPPORT, AS DETERMINED BY QUALIFIED ENGINEERING PERSONNEL, FOR EXAMPLE: "80". See table 1855/1 a range [0 through 999]
