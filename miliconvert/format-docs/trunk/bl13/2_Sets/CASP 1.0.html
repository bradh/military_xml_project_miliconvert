Concept/Explanation/Examples

1

SHIP ID

 

 

 

 

1A

SHIP IDENTIFICATION

 

ENTER THE IDENTIFICATION OF THE SHIP BY TYPE AND HULL NUMBER, FOR EXAMPLE: "CVN-F2345". See table 2021 which is a composite

2

SHIP CLASS

 

 

 

 

2A

SHIP CLASS NAME

 

ENTER A LITERAL GIVING THE SHIP CLASS NAME, FOR EXAMPLE: "UDALOY". See table 1022/227 which is an instructive entry

3

SHIP START POSITION

 

THE GEOGRAPHIC LOCATION OF START OF THE SHIP DESIGNATED FOR CASP.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

4

SHIP START TIME

 

THE DAY-TIME AND MONTH OR DTG OF START OF THE SHIP DESIGNATED FOR CASP FROM THE POSITION AT SEQ. 3.

 

 

4A

DTG OF START

 

ENTER THE DTG OF START OF THE SHIP DESIGNATED FOR CASP FROM THE POSITION AT SEQ. 3., FOR EXAMPLE: "182135ZNOV2004". See table 2033 which is a composite

 

 

4B

DAY-TIME AND MONTH OF START

 

ENTER THE DAY-TIME AND MONTH OF START OF THE SHIP DESIGNATED FOR CASP FROM THE POSITION AT SEQ. 3., FOR EXAMPLE: "182135ZNOV". See table 2030 which is a composite

5

SHIP END POSITION

 

THE GEOGRAPHIC LOCATION OF END OF THE SHIP DESIGNATED FOR CASP.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

6

SHIP STOP TIME

 

THE DAY-TIME AND MONTH OR DTG OF STOP.  CAN ONLY BE USED IN CONJUNCTION WITH SHIP END POSITION SPECIFIED IN THIS SET.

 

 

6A

DTG OF STOP

 

ENTER THE DTG OF STOP OF THE SHIP DESIGNATED FOR CASP AT THE POSITION AT SEQ. 3., FOR EXAMPLE: "221605ZNOV2004". See table 2033 which is a composite

 

 

6B

DAY-TIME AND MONTH OF STOP

 

ENTER THE DAY-TIME AND MONTH OF STOP OF THE SHIP DESIGNATED FOR CASP AT THE POSITION AT SEQ. 3., FOR EXAMPLE: "221605ZNOV". See table 2030 which is a composite

7

COORDINATED AIR/SEA PROCEDURE CATEGORY

 

 

 

 

7A

COORDINATED AIR SEA PROCEDURE CATEGORY

 

ENTER THE CODE FOR THE COORDINATED AIR/SEA PROCEDURES CATEGORY, FOR EXAMPLE: "3". See table 1814/1 which contains a list of data items and associated data codes

8

DTG OF START OF CASP DUTIES

 

 

 

 

8A

DTG OF START

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF THE START OF THE TRANSFER OF AUTHORITY TIMEFRAME, FOR EXAMPLE: "190800ZNOV2004". See table 2033 which is a composite

9

DTG OF STOP OF CASP DUTIES

 

 

 

 

9A

DTG OF STOP

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF THE STOP OF THE TRANSFER OF AUTHORITY TIMEFRAME, FOR EXAMPLE: "221325ZNOV2004". See table 2033 which is a composite

10

UNIT REMARKS

 

 

 

 

10A

REMARKS

 

ENTER, AS LITERAL, REMARKS ABOUT THE SHIP, FOR EXAMPLE: "TAC 24 HRS 7". See table 1357/28 which is an instructive entry
