TARGET SIZE

 

IDENTIFY DIMENSIONS OF THE TARGET EITHER AS LENGTH AND WIDTH OR RADIUS:

 

 

1A

TARGET DIMENSIONS (LENGTH-WIDTH)

TARGDIM

ENTER "TARGDIM:" FOLLOWED BY THE LINEAR DIMENSIONS OF THE TARGET (LENGTH-WIDTH) INCLUDING THE UNIT OF MEASURE, FOR EXAMPLE: "TARGDIM:400-300-M". See table 2683 which is a composite

 

 

1B

TARGET RADIUS, FT, YD, SM, NM, M, KM, HF

TARGRAD

ENTER THE ENTER "TARGRAD:" FOLLOWED BY RADIUS INFORMATION DESCRIBING THE DIMENSIONS OF A CIRCULAR TARGET, FOR EXAMPLE: "TARGRAD:7.5KM". See table 2178 which is a composite

2

TARGET HEIGHT, FEET OR METRES

 

THIS FIELD MUST EQUAL "CIR" IN THIS SET, IF FIELD 1 EQUALS ALTERNATIVE "B" "TARGRAD".

 

 

2A

TARGET HEIGHT, FEET OR METRES

 

ENTER A MEASUREMENT IN FEET OR METRES TO GIVE THE VERTICAL DIMENSION OF THE TARGET, FOR EXAMPLE: "32M". See table 2132 which is a composite
