No

Designator

Field Desc

Concept/Explanation/Examples

1

NATIONALITY OR COMMAND

 

SPECIFIES THE NATIONALITY OR COMMAND OF THE REQUIRED FORCE.

 

 

1A

NATIONALITY

 

ENTER THE NATIONALITY OF THE REQUIRED FORCE, FOR EXAMPLE: "US". See table 1026/2 which contains a list of data items and associated data codes

 

 

1B

NATO COMMAND DESIGNATOR

 

ENTER THE NATO DESIGNATOR FROM AAP1 OF THE COMMAND OF THE REQUIRED FORCE, FOR EXAMPLE: "ACE". See table 1515/3 which contains a list of data items and associated data codes

2

ARMED SERVICE

 

 

 

 

2A

ARMED SERVICE

 

ENTER THE CODE FOR THE ARMED SERVICE OF THE REQUIRED FORCE, FOR EXAMPLE: "F". See table 1107/1 which contains a list of data items and associated data codes

3

NUMBER OF UNITS

 

 

 

 

3A

NUMBER OF UNITS

 

ENTER THE NUMBER OF UNITS REQUIRED, FOR EXAMPLE: "5". See table 1044/102 a range [0 through 999]

4

TYPE OF UNIT

 

SPECIFIES THE TYPE OF UNIT REQUIRED.

 

 

4A

NAVAL VESSEL TYPE

SHIP

ENTER FIELD DESCRIPTOR "SHIP:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE NAVAL VESSEL TYPE, FOR EXAMPLE: "SHIP:DD". See table 1166/3 which contains a list of data items and associated data codes

 

 

4B

AIRCRAFT CODE

ACFT

ENTER FIELD DESCRIPTOR "ACFT:" FOLLOWED BY THE AIRCRAFT TYPE, FOR EXAMPLE: "ACFT:F18". See table 1015/2 which contains a list of data items and associated data codes

 

 

4C

ECHELON LEVEL

ECHLVL

ENTER FIELD DESCRIPTOR "ECHLVL:" FOLLOWED BY THE ECHELON LEVEL OF THE UNIT, FOR EXAMPLE: "ECHLVL:DIV". See table 1045/1 which contains a list of data items and associated data codes

 

 

4D

UNIT IDENTIFICATION CODE (UIC)

 

ENTER THE UNIT IDENTIFICATION CODE AS A NATIONALITY CODE, FOLLOWED BY AN ARMED SERVICE INDICATOR CODE, FOLLOWED BY THE ACE FILE SEQUENTIAL LOCATION NUMBER OF THE UNIT, FOR EXAMPLE: "USA123456". See table 2141 which is a composite
