1

AMMUNITION SHORT TITLE/SPECIFICATION NAME

 

 

 

 

1A

AMMUNITION SHORT TITLE SPECIFICATION NAME

 

ENTER THE LITERAL AMMUNITION SPECIFICATION NAME, FOR EXAMPLE: "GROUND MINE". See table 1022/86 which is an instructive entry

2

ORDNANCE WEAPON TYPE

 

 

 

 

2A

ORDNANCE WEAPON TYPE

 

ENTER A CODE TO DEFINE THE UXO WEAPON TYPE, FOR EXAMPLE: "MINES". See table 1407/14 which contains a list of data items and associated data codes

3

FUZE POSITION

 

 

 

 

3A

FUZE POSITION

 

ENTER A CODE TO PROVIDE THE SPECIFIC POSITION OF THE FUZE, FOR EXAMPLE: "FRONT". See table 1223/2 which contains a list of data items and associated data codes

4

FUZE INTERNAL INDICATOR

 

 

 

 

4A

FUZE INTERNAL INDICATOR

 

IF USED, ENTER THE INDICATOR ABOUT THE  INTERNAL POSITION OF FUZE, FOR EXAMPLE: "YES". See table 1027/37 which contains a list of data items and associated data codes

5

FUZE TYPE

 

 

 

 

5A

FUZE TYPE

 

IF USED, ENTER A DESCRIPTION OF THE TYPE OF FUZE ASSOCIATED WITH UXO, FOR EXAMPLE: "EXPLODER MKXX". See table 1192/1 which is an instructive entry

6

FUZE STATUS

 

 

 

 

6A

FUZE STATUS

 

IF USED, ENTER A CODE TO QUALIFY THE STATUS OF THE FUZE, FOR EXAMPLE: "02". See table 1130/38 which contains a list of data items and associated data codes
