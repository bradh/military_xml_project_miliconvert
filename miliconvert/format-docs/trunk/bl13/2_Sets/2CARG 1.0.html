Concept/Explanation/Examples

1

DATA ENTRY

 

 

 

 

1A

DATA ENTRY

 

ENTER THE TWO DIGIT NUMBER (01 THROUGH 99) THAT TIES THE INFORMATION IN THIS LINE TO RELATED DATA LINES IN SETS CARG AND 2 DANGER, IN ASCENDING ORDER, FOR EXAMPLE: "01". See table 1348/1 a range [01 through 99]

2

TYPE OF AIR CARGO

 

 

 

 

2A

TYPE OF AIR CARGO

 

ENTER THE DATA CODE FOR THE TYPE OF AIR CARGO, FOR EXAMPLE: "TANK".  WHEN EITHER AIRF OR HELI IS ENTERED, THE TYPE OF AIRCRAFT OR HELICOPTER IS TO BE STATED IN A SUBSEQUENT AMPN SET.  IF ROLL IS ENTERED, DETAILS SUCH AS TYPE, CENTER OF GRAVITY, WHEEL/AXLE LOAD, WHEELS/TRACKS, ETC..., MAY BE GIVEN IN A SUBSEQUENT AMPN SET. See table 1173/6 which contains a list of data items and associated data codes

3

NUMBER OF CARGO ITEMS

 

 

 

 

3A

NUMBER OF CARGO ITEMS

 

ENTER THE QUANTITY OF ITEMS (1 THROUGH 999) ENTERED IN FIELD 2, FOR EXAMPLE: "5". See table 1044/149 a range [1 through 999]

4

DANGEROUS CARGO INDICATOR

 

 

 

 

4A

DANGEROUS CARGO INDICATOR

 

ENTER THE DATA CODE THAT INDICATES IF DANGEROUS CARGO IS BEING TRANSPORTED, FOR EXAMPLE: "Y". IF ENTRY EQUALS "Y" THEN INFORMATION ON THE NATURE OF THE DANGEROUS CARGO IS TO BE GIVEN IN SUBSEQUENT SET 2 DANGER. See table 1359/4 which contains a list of data items and associated data codes

5

EXTERNAL CARGO INDICATOR

 

 

 

 

5A

EXTERNAL CARGO INDICATOR

 

ENTER THE DATA CODE THAT INDICATES IF CARGO IS TO BE CARRIED EXTERNALLY TO THE AIRCRAFT, FOR EXAMPLE: "Y". See table 1359/2 which contains a list of data items and associated data codes

6

INDIVIDUAL CARGO WEIGHT

 

 

 

 

6A

INDIVIDUAL CARGO WEIGHT

 

ENTER THE INDIVIDUAL WEIGHT OF THE CARGO REPORTEDIN FIELD 2. ENTER THE CONTEXT QUANTITY, DECIMAL POINT PERMITTED, 8 CHARACTER MAXIMUM (1 THROUGH 999999.9) AND THE UNIT OF MEASUREMENT, FOR EXAMPLE: "109.5KG". See table 2185 which is a composite

7

CARGO WEIGHT

 

 

 

 

7A

CARGO WEIGHT

 

ENTER THE TOTAL WEIGHT OF THE CARGO REPORTED IN FIELD 2. ENTER THE CONTEXT QUANTITY, DECIMAL POINT PERMITTED, 8 CHARACTER MAXIMUM (1 THROUGH 999999.9) AND THE UNIT OF MEASUREMENT, FOR EXAMPLE: "109.5KG". THE TOTAL WEIGHT OF THE CARGO TO BE TRANSPORTED IS THE SUM OF THE ENTRIES IN FIELD 7. IF CLARIFICATION IS NEEDED, PROVIDE IN A SUBSEQUENT AMPN SET. See table 2185 which is a composite

8

AIR CARGO LINEAR DIMENSIONS

 

 

 

 

8A

AIR CARGO LINEAR DIMENSIONS

 

ENTER THE DIMENSION OF THE CARGO REPORTED IN FIELD 2. ENTER A CONTEXT QUANTITY, DECIMAL POINT PERMITTED (0 THROUGH 99999) OR (.0001 THROUGH 999.9),  A CHARACTER "X", A CONTEXT QUANTITY, DECIMAL POINT PERMITTED (0 THROUGH 99999) OR (.0001 THROUGH 999.9), A "X" CHARACTER, A CONTEXT QUANTITY, DECIMAL POINT PERMITTED (0 THROUGH 99999) OR (.0001 THROUGH 999.9), AND UNIT OF LINEAR MEASUREMENT, FOR EXAMPLE: "180X75.5X62.5IN". See table 2182 which is a composite
