Designator

Field Desc

Concept/Explanation/Examples

1

NATIONALITY

 

THE NATIONALITY OF THE ALLOCATED GBAD UNIT.

 

 

1A

NATIONALITY (2A)

 

ENTER THE 2 LETTER NATO COUNTRY CODE CORRESPONDING TO THE NATIONALITY OF THE ALLOCATED GBAD UNIT, FOR EXAMPLE: "IT". See table 1262/5 which contains a list of data items and associated data codes

 

 

1B

NATIONALITY (3A)

 

ENTER THE 3 LETTER COUNTRY CODE CORRESPONDING TO THE NATIONALITY OF THE ALLOCATED ASACS UNIT, IN ACCORDANCE WITH STANAG 1059, ED. 8, FOR EXAMPLE: "ITA". See table 1262/2 which contains a list of data items and associated data codes

 

 

1C

EXERCISE GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE CORRESPONDING TO THE FICTITOUS NATIONALITY OF THE GBAD UNIT ALLOCATED FOR THE EXERCISE, IN ACCORDANCE WITH STANAG 1059, ED. 8, FOR EXAMPLE: "XI". See table 1262/4 which contains a list of data items and associated data codes

2

ARMED SERVICE CODE

 

 

 

 

2A

ARMED SERVICE CODE

 

ENTER THE ALPHABETIC CHARACTER CODE CORRESPONDING TO THE ARMED SERVICE TO WHICH THE ALLOCATED ASACS UNITS BELONG, FOR EXAMPLE: "F". See table 1107/2 which contains a list of data items and associated data codes

3

UNIT IDENTIFIER

 

THE NAME OR UNIT IDENTIFICATION CODE (UIC) CORRESPONDING TO THE ALLOCATED UNIT.

 

 

3A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME CORRESPONDING TO THE ALLOCATED GBAD UNIT, FOR EXAMPLE: "UNIT:65 AEROGRUPPO". See table 1028/43 which is an instructive entry

 

 

3B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (USING THE 2 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED GBAD UNIT, FOR EXAMPLE: "UIC2:ITF065122". See table 2243 which is a composite

 

 

3C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (USING THE 3 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED GBAD UNIT, FOR EXAMPLE: "UIC3:ITAF065122". See table 2241 which is a composite

4

BASE IDENTIFIER

 

THE GEOGRAPHICAL LOCATION OF THE BASE WHERE THE ALLOCATED GBAD UNIT IS SITUATED.

 

 

4A

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR OF THE PLACE WHERE THE ALLOCATED GBAD UNIT IS SITUATED, FOR EXAMPLE: "ICAO:EBGL". See table 1125/1 which is an instructive entry

 

 

4B

PLACE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE BASE WHERE THE ALLOCATED GBAD UNIT IS SITUATED, FOR EXAMPLE: "NAME:GLONS". See table 1022/170 which is an instructive entry

 

 

4C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

5

LOCATION IDENTIFIER

 

 

 

 

5A

LOCATION OR POSITION NAME

 

ENTER AS A LITERAL THE NAME OF THE LOCATION WHERE THE ALLOCATED GBAD UNIT IS SITUATED, FOR EXAMPLE "GLONS". See table 1022/6 which is an instructive entry

6

GROUND BASED AIR DEFENCE SYSTEMS

 

THE TYPE AND MODEL NUMBER OF THE WEAPONS ASSIGNED/ATTACHED TO THE ALLOCATED GBAD UNIT.

 

 

6A

SURFACE-TO-AIR WEAPON SYSTEM

SAW

ENTER "SAW:" FOLLOWED BY THE TYPE OF SURFACE-TO-AIR WEAPON ASSIGNED/ATTACHED TO THE ALLOCATED GBAD UNIT, FOR EXAMPLE: "SAW:CROTAL". See table 1407/2 which contains a list of data items and associated data codes

 

 

6B

SURFACE-TO-AIR WEAPON SYSTEM TYPE, OTHER

OTHSAW

ENTER "OTHSAW:" FOLLOWED BY THE TYPE OF SURFACE-TO-AIR WEAPON, NOT CONTAINED IN THE WEAPON CODES LIST, ASSIGNED/ATTACHED TO THE ALLOCATED GBAD UNIT, FOR EXAMPLE:"OTHSAW: SA-XYZ11" See table 1407/24 which is an instructive entry

7

PRIMARY ROLE

 

 

 

 

7A

PRIMARY SURFACE-TO-AIR MISSION TYPE

 

ENTER THE ALPHABETIC CHARACTER CODE CORRESPONDING TO THE TYPE OF PRIMARY ROLE OF THE ALLOCATED UNIT, FOR EXAMPLE: "ABT". See table 1285/36 which contains a list of data items and associated data codes

8

SECONDARY ROLE

 

 

 

 

8A

SECONDARY SURFACE-TO-AIR MISSION TYPE

 

ENTER THE ALPHABETIC CHARACTER CODE CORRESPONDING TO THE TYPE OF SECONDARY ROLE OF THE ALLOCATED UNIT, FOR EXAMPLE: "EAD". See table 1285/37 which contains a list of data items and associated data codes

9

FIRE UNITS NUMBER

 

 

 

 

9A

NUMBER OF FIRE UNITS

 

ENTER THE NUMBER OF FIRE UNITS ASSIGNED/ATTACHED TO THE ALLOCATED UNIT, FOR EXAMPLE: "5". See table 1044/37 a range [0 through 99]

10

LAUNCHERS NUMBER

 

 

 

 

10A

NUMBER OF LAUNCHERS

 

ENTER THE NUMBER OF LAUNCHERS ASSIGNED/ATTACHED TO THE ALLOCATED UNIT, FOR EXAMPLE: "15". See table 1044/38 a range [0 through 999]

11

MISSILES NUMBER

 

 

 

 

11A

NUMBER OF MISSILES

 

ENTER THE NUMBER OF MISSILES ASSIGNED/ATTACHED TO THE ALLOCATED GBAD UNIT, FOR EXAMPLE: "105". See table 1044/323 a range [0 through 999]

12

START OF ALLOCATION PERIOD

 

 

 

 

12A

DTG OF START

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF THE START OF THE ALLOCATION TIME PERIOD, FOR EXAMPLE: "141325ZFEB2003". See table 2033 which is a composite

ENTER THE DAY-TIME, MONTH, AND YEAR OF THE STOP OF THE ALLOCATION TIME PERIOD, FOR EXAMPLE: "141925ZFEB2003". See table 2033 which is a composite

14A

REMARKS
