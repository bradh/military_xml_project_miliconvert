1

AIRCRAFT TYPE OR ROLE

 

PROVIDES THE TYPE OR THE ROLE OF AIRCRAFT INVOLVED IN THE INCIDENT.

 

 

1A

AIRCRAFT TYPE

TYPE

ENTER "TYPE:" FOLLOWED BY THE AIRCRAFT TYPE, FOR EXAMPLE: "TYPE:TORNADO". See table 1015/1 which is an instructive entry

 

 

1B

AIRCRAFT ROLE

ROLE

ENTER "ROLE:" FOLLOWED BY THE AIRCRAFT ROLE, FOR EXAMPLE: "ROLE:FGD". See table 1010/6 which contains a list of data items and associated data codes

2

DAY-TIME OF INCIDENT

 

 

 

 

2A

DAY-TIME OF INCIDENT

 

ENTER THE DAY-TIME OF THE INCIDENT, FOR EXAMPLE: "211414Z". See table 2000 which is a composite

3

INCIDENT LOCATION

 

PROVIDES THE LOCATION OF THE INCIDENT.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

GEOGRAPHIC POSITION, GEOREF, MINUTE

GEOREFM

ENTER "GEOREFM:" FOLLOWED BY THE LOCATION OF THE INCIDENT USING GEOREF MINUTE, FOR EXAMPLE: "GEOREFM:BBDF4500". See table 2017 which is a composite

 

 

3C

BEARING AND DISTANCE FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

3D

LOCATION OR POSITION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE POSITION NAME, FOR EXAMPLE: "LOC:SEMBACH". See table 1022/6 which is an instructive entry

 

 

3E

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE INCIDENT LOCATION BY USING GEOREF CENTIMINUTE, FOR EXAMPLE: "GEO:BBDF45350021". See table 2018 which is a composite

4

LOCATION QUALIFIER

 

 

 

 

4A

LOCATION QUALIFIER

 

ENTER THE CODE TO DENOTE ACTUAL OR ESTIMATED COORDINATES, FOR EXAMPLE: "AA". See table 1118/1 which contains a list of data items and associated data codes

5

ALTITUDE IN HUNDREDS OF FEET

 

 

 

 

5A

ALTITUDE, HUNDREDS OF FEET

 

ENTER THE ALTITUDE OF THE INCIDENT AIRCRAFT IN HUNDREDS OF FEET, FOR EXAMPLE: "200". See table 1255/14 a range [000 through 999]

6

AIRCRAFT CALL SIGN

 

 

 

 

6A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN OF THE LOST AIRCRAFT, FOR EXAMPLE: "JOLLY ONE". See table 1129/31 which is an instructive entry

7

COLOUR

 

 

 

 

7A

COLOUR

 

ENTER THE PREDOMINANT COLOR OF THE AIRCRAFT, FOR EXAMPLE: "GREY". See table 1138/1 which contains a list of data items and associated data codes

8

CAMOUFLAGE INDICATOR

 

 

 

 

8A

CAMOUFLAGE INDICATOR

 

ENTER THE INDICATOR FOR THE TYPE CAMOUFLAGE USED, FOR EXAMPLE: "DSG". See table 1402/1 which contains a list of data items and associated data codes

9

AIRCRAFT TAIL LETTERS AND SIDE NUMBER

 

 

 

 

9A

AIRCRAFT TAIL LETTERS AND SIDE NUMBER

 

ENTER THE TAIL LETTERS AND SIDE NUMBER OF THE LOST AIRCRAFT, FOR EXAMPLE: "FR8904". See table 1012/97 which is an instructive entry

10

CAUSE OF AIRCRAFT LOSS

 

 

 

 

10A

CAUSE OF AIRCRAFT LOSS

 

ENTER THE CAUSE OF THE AIRCRAFT INCIDENT OR LOSS, FOR EXAMPLE: "GUN-AA". See table 1374/1 which contains a list of data items and associated data codes

11

COUNT OF PERSONNEL ON BOARD

 

PROVIDES THE NUMBER OF PERSONNEL ON BOARD OF THE AIRCRAFT.

 

 

11A

COUNT OF PERSONNEL ON BOARD

 

ENTER THE TOTAL NUMBER OF PERSONNEL ON BOARD THE AIRCRAFT, FOR EXAMPLE: "2". See table 1044/132 a range [0 through 999]

12

DATA QUALIFIER

 

MUST BE USED IF COUNT OF PERSONNEL ON BOARD IS SPECIFIED WITHIN THIS SET.

 

 

12A

DATA QUALIFIER

 

ENTER THE DATA QUALIFIER CODE, FOR EXAMPLE: "ACT". See table 1130/2 which contains a list of data items and associated data codes

13

PERSONNEL COUNT

 

 

 

 

13A

PERSONNEL COUNT

 

ENTER THE NUMBER OF PERSONNEL (CREW AND/OR PASSENGERS) WITH THE SAME SAR STATUS, FOR EXAMPLE: "2". See table 1044/133 a range [0 through 999]

14

DOWNED AIRCRAFT CREW STATUS

 

MUST BE USED IF PERSONNEL COUNT IS SPECIFIED WITHIN THIS SET.

 

 

14A

DOWNED AIRCRAFT CREW STATUS

 

ENTER THE DISPOSITION OF PERSONNEL REPORTED IN FIELD 13, FOR EXAMPLE: "MIA". See table 1376/1 which contains a list of data items and associated data codes
