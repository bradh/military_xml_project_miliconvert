SECTOR NUMBER

 

 

 

 

1A

SECTOR NUMBER

 

ENTER THE SAFE SECTOR LETTER AND NUMBER (SPELLED PHONETICALLY) SEPARATED BY A BLANK, FOR EXAMPLE: "ALFA ONE". See table 2408 which is a composite

2

SECTOR ORIGIN

 

THE GEOGRAPHICAL LOCATION USED AS SECTOR ORIGIN.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

ORIGIN PLACE NAME

PLACE

ENTER "PLACE:" FOLLOWED BY THE PLACE NAME OF THE SECTOR ORIGIN, FOR EXAMPLE: "PLACE:MEWSTONE". See table 1022/109 which is an instructive entry

 

 

2H

ORIGIN, BEARING AND DISTANCE IN NM FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE SECTOR ORIGIN AS A BEARING AND DISTANCE FROM A REFERENCE POINT, FOR EXAMPLE: "BAR:270T-ZZ-2.5NM". See table 2307 which is a composite

 

 

2I

UNIT NAME

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OF THE UNIT ON WHICH THE SECTOR ORIGINATES, FOR EXAMPLE: "UNIT:DARING". See table 1022/48 which is an instructive entry

 

 

2J

ATP-1 POSITION DESIGNATOR

RP

ENTER "RP:" FOLLOWED BY THE ATP-1 STANDARD POSITION FROM WHICH THE SECTOR ORIGINATES, FOR EXAMPLE: "RP:ZZ". See table 1612/1 which contains a list of data items and associated data codes

3

OUTER RANGE

 

 

 

 

3A

OUTER RANGE

 

ENTER THE OUTER RANGE LIMIT OF THE SECTOR IN NAUTICAL MILES, FOR EXAMPLE: "50NM". See table 2158 which is a composite

4

BEARING, QUALIFIED

 

 

 

 

4A

BEARING, QUALIFIED

 

ENTER THE SECTOR CENTRE BEARING AND ANGULAR REFERENCE, FOR EXAMPLE: "130T". See table 2159 which is a composite

5

SECTOR WIDTH

 

 

 

 

5A

SECTOR WIDTH

 

ENTER THE SECTOR WIDTH IN DEGREES EITHER SIDE OF THE CENTRE BEARING IN FIELD 4, FOR EXAMPLE: "20". See table 1043/16 a range [0 through 180]

6

UPPER FLIGHT LEVEL OR ALTITUDE LIMIT

 

SPECIFIES THE UPPER FLIGHT LEVEL OR ALTITUDE LIMIT FOR THE SAFETY SECTOR.

 

 

6A

MAXIMUM FLIGHT LEVEL

 

ENTER "FL" PLUS THE MAXIMUM FLIGHT LEVEL, FOR EXAMPLE: "FL345". See table 2129 which is a composite

 

 

6B

MAXIMUM ALTITUDE IN HUNDREDS OF FEET

 

ENTER "ALT" PLUS THE MAXIMUM ALTITUDE IN HUNDREDS OF FEET, FOR EXAMPLE: "ALT445". See table 2164 which is a composite

7

START TIME

 

SPECIFIES THE DATE/TIME AT WHICH THE SECTOR IS TO BE ESTABLISHED.

 

 

7A

DTG

 

ENTER AS A DTG THE DATE AND TIME AT WHICH THE SAFETY SECTOR IS TO BE ESTABLISHED, FOR EXAMPLE: "100700ZOCT2003". See table 2033 which is a composite

 

 

7B

DAY-TIME START

 

ENTER THE DAY-TIME AT WHICH THE SAFETY SECTOR IS TO BE ESTABLISHED, FOR EXAMPLE: "100700Z". See table 2000 which is a composite

 

 

7C

DAY-TIME AND MONTH OF START

 

ENTER THE DAY-TIME AND MONTH AT WHICH THE SAFETY SECTOR IS TO BE ESTABLISHED, FOR EXAMPLE: "100700ZOCT". See table 2030 which is a composite

8

STOP TIME

 

SPECIFIES THE DATE/TIME AT WHICH THE SAFETY SECTOR IS TO BE DISESTABLISHED.

 

 

8A

DTG

 

ENTER AS A DTG THE DATE AND TIME AT WHICH THE SAFETY SECTOR IS TO BE DISESTABLISHED, FOR EXAMPLE: "101700ZOCT2003". See table 2033 which is a composite

 

 

8B

DAY-TIME STOP

 

ENTER THE DAY-TIME AT WHICH THE SAFETY SECTOR IS TO BE DISESTABLISHED, FOR EXAMPLE: "101700Z". See table 2000 which is a composite

 

 

8C

DAY-TIME AND MONTH OF STOP

 

ENTER THE DAY-TIME AND MONTH AT WHICH THE SAFETY SECTOR IS TO BE DISESTABLISHED, FOR EXAMPLE: "101700ZOCT". See table 2030 which is a composite
