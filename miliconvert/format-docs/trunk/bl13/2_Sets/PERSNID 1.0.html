1

BIOGRAPHIC KEY

 

 

 

 

1A

BIOGRAPHIC KEY

 

ENTER THE CONCATENATED KEY (BIOGRAPHIC INDICATOR, RECORD ORIGINATOR AND ACCESSION NUMBER) TO IDENTIFY A CERTAIN BIOGRAPHIC RECORD, FOR EXAMPLE: "B-DA99-12345". See table 2445 which is a composite

2

LAST NAME

 

 

 

 

2A

LAST NAME

 

ENTER THE SURNAME OR GIVEN NAME OF THE INDIVIDUAL THAT THE BIOGRAPHIC RECORD RELATES TO, FOR EXAMPLE: "POMMEROY". See table 1022/255 which is an instructive entry

3

FIRST NAME(S)

 

 

 

 

3A

FIRST NAME(S)

 

ENTER THE INDIVIDUALS FIRST NAME, FOR EXAMPLE: "PAUL." See table 1022/256 which is an instructive entry

4

MIDDLE NAME(S)/INITIAL(S)

 

 

 

 

4A

MIDDLE NAME(S) INITIAL(S)

 

IF USED, ENTER THE INDIVIDUAL'S MIDDLE NAME(S), FOR EXAMPLE: "HENRY, JAMES". See table 1022/257 which is an instructive entry

5

BIRTH NAME

 

 

 

 

5A

BIRTH NAME

 

IF USED, ENTER THE GIVEN NAME THAT WAS CONFERRED UPON THE INDIVIDUAL AT BIRTH, FOR EXAMPLE: "MILLER". See table 1022/258 which is an instructive entry

6

PERSONAL TITLE

 

 

 

 

6A

PERSONAL TITLE

 

IF USED, ENTER THE INDIVIDUAL'S PERSONAL TITLE, FOR EXAMPLE: "COLONEL". See table 1022/259 which is an instructive entry

7

GENDER, ABBREVIATED

 

 

 

 

7A

GENDER, ABBREVIATED

 

ENTER "M" OR "F" TO IDENTIFY IF THE INDIVIDUAL IS MALE OR FEMALE. See table 1286/13 which contains a list of data items and associated data codes

8

HEIGHT OF PERSON, FEET OR METRES

 

 

 

 

8A

HEIGHT OF PERSON, FEET OR METRES

 

IF USED, ENTER THE PERSON'S HEIGHT IN FOOT OR METRE, FOR EXAMPLE: "1.86M". See table 2596 which is a composite

9

COLOUR OF EYES

 

 

 

 

9A

COLOUR OF EYES

 

IF USED, ENTER THE COLOUR OF EYES, FOR EXAMPLE: BLUE". See table 1138/11 which contains a list of data items and associated data codes

10

DATE OF BIRTH, YYYYMMDD

 

 

 

 

10A

DATE OF BIRTH, YYYYMMDD

 

ENTER THE DATE OF BIRTH IN THE FORM YYYYMMDD, FOR EXAMPLE: "19201226". See table 2053 which is a composite

11

PLACE OF BIRTH

 

 

 

 

11A

PLACE OF BIRTH

 

IF USED, MAKE A LITERAL ENTRY TO IDENTIFY THE PLACE WHERE THE INDIVIDUAL WAS BORN, FOR EXAMPLE: "BUTTERMERE, STUKELEY MEADOWS". See table 1022/309 which is an instructive entry

12

COUNTRY OF BIRTH

 

 

 

 

12A

COUNTRY OF BIRTH

 

IF USED, ENTER A CODE TO IDENTIFY THE COUNTRY IN WHICH THE INDIVIDUAL WAS BORN, FOR EXAMPLE: "UK". See table 1026/13 which contains a list of data items and associated data codes

13

DATE OF DEATH, YYYYMMDD

 

 

 

 

13A

DATE OF DEATH, YYYYMMDD

 

IF USED, ENTER THE DATE (YYYYMMDD) WHEN THE INDIVIDUAL DIED, FOR EXAMPLE: "20000214". See table 2053 which is a composite

14

PLACE OF DEATH

 

 

 

 

14A

PLACE OF DEATH

 

IF USED, MAKE A LITERAL ENTRY TO IDENTIFY THE PLACE WHERE THE INDIVIDUAL DIED, FOR EXAMPLE: "BELET HUEN". See table 1022/310 which is an instructive entry

15

COUNTRY

 

 

 

 

15A

COUNTRY

 

IF USED, ENTER A CODE IDENTIFYING THE COUNTRY OF DEATH, FOR EXAMPLE: "SO". See table 1026/18 which contains a list of data items and associated data codes

16

ALLEGIANCE

 

 

 

 

16A

ALLEGIANCE

 

ENTER A CODE IDENTIFYING THE COUNTRY OR POLITICAL ENTITY TO WHICH THE INDIVIDUAL OWES ITS ALLEGIANCE, FOR EXAMPLE: "UK". See table 1026/17 which contains a list of data items and associated data codes

17

RELIGION

 

 

 

 

17A

RELIGION

 

IF USED, ENTER THE RELIGION OF THE INDIVIDUAL, FOR EXAMPLE: "BAPTIST". See table 1016/1 which contains a list of data items and associated data codes

18

ETHNIC GROUP

 

 

 

 

18A

ETHNIC GROUP

 

IF USED, ENTER THE ETHNIC GROUP THE INDIVIDUAL BELONGS TO, FOR EXAMPLE: "ETHNIC ALBANIEN". See table 1646/1 which is an instructive entry

19

MARITAL STATUS

 

 

 

 

19A

MARITAL STATUS

 

ENTER THE MARITAL STAE OF THE INDIVIDUAL, FOR EXAMPLE: "D". See table 1727/1 which contains a list of data items and associated data codes

20

NUMBER OF CHILDREN

 

 

 

 

20A

NUMBER OF CHILDREN

 

IF USED, ENTER THE NUMBER OF CHILDREN THE INDIVIDUAL HAS, FOR EXAMPLE: "2". See table 1044/337 a range [0 through 99]

21

IMAGERY KEY

 

 

 

 

21A

IMAGERY KEY

 

IF USED, ENTER THE KEY OF ANY IMAGERY THAT MAY BE RELATED TO THE INDIVIDUAL, FOR EXAMPLE: "I-UK22-12345". See table 2585 which is a composite

22

DATE-TIME (Z) IMAGERY COLLECTED, YYYYMMDDHHMMSS

 

 

 

 

22A

DATE-TIME (Z) IMAGERY COLLECTED, YYYYMMDDHHMMSS

 

IF USED, ENTER THE DATE/TIME (YYYYMMDDHHMMSS) WHEN THE IMAGERY WAS COLLECTED, FOR EXAMPLE: "20001213164734". REPEATING AS NECESSARY. See table 2417 which is a composite

REMARKS
