Field Desc

Concept/Explanation/Examples

1

FLAG

 

 

 

 

1A

FLAG

 

ENTER THE COUNTRY CODE FOR THE FLAG UNDER WHICH THE SPECIFIED VESSEL SAILS, FOR EXAMPLE: "FR". See table 1026/3 which contains a list of data items and associated data codes

2

NAVAL VESSEL TYPE

 

 

 

 

2A

NAVAL VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE NAVAL VESSEL TYPE, FOR EXAMPLE: "FFGH". See table 1166/3 which contains a list of data items and associated data codes

3

UNIT IDENTIFICATION

 

SPECIFIES THE IDENTITY OF THE UNIT.

 

 

3A

SHIP NAME

 

ENTER THE SHIP NAME, FOR EXAMPLE: "CHARLES DE GAULLE". See table 1022/49 which is an instructive entry

 

 

3B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER FIELD DESCRIPTOR "UIC:" FOLLOWED BY THE COUNTRY CODE, THE ARMED SERVICE INDICATOR CODE AND THE ACE FILE SEQUENTIAL LOCATION NUMBER OF THE UNIT, FOR EXAMPLE: "UIC:USA123456". See table 2141 which is a composite

4

UNIT LOCATION

 

SPECIFIES THE GEOGRAPHIC LOCATION OF THE UNIT.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

PLACE NAME

NAME

ENTER FIELD DESCRIPTOR "NAME:"  FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "NAME:POINT X-RAY". See table 1022/170 which is an instructive entry

 

 

4H

TRUE BEARING AND DISTANCE IN NM FROM REFERENCE POINT

TBRN

ENTER THE POSITION AS A TRUE BEARING AND DISTANCE IN NM FROM A REFERENCE POINT, FOR EXAMPLE: "TBRN:134-DESSAU-55". See table 2016 which is a composite

5

DAY-TIME AT LOCATION

 

PROVIDES THE DAY-TIME AT WHICH THE UNIT WAS IN THE SPECIFIED LOCATION.

 

 

5A

DAY-TIME OF POSITION

 

ENTER THE DAY-TIME OF POSITION, FOR EXAMPLE: "311002Z". See table 2000 which is a composite

 

 

5B

DAY-TIME AND MONTH

 

ENTER THE DAY-TIME AND MONTH, FOR EXAMPLE: "201050ZDEC". See table 2030 which is a composite

 

 

5C

DTG

 

ENTER THE DTG, FOR EXAMPLE "142000ZJUN2002". See table 2033 which is a composite

 

 

5D

DAY-MONTH NAME

 

IF THE TIME IS UNKNOWN, ENTER THE DAY-MONTH NAME, FOR EXAMPLE: "16SEP". See table 2067 which is a composite

6

VESSEL MOVEMENT

 

PROVIDES DETAILS OF THE MOVEMENT OF THE SPECIFIED VESSEL.

 

 

6A

COURSE IN DEGREES TRUE AND SPEED IN KNOTS

 

ENTER THE COURSE IN DEGREES TRUE, A HYPHEN AND THE SPEED IN KNOTS OF THE VESSEL, FOR EXAMPLE: "265-13". See table 2025 which is a composite

 

 

6B

NAVAL FORCE MOVEMENT STATUS

 

ENTER THE NAVAL FORCE MOVEMENT STATUS APPLICABLE TO THE VESSEL, FOR EXAMPLE: "MAN". See table 1341/2 which contains a list of data items and associated data codes
