1A

AIRSPACE CONTROL MEANS STATUS UPDATE

 

ENTER THE STATUS OF THE DESIGNATED AIRSPACE CONTROL MEANS, FOR EXAMPLE: "DELETE".   FOR A NEW AIRSPACE CONTROL MEANS ENTER "ADD" See table 1156/5 which contains a list of data items and associated data codes

2

AIRSPACE CONTROL MEANS IDENTIFIER

 

ENTER THE AIRSPACE CONTROL MEANS IDENTIFIER OR NAME.  CAN ONLY BE USED IN CONJUNCTION WITH AIRSPACE CONTROL MEANS STATUS UPDATE "DELETE".  USE ONE OF THE FOLLOWING:

 

 

2A

AIRSPACE CONTROL MEANS IDENTIFYING DESIGNATOR

DESIGN

ENTER: "DESIGN:"  FOLLOWED BY THE AIRSPACE CONTROL MEANS IDENTIFYING DESIGNATOR, FOR EXAMPLE: "DESIGN:C12". See table 1515/12 which is an instructive entry

 

 

2B

AIRSPACE CONTROL MEANS NAME

NAME

ENTER: "NAME:" FOLLOWED BY THE AIRSPACE CONTROL MEANS NAME, FOR EXAMPLE: "NAME:TEXACO". See table 1022/27 which is an instructive entry

 

 

2C

AIRSPACE CONTROL MEANS DESIGNATOR RANGE

RANG

ENTER: "RANG:" FOLLOWED BY THE AIRSPACE CONTROL MEANS DESIGNATOR RANGE, FOR EXAMPLE: "RANG:C21-C25". See table 2165 which is a composite
