Concept/Explanation/Examples

1

PASSAGE DESIGNATOR

 

 

 

 

1A

PASSAGE DESIGNATOR

 

ENTER PASSAGE DESIGNATOR, FOR EXAMPLE: "LEAD". See table 1142/1 which contains a list of data items and associated data codes

2

START TIME OF TRANSIT

 

IF USED, ENTER START TIME OF TRANSIT IN ONE OF THE FOLLOWING WAYS:

 

 

2A

DAY-TIME OF TRANSIT START

 

ENTER DAY-TIME COMPRISING DAY [01-31], HOUR [00-23], MINUTE [00-59] AND TIME ZONE, FOR EXAMPLE: "110845Z". See table 2000 which is a composite

 

 

2B

DAY-TIME AND MONTH OF TRANSIT START

 

ENTER DAY-TIME AND MONTH.  ENTER DAY [01-31], HOUR [00-23], MINUTE [00-59], TIME ZONE, AND MONTH NAME, FOR EXAMPLE: "110845ZMAR". See table 2030 which is a composite

 

 

2C

DTG OF TRANSIT START

 

ENTER DAY-TIME GROUP CONSISTING OF DAY [01-31], HOUR [00-23], MINUTE [00-59], TIME ZONE, MONTH NAME, AND YEAR [00-99], FOR EXAMPLE: "110845ZMAR97". See table 2033 which is a composite

3

RECOMMENDED TRANSIT SPEED, KNOTS

 

 

 

 

3A

RECOMMENDED TRANSIT SPEED, KNOTS

 

IF USED, ENTER RECOMMENDED TRANSIT SPEED [0-9999, 0 TO 3 DECIMAL PLACES]  IN KNOTS, FOR EXAMPLE: "22.5KTS". See table 2265 which is a composite

4

SHIP TRANSIT LONGITUDINAL INTERVAL, METRES OR YARDS

 

 

 

 

4A

SHIP TRANSIT LONGITUDINAL INTERVAL, METRES OR YARDS

 

ENTER LONGITUDINAL SEPARATION BETWEEN SHIPS GIVEN AS 1-4 FIGURES [1-9999], PLUS "M" (METRES) OR "YD" (YARDS), FOR EXAMPLE: "155M". See table 2208 which is a composite

5

GROUP TRANSIT LONGITUDINAL INTERVAL, NAUTICAL MILES

 

 

 

 

5A

GROUP TRANSIT LONGITUDINAL INTERVAL, NAUTICAL MILES

 

ENTER LONGITUDINAL SEPARATION BETWEEN GROUPS GIVEN AS 1-3 FIGURES [1-999], FOR EXAMPLE: "13". See table 1089/15 a range [0 through 999]

6

NAME OF LEAD SHIP IN GROUP

 

 

 

 

6A

NAME OF LEAD SHIP IN GROUP

 

IF USED, ENTER NAME [LITERAL] OF LEAD SHIP IN GROUP, FOR EXAMPLE: "VEER". See table 1022/54 which is an instructive entry

7

NUMBER OF SHIPS IN GROUP

 

 

 

 

7A

NUMBER OF SHIPS IN GROUP

 

IF USED, ENTER NUMBER OF SHIPS [1-99], FOR EXAMPLE: "22". See table 1044/75 a range [0 through 99]
