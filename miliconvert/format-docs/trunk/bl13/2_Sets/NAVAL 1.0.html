Concept/Explanation/Examples

1

CONTACT SERIAL NUMBER

 

 

 

 

1A

CONTACT SERIAL NUMBER

 

ENTER THE SEQUENTIAL SERIAL NUMBER OF THE CONTACT BY TYPE WITHIN A SINGLE MESSAGE, FOR EXAMPLE: "3". See table 1012/67 a range [1 through 999]

2

REPORTING UNIT NAME

 

 

 

 

2A

UNIT

 

ENTER THE NAME OF THE REPORTING UNIT, FOR EXAMPLE: "LOWESTOFT ". See table 1022/57 which is an instructive entry

3

LOCATING REPORT NUMBER

 

SPECIFIES THE LOCATING REPORT NUMBER.

 

 

3A

LOCATING REPORT NUMBER

 

ENTER THE LOCATING REPORT NUMBER, FOR EXAMPLE: "LR3". See table 2296 which is a composite

 

 

3B

NOT REPORTED IN FLIGHT

 

ENTER THE CODE "NRIF" SPECIFYING NOT REPORTED IN FLIGHT. See table 1449/1 which contains a list of data items and associated data codes

4

CONTACT IDENTIFICATION

 

SPECIFIES THE IDENTIFICATION OF THE NAVAL CONTACT.

 

 

4A

NAME OF SHIP CLASS

CLASS

ENTER FIELD DESCRIPTOR "CLASS:" FOLLOWED BY THE NAME OF THE SHIP CLASS, FOR EXAMPLE: "CLASS:MOSKVA". See table 1022/35 which is an instructive entry

 

 

4B

UNIT

 

ENTER THE NAME OF THE NAVAL SURFACE CONTACT, FOR EXAMPLE: "CHARLES DE GAULLE". See table 1022/57 which is an instructive entry

5

NAVAL VESSEL TYPE

 

 

 

 

5A

NAVAL VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE NAVAL VESSEL TYPE BEING REPORTED, FOR EXAMPLE: "AE". See table 1166/3 which contains a list of data items and associated data codes

6

FLAG

 

 

 

 

6A

FLAG

 

ENTER THE COUNTRY CODE FOR THE FLAG UNDER WHICH THE VESSEL SAILS, FOR EXAMPLE: "FR". See table 1026/3 which contains a list of data items and associated data codes

7

CONTACT PHOTO STATUS

 

 

 

 

7A

CONTACT PHOTO STATUS

 

ENTER THE INDICATION OF WHETHER OR NOT PHOTOS WERE TAKEN OF THE CONTACT, FOR EXAMPLE: "PHT". See table 1450/1 which contains a list of data items and associated data codes

8

PENNANT NUMBER

 

 

 

 

8A

PENNANT NUMBER

 

ENTER THE PENNANT NUMBER OF THE NAVAL SURFACE CONTACT, FOR EXAMPLE: "FR962". See table 1012/46 which is an instructive entry

9

SHIP CONTROL NUMBER

 

 

 

 

9A

SHIP CONTROL NUMBER

 

ENTER THE SHIP CONTROL NUMBER ASSIGNED TO THE NAVAL SURFACE CONTACT, FOR EXAMPLE: "A12345". See table 2282 which is a composite

10

NATO DESIGNATOR OR DISCRETE IDENTIFIER

 

SPECIFIES THE NATO DESIGNATION OR IDENTIFIER ASSIGNED TO THE CONTACT.

 

 

10A

NATO SURFACE CONTACT DESIGNATOR

DESIG

ENTER FIELD DESCRIPTOR "DESIG:" FOLLOWED BY THE NATO SURFACE CONTACT DESIGNATOR, FOR EXAMPLE: "DESIG:32AA". See table 2154 which is a composite

 

 

10B

DISCRETE IDENTIFIER

 

ENTER THE DISCRETE IDENTIFIER (A MARITIME REPORTING CODE FOR TRACKS THAT ARE NOT IDENTIFIED AS "FRIEND", "ASSUMED FRIEND" OR "UNKNOWN") FOR THE CONTACT, FOR EXAMPLE: "1234". See table 1625/25 which is an alphanumeric range

11

VESSEL SUSPICION CODE

 

 

 

 

11A

VESSEL SUSPICION CODE

 

ENTER THE NUMERIC CODE INDICATING THE LEVEL OF SUSPICION CONCERNING THE CONTACT, FOR EXAMPLE: "01". See table 1698/2 which contains a list of data items and associated data codes
