Concept/Explanation/Examples

1

RESIDUAL MISSION INDICATOR

 

INDICATES WHETHER THE MISSION IS OR WILL BE A RESIDUAL MISSION.  MISSIONS THAT SPAN MORE THAN ONE TASKING PERIOD WILL REAPPEAR IN EACH APPLICABLE MESSAGE FOR COORDINATION AND FLIGHT SAFETY PURPOSES.  THE FIRST TIME THIS TYPE OF MISSION IS PUBLISHED, IT IS MARKED AS "FORWARDED."  IN SUBSEQUENT APPLICABLE MESSAGES, THE MISSION LINES ARE COPIED VERBATIM AND MARKED AS "RESIDUAL."  MISSIONS THAT DO NOT SPAN MULTIPLE TASKING PERIODS ARE MARKED "NON-RESIDUAL."

 

 

1A

RESIDUAL MISSION INDICATOR

 

ENTER THE INDICATION CODE CONCERNING THE RESIDUALITY OF THE MISSION, FOR EXAMPLE: "N" FOR A NON-RESIDUAL MISSION. See table 1084/8 which contains a list of data items and associated data codes

2

MISSION NUMBER

 

THE IDENTIFICATION NUMBER OF A MISSSION.

 

 

2A

MISSION NUMBER

 

ENTER THE NUMBER OR ALPHANUMERIC CODE IDENTIFYING A MISSION, FOR EXAMPLE "TYU876". See table 1012/69 which is an instructive entry

3

PRIMARY AERIAL MISSION TYPE

 

PROVIDES INFORMATION ABOUT THE TYPE OF PRIMARY AERIAL MISSION TO BE PERFORMED BY THE TASKED UNIT.

 

 

3A

PRIMARY AERIAL MISSION TYPE

 

ENTER THE PRIMARY TYPE OF MISSION, FOR EXAMPLE "GCAS". See table 1285/31 which contains a list of data items and associated data codes

4

SECONDARY AERIAL MISSION TYPE

 

PROVIDES INFORMATION ABOUT THE TYPE OF THE SECONDARY AERIAL MISSION TO BE PERFORMED BY THE TASKED UNIT.

 

 

4A

SECONDARY AERIAL MISSION TYPE

 

ENTER THE SECONDARY MISSON TYPE, FOR EXAMPLE "GFAC". See table 1285/32 which contains a list of data items and associated data codes

5

MISSILE ALERT STATUS

 

PROVIDES INFORMATION ABOUT THE STATUS OF READINESS OF THE SURFACE/SUBSURFACE LAUNCHED MISSILE UNIT.  CAN ONLY BE USED IN CONJUNCTION WITH GROUND ALERT MISSION TYPES WITHIN THIS SET.
