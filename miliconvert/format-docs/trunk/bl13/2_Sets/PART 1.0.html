MESSAGE PART IDENTIFIER

MESSAGE PART IDENTIFIER

 

ENTER THE MESSAGE PART IDENTIFIER, FOR EXAMPLE: "A9". See table 1102/2 which is an alphanumeric range

MESSAGE PART DENOMINATION

 

 

 

 

2A

MESSAGE PART DENOMINATION

 

IF USED, ENTER A LITERAL WORD OR PHRASE THAT IS A SUBHEADING OR MORE SPECIFIC DESCRIPTION OF THE PART IDENTIFIED IN FIELD 1, FOR EXAMPLE: "NATO AVAIL". See table 1081/1 which is an instructive entry

3

SECURITY CLASSIFICATION, ABBREVIATED

 

 

 

 

3A

SECURITY CLASSIFICATION, ABBREVIATED

 

IF USED, ENTER THE SECURITY CLASSIFICATION OF THE INFORMATION CONTAINED IN THIS MESSAGE PART, FOR EXAMPLE: "NS".  THE SECURITY CLASSIFICATION APPLIES TO INFORMATION CONTAINED BETWEEN CONSECUTIVE PART4A SETS.  IF ONLY A SINGLE PART4A IS USED, THE CLASSIFICATION APPLIES TO ALL SETS FOLLOWING IT. See table 1288/1 which contains a list of data items and associated data codes

4

PART TITLE

 

 

 

 

4A

PART TITLE

 

IF USED, ENTER A LITERAL TITLE THAT IS A SUBHEADING OF FIELD 2, OR WHICH REFLECTS THE SUBJECT MATTER OF SET FORMATS THAT FOLLOW, FOR EXAMPLE: "AIRCRAFT AVAILABILITY". See table 1356/2 which is an instructive entry

5

SPECIAL HANDLING DESIGNATOR

 

 

 

 

5A

SPECIAL HANDLING DESIGNATOR

 

IF USED, ENTER THE SPECIAL HANDLING DESIGNATOR: "CRYPTO SECURITY". See table 1101/1 which contains a list of data items and associated data codes
