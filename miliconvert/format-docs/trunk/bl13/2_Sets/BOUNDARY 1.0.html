LOWER LIMIT

 

ENTER THE LOWER BOUNDARY LIMIT IN ONE OF THE FOLLOWING WAYS:

 

 

1A

BOUNDARY LOWER LIMIT, SPEED IN KNOTS

SPD

ENTER "SPD:" FOLLOWED BY THE LOWER SPEED LIMIT, FOR EXAMPLE: "SPD:05KTS". See table 2370 which is a composite

 

 

1B

BOUNDARY LOWER LIMIT, FLIGHT LEVEL

 

ENTER "FL" FOLLOWED BY THE LOWER FLIGHT LEVEL LIMIT IN HUNDREDS OF FEET, FOR EXAMPLE: "FL180". See table 2129 which is a composite

 

 

1C

BOUNDARY LOWER LIMIT, ALTITUDE IN HUNDREDS OF FEET

 

ENTER "ALT" FOLLOWED BY THE LOWER ALTITUDE LEVEL LIMIT IN HUNDREDS OF FEET, FOR EXAMPLE: "ALT125". See table 2164 which is a composite

 

 

1D

LOWER LIMIT, HEADING

HDG

ENTER "HDG:" FOLLOWED BY THE LOWER HEADING LIMIT IN DEGREES (000-359) FOLLOWED BY THE ANGULAR MEASUREMENT INDICATOR, FOR EXAMPLE: "HDG:180T". See table 2368 which is a composite

 

 

1E

ABOVE MEAN SEA LEVEL DATA QUALIFIER

 

ENTER THE ABOVE MEAN SEA LEVEL QUALIFIER "AMSL". See table 1130/41 which contains a list of data items and associated data codes
