Designator

Field Desc

Concept/Explanation/Examples

1

FREQUENCY

 

PROVIDES THE FREQUENCY FOR THE IMAGERY DATA LINK.

 

 

1A

FREQUENCY IN MEGAHERTZ

FREQ

ENTER "FREQ:" FOLLOWED BY THE SPECIFIED DATA LINK FREQUENCY USED TO TRANSMIT IMAGERY PRODUCTS, FOR EXAMPLE: "FREQ:244.0". See table 1068/1 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

1B

FREQUENCY DESIGNATOR

DESIG

ENTER "DESIG:" FOLLOWED BY THE FREQUENCY DESIGNATOR FOR TRANSMITTING THE IMAGERY, FOR EXAMPLE: "DESIG:PINK". See table 1396/1 which is an instructive entry

2

IMAGERY RECEIVER CALL SIGN

 

 

 

 

2A

IMAGERY RECEIVER CALL SIGN

 

ENTER THE CALL SIGN OF THE IMAGERY RECEIVER, FOR EXAMPLE: "JOHANN23". See table 1129/32 which is an instructive entry

3

REPORT-IN POINT

 

PROVIDES THE REPORT-IN POINT LOCATION FOR THE IMAGERY DOWN LINK.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

3C

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME OF THE LOCATION, FOR EXAMPLE: "LOC:FORT BRAGG". See table 1022/4 which is an instructive entry

4

LINK 16 CALL SIGN

 

 

 

 

4A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE LINK 16 ABBREVIATED CALL SIGN IF THE IMAGERY RECEIVER IS LINK 16 EQUIPPED, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry

5

DAY-TIME OF LINK START

 

 

 

 

5A

DAY-TIME OF LINK START

 

ENTER THE SCHEDULED OR ACTUAL DAY-TIME TO BEGIN A DATA LINK ACTIVITY, FOR EXAMPLE: "141325Z". See table 2000 which is a composite

6

DAY-TIME OF LINK STOP

 

CAN ONLY BE USED IN CONJUNCTION WITH A SPECIFIED DAY-TIME OF LINK START.

 

 

6A

DAY-TIME OF LINK STOP

 

ENTER THE SCHEDULED OR ACTUAL DAY-TIME TO END A DATA LINK ACTIVITY, FOR EXAMPLE: "141330Z". See table 2000 which is a composite

7

IMAGERY RECEIVER LOCATION

 

PROVIDES THE IMAGERY RECEIVER LOCATION.

 

 

7A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

7B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

7C

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME OF THE RECEIVER LOCATION, FOR EXAMPLE: "LOC:FORT BENNINGS". See table 1022/4 which is an instructive entry
