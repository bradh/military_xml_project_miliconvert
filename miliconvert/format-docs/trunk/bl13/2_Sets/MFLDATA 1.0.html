1

MINEFIELD DESIGNATOR

 

 

 

 

1A

MINEFIELD DESIGNATOR

 

ENTER THE DESIGNATOR OBTAINED FROM THE MINE FIELD PLANNING FOLDER OR FROM THE OPERATIONAL PLANNING AUTHORITY, FOR EXAMPLE "123455". See table 1012/90 which is a regular expression

2

DELIVERY VEHICLE

 

SPECIFIES THE TYPE OF VEHICLE USED TO LAY THE MINE FIELD.

 

 

2A

NAVAL VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE NAVAL VESSEL TYPE, FOR EXAMPLE: "SSN". See table 1166/3 which contains a list of data items and associated data codes

 

 

2B

AIRCRAFT TYPE MODEL

ACTYPE

ENTER FIELD DESCRIPTOR "ACTYPE:" FOLLOWED BY THE AIRCRAFT TYPE/MODEL, FOR EXAMPLE: "ACTYPE:P3C". See table 1015/4 which contains a list of data items and associated data codes

3

NAVIGATION METHOD

 

 

 

 

3A

NAVIGATION METHOD

 

ENTER THE METHOD OF NAVIGATION USED IN FIXING POSITION OR DELIVERY ACCURACY, FOR EXAMPLE: "VISUAL". See table 1486/1 which contains a list of data items and associated data codes

4

TIME COMPLETED

 

SPECIFIES THE DATE AND/OR TIME WHEN THE MINE FIELD LAYING WAS COMPLETED.

 

 

4A

DAY-TIME OF COMPLETION

 

ENTER THE DAY, TIME AND TIME ZONE, FOR EXAMPLE: "031132Z". See table 2000 which is a composite

 

 

4B

DAY-TIME AND MONTH OF COMPLETION

 

ENTER THE DAY-TIME MONTH, FOR EXAMPLE: "071005ZJAN". See table 2030 which is a composite

 

 

4C

MONTH DAY-TIME OF COMPLETION

 

ENTER THE MONTH-DAY-TIME, FOR EXAMPLE: "JAN121200Z". See table 2251 which is a composite

 

 

4D

DTG
