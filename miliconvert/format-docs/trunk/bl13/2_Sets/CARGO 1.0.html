CARGO CLASSIFICATION

 

 

 

 

1A

CARGO CLASSIFICATION

 

ENTER THE CLASSIFICATION OF THE CARGO, FOR EXAMPLE: "C". See table 1012/199 which contains a list of data items and associated data codes

2

PAX OR CARGO TYPE

 

THE CATEGORY OF PASSENGERS OR THE TYPE OF CARGO TO BE TRANSPORTED.

 

 

2A

PASSENGER CATEGORY

PA

ENTER "PA:" FOLLOWED BY THE PASSENGER CATEGORY, FOR EXAMPLE "PA:CBT" See table 1173/3 which contains a list of data items and associated data codes

 

 

2B

CARGO TYPE, GENERIC

CT

ENTER "CT:" FOLLOWED BY THE TYPE OF CARGO, FOR EXAMPLE "CT:C-FOOD". See table 1173/15 which is an instructive entry

3

QUANTITY OF LOAD TYPE

 

 

 

 

3A

CONTEXT QUANTITY, K PERMITTED, 4 CHARACTERS MAX

 

ENTER THE QUANTITY OF LOADED FREIGHT OR PERSONS, FOR EXAMPLE "69K". See table 1390/3 which is an alphanumeric range

4

CARGO WEIGHT

 

THE WEIGHT OF THE CARGO, IN EITHER TENTHS OF SHORT TONS OR TENTHS OF TONS.

 

 

4A

BULK CARGO WEIGHT IN SHORT TONS IN TENTHS

STON

ENTER "STON:", FOLLOWED BY THE BULK CARGO WEIGHT IN SHORT TONS IN TENTHS, FOR EXAMPLE "STON:69.5". See table 1800/2 a range [0.1 through 999.9] (1 to 1 decimal places)

 

 

4B

BULK CARGO WEIGHT IN TONS IN TENTHS

TON

ENTER "TON:", FOLLOWED BY BULK CARGO WEIGHT IN TONS IN TENTHS, FOR EXAMPLE:"TON:69.5" See table 1440/5 a range [0.1 through 999.9] (1 to 1 decimal places)

5

CARGO SIZE

 

 

 

 

5A

AIR CARGO SIZE

SIZE

ENTER "SIZE:" FOLLOWED BY A DESCRIPTION OF THE CUBIC SIZE OF THE AIR CARGO, FOR EXAMPLE: "SIZE:1KCUM". See table 2506 which is a composite

6

CARGO LENGTH

 

 

 

 

6A

CARGO LINEAR DIMENSION, LENGTH

CARGO WIDTH

 

 

 

 

7A

CARGO LINEAR DIMENSION, WIDTH

 

ENTER THE WIDTH OF THE AIR CARGO, FOR EXAMPLE: "3.45M". See table 2507 which is a composite

8

CARGO HEIGHT

 

 

 

 

8A

CARGO LINEAR DIMENSION, HEIGHT

 

ENTER THE HEIGHT OF THE AIR CARGO, FOR EXAMPLE: "3.05M". See table 2507 which is a composite

9

HOIST REQUIREMENT INDICATOR

 

 

 

 

9A

HOIST REQUIREMENT INDICATOR

 

ENTER IF THE FREIGHT NEEDS TO BE RAISED UP TO THE AIRCRAFT CARGO HOLD, YES OR NO, FOR EXAMPLE:"YES". See table 1027/52 which contains a list of data items and associated data codes

10

SUPPORT EQUIPMENT REQUIRED

 

 

 

 

10A

TYPE OF SUPPORT EQUIPMENT

 

ENTER THE TYPE OF SUPPORT EQUIPMENT REQUIRED FOR LOADING AND UNLOADING THE CARGO, FOR EXAMPLE: "FORK LIFT". See table 1135/9 which is an instructive entry

11

HAZARDOUS CARGO DESIGNATOR, ABBREVIATED

 

 

 

 

11A

HAZARDOUS CARGO DESIGNATOR, ABBREVIATED

 

ENTER THE APPROPRIATE HAZARDOUS CARGO DESIGNATOR, FOR EXAMPLE "RMD". See table 1221/3 which is an instructive entry

12

SINGLE DAGGER REQUIRED

 

 

 

 

12A

SINGLE DAGGER REQUIRED

 

ENTER WHETHER THERE IS A REQUIREMENT FOR CARRYING MATERIAL CODED WITH SINGLE DAGGER, "Y" FOR YES OR "N" FOR NO, FOR EXAMPLE "Y". See table 1359/22 which contains a list of data items and associated data codes
