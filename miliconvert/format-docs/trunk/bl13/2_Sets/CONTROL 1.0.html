No

Designator

Field Desc

Concept/Explanation/Examples

1

MISSION NUMBER

 

PROVIDES THE MISSION NUMBER ASSIGNED TO THE SPECIFIC MISSION TO BE CONTROLLED.

 

 

1A

MISSION NUMBER

 

ENTER THE MISSION NUMBER ASSIGNED TO THE SPECIFIC MISSION TO BE CONTROLLED, FOR EXAMPLE: "D123F". See table 1012/69 which is an instructive entry

2

UNIT IDENTIFICATION

 

THE UNIT IDENTIFICATION.

 

 

2A

UNIT IDENTIFICATION CODE (UIC)

UIC2

ENTER "UIC2:" FOLLOWED BY THE IDENTIFICATION CODE OF THE GROUND COORDINATION UNIT, FOR EXAMPLE: "UIC2:GEF1234". See table 2141 which is a composite

 

 

2B

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE DESIGNATOR OF THE GROUND COORDINATION UNIT, FOR EXAMPLE: "UNIT:ARS BROCKZETEL". See table 1028/43 which is an instructive entry

 

 

2C

BASE IDENTIFICATION CODE

BASE

ENTER "BASE:" FOLLOWED BY THE BASE IDENTIFICATION CODE FOR THE UNIT, FOR EXAMPLE: "BASE:GEF123401". See table 2141 which is a composite

 

 

2D

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE LOCATION OF THE GROUND COORDINATION UNIT USING AN ICAO BASE CODE, FOR EXAMPLE: "ICAO:ETME". See table 1125/2 which is an instructive entry

3

AIRCRAFT NUMBER

 

PROVIDES THE NUMBER OF AIRCRAFT PARTICIPATING IN THE PARTICULAR MISSION.

 

 

3A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT PARTICIPATING IN THE PARTICULAR MISSION, FOR EXAMPLE: "4". See table 1044/322 a range [1 through 99]

4

TYPE OF AIRCRAFT

 

THE TYPE OF AIRCRAFT PARTICIPATING IN THE MISSION BEING CONTROLLED.

 

 

4A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE AND MODEL NUMBER OF THE AIRCRAFT, FOR EXAMPLE: "ACTYP:FA18". See table 1015/20 which contains a list of data items and associated data codes

 

 

4B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER THE AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST, FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

5

PRIMARY AERIAL MISSION TYPE

 

PROVIDES THE CODE FOR THE TYPE OF PRIMARY MISSION BEING CONTROLLED.

 

 

5A

PRIMARY AERIAL MISSION TYPE

 

ENTER THE CODE FOR THE TYPE OF PRIMARY MISSION BEING CONTROLLED, FOR EXAMPLE: "OCA". See table 1285/31 which contains a list of data items and associated data codes

6

SECONDARY AERIAL MISSION TYPE

 

PROVIDES THE CODE FOR THE TYPE OF SECONDARY MISSION BEING CONTROLLED.

 

 

6A

SECONDARY AERIAL MISSION TYPE

 

ENTER THE CODE FOR THE TYPE OF SECONDARY MISSION BEING CONTROLLED, FOR EXAMPLE: "CAS". See table 1285/32 which contains a list of data items and associated data codes

7

STATION TIME OR AIRCRAFT ALERT STATUS

 

THE TIME ON STATION OR THE ALERT STATUS OF THE AIRCRAFT TO BE AIRBORNE.

 

 

7A

DAY-TIME ON STATION

ONSTN

ENTER "ONSTN:" FOLLOWED BY THE DAY-TIME ON STATION, FOR EXAMPLE: "ONSTN:141325Z". See table 2000 which is a composite

 

 

7B

AIRCRAFT ALERT STATUS, HOURS

 

ENTER THE READINESS STATUS EXPRESSED AS AN INDICATION OF THE TIME IN HOURS FOR THE AIRCRAFT TO BE AIRBORNE AFTER THE LAUNCH ORDER IS RECEIVED, FOR EXAMPLE "12H". See table 2139 which is a composite

 

 

7C

AIRCRAFT ALERT STATUS, MINUTES

 

ENTER THE READINESS STATUS EXPRESSED AS AN INDICATION OF THE TIME IN MINUTES FOR THE AIRCRAFT TO BE AIRBORNE AFTER THE LAUNCH ORDER IS RECEIVED, FOR EXAMPLE "30M". See table 2142 which is a composite

 

 

7D

AIRCRAFT ALERT CODE

 

ENTER THE APPROPRIATE READINESS STATUS CODE, FOR EXAMPLE "RED". See table 1798/2 which contains a list of data items and associated data codes

8

PRIMARY CONFIGURATION LOAD

 

PROVIDES THE PRIMARY CONFIGURATION LOAD FOR THE AIRCRAFT PARTICIPATING IN THE MISSION BEING CONTROLLED.

 

 

8A

PRIMARY CONFIGURATION LOAD

 

ENTER THE PRIMARY CONFIGURATION LOAD FOR THE CONTROLLED AIRCRAFT, FOR EXAMPLE: "2AGM65". See table 1012/258 which is an instructive entry

9

CALL SIGN

 

PROVIDES THE CALL SIGN ASSIGNED TO THE MISSION AIRCRAFT.

 

 

9A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN ASSIGNED TO THE MISSION AIRCRAFT, FOR EXAMPLE: "HAMMER 21". See table 1129/31 which is an instructive entry

10

CONTACT POINT

 

THE GEOGRAPHIC LOCATION OF THE CONTACT POINT FOR THE AIRCRAFT PARTICIPATING IN THE MISSION BEING CONTROLLED.

 

 

10A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

10B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

10C

GEOGRAPHIC POSITION, GEOREF, MINUTE

GEOREFM

ENTER "GEOREFM:" FOLLOWED BY THE LOCATION OF THE ROUTE POINT USING GEOREF MINUTE, FOR EXAMPLE: "GEOREFM:AABB1535". See table 2017 which is a composite
