Field Desc

Concept/Explanation/Examples

1

COMMON NATO TRACK NUMBER

 

PROVIDES THE TRACK NUMBER ASSIGNED TO THE CONTACT, EITHER NATO OR NATIONAL.

 

 

1A

COMMON NATO TRACK NUMBER

NTN

ENTER "NTN:" FOLLOWED BY THE NATO TRACK NUMBER ASSIGNED TO THE CONTACT, FOR EXAMPLE: "NTN:AB123". See table 1625/5 which is an instructive entry

 

 

1B

COMMON NATO TRACK NUMBER, 6 DIGIT

TN

ENTER "TN:" FOLLOWED BY THE  TRACK NUMBER FOR THE TARGET. FOR EXAMPLE: "TN:BB5ZAV". See table 1625/6 which is an instructive entry

2

NUMBER OF AIRCRAFT INVOLVED

 

 

 

 

2A

NUMBER OF AIRCRAFT INVOLVED

 

ENTER THE NUMBER OF TARGET AIRCRAFT. FOR EXAMPLE: "2". See table 1044/2 a range [0 through 999]

3

ENEMY HEADING IN DEGREES

 

 

 

 

3A

ENEMY HEADING IN DEGREES

 

ENTER THE  HEADING FOR THE TARGET AIRCRAFT. FOR EXAMPLE: "280T". See table 2065 which is a composite

4

ENEMY AIRCRAFT SPEED

 

 

 

 

4A

ENEMY AIRCRAFT SPEED

 

ENTER THE SPEED OF THE TARGET AIRCRAFT. FOR EXAMPLE: "550KTS". See table 2063 which is a composite

5

ALTITUDE OF TARGET

 

ENTER THE ALTITUDE OF THE TARGET AIRCRAFT USING ONE OF THE FOLLOWING:

 

 

5A

ENEMY ALTITUDE IN HUNDREDS OF FEET

 

THE ALTITUDE OF THE TARGET AIRCRAFT IN HUNDERDS OF FEET. FOR EXAMPLE: "050" See table 1255/15 a range [000 through 999]

 

 

5B

FLIGHT LEVEL

FL

ENTER "FL:" FOLLOWED BY THE FLIGHT LEVEL OF THE TARGET AIRCRAFT. FOR EXAMPLE: "FL:150" See table 2129 which is a composite

 

 

5C

ALTITUDE BAND

AB

ENTER "AB:" FOLLOWED BY THE ALTITUDE BAND OF THE TARGET AIRCRAFT. FOR EXAMPLE: "AB:350-370" See table 2100 which is a composite
