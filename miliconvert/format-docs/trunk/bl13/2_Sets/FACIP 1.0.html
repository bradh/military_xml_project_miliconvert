Concept/Explanation/Examples

1

INITIAL POINT IDENTIFIER

 

 

 

 

1A

INITIAL POINT IDENTIFIER

 

ENTER THE INITIAL POINT IDENTIFIER FOR CONTACTING THE FORWARD AIR CONTROLLER, FOR EXAMPLE: "IP GREEN ONE". See table 1261/6 which is an instructive entry

2

POINT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE INITIAL POINT AT WHICH THE AIRCRAFT SHOULD CONTACT THE FAC.

 

 

2A

POSITION OR POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF A CONTACT POINT, FOR EXAMPLE: "NAME:POINT ALFA". See table 1022/18 which is an instructive entry

 

 

2B

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2C

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

3

INITIAL POINT ELEVATION

 

 

 

 

3A

INITIAL POINT ELEVATION

 

ENTER THE ELEVATION OF THE CONTACT POINT MEASURED IN FEET OR METRES ABOVE OR BELOW MEAN SEA LEVEL OR ABOVE THE REFERENCE ELLIPSOID, FOR EXAMPLE: "200FTMSL". See table 2492 which is a composite

4

INITIAL POINT DESCRIPTION

 

 

 

 

4A

INITIAL POINT DESCRIPTION

 

ENTER, AS A LITERAL, THE INITIAL POINT DESCRIPTION, FOR EXAMPLE: "CHURCH STEEPLE IN ST LO". See table 1363/19 which is an instructive entry
