1

TYPE OF RADIO EQUIPMENT

 

 

 

 

1A

TYPE OF RADIO EQUIPMENT

 

ENTER THE CODE FOR THE TYPE OF RADIO EQUIPMENT IN THE CORRECT ORDER, FOR EXAMPLE: "TRANSMIT". See table 1135/17 which contains a list of data items and associated data codes

2

NUMBER OF ITEMS OF RADIO EQUIPMENT

 

 

 

 

2A

NUMBER OF ITEMS OF RADIO EQUIPMENT

 

ENTER THE NUMBER OF ITEMS OF RADIO EQUIPMENT SPECIFIED IN FIELD 1, FOR EXAMPLE: "5". See table 1044/231 a range [1 through 99]

3

TRANSMISSION BAND

 

 

 

 

3A

TRANSMISSION BAND

 

ENTER THE TRANSMISSION BAND, FOR EXAMPLE: "UHF". See table 1175/3 which contains a list of data items and associated data codes

4

EMISSION TYPE

 

ENTER THE EMISSION TYPE USING ONE OF THE FOLLOWING:

 

 

4A

DESIGNATION OF EMISSIONS

 

THE DESIGNATION OF EMISSIONS AS LISTED IN ACP 176 NS-1 PARA 502, FOR EXAMPLE: "J3E". See table 1621/1 which contains a list of data items and associated data codes

 

 

4B

CRYSTAL RADIO CODE

 

IF A CRYSTAL RADIO ENTER: "XTAL". See table 1181/5 which contains a list of data items and associated data codes

5

FREQUENCY SEPARATION

 

ENTER THE FREQUENCY SEPARATION USING ONE OF THE FOLLOWING:

 

 

5A

FREQUENCY SEPARATION, KILOHERTZ OR MEGAHERTZ

 

THE FREQUENCY AND THE UNIT OF FREQUENCY MEASUREMENT, FOR EXAMPLE: "500KHZ". See table 2385 which is a composite

 

 

5B

FREQUENCY SEPARATION IN PERCENTAGE

PCT

ENTER FIELD DESCRIPTOR "PCT:" FOLLOWED BY THE FREQUENCY SEPARATION AS A PERCENTAGE, FOR EXAMPLE: "PCT:1". See table 1064/34 a range [0 through 99]

6

FREQUENCY LIMITATION

 

ENTER THE FREQUENCY LIMITATION USING ONE OF THE FOLLOWING:

 

 

6A

FREQUENCY LIMITATION, KILOHERTZ OR MEGAHERTZ

 

A SINGLE FREQUENCY IN MHZ OR KHZ WHICH CANNOT BE USED, FOR EXAMPLE: 1234.500KHZ". See table 2385 which is a composite

 

 

6B

BAND LIMITATIONS

 

THE LOWER AND UPPER FREQUENCY IN MHZ OR KHZ WHICH CANNOT BE USED, FOR EXAMPLE: "250-500KHZ". See table 2377 which is a composite
