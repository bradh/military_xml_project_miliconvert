1

WEAPON TYPE

 

THE WEAPON TYPE DESCRIBED AS ONE OF THE FOLLOWING:

 

 

1A

MARITIME ABOVE WATER WEAPON SYSTEMS

 

ENTER THE WEAPON TYPE, FOR EXAMPLE: "MM40". See table 1407/55 which contains a list of data items and associated data codes

 

 

1B

WEAPON TYPE, GENERIC

GEN

ONLY TO BE USED IF THE WEAPON SYSTEM IS NOT AVAILABLE IN ALT A.  ENTER "GEN:" FOLLOWED BY THE NAME OF A WEAPON SYSTEM, FOR EXAMPLE: "GEN:OERLIKON". See table 1407/1 which is an instructive entry

2

SALVO POLICY

 

 

 

 

2A

SALVO POLICY

 

ENTER THE REQUIRED POLICY, EITHER "SHOOT LOOK" OR "SHOOT SHOOT LOOK" FOR MISSILES, OR "SHOOT LOOK SHOOT" FOR GUNS, FOR EXAMPLE: "SHOOT LOOK". See table 1533/3 which contains a list of data items and associated data codes

3

SALVO INTERVAL IN MINUTES

 

 

 

 

3A

SALVO INTERVAL IN MINUTES

 

ENTER THE REQUIRED INTERVAL IN MINUTES, FOR EXAMPLE: "2". See table 1072/18 a range [1 through 99]

4

SALVO SIZE

 

 

 

 

4A

SALVO SIZE

 

ENTER THE SALVO SIZE IN NUMBER OF ROUNDS PER WEAPON, FOR EXAMPLE: "12". See table 1044/237 a range [1 through 99]

5

NUMBER OF WEAPONS/ROUNDS

 

 

 

 

5A

NUMBER OF WEAPONS ROUNDS

 

IF USED, SPECIFIES THE AMOUNT OF WEAPONS TO BE EXPENDED DURING A SINGLE ENGAGEMENT, EXPRESSED AS NUMBER OF WEAPONS OR ROUNDS, FOR EXAMPLE: "4". See table 1044/58 a range [1 through 999]
