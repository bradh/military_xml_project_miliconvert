CALL SIGN

 

 

 

 

1A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN OF THE FORWARD AIR CONTROLLER, FOR EXAMPLE: "NAIL 10". See table 1129/31 which is an instructive entry

2

PRIMARY COMMUNICATION PARAMETERS

 

PROVIDES THE PRIMARY COMMUNICATION PARAMETERS AS REQUIRED ASSIGNED TO THE FORWARD AIR CONTROLLER (FAC), AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.

 

 

2A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY TO BE USED BY THE FAC, FOR EXAMPLE: "PFREQ:235.7". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

2B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR FOR CONTACTING THE FAC, FOR EXAMPLE: "PDESIG:TAD001". See table 1396/2 which is an instructive entry

 

 

2C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

2D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQK:BA12-CD34-12082000". See table 2535 which is a composite

3

SECONDARY COMMUNICATIONS PARAMETERS

 

PROVIDES THE SECONDARY COMMUNICATIONS PARAMETERS AS REQUIRED  ASSIGNED TO THE FORWARD AIR CONTROLLER (FAC), AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.

 

 

3A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE SECONDARY FREQUENCY TO BE USED BY THE FAC, FOR EXAMPLE: "SFREQ:250.25". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

3B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY DESIGNATOR FOR CONTACTING THE FAC, FOR EXAMPLE: "SDESIG:RED11". See table 1396/3 which is an instructive entry

 

 

3C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

SHVQKM

ENTER "SHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

3D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

SHVQK

ENTER "SHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQK:AB12-CD34-12082000". See table 2535 which is a composite

4

LINK 16 CALL SIGN

 

 

 

 

4A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE LINK 16 ABBREVIATED CALL SIGN IF THE FAC IS LINK 16 EQUIPPED, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry

5

CONTACT POINT

 

THE GEOGRAPHIC LOCATION OF THE POINT AT WHICH THE AIRCRAFT SHOULD CONTACT THE FAC.

 

 

5A

POSITION OR POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF A CONTACT POINT, FOR EXAMPLE: "NAME:POINT ALFA". See table 1022/18 which is an instructive entry

 

 

5B

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

6

SUPPORTED UNIT IDENTIFIER

 

PROVIDES THE UNIT IDENTIFICATION.

 

 

6A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIQUE IDENTIFICATION OF THE UNIT BEING SUPPORTED, FOR EXAMPLE: "UNIT:1BDE". See table 1028/43 which is an instructive entry

 

 

6B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE IDENTIFICATION CODE OF THE GROUND UNIT BEING SUPPORTED, FOR EXAMPLE: "UIC:GEF1234". See table 2141 which is a composite

7

FAC UNIT REFERENCE NUMBER

 

 

 

 

7A

FAC UNIT REFERENCE NUMBER

URN

ENTER "URN:" FOLLOWED BY THE UNIT REFERENCE NUMBER IDENTIFYING THE AIRBORNE OR GROUNDBASED FORWARD AIR CONTROLLER (FAC) TO CONTACT, FOR EXAMPLE "URN: 13578642". See table 1028/46 a range [00000000 through 16777215]

8

FAC INTERNET ADDRESS

 

PROVIDES THE IP ADDRESS OF THE COMMUNICATION DEVICE USED BY THE AIRBORNE OR GROUNDBASED FORWARD AIR CONTROLLER (FAC) TO CONTACT.

 

 

8A

FAC INTERNET ADDRESS, IPV4

IPV4

ENTER "IPV4:" FOLLOWED BY THE IP ADDRESS VERSION 4 OF THE COMMUNICATION DEVICE USED BY THE AIRBORNE OR GROUNDBASED FORWARD AIR CONTROLLER (FAC) TO CONTACT, FOR EXAMPLE "IPV4:192.168.46.120". See table 2004 which is a composite

 

 

8B

FORWARD AIR CONTROLLER INTERNET ADDRESS, IPV6

IPV6

ENTER "IPV6:" FOLLOWED BY THE IP ADDRESS VERSION 6 OF THE COMMUNICATION DEVICE USED BY THE AIRBORNE OR GROUNDBASED FORWARD AIR CONTROLLER (FAC) TO CONTACT. See table 1999/4 which is an instructive entry

9

FAC DATA LINK ADDRESS

 

 

 

 

9A

FAC DATA LINK ADDRESS

 

ENTER THE DATA LINK ADDRESS FOR THE AIRBORNE OR GROUNDBASED FORWARD AIR CONTROLLER (FAC) TO CONTACT THAT REPRESENTS THE SOURCE/DESTINATION OF THE MESSAGE, FOR EXAMPLE: "080". See table 1012/165 a range [000 through 127]
