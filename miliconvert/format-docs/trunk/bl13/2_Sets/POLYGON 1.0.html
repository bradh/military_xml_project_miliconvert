POLYGON POINTS

 

THE POLYGON IS DRAWN CLOCKWISE WITH AT LEAST THREE POINTS. THE LAST POINT WILL AUTOMATICALLY BE THE SAME AS THE FIRST, SO THE POLYGON IS CLOSED. THEREFORE IT DOES NOT NEED TO BE SPECIFIED.  ENTER THE INDIVIDUAL POINTS THAT ARE NEEDED TO CONSTRUCT THE POLYGON SHAPE, REPEAT THE FIELD AS NECESSARY.
