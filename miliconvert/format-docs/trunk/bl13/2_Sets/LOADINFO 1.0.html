DATA ENTRY

 

ENTER THE TWO-DIGIT NUMBER (01 THROUGH 99) THAT TIES THE INFORMATION IN THIS SET TO ENTRIES IN SET 2LOAD, FOR EXAMPLE: "01". See table 1348/1 a range [01 through 99]

2A

TOTAL WEIGHT OF AIR CARGO

 

ENTER THE TOTAL WEIGHT OF THE CARGO.  ENTER THE CONTEXT QUANTITY (1 THROUGH 999999.9) (DECIMAL POINT PERMITTED) AND THE UNIT OF WEIGHT MEASUREMENT, FOR EXAMPLE: "12530KG". See table 2185 which is a composite

HEAVIEST UNIT WEIGHT

 

 

 

 

3A

HEAVIEST UNIT WEIGHT

HSI

ENTER THE FIELD DESCRIPTOR "HSI:" FOLLOWED BY THE WEIGHT OF THE HEAVIEST SINGLE UNIT OF CARGO.  ENTER THE CONTEXT QUANTITY (1 THROUGH 999999.9) (DECIMAL POINT PERMITTED) AND THE UNIT OF WEIGHT MEASUREMENT, FOR EXAMPLE: "HSI4700KG". See table 2185 which is a composite

TYPE OF HEAVIEST AIR CARGO

 

ENTER THE TYPE OF THE HEAVIEST SINGLE UNIT OF CARGO, FOR EXAMPLE: "PALL". See table 1173/7 which contains a list of data items and associated data codes

DIMENSIONS OF LARGEST UNIT OF AIR CARGO

LSI

ENTER THE FIELD DESCRIPTOR "LSI:' FOLLOWED BY THE DIMENSION OF THE LARGEST UNIT OF CARGO.  ENTER THE CONTEXT QUANTITY (0 THROUGH 99999) (DECIMAL POINT PERMITTED,  THE DISTANCE DELIMITER "X", THE CONTEXT QUANTITY (O THROUGH 99999) (DECIMAL POINT PERMITTED), THE DISTANCE DELIMITER "X", THE CONTEXT QUANTITY (0 THROUGH 99999) (DECIMAL POINT PERMITTED), AND THE UNIT OF LINEAR MEASUREMENT, FOR EXAMPLE: "LSI:200X190X185CM". See table 2182 which is a composite
