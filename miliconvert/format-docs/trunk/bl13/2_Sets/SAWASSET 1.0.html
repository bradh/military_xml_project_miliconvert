Concept/Explanation/Examples

1

UNIT IDENTIFICATION

 

PROVIDES THE UNIT IDENTIFICATION OF THE SAW UNIT WHICH IS ASSIGNED OR ATTACHED TO A PARENT COMMAND AND CONTROL UNIT.

 

 

1A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UNIT IDENTIFICATION CODE FOR THE UNIT, FOR EXAMPLE: "UIC:GEF1234". See table 2141 which is a composite

 

 

1B

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT NAME, FOR EXAMPLE: "UNIT:1ST FTR WG". See table 1028/43 which is an instructive entry

2

ORDNANCE/WEAPON TYPE, SURFACE-TO-AIR

 

PROVIDES THE WEAPON TYPE TO BE TASKED.

 

 

2A

SURFACE-TO-AIR WEAPON SYSTEM

SAW

ENTER "SAW:" FOLLOWED BY THE TYPE OF SURFACE-TO-AIR WEAPON ASSIGNED/ATTACHED, FOR EXAMPLE: "SAW:CROTAL". See table 1407/2 which contains a list of data items and associated data codes

 

 

2B

SURFACE-TO-AIR WEAPON SYSTEM TYPE, OTHER

OTHSAW

ENTER "OTHSAW:" FOLLOWED BY THE ALTERNATE TYPE OF SURFACE-TO-AIR WEAPON ASSIGNED/ATTACHED, FOR WHICH NO CODE IS AVAILABLE, FOR EXAMPLE "OTHSAW:SA-XYZ11". See table 1407/24 which is an instructive entry
