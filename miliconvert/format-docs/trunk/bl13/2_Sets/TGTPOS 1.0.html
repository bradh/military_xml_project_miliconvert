No

Designator

Field Desc

Concept/Explanation/Examples

1

TARGET DESCRIPTION

 

A DESCRIPTION OF THE TARGET, EITHER SHIP OR NON-SHIP.

 

 

1A

NON-SHIP TARGET TYPE

 

ENTER A DESCRIPTION OF A TARGET WHICH IS NOT A SHIP, FOR EXAMPLE: "AFBASE". See table 1650/6 which contains a list of data items and associated data codes

 

 

1B

SHIP TARGET TYPE

 

ENTER THE DESCRIPTION OF A TARGET WHICH IS A SHIP, FOR EXAMPLE "PTL". See table 1166/11 which contains a list of data items and associated data codes

2

TARGET NAME

 

THE NAME OF THE TARGET.

 

 

2A

TARGET FACILITY NAME

 

ENTER THE NAME OF THE TARGET, FOR EXAMPLE: "KRIVAK". See table 1022/234 which is an instructive entry

3

TARGET POSITION IDENTIFIER

 

THE NAME USED TO IDENTIFY THE TARGET POSITION.

 

 

3A

TARGET IDENTIFIER, ABBREVIATED

 

ENTER THE NAME USED TO IDENTIFY THE TARGET POSITION, FOR EXAMPLE: "DAUNE PASS". See table 1416/8 which is an instructive entry

4

TARGET AREA DESCRIPTION

 

A DESCRIPTION IN PLAIN WORDS OF THE AREA WHERE THE TARGET IS POSITIONED.

 

 

4A

TARGET AREA DESCRIPTION

 

ENTER THE TARGET AREA DESCRIPTION, FOR EXAMPLE: "MARSHALLING YARD". See table 1363/6 which is an instructive entry

5

REQUEST IDENTIFIER

 

THE SERIAL IDENTIFIER OF THE REQUEST.

 

 

5A

REQUEST IDENTIFIER

 

ENTER THE REQUEST IDENTIFIER, FOR EXAMPLE:"AW168GT30". See table 1012/144 which is an instructive entry

6

POINT OF IMPACT

 

AN INDICATION OF THE DESIRED POINT OF IMPACT, EITHER AS DESIRED MEAN POINT OF IMPACT OR AS MOBILE TARGET NUMBER.

 

 

6A

DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER

DMPI

ENTER "DMPI:", FOLLOWED BY THE DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER, FOR EXAMPLE "DMPI:BL-129". See table 1416/9 which is an instructive entry

 

 

6B

MOBILE TARGET NUMBER

MT

ENTER "MT:", FOLLOWED BY THE MOBILE TARGET NUMBER ASSIGNED TO THE SPECIFIC TARGET, FOR EXAMPLE: "MT:CA53363". See table 1012/295 which is an instructive entry

7

DESIRED MEAN POINT OF IMPACT (DMPI) DESCRIPTION.

 

A PLAIN LANGUAGE DESCRIPTION OF THE DESIRED MEAN POINT OF IMPACT.

 

 

7A

DESIRED MEAN POINT OF IMPACT DESCRIPTION

 

ENTER A PLAIN LANGUAGE DESCRIPTION OF THE DESIRED MEAN POINT OF IMPACT (DMPI), FOR EXAMPLE "SHIP TOPMAST". See table 1363/44 which is an instructive entry

8

TARGET POSITION

 

THE GEOGRAPHIC LOCATION OF THE TARGET.

 

 

8A

TARGET POSITION, LAT/LONG, CENTISECONDS

LATCS

ENTER "LATCS:", FOLLWED BY THE TARGET POSITION IN LAT/LONG, CENTISECONDS, FOR EXAMPLE: "LATCS:58253010N015205522W". See table 2488 which is a composite

 

 

8B

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

9

TARGET ELEVATION

 

THE ELEVATION OF THE TARGET IN FEET OR METRES

 

 

9A

TARGET ELEVATION IN FEET OR METRES

 

ENTER THE TARGET ELEVATION, SPECIFYING THE UNIT OF MEASUREMENT, FEET (FT) OR METRES (M), FOR EXAMPLE: "2910FT". See table 2148 which is a composite

10

TARGET HEADING

 

THE TARGET HEADING  IN DEGREES, TRUE, GRID, MAGNETIC OR RELATIVE.

 

 

10A

MOBILE TARGET HEADING IN DEGREES

 

ENTER THE TARGET HEADING IN DEGREES, SPECIFYING THE ANGULAR MEASUREMENT REFERENCE TRUE NORTH (T), MAGNETIC NORTH (M), GRID NORTH (G) OR RELATIVE (R), FOR EXAMPLE: "100M". See table 2065 which is a composite

11

TARGET SPEED

 

THE MOBILE TARGET SPEED, IN KPH, KTS, MPH.

 

 

11A

MOBILE TARGET SPEED IN KPH, KTS, MPH

 

ENTER THE TARGET SPEED, SPECIFIYNG THE UNIT OF MEASUREMENT KPH, KTS OR MPH, FOR EXAMPLE: "420KTS". See table 2461 which is a composite
