No

Designator

Field Desc

Concept/Explanation/Examples

1

MISSION NUMBER

 

 

 

 

1A

MISSION NUMBER

 

ENTER THE MISSION NUMBER OF THE AIRCRAFT BEING REFUELLED, FOR EXAMPLE: "0151N". See table 1012/69 which is an instructive entry

2

RECEIVER CALL SIGN

 

 

 

 

2A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN OF THE AIRCRAFT THAT IS TO RECEIVE FUEL,  FOR EXAMPLE: "HAMMER 51". See table 1129/31 which is an instructive entry

3

AIRCRAFT NUMBER

 

 

 

 

3A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT ALLOCATED TO THE MISSION, FOR EXAMPLE: "2". See table 1044/322 a range [1 through 99]

4

TYPE OF AIRCRAFT

 

PROVIDES THE TYPE OF AIRCRAFT BEING REFUELLED.

 

 

4A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE AND MODEL NUMBER OF THE AIRCRAFT, FOR EXAMPLE: "ACTYP:FA18". See table 1015/20 which contains a list of data items and associated data codes

 

 

4B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF THE AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST, FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

5

OFF LOAD FUEL QUANTITY

 

 

 

 

5A

OFF LOAD FUEL QUANTITY (HKG OR KLB)

 

ENTER THE TOTAL AMOUNT OF FUEL IN THOUSANDS OF POUNDS OR HUNDREDS OF KILOGRAMS TO BE TRANSFERRED TO A FLIGHT OF ONE OR MORE AIRCRAFT DURING INFLIGHT REFUELLING, FOR EXAMPLE: "50.80KLB". See table 2468 which is a composite

6

AIR REFUELLING CONTROL DAY-TIME

 

 

 

 

6A

DAY-TIME OF AIR REFUELLING CONTROL

 

ENTER THE DAY-TIME THE AIR REFUELLING AIRCRAFT WILL ARRIVE AT THE AIR REFUELLING CONTROL POINT, FOR EXAMPLE: "141325Z". See table 2000 which is a composite

7

CELL SEQUENCE NUMBER(S)

 

PROVIDES THE TANKER(S) POSITION WITHIN THE AIR REFUELING CELL.

 

 

7A

CELL SEQUENCE NUMBER

NUM

ENTER "NUM:" FOLLOWED BY THE RELATIVE NUMBER OF A SINGLE TANKER AIRCRAFT IN A FLIGHT, THE LEAD TANKER IS 1, FOR EXAMPLE:"NUM:3". See table 1012/253 a range [1 through 9]

 

 

7B

CELL SEQUENCE NUMBERS

NUMS

ENTER "NUMS:" FOLLOWED BY THE RELATIVE NUMBERS OF TWO TANKER AIRCRAFT IN A CELL, SEPARATED BY A COMMA. THE LEAD TANKER IS 1, FOR EXAMPLE: "NUMS:1,3". See table 2436 which is a composite

8

REFUELLING FUEL TYPE

 

PROVIDES THE TYPE OF FUEL TO BE OFFLOADED BY THE TANKER DURING REFUELLING.

 

 

8A

REFUELLING FUEL GRADE

GRADE

ENTER "GRADE:" FOLLOWED BY THE CODE FOR THE TYPE OF FUEL TO BE OFFLOADED BY A TANKER DURING AERIAL REFUELLING, FOR EXAMPLE: "GRADE:JP8". See table 1803/4 which contains a list of data items and associated data codes

 

 

8B

NATO STANDARD AVIATION FUEL TYPE

NFUEL

ENTER "NFUEL:" FOLLOWED BY THE CODE FOR THE  NATO STANDARD FUEL TYPE, FOR EXAMPLE: "NFUEL:F34". See table 1172/3 which contains a list of data items and associated data codes

9

AERIAL REFUELLING SYSTEM

 

 

 

 

9A

AERIAL REFUELLING SYSTEM, SPECIFIC

 

ENTER THE SYSTEM USED BY AN AIRCRAFT FOR AERIAL REFUELLING, FOR EXAMPLE: "CLD". See table 1579/2 which contains a list of data items and associated data codes

10

RECEIVER UNIT REFERENCE NUMBER

 

 

 

 

10A

RECEIVER UNIT REFERENCE NUMBER

URN

ENTER "URN:" FOLLOWED BY THE URN REFERENCE NUMBER IDENTIFYING THE AIRCRAFT RECEIVING THE AIR-TO-AIR REFUELLING, FOR EXAMPLE "URN:14280140". See table 1028/47 a range [00000000 through 16777215]

11

RECEIVER INTERNET ADDRESS

 

THE IP ADDRESS OF THE COMMUNICATION DEVICE USED BY THE AIRCRAFT RECEIVING THE AIR-TO-AIR REFUELLING.

 

 

11A

RECEIVER INTERNET ADDRESS, IPV4

IPV4

ENTER "IPV4:" FOLLOWED BY THE IP ADDRESS VERSION 4 OF THE COMMUNICATION DEVICE USED BY THE AIRCRAFT RECEIVING THE AIR-TO-AIR REFUELLING, FOR EXAMPLE "IPV4:192.168.6.125". See table 2004 which is a composite

 

 

11B

RECEIVER INTERNET ADDRESS, IPV6

IPV6

ENTER "IPV6:" FOLLOWED BY THE IP ADDRESS VERSION 6 OF THE COMMUNICATION DEVICE USED BY THE AIRCRAFT RECEIVING THE AIR-TO-AIR REFUELLING. See table 1999/5 which is an instructive entry

12

RECEIVER DATA LINK ADDRESS

 

 

 

 

12A

RECEIVER DATA LINK ADDRESS

 

ENTER THE DATA LINK ADDRESS FOR THE AIRCRAFT RECEIVING THE  AIR-TO-AIR REFUELLING THAT REPRESENTS THE SOURCE/DESTINATION OF THE MESSAGE, FOR EXAMPLE: "072". See table 1012/167 a range [000 through 127]
