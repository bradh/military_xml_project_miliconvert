No

Designator

Field Desc

Concept/Explanation/Examples

1

QUANTITY OF ORDNANCE

 

 

 

 

1A

QUANTITY OF ORDNANCE

 

ENTER THE QUANTITY OF ORDNANCE TO BE DELIVERED ON THE TARGET, FOR EXAMPLE: "145". See table 1044/131 a range [0 through 999]

2

WEAPON TYPE

 

 

 

 

2A

WEAPON TYPE, GENERIC

 

ENTER THE GENERIC TYPE OF WEAPON, FOR EXAMPLE: "CRUISE  MISSILE". See table 1407/1 which is an instructive entry

3

LASER/DATA LINK CODE

 

PROVIDES A CODE THAT ENSURES CORRELATION BETWEEN THE TARGET DESIGNATION SYSTEM AND THE WEAPON DELIVERY SYSTEM.

 

 

3A

LASER CORRELATION CODE

LC

ENTER "LC:" FOLLOWED BY THE LASER CODE, FOR EXAMPLE: "LC:3290". See table 1012/254 a range [0000 through 9999]

 

 

3B

DATA LINK FREQUENCY CODE

DL

ENTER "DL:" FOLLOWED BY THE DATA LINK FREQUENCY CODE, FOR EXAMPLE: "DL:A1". See table 1396/4 which is an instructive entry

4

GUIDANCE FREQUENCY

 

PROVIDES THE GUIDANCE FREQUENCY FOR THE PRECISION GUIDED WEAPON. CAN ONLY BE USED IN CONJUNCTION WITH A LASER/DATA LINK CODE SPECIFIED IN THIS SET.

 

 

4A

GUIDANCE FREQUENCY IN MEGAHERTZ

FREQ

ENTER "FREQ:" FOLLOWED BY THE FREQUENCY USED FOR IN-FLIGHT/TERMINAL GUIDANCE OF PRECISION MUNITIONS, FOR EXAMPLE: "FREQ:123.456". See table 1068/11 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

4B

GUIDANCE FREQUENCY DESIGNATOR

DESIG

ENTER "DESIG:" FOLLOWED BY THE FREQUENCY DESIGNATOR USED FOR IN-FLIGHT/TERMINAL GUIDANCE OF PRECISION MUNITIONS, FOR EXAMPLE: "DESIG:RED01". See table 1396/7 which is an instructive entry

5

CONTACT CALL SIGN

 

 

 

 

5A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN OF PRECISION GUIDED MUNITION, FOR EXAMPLE: "BRAVO CHARLIE 35". See table 1129/31 which is an instructive entry

6

PRIMARY COMMUNICATION PARAMETERS

 

PROVIDES THE PRIMARY COMMUNICATION PARAMETERS FOR CONTACTING THE PRECISION GUIDED MUNITION.

 

 

6A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY IN MEGAHERTZ, FOR EXAMPLE: "PFREQ:243.5". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

6B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR, FOR EXAMPLE: "PDESIG:TAD001". See table 1396/2 which is an instructive entry

 

 

6C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

6D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQK:AB12-CD34-12082000". See table 2535 which is a composite

7

SECONDARY COMMUNICATIONS PARAMETERS

 

PROVIDES THE SECONDARY COMMUNICATION PARAMETERS.  CAN ONLY BE USED IN CONJUNCTION WITH PRIMARY COMMUNICATION PARAMETERS SPECIFIED IN THIS SET.

 

 

7A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE  SECONDARY FREQUENCY TO BE USED, FOR EXAMPLE: "SFREQ:350.5". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

7B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY DESIGNATOR, FOR EXAMPLE: "SDESIG:TAD002". See table 1396/3 which is an instructive entry

 

 

7C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

SHVQKM

ENTER "SHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

7D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

SHVQK

ENTER "SHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQK:AB12-CD34-12082000". See table 2535 which is a composite

8

LINK 16 CALL SIGN

 

 

 

 

8A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE LINK 16 ABBREVIATED CALL SIGN, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry
