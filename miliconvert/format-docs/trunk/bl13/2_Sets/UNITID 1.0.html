1

UNIT KEY

 

 

 

 

1A

UNIT KEY

 

ENTER THE KEY WHICH CLEARLY IDENTIFIES THE UNIT, FOR EXAMPLE: "U-GE12-A-GE-AB123". See table 2443 which is a composite

2

UNIT NAME, EXPANDED

 

 

 

 

2A

UNIT NAME, EXPANDED

 

ENTER THE TRANSLATED UNIT NAME OR IDENTIFICATION GIVEN THE UNIT IN APPROPRIATE ORDERS OR COMMUNICATIONS WITHIN THE NATIONAL MILITARY OR CIVILIAN ESTABLISHMENT OF THE COUNTRY OF ALLEGIANCE, FOR EXAMPLE: "IRON HEADS". See table 1022/253 which is an instructive entry

3

ALTERNATE ALLEGIANCE

 

 

 

 

3A

ALTERNATE ALLEGIANCE

 

ENTER THE CODE OF THE COUNTRY OR POLITICAL ENTITY TO WHICH THE UNIT ALSO OWES ALLEGIANCE; FOR EXAMPLE: "AR". See table 1026/25 which contains a list of data items and associated data codes

4

FORCE

 

 

 

 

4A

FORCE

 

ENTER THE FORCE CODE TO IDENTIFY THE UNIT, FOR EXAMPLE: "AMD". See table 1107/4 which contains a list of data items and associated data codes

5

FORCE NAME

 

 

 

 

5A

FORCE NAME

 

MAKE A LITERAL ENTRY GIVING THE SPECIFIC NAME FOR A GIVEN FORCE, FOR EXAMPLE: "JOHNSTON FORCE." See table 1022/299 which is an instructive entry

6

UNIT ECHELON (NATIVE)

 

 

 

 

6A

UNIT ECHELON (NATIVE)

 

ENTER  A VALUE WHICH REFLECTS THE NATIONAL TERM FOR THE LEVEL AT WHICH THE UNIT EXISTS IN ITS OWN COMMAND STRUCTURE, FOR EXAMPLE: "BDE". See table 1045/9 which contains a list of data items and associated data codes

7

COMMAND LEVEL (NATO)

 

 

 

 

7A

COMMAND LEVEL (NATO)

 

ENTER THE NATO TERM FOR THE UNIT'S POSITION IN THE CHAIN OF COMMAND, FOR EXAMPLE: "CPS". See table 1045/8 which contains a list of data items and associated data codes

8

PREDECESSOR UNIT KEY

 

 

 

 

8A

PREDECESSOR UNIT KEY

 

ENTER THE UNIT KEY OF THE UNIT WHICH IS IMMEDIATELY SUPERIOR TO THE NAMED UNIT IN THE CHAIN OF COMMAND, FOR EXAMPLE: "U-UK-A-US-AB123". See table 2443 which is a composite

9

REMARKS

 

 

 

 

9A

REMARKS

 

ENTER ANY ADDITIONAL INFORMATION WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS IN THIS SET. See table 1357/28 which is an instructive entry
