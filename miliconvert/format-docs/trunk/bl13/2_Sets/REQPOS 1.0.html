Designator

Field Desc

Concept/Explanation/Examples

1

REQUEST LOCATION

 

 

 

 

1A

REQUEST LOCATION

 

ENTER TYPE OF LOCATION, FOR EXAMPLE: "RADIUS". See table 1284/6 which contains a list of data items and associated data codes

2

LOCATION

 

THE GEOGRAPHICAL LOCATION OF PLACE OF INTEREST.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE GEOGRAPHIC LOCATION OF THE RENDEZVOUS/BREAKUP ACTIVITY IN GEOREF, CENTIMINUTE, FOR EXAMPLE: "GEO:AABB15233527". See table 2018 which is a composite

3

TARGET

 

ENTER ONE OF THE FOLLOWING TARGET DATA SPECIFYING SHAPE AND SIZE OF A TARGET.  IF FIELD 1 OF THIS SET CONTAINS "RADIUS", ALTERNATIVE "A" OF THIS FIELD MUST BE USED.  IF FIELD 1 OF THIS SET CONTAINS "WIDTH", ALTERNATIVE "B" OF THIS FIELD MUST BE USED.  IF FIELD 1 OF THIS SET CONTAINS "ELLIPSE", ALTERNATIVE "C" OF THIS FIELD MUST BE USED.

 

 

3A

TARGET RADIUS, FT, YD, SM, NM, M, KM, HF

RAD

ENTER "RAD:" FOLLOWED BY THE RADIUS OF A CIRCULAR TARGET AND THE UNIT OF MEASUREMENT, FOR EXAMPLE: "100M". See table 2178 which is a composite

 

 

3B

TARGET WIDTH, FT, YD, SM, NM, M, KM, HF

WDTH

ENTER "WDTH:" FOLLOWED BY THE WIDTH OF A RECTANGULAR TARGET AND THE UNIT OF MEASUREMENT, FOR EXAMPLE: "100M". See table 2179 which is a composite

 

 

3C

ELLIPTICAL TARGET AREA

ELP

ENTER "ELP:" FOLLOWED BY THE VALUE /UNIT OF MEASUREMENT FOR THE SEMI-MAJOR AND SEMI-MINOR AXIS AND THE ORIENTATION OF THE SEMI-MAJOR AXIS OF AN ELLIPTICAL AREA, FOR EXAMPLE: "ELP:10KM-7KM-120.0". See table 2180 which is a composite

4

ELEVATION IN METRES

 

 

 

 

4A

ELEVATION IN METRES

 

ENTER THE ELEVATION OF THE LOCATION IN METRES [0-9999], FOR EXAMPLE: "555". See table 1344/2 a range [-610 through 9999]

5

DAY-TIME AND MONTH

 

 

 

 

5A

DAY-TIME AND MONTH

 

ENTER DAY-TIME AND MONTH OF THE LAST KNOWN INFORMATION [DAY, HOUR, MINUTE, TIME ZONE, MONTH NAME], FOR EXAMPLE: "221635AJAN". See table 2030 which is a composite

6

COURSE OR DIRECTION OF MOVEMENT

 

IF USED, ENTER TARGET COURSE OR DIRECTION OF MOVEMENT THAT THE MOVING ITEM IS TAKING IN ONE OF THE FOLLOWING WAYS:

 

 

6A

TARGET COURSE IN DEGREES

 

ENTER TARGET COURSE WHAT CONSISTS OF THE DEGREE NUMBER [000-359] AND THE ANGULAR MEASUREMENT REFERENCE, FOR EXAMPLE: "180M". See table 2065 which is a composite

 

 

6B

DIRECTION OF MOVEMENT IN MILS

MILS

ENTER "MILS:" FOLLOWED BY THE DIRECTION OF MOVEMENT IN MILS [0-6400], FOR EXAMPLE: "MILS:2995". See table 1048/5 a range [0 through 6400]

 

 

6C

DIRECTION OF MOVEMENT IN DEGREES

DEG

ENTER "DEG:" FOLLOWED BY THE DIRECTION OF MOVEMENT IN DEGREES [000-359], FOR EXAMPLE: "DEG:180". See table 1043/9 a range [001 through 360]

 

 

6D

DIRECTION OF MOVEMENT (COMPASS POINT)

 

ENTER DIRECTION OF MOVEMENT IN ONE OF SIXTEEN POINTS OF THE COMPASS, FOR EXAMPLE:  "SSW". See table 1161/1 which contains a list of data items and associated data codes

7

TARGET SPEED

 

IF USED, ENTER THE SPEED AT WHICH THE TARGET IS TRAVELLING IN ONE OF THE FOLLOWING WAYS:

 

 

7A

TARGET SPEED

 

ENTER SPEED VALUE FOLLOWED BY THE UNIT OF SPEED MEASUREMENT, FOR EXAMPLE: "2KPH". See table 2063 which is a composite

 

 

7B

APPROXIMATE SPEED

 

ENTER CODE SPECIFYING TARGET SPEED APPROXIMATELY, FOR EXAMPLE: "FAST". See table 1113/5 which contains a list of data items and associated data codes

8

ADDITIONAL LOCATION

 

THE GEOGRAPHICAL POSITION OF THE ADDITIONAL POINTS OF INTEREST.  MUST BE USED ONLY IF FIELD 1 OF THIS SET EQUALS "MULTIPLE" OR "LINE" OR "TRACEPNT" OR "WIDTH".

 

 

8A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES
