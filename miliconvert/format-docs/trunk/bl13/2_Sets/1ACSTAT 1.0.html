AIRCRAFT IDENTIFICATION NUMBER

 

 

 

 

1A

AIRCRAFT IDENTIFICATION NUMBER

 

ENTER THE E-3 AIRCRAFT IDENTIFICATION NUMBER FOUND ON THE TAIL SURFACE, FOR EXAMPLE: "442". See table 1012/84 which is an instructive entry

2

ICAO BASE CODE

 

 

 

 

2A

ICAO BASE CODE

 

A 4 LETTER INTERNATIONAL CODE TO DESIGNATE A PARTICULAR AIRFIELD THAT THE AIRCRAFT IN QUESTION IS LOCATED, AT TIME OF THE REPORT, FOR EXAMPLE: "LICT". See table 1125/2 which is an instructive entry

3

AIRCRAFT STATUS

 

 

 

 

3A

AIRCRAFT STATUS

 

A CODE THAT GIVES A SHORT DESCRIPTION AS TO THE MISSION CAPABILITIES OF THE AIRCRAFT ITSELF, FOR EXAMPLE: "NMCM". See table 1108/3 which contains a list of data items and associated data codes

4

AVIONICS STATUS

 

 

 

 

4A

AVIONICS STATUS

 

ENTER THE CODE THAT GIVES A SHORT DESCRIPTION AS TO THE CAPABILITIES OF THE MISSION AVIONICS OF THE AIRCRAFT IN QUESTION, FOR EXAMPLE: "FMC". See table 1108/4 which contains a list of data items and associated data codes

REASONS

 

A SHORT DESCRIPTION AS TO WHY THE AIRCRAFT IS NOT "FMC". See table 1357/19 which is an instructive entry
