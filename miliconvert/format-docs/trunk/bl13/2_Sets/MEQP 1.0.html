Concept/Explanation/Examples

1

UNIT IDENTIFICATION

 

PROVIDES THE IDENTIFICATION OF THE UNIT IN ONE OF THE FOLLOWING WAYS:

 

 

1A

SHIP NAME

 

ENTER THE UNIT NAME ON WHICH EQUIPMENT IS LOCATED, FOR EXAMPLE: "SARATOGA". See table 1022/49 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER THE FIELD DESCRIPTOR CODE "UIC" FOLLOWED BY THE UNIT IDENTIFICATION CODE, FOR EXAMPLE: "UIC:ITN1234". See table 2141 which is a composite

2

WEAPON SYSTEM

 

THE WEAPON SYSTEM DESCRIBED USING ONE OF THE FOLLOWING:

 

 

2A

MARITIME WEAPON SYSTEMS

 

ENTER THE WEAPON TYPE, FOR EXAMPLE: "MK46". See table 1407/54 which contains a list of data items and associated data codes

 

 

2B

WEAPON TYPE, GENERIC

GEN

ONLY TO BE USED IF THE WEAPON SYSTEM IS NOT AVAILABLE IN ALT A.  ENTER "GEN:" FOLLOWED BY THE NAME OF A WEAPON SYSTEM, FOR EXAMPLE "GEN:49MM". See table 1407/1 which is an instructive entry

 

 

2C

AIRCRAFT BY TYPE, MODEL OR CODE

AC

ENTER "AC:" FOLLOWED BY THE AIRCRAFT TYPE, FOR EXAMPLE: "AC:E2". See table 1015/20 which contains a list of data items and associated data codes

QUANTITY OF EQUIPMENT

 

ENTER THE QUANTITY OF EQUIPMENT HELD, FOR EXAMPLE: "1". See table 1044/179 a range [1 through 99]
