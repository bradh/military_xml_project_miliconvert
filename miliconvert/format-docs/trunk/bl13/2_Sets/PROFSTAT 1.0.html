1

ALLEGIANCE

 

 

 

 

1A

ALLEGIANCE

 

INDICATES THE COUNTRY TO WHICH THE INDIVIDUAL GIVES HIS LOYALTY, FOR EXAMPLE: "GE". See table 1026/17 which contains a list of data items and associated data codes

2

SERVICE FORMATION

 

 

 

 

2A

SERVICE FORMATION

 

ENTER THE BASIC MILITARY OR PARAMILITARY SUB-DIVISION OF THE ARMED FORCES TO WHICH AN INDIVIDUAL BELONGS, FOR EXAMPLE: "AR". See table 1107/6 which contains a list of data items and associated data codes

3

SERVICE NUMBER

 

 

 

 

3A

SERVICE NUMBER

 

ENTER THE INDIVIDUAL'S SERVICE NUMBER, FOR EXAMPLE: "22081951-M-117587". See table 1012/325 which is an instructive entry

4

NATIONAL MILITARY RANK

 

 

 

 

4A

NATIONAL MILITARY RANK

 

ENTER THE NATIVE RANK OF THE INDIVIDUAL, FOR EXAMPLE: "OBERSTLT". See table 1046/3 which contains a list of data items and associated data codes

5

RANK (NATO/US), MIL OR CIV

 

 

 

 

5A

RANK (NATO US), MIL OR CIV

 

ENTER  THE MILITARY GRADE AS COMMONLY USED IN NATO/US SYSTEMS OR A SES LEVEL RESPECTIVELY, FOR EXAMPLE: "O4". See table 1046/6 which contains a list of data items and associated data codes

6

SPECIALISATION, PERSONNEL

 

 

 

 

6A

SPECIALISATION, PERSONNEL

 

ENTER A CODE TO IDENTIFY THE PRINCIPAL SPECIALTY THAT THE INDIVIDUAL IS TRAINED TO PERFORM,FOR EXAMPLE: "EC". See table 1307/3 which contains a list of data items and associated data codes

7

PROFESSION

 

 

 

 

7A

PROFESSION

 

DESCRIBES THE CIVILIAN PROFESSION, TRADE, OR OCCUPATION OF THE INDIVIDUAL, FOR EXAMPLE: "COMMUNICATIONS ENGINEER". See table 1046/4 which is an instructive entry

8

PROMOTION DATE, YYYYMMDD

 

 

 

 

8A

PROMOTION DATE, YYYYMMDD

 

IDENTIFIES THE YEAR, MONTH, AND DAY IN WHICH THE INDIVIDUAL OBTAINED HIS CURRENT RANK OR POSITION IN THE FORMAT YYYYMMDD, FOR EXAMPLE: "19940401". See table 2053 which is a composite

9A

REMARKS
