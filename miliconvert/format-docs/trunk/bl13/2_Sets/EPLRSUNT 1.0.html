UNIT DESIGNATOR

 

 

 

 

1A

UNIT DESIGNATOR

 

ENTER THE IDENTIFICATION OF A MILITARY, PARA-MILITARY OR GOVERNMENT AGENCY UNIT, AS USED IN OFFICIAL COMMUNICATIONS WITH MILITARY ESTABLISHMENTS, FOR EXAMPLE: "AATC F-16C". See table 1028/43 which is an instructive entry

2

POC PHONE NUMBER

 

THE SADL SQUADRON POC INFORMATION DESCRIBED IN ONE OF THE FOLLOWING WAYS:

 

 

2A

NON-SECURE TELEPHONE

TEL

ENTER "TEL:" FOLLOWED BY THE PHONE NUMBER OF THE POINT OF CONTACT AT UNIT, FOR EXAMPLE: "TEL:DSN 228-4567". See table 1361/3 which is an instructive entry

 

 

2B

SECURE TELEPHONE NUMBER

SECTEL

ENTER "SECTEL:" FOLLOWED BY THE SECURE PHONE NUMBER OF THE POINT OF CONTACT AT THE UNIT, FOR EXAMPLE:"SECTEL:234-9876". See table 1361/4 which is an instructive entry

AIR TO GROUND DIVISION KEY

 

THE KEY FOR ESTABLISHING THE SADL NET PLAYERS THAT WILL BE DISPLAYED. FOR EXAMPLE: "02" See table 2809 which is a composite

FREQUENCY CHANNEL

 

 

 

 

5A

FREQUENCY CHANNEL

 

THE FREQUENCY IN WHICH THE NETWORK AND GATEWAY WILL OPERATE, FOR EXAMPLE: "37". See table 2808 which is a composite

GATEWAY RANGE SETTING

SADL POWER LEVEL IN WATTS

BRIGADE LOGICAL CHANNEL NUMBER

 

 

 

 

9A

BRIGADE LOGICAL CHANNEL NUMBER

BATTALION LOGICAL CHANNEL NUMBER

BATTALION LOGICAL CHANNEL NUMBER
