1

RADIO FREQUENCY

 

ENTER THE RADIO FREQUENCY IN ONE OF THE FOLLWING WAYS:

 

 

1A

RADIO FREQUENCY

 

THE RADIO FREQUENCY.  ENTER THE CONTEXT QUANTITY (0-99999999999, 0 TO 10 DECIMAL PLACES), AND THE UNIT OF ELECTROMAGNETIC EMISSION MEASUREMENT,  FOR EXAMPLE: "240.5MHZ". See table 2064 which is a composite

 

 

1B

FREQUENCY BAND

 

THE CODE TO PROVIDE THE CLASSIFICATION OF ELECTROMAGNETIC ENERGY BY GROUPING RANGES OF FREQUENCIES INTO DISTINCT GROUPS, FOR EXAMPLE: "A-BAND". See table 1264/1 which contains a list of data items and associated data codes

2

PULSE REPETITION FREQUENCY

 

 

 

 

2A

PULSE REPETITION FREQUENCY

 

ENTER THE PREFIX "PRF" FOLLOWED BY THE PULSE REPETITION FREQUENCY IN PULSES PER SECOND, FOR EXAMPLE: "PRF4.6". See table 2177 which is a composite

3

PULSE REPETITION INTERVAL IN MICROSECONDS

 

 

 

 

3A

PULSE REPETITION INTERVAL IN MICROSECONDS

 

ENTER THE PREFIX "PRI" FOLLOWED BY THE PULSE REPETITION INTERVAL IN MICROSECONDS, FOR EXAMPLE: "PRI56.8". See table 2176 which is a composite

4

PULSE WIDTH

 

 

 

 

4A

PULSE WIDTH

 

ENTER THE PREFIX "PW" FOLLOWED BY THE PULSE WIDTH IN MICROSECONDS, FOR EXAMPLE: "PW125". See table 2175 which is a composite

5

SCAN RATE

 

ENTER THE SCAN RATE IN ONE OF THE FOLLOWING WAYS:

 

 

5A

SCAN RATE (HZ)

 

ENTER THE THE RATE REQUIRED FOR A BEAM OF RADIO FREQUENCY ENERGY TO COMPLETE A GIVEN ELECTRONIC EMITTER SCAN PATTERN EXPRESSED IN HERTZ (HZ), FOR EXAMPLE: "3HZ". See table 2026 which is a composite

 

 

5B

SCAN RATE (SPR)

 

ENTER THE THE RATE REQUIRED FOR A BEAM OF RADIO FREQUENCY ENERGY TO COMPLETE A GIVEN ELECTRONIC EMITTER SCAN PATTERN, EXPRESSED IN SECONDS PER REVOLUTION (SPR), FOR EXAMPLE: "1.5SPR". See table 2008 which is a composite

 

 

5C

SCAN RATE (RPM)

 

ENTER THE THE RATE REQUIRED FOR A BEAM OF RADIO FREQUENCY ENERGY TO COMPLETE A GIVEN ELECTRONIC EMITTER SCAN PATTERN EXPRESSED IN REVOLUTIONS PER MINUTE (RPM), FOR EXAMPLE: "4RPM". See table 2174 which is a composite

6

SCAN TYPE

 

 

 

 

6A

SCAN TYPE

 

ENTER THE PATTERN OR SCAN TYPE OF THE EMITTER RADIATION, FOR EXAMPLE: "SPIR". See table 1105/1 which contains a list of data items and associated data codes

EMISSION INTERCEPT DURATION

 

 

 

 

7A

EMISSION INTERCEPT DURATION

 

ENTER THE TIME PERIOD DURING WHICH AN INTERCEPTED ELECTROMAGNETIC EMISSION IS RECEIVED, FOR EXAMPLE: "12H10M30S". See table 2095 which is a composite

8

8A

PRF VARIATION

 

ENTER A CODE TO PROVIDE THE VARIATION IN THE FREQUENCY OF THE PULSE REPETITION, FOR EXAMPLE: "STDY". See table 1260/1 which contains a list of data items and associated data codes
