AERIAL MISSION TYPE

 

 

 

 

1A

AERIAL MISSION TYPE

2

TARGET DESCRIPTION

 

 

 

 

2A

TARGET DESCRIPTION

 

ENTER A DESCRIPTION OF THE TARGET IN PLAIN LANGUAGE, FOR EXAMPLE: "AIRCRAFT HANGAR". See table 1363/4 which is an instructive entry

3

TARGET NOMINATING AUTHORITY

 

 

 

 

3A

TARGET NOMINATING AUTHORITY

 

ENTER THE TYPE OF TARGET NOMINATING AUTHORITY, FOR EXAMPLE: "LCC". See table 1874/7 which contains a list of data items and associated data codes

4

TARGET CATEGORY

 

 

 

 

4A

TARGET CATEGORY

 

ENTER THE TARGET CATEGORY, LETHAL OR NON LETHAL, IAW ACO DIRECTIVE 80-70, ANNEX D, FOR EXAMPLE: "NON LETHAL". See table 1309/7 which contains a list of data items and associated data codes

5

TARGET NAME

 

 

 

 

5A

TARGET NAME

 

ENTER THE NAME OF THE TARGET, FOR EXAMPLE: "DUGOUT BRAVO". See table 1022/97 which is an instructive entry

6

GEOGRAPHICAL ENTITY

 

THE CODE OF THE COUNTRY OR OTHER GEOGRAPHICAL ENTITY WHERE THE TARGET IS SITUATED.

 

 

6A

GEOGRAPHICAL ENTITY (2A)

 

ENTER THE CODE (2A) FOR THE COUNTRY OR OTHER GEOGRAPHICAL ENTITY WHERE THE TARGET IS SITUATED, FOR EXAMPLE: "BE". See table 1262/6 which contains a list of data items and associated data codes

 

 

6B

GEOGRAPHICAL ENTITY (3A)

 

ENTER THE CODE  (3A) FOR THE COUNTRY OR OTHER GEOGRAPHICAL ENTITY WHERE THE TARGET IS SITUATED, FOR EXAMPLE: "BEL". See table 1262/1 which contains a list of data items and associated data codes

 

 

6C

EXERCISE GEOGRAPHICAL ENTITY

 

ENTER THE CODE  FOR THE FICTITIOUS COUNTRY WHERE THE TARGET IS SITUATED, FOR EXAMPLE: "XW". See table 1262/4 which contains a list of data items and associated data codes

7

TARGET IDENTIFIER

 

AN UNIVOCAL TARGET IDENTIFIER, USING EITHER THE AMERICAN - BRITISH - CANADIAN - AUSTRALIAN (ABCA) TARGET NUMBERING SYSTEM, OR THE BASIC ENCYCLOPEDIA NUMBER.

 

 

7A

BASIC ENCYCLOPEDIA NUMBER

BEN

ENTER "BEN:" FOLLOWED BY THE BASIC ENCYCLOPEDIA NUMBER, FOR EXAMPLE: "BEN:1234-12345". See table 2002 which is a composite

 

 

7B

ABCA TARGET NUMBER

ABCA

ENTER "ABCA:" FOLLOWED BY THE ABCA TARGET NUMBER, FOR EXAMPLE: "ABCA:ABB1234". See table 2120 which is a composite

8

TARGET AREA DESCRIPTION

 

 

 

 

8A

TARGET AREA DESCRIPTION

 

ENTER A PLAIN LANGUAGE DESCRIPTION OF THE AREA WHERE THE TARGET IS SITUATED, FOR EXAMPLE: "LASCAR PREFAB". See table 1363/6 which is an instructive entry

9

TARGET POSITION

 

THE GEOGRAPHIC LOCATION OF THE LAST OBSERVED TARGET POSITION.

 

 

9A

TARGET POSITION, LAT/LONG, CENTISECONDS

LATCS

ENTER "LATCS:" FOLLOWED BY THE LAST OBSERVED TARGET POSITION IN LAT/LONG, CENTISECONDS, FOR EXAMPLE: "LATCS:50451030N005322015E". See table 2488 which is a composite

 

 

9B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

9C

TARGET LOCATION, GEOREF, MINUTE

GEOREFM

ENTER "GEOREFM:" FOLLOWED BY THE LAST OBSERVED TARGET POSITION IN GEOREF COORDINATES APPROXIMATED TO THE MINUTE, FOR EXAMPLE: "GEOREFM:AKMJ3548". See table 2017 which is a composite

 

 

9D

TARGET POSITION, TRUE BEARING AND DISTANCE IN NM

TBR

ENTER "TBR:" FOLLOWED BY THE LAST OBSERVED TARGET VESSEL POSITION AS TRUE BEARING AND RANGE IN NAUTIC MILES FROM A REFERENCE POINT, FOR EXAMPLE: "TBR:060-POINT BRAVO-18". See table 2016 which is a composite

 

 

9E

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

10

DAY-TIME OF POSITION

 

 

 

 

10A

DTG OF OBSERVATION, ZULU

 

ENTER THE DTG ZULU OF LAST OBSERVED TARGET POSITION, FOR EXAMPLE: "251330ZDEC2004". See table 2323 which is a composite

11

TARGET COURSE

 

THE TARGET COURSE EITHER IN DEGREES, OR IN DEGREES WITH SPECIFICATION IF TRUE, GRID MAGNETIC OR RELATIVE, OR REFERRING TO A COMPASS POINT.

 

 

11A

TARGET COURSE IN DEGREES

COURSE

ENTER "COURSE:" FOLLOWED BY THE TARGET COURSE, ANGLE IN DEGREES, TRUE, GRID MAGNETIC OR RELATIVE, FOR EXAMPLE: "COURSE:210T". See table 2065 which is a composite

 

 

11B

DIRECTION OF MOVEMENT (COMPASS POINT)

DIR

ENTER "DIR:" FOLLOWED BY THE DIRECTION OF MOVEMENT (COMPASS POINT), FOR EXAMPLE: "DIR:SSW". See table 1161/1 which contains a list of data items and associated data codes

 

 

11C

DIRECTION OF MOVEMENT IN DEGREES

DIRMOV

ENTER "DIRMOV:" FOLLOWED BY THE DIRECTION OF MOVEMENT IN DEGREES, FOR EXAMPLE: "DIRMOV:342". See table 1043/9 a range [001 through 360]

12

TARGET SPEED

 

THE TARGET SPEED EITHER IN MILES PER HOUR/ KNOTS/ KILOMETRES PER HOUR/ METRES PER SECOND, OR AS A GENERIC DEFINITION OF SLOW, MEDIUM, FAST OR ZERO.

 

 

12A

TARGET SPEED

SPEED

ENTER "SPEED:" FOLLOWED BY THE TARGET SPEED, FOR EXAMPLE: "SPEED:80KMH". See table 2063 which is a composite

 

 

12B

APPROXIMATE SPEED

APPXSPD

ENTER "APPXSPD:" FOLLOWED BY THE APPROXIMATE SPEED, FOR EXAMPLE: "APPXSPD:ZERO". See table 1113/5 which contains a list of data items and associated data codes
