1

SUPPORTING UNIT NAME

 

 

 

 

1A

SUPPORTING UNIT NAME

 

ENTER THE NAME OF THE UNIT ASSIGNED TO NAVAL GUNFIRE SUPPORT DUTIES, FOR EXAMPLE: "A BURKE". See table 1022/196 which is an instructive entry

2

FIRE SUPPORT AREA OR STATION DESIGNATOR

 

PROVIDES THE DESIGNATOR OF THE FIRE SUPPORT AREA OR STATION TO WHICH THE SPECIFIED UNIT IS ASSIGNED.

 

 

2A

FIRE SUPPORT AREA DESIGNATOR

 

ENTER THE FIRE SUPPORT AREA DESIGNATOR, FOR EXAMPLE: "FSA IX". See table 1012/210 which contains a list of data items and associated data codes

 

 

2B

FIRE SUPPORT STATION DESIGNATOR

 

ENTER THE FIRE SUPPORT STATION DESIGNATOR, FOR EXAMPLE: "FSS8". See table 1012/209 which is an alphanumeric range

3

NAVAL GUNFIRE SUPPORT MISSION TYPE

 

 

 

 

3A

NAVAL GUNFIRE SUPPORT MISSION TYPE

 

ENTER THE TYPE OF NAVAL GUNFIRE SUPPORT MISSION, FOR EXAMPLE: "COVER". See table 1285/43 which is an instructive entry

4

ZONE OF FIRE DESIGNATOR

 

 

 

 

4A

ZONE OF FIRE DESIGNATOR

 

ENTER THE DESIGNATOR OF THE ZONE OF FIRE TO WHICH THE UNIT IS ASSIGNED, FOR EXAMPLE: "ZF7". See table 1012/211 which is an alphanumeric range

5

SUPPORTED UNIT NAME

 

 

 

 

5A

SUPPORTED UNIT NAME

 

ENTER THE NAME OF THE UNIT BEING SUPPORTED BY NAVAL GUNFIRE, FOR EXAMPLE: "FIRST MARDIV". See table 1022/332 which is an instructive entry

6

DTG OF START
