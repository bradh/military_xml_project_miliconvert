1

PLATFORM TYPE

 

SPECIFIES THE TYPE OF PLATFORM WHICH DETECTED THE MINE(S).

 

 

1A

SUBMARINE TYPE

SUB

ENTER FIELD DESCRIPTOR "SUB:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE SUBMARINE TYPE, FOR EXAMPLE: "SUB:SSK". See table 1166/2 which contains a list of data items and associated data codes

 

 

1B

SURFACE VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE SURFACE SHIP TYPE, FOR EXAMPLE: "MC". See table 1166/7 which contains a list of data items and associated data codes

 

 

1C

AIRCRAFT CATEGORY

AIR

ENTER FIELD DESCRIPTOR "AIR:" FOLLOWED BY THE AIRCRAFT CATEGORY, FOR EXAMPLE: "HELO". See table 1549/1 which contains a list of data items and associated data codes

2

PLATFORM IDENTIFIER

 

SPECIFIES THE IDENTITY OF THE PLATFORM DETECTING THE MINE(S).

 

 

2A

UNIT NAME

 

ENTER THE UNIT NAME, FOR EXAMPLE: "ALLSUND". See table 1022/48 which is an instructive entry

 

 

2B

CALL SIGN

CALLSIGN

ENTER FIELD DESCRIPTOR "CALLSIGN:" FOLLOWED BY THE UNIT CALL SIGN, FOR EXAMPLE: "CALLSIGN:AB123". See table 1129/1 which is an instructive entry

3

GEODETIC DATUM

 

 

 

 

3A

GEODETIC DATUM

 

ENTER THE GEODETIC DATUM USED FOR POSITION REFERENCES, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

4

POSITION OF MINE(S)

 

SPECIFIES THE GEOGRAPHIC LOCATION OF THE DETECTED MINE(S).

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE GEOREF LOCATION, CENTIMINUTE, FOR EXAMPLE: "GEO:BBAA33443344". See table 2018 which is a composite

 

 

4H

PLACE NAME

NAME

ENTER FIELD DESCRIPTOR "NAME:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "NAME:ZULU". See table 1022/170 which is an instructive entry

 

 

4I

TRUE BEARING AND DISTANCE IN NM FROM REFERENCE POINT

TBR

ENTER THE POSITION AS A TRUE BEARING AND DISTANCE IN NAUTICAL MILES FROM A NAMED REFERENCE POINT, FOR EXAMPLE: "TBR:191-LIZARD POINT LIGHT-47.4". See table 2016 which is a composite

 

 

4J

Q-ROUTE POSITION

QPOS

ENTER FIELD DESCRIPTOR "QPOS:" FOLLOWED BY THE Q-ROUTE POSITION, FOR EXAMPLE: "QPOS:171Z87.4MS645". See table 2203 which is a composite
