TRP DESIGNATOR

 

THE GEOGRAPHICAL POSITION OF THE TRP.

 

 

1A

TIMING REFERENCE POINT (TRP) DESIGNATOR

 

ENTER THE NAME OR IDENTIFICATION OF A POINT USED AS A TIMING REFERENCE, FOR EXAMPLE: "ZA". See table 1796/2 which is an instructive entry

2

TRP LOCATION

 

THE GEOGRAPHIC LOCATION OF THE TIMING REFERENCE POINT (TRP).

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME OR DESIGNATION USED TO IDENTIFY A TIMING REFERENCE POINT (TRP), FOR EXAMPLE: "LOC:CHARLIE". See table 1022/4 which is an instructive entry

ALTITUDE

 

 

 

 

3A

ALTITUDE, HUNDREDS OF FEET

 

ENTER THE ALTITUDE OF THE TIMING REFERENCE POINT, FOR EXAMPLE: "310". See table 1255/14 a range [000 through 999]

TRP DESCRIPTION

 

 

 

 

4A

TIME REFERENCE POINT (TRP) DESCRIPTION

 

ENTER THE DISTINGUISHING FEATURES AND CHARACTERISTICS OF THE TIMING REFERENCE POINT (TRP), FOR EXAMPLE: "CTR BRIDGE". See table 1363/47 which is an instructive entry

5

PRIMARY COMMUNICATION PARAMETERS

 

THE PRIMARY COMMUNICATION PARAMETERS TO BE USED BY THE AIRCRAFT AT THE TRP.

 

 

5A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY TO BE USED, FOR EXAMPLE: "PFREQ:244.0". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

5B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR TO BE USED, FOR EXAMPLE: "PDESIG:TAD001". See table 1396/2 which is an instructive entry

 

 

5C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQKM:2B-4A-03042001-74A56-356.7MHZ". See table 2534 which is a composite

 

 

5D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQK:3B-4X-04112001". See table 2535 which is a composite

6

SECONDARY COMMUNICATION PARAMETERS

 

PROVIDES THE SECONDARY COMMUNICATION PARAMETERS TO BE USED BY THE AIRCRAFT AT THE TRP, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.  CAN ONLY BE USED IN CONJUNCTION WITH PRIMARY COMMUNICATION PARAMETERS.

 

 

6A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE SECONDARY FREQUENCY TO BE USED BY AN AIRCRAFT, AGENCY, OR NET, FOR EXAMPLE: "SFREQ:250.25". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

6B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY FOR CONTACTING AN AIRCRAFT OR AGENCY, FOR EXAMPLE: "SDESIG:TAD002". See table 1396/3 which is an instructive entry

 

 

6C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

SHVQKM

ENTER "SHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQKM:2B-6D-03042001-74A56-356.7HZ". See table 2534 which is a composite

 

 

6D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

SHVQK

ENTER "SHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQK:6B-4X-04112001". See table 2535 which is a composite
