Concept/Explanation/Examples

1

TIME TASKED ON TARGET

 

EITHER THE DAY-TIME AND MONTH OR DAY-TIME TO THE SECOND TASKED ON TARGET.

 

 

1A

DAY-TIME AND MONTH TASKED ON TARGET

TOT

ENTER "TOT:" FOLLOWED BY THE DAY-TIME WHEN THE REQUESTED MISSION IS SUPPOSED TO BE ON TARGET, FOR EXAMPLE: "TOT:231550ZAUG". See table 2030 which is a composite

 

 

1B

DAY-TIME ON TARGET TO THE SECOND, ZULU

TOTS

ENTER "TOTS:" FOLLOWED BY THE DAY, HOUR, MINUTE, SECOND, TIME ZONE, WHEN THE REQUESTED MISSION IS SUPPOSED TO BE ON TARGET, FOR EXAMPLE:"TOTS:23155040Z". See table 2483 which is a composite

2

NOT EARLIER THAN DAY-TIME AND MONTH

 

REQUIRED ONLY IF TIME TASKED ON TARGET HAS NOT BEEN SPECIFIED WITHIN THIS SET.

 

 

2A

NOT EARLIER THAN DAY-TIME AND MONTH

 

ENTER THE NOT EARLIER THAN DAY-TIME AND MONTH ON TARGET, FOR EXAMPLE: "141320ZFEB". See table 2030 which is a composite

3

NOT LATER THAN DTG

 

THE LATEST DATE-TIME WHEN THE MISSION IS SUPPOSED TO BE ON TARGET.  REQUIRED ONLY IF TIME TASKED ON TARGET HAS NOT BEEN SPECIFIED WITHIN THIS SET.

 

 

3A

NO LATER THAN DTG

NLT

ENTER "NLT:" FOLLOWED BY THE LATEST DAY-TIME WHEN THE MISSION IS SUPPOSED TO BE ON TARGET, FOR EXAMPLE: "NLT:261556ZMAY2006". See table 2033 which is a composite

 

 

3B

DAY-TIME AND MONTH OFF TARGET

DTMOFF

ENTER "DTMOFF:" FOLLOWED BY THE DAY-TIME AND MONTH WHEN THE MISSION IS DUE TO BE OFF TARGET, FOR EXAMPLE: "DTMOFF:251559ZMAY". See table 2030 which is a composite

 

 

3C

NOT LATER THAN DAY-TIME

NLDT

ENTER "NLDT:" FOLLOWED BY THE LATEST DAY-TIME ON TARGET, FOR EXAMPLE "NLDT:131408Z". See table 2000 which is a composite

NON-SHIP TARGET TYPE

 

 

 

 

4A

NON-SHIP TARGET TYPE

 

ENTER THE CODE MATCHING THE NON-SHIP TARGET TYPE OF THE MISSION, FOR EXAMPLE: "ADAH". See table 1650/6 which contains a list of data items and associated data codes

TARGET IDENTIFIER

 

 

 

 

5A

TARGET IDENTIFIER, ABBREVIATED

GEODETIC DATUM

 

 

 

 

6A

GEODETIC DATUM

 

ENTER THE GEODETIC DATUM, IF THE ONE REQUIRED IS DIFFERENT FROM THE ONE GENERALLY USED IN THE MTF, FOR EXAMPLE: "BOO". See table 1195/1 which contains a list of data items and associated data codes

7

TARGET/FACILITY NAME

 

 

 

 

7A

TARGET FACILITY NAME

 

ENTER THE NAME OF THE TARGET FACILITY, FOR EXAMPLE "FORT SREDNIK". See table 1022/234 which is an instructive entry

8

TARGET POSITION

 

THE GEOGRAPHICAL LOCATION OF THE TARGET.

 

 

8A

GEOGRAPHIC LOCATION, LAT/LONG, CENTISECONDS

LATCS

ENTER "LATCS:", FOLLOWED BY THE LOCATION COORDINATES IN LAT/LONG CENTISECONDS, FOR EXAMPLE: "LATCS:35102530N079012545W". See table 2488 which is a composite

 

 

8B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

8C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

9

TARGET STRENGTH

 

 

 

 

9A

TARGET STRENGTH

10

COMPONENT TARGET IDENTIFIER

 

THE IDENTIFIER ASSIGNED TO THE TARGET BY THE RELEVANT COMPONENT, OR THE NUMBERING SCHEME USED TO IDENTIFY A TARGET ON A VARIABLE MESSAGE FORMAT (VMF) INTERFACE.

 

 

10A

COMPONENT TARGET IDENTIFIER

CTGT

ENTER "CTGT:" FOLLOWED BY THE APPROPRIATE COMPONENT TARGET IDENTIFIER, FOR EXAMPLE: "CTGT:F1C4N3R4" See table 1416/5 which is an instructive entry

 

 

10B

TARGET ENTITY ID REFERENCE NUMBER

EIRN
