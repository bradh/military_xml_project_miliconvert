Designator

Field Desc

Concept/Explanation/Examples

1

DAY-TIME START

 

 

 

 

1A

DAY-TIME START

 

ENTER THE DAY-TIME OF THE START OF THE ESCORT MISSION, FOR EXAMPLE: "281000Z". See table 2000 which is a composite

2

NUMBER ESCORTED

 

 

 

 

2A

NUMBER ESCORTED

 

ENTER A COUNT OF THE AIRCRAFT OR VEHICLES TO BE ESCORTED, FOR EXAMPLE: "03". See table 1044/330 a range [00 through 99]

3

ESCORTED AIRCRAFT OR VEHICLE

 

A DESCRIPTION OF THE AIRCRAFT OR VEHICLE TO BE ESCORTED, USING EITHER ITS TYPE AND MODEL NUMBER, OR ITS NAME, OR THE IDENTIFIER OF THE GENERIC CATEGORY TO WHICH IT BELONGS.

 

 

3A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE AND MODEL NUMBER OF AN AIRCRAFT BEING ESCORTED, FOR EXAMPLE: "ACTYP:C130E". See table 1015/20 which contains a list of data items and associated data codes

 

 

3B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF AN AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST, FOR EXAMPLE: "OTHAC:AX10". See table 1015/18 which is an instructive entry

 

 

3C

VEHICLE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME AND/OR TYPE OF THE VEHICLE BEING ESCORTED, FOR EXAMPLE: "NAME:M1A2". See table 1022/224 which is an instructive entry

 

 

3D

AIRCRAFT CATEGORY, ABBREVIATED

ACCAT

ENTER "ACCAT:" FOLLOWED BY AN IDENTIFIER OF THE GENERIC CATEGORY OF AIRCRAFT, FOR EXAMPLE: "ACCAT:CRGTRS". See table 1549/2 which contains a list of data items and associated data codes

4

PRIMARY AERIAL MISSION TYPE

 

 

 

 

4A

PRIMARY AERIAL MISSION TYPE

 

ENTER THE CODE FOR THE TYPE OF AIR MISSION TASKED, FOR EXAMPLE: "CAS". See table 1285/31 which contains a list of data items and associated data codes

5

START POSITION

 

THE GEOGRAPHIC LOCATION WHERE THE ESCORT PARTICIPANTS WILL JOIN UP.

 

 

5A

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION, FOR EXAMPLE: "ICAO:KILA". See table 1125/2 which is an instructive entry

 

 

5B

PLACE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME ASSIGNED TO THE LOCATION, FOR EXAMPLE: "NAME:GARGANTA DEL DIABLO". See table 1022/170 which is an instructive entry

 

 

5C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

6

DAY-TIME STOP

 

 

 

 

6A

DAY-TIME STOP

 

ENTER THE DAY-TIME OF THE END OF THE ESCORT MISSION, FOR EXAMPLE: "281200Z". See table 2000 which is a composite

7

STOP POSITION

 

THE GEOGRAPHIC LOCATION WHERE ESCORT MISSION WILL STOP.

 

 

7A

ICAO BASE CODE

ICAO

7B

PLACE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME ASSIGNED TO THE LOCATION, FOR EXAMPLE: "NAME:GARGANTA DEL DIABLO". See table 1022/170 which is an instructive entry

 

 

7C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

7D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

8

ESCORTED AIRCRAFT CALL SIGN

 

 

 

 

8A

ESCORTED CALL SIGN

 

ENTER THE CALL SIGN OF THE ESCORTED AIRCRAFT OR ESCORTED VEHICLES, FOR EXAMPLE: "HERKY 10". See table 1129/38 which is an instructive entry

9

MISSION NUMBER SUPPORTED

 

 

 

 

9A

MISSION NUMBER SUPPORTED

 

ENTER THE MISSION NUMBER SUPPORTED, FOR EXAMPLE "FJ892T". See table 1012/279 which is an instructive entry

10

PRIMARY COMMUNICATION PARAMETERS

 

THE PRIMARY COMMUNICATION PARAMETERS ASSIGNED TO THE ESCORTED AIRCRAFT OR VEHICLES, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.

 

 

10A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY TO BE USED BY THE ESCORTED AIRCRAFT, FOR EXAMPLE: "PFREQ:290.8". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

10B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR FOR CONTACTING THE ESCORTED AIRCRAFT, FOR EXAMPLE: "PDESIG:TAD001". See table 1396/2 which is an instructive entry

 

 

10C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

10D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQK:AB12-CD34-12082000". See table 2535 which is a composite

11

SECONDARY COMMUNICATION PARAMETERS

 

THE SECONDARY COMMUNICATION PARAMETERS ASSIGNED TO THE ESCORTED AIRCRAFT OR VEHICLES, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.  CAN ONLY BE USED IF PRIMARY COMMUNICATION PARAMETERS HAVE BEEN SPECIFIED IN THIS SET.

 

 

11A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE SECONDARY FREQUENCY TO BE USED BY THE ESCORTED AIRCRAFT, FOR EXAMPLE: "SFREQ:341.2". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

11B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY DESIGNATOR FOR CONTACTING THE ESCORTED AIRCRAFT, FOR EXAMPLE: "SDESIG:TAD002". See table 1396/3 which is an instructive entry

 

 

11C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

SHVQKM

ENTER "SHVQKM:"FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

11D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

SHVQK

ENTER "SHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQK:AB12-CD34-12082000". See table 2535 which is a composite
