INTERCEPTED AIRCRAFT TYPE

 

THE AIRCRAFT TYPE AND MODEL OF THE INTERCEPTED AIRCRAFT AS A CODED OR LITERAL VALUE.

 

 

7A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE CODE FOR TYPE AND MODEL OF THE AIRCRAFT, FOR EXAMPLE: "ACTYP:FLKA". See table 1015/20 which contains a list of data items and associated data codes

 

 

7B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC
