PARENT UNIT IDENTITY

 

ENTER THE IDENTITY OF THE AIRCRAFT PARENT UNIT USING ONE OF THE FOLLOWING:

 

 

1A

UNIT

 

ENTER THE SHIP NAME, FOR EXAMPLE: "SARATOGA". See table 1022/57 which is an instructive entry

 

 

1B

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LETTER CODE, FOR EXAMPLE: "ICAO:ABCD". See table 1125/2 which is an instructive entry

 

 

1C

BASE NAME

BASE

ENTER "BASE:" FOLLOWED BY THE BASE NAME, FOR EXAMPLE: "BASE:CULDROSE". See table 1022/104 which is an instructive entry

2

AIRCRAFT CODE

 

 

 

 

2A

AIRCRAFT CODE

 

ENTER THE AIRCRAFT TYPE ON ALERT, FOR EXAMPLE: "F14", REPEATING THIS AND SUBSEQUENT FIELDS AS A GROUP AS NECESSARY. See table 1015/2 which contains a list of data items and associated data codes

3

NUMBER OF AIRCRAFT, 1-99

 

 

 

 

3A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT OF THE SPECIFIED TYPE ON ALERT, FOR EXAMPLE: "2". See table 1044/322 a range [1 through 99]

4

ALERT STATUS

 

 

 

 

4A

ALERT STATUS

 

ENTER THE AIRCRAFT ALERT STATUS IN MINUTES FOLLOWED BY "MIN" OR IN HOURS FOLLOWED BY "HR", FOR EXAMPLE: "15MIN". See table 2161 which is a composite

5

DTG OF START

 

 

 

 

5A

DTG OF START

 

ENTER THE DTG WHEN THE ALERT STATE FOR THE SPECIFIED AIRCRAFT IS TO BE ASSUMED, FOR EXAMPLE "091800ZJUN2004". See table 2033 which is a composite

6

END TIME

 

SPECIFIES THE TIME WHEN THE ALERT STATE FOR THE SPECIFIED AIRCRAFT WILL TERMINATE.

 

 

6A

DTG OF STOP

 

ENTER THE DTG OF ALERT STATE TERMINATION, FOR EXAMPLE "120930ZJUN2004". See table 2033 which is a composite

 

 

6B

TIME, NON-DISCRETE INDICATOR

 

ENTER A NON-DISCRETE TIME INDICATOR FOR THE ALERT STATE TERMINATION, FOR EXAMPLE: "UFN". See table 1084/3 which contains a list of data items and associated data codes

7

WEAPON LOAD

 

DESCRIBES THE WEAPON LOAD OF THE SPECIFIED AIRCRAFT.

 

 

7A

AIRCRAFT WEAPON LOAD

 

ENTER THE NUMBER AND TYPE OF WEAPONS OR "NA" IF NOT APPLICABLE, FOR EXAMPLE: "4 AIM-7R PLUS 2 AIM-9L". See table 1514/1 which is an instructive entry

 

 

7B

IDENTIFIER, NON-DISCRETE INDICATOR

ID

ENTER "ID:" FOLLOWED BY A NON-DISCRETE INDICATOR OF WEAPON INFORMATION, FOR EXAMPLE: "ID:AS SIGNALED". See table 1084/4 which contains a list of data items and associated data codes

8

CALL SIGN

 

 

 

 

8A

CALL SIGN

 

ENTER THE CALL SIGN, FOR EXAMPLE: "1AC". See table 1129/1 which is an instructive entry
