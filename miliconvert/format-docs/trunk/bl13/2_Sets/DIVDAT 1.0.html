1

MINE REFERENCE NUMBER

 

ENTER MINE REFERENCE NUMBER (MRN) USING ONE OF THE FOLLOWING ALTERNATIVES:

 

 

1A

MINE REFERENCE NUMBER

MRN

ENTER "MRN:" FOLLOWED BY THE MINE REFERENCE NUMBER [LITERAL], FOR EXAMPLE: "MRN:JE01". See table 1012/186 which is an instructive entry

 

 

1B

CONTACT PLOT DESIGNATOR

 

ENTER CONTACT PLOT DESIGNATOR [LITERAL], FOR EXAMPLE: "5123B". See table 1012/187 which is an instructive entry

2

ARM THRUST

 

IF USED, ENTER ARM THRUST USING ONE OF THE FOLLOWING ALTERNATIVES:

 

 

2A

SINGLE ARM THRUST RESULT CODE

 

ENTER THE SINGLE ARM THRUST RESULT CODE, FOR EXAMPLE: "A". See table 1184/2 which contains a list of data items and associated data codes

 

 

2B

DEPTH OF PENETRATION OF ARM THRUSTS IN CENTIMETRES

 

ENTER THE DEPTH OF PENETRATION OF ARM THRUSTS IN CENTIMETERS [0-99], FOR EXAMPLE: "22". See table 1247/10 a range [0 through 99]
