Concept/Explanation/Examples

1

ANCHORAGE IDENTIFIER

 

 

 

 

1A

ANCHORAGE IDENTIFIER

 

ENTER THE IDENTIFIER OF AN ANCHORAGE, FOR EXAMPLE: "BRAVO". See table 1012/696 which is an instructive entry

2

ANCHORAGE LOCATION

 

THE GEOGRAPHIC LOCATION OF THE ANCHORAGE.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

3

GEODETIC DATUM

 

 

 

 

3A

GEODETIC DATUM

 

ENTER THE GEODETIC DATUM UPON WHICH THE ANCHORAGE LOCATION IS BASED IF THIS IS DIFFERENT TO THE MAIN MESSAGE GEODETIC DATUM, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

4

ASSIGNED SHIP NAME OR CALL SIGN

 

SPECIFIES THE NAME OR CALL SIGN OF THE SHIP ASSIGNED TO THE ANCHORAGE.

 

 

4A

SHIP NAME

 

ENTER THE NAME OF THE SHIP ASSIGNED TO THE ANCHORAGE, FOR EXAMPLE: "ARK ROYAL". See table 1022/49 which is an instructive entry

 

 

4B

CALL SIGN

CALLSIGN

ENTER "CALLSIGN:" FOLLOWED BY THE CALL SIGN OF THE SHIP ASSIGNED TO THE ANCHORAGE, FOR EXAMPLE: "CALLSIGN:W1C". See table 1129/1 which is an instructive entry

5

DTG OF START

 

 

 

 

5A

DTG OF START

 

ENTER THE DTG AT WHICH THE SHIP IS DIRECTED TO ANCHOR, FOR EXAMPLE: "221100ZMAR2005". See table 2033 which is a composite

6

DTG OF STOP

 

 

 

 

6A

DTG OF STOP

 

ENTER THE DTG AT WHICH THE SHIP IS EXPECTED TO WEIGH ANCHOR, FOR EXAMPLE: "230800ZMAR2005". See table 2033 which is a composite
