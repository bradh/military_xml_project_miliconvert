No

Designator

Field Desc

Concept/Explanation/Examples

1

TIME OF POSITION

 

PROVIDES THE TIME THE FRIENDLY GROUND FORCES ARE AT THE POSITION BEING REPORTED USING EITHER A RELATIVE TIME INDICATION OR A DAY-TIME INDICATION.

 

 

1A

RELATIVE TIME AT POSITION

RTIME

ENTER "RTIME:" FOLLOWED BY THE TIME THE FRIENDLY GROUND FORCES WERE KNOWN TO BE AT THE POSITION, FOR EXAMPLE: "RTIME:LP0345". See table 2482 which is a composite

 

 

1B

DAY-TIME OF POSITION

DTIME

ENTER "DTIME:" FOLLOWED BY THE DAY-TIME OF THE POSITION, FOR EXAMPLE: "DTIME:141325Z". See table 2000 which is a composite

2

FRIENDLY UNIT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE FRIENDLY GROUND FORCES.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

BEARING AND DISTANCE FROM REFERENCE POINT

TBR

ENTER "TBR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "TBR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

2C

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

3

GEODETIC DATUM

 

 

 

 

3A

GEODETIC DATUM

 

IF THE DEFAULT GEODETIC DATUM SPECIFIED IN THE BEGINNING OF THE MESSAGE DOES NOT APPLY TO THE LOCATION OF THE GROUND FRIENDLY FORCES, ENTER THE CORRECT GEODATUM, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

4

UNIT IDENTIFIER

 

PROVIDES THE IDENTIFICATION OF THE FRIENDLY GROUND FORCES UNIT USING EITHER ITS OFFICIAL NAME OR ITS UNIT IDENTIFICATION CODE (UIC).

 

 

4A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT NAME, FOR EXAMPLE: "UNIT:2ND DIVISION". See table 1028/43 which is an instructive entry

 

 

4B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UNIT IDENTIFICATION CODE,  FOR EXAMPLE: "UIC:GEA1234". See table 2141 which is a composite

5

FRIENDLY FORCE COURSE AND SPEED

 

 

 

 

5A

FRIENDLY FORCE COURSE (TRUE) AND SPEED (KNOTS)

 

ENTER THE COURSE AND SPEED OF THE MOVEMENT OF THE FRIENDLY FORCES, FOR EXAMPLE: "120T10KTS". See table 2037 which is a composite

6

MARKING METHOD

 

 

 

 

6A

METHOD OF MARKING FRIENDLY POSITIONS

 

ENTER THE METHOD OF MARKING THE FRIENDLY FORCES, FOR EXAMPLE: "PANELS". See table 1613/2 which contains a list of data items and associated data codes

7

MARKING COLOUR

 

CAN ONLY BE USED IN CONJUNCTION WITH MARKING METHOD SPECIFIED IN THIS SET.

 

 

7A

COLOUR

 

ENTER THE COLOUR OF THE MARKING METHOD, FOR EXAMPLE: "RED". See table 1138/1 which contains a list of data items and associated data codes
