Concept/Explanation/Examples

1

PRIORITY IDENTIFIER

 

 

 

 

1A

PRIORITY IDENTIFIER

 

ENTER THE PRIORITY FOR DETECTION OF AN EMITTER AS AN INTEGER 01-99 WHERE THE GREATEST PRIORITY IS "01", FOR EXAMPLE:"03". See table 1076/20 a range [01 through 99]

2

EMITTER CATEGORY

 

 

 

 

2A

EMITTER CATEGORY

 

IDENTIFY THE EMITTER ASSESSED AS EITHER THE THREAT, TARGET,  CONTACT OF INTEREST(COI) OR CRITICAL CONTACT OF INTEREST(CCOI)  FOR EXAMPLE: "TARGET". See table 1121/5 which contains a list of data items and associated data codes

3

EMITTER NATO DESIGNATION, NICK NAME OR SPOT NUMBER

 

THE EMITTER NATO DESIGNATION, NICKNAME OR SPOT NUMBER:

 

 

3A

EMITTER DESIGNATION

DESIG

ENTER "DESIG:" FOLLOWED BY THE EMITTER DESIGNATION, FOR EXAMPLE: "DESIG:M623S". See table 1121/1 which is an instructive entry

 

 

3B

EMITTER NATO NICKNAME

NICK

ENTER "NICK:" FOLLOWED BY THE NATO NICKNAME FOR THE EMITTER, FOR EXAMPLE: "NICK:BELT BUCKLE". See table 1121/3 which is an instructive entry

 

 

3C

SPOT NUMBER

SPOT

ENTER "SPOT:" FOLLOWED BY THE 5 DIGIT SPOT NUMBER ASSIGNED TO THE THREAT/TARGET EMITTER, FOR EXAMPLE: "SPOT:06217". See table 1012/125 a range [00000 through 99999]

4

THE NATION TO WHICH THE DESIGNATED EMITTER BELONGS.

 

 

 

 

4A

COUNTRY OF EMITTER

 

ENTER THE 2-LETTER COUNTRY CODE FOR THE NATION TO WHICH THE EMITTER BELONGS, FOR EXAMPLE: "UK". See table 1026/34 which contains a list of data items and associated data codes
