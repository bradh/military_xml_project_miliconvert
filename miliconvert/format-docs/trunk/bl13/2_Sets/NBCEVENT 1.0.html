NBC  REPORT

 

PROVIDES THE TYPE OF NUCLEAR, BIOLOGICAL, CHEMICAL, ROTA EVENT REPORT, OR WEATHER REPORT, IN ONE OF THE FOLLOWING WAYS:

 

 

1A

TYPE OF NBC REPORT

 

ENTER THE TYPE OF NBC REPORT OR ROTA EVENT REPORT, FOR EXAMPLE: "CHEM". See table 1018/8 which contains a list of data items and associated data codes

 

 

1B

TYPE OF NBC WEATHER REPORT

WEA

ENTER "WEA:" FOLLOWED BY THE TYPE OF NBC WEATHER REPORT, FOR EXAMPLE: "WEA:BWF". See table 1018/3 which contains a list of data items and associated data codes

VALIDATION CODE

VALIDATION CODE

 

GIVES THE CODE ALLOWING THE RECIPIENT TO KNOW THAT THE MESSAGE IS TRANSMITTED BY AN AUTHORIZED USER, FOR EXAMPLE: "NOIR 12". See table 1742/1 which is an instructive entry
