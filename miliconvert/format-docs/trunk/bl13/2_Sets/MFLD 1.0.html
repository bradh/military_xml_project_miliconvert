1

MINEFIELD IDENTIFICATION

 

ENTER IDENTIFICATION OF MINEFIELD IN ONE OF THE FOLLOWING WAYS:

 

 

1A

IDENTIFICATION OF MINEFIELD

IDCODE

ENTER "IDCODE:" FOLLOWED BY THE IDENTIFICATION OF MINEFIELD WHICH CONSISTS OF NATIONALITY, MARITIME COMMAND CODE, MINEFIELD SEQUENTIAL NUMBER [00-99], MINEFIELD SEGMENT IDENTIFIER [A-Z], AND MINEFIELD PRIORITY, FOR EXAMPLE: "IDCODE:UR120015D2". See table 2215 which is a composite

 

 

1B

MINEFIELD IDENTIFIER

 

ENTER THE NAME OF MINEFIELD [LITERAL], FOR EXAMPLE: "LAGUNE GREEN". See table 1261/9 which is an instructive entry

2

TYPE OF MINE

 

ENTER TYPE OF MINE IN ONE OF THE FOLLOWING WAYS:

 

 

2A

CLASSIFICATION OF MINE TYPE

 

ENTER THE CLASSIFICATION OF MINE TYPE, FOR EXAMPLE: "GROUND". See table 1210/2 which contains a list of data items and associated data codes

 

 

2B

MINE MARK MOD

MKMOD

ENTER "MKMOD:" FOLLOWED BY THE MINE MAR/MINE MOD [LITERAL], FOR EXAMPLE: "MKMOD:ABC1824". See table 1022/193 which is an instructive entry

3

NUMBER OF MINES TO BE LAID

 

 

 

 

3A

NUMBER OF MINES TO BE LAID

 

ENTER NUMBER [1-999] OF MINES TO BE LAID, FOR EXAMPLE: "62". See table 1044/60 a range [0 through 999]

4

GEODETIC DATUM

 

 

 

 

4A

GEODETIC DATUM

 

ENTER GEODETIC DATUM, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

5

MINE LINE POSITION

 

PROVIDES THE MINE LINE POSITION INFORMATION.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

5C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

5D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

5E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

5F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

5G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE GEOREF LOCATION, CENTIMINUTE, FOR EXAMPLE: "GEO:BBAA33443344". See table 2018 which is a composite

 

 

5H

MINE LINE POSITION, PLACE NAME

PLACE

ENTER "PLACE:" FOLLOWED BY THE PLACE NAME [LITERAL], FOR EXAMPLE: "PLACE:ZULU". See table 1022/203 which is an instructive entry

 

 

5I

MINE LINE, TRUE BEARING AND DISTANCE FROM REF POINT

BAR

ENTER THE MINE LINE TRUE BEARING AND DISTANCE FROM REF POINT, FOR EXAMPLE: "BAR:191.1T-AAA-87.4NM". See table 2047 which is a composite

6

MINE LINE ORIENTATION AND SPACING

 

 

 

 

6A

MINE LINE ORIENTATION AND SPACING

 

ENTER MINE LINE ORIENTATION AND SPACING GIVEN AS ORIENTATION [000-359], PLUS "T" (TRUE), HYPHEN, SPACING [50-999], PLUS "M" (METRES) OR "YD" (YARDS), FOR EXAMPLE: "240T-500M". See table 2216 which is a composite

 

 

6B

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

6C

MINE LINE, Q-ROUTE POSITION

POS

ENTER "POS:" FOLLOWED BY THE Q-ROUTE POSITION, FOR EXAMPLE: "POS:171Z87.4MS645". See table 2203 which is a composite

7

DEPTH IN METRES, 1-999

 

 

 

 

7A

DEPTH IN METRES, 1-999

 

ENTER DEPTH IN METRES [0-999], FOR EXAMPLE: "50". See table 1247/6 a range [1 through 999]

8

ARMING DELAY

 

ENTER ARMING DELAY IN ONE OF THE FOLLOWING WAYS:

 

 

8A

ARMING DELAY, MINUTES

 

ENTER THE MINUTES [1-59], PLUS "MIN", FOR EXAMPLE: "12MIN". See table 2096 which is a composite

 

 

8B

ARMING DELAY, HOURS AND MINUTES

 

ENTER THE HOURS [1-23], PLUS "HR",  PLUS MINUTES [1-59], PLUS "MIN", FOR EXAMPLE: "1HR12MIN". See table 2097 which is a composite

 

 

8C

ARMING DELAY, DAYS, HOURS AND MINUTES

 

ENTER THE DAYS [1-999], PLUS "DAY", PLUS HOURS [1-23], PLUS "HR",  PLUS MINUTES [1-59], PLUS "MIN", FOR EXAMPLE: "22DAY23HR59MIN". See table 2098 which is a composite

9

SHIP COUNT

 

ENTER SHIP COUNT IN ONE OF THE FOLLOWING WAYS:

 

 

9A

COUNT OF SHIPS PASSED

 

ENTER THE COUNT OF SHIPS PASSED [0-99], FOR EXAMPLE: "4". See table 1044/79 a range [0 through 99]

 

 

9B

ESTIMATED COUNT OF SHIPS

 

ENTER THE ESTIMATED COUNT OF SHIPS GIVEN AS 'FROM' [0-99], HYPHEN, 'TO' [0-99], FOR EXAMPLE: "12-15". See table 2012 which is a composite

10

LAYING REFERENCE NUMBER (LRN)

 

 

 

 

10A

LAYING REFERENCE NUMBER (LRN)

 

ENTER THE LAYING REFERENCE NUMBER (LRN) [LITERAL], FOR EXAMPLE: "42BMW001-250". See table 1012/188 which is an instructive entry
