Concept/Explanation/Examples

1

MISSION IDENTIFIER

 

THE UNIQUE IDENTIFIER FOR THE MISSION.

 

 

1A

MISSION NUMBER

MSNO

ENTER "MSNO:" FOLLOWED BY THE MISSION NUMBER FOR THE MISSION BEING REPORTED, FOR EXAMPLE: "MSNO:A134". See table 1012/69 which is an instructive entry

 

 

1B

CALL SIGN, ABBREVIATED

CSIGN

ENTER "CSIGN:" FOLLOWED BY THE CALLSIGN OF THE MISSION, FOR EXAMPLE: "CSIGN:DELTA1". See table 1129/34 which is an instructive entry

2

REQUEST IDENTIFIER

 

 

 

 

2A

REQUEST IDENTIFIER

 

ENTER THE REQUEST IDENTIFIER FOR THE MISSION, FOR EXAMPLE: "MC2674". See table 1012/144 which is an instructive entry

3

TARGET LOCATION OR TARGET VESSEL TYPE

 

THE GEOGRAPHIC LOCATION OF THE TARGET OR THE TARGET VESSEL TYPE.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

3C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

3D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

3E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

3F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

3G

TARGET POSITION IDENTIFIER

ID

ENTER "ID:" FOLLOWED BY THE TARGET IDENTIFIER FOR THE MISSION AS SPECIFIED IN THE ATO, FOR EXAMPLE: "ID:B1234F12345" See table 1416/2 which is an instructive entry

 

 

3H

VESSEL TYPE

VSL

ENTER "VSL:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE TARGET VESSEL TYPE, FOR EXAMPLE: "VSL:CG". See table 1166/1 which contains a list of data items and associated data codes

4

TIME ON TARGET

 

THE TIME ON TARGET.

 

 

4A

TIME ON TARGET (HOUR-MINUTE)

 

ENTER THE ON TARGET TIME, FOR EXAMPLE:"0831". See table 2014 which is a composite

5

AIR MISSION RESULTS

 

 

 

 

5A

AIR MISSION RESULTS

 

ENTER THE RESULTS OF THE MISSION, FOR EXAMPLE: "S".  (RESULT APPLIES TO THE MISSION ACCOMPLISHMENT, FOR EXAMPLE; BOMBS DROPPED ON TARGET OR AIRDROP ACCOMPLISHED. IT DOES NOT, HOWEVER, INDICATE WHETHER A TARGET WAS DESTROYED. "S" (SUCCESSFUL) INDICATES A REQUIREMENT FOR FOLLOW UP ASSESSMENT TO DETERMINE THE DEGREE OF SUCCESS. "U" (UNSUCCESSFUL) INDICATES THE NEED TO BEGIN ACTIONS TO REACCOMPLISH THE MISSION IF PRIORITIES DICTATE.) See table 1088/6 which contains a list of data items and associated data codes
