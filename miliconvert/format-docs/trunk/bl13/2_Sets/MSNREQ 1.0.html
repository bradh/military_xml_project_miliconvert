AIRCRAFT ALERT CODE

 

THE ALERT STATUS ASSIGNED TO THE MISSION AIRCRAFT.  CAN ONLY BE USED IN CONJUNCTION WITH GROUND ALERT AERIAL MISSION TYPES.
