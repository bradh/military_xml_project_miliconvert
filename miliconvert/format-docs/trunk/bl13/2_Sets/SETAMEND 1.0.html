Designator

Field Desc

Concept/Explanation/Examples

1

AMENDMENT SET IDENTIFIER

 

 

 

 

1A

AMENDMENT SET IDENTIFIER

 

ENTER SET IDENTIFIER OF THE SET TO BE AMENDED, FOR EXAMPLE: "JAMCON". See table 1369/3 which is an instructive entry

2

REPEATED SET OCCURRENCE

 

 

 

 

2A

REPEATED SET OCCURRENCE

OCC

ENTER "OCC:" FOLLOWED BY THE REPETITION NUMBER OF THE SET TO BE REPEATED, FOR EXAMPLE: "OCC:SECOND". See table 1597/1 which contains a list of data items and associated data codes

3

AMENDMENT TYPE

 

 

 

 

3A

AMENDMENT TYPE

 

ENTER THE TYPE OF MESSAGE AMENDMENT BEING MADE, FOR EXAMPLE: "REP". See table 1653/1 which contains a list of data items and associated data codes

4

FORMETS SET TYPE

 

 

 

 

4A

FORMETS SET TYPE

 

ENTER THE FORMET SET TYPE, FOR EXAMPLE: "LINEAR". See table 1596/1 which contains a list of data items and associated data codes

5

SET SEQUENCE NUMBER

 

 

 

 

5A

SET SEQUENCE NUMBER

SEQ

ENTER "SEQ:" FOLLOWED BY THE SET SEQUENCE NUMBER OF THE SET FORMAT COUNTED FROM THE FIRST SET (EXER OR OPER) OF THE REFERENCED MESSAGE, FOR EXAMPLE: "32". See table 1012/233 a range [1 through 999]

6

DATA LINE SEQUENCE NUMBER

 

 

 

 

6A

DATA LINE SEQUENCE NUMBER

LINE

ENTER "LINE:" FOLLOWED BY THE NUMBER INDICATING THE POSITION OF THE TARGETED LINE WITHIN THE PARTICULAR COLUMNAR SET FORMAT BEING AMENDED.  THE HEADER LINE IS ALWAYS DATA LINE SEQUENCE NUMBER 1, WITH FOLLOWING DATA LINES NUMBERED SEQUENTIALLY BEGINNING WITH 2, FOR EXAMPLE: "3". See table 1012/234 a range [1 through 999]

 

 

Structured Notation:

[^]F6 RP ([^]F4 EQ /COLUMNAR/)

 

 

Structured Language:      

FIELD 6 IN SET ^ (SETAMEND) IS REQUIRED IF FIELD 4 IN SET ^ (SETAMEND) EQUALS /COLUMNAR/, OTHERWISE IT IS PROHIBITED.
