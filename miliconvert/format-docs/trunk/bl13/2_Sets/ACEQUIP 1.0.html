Designator

Field Desc

Concept/Explanation/Examples

1

RADAR WARNING RECEIVER TYPE

 

PROVIDES THE TYPE OF RADAR WARNING RECEIVING DEVICE THE REPORTED AIRCRAFT IS EQUIPPED WITH.

 

 

1A

RADAR WARNING RECEIVER TYPE

 

ENTER THE TYPE OF RADAR WARNING RECEIVING DEVICE THE REPORTED AIRCRAFT IS EQUIPPED WITH, FOR EXAMPLE: "ALQ125". See table 1135/25 which contains a list of data items and associated data codes

2

RADAR TYPE/FUNCTION

 

PROVIDES THE PRIMARY TYPE OR FUNCTION OF THE RADAR SYSTEM INSTALLED ABOARD THE AIRCRAFT REPORTED.

 

 

2A

RADAR TYPE FUNCTION

 

ENTER THE PRIMARY TYPE OR FUNCTION OF THE RADAR SYSTEM INSTALLED ABOARD THE AIRCRAFT REPORTED, FOR EXAMPLE "ET". See table 1471/4 which contains a list of data items and associated data codes

3

SOFTWARE RELEASE RADAR TYPE

 

PROVIDES THE UNIQUE DESIGNATOR OF THE RELEASE OF THE SOFTWARE INSTALLED ON THE ITEM OF EQUIPMENT REPORTED AT SEQ. 2 OF THE SET.

 

 

3A

SOFTWARE RELEASE

 

ENTER THE UNIQUE DESIGNATOR OF THE RELEASE OF THE SOFTWARE INSTALLED ON THE ITEM OF EQUIPMENT REPORTED AT SEQ. 2 OF THE SET, FOR EXAMPLE: "2.0.1". See table 1012/363 which is an instructive entry

4

COUNTERMEASURES EQUIPMENT

 

IF USED, PROVIDES THE TYPE OF COUNTERMEASURES EQUIPMENT INSTALLED ON REPORTED MISSION AIRCRAFT.

 

 

4A

COUNTERMEASURES EQUIPMENT

 

ENTER THE TYPE OF COUNTERMEASURES EQUIPMENT INSTALLED ON REPORTED MISSION AIRCRAFT, FOR EXAMPLE: "ALQ135". See table 1135/65 which contains a list of data items and associated data codes

5

SOFTWARE RELEASE COUNTERMEASURE

 

PROVIDES THE UNIQUE DESIGNATOR OF THE RELEASE OF THE SOFTWARE INSTALLED ON THE COUNTERMEASURE ITEM OF EQUIPMENT REPORTED AT SEQ. 4 OF THE SET.

 

 

5A

SOFTWARE RELEASE

 

ENTER THE RELEASE OF THE SOFTWARE USED FOR THE COUNTERMEASURE EQUIPMENT REPORTED AT SEQ. 4 OF THE SET, FOR EXAMPLE: "2.5.1". See table 1012/363 which is an instructive entry

6

COUNTERMEASURES DISPENSER

 

IF USED, PROVIDES THE TYPE OF COUNTERMEASURE DISPENSER INSTALLED ON REPORTED MISSION AIRCRAFT.

 

 

6A

COUNTERMEASURES DISPENSER

 

ENTER THE TYPE OF COUNTERMEASURE DISPENSER INSTALLED ON REPORTED MISSION AIRCRAFT, FOR EXAMPLE: "ALE45". See table 1135/66 which contains a list of data items and associated data codes

7

TOWED DECOYS

 

IF USED, PROVIDES THE TYPE OF DECOYS TOWED DURING MISSION OPERATIONS.

 

 

7A

TOWED DECOYS

 

ENTERTHE TYPE OF DECOYS TOWED DURING MISSION OPERATIONS, FOR EXAMPLE: "ALE50". See table 1135/67 which contains a list of data items and associated data codes

8

MATRIX CAPABILITIES

 

IF USED, PROVIDES THE TYPE OF COUNTERMEASURES LINKING SYSTEM INSTALLED ON REPORTED MISSION AIRCRAFT.

 

 

8A

MATRIX CAPABILITIES

 

ENTER THE TYPE OF COUNTERMEASURES LINKING SYSTEM INSTALLED ON REPORTED MISSION AIRCRAFT, FOR EXAMPLE: "B4208". See table 1135/69 which contains a list of data items and associated data codes

9

TARGETING AND/OR NAVIGATION POD

 

IF USED, PROVIDES THE SPECIFIC TYPE OF TARGETING AND/OR NAVIGATION POD EMPLOYED BY TASKED MISSION AIRCRAFT.

 

 

9A

TARGETING AND OR NAVIGATION POD

 

ENTER THE SPECIFIC TYPE OF TARGETING AND/OR NAVIGATION POD EMPLOYED BY TASKED MISSION AIRCRAFT, FOR EXAMPLE: "AN-AAQ-28". See table 1135/70 which contains a list of data items and associated data codes
