Concept/Explanation/Examples

1

MISSION NUMBER

 

 

 

 

1A

MISSION NUMBER

 

ENTER THE MISSION NUMBER, FOR EXAMPLE: "A195" See table 1012/69 which is an instructive entry

2

PRIMARY CONFIGURATION CODE

 

 

 

 

2A

PRIMARY CONFIGURATION LOAD

 

ENTER THE PRIMARY CONFIGURATION LOAD FOR THE AIRCRAFT.  FOR EXAMPLE: "2HARM". See table 1012/258 which is an instructive entry

3

DAY-TIME OF DEPARTURE

 

 

 

 

3A

DTG OF DEPARTURE

 

ENTER THE ACTUAL DATE-TIME OF DEPARTURE FOR THE MISSION, FOR EXAMPLE: "011200ZFEB2006" See table 2033 which is a composite

4

ESTIMATED DTG OF DEPARTURE

 

MUST NOT BE USED IF DAY-TIME OF DEPARTURE IS SPECIFIED.

 

 

4A

ESTIMATED DTG OF DEPARTURE

 

ENTER THE ESTIMATED DTG OF DEPARTURE  FOR THE MISSION, FOR EXAMPLE: "012300ZFEB2006" See table 2033 which is a composite

5

NUMBER OF AIRCRAFT AIRBORNE

 

MUST BE USED IN CONJUNCTION WITH DAY-TIME OF DEPARTURE.

 

 

5A

NUMBER OF AIRCRAFT AIRBORNE (00-99)

 

ENTER THE NUMBER OF AIRCRAFT AIRBORNE FOR THE MISSION, FOR EXAMPLE: "02" See table 1044/430 a range [00 through 99]

6

NUMBER OF AIRCRAFT ABORTED

 

MUST BE USED IF NEITHER "DAY-TIME OF DEPARTURE" NOR "ESTIMATED DTG OF DEPARTURE" ARE SPECIFIED.

 

 

6A

NUMBER OF AIRCRAFT ABORTED

 

ENTER THE NUMBER OF AIRCRAFT ABORTED IN THE MISSION, FOR EXAMPLE: "01" See table 1044/431 a range [00 through 99]

7

REASON AIRCRAFT ABORTED

 

MUST BE USED IN CONJUNCTION WITH NUMBER OF AIRCRAFT ABORTED.

 

 

7A

REASON AIRCRAFT ABORTED

 

ENTER THE CODE INDICATING THE PRIMARY REASON THE AIRCRAFT ABORTED, FOR EXAMPLE: "MAINT" See table 1212/15 which contains a list of data items and associated data codes
