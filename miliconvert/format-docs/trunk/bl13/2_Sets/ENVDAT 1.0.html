1

WEATHER SATELLITE IMAGERY CAPABILITY

 

 

 

 

1A

AFFIRMATIVE OR NEGATIVE INDICATOR

 

ENTER AFFIRMATIVE OR NEGATIVE  INDICATOR WHETHER OR NOT THE UNIT IS CAPABLE OF RECEIVING WEATHER SATELLITE IMAGERY, FOR EXAMPLE: "NO". See table 1027/1 which contains a list of data items and associated data codes

2

UPPER AIR SOUNDING SYSTEM

 

 

 

 

2A

NAME OF UPPER AIR SOUNDING SYSTEM

 

ENTER THE NAME OF THE ENVIRONMENTAL MONITORING SYSTEM USED FOR TAKING UPPER AIR SOUNDINGS, FOR EXAMPLE: "AUASS". See table 1022/20 which is an instructive entry

3

ATMOSPHERIC RANGE PREDICTION MODEL

 

 

 

 

3A

NAME OF ATMOSPHERIC RANGE PREDICTION MODEL

 

ENTER THE NAME OF THE MODEL USED FOR DETERMINING ATMOSPHERIC ELECTROMAGNETIC PROPAGATION RANGES, FOR EXAMPLE: "IREPS". See table 1022/28 which is an instructive entry

4

ATMOSPHERIC RANGE PREDICTION COMPUTER

 

 

 

 

4A

NAME OF ATMOSPHERIC RANGE PREDICTION COMPUTER

 

ENTER THE NAME OF THE COMPUTER SYSTEM ON WHICH THE ATMOSPHERIC RANGE PREDICTION MODEL IS RUN, FOR EXAMPLE: "PDT". See table 1022/58 which is an instructive entry

5

BATHYTHERMOGRAPH SYSTEM

 

 

 

 

5A

NAME OF THERMOGRAPH SYSTEM

 

ENTER THE NAME OF THE BATHYTHERMOGRAPHIC SYSTEM, FOR EXAMPLE: "X8X". See table 1022/61 which is an instructive entry

6

SONAR RANGE PREDICTION MODEL

 

 

 

 

6A

NAME OF SONAR RANGE PREDICTION MODEL

 

ENTER THE NAME OF THE MODEL OR COMPUTER PROGRAMME USED FOR DETERMINING SONAR PROPAGATION RANGES, FOR EXAMPLE: "HODGSON". See table 1022/65 which is an instructive entry

ENTER THE NAME OF THE COMPUTER SYSTEM ON WHICH THE SONAR RANGE PREDICTION MODEL IS RUN, FOR EXAMPLE: "SONAR 2090". See table 1022/70 which is an instructive entry
