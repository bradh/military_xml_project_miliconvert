Concept/Explanation/Examples

1

CENTRE OF CIRCLE

 

THE GEOGRAPHIC LOCATION OR OTHER POINT WHICH REPRESENTS THE CENTRE OF THE CIRCLE.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

1B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

1C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

1D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

1E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

1F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

1G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE LOCATION OF THE CENTRE OF THE CIRCLE AS GEOREF, CENTIMINUTE, FOR EXAMPLE: "GEO:ABAB11223344". See table 2018 which is a composite

 

 

1H

POSITION OR POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE PRE-DEFINED NAME OR DESIGNATOR FOR THE CENTRE OF THE CIRCLE, FOR EXAMPLE: "NAME:AA". See table 1022/18 which is an instructive entry

 

 

1I

TRUE BEARING AND DISTANCE IN NM FROM REFERENCE POINT

TBRN

ENTER "TBRN:" FOLLOWED BY THE LOCATION OF THE CENTRE OF THE CIRCLE AS A TRUE BEARING AND DISTANCE IN NAUTICAL MILES FROM A SPECIFIED REFERENCE POINT, FOR EXAMPLE: "TBRN:205-PORTLAND BILL LT-34.5". See table 2016 which is a composite

 

 

1J

Q-ROUTE POSITION

QPOS

ENTER "QPOS:" FOLLOWED BY THE LOCATION OF THE CENTRE OF THE CIRCLE AS A Q-ROUTE POSITION, FOR EXAMPLE: "QPOS:121R1234PS121". See table 2203 which is a composite

 

 

1K

UNIT NAME

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OR DESIGNATION OF THE UNIT UPON WHICH THE CIRCLE IS CENTRED, FOR EXAMPLE: "UNIT:INVINCIBLE". See table 1022/48 which is an instructive entry

 

 

1L

VERIFIED TRUE BEARING AND DISTANCE FROM REF POINT

BAR

ENTER "BAR:" FOLLOWED BY THE LOCATION OF THE CENTRE OF THE CIRCLE AS A TRUE BEARING FROM A NAMED REFERENCE POINT, FOLLOWED BY THE LETTER "T", THE BEARING VERIFICATION DIGIT, A HYPHEN, THE REF. POINT NAME, A HYPHEN, THE DISTANCE FROM THE REF POINT, THE UNIT OF DISTANCE MEASUREMENT AND THE DISTANCE VERIFICATION DIGIT, FOR EXAMPLE: "BAR:123.4T0-MEWSTONE-12.3NM6". See table 2202 which is a composite

 

 

1M

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES VERIFIED

SUB

ENTER THE LATITUDE AND LONGITUDE OF THE CENTRE OF THE CIRCLE UP TO 1/10,000 OF A MINUTE, VERIFIED, FOR EXAMPLE: "2010.30N6-01040.20W7". See table 2577 which is a composite
