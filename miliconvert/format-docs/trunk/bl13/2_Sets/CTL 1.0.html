No

Designator

Field Desc

Concept/Explanation/Examples

1

TARGET AREA

 

 

 

 

1A

TARGET AREA DESCRIPTION

 

ENTER A PLAIN LANGUAGE DESCRIPTION OF THE FEATURES OF THE AREA WHERE THE TARGET IS LOCATED, FOR EXAMPLE: "AMMUNITION DUMP". See table 1363/6 which is an instructive entry

2

DARB PRIORITY

 

 

 

 

2A

PREPLANNED PRIORITY

 

ENTER THE PRIORITY GIVEN TO THE TARGET BY THE DAILY ASSET RECONNAISSANCE BOARD, FOR EXAMPLE: "3". See table 1076/1 which contains a list of data items and associated data codes

3

COMPOUND COMMAND PRIORITY

 

 

 

 

3A

COMPOUND COMMAND PRIORITY

 

ENTER THE 2 CHARACTER CODE FOR THE COMPOUND COMMAND  PRIORITY ASSOCIATED WITH THE TARGET, FOR EXAMPLE: "3B". See table 1076/17 which is an alphanumeric range

4

RECONNAISSANCE CATEGORY

 

THE RECONNAISSANCE CATEGORY ASSOCIATED WITH THE TARGET ACCORDING TO STANAG 3596.

 

 

4A

TARGET CATEGORY CODE

CAT

ENTER "CAT:" FOLLOWED BY THE CATEGORY OF THE RECONNAISSANCE MISSION ASSOCIATED WITH THE TARGET, FOR EXAMPLE: "CAT:05". See table 1309/2 which contains a list of data items and associated data codes

 

 

4B

TARGET CATEGORY PURPOSE

PUR

ENTER "PUR:" FOLLOWED BY THE ONE CHARACTER CODE FOR THE RECONNAISSANCE MISSION PURPOSE, FOR EXAMPLE: "PUR:A". See table 1435/2 which contains a list of data items and associated data codes

 

 

4C

ESSENTIAL ELEMENTS OF INFORMATION (EEI) OF TARGET CATEGORY

EEI

ENTER "EEI:" FOLLOWED BY THE CODE FOR THE TARGET CATEGORY ESSENTIAL ELEMENTS OF INFORMATION THAT REPRESENTS THE RECONNAISSANCE OBJECTIVES ASSOCIATED WITH INDIVIDUAL TARGET CATEGORIES, FOR EXAMPLE:"EEI:01-7". See table 1802/1 which contains a list of data items and associated data codes

5

TARGET IDENTIFICATION (BE NUMBER OR TIN)

 

IDENTIFIES THE TARGET USING EITHER THE ASSOCIATED BASIC ENCYCLOPEDIA (BE) NUMBER WITH OR WITHOUT SUFFIX OR ITS IDENTITY ID REFERENCE NUMBER (EIRN).

 

 

5A

BASIC ENCYCLOPEDIA NUMBER

BEN

ENTER "BEN:" FOLLOWED BY THE BASIC ENCYCLOPEDIA NUMBER ASSOCIATED WITH THE TARGET, FOR EXAMPLE: "BEN:1635LG1A22". See table 2002 which is a composite

 

 

5B

TARGET IDENTIFIER BE NUMBER WITH SUFFIX

BENS

ENTER "BENS:", FOLLOWED BY THE BASIC ENCYCLOPEDIA NUMBER WITH SUFFIX ASSOCIATED WITH THE TARGET, FOR EXAMPLE: "BENS:A1635LG1A2268". See table 2122 which is a composite

 

 

5C

TARGET ENTITY ID REFERENCE NUMBER

EIRN

ENTER "EIRN:" FOLLOWED BY THE TARGET IDENTITY ID REFERENCE NUMBER, FOR EXAMPLE: "EIRN:150000114294960011". See table 2085 which is a composite

6

RECONNAISSANCE MISSION TYPE

 

 

 

 

6A

RECONNAISSANCE MISSION TYPE IDENTIFIER

 

ENTER THE IDENTIFIER FOR THE TYPE OF RECONNAISSANCE MISSION ASSOCIATED WITH THE TARGET, FOR EXAMPLE: "IIC". See table 1285/26 which contains a list of data items and associated data codes

7

START AND END TIME OVER TARGET

 

 

 

 

7A

START AND END TIME OVER TARGET

 

ENTER THE START AND END TIME (ZULU) TO BE OVER TARGET, SEPARATED BY AN HYPHEN, FOR EXAMPLE: "312350ZDEC2004-010010ZJAN2005". See table 2117 which is a composite

8

TYPE OF INTELLIGENCE TASK

 

 

 

 

8A

TYPE OF INTELLIGENCE TASK

 

ENTER THE CODE FOR THE INTELLIGENCE TASK TO BE PERFORMED ON THE RECONNAISSANCE MISSION, FOR EXAMPLE: "DES". See table 1285/28 which contains a list of data items and associated data codes

9

LATEST TIME OF REPORT

 

 

 

 

9A

DTG OF REPORT, ZULU

 

ENTER THE DATE-TIME GROUP OF THE MOST RECENT REPORT ABOUT THE TARGET, FOR EXAMPLE: "192120ZJAN2003". See table 2323 which is a composite

10

REMARKS

 

 

 

 

10A

REMARKS

 

ENTER A PLAIN TEXT REMARK CONCERNING THE TARGET, FOR EXAMPLE: "OBSERVATIONS NOT AVAILABLE BEFORE MAY 1998". See table 1357/28 which is an instructive entry

11

REFERENCE

 

 

 

 

11A

REFERENCE

 

ENTER THE REFERENCE CCIRM (COLLECTION COORDINATION INTELLIGENCE REQUIREMENT MANAGEMENT) OR IOP (INFORMATION OPERATION PRIORITY) NUMBER, FOR EXAMPLE "SAF12-24P". See table 1490/1 which is an instructive entry
