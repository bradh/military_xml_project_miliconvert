Concept/Explanation/Examples

1

DATE-TIME GROUP

 

SPECIFY THE DAY-TIME OF POSITION IN ONE OF THE FOLLOWING WAYS:

 

 

1A

DAY-TIME OF POSITION

 

ENTER THE DAY-TIME OF POSITION, FOR EXAMPLE: "210530Z". See table 2000 which is a composite

2

GEOGRAPHIC LOCATION

 

THE ASSIGNED GEOGRAPHIC LOCATION.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

3

CONTACT COURSE

 

 

 

 

3A

CONTACT COURSE, DEGREES TRUE

 

ENTER THE CONTACT'S COURSE EXPRESSED IN DEGREES TRUE, FOR EXAMPLE: "235T". See table 2258 which is a composite

4

CONTACT SPEED

 

 

 

 

4A

CONTACT SPEED, KNOTS

 

ENTER THE CONTACT'S SPEED IN KNOTS (DECIMAL ALLOWED), FOR EXAMPLE: "12.5KTS". See table 2265 which is a composite

5

ASW MISSION EVENT

 

 

 

 

5A

ASW MISSION EVENT

 

ENTER THE CODE TO INDICATE THE MISSION EVE NT THAT TRANSPIRED, FOR EXAMPLE: "NOFIX".{THIS FUD IS TO REPLACE THE AGREED FUD AND MODIFIES DI/DCs.} See table 1224/3 which contains a list of data items and associated data codes

6

SENSOR PLATFORM

 

 

 

 

6A

SENSOR PLATFORM

 

ENTER THE PLATFORM ON WHICH A SENSOR IS USED TO GATHER ELECTRONIC AND/OR ACOUSTIC DATA, SELECT FROM THE FOLLOWING "AIR", "SUB", "SURF", FOR EXAMPLE: "AIR". See table 1473/2 which contains a list of data items and associated data codes

7

ACOUSTIC SENSOR

 

 

 

 

7A

ACOUSTIC SENSOR

 

ENTER THE TYPE OF ACOUSTIC SENSOR IN CONTACT AT THE TIME AND POSITION REPORTED, FOR EXAMPLE: "CASS". See table 1434/5 which contains a list of data items and associated data codes

8

ACOUSTIC INFORMATION SOURCE

 

 

 

 

8A

ACOUSTIC INFORMATION SOURCE

 

ENTER THE SOURCE OF POSITIONAL DATA USED WHEN INFORMATION IS OBTAINED FROM A SOURCE OTHER THAN A SENSOR, FOR EXAMPLE: "FLIR". See table 1474/1 which contains a list of data items and associated data codes

9

NUMBER OF DETECTING SENSORS

 

 

 

 

9A

NUMBER OF DETECTING SENSORS

 

ENTER THE NUMBER OF DETECTING SENSORS IN THE EVENT REPORTED, EXPRESSED IN ONE TO TWO NUMERICS, FOR EXAMPLE: "14". See table 1044/120 a range [1 through 99]

10

NAVIGATION ACCURACY

 

 

 

 

10A

NAVIGATION ACCURACY, YARDS, METRES OR NAUTICAL MILES

 

ENTER THE POSITION ACCURACY EXPRESSED AS ONE TO FOUR NUMERICS (DECIMALS ALLOWED) PLUS "NM" (NAUTICAL MILES), "YD" (YARDS) OR "M" (METRES), FOR EXAMPLE: "100YD". See table 2284 which is a composite

11

ALPHAMONTH-YEAR

 

 

 

 

11A

ALPHAMONTH-YEAR

 

ENTER THE MONTH AND YEAR OF EVENT, FOR EXAMPLE: "FEB99".{THIS IS A NEW FFIRN/FUD COMPOSED OF AGREED ELEMENTALS BUILT AS A NEW FIELD IN THIS SET} See table 2556 which is a composite
