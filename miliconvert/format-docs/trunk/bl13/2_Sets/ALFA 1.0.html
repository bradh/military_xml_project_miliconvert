Concept/Explanation/Examples

1

GEOGRAPHIC LOCATION

 

PROVIDES THE GEOGRAPHIC LOCATION IN ONE OF THE TWO FOLLOWING WAYS:

 

 

1A

NATIONALITY

NAT

ENTER "NAT:" FOLLOWED BY THE  COUNTRY CODE, FOR EXAMPLE: "NAT:UK". See table 1026/2 which contains a list of data items and associated data codes

 

 

1B

AREA CONTROL CENTRE CODE

ACC

ENTER "ACC:" FOLLOWED BY THE  CODE SPECIFYING THE LOCATION OF AREA CONTROL CENTER, FOR EXAMPLE: "ACC:D45". See table 1080/2 which is an instructive entry

2

CODE FOR ORIGINATOR

 

 

 

 

2A

CODE FOR ORIGINATOR

 

ENTER THE UNIT CODE WHICH IDENTIFIES THE UNIT ISSUING THE MESSAGE, FOR EXAMPLE: "1HF4". See table 1029/3 which is an instructive entry

3

SEQUENCE NUMBER

 

 

 

 

3A

SEQUENCE NUMBER

 

ENTER THE STRIKE SEQUENCE NUMBER, FOR EXAMPLE: "2FG T24". See table 1012/57 which is an instructive entry

4

TYPE OF INCIDENT

 

 

 

 

4A

TYPE OF INCIDENT

 

ENTER THE CODE SPECIFYING THE TYPE OF NBC INCIDENT, FOR EXAMPLE: "RB". See table 1211/10 which contains a list of data items and associated data codes

5

GRADING OF MESSAGE/REPORT

 

 

 

 

5A

GRADING OF MESSAGE REPORT

 

ENTER A NUMERIC CODE QUALIFYING THE LEVEL OF RELIABILITY GIVEN TO THE MESSAGE, FROM 1,  VERY LOW QUALITY, TO 999, VERY HIGH QUALITY,  FOR EXAMPLE: "14". See table 1202/4 a range [1 through 999]
