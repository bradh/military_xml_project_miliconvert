No

Designator

Field Desc

Concept/Explanation/Examples

1

REPORTING AIRCRAFT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE REPORTING AIRCRAFT WHEN THE SIGNAL WAS INTERCEPTED.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGTITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

2

DTG OF ELECTRONIC INTERCEPT

 

PROVIDES THE DTG WHEN THE SIGNAL WAS INTERCEPTED.

 

 

2A

DTG OF ELECTRONIC INTERCEPT

 

ENTER THE APPROXIMATE DTG WHEN THE SIGNAL WAS INTERCEPTED, FOR EXAMPLE: "201200ZJUL2005". See table 2033 which is a composite

3

AIRCRAFT ALTITUDE

 

PROVIDES THE ALTITUDE OF THE REPORTING AIRCRAFT WHEN THE SIGNAL WAS INTERCEPTED.

 

 

3A

AIRCRAFT ALTITUDE IN HUNDREDS OF FEET

 

ENTER THE ALTITUDE OF THE AIRCRAFT WHEN IT EMITTED THE INTERCEPTED SIGNAL, USING THREE DIGITS, FOR EXAMPLE, "210". See table 1255/21 a range [000 through 999]

4

REPORTING AIRCRAFT HEADING

 

PROVIDES THE HEADING OF THE REPORTING AIRCRAFT WHEN THE SIGNAL WAS INTERCEPTED.

 

 

4A

REPORTING AIRCRAFT HEADING IN DEGREES

 

ENTER THE HEADING OF THE REPORTING AIRCRAFT WHEN THE SIGNAL WAS INTERCEPTED, WITH INDICATION IF TRUE, MAGNETIC, GRID OR RELATIVE, FOR EXAMPLE "190T". See table 2065 which is a composite

5

REPORTING AIRCRAFT SPEED

 

PROVIDES THE SPEED OF THE REPORTING AIRCRAFT.

 

 

5A

AIRCRAFT AIRSPEED IN KPH, KTS, MPH

 

ENTER THE AIRSPEED OF THE REPORTING AIRCRAFT, FOR EXAMPLE "445KTS". See table 2461 which is a composite

6

RADAR WARNING RECEIVER INDICATION

 

PROVIDES A YES/NO INDICATION WHETHER ANY RADAR WARNING RECEIVER INDICATION WAS RECEIVED IN ASSOCIATION WITH THE EMISSION.

 

 

6A

RADAR WARNING RECEIVER INDICATIONS RECEIVED INDICATOR

 

ENTER AN INDICATION WHETHER A RADAR WARNING WAS RECEIVED OR NOT CONCERNING THE INTERCEPTED SIGNAL, FOR EXAMPLE:"NO". See table 1027/89 which contains a list of data items and associated data codes

7

DURATION OF SIGNAL

 

PROVIDES THE DURATION OF THE INTERCEPTED SIGNAL, IN HOURS, MINUTES OR SECONDS.

 

 

7A

DURATION OF SIGNAL, HOURS, MINUTES, SECONDS

 

ENTER THE DURATION OF THE INTERCEPTED SIGNAL, IN HOURS, MINUTE OR SECONDS, FOR EXAMPLE "140MIN". See table 2521 which is a composite

8

BEARING OF SIGNAL

 

PROVIDES THE BEARING OF THE EMITTER FROM THE REPORTING AIRCRAFT OR FROM THE BULLSEYE.

 

 

8A

BEARING OF SIGNAL FROM INTERCEPTING AIRCRAFT, DEGREES TRUE

BEARING

ENTER "BEARING:", FOLLOWED BY THE BEARING OF THE EMITTER FROM THE REPORTING AIRCRAFT, ANGLE IN DEGREES AND INDICATION FOR TRUE NORTH, FOR EXAMPLE: "BEARING:120T". See table 2253 which is a composite

 

 

8B

BEARING OF SIGNAL FROM BULLSEYE, DEGREES TRUE

BULL

ENTER "BULL:", FOLLOWED BY THE BEARING OF THE EMITTER FROM THE BULLSEYE, ANGLE IN DEGREES AND INDICATION FOR TRUE NORTH, FOR EXAMPLE: "BULL:120T". See table 2253 which is a composite

9

INTERCEPT SERIAL NUMBER

 

PROVIDES THE SERIAL NUMBER OF THE INTERCEPT.

 

 

9A

INTERCEPT SERIAL NUMBER

 

ENTER THE INTERCEPT SERIAL NUMBER, FOR EXAMPLE: "015". See table 1012/25 a range [000 through 999]

10

SIGNAL INTERCEPT DISTANCE FROM AIRCRAFT

 

PROVIDES THE DISTANCE OF THE EMITTER FROM THE REPORTING AIRCRAFT IN NAUTICAL MILES.

 

 

10A

SIGNAL INTERCEPT DISTANCE IN NAUTICAL MILES

 

ENTER THE DISTANCE OF THE EMITTER FROM THE REPORTING AIRCRAFT IN NAUTICAL MILES, FOR EXAMPLE: "100". See table 1089/16 a range [0 through 999]

11

SIGNAL POSITION

 

PROVIDES THE GEOGRAPHICAL COORDINATES OF THE EMITTER.

 

 

11A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGTITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

12

EMITTER NAME

 

PROVIDES THE NAME OF THE EMITTER OF THE INTERCEPTED SIGNAL.

 

 

12A

EMITTER NAME

 

ENTER THE ASSIGNED NAME OF THE EMITTER OR STATION EMITTING THE INTERCEPTED SIGNAL, FOR EXAMPLE: "BIG BIRD". See table 1022/225 which is an instructive entry

13

ELINT NOTATION

 

 

 

 

13A

ELINT NOTATION

 

ENTER A 5-CHARACTER ELINT NOTATION FOR NON-COMMUNICATIONS ELECTRONICS EMISSIONS RELATED TO SUCH EQUIPMENT. THE ELINT NOTATION IS EXTRACTED FROM THE LATEST EDITION OF THE ELINT PARAMETER LIST, FOR EXAMPLE: "CA220". See table 1012/34 which is an instructive entry

14

EMITTER DESIGNATION

 

 

 

 

14A

EMITTER DESIGNATION

 

ENTER THE EMITTER DESIGNATION, FOR EXAMPLE: "M343S". See table 1121/1 which is an instructive entry

15

DESCRIPTION OF TONE RECEIVED

 

PROVIDES A PLAIN WORDS DESCRIPTION OF THE TONE RECEIVED BY THE INTERCEPTING EQUIPMENT.

 

 

15A

DESCRIPTION OF TONE RECEIVED

 

ENTER A PLAIN WORDS DESCRIPTION OF THE TONE RECEIVED BY THE INTERCEPTING EQUIPMENT, FOR EXAMPLE "LOUD RIPPLE". See table 1363/26 which is an instructive entry

16

PULSE REPETITION INTERVAL ACTIVITY CODE

 

PROVIDES THE CODES CORRESPONDING TO THE MODULATION CHARACTERISTIC OF THE EMITTER SIGNAL.

 

 

16A

PULSE REPETITION INTERVAL ACTIVITY CODE

PRI

ENTER "PRI:" FOLLOWED BY THE MODULATION CHARACTERISTIC OF AN EMITTER SIGNAL, FOR EXAMPLE: "PRI:P". See table 1471/6 which contains a list of data items and associated data codes

 

 

16B

PULSE REPETITION FREQUENCY VARIATION

PRF

ENTER "PRF:" FOLLOWED BY THE VARIATION IN THE FREQUENCY OF THE PULSE REPETITION, FOR EXAMPLE: "PRF:PLSGP". See table 1471/7 which contains a list of data items and associated data codes

17

PRIMARY FREQUENCY IN MEGAHERTZ

 

 

 

 

17A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

 

ENTER THE RECEIVED FREQUENCY OF THE INTERCEPTED SIGNAL MEASURED IN MEGAHERTZ, FOR EXAMPLE: "2025.360". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

18

EMITTER EVALUATION

 

PROVIDES THE EMITTER EVALUATION NICKNAME.

 

 

18A

EMITTER EVALUATION

 

ENTER THE EMITTER EVALUATION NICKNAME, FOR EXAMPLE: "HEADNET". See table 1121/2 which is an instructive entry

19

ELECTRONIC PROTECTION CONFIDENCE LEVEL

 

PROVIDES THE LEVEL OF CONFIDENCE OF THE INFORMATION PROVIDED.

 

 

19A

ELECTRONIC PROTECTION CONFIDENCE LEVEL

CONF

ENTER "CONF:" FOLLOWED BY THE LEVEL OF CERTAINTY IN ELECTRONIC PROTECTION INFORMATION REPORTED, FOR EXAMPLE: "CONF:1". See table 1123/12 which contains a list of data items and associated data codes

 

 

19B

ELECTRONIC PROTECTION CREDIBILITY

CRED

ENTER "CRED:", FOLLOWED BY THE LEVEL OF CREDIBILITY OF ELECTRONIC PROTECTION INFORMATION REPORTED, FOR EXAMPLE: "CRED:MED". See table 1123/13 which contains a list of data items and associated data codes
