No

Designator

Field Desc

Concept/Explanation/Examples

1

RECONNAISSANCE MISSION LOCATION

 

THE GEOGRAPHIC LOCATION OF THE POSITION TRACE PLOT.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

1B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

2

TARGET POSITION

 

IF USED, PROVIDES THE POSITION WHERE THE TARGET IS SITUATED EITHER BY ITS NAME OR BY ITS GEOGRAPHICAL COORDINATES.

 

 

2A

TARGET POSITION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE TARGET POSITION NAME,  FOR EXAMPLE: "NAME:HAMBURGER". See table 1022/81 which is an instructive entry

 

 

2B

TARGET POSITION IDENTIFIER

ID

ENTER "ID:" FOLLOWED BY THE UNIQUE IDENTIFIER OF THE TARGET, FOR EXAMPLE: "ID:B1234F12345". See table 1416/2 which is an instructive entry

3

AREA DESCRIPTION

 

IF USED, PROVIDES THE GEOMETRIC DETAILS OF THE AREA OF RECONNAISSANCE.

 

 

3A

RADIUS, IN, FT, YD, KF, SM, NM, MM, CM, M, KM, HM

CIRC

ENTER "CIRC:" FOLLOWED BY THE RADIUS OF THE CIRCLE AROUND THE LOCATION BEING REPORTED IN FIELD 1, FOR EXAMPLE: "CIRC:3NM". See table 2490 which is a composite

 

 

3B

WIDTH, IN, FT, YD, KF, SM, NM, MM, CM, M, KM, HM

WDTH

ENTER "WDTH:" FOLLOWED BY THE LINEAR DISTANCE FROM SIDE TO SIDE OF A LINEAR TARGET, SUCH AS AN HIGHWAY, A RIVER OR A RUNWAY,  FOR EXAMPLE: "WDTH:100YD". See table 2491 which is a composite

 

 

3C

ELLIPTICAL AREA, DIMENSIONS AND MAJOR AXIS ORIENTATION

ELL

ENTER "ELL:" FOLLOWED BY THE DIMENSIONS AND ORIENTATION WHICH DESIGNATE AN ELLIPTICAL AREA AROUND THE LOCATION REPORTED IN FIELD 1, FOR EXAMPLE: "ELL:2000YD-1000YD-135.5". See table 2494 which is a composite

 

 

3D

SECTOR GEOMETRY WITH OUTER RANGE

SECTO

ENTER "SECTO:" FOLLOWED BY SECTOR GEOMETRY OF THE RECONNAISSANCE AREA USING AN OUTER RANGE, FOR EXAMPLE: "SECTO:180240T27NM". See table 2526 which is a composite

 

 

3E

SECTOR GEOMETRY WITH INNER AND OUTER RANGE

SECTIO

ENTER "SECTIO:" FOLLOWED BY THE SECTOR GEOMETRY OF THE RECONNAISSANCE AREA USING AN INNER AND OUTER RANGE, FOR EXAMPLE: "SECTIO:058190T45-60NM". See table 2527 which is a composite

4

TARGET ELEVATION

 

IF USED, PROVIDES THE TARGET ELEVATION EITHER ABOVE MEAN SEA LEVEL (MSL) OR ABOVE THE REFERENC ELLIPSOID (HAE).

 

 

4A

TARGET ELEVATION

 

ENTER THE ELEVATION OF THE TARGET MEASURED ABOVE OR BELOW THE MEAN SEA LEVEL OR ABOVE THE REFERNCE ELLIPSOID, FOR EXAMPLE: "-90FTMSL". See table 2492 which is a composite

5

GEODETIC DATUM

 

IF USED, PROVIDES THE GEODATIC DATUM CONCERNING THE TARGET.

 

 

5A

GEODETIC DATUM

 

IF THE DEFAULT GEODETIC DATUM SPECIFIED IN THE BEGINNING OF THE MESSAGE DOES NOT APPLY TO THE LOCATION OF THE TARGET, ENTER THE CORRECT GEODATUM, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

6

TRACE POINT

 

PROVIDES THE GEOGRAPHIC LOCATION OF THE TARGET.

 

 

6A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

6B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite
