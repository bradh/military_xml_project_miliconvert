Concept/Explanation/Examples

1

ORGANIZATION IDENTIFIER

 

 

 

 

1A

ORGANIZATION IDENTIFIER

 

ENTER THE NAME OR IDENTIFIER OF ORGANIZATION OF WHICH LOCATION IS BEING SPECIFIED, FOR EXAMPLE: "UK-NL AMPHIBIOUS GROUP". See table 1380/1 which is an instructive entry

2

ORGANIZATION DESCRIPTION

 

 

 

 

2A

ORGANIZATION DESCRIPTION

 

ENTER THE DESCRIPTION OF THE ORGANIZATION OF WHICH LOCATION IS BEING SPECIFIED, FOR EXAMPLE: "P3C SQDN". See table 1380/8 which is an instructive entry

3

NEW LOCATION OR SHIP NAME

 

PROVIDES THE NAME OF THE LOCATION OR SHIP TO WHICH THE ORGANIZATION HAS BEEN RELOCATED:

 

 

3A

SHIP NAME

SHIP

ENTER "SHIP:" FOLLOWED BY THE NAME OF THE SHIP TO WHICH THE ORGANIZATION HAS BEEN RELOCATED, FOR EXAMPLE: "SHIP:ARK ROYAL". See table 1022/49 which is an instructive entry

 

 

3B

PLACE NAME, EXTENDED

PLACE

ENTER "PLACE:" FOLLOWED BY THE NAME OF THE LOCATION TO WHICH THE ORGANISATION HAS BEEN RELOCATED, FOR EXAMPLE: "PLACE:SOUDA BAY". See table 1022/261 which is an instructive entry

4

LOCATION OR SHIP NATIONALITY

 

PROVIDES THE NATIONALITY OF THE LOCATION OR SHIP TO WHICH THE ORGANIZATION HAS BEEN RELOCATED:

 

 

4A

GEOGRAPHICAL ENTITY (2A)

 

ENTER THE TWO-LETTER COUNTRY CODE OF THE LOCATION OR SHIP TO WHICH THE ORGANIZATION HAS BEEN RELOCATED, FOR EXAMPLE: "GR". See table 1262/6 which contains a list of data items and associated data codes

 

 

4B

EXERCISE GEOGRAPHICAL ENTITY

EXEN

ENTER "EXEN:" FOLLOWED BY THE TWO-LETTER EXERCISE COUNTRY CODE OF THE LOCATION OR SHIP TO WHICH THE ORGANIZATION HAS BEEN RELOCATED, FOR EXAMPLE: "EXEN:XB". See table 1262/4 which contains a list of data items and associated data codes

5

DTG ORGANIZATION RELOCATED

 

 

 

 

5A

DTG RELOCATED

 

ENTER THE DATE-TIME GROUP AT WHICH THE ORGANIZATION WAS RELOCATED TO THE NEW LOCATION, FOR EXAMPLE: "172345ZAUG2001". See table 2033 which is a composite
