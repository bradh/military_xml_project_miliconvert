No

Designator

Field Desc

Concept/Explanation/Examples

1

BASE IDENTIFIER

 

THE IDENTIFIER FOR THE BASE BEING REPORTED.

 

 

1A

BASE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE AIR BASE, FOR EXAMPLE: "NAME:RAMSTEIN AB". See table 1022/104 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 2 LETTER COUNTRY CODE), FOR THE UNIT, FOR EXAMPLE: "UIC2:GEF1076". See table 2243 which is a composite

 

 

1C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE(THIS ALTERNATIVE IS FOR THE 3 LETTER COUNTRY CODE), FOR THE UNIT, FOR EXAMPLE: "UIC3:DEUF1076". See table 2241 which is a composite

 

 

1D

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO INDICATOR OF THE LOCATION, FOR EXAMPLE: "ICAO:KKIL". See table 1125/1 which is an instructive entry

2

BASE LOCATION

 

THE GEOGRAPHICAL LOCATION OF THE BASE OR UNIT BEING REPORTED.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE BASE LOCATION IN GEOREF, CENTIMINUTE, FOR EXAMPLE: "GEO:AABB15233527". See table 2018 which is a composite

3

ELEVATION

 

THE ELEVATION OF THE BASE BEING REPORTED IN FEET OR METRES.

 

 

3A

ELEVATION IN FEET

FT

ENTER "FT:" FOLLOWED BY THE ELEVATION OF THE BASE IN FEET, FOR EXAMPLE: "FT:120". See table 1799/1 a range [-1290 through 29028]

 

 

3B

ELEVATION IN METRES ABOVE MSL

MT

ENTER "MT:" FOLLOWED BY THE ELEVATION IN METRES ABOVE MEAN SEA LEVEL, FOR EXAMPLE: "MT:50". See table 1344/1 a range [-610 through 9999]
