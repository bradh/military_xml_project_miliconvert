SUBMARINE TRACK IDENTIFIERS

 

 

 

 

1A

SUBMARINE TRACK IDENTIFIERS

 

PROVIDE A UNIQUE IDENTIFIER FOR THE FIRST POSITION IN EACH SEGMENT ALONG THE PLANNED ROUTE.  ENTER TWO LETTER CODE AS FOLLOWS:A.  FIRST LETTER USED THROUGHOUT THE MESSAGE IS CHOSEN AT RANDOM, FOR EXAMPLE.  FIRST LETTER OF SUBMARINE NAMEB.  SECOND LETTER IN ALPHABETICAL ORDER, STARTING WITH "A" TO INDICATE THE SEQUENCE OF THE POSITION ALONG THE TRACK.FOR EXAMPLE: "SC". See table 2226 which is a composite

2

LOCATION

 

THE GEOGRAPHIC LOCATION OF THE FIRST PART OF THE MOVEMENT IN EACH SEGMENT.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

DATE-TIME MONTH, VERIFIED

 

 

 

 

3A

DATE-TIME MONTH, VERIFIED

 

GIVEN AS A VERIFIED DATE TIME:A. DEPARTURE FROM THE START POSITION WHEN THE SUBMARINE CONDUCTS A TRANSIT IN A MOVING HAVEN, FOR EXAMPLE, WHEN THE SUBTRKST SET IS FOLLOWED BY SUBTRK AND SUBMAN SETS, ORB. START OF DEPLOYMENT IS AN SPA, ISA OR LANE, THAT IS, WHEN THE SUBTRKST SET IS FOLLOWED BY A SUBAREA OR SUBLANE SET.FOR EXAMPLE: "121200Z6JUN". See table 2032 which is a composite

Notes:   

1.  THE DATE-TIME GROUP (DTG) IS GENERALLY NOT THE SAME AS IN THE FOLLOWING AND PRECEDING SUBAREA OR SUBLANE SET (SEE FIELDS 3 OF SOPAREA AND SUBLANE SETS).2.  THE SUBTRKST SET IS FOLLOWED BY SUBTRK SET(S) TO DESCRIBE THE TRACK TO BE FOLLOWED THROUGH TWO OR MORE INTERMEDIATE (VIA) POSITIONS WHEN THE SUBMARINE MOVES IN A SPECIFIC MOVING HAVEN WHICH DOES NOT CHANGE ITS SIZE, SPEED OF ADVANCE OR MODE OF OPERATIONS.3.  THE FIRST SUBTRKST POSITION AND THE LAST SUBTRK POSITION SHOULD REFLECT THE MOVESUM LINE.4.  TIMES AND POSITIONS IN A SUBTRKST SET FOR A SUBMARINE ENTERING OR DEPARTING AN AREA MUST EITHER COINCIDE WITH THE AREA'S BORDERS AND EFFECTIVE EXPIRY TIME, OR BE CONTAINED WITHIN THEM.
