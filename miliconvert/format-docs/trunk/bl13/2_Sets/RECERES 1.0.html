2A

STATUS OF TARGET OPERABILITY

 

ENTER TARGET OPERABILITY STATUS, FOR EXAMPLE: "PARTOPS". See table 1236/24 which contains a list of data items and associated data codes

STATUS OF TARGET SERVICEABILITY

 

 

 

 

3A

STATUS OF TARGET SERVICEABILITY

 

ENTER TARGET SERVICEABILITY STATUS, FOR EXAMPLE: "REP". See table 1236/25 which contains a list of data items and associated data codes

4

NUMBER OF TAGRGET ELEMENTS

 

GIVE TARGET INFORMATION USING ONE OF THE FOLLOWING FIELDS:

 

 

4A

NUMBER OF TARGET ELEMENTS

TEN

ENTER "TEN:" FOLLOWED BY THE NUMBER OF TARGET ELEMENTS, FOR EXAMPLE: "TEN:1". See table 1044/83 a range [1 through 9999]

 

 

4B

NUMBER OF ITEMS OF EQUIPMENT COUNTED OBSERVED

EQPN

ENTER "EQPN:" FOLLOWED BY THE NUMBER OF PIECES OF EQUIPMENT WHICH WERE COUNTED OR OBSERVED, FOR EXAMPLE: "EQPN:22". See table 1044/97 a range [0 through 9999]

5

TARGET PERCENTAGE COVERED

 

 

 

 

5A

TARGET PERCENTAGE COVERED

 

ENTER PERCENTAGE COVER OF TARGET OR DEPLOYED UNIT, FOR EXAMPLE: "45". See table 1064/43 a range [0 through 100]

6

EVALUATION OF ACCURACY

 

 

 

 

6A

EVALUATION OF ACCURACY

 

ENTER A ONE DIGIT NUMERIC CODE RATING THE ACCURACY OF THE RECONNAISSANCE MISSION RESULTS, FOR EXAMPLE: "1". See table 1251/1 which contains a list of data items and associated data codes

7

DESCRIPTION OF TARGET ELEMENT OR EQUIPMENT

 

ENTER EITHER A DESCRIPTION OF THE TARGET ELEMENT OR AN EQUIPMENT:

 

 

7A

TARGET ELEMENT DESCRIPTION

TED

ENTER "TED:" FOLLOWED BY THE TARGET ELEMENT DESCRIPTION, FOR EXAMPLE: "AMMO STORAGE". See table 1363/7 which is an instructive entry

 

 

7B

EQUIPMENT TYPE

EQPT

ENTER "EQPT:" FOLLOWED BY THE EQUIPMENT TYPE, FOR EXAMPLE: "EQPT:BACKNET". See table 1135/1 which is an instructive entry

8

ACTIVITY TYPE, LITERAL

 

 

 

 

8A

ACTIVITY TYPE, LITERAL

 

ENTER A LITERAL GIVING THE TYPE OF ACTIVITY, FOR EXAMPLE: "STATIC". See table 1299/10 which is an instructive entry

9

CHANGE SIGNIFICANCE

 

 

 

 

9A

CHANGE SIGNIFICANCE

 

ENTER CHANGE SIGNIFICANCE, FOR EXAMPLE: "MIN". See table 1671/1 which contains a list of data items and associated data codes

10

IMAGERY CONFIRMATION

 

 

 

 

10A

IMAGERY CONFIRMATION

 

ENTER POSITIVE OR NEGATIVE CONFIRMATION THAT TARGET WAS IMAGED, FOR EXAMPLE: "YES". See table 1027/7 which contains a list of data items and associated data codes

11

TARGET DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER

 

 

 

 

11A

TARGET DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER

 

ENTER THE DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER, FOR EXAMPLE: "QRS 1236-RTY". See table 1416/7 which is an instructive entry

12

WEAPON IMPACT POINT

 

THE GEOGRAPHIC LOCATION OF THE WEAPON POINT OF IMPACT.

 

 

12A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

12B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

12C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

12D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

12E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

12F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

12G

TRUE BEARING AND DISTANCE

TBR

ENTER "TBR:" FOLLOWED BY THE LOCATION EXPRESSED AS A TRUE BEARING AND DISTANCE IN NAUTICAL MILES FROM A SPECIFIED REFERENCE POINT, FOR EXAMPLE: "TBR: 205-PORTLAND BILL LT-34.5". See table 2410 which is a composite

 

 

12H

AUTOMATED TACTICAL TARGET GRAPHIC ANNOTATION

NOT

ENTER "NOT:" FOLLOWED BY THE AUTOMATED TACTICAL TARGET GRAPHIC ANNOTATION, FOR EXAMPLE: "NOT: "A01". See table 1411/2 which is an instructive entry

13

DEGRADATION

 

DESCRIBE THE EXTENT OF DEGRADATION EITHER WITH A CODE OR GIVING A PERCENTAGE:

 

 

13A

EXTENT OF DEGRADATION

 

ENTER A CODE DESCRIBING THE EXTENT OF DEGRADATION, FOR EXAMPLE: "A". See table 1088/8 which contains a list of data items and associated data codes

 

 

13B

DEGRADATION PERCENTAGE

 

ENTER AN ESTIMATE OF THE DEGRADATION OF TARGET AS A PERCENTAGE FIGURE [0-100], FOR EXAMPLE: "20". See table 1064/42 a range [1 through 100]

14

TYPE OF REPAIR ON TARGET

 

 

 

 

14A

TYPE OF REPAIR ON TARGET

 

ENTER A LITERAL DESCRIPTION OF THE TYPE OF REPAIR BEING CARRIED OUT ON THE TARGET, FOR EXAMPLE: "RESURFACING OF RUNWAY". See table 1676/2 which is an instructive entry

15

EFFORT REQUIRED TO REPAIR

 

 

 

 

15A

EFFORT REQUIRED TO REPAIR

 

ENTER AN ESTIMATE OF TIME AND RESOURCES LIKELY TO BE REQUIRED TO GET THE TARGET RESTORED, FOR EXAMPLE: "100 MAN DAYS". See table 1729/2 which is an instructive entry

16

REASON TARGET OPERATIONAL STATUS IS DEGRADED

 

 

 

 

16A

REASON TARGET OPERATIONAL STATUS IS DEGRADED

 

ENTER THE REASON WHY THE TARGET OPERATIONAL STATUS IS DEGRADED, FOR EXAMPLE: "CONV". REPEATING AS NECESSARY. See table 1235/2 which contains a list of data items and associated data codes
