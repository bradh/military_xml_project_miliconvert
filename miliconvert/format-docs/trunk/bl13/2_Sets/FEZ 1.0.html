1

PHONETIC ALPHABETIC IDENTIFIER

 

 

 

 

1A

PHONETIC ALPHABETIC IDENTIFIER

 

ENTER THE FIGHTER ENGAGEMENT ZONE OR AREA DESIGNATOR (SPELLED PHONETICALLY), FOR EXAMPLE: "ALFA". See table 1128/2 which contains a list of data items and associated data codes

2

ZONE OR AREA ORIGIN

 

ENTER THE FIGHTER ENGAGEMENT ZONE OR AREA OF ORIGIN IN ONE OF THE FOLLOWING WAYS:

 

 

2A

ORIGIN PLACE NAME

PLACE

ENTER "PLACE:" FOLLOWED BY THE ORIGIN PLACE NAME, FOR EXAMPLE: "PLACE:ZULU ZULU". See table 1022/109 which is an instructive entry

 

 

2B

UNIT

 

ENTER THE LITERAL NAME OF THE UNIT THAT WILL BE THE ORIGIN, FOR EXAMPLE: "DE RUYTER". See table 1022/57 which is an instructive entry

3

OUTER RANGE

 

 

 

 

3A

OUTER RANGE

 

ENTER THE OUTER RANGE OF THE ZONE FROM THE ORIGIN, FOR EXAMPLE: "24NM". See table 2158 which is a composite

4

LOWER ALTITUDE LIMIT

 

ENTER THE LOWER FLIGHT LEVEL OR ALTITUDE IN ONE OF THE FOLLOWING WAYS:

 

 

4A

MINIMUM ALTITUDE IN HUNDREDS OF FEET

 

ENTER "ALT" PLUS THE MARITIME MINIMUM ALTITUDE IN HUNDREDS OF FEET, FOR EXAMPLE: "ALT125". See table 2164 which is a composite

 

 

4B

MINIMUM FLIGHT LEVEL

 

ENTER "FL" PLUS THE MINIMUM FLIGHT LEVEL IN HUNDREDS OF FEET, FOR EXAMPLE: "FL124". See table 2129 which is a composite

5

UPPER ALTITUDE LIMIT

 

ENTER THE UPPER FLIGHT LEVEL OR ALTITUDE IN ONE OF THE FOLLOWING WAYS:

 

 

5A

MAXIMUM ALTITUDE IN HUNDREDS OF FEET

 

ENTER "ALT" PLUS THE MAXIMUM MARITIME ALTITUDE IN HUNDREDS OF FEET, FOR EXAMPLE: "ALT445". See table 2164 which is a composite

 

 

5B

MAXIMUM FLIGHT LEVEL

 

ENTER "FL" PLUS THE MAXIMUM FLIGHT LEVEL IN HUNDREDS OF FEET, FOR EXAMPLE: "FL345". See table 2129 which is a composite
