3

NEW ETA AT DOWNLINE STATION

 

ENTER ONE OF THE FOLLOWING:

 

 

3A

NEW ETA DAY-TIME AND MONTH AT DOWNLINE STATION

 

ENTER THE NEW ESTIMATE TIME OF ARRIVAL AT THE DOWNLINE LOCATION.  ENTER THE DATE (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE AND MONTH, FOR EXAMPLE: "041405ZAPR". See table 2030 which is a composite

 

 

3B

TIME QUALIFIER TO BE DETERMINED

 

ENTER THE DATA CODE FOR 'TO BE DETERMINED' IF ARRIVAL TIME CANNOT BE SPECIFIED AT THE TIME OF REPORTING, AND EXPLAIN IN A SUBSEQUENT FIELD, FOR EXAMPLE: "TBD". See table 1220/10 which contains a list of data items and associated data codes
