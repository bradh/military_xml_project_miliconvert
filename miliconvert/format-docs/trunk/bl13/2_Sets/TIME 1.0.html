DATE-TIME INDICATOR

 

ENTER THE DATE AND TIME IN ONE OF THE FOLLOWING WAYS:

 

 

2A

DTG, ZULU

 

DATE-TIME GROUP.  ENTER THE DAY (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE, MONTH, AND YEAR (00-99), FOR EXAMPLE: "261752ZNOV96". See table 2323 which is a composite

DAY-TIME AND MONTH

DTM

ENTER "DTM:" FOLLOWED BY DAY-TIME AND MONTH.  ENTER THE DAY (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE AND MONTH,  FOR EXAMPLE: "DTM:261752ZNOV". See table 2030 which is a composite

REFERENCE TIME INDICATOR.  ENTER REFERENCE TIME INDICATOR, FOR EXAMPLE: "D". See table 1258/1 which contains a list of data items and associated data codes
