Concept/Explanation/Examples

1

REQUESTING COMPONENT COMMAND

 

PROVIDES THE COMPONENT COMMAND THAT WILL RECEIVE THE SORTIES ALLOTTED BY THE JOINT FORCE COMMANDER.

 

 

1A

REQUESTING COMPONENT COMMAND

 

ENTER THE COMPONENT COMMAND THAT WILL RECEIVE THE SORTIES ALLOTTED BY THE JOINT FORCE COMMANDER, FOR EXAMPLE: "CCAIR IZMIR". See table 1037/4 which is an instructive entry

2

REQUEST IDENTIFIER

 

PROVIDES THE IDENTIFYING NUMBER ASSIGNED TO THE SPECIFIC MISSION REQUEST BY THE REQUESTING ORGANIZATION.

 

 

2A

REQUEST IDENTIFIER

 

ENTER THE IDENTIFYING NUMBER ASSIGNED TO THE SPECIFIC MISSION REQUEST BY THE REQUESTING ORGANIZATION, FOR EXAMPLE: "1F785I". See table 1012/144 which is an instructive entry

3

REQUEST STATUS INDICATOR

 

PROVIDES THE CODE THAT INDICATES WHETHER THE REQUEST WAS APPROVED OR DISAPPROVED.

 

 

3A

REQUEST STATUS INDICATOR, ABBREVIATED

 

ENTER THE CODE THAT INDICATES WHETHER THE REQUEST WAS APPROVED OR DISAPPROVED, FOR EXAMPLE: "APP". See table 1570/3 which contains a list of data items and associated data codes

4

MISSION NUMBER

 

PROVIDES THE MISSION NUMBER ASSIGNED TO THE REQUEST IF APPROVED.

 

 

4A

MISSION NUMBER

 

ENTER THE MISSION NUMBER ASSIGNED TO THE REQUEST IF APPROVED, FOR EXAMPLE: "0175D". See table 1012/69 which is an instructive entry

5

COMPLYING COMPONENT COMMAND

 

PROVIDES THE COMPONENT COMMAND OR UNIT THAT WILL PROVIDE THE SORTIES TASKED BY THE JOINT FORCE COMMANDER.

 

 

5A

COMPLYING COMPONENT COMMAND

 

ENTER THE COMPONENT COMMAND OR UNIT THAT WILL PROVIDE THE SORTIES TASKED BY THE JOINT FORCE COMMANDER, FOR EXAMPLE: "CVN 68 VA-165". See table 1037/5 which is an instructive entry

6

REVISED PRIORITY

 

PROVIDES THE REVISED PRIORITY NUMBER OR CODE ASSIGNED TO THE MISSION BY THE JOINT FORCE COMMANDER.

 

 

6A

REVISED PRIORITY, NUMBER

 

ENTER THE NUMBER FOR THE REVISED PRIORITY ASSIGNED TO THE MISSION BY THE JOINT FORCE COMMANDER, FOR EXAMPLE: "1". See table 1076/9 which contains a list of data items and associated data codes

 

 

6B

REVISED PRIORITY, CODE

 

ENTER THE CODE FOR THE REVISED PRIORITY ASSIGNED TO THE MISSION BY THE JOINT FORCE COMMANDER, FOR EXAMPLE: "1D". See table 2190 which is a composite

7

COMMENT

 

PROVIDES ADDITIONAL INFORMATION TO CLARIFY THE ALLOTMENT DATA.

 

 

7A

COMMENTS

 

ENTER ADDITIONAL INFORMATION TO CLARIFY THE ALLOTMENT DATA, FOR EXAMPLE: "SQUADRON IS OK". See table 1357/1 which is an instructive entry
