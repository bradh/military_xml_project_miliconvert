RECTANGULAR MOVING HAVEN DIMENSIONS, NM

 

 

 

 

3A

RECTANGULAR MOVING HAVEN DIMENSIONS, NM

 

ENTER THE DIMENSIONS OF THE RECTANGULAR MOVING HAVEN (MHN) IN NAUTICAL MILES. THIS IS MEASURED FROM A REFERENCE POINT, WHICH IS THE THE DATUM FOR ALL REFERENCES WITHIN THE MHN. THE BOX IS MEASURED FIRSTLY THE DISTANCE AHEAD OF THE REFERENCE POINT, FOLLOWED BY THE DISTANCE ASTERN AND EITHER SIDE. FOR EXAMPLE "20-30-5" DESCRIBES A BOX WHICH MEASURES 20 NAUTICAL MILES AHEAD OF THE REFERENCE POINT, 30 NAUTICAL MILES ASTERN AND 5 NAUTICAL MILES TO EITHER SIDE. See table 2214 which is a composite

Notes:   

THE ALPHABETIC IDENTIFIER ALLOWS THE DATA TO BE ANCHORED TO A PARTICULAR LEG OF THE VOYAGE. THE RECTANGULAR MOVING HAVEN IS A TECHNIQUE TAKEN FROM SUBMARINE OPERATIONS AND ALLOWS A VESSEL OR CONVOY TO PROCEED WITHIN A MOVING PIECE OF WATER.
