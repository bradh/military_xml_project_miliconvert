1

ICAO BASE CODE

 

 

 

 

1A

ICAO BASE CODE

 

A FOUR LETTER CODE DESIGNATING THE AIRFIELD WHERE THE PERSONNEL ARE LOCATED, FOR EXAMPLE: "LICT". See table 1125/2 which is an instructive entry

2

PERSONNEL QUALIFICATION DESIGNATOR

 

ENTER ONE OF THE FOLLOWING, REPEATING AS NECESSARY:

 

 

2A

PERSONNEL QUALIFICATION DESIGNATOR

 

ENTER THE CODE INDICATING THE SPECIALITY AREA TO BE REPORTED, FOR EXAMPLE: "ENG". See table 1286/6 which contains a list of data items and associated data codes

 

 

2B

FULLY MANNED STATUS

 

ENTER THE CODE INDICATING ALL SPECIALITY AREAS ARE AVAILABLE AND FULLY MANNED, FOR EXAMPLE: "FULLMAN". See table 1236/23 which contains a list of data items and associated data codes

3

NUMBER OF PERSONNEL QUALIFIED

 

 

 

 

3A

NUMBER OF PERSONNEL QUALIFIED

 

THE TOTAL NUMBER QUALIFIED IN THE SPECIALITY IDENTIFIED IN FIELD 2, FOR EXAMPLE: "29". See table 1044/226 a range [0 through 999]

4

NUMBER OF PERSONNEL UNAVAILABLE

 

 

 

 

4A

NUMBER OF PERSONNEL UNAVAILABLE

 

THE SHORTFALL OF PERSONNEL THAT ARE NOT AVAILABLE TO PERFORM THE SPECIALITY REPORTED IN FIELD 2, FOR EXAMPLE: "1". See table 1044/227 a range [0 through 999]

COMMENTS

 

PROVIDE DETAILS ON CAUSE/SOLUTION OF SHORTFALL, FOR EXAMPLE: "AT TRG COURSE AT EDNG". See table 1357/1 which is an instructive entry
