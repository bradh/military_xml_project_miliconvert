Field Desc

Concept/Explanation/Examples

1

RESIDUAL MISSION INDICATOR

 

 

 

 

1A

RESIDUAL MISSION INDICATOR

 

ENTER THE INDICATOR AS TO WHETHER THE MISSION BEGAN IN THE PREVIOUS ATO, IS ENTIRELY IN THIS ATO, OR WILL BE CARRIED OVER TO THE NEXT ATO, FOR EXAMPLE: "R". See table 1084/8 which contains a list of data items and associated data codes

2

MISSION NUMBER

 

 

 

 

2A

MISSION NUMBER

 

ENTER THE IDENTIFYING NUMBER ASSIGNED TO THE MISSION, FOR EXAMPLE: "X107". See table 1012/69 which is an instructive entry

3

AIRLIFT MISSION NUMBER OR EVENT NUMBER

 

THE AIRLIFT MISSION NUMBER OR EVENT NUMBER ASSIGNED TO THE AIRLIFT MISSION.

 

 

3A

AIRLIFT MISSION NUMBER

AMC

ENTER "AMC:" FOLLOWED BY  THE AIRLIFT MISSION NUMBER ASSIGNED TO THE AIRLIFT, FOR EXAMPLE: "AMC:JJXD123HA045". See table 1012/241 which is an instructive entry

 

 

3B

AIRCRAFT EVENT NUMBER

EVT

ENTER "EVT:" FOLLOWED BY THE NUMBER USED TO DESIGNATE THE AIRLIFT,  FOR EXAMPLE: "EVT:123" See table 1012/23 a range [001 through 999]

4

PACKAGE IDENTIFICATION

 

 

 

 

4A

PACKAGE IDENTIFICATION

 

ENTER THE PACKAGE IDENTIFIER THAT DESIGNATES THE  COMPOSITE FORCE PACKAGE TO WHICH THE MISSION IS ASSIGNED, FOR EXAMPLE: "ZZ". See table 1028/38 which is an instructive entry

5

MISSION COMMANDER DESIGNATOR

 

 

 

 

5A

MISSION COMMANDER DESIGNATOR

 

ENTER "MC" IF THE TASKED MISSION SERVES AS THE MISSION COMMANDER FOR THE PACKAGE IDENTIFIED IN FIELD 4. See table 1515/5 which contains a list of data items and associated data codes

6

COMPOSITE AIR OPERATIONS (COMAO) IDENTIFIER

 

 

 

 

6A

COMPOSITE AIR OPERATIONS (COMAO) IDENTIFIER

 

ENTER THE IDENTIFIER FOR THE COMPOSITE AIR OPERATIONS (COMAO), FOR EXAMPLE: "ABC-123". See table 1012/277 which is an instructive entry

7

PRIMARY AERIAL MISSION TYPE

 

 

 

 

7A

PRIMARY AERIAL MISSION TYPE

 

ENTER THE CODE FOR THE TYPE OF AIR MISSION TASKED, FOR EXAMPLE: "CAS". See table 1285/31 which contains a list of data items and associated data codes

8

SECONDARY AERIAL MISSION TYPE

 

IT PROVIDES THE CODE FOR THE ALTERNATE TYPE OF AIR MISSION TASKED.

 

 

8A

SECONDARY AERIAL MISSION TYPE

 

ENTER THE CODE FOR THE ALTERNATE TYPE OF AIR MISSION TASKED, FOR EXAMPLE: "SEAD". See table 1285/32 which contains a list of data items and associated data codes

9

AIRCRAFT ALERT STATUS

 

PROVIDES THE READINESS STATUS EXPRESSED AS AN INDICATION OF THE TIME FOR AN AIRCRAFT TO BE AIRBORNE AFTER THE LAUNCH ORDER IS RECEIVED. CAN ONLY BE USED IN CONJUNCTION WITH GROUND ALERT MISSION TYPES WITHIN THIS SET.

 

 

9A

AIRCRAFT ALERT STATUS, HOURS

 

IF REQUIRED, ENTER THE READINESS STATUS EXPRESSED AS AN INDICATION OF THE TIME IN HOURS FOR AN AIRCRAFT TO BE AIRBORNE AFTER THE LAUNCH ORDER IS RECEIVED, FOR EXAMPLE "12H". See table 2139 which is a composite

 

 

9B

AIRCRAFT ALERT STATUS, MINUTES

 

IF REQUIRED, ENTER THE READINESS STATUS EXPRESSED AS AN INDICATION OF THE TIME IN MINUTES FOR AN AIRCRAFT TO BE AIRBORNE AFTER THE LAUNCH ORDER IS RECEIVED, FOR EXAMPLE "30M". See table 2142 which is a composite

 

 

9C

AIRCRAFT ALERT CODE

 

ENTER THE APPROPRIATE READINESS STATUS CODE, FOR EXAMPLE "RED". See table 1798/2 which contains a list of data items and associated data codes

 

 

9D

AIRCRAFT ALERT READINESS STATE

 

ENTER THE AIRCRAFT ALERT READINESS STATE,"RS" FOLLOWED BY A 2-DIGIT NUMBER, FOR EXAMPLE: "RS10". See table 1798/3 which is an alphanumeric range

10

DEPARTURE LOCATION

 

THE GEOGRAPHIC LOCATION OF DEPARTURE OF THE MISSION.

 

 

10A

ICAO DEPARTURE LOCATION

DEPLOC

ENTER "DEPLOC:" FOLLOWED BY THE ICAO INDICATOR OF THE BASE FROM WHICH THE AIRCRAFT WILL DEPART, FOR EXAMPLE: "DEPLOC:KGZ6". See table 1125/4 which is an instructive entry

 

 

10B

DEPARTURE LOCATION NAME

DEPNAME

ENTER "DEPNAME:" FOLLOWED BY THE NAME OF THE DESIGNATED PLACE, FOR EXAMPLE: "DEPNAME:TUZLOC". See table 1022/84 which is an instructive entry

 

 

10C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

10D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

11

DEPARTURE DATE

 

 

 

 

11A

DAY-TIME AND MONTH OF DEPARTURE

 

ENTER THE DAY-TIME AND MONTH OF DEPARTURE FOR THE MISSION, FOR EXAMPLE: "281030ZFEB". See table 2030 which is a composite

12

RECOVERY LOCATION

 

THE GEOGRAPHIC LOCATION OF RECOVERY FOR THE MISSION.

 

 

12A

ICAO RECOVERY BASE

ARRLOC

ENTER "ARRLOC:" FOLLOWED BY THE ICAO INDICATOR OF THE BASE AT WHICH AIRCRAFT WILL LAND, FOR EXAMPLE: "ARRLOC:KDZ7". See table 1125/5 which is an instructive entry

 

 

12B

RECOVERY LOCATION NAME

ARRNAME

ENTER "ARRNAME:" FOLLOWED BY THE NAME OF DESIGNATED PLACE, FOR EXAMPLE: "ARRNAME:CV96". See table 1022/10 which is an instructive entry

 

 

12C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

12D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

13

RECOVERY DATE

 

 

 

 

13A

DAY-TIME AND MONTH OF RECOVERY

 

ENTER THE DAY-TIME AND MONTH OF RECOVERY, FOR EXAMPLE: "211045ZAPR". See table 2030 which is a composite
