No

Designator

Field Desc

Concept/Explanation/Examples

1

AIRCRAFT CONTROL AGENCY

 

 

 

 

1A

TYPE OF AIRCRAFT CONTROL AGENCY

 

ENTER THE CODE FOR THE TYPE OF CONTROL AGENCY, FOR EXAMPLE: "CRC". See table 1395/5 which contains a list of data items and associated data codes

2

CONTACT CALL SIGN OR NAME

 

PROVIDES THE NAME OR CALL SIGN OF THE CONTROL AGENCY.

 

 

2A

CONTACT CALL SIGN

CALL

ENTER "CALL:" FOLLOWED BY THE CALL SIGN WHICH IDENTIFIES  THE CONTROL AGENCY, FOR EXAMPLE: "CALL:INVERT". See table 1129/31 which is an instructive entry

 

 

2B

CONTACT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME WHICH IDENTIFIES  THE CONTROL AGENCY, FOR EXAMPLE: "NAME:RED CONTROL". See table 1022/7 which is an instructive entry

3

MISSION NUMBER

 

 

 

 

3A

MISSION NUMBER

 

ENTER THE MISSION NUMBER FOR THE CONTROL AGENCY IF APPLICABLE, FOR EXAMPLE: "MN001". See table 1012/69 which is an instructive entry

4

TACAN CHANNEL NUMBER

 

 

 

 

4A

TACAN CHANNEL WITH MODES

 

ENTER THE TACAN CHANNEL FOR THE CONTROL  AGENCY, FOR EXAMPLE: "10Y". See table 2103 which is a composite

5

DAY-TIME AVAILABLE FROM

 

 

 

 

5A

DAY-TIME AVAILABLE FROM

 

ENTER THE DAY-TIME THE CONTACT AGENCY IS AVAILABLE FOR CONTROL, FOR EXAMPLE: "010020Z". See table 2000 which is a composite

6

DAY-TIME AVAILABLE TO

 

CAN ONLY BE USED IN CONJUNCTION WITH DAY-TIME AVAILABLE FROM SPECIFIED IN THIS SET.

 

 

6A

DAY-TIME AVAILABLE TO

 

ENTER THE DAY-TIME THE CONTROL AGENCY IS  NO LONGER AVAILABLE  FOR CONTACT, FOR EXAMPLE: "020020Z". See table 2000 which is a composite

7

PRIMARY COMMUNICATION PARAMETERS

 

THE PRIMARY COMMUNICATION PARAMETERS FOR CONTACTING THE CONTROLLING AGENCY.

 

 

7A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY TO BE USED FOR CONTACTING THE CONTROL AGENCY, FOR EXAMPLE: "PFREQ:342.5". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

7B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR FOR CONTACTING THE CONTROL  AGENCY, FOR EXAMPLE:  "PDESIG:TAD001". See table 1396/2 which is an instructive entry

 

 

7C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE  PRIMARY HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER FOR CONTACTING THE CONTROL  AGENCY, FOR EXAMPLE:  "PHVQKM:12-333-01021999-MICKEY1-350.7MHZ". See table 2534 which is a composite

 

 

7D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE  PRIMARY HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER FOR CONTACTING THE CONTROL  AGENCY, FOR EXAMPLE:  "PHVQK:12-333-01021999". See table 2535 which is a composite

8

SECONDARY COMMUNICATION PARAMETERS

 

THE SECONDARY COMMUNICATION PARAMETERS FOR CONTACTING THE CONTROLLING AGENCY.

 

 

8A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE SECONDARY FREQUENCY TO BE USED FOR CONTACTING THE CONTROL AGENCY, FOR EXAMPLE: "SFREQ:350.5". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

8B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY DESIGNATOR FOR CONTACTING THE CONTROL  AGENCY, FOR EXAMPLE: "SDESIG:TAD002". See table 1396/3 which is an instructive entry

 

 

8C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

SHVQKM

ENTER "SHVQKM:" FOLLOWED BY THE  SECONDARY HAVE QUICK PARAMETERS WITH MICKEY NET FOR CONTACTING THE CONTROL  AGENCY, FOR EXAMPLE:  "SHVQKM:15-333-01021999-MICKEY1-350.7HZ". See table 2534 which is a composite

 

 

8D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

SHVQK

ENTER "SHVQK:" FOLLOWED BY THE SECONDARY HAVE QUICK PARAMETERS WITHOUT MICKEY NET FOR CONTACTING THE CONTROL  AGENCY, FOR EXAMPLE:  "SHVQK:15-333-01021999". See table 2535 which is a composite

9

LINK 16 CALL SIGN

 

 

 

 

9A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE LINK 16 ABBREVIATED CALL SIGN, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry

10

CONTACT NAME OR LOCATION

 

THE GEOGRAPHICAL LOCATION OF THE POINT AT WHICH THE AIRCRAFT SHOULD MAKE CONTACT WITH THE CONTROLLING AGENCY.

 

 

10A

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE LOCATION OF THE POINT AT WHICH AN AIRCRAFT IS TO CONTACT A CONTROL AGENCY, FOR EXAMPLE: "LOC:ALFA". See table 1022/4 which is an instructive entry

 

 

10B

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

10C

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

11

CONTACT ALTITUDE

 

CAN ONLY BE USED IN CONJUNCTION WITH CONTACT NAME OR LOCATION SPECIFIED IN THIS SET.

 

 

11A

ALTITUDE, HUNDREDS OF FEET

 

ENTER THE CONTACT ALTITUDE IN HUNDREDS OF FEET, FOR EXAMPLE: "075". See table 1255/14 a range [000 through 999]

12

CONTACT INSTRUCTIONS

 

 

 

 

12A

CONTACT INSTRUCTIONS

 

ENTER, AS A LITERAL, ANY CONTACT INSTRUCTIONS THAT MAY ALSO BE REQUIRED, FOR EXAMPLE: "THE SYSTEM MAY BE DOWN FOR MAINTENANCE BETWEEN 1200 AND 1300Z." See table 1822/4 which is an instructive entry
