SUBMARINE TYPE

 

 

 

 

1A

SUBMARINE TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE TYPE OF SUBMARINE INVOLVED IN THE INCIDENT, FOR EXAMPLE: "SSN". See table 1166/2 which contains a list of data items and associated data codes

2

DAY-TIME LAST COMMUNICATION

 

 

 

 

2A

DAY-TIME LAST COMMUNICATION

 

ENTER THE DAY-TIME OF THE LAST KNOWN COMMUNICATION WITH THE INCIDENT SUBMARINE, FOR EXAMPLE: "302323G". See table 2000 which is a composite

3

LAST KNOW LOCATION

 

THE GEOGRAPHIC LOCATION OF THE SUBMARINE LOST OR DISTRESSED AT THE LAST COMMUNICATION.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

4

LOCATION QUALIFIER

 

 

 

 

4A

LOCATION QUALIFIER

 

ENTER THE LOCATION QUALIFIER, FOR EXAMPLE: "AA". See table 1118/1 which contains a list of data items and associated data codes

FLAG

 

 

 

 

5A

GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE FOR THE NATION OF ORIGIN OF THE INCIDENT SUBMARINE, FOR EXAMPLE: "IT". See table 1026/1 which contains a list of data items and associated data codes

6

DEPTH

 

 

 

 

6A

DEPTH

 

ENTER THE DEPTH OF THE INCIDENT SUBMARINE, FOR EXAMPLE: "2000FT". See table 2058 which is a composite

COURSE AND SPEED

 

 

 

 

7A

COURSE IN DEGREES TRUE AND SPEED IN KNOTS

 

ENTER THE LAST KNOWN TRUE COURSE AND SPEED IN KNOTS OF THE INCIDENT SUBMARINE, FOR EXAMPLE: "170-17". See table 2025 which is a composite

COUNT OF PERSONNEL ON BOARD

POB
