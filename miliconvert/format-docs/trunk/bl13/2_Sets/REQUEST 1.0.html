REQUESTING AUTHORITY

 

 

 

 

1A

COMMAND ENGAGEMENT AUTHORITY

 

ENTER THE CODE REPRESENTING THE UNIT, COMMAND OR AGENCY MAKING THE REQUEST, FOR EXAMPLE: "A". See table 1028/50 which is an instructive entry

SERIAL NUMBER OF REQUEST

 

 

 

 

2A

SERIAL NUMBER OF REQUEST

 

ENTER THE NUMBER USED TO IDENTIFY THE REQUEST, FOR EXAMPLE: "34". See table 1012/68 a range [1 through 999]

3

TYPE OF ASSET

 

SPECIFIES THE TYPE OF ASSET REQUESTED.

 

 

3A

AIRCRAFT TYPE

ACTYP

ENTER FIELD DESCRIPTOR "ACTYP:" FOLLOWED BY THE AIRCRAFT TYPE, FOR EXAMPLE: "ACTYP:C-130". See table 1015/1 which is an instructive entry

 

 

3B

VESSEL TYPE

SHIPTYP

ENTER FIELD DESCRIPTOR "SHIPTYP:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE VESSEL TYPE, FOR EXAMPLE: "SHIPTYP:DDG". See table 1166/1 which contains a list of data items and associated data codes

 

 

3C

EQUIPMENT TYPE

EQPMT

ENTER FIELD DESCRIPTOR "EQPMT:" FOLLOWED BY THE EQUIPMENT TYPE, FOR EXAMPLE: "EQPMT:VULCAN". See table 1135/1 which is an instructive entry

 

 

3D

TYPE OF SUPPORT ASSETS

ASSETYP

ENTER FIELD DESCRIPTOR "ASSETYP:" FOLLOWED BY THE TYPE OF SUPPORT ASSETS REQUESTED, FOR EXAMPLE: "ASSETYP:PLATOON". See table 1317/6 which is an instructive entry

4

NUMBER REQUESTED

 

 

 

 

4A

NUMBER REQUESTED

 

ENTER THE NUMBER OF ASSETS REQUESTED, FOR EXAMPLE: "44". See table 1044/134 a range [0 through 9999]
