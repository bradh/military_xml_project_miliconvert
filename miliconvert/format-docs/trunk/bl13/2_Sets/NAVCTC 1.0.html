No

Designator

Field Desc

Concept/Explanation/Examples

1

CONTACT NUMBER

 

PROVIDES THE NUMBER ASSIGNED TO THE NAVAL CONTACT.

 

 

1A

LINK 11 11B TRACK NUMBER

DLRN

ENTER "DLRN:" FOLLOWED BY THE DATA LINK REFERENCE NUMBER ASSIGNED TO THE CONTACT, FOR EXAMPLE: "DLRN:0045". See table 1625/19 which is an alphanumeric range

 

 

1B

COMMON NATO TRACK NUMBER

NTN

ENTER "NTN:" FOLLOWED BY THE NATO TRACK NUMBER ASSIGNED TO THE CONTACT, FOR EXAMPLE: "NTN:AB123". See table 1625/5 which is an instructive entry

 

 

1C

LINK 16 TRACK NUMBER

LNK

ENTER "LNK:" FOLLOWED BY THE LINK 16 TRACK NUMBER ASSIGNED TO THE CONTACT, FOR EXAMPLE: "LNK:A0001". See table 1625/26 which is an alphanumeric range

2

SHIP CLASS OR NAME

 

PROVIDES THE NAME OF THE SHIP OR VESSEL CLASS REPRESENTED BY THE CONTACT.

 

 

2A

SHIP NAME

NAME

ENTER "NAME:" FOLLOWED BY THE SHIP NAME, FOR EXAMPLE: "NAME:MOSKVA". See table 1022/49 which is an instructive entry

 

 

2B

SHIP CLASS NAME

CLASS

ENTER "CLASS:" FOLLOWED BY THE CLASS OF THE SHIP, FOR EXAMPLE: "CLASS:ROOSKI". See table 1022/227 which is an instructive entry

3

COUNTRY

 

 

 

 

3A

GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE OF THE REPORTED CONTACT, FOR EXAMPLE: "RS". See table 1026/1 which contains a list of data items and associated data codes
