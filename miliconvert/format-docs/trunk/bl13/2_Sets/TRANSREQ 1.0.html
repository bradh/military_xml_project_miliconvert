1

MISSION NUMBER

 

 

 

 

1A

MISSION NUMBER

 

ENTER THE IDENTIFYING NUMBER ASSIGNED TO THE MISSION, FOR EXAMPLE: "D123HA". See table 1012/69 which is an instructive entry

2

TRANSPORTATION REQUIREMENT

 

PROVIDES THE TRANSPORTATION REQUIREMENT INFORMATION FOR THE CARGO AND/OR PASSENGERS TO BE ONLOADED.

 

 

2A

TRANSPORTATION REQUIREMENT IDENTIFIER

ID

ENTER "ID:" FOLLOWED BY THE DESIGNATION USED TO IDENTIFY A TRANSPORTING REQUIREMENT, FOR EXAMPLE: "ID:A12A1". See table 1102/11 which is an instructive entry

 

 

2B

UNIT LINE NUMBER

ULN

ENTER "ULN:" FOLLOWED BY THE UNIT LINE NUMBER, FOR EXAMPLE: "ULN:AZ12345". See table 1012/246 which is an instructive entry

 

 

2C

UNIT IDENTIFICATION CODE, SPECIFIC

UIC

ENTER "UIC:" FOLLOWED BY THE UNIQUE IDENTIFICATION OF A UNIT LEVEL ORGANISATION, FOR EXAMPLE: "UIC:F12345". See table 1028/39 which is an instructive entry

 

 

2D

PERSONNEL INCREMENT NUMBER

PIN

ENTER "PIN:" FOLLOWED BY THE IDENTIFIER THAT UNIQUELY DEFINES EACH NON UNIT PERSONNEL RECORD, FOR EXAMPLE: "PIN:AZ12345". See table 1012/251 which is an instructive entry

 

 

2E

CARGO INCREMENT NUMBER

CIN

ENTER "CIN:" FOLLOWED BY THE IDENTIFIER THAT UNIQUELY DEFINES EACH NON UNIT CARGO RECORD, FOR EXAMPLE: "CIN:AZ12345". See table 1012/252 which is an instructive entry

3

ONLOAD LEG REFERENCE NUMBER

 

 

 

 

3A

ONLOAD LEG REFERENCE NUMBER

 

ENTER THE ONLOAD LEG REFERENCE NUMBER, FOR EXAMPLE: "A1". See table 1012/249 which is an instructive entry

4

OFFLOAD LEG REFERENCE NUMBER

 

 

 

 

4A

OFFLOAD LEG REFERENCE NUMBER

 

ENTER THE OFFLOAD LEG REFERENCE NUMBER, FOR EXAMPLE: "A2". See table 1012/250 which is an instructive entry

5

PASSENGERS NUMBER

 

 

 

 

5A

NUMBER OF PASSENGERS

 

ENTER THE TOTAL NUMBER OF PASSENGERS, FOR EXAMPLE: "235". See table 1044/88 a range [1 through 9999]

6

BULK CARGO WEIGHT

 

PROVIDES THE WEIGHT OF THE BULK CARGO, THAT IS GENERALLY SHIPPED IN VOLUME WHERE THE TRANSPORTATION CONVEYANCE IS THE ONLY EXTERNAL CONTAINER, SUCH AS LIQUIDS, ORE OR GRAIN.

 

 

6A

BULK CARGO WEIGHT IN SHORT TONS IN TENTHS

STON

ENTER "STON:" FOLLOWED BY THE WEIGHT OF CARGO MEASURED IN SHORT TONS TO THE NEAREST TENTH, FOR EXAMPLE: "STON:25.2". See table 1800/2 a range [0.1 through 999.9] (1 to 1 decimal places)

 

 

6B

BULK CARGO WEIGHT IN TONS IN TENTHS

TON

ENTER "TON:" FOLLOWED BY THE WEIGHT OF CARGO MEASURED IN TONS TO THE NEAREST TENTH, FOR EXAMPLE: "TON:21.5". See table 1440/5 a range [0.1 through 999.9] (1 to 1 decimal places)

7

HOIST REQUIREMENT INDICATOR

 

 

 

 

7A

HOIST REQUIREMENT INDICATOR

 

ENTER AN INDICATOR OF WHETHER A REQUIREMENT EXISTS FOR A HOIST, FOR EXAMPLE: "NO". See table 1027/52 which contains a list of data items and associated data codes

8

TYPE OF SUPPORT EQUIPMENT

 

 

 

 

8A

TYPE OF SUPPORT EQUIPMENT

 

ENTER THE TYPE OF SUPPORT EQUIPMENT REQUIRED TO ONLOAD OR OFFLOAD CARGO FROM THE AIRCRAFT, FOR EXAMPLE: "25K LOADER". See table 1135/9 which is an instructive entry

9

HAZARDOUS CARGO DESIGNATOR

 

 

 

 

9A

HAZARDOUS CARGO DESIGNATOR, ABBREVIATED

 

ENTER THE APPROPRIATE REFERENCE NUMBER THAT INDICATES THE TYPE OF HAZARD ASSOCIATED WITH THIS CARGO AND ACKNOWLEDGES THAT THE APPROPRIATE PRECAUTIONS HAVE BEEN TAKEN, FOR EXAMPLE: "A12.1". See table 1221/3 which is an instructive entry

10

SINGLE DAGGER REQUIRED

 

 

 

 

10A

SINGLE DAGGER REQUIRED

 

ENTER THE CODE TO INDICATE IF A SINGLE DAGGER REQUIREMENT EXISTS FOR CARGO REQUIRING SPECIAL HANDLING, FOR EXAMPLE: "Y". See table 1359/22 which contains a list of data items and associated data codes

11

NET EXPLOSIVE WEIGHT

 

 

 

 

11A

NET EXPLOSIVE WEIGHT

 

ENTER THE NET WEIGHT OF THE CARGO THAT IS EXPLOSIVE, FOR EXAMPLE: "1000LB". See table 2487 which is a composite

12

OVERSIZE CARGO WEIGHT

 

PROVIDES THE WEIGHT OF THE OVERSIZE CARGO, THAT EXCEEDS THE USABLE DIMENSION OF A 436L PALLET, 104" X 84" X 96", OR A HEIGHT SET BY THE PARTICULAR MODEL AIRCRAFT.

 

 

12A

OVERSIZE CARGO WEIGHT IN SHORT TONS IN TENTHS

STON

ENTER "STON:" FOLLOWED BY THE OVERSIZE CARGO WEIGHT MEASURED IN SHORT TONS TO THE NEAREST TENTH, FOR EXAMPLE: "STON:2.0". See table 1800/3 a range [0.1 through 999.9] (1 to 1 decimal places)

 

 

12B

OVERSIZE CARGO WEIGHT IN TONS IN TENTHS

TON

ENTER "TON:" FOLLOWED BY THE OVERSIZE CARGO WEIGHT MEASURED IN TONS TO THE NEAREST TENTH, FOR EXAMPLE: "TON:12.4". See table 1440/3 a range [0.1 through 999.9] (1 to 1 decimal places)

13

OUTSIZE CARGO WEIGHT

 

PROVIDES THE WEIGHT OF THE OUTSIZE CARGO, THAT EXCEEDS 1,090" X 117" X 105".

 

 

13A

OUTSIZE CARGO WEIGHT IN SHORT TONS IN TENTHS

STON

ENTER "STON:" FOLLOWED BY THE OUTSIZE CARGO WEIGHT MEASURED IN SHORT TONS TO THE NEAREST TENTH, FOR EXAMPLE: "STON:10.5". See table 1800/4 a range [0.1 through 999.9] (1 to 1 decimal places)

 

 

13B

OUTSIZE CARGO WEIGHT IN TONS IN TENTHS

TON

ENTER "TON:" FOLLOWED BY THE OUTSIZE CARGO WEIGHT MEASURED IN TONS TO THE NEAREST TENTH, FOR EXAMPLE: "TON:20.5". See table 1440/4 a range [0.1 through 999.9] (1 to 1 decimal places)

14

15

AMBULATORY PATIENTS NUMBER

 

 

 

 

15A

NUMBER OF AMBULATORY PATIENTS

 

ENTER THE NUMBER OF AMBULATORY PATIENTS, FOR EXAMPLE: "45". See table 1044/326 a range [0 through 999]

16

ATTENDANTS NUMBER

 

 

 

 

16A

NUMBER OF ATTENDANTS

 

ENTER THE NUMBER OF ATTENDANTS FOR THE AMBULATORY, FOR EXAMPLE: "3". See table 1044/327 a range [0 through 999]

17

CONTACT NAME

 

 

 

 

17A

CONTACT NAME

 

ENTER THE NAME OF THE PERSON OR ORGANISATION TO BE CONTACTED, FOR EXAMPLE: "DOCTOR PATCH ADAMS". See table 1022/7 which is an instructive entry
