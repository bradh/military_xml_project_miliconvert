STANDARD EMITTER DESIGNATION

 

ENTER THE LITERAL NICKNAME OF INTERCEPTED EMITTER, FOR EXAMPLE: "SMALL BEAR". See table 1121/4 which is an instructive entry

2A

CREDIBILITY

 

ENTER THE LEVEL OF OPERATOR'S CONFIDENCE IN THE ACCURACY OF EMITTER EVALUATION, FOR EXAMPLE: "LOW". See table 1461/1 which contains a list of data items and associated data codes

3

ELINT NOTATION OR SORTING CODE

 

PROVIDES THE ELINT NOTATION OR SORTING CODE IN ONE OF THE FOLLOWING WAYS:

 

 

3A

ELINT NOTATION

 

ENTER A 5-CHARACTER ELINT NOTATION FOR NON-COMMUNICATIONS ELECTRONIC EMISSIONS RELATED TO SUCH EQUIPMENTS, FOR EXAMPLE: "FG23R". See table 1012/34 which is an instructive entry

 

 

3B

SORTING CODE

 

ENTER A 4-CHARACTER CODE WHICH FACILITATES REPORTING EQUIPMENT TYPE THAT HAS NO ASSOCIATED ELINT NOTATION, FOR EXAMPLE: "F34R". See table 1012/76 which is an instructive entry

4

RADIO/RADAR FUNCTION

 

 

 

 

4A

RADIO RADAR FUNCTION

 

ENTER THE CODE SPECIFYING THE RADIO/RADAR FUNCTION, FOR EXAMPLE: "AQ". See table 1471/1 which contains a list of data items and associated data codes

5

CRYSTAL FREQUENCY

 

 

 

 

5A

CRYSTAL FREQUENCY

 

ENTER THE CRYSTAL FREQUENCY, EXPRESSED AS ONE TO EIGHT NUMERALS PLUS "HZ", "KHZ", "MHZ" OR "GHZ", DECIMAL ALLOWED, FOR EXAMPLE: "1234.5KHZ". See table 2064 which is a composite

6

CRYSTAL COUNTDOWN

 

 

 

 

6A

CRYSTAL COUNTDOWN

 

ENTER THE CHARACTERISTICS OF SOME RADARS RELATED TO PULSE REPETITION INTERVAL, EXPRESSED BY ONE TO THREE NUMERICS, FOR EXAMPLE: "25". See table 1472/1 a range [0 through 999]
