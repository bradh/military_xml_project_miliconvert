No

Designator

Field Desc

Concept/Explanation/Examples

1

AIRCRAFT CALL SIGN

 

PROVIDES THE CALL SIGN OF THE AIRCRAFT INVOLVED IN THE AIR MISSION.

 

 

1A

CONTACT CALL SIGN

 

ENTER THE AIRCRAFT CALL SIGN, FOR EXAMPLE "RICKY ONE". See table 1129/31 which is an instructive entry

2

AIRCRAFT BY TYPE, MODEL OR CODE

 

PROVIDES THE TYPE, MODEL OR CODE OF THE AIRCRAFT INVOLVED IN THE AIR MISSION REPORTED.

 

 

2A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE OF AIRCRAFT EXECUTING THE AIR MISSION, FOR EXAMPLE "ACTYP:FA18D". See table 1015/20 which contains a list of data items and associated data codes

 

 

2B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF THE AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST, FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

3

UNIT DESIGNATOR

 

PROVIDES THE DESIGNATOR OF THE UNIT TO WHICH THE AIRCRAFT INVOLVED IN THE AIR MISSION BELONG.

 

 

3A

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 2 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "UIC2:FRJ123456". See table 2243 which is a composite

 

 

3B

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 3 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "UIC3:FRAJ123456". See table 2241 which is a composite

 

 

3C

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OF THE UNIT CORRESPONDING TO THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "UNIT:GRUPO 12". See table 1028/43 which is an instructive entry

4

AIRCRAFT IDENTIFICATION NUMBER

 

PROVIDES THE TAIL OR SIDE IDENTIFICATION NUMBER OF THE AIRCRAFT INVOLVED IN THE AIR MISSION.

 

 

4A

AIRCRAFT IDENTIFICATION NUMBER

TAIL

ENTER "TAIL:" FOLLOWED BY THE AIRCRAFT TAIL IDENTIFICATION NUMBER, FOR EXAMPLE: "TAIL:MC28361". See table 1012/84 which is an instructive entry

 

 

4B

AIRCRAFT SIDE NUMBER

SIDE

ENTER "SIDE:" FOLLOWED BY THE SIDE IDENTIFICATION NUMBER OF THE AIRCRAFT, FOR EXAMPLE: "SIDE:F164". See table 1012/92 which is an instructive entry

5

PRIMARY CONFIGURATION LOAD

 

PROVIDES THE PRIMARY ORDNANCE MIX CARRIED ON THE MISSION AIRCRAFT.

 

 

5A

PRIMARY CONFIGURATION CODE

 

ENTER THE PRIMARY CONFIGURATION CODE FOR THE AIRCRAFT INVOLVED IN THE AIR MISSION, FOR EXAMPLE: "2HARM". See table 1012/264 which is an instructive entry

6

SECONDARY CONFIGURATION LOAD

 

IF USED, PROVIDES THE SECONDARY ORDNANCE MIX CARRIED ON THE MISSION AIRCRAFT.

 

 

6A

SECONDARY CONFIGURATION CODE

 

ENTER THE SECONDARY CONFIGURATION CODE FOR THE AIRCRAFT INVOLVED IN THE AIR MISSION, FOR EXAMPLE: "2S2WG". See table 1012/265 which is an instructive entry

7

TANKER SUPPORT RECEIVED

 

IF USED, PROVIDES INDICATION WHETHER THE AIR MISSION REPORTED HAS RECEIVED OR NOT SUPPORT BY A TANKER AIRCRAFT.

 

 

7A

TANKER SUPPORT RECEIVED INDICATOR

 

ENTER THE INDICATION WHETHER THE TANKER SUPPORT WAS RECEIVED BY THE AIRCRAFT INVOLVED IN THE AIR MISSION, FOR EXAMPLE "YES". See table 1027/86 which contains a list of data items and associated data codes

8

INFLIGHT REPORT FILED

 

PROVIDES INDICATION WHETHER THE INFLIGHT REPORT OF THE AIR MISSION WAS FILED OR NOT.

 

 

8A

INFLIGHT REPORT FILED INDICATOR

 

ENTER THE INDICATION WHETHER THE INFLIGHT REPORT OF THE AIR MISSION WAS FILED OR NOT, FOR EXAMPLE "NO". See table 1027/87 which contains a list of data items and associated data codes

9

END OF DEBRIEF DATE-TIME GROUP

 

PROVIDES THE DAY-TIME OF END OF THE DEBRIEFING OF THE AIR MISSION REPORTED.

 

 

9A

END OF DEBRIEF DTG

 

ENTER THE DAY-TIME OF END OF THE DEBRIEFING, FOR EXAMPLE: "121930ZMAY2006". See table 2033 which is a composite

10

ICAO DEPARTURE LOCATION

 

PROVIDES THE DEPARTURE LOCATION OF THE AIR MISSION REPORTED, USING THE ICAO BASE CODE, THE LOCATION NAME OR ITS GEOGRAPHICAL COORDINATES.

 

 

10A

ICAO DEPARTURE LOCATION

LOC

ENTER "LOC:" FOLLOWED BY THE ICAO INDICATOR OF THE BASE AT WHICH THE AIRCRAFT DEPARTED, FOR EXAMPLE: "LOC:KGZ6". See table 1125/4 which is an instructive entry

 

 

10B

DEPARTURE LOCATION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE DESIGNATED PLACE OF DEPARTURE OF THE AIR MISSION, FOR EXAMPLE: "NAME:TUZLOC". See table 1022/84 which is an instructive entry

 

 

10C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

10D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

11

DTG OF DEPARTURE

 

PROVIDES THE ACTUAL DATE-TIME OF DEPARTURE OF THE AIR MISSION REPORTED.

 

 

11A

DTG OF DEPARTURE

 

ENTER THE ACTUAL DATE-TIME OF DEPARTURE OF THE AIR  MISSION REPORTED, FOR EXAMPLE: "281030ZFEB2006". See table 2033 which is a composite

12

ICAO RECOVERY BASE

 

PROVIDES THE ARRIVAL LOCATION OF THE AIR MISSION REPORTED, USING THE ICAO BASE CODE, THE LOCATION NAME OR ITS GEOGRAPHICAL COORDINATES.

 

 

12A

ICAO RECOVERY BASE

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO INDICATOR OF THE BASE AT WHICH AIRCRAFT HAS LANDED, FOR EXAMPLE: "ICAO:KDZ7". See table 1125/5 which is an instructive entry

 

 

12B

RECOVERY LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME OF DESIGNATED PLACE, FOR EXAMPLE: "LOC:CV96". See table 1022/10 which is an instructive entry

 

 

12C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

12D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

13

DTG OF ARRIVAL

 

PROVIDES THE ACTUAL DAY-TIME OF ARRIVAL OF THE AIR MISSION REPORTED.

 

 

13A

DTG OF ARRIVAL

 

ENTER THE ACTUAL DATE-TIME OF ARRIVAL OF THE AIR  MISSION REPORTED, FOR EXAMPLE: "281115ZFEB2006". See table 2033 which is a composite
