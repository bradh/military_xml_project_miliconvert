Concept/Explanation/Examples

1

SERIAL NUMBER

 

 

 

 

1A

SERIAL NUMBER

 

ENTER THE NUMBER OF THIS SPECIFIC DEPLOYMENT AS IT IS FIXED IN THE ALLIED DISPOSITION LIST (ADL), FOR EXAMPLE: "132". See table 1012/130 a range [001 through 999]

2

UNIT NAME

 

 

 

 

2A

UNIT NAME

 

ENTER THE NAME OF UNIT, WHICH HAS STARTED THE PLANNED DEPLOYMENT, FOR EXAMPLE: "4TH (UK) ARM BDE". See table 1022/48 which is an instructive entry

3

ARRIVAL LOCATION

 

THE GEOGRAPHIC LOCATION OF ARRIVAL.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

3C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

3D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

3E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

3F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

3G

ARRIVAL PLACE NAME

APN

ENTER "APN" FOLLOWED BY THE NAME OF THE ARRIVAL LOCATION, FOR EXAMPLE: "APN:DORTMUND". See table 1022/41 which is an instructive entry

4

ESTIMATED DAY-TIME AND MONTH OF ARRIVAL

 

 

 

 

4A

ESTIMATED DAY-TIME AND MONTH OF ARRIVAL

 

ENTER THE ESTIMATED DAY (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE INDICATOR AND MONTH OF ARRIVAL (ETA) AT THE LOCATION IN FIELD 3, FOR EXAMPLE: "261815ZMAR". See table 2030 which is a composite

5

DAY-TIME AND MONTH OF ARRIVAL

 

 

 

 

5A

DAY-TIME AND MONTH OF ARRIVAL

 

ENTER THE ACTUAL DAY (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE INDICATOR AND MONTH OF ARRIVAL (ATA) AT THE LOCATION IN FIELD 3, FOR EXAMPLE: "261900ZMAR". See table 2030 which is a composite

6

REASON FOR DELAY

 

 

 

 

6A

REASON FOR DELAY

 

INDICATE THE REASON FOR DELAY, FOR EXAMPLE: "WEATHER".  USE ONLY WHEN THERE ARE DIFFERENCES BETWEEN ETA AND ATA. See table 1318/2 which contains a list of data items and associated data codes

7

MODE OF TRANSPORTATION

 

 

 

 

7A

MODE OF TRANSPORTATION

 

INDICATE THE MODE OF TRANSPORTATION, FOR EXAMPLE: "RAIL". See table 1269/1 which contains a list of data items and associated data codes
