UNIT IDENTIFICATION

 

PROVIDES THE IDENTIFICATION OF THE UNIT.

 

 

1A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UNIT WHICH WILL FULFILL THE AEW REQUEST, FOR EXAMPLE: "UIC:USF2141". See table 2141 which is a composite

 

 

1B

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE DESIGNATOR OF THE UNIT WHICH WILL FULFILL THE AEW REQUEST, FOR EXAMPLE: "UNIT:552 AWACS". See table 1028/43 which is an instructive entry

2

AEW AIRCRAFT

 

 

 

 

2A

AEW AIRCRAFT

 

ENTER THE TYPE OF AEW AIRCRAFT THAT WILL PARTICIPATE IN THE MISSION, FOR EXAMPLE: "E3A". See table 1015/14 which contains a list of data items and associated data codes

AIRCRAFT CALL SIGN

 

 

 

 

3A

AIRCRAFT CALL SIGN

 

ENTER THE CALL SIGN FOR THE AIRCRAFT FLIGHT DECK, FOR EXAMPLE: "LOOKER 01". See table 1129/9 which is an instructive entry

MISSION CALL SIGN

 

 

 

 

4A

MISSION CALL SIGN

 

ENTER THE CALL SIGN FOR THE MISSION, FOR EXAMPLE: "SCREEN 01". See table 1129/21 which is an instructive entry

5

DEPARTURE LOCATION

 

PROVIDES THE DEPARTURE LOCATION FOR THE AEW AIRCRAFT.

 

 

5A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE DEPARTURE LOCATION CODE, FOR EXAMPLE: "UIC:GEF1234". See table 2141 which is a composite

 

 

5B

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE DEPARTURE LOCATION ICAO CODE, FOR EXAMPLE: "ICAO:KILA". See table 1125/2 which is an instructive entry

 

 

5C

LOCATION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE PLACE NAME OF THE DEPARTURE LOCATION, FOR EXAMPLE: "NAME:HOME FIELD". See table 1022/4 which is an instructive entry

7B

ICAO BASE CODE

ICAO

7C

LOCATION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE LANDING LOCATION NAME, FOR EXAMPLE: "NAME:AWAY FIELD". See table 1022/4 which is an instructive entry

ARRIVAL TIME
