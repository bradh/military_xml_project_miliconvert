1

COMMAND AND CONTROL UNIT IDENTIFIER

 

THE IDENTIFIER OF THE COMMAND AND CONTROL UNIT BEING REPORTED.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT IDENTIFIER OF THE COMMAND AND CONTROL UNIT BEING REPORTED, FOR EXAMPLE: "UNIT:KING42". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY ENTER THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 2 LETTER COUNTRY CODE), OF THE COMMAND AND CONTROL UNIT BEING REPORTED, FOR EXAMPLE: "UIC2:UKJ12345". See table 2243 which is a composite

 

 

1C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 3 LETTER COUNTRY CODE), OF THE COMMAND AND CONTROL UNIT BEING REPORTED, FOR EXAMPLE: "UIC3:GBRJ12345". See table 2241 which is a composite

2

COMMAND AND CONTROL UNIT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE COMMAND AND CONTROL UNIT BEING REPORTED.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR, FOR EXAMPLE: "ICAO:KX4Q". See table 1125/2 which is an instructive entry

 

 

2H

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE LOCATION NAME, FOR EXAMPLE "LOC:DECOPRO". See table 1022/4 which is an instructive entry

3

UNIT OPERATIONAL STATUS

 

PROVIDES INFORMATION ABOUT THE ABILITY OF THE COMMAND AND CONTROL UNIT TO SUPPORT ITS ASSIGNED OPERATIONS.  FOR UNIT OPERATIONAL STATUS OTHER THAN "OPR" AN ESTIMATED DATE TIME IN COMMISSION AND A NEXT OPERATIONAL STATUS MUST BE SPECIFIED IN THIS SET.

 

 

3A

UNIT OPERATIONAL STATUS

 

ENTER THE OPERATIONAL STATUS FOR THE COMMAND AND CONTROL UNIT BEING REPORTED, FOR EXAMPLE: "LOP". See table 1236/34 which contains a list of data items and associated data codes

4

ESTIMATED DATE TIME IN COMMISSION

 

PROVIDES THE ESTIMATED DATE TIME WHEN THE OPERATIONAL STATUS OF THE COMMAND AND CONTROL UNIT BEING REPORTED WILL CHANGE.

 

 

4A

ESTIMATED DTG CURRENT STATUS EXPECTED TO END

 

ENTER THE ESTIMATED DATE TIME WHEN THE OPERATIONAL STATUS OF THE COMMAND AND CONTROL UNIT BEING REPORTED WILL CHANGE, FOR EXAMPLE: "251325ZJAN2006". See table 2033 which is a composite

5

NEXT OPERATIONAL STATUS

 

PROVIDES THE ANTICIPATED OPERATIONAL STATUS THAT THE COMMAND AND CONTROL UNIT WILL ASSUME AFTER THE CURRENT.

 

 

5A

NEXT BASE OPERATIONAL STATUS

 

ENTER THE NEXT OPERATIONAL STATUS THE BASE/UNIT IS EXPECTED TO HAVE. FOR EXAMPLE: "OPR". See table 1236/39 which contains a list of data items and associated data codes

6

TYPE OF COMMAND AND CONTROL AGENCY

 

PROVIDES INFORMATION ABOUT THE FUNCTIONS AND TYPE OF THE COMMAND AND CONTROL UNIT BEING REPORTED.

 

 

6A

TYPE OF COMMAND AND CONTROL AGENCY

 

ENTER THE COMAND AND CONTROL TYPE  FOR THE BASE/UNIT. FOR EXAMPLE: "ACC". See table 1874/4 which contains a list of data items and associated data codes

7

COMMAND AND CONTROL MODE

 

PROVIDES THE MODE OF ACTIVATION OF THE COMMAND AND CONTROL UNIT BEING REPORTED.

 

 

7A

COMMAND AND CONTROL MODE

 

ENTER THE COMMAND AND CONTROL MODE FOR THE BASE/UNIT. FOR EXAMPLE: "BAC". See table 1874/2 which contains a list of data items and associated data codes
