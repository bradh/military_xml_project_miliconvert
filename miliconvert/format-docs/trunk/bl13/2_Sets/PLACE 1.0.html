Notes:   

FIELD 2 INSTALLATION OR LOCATION IS USED TO REPORT AN INSTALLATION, FACILITY OR A POINT ON A MAP THAT IS NOT RELATED TO A SPECIFIC OWN UNIT, BUT IS A FEATURE THAT IS OF INTEREST TO A COMMANDER (E.G. ABANDONED AIRFIELD, CROSSROADS ON A MAP ETC.).  FOR INSTALLATIONS OR LOCATIONS RELATED TO UNITS SEE SET "LOCATION".
