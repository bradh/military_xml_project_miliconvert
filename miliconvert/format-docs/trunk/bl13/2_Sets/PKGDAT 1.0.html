Designator

Field Desc

Concept/Explanation/Examples

1

UNIT IDENTIFICATION

 

PROVIDES THE UNIT ASSIGNED TO THE PACKAGE.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE DESIGNATOR OF THE UNIT TASKED TO PERFORM THE SPECIFIC MISSION IN THE PACKAGE, FOR EXAMPLE: "UNIT:123 FS". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UIC OF THE UNIT TASKED TO PERFORM THE SPECIFIC MISSION IN THE PACKAGE, FOR EXAMPLE "UIC:USF2141". See table 2141 which is a composite

2

MISSION NUMBER

 

 

 

 

2A

MISSION NUMBER

 

ENTER THE IDENTIFYING NUMBER ASSIGNED TO THE SPECIFIC MISSION, FOR EXAMPLE: "123DAH". See table 1012/69 which is an instructive entry

3

PRIMARY AERIAL MISSION TYPE

 

 

 

 

3A

PRIMARY AERIAL MISSION TYPE

 

ENTER THE CODE FOR THE MISSION TYPE OF THE TASKED AIR MISSION, FOR EXAMPLE: "CAS". See table 1285/31 which contains a list of data items and associated data codes

4

AIRCRAFT NUMBER

 

 

 

 

4A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT PARTICIPATING IN THE PARTICULAR MISSION, FOR EXAMPLE: "4". See table 1044/322 a range [1 through 99]

5

AIRCRAFT TYPE

 

PROVIDES THE TYPE OF AIRCRAFT ASSIGNED TO THE PACKAGE.

 

 

5A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE AND MODEL NUMBER OF THE AIRCRAFT, FOR EXAMPLE: "ACTYP:FA18". See table 1015/20 which contains a list of data items and associated data codes

 

 

5B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF AN AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST, FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

6

CALL SIGN

 

 

 

 

6A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN ASSIGNED TO MISSION AIRCRAFT, FOR EXAMPLE: "HAMMER 12". See table 1129/31 which is an instructive entry
