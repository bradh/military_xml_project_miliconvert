1

SURFACE-TO-AIR WEAPON UNIT IDENTIFIER

 

THE IDENTIFICATION OF THE SURFACE-TO-AIR WEAPON UNIT BEING REPORTED.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE IDENTIFIER OF THE SURFACE-TO-AIR WEAPON UNIT BEING REPORTED, FOR EXAMPLE: "UNIT:ATLAS11". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 2 LETTER COUNTRY CODE), OF THE UNIT BEING REPORTED, FOR EXAMPLE: "UIC2:FRJ12345". See table 2243 which is a composite

 

 

1C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 3 LETTER COUNTRY CODE), OF THE UNIT BEING REPORTED, FOR EXAMPLE: "UIC3:FRAJ12345". See table 2241 which is a composite

SURFACE-TO-AIR WEAPON UNIT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE SURFACE-TO-AIR WEAPON UNIT.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO BASE CODE, FOR EXAMPLE "ICAO:LIPZ". See table 1125/2 which is an instructive entry

 

 

2H

LOCATION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE LOCATION NAME, FOR EXAMPLE "NAME:DECOPRO". See table 1022/4 which is an instructive entry

3

ORGANIZATION TYPE

 

THE TYPE OF ORGANIZATION TO WHICH THE SURFACE-TO-AIR WEAPON UNIT BEING REPORTED BELONGS.

 

 

3A

ORGANIZATION TYPE

TYP

ENTER "TYP:" FOLLOWED BY THE TYPE OF SURFACE-TO-AIR WEAPON UNIT BEING REPORTED, FOR EXAMPLE: "TYP:ML". See table 1311/1 which contains a list of data items and associated data codes

 

 

3B

SURFACE-TO-AIR WEAPON SYSTEM

SYS

ENTER "SYS:" FOLLOWED BY THE SYSTEM OF SURFACE-TO-AIR WEAPON UNIT BEING REPORTED, FOR EXAMPLE: "SYS:CROTAL". See table 1407/2 which contains a list of data items and associated data codes

4

SURFACE-TO-AIR WEAPON UNIT READINESS STATE

 

THE SURFACE-TO-AIR WEAPON UNIT STATE OF READINESS.

 

 

4A

SAW READINESS STATE

 

ENTER THE CURRENT WEAPON READINESS STATE, FOR EXAMPLE: "1". See table 1236/9 which contains a list of data items and associated data codes

5

SURFACE-TO-AIR WEAPONS OPERATIONAL STATUS

 

THE OPERATIONAL STATUS OF THE SURFACE-TO-AIR WEAPONS UNIT BEING REPORTED.

 

 

5A

SURFACE-TO-AIR WEAPONS OPERATIONAL STATUS

 

ENTER THE SURFACE-TO-AIR WEAPONS UNIT OPERATIONAL STATUS, FOR EXAMPLE: "LOP". See table 1236/37 which contains a list of data items and associated data codes

6

ESTIMATED DAY-TIME AND MONTH IN COMMISSION

 

THE ESTIMATED DATE TIME WHEN THE OPERATIONAL STATUS OF THE SURFACE-TO-AIR WEAPON UNIT BEING REPORTED WILL CHANGE.

 

 

6A

ESTIMATED DTG CURRENT STATUS EXPECTED TO END

 

ENTER THE ESTIMATED DATE TIME UNTIL WHICH THE CURRENT READINESS STATE IS EXPECTED TO LAST, FOR EXAMPLE "271030ZJUN2005". See table 2033 which is a composite
