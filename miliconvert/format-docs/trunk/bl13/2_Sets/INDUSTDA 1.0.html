1

ANNUAL CAPACITY

 

ENTER THE ACTUAL OR POTENTIAL CAPACITY OF THE FACILITY EITHER TO STORE OR TO PRODUCE ITEMS:

 

 

1A

STORAGE CAPACITY

CAPSTOR

ENTER "CAPSTOR:" FOLLOWED BY AN UP TO 10 DIGITS RATING  OF THE ACTUAL POTENTIAL OF THE INDUSTRIAL FACILITY TO STORE ITEMS AS DESCRIBED IN FIELD 7 IN A YEAR'S TIMEFRAME, FOR EXAMPLE: "CAPSTOR:100.60". See table 1869/3 a range [0 through 99999.0000] (0 to 4 decimal places)

 

 

1B

PRODUCTION CAPACITY

CAPPROD

ENTER "CAPPROD:" FOLLOWED BY AN UP TO 10 DIGITS RATING  OF THE ACTUAL POTENTIAL OF THE INDUSTRIAL FACILITY TO MANUFACTURE ITEMS AS DESCRIBED IN FIELD 7 IN A YEAR'S TIMEFRAME, FOR EXAMPLE: "CAPPROD:100.60". See table 1869/4 a range [0 through 99999.0000] (0 to 4 decimal places)

2

CAPACITY DATE-TIME (Z), YYYYMMDDHHMMSS

 

 

 

 

2A

CAPACITY DATE-TIME (Z), YYYYMMDDHHMMSS

CAPTIME

ENTER  "CAPTIME:" FOLLOWED BY THE DATE/TIME ON WHICH THE VALUE IN THE CAPACITY FIELD WAS ASSESSED, FOR EXAMPLE: "CAPTIME:20010406231619". See table 2417 which is a composite

3

VALIDITY

 

 

 

 

3A

VALIDITY

 

ENTER A ONE DIGIT NUMERIC  CODE TO GIVE A RATING OF THE CONFIDENCE THE ANALYST HAS IN THE FIGURES QUOTED, FOR EXAMPLE: "6". See table 1266/2 which contains a list of data items and associated data codes

4

CAPACITY VALIDITY DATE-TIME (Z), YYYYMMDDHHMMSS

 

 

 

 

4A

CAPACITY VALIDITY DATE-TIME (Z), YYYYMMDDHHMMSS

CAVATIM

ENTER "CAVATIM:" FOLLOWED BY THE DATE/TIME WHEN THE CAPACITY VALITY WAS RATED, FOR EXAMPLE: "CAVATIM:20010406231619". See table 2417 which is a composite

5

CAPACITY MAXIMUM

 

 

 

 

5A

CAPACITY MAXIMUM

CAPMAX

ENTER  "CAPMAX:" FOLLOWED BY UP TO 10 DIGITS GIVING THE ESTIMATED  MAXIMUM  POTENTIAL OF THE INDUSTRIAL FACILITY TO PRODUCE OR STORE (IDENTIFIED BY THE ENTRY IN FIELD 1) ITEMS DESCRIBED IN FIELD 7, FOR EXAMPLE: "CAPMAX:200". See table 1869/2 a range [0 through 99999.0000] (0 to 4 decimal places)

6

CAPACITY UNIT OF MEASUREMENT

 

 

 

 

6A

CAPACITY UNIT OF MEASUREMENT

 

ENTER THE UNIT OF MEASURE APPROPRIATE FOR THE CAPACITY FIGURES GIVEN IN FIELDS 1 AND 5, FOR EXAMPLE: "MTPD". See table 1008/104 which contains a list of data items and associated data codes

7

INDUSTRIAL ITEM

 

 

 

 

7A

INDUSTRIAL ITEM

 

ENTER A CODE TO INDICATE THE KEY FEATURE OF THE PRODUCT OR SERVICE ASSOCIATED WITH THE FACILITY, FOR EXAMPLE: "CELLULOSE". See table 1870/1 which contains a list of data items and associated data codes

8

ITEM AVAILABILITY

 

 

 

 

8A

ITEM AVAILABILITY

 

ENTER A CODE IDENTIFYING THE AVAILIBILITY OF EACH SPECIFIC FEATURED PRODUCT OR SERVICE ASSOCIATED WITH THE FACILITY, FOR EXAMPLE: " LONG". See table 1868/3 which contains a list of data items and associated data codes

9

ITEM COUNTRY CODE

 

 

 

 

9A

ITEM COUNTRY CODE

 

ENTER A TWO DIGIT CODE REPRESENTING THE PRINCIPAL COUNTRY THAT THE MATERIAL IS EITHER BEING IMPORTED FROM OR EXPORTED TO, FOR EXAMPLE: "BU". See table 1026/22 which contains a list of data items and associated data codes

10

ITEM PERCENT IMPORTED

 

 

 

 

10A

ITEM PERCENT IMPORTED

IMP

ENTER "IMP:" FOLLOWED BY  A NUMERIC FIGURE GIVING THE PERCENTAGE OF FACILITY'S PRODUCT OR SERVICE THAT IS IMPORTED FROM FOREIGN COUNTRIES, FOR EXAMPLE: "IMP:100". See table 1064/38 a range [0 through 100]

11

ITEM PERCENT EXPORTED

 

 

 

 

11A

ITEM PERCENT EXPORTED

EXP

ENTER "EXP:" FOLLOWED BY  A NUMERIC FIGURE GIVING THE PERCENTAGE OF FACILITY'S PRODUCT OR SERVICE THAT IS EXPORTED TO FOREIGN COUNTRIES, FOR EXAMPLE: "EXP:100". See table 1064/46 a range [0 through 100]

12

ITEM TYPE

 

 

 

 

12A

ITEM TYPE

 

ENTER AN ALPHABETIC CODE INDICATING THE PRODUCTION INTENT OF THE TYPE OF MATERIAL BEING REFERENCED IN CONTEXT OF THE FACILITY, FOR EXAMPLE: "SERV". See table 1862/4 which contains a list of data items and associated data codes

13

FACILITY INTEGRATION

 

 

 

 

13A

FACILITY INTEGRATION

 

ENTER A CODE TO INDICATE THE EXTENT TO WHICH THE FACILITY'S OPERATION IS INTEGRATED, FOR EXAMPLE: "NOIN".   See table 1760/2 which contains a list of data items and associated data codes

14

LINE QUANTITY

 

 

 

 

14A

LINE QUANTITY

LINES

ENTER "LINES:" FOLLOWED BY  THE  NUMBER OF PRODUCTION LINES RUNNING IN THE INDUSTRIAL FACILITY, FOR EXAMPLE: "LINES:10". See table 1044/380 a range [1 through 999]

15

CRITICAL INDUSTRY

 

 

 

 

15A

CRITICAL INDUSTRY

CRITICAL

ENTER "CRITICAL:" FOLLOWED BY A ONE DIGIT CODE INDICATING KEY INDUSTRIAL GROUPS WHICH ARE CRITICAL TO THE COUNTRY'S MILITARY, ECONOMIC, AND POLITICAL SUCCESS, FOR EXAMPLE: "CRITICAL:1". See table 1559/2 which contains a list of data items and associated data codes

16

PERCENTAGE OF FOREIGN TECHNOLOGY

 

 

 

 

16A

PERCENTAGE OF FOREIGN TECHNOLOGY

PERCENT

ENTER "PERCENT:" FOLLOWED BY A NUMERICAL FIGURE GIVING THE PERCENTAGE OF THE FACILITY WHICH IS DEPENDANT ON FOREIGN TECHNOLOGY OR THE PERCENTAGE OF TECHNOLOGY PROVIDED BY THE FOREIGN COUNTRY, FOR EXAMPLE: "PERCENT:55" See table 1064/39 a range [0 through 100]

17

MODERNISATION

 

 

 

 

17A

MODERNISATION

 

ENTER A CODE DESCRIBING THE EXTENT TO WHICH THE FACILITY PROCESS AND EQUIPMENT HAVE BEEN MODERNIZED, FOR EXAMPLE: "OLDOUT" See table 1584/11 which contains a list of data items and associated data codes

18

OPEN STORAGE COVERED

 

 

 

 

18A

OPEN STORAGE COVERED

STORECOV

ENTER "STORECOV:" FOLLOWED BY THE TOTAL AREA IN 100 SQUARE METRES OF FACILITY OPEN STORAGE WHICH IS COVERED, FOR EXAMPLE: "STORECOV:600". See table 1656/2 a range [0 through 99999.0000] (0 to 4 decimal places)

19

OPEN STORAGE TOTAL

 

 

 

 

19A

OPEN STORAGE TOTAL

OSTORTOT

ENTER "OSTORTOT:" FOLLOWED BY THE TOTAL AREA IN 100 SQUARE METRES OF FACILITY OPEN STORAGE, FOR EXAMPLE: "OSTORTOT:900". See table 1656/3 a range [0 through 99999.0000] (0 to 4 decimal places)

20

ANNUAL INDUSTRIAL OUTPUT

 

 

 

 

20A

ANNUAL INDUSTRIAL OUTPUT

OUTPUT

ENTER "OUTPUT:" FOLLOWED BY A QUANTITY FIGURE GIVING THE ANUAL INDUSTRIAL OUTPUT OF A PRODUCTION FACILITY, FOR EXAMPLE: "OUTPUT:123444". See table 1659/2 a range [0 through 99999.0000] (0 to 4 decimal places)

21

OUTPUT DATE, DDMMYYYY

 

MUST ONLY CONTAIN A HYPHEN, IF FIELD 20 OF THIS SET CONTAINS A HYPHEN.

 

 

21A

OUTPUT DATE, DDMMYYYY

OUTDATE

ENTER "OUTDATE:" FOLLOWED BY THE DATE ON WHICH THE VALUE IN THE OUTPUT FIELD IS BASED, FOR EXAMPLE: "OUTDATE:03072001". See table 2052 which is a composite

22

VALIDITY, OUTPUT

 

MUST ONLY CONTAIN A HYPHEN, IF FIELD 20 OF THIS SET CONTAINS A HYPHEN.

 

 

22A

VALIDITY

 

ENTER A CODE GIVING AN EVALUATION OF THE OUTPUT AND OUTPUT MAXIMUM FIELD, FOR EXAMPLE: "1".  See table 1266/2 which contains a list of data items and associated data codes

23

OUTPUT VALIDITY DATE, DDMMYYYY

 

MUST ONLY CONTAIN A HYPHEN, IF FIELD 20 OF THIS SET CONTAINS A HYPHEN.

 

 

23A

OUTPUT VALIDITY DATE, DDMMYYYY

 

ENTER THE DATE ON WHICH THE OUTPUT VALIDITY WAS LAST EVALUATED,  FOR EXAMPLE: "03072001". See table 2052 which is a composite

24

MAXIMUM ANNUAL OUTPUT

 

 

 

 

24A

MAXIMUM ANNUAL OUTPUT

OUTMAX

ENTER "OUTMAX:" FOLLOWED BY THE MAXIMUM ANNUAL INDUSTRIAL OUTPUT POSSIBLE FOR THIS INDUSTRIAL FACILITY, FOR EXAMPLE: "OUTMAX:3500". See table 1659/3 a range [0 through 99999.0000] (0 to 4 decimal places)

25

OUTPUT UNIT OF MEASUREMENT

 

THIS FIELD MUST HAVE A VALID ENTRY OTHER THAN HYPHEN, IF FIELD 20 AND/OR FIELD 24 HOLD AN ENTRY OTHER THAN A HYPHEN.

 

 

25A

OUTPUT UNIT OF MEASUREMENT

 

ENTER THE UNIT OF MEASURE FOR  "OUTPUT"AND "OUTPUT MAXIMUM", FOR EXAMPLE: "MTPD". See table 1008/103 which contains a list of data items and associated data codes

26

PRODUCTION SCALE

 

 

 

 

26A

PRODUCTION SCALE

 

ENTER A CODE TO EVALUATE MANNER IN WHICH PRODUCTION AT THE FACILITY IS CARRIED OUT, FOR EXAMPLE: "PILOT". See table 1809/1 which contains a list of data items and associated data codes

27

TURN AROUND TIME, DDDHHMM

 

 

 

 

27A

TURN AROUND TIME, DDDHHMM

 

ENTER THE TIME TAKEN TO MANUFACTURE OR PRODUCE THE ITEM DESCRIBED IN FIELD 7 EXPRESSED AS  DDDHHMM, FOR EXAMPLE: "3201637". See table 2599 which is a composite

28

POWER SOURCE

 

 

 

 

28A

POWER SOURCE

INDUSTRIAL ROLE LEVEL

 

 

 

 

29A

INDUSTRIAL ROLE LEVEL

ROLE TYPE, INDUSTRIAL PLANT
