Concept/Explanation/Examples

1

TARGET LOCATION

 

THE GEOGRAPHIC LOCATION OF THE TARGET.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

1B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

1C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

1D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

1E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

1F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

1G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE GEOGRAPHIC LOCATION OF THE TARGET AS GEOREF, CENTIMINUTE, FOR EXAMPLE: "GEO:KQAA24151224". See table 2018 which is a composite

 

 

1H

POSITION OR POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE TARGET POSITION NAME, FOR EXAMPLE: "NAME:BRAND". See table 1022/18 which is an instructive entry

 

 

1I

TARGET POSITION, TRUE BEARING AND DISTANCE IN NM

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:050T-BRAND-15NM". See table 2016 which is a composite

 

 

1J

BASIC ENCYCLOPEDIA NUMBER

BE

ENTER "BE:" FOLLOWED BY THE  BASIC ENCYCLOPEDIA NUMBER, FOR EXAMPLE: "BE:1234-12345". See table 2002 which is a composite

 

 

1K

ACE LINE SEARCH OPERATIONAL TASK NUMBER

AOP

ENTER "AOP:" FOLLOWED BY ACE LINE SEARCH OPERATIONAL TASK NUMBER, FOR EXAMPLE: "AOP:ACE3530". See table 1428/2 which is an instructive entry

 

 

1L

ACE LINE SEARCH EXERCISE TASK NUMBER

AEX

ENTER "AEX:" FOLLOWED BY THE ACE LINE SEARCH EXERCISE TASK NUMBER, FOR EXAMPLE: "AEX:ACE1213". See table 1428/3 which is an instructive entry

2

DAY-TIME OF LOCATION.

 

THE DAY-TIME WHEN THE TARGET WAS AT THE LOCATION.

 

 

2A

DAY-TIME OF POSITION

 

DAY-TIME OF POSITION, FOR EXAMPLE: "061500Z". See table 2000 which is a composite

3

TARGET COURSE

 

THE COURSE OF THE TARGET.

 

 

3A

TARGET COURSE IN DEGREES

 

ENTER THE TARGET COURSE, FOR EXAMPLE: "030T" OR "030M" OR "030G". See table 2065 which is a composite

 

 

3B

DIRECTION OF MOVEMENT (COMPASS POINT)

 

ENTER THE DIRECTION OF MOVEMENT EXPRESSED BY COMPASS POINT, FOR EXAMPLE: "NNE" OR "SSW". See table 1161/1 which contains a list of data items and associated data codes

 

 

3C

DIRECTION OF MOVEMENT IN DEGREES

 

ENTER THE DIRECTION OF MOVEMENT EXPRESSED IN DEGREES, FOR EXAMPLE: "030". See table 1043/9 a range [001 through 360]

4

TARGET SPEED

 

THE SPEED OF THE TARGET.

 

 

4A

TARGET SPEED

 

ENTER THE TARGET SPEED, FOR EXAMPLE: "350KPH" OR "220KTS". See table 2063 which is a composite

 

 

4B

APPROXIMATE SPEED

 

ENTER THE  APPROXIMATE SPEED, FOR EXAMPLE: "FAST". See table 1113/5 which contains a list of data items and associated data codes

5

TARGET ALTITUDE

 

THE ALTITUDE OF THE TARGET.

 

 

5A

TARGET ALTITUDE, FLIGHT LEVEL

 

ENTER THE FLIGHT LEVEL, FOR EXAMPLE: "FL250". See table 2129 which is a composite

 

 

5B

TARGET ALTITUDE IN HUNDREDS OF FEET

 

ENTER THE TARGET ALTITUDE IN HUNDREDS OF FEET, FOR EXAMPLE: "120". See table 1255/8 a range [000 through 999]

 

 

5C

ALTITUDE IN FEET ABOVE GROUND LEVEL (AGL)

AGL

ENTER "AGL:" FOLLOWED BY THE FEET ABOVE GROUND LEVEL (AGL), FOR EXAMPLE: "AGL:200". See table 1290/2 a range [0 through 99999]
