1

RADAR WARNING RECEIVER TYPE

 

PROVIDES THE TYPE OF RADAR WARNING RECEIVER EQUIPPING THE AIRCRAFT.

 

 

1A

RADAR WARNING RECEIVER TYPE

RWR

ENTER "RWR:" FOLLOWED BY THE TYPE OF RADAR WARNING RECEIVER EQUIPPING THE AIRCRAFT, FOR EXAMPLE "RWR:ALR69". See table 1135/25 which contains a list of data items and associated data codes

2

ALE-50 ONBOARD

 

PROVIDES AN INDICATION, YES OR NO,  AS TO WHETHER THE MISSION AIRCRAFT IS EQUIPPED WITH ALE-50.

 

 

2A

ALE-50 ONBOARD INDICATOR

 

ENTER AN INDICATION, YES OR NO, AS TO WHETHER THE MISSION AIRCRAFT IS EQUIPPED WITH ALE-50, FOR EXAMPLE "YES". See table 1027/111 which contains a list of data items and associated data codes

3

RADAR WARNING RECEIVER OPERATIONAL FLIGHT PROGRAM

 

IF USED, PROVIDES THE RADAR WARNING RECEIVER OPERATIONAL FLIGHT PROGRAM FOR THE MISSION AIRCRAFT.

 

 

3A

RADAR WARNING RECEIVER OPERATIONAL FLIGHT PROGRAM

OFP

ENTER "OFP:" FOLLOWED BY THE RADAR WARNING RECEIVER OPERATIONAL FLIGHT PROGRAM FOR THE MISSION AIRCRAFT, FOR EXAMPLE, "OFP:MILLIPEDE OFP MK III". See table 1022/100 which is an instructive entry

4

RADAR WARNING RECEIVER MULTI FUNCTION DISPLAY

 

IF USED, PROVIDES THE TYPE OF RADAR WARNING RECEIVER MULTI-FUNCTION DISPLAY EQUIPPING THE MISSION AIRCRAFT.

 

 

4A

RADAR WARNING RECEIVER MULTI FUNCTION DISPLAY

MFD

ENTER "MFD:" THE TYPE OF RADAR WARNING RECEIVER MULTI-FUNCTION DISPLAY EQUIPPING THE MISSION AIRCRAFT, FOR EXAMPLE, "MFD:MLUMFD". See table 1022/105 which is an instructive entry

5

RADAR WARNING RECEIVER SYMBOLOGY, STEADY

 

IF USED, PROVIDES A DESCRIPTION OF THE RADAR WARNING RECEIVER STEADY SYMBOL INDICATIONS.

 

 

5A

RADAR WARNING RECEIVER SYMBOLOGY, STEADY

 

ENTER A DESCRIPTION OF THE RADAR WARNING RECEIVER STEADY SYMBOL INDICATIONS, FOR EXAMPLE "HAWK". See table 1022/116 which is an instructive entry

6

RADAR WARNING RECEIVER SYMBOLOGY, MIPPLE

 

IF USED, PROVIDES A DESCRIPTION OF THE RADAR WARNING RECEIVER MIPPLE INDICATIONS

 

 

6A

RADAR WARNING RECEIVER SYMBOLOGY, MIPPLE

 

ENTER A DESCRIPTION OF THE RADAR WARNING RECEIVER MIPPLE INDICATIONS, FOR EXAMPLE: "CS, BB, HA, HK" See table 1022/117 which is an instructive entry

7

RADAR WARNING RECEIVER SYMBOLOGY, RESOLVED TO

 

PROVIDES A DESCRIPTION OF THE RADAR WARNING RECEIVER MIPPLE SYMBOL INDICATIONS RESOLVED TO.  CAN ONLY BE USED IN CONJUNCTION WITH RADAR WARNING RECEIVER MIPPLE INDICATIONS IN THIS SET.

 

 

7A

RADAR WARNING RECEIVER SYMBOLOGY, RESOLVED TO

 

ENTER A DESCRIPTION OF THE RADAR WARNING RECEIVER MIPPLE SYMBOL INDICATIONS RESOLVED TO, FOR EXAMPLE: "HA, HK". See table 1022/73 which is an instructive entry

8

RADAR WARNING RECEIVER SYMBOLOGY, TEMPORARY

 

IF USED, PROVIDES A DESCRIPTION OF THE RADAR WARNING RECEIVER TEMPORARY INDICATIONS.

 

 

8A

RADAR WARNING RECEIVER SYMBOLOGY, TEMPORARY

 

ENTER A DESCRIPTION OF THE RADAR WARNING RECEIVER TEMPORARY INDICATIONS, FOR EXAMPLE: "15, 31". See table 1022/85 which is an instructive entry

9

RADAR WARNING RECEIVER AUDIO HEARD

 

PROVIDES AN INDICATOR, YES OR NO, AS TO WHETHER RADAR WARNING RECEIVER AUDIO WAS HEARD DURING FLIGHT OPERATIONS.

 

 

9A

RADAR WARNING RECEIVER AUDIO HEARD

 

ENTER AN INDICATOR, YES OR NO, AS TO WHETHER THE RADAR WARNING RECEIVER AUDIO WAS HEARD DURING FLIGHT OPERATIONS, FOR EXAMPLE "YES". See table 1027/112 which contains a list of data items and associated data codes

10

RADAR WARNING RECEIVER AUDIO TYPE

 

USED IF A YES INDICATOR WAS ENTERED AT THE PREVIOUS FIELD, PROVIDES THE TYPE OF AUDIBLE WARNING HEARD.

 

 

10A

RADAR WARNING RECEIVER AUDIO TYPE

 

ENTER THE TYPE OF AUDIBLE WARNING THAT WAS HEARD DURING THE MISSION, FOR EXAMPLE: "HIGH-PITCHED AUDIO TONE HEARD ONCE". See table 1862/7 which is an instructive entry

11

RADAR WARNING RECEIVER MISSILE LAUNCH WARNING INDICATOR

 

PROVIDES AN INDICATION, YES OR NO, AS TO WHETHER RADAR WARNING RECEIVER MISSILE LAUNCH INDICATIONS WERE RECEIVED DURING ENGAGEMENT.

 

 

11A

RADAR WARNING RECEIVER MISSILE LAUNCH WARNING INDICATOR

 

ENTER AN INDICATION, YES OR NO, AS TO WHETHER RADAR WARNING RECEIVER MISSILE LAUNCH INDICATIONS WERE RECEIVED DURING ENGAGEMENT, FOR EXAMPLE: "YES". See table 1027/113 which contains a list of data items and associated data codes

12

RADAR WARNING RECEIVER SYMBOL POSITION ON SCOPE

 

PROVIDES THE POSITION OF THE INDICATED MISSILE LAUNCH AS DISPLAYED ON THE SCOPE OF THE RWR AT TIME OF ENCOUNTER.

 

 

12A

RADAR WARNING RECEIVER SYMBOL POSITION ON SCOPE

 

PROVIDES THE POSITION OF RWR INDICATIONS AS DISPLAYED ON THE SCOPE AT TIME OF ENCOUNTER, FOR EXAMPLE "BEARING 315 RANGE 50NM". See table 1022/95 which is an instructive entry

13

ENEMY MISSILE LAUNCHED

 

PROVIDES AN INDICATOR, YES OR NO,  AS TO WHETHER AN ACTUAL MISSILE WAS LAUNCHED DURING ENGAGEMENT.

 

 

13A

ENEMY MISSILE LAUNCHED

 

ENTER AN INDICATOR, YES OR NO,  AS TO WHETHER AN ACTUAL MISSILE WAS LAUNCHED DURING ENGAGEMENT, FOR EXAMPLE: "YES". See table 1027/114 which contains a list of data items and associated data codes

14

RADAR WARNING RECEIVER CREDIBILITY

 

PROVIDES THE CONFIDENCE LEVEL OF RADAR WARNING RECEIVER INDICATIONS.

 

 

14A

RADAR WARNING RECEIVER CONFIDENCE

 

ENTER THE CONFIDENCE LEVEL OF RADAR WARNING RECEIVER INDICATIONS, FOR EXAMPLE: "HIGH". See table 1123/14 which contains a list of data items and associated data codes

15

RADAR WARNING RECEIVER AUDIO CORRELATION TO TTR

 

PROVIDES AN INDICATOR, YES OR NO, AS TO WHETHER RADAR WARNING RECEIVER AUDIO INDICATIONS WERE CORRELATED TO TARGET TRACKING RADAR.

 

 

15A

RADAR WARNING RECEIVER AUDIO CORRELATION TO TTR

 

ENTER AN INDICATOR, YES OR NO, AS TO WHETHER RADAR WARNING RECEIVER AUDIO INDICATIONS WERE CORRELATED TO TARGET TRACKING RADAR, FOR EXAMPLE: "YES". See table 1027/115 which contains a list of data items and associated data codes
