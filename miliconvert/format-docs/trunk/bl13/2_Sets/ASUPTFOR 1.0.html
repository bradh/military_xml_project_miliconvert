No

Designator

Field Desc

Concept/Explanation/Examples

1

AERIAL MISSION SUPPORT TYPE

 

 

 

 

1A

PRIMARY AERIAL MISSION TYPE

 

ENTER THE CODE FOR THE TYPE MISSION PROVIDING THE SUPPORT, FOR EXAMPLE: "CAS". See table 1285/31 which contains a list of data items and associated data codes

2

UNIT IDENTIFIER

 

PROVIDES THE UNIT IDENTIFICATION OF THE UNIT BEING SUPPORTED, USING A DESIGNATOR OR A UIC CODE.

 

 

2A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE DESIGNATOR OF THE UNIT THAT IS TASKED TO PERFORM THE SUPPORT  MISSION, FOR EXAMPLE: "UNIT:ARS BROCKZETEL". See table 1028/43 which is an instructive entry

 

 

2B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE IDENTIFICATION CODE OF THE UNIT BEING SUPPORTED, FOR EXAMPLE: "UIC:GEF1234". See table 2141 which is a composite

3

MISSION NUMBER

 

 

 

 

3A

MISSION NUMBER

 

ENTER THE MISSION NUMBER ASSIGNED TO THE SUPPORTING MISSION, FOR EXAMPLE: "D171HA". See table 1012/69 which is an instructive entry

4

CALL SIGN

 

 

 

 

4A

CONTACT CALL SIGN

 

ENTER THE CALL  SIGN ASSIGNED TO THE SUPPORTING AIRCRAFT, FOR EXAMPLE: "HAMMER 53". See table 1129/31 which is an instructive entry

5

LINK 16 CALL SIGN

 

 

 

 

5A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE LINK 16 ABBREVIATED CALL SIGN IF THE SUPPORTING AIRCRAFT IS LINK 16 EQUIPPED, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry

6

NUMBER OF ASSETS

 

 

 

 

6A

NUMBER OF ASSETS, 1-999

 

ENTER THE NUMBER OF ASSETS PARTICIPATING IN THE SUPPORTING MISSION, FOR EXAMPLE: "1". See table 1044/802 a range [1 through 999]

7

SUPPORTING AIRCRAFT OR VEHICLE

 

PROVIDES THE TYPE OF AIRCRAFT OR VEHICLE PROVIDING THE SUPPORT.  CAN ONLY BE USED IN CONJUNCTION WITH NUMBER OF ASSETS SPECIFIED IN THIS SET.

 

 

7A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE CODE FOR TYPE AND MODEL NUMBER OF THE AIRCRAFT, FOR EXAMPLE: "ACTYP:FA18". See table 1015/20 which contains a list of data items and associated data codes

 

 

7B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF THE AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST, FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

 

 

7C

VEHICLE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE VEHICLES, FOR EXAMPLE: "NAME:GAZELLE 321". See table 1022/224 which is an instructive entry

 

 

7D

AIRCRAFT CATEGORY, ABBREVIATED

ACCAT

ENTER "ACCAT:" FOLLOWED BY THE AIRCRAFT CATEGORY, ABBREVIATED, FOR EXAMPLE: "ACCAT:CRGTRS". See table 1549/2 which contains a list of data items and associated data codes

8

PRIMARY COMMUNICATION PARAMETERS

 

PROVIDES THE PRIMARY COMMUNICATION PARAMETERS REQUIRED FOR CONTACTING THE SUPPORTING MISSION, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.

 

 

8A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY TO BE USED BY THE SUPORTING AIRCRAFT, FOR EXAMPLE: "PFREQ:342.5". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

8B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR FOR CONTACTING THE SUPPORTING AIRCRAFT, FOR EXAMPLE: "PDESIG:TAD001". See table 1396/2 which is an instructive entry

 

 

8C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQKM:AB12-CD34-12082000-3EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

8D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQK:AB12-CD34-12082000". See table 2535 which is a composite

9

SECONDARY COMMUNICATION PARAMETERS

 

PROVIDES THE SECONDARY COMMUNICATION PARAMETERS REQUIRED FOR CONTACTING THE SUPPORTING MISSION, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.  CAN ONLY BE USED IN CONJUNCTION WITH PRIMARY COMMUNICATION PARAMETERS SPECIFIED IN THIS SET.

 

 

9A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE SECONDARY FREQUENCY TO BE USED BY THE SUPPORTING AIRCRAFT, FOR EXAMPLE: "SFREQ:350.5". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

9B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY DESIGNATOR FOR CONTACTING THE SUPPORTING AIRCRAFT, FOR EXAMPLE: "SDESIG:TAD002". See table 1396/3 which is an instructive entry

 

 

9C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

SHVQKM

ENTER "SHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

9D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

SHVQK

ENTER "SHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQK:AB12-CD34-12082000". See table 2535 which is a composite

10

AIRCRAFT UNIT REFERENCE NUMBER

 

 

 

 

10A

AIRCRAFT UNIT REFERENCE NUMBER

URN

ENTER "URN:" FOLLOWED BY  THE UNIT REFERENCE NUMBER IDENTIFYING THE AIRCRAFT CONDUCTING THE SUPPORTING MISSION, FOR EXAMPLE "URN:12543678" See table 1028/45 a range [00000000 through 16777215]

11

AIRCRAFT INTERNET ADDRESS

 

IT PROVIDES THE IP ADDRESS OF THE COMMUNICATION DEVICE USED BY THE AIRCRAFT PERFORMING THE SUPPORTING MISSION.

 

 

11A

AIRCRAFT INTERNET ADDRESS, IPV4

IPV4

ENTER "IPV4:" FOLLOWED BY THE IP ADDRESS OF THE COMMUNICATION DEVICE USED BY THE AIRCRAFT PERFORMING THE SUPPORTING MISSION, FOR EXAMPLE "IPV4:192.168.29.100". See table 2004 which is a composite

 

 

11B

AIRCRAFT INTERNET ADDRESS, IPV6

IPV6

ENTER "IPV6:" FOLLOWED BY THE IP ADDRESS VERSION 6 OF THE COMMUNICATION DEVICE USED BY THE AIRCRAFT PERFORMING THE SUPPORTING MISSION. See table 1999/3 which is an instructive entry

12

AIRCRAFT DATA LINK ADDRESS

 

 

 

 

12A

AIRCRAFT DATA LINK ADDRESS

 

ENTER THE DATA LINK ADDRESS FOR THE SUPPORTING MISSION AIRCRAFT THAT REPRESENTS THE SOURCE/DESTINATION OF THE MESSAGE, FOR EXAMPLE: "075". See table 1012/126 a range [000 through 127]
