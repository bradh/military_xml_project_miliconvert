UNIT DUTY STATUS

 

ENTER A CODE INDICATING THE AVAILABILITY OF A UNIT TO PARTICIPATE IN THE FULL TIME MILITARY ACTIVITY, FOR EXAMPLE: "A". See table 1584/10 which contains a list of data items and associated data codes

2

UNIT OPERATIONAL READINESS STATUS

 

 

 

 

2A

UNIT OPERATIONAL READINESS STATUS

 

ENTER A CODE INDICATING THE DEGREE TO WHICH A UNIT IN PEACETIME IS READY TO PERFORM THE OVERALL OPERATIONAL MISSION(S) FOR WHICH IT WAS ORGANIZED AND EQUIPPED, FOR EXAMPLE: "Z". See table 1236/30 which contains a list of data items and associated data codes

3

DEPLOYMENT STATUS

 

 

 

 

3A

DEPLOYMENT STATUS

 

IF USED, ENTER A CODE INDICATING THE PROPORTION OF THE UNIT THAT HAS DEPLOYED, FOR EXAMPLE: "ND". See table 1584/14 which contains a list of data items and associated data codes

4

UNIT COMBAT EFFECTIVENESS IN PERCENT

 

 

 

 

4A

UNIT COMBAT EFFECTIVENESS IN PERCENT

 

ENTER AN UP TO THREE DIGIT FIGURE GIVING THE LEVEL OF EFFECTIVENESS  TO WHICH A UNIT IS READY TO PERFORM THE OVERALL OPERATIONAL MISSION(S) FOR WHICH IT WAS ORGANIZED AND EQUIPPED, EXPRESSED AS A PERCENTAGE, FOR EXAMPLE: "95". See table 1064/60 a range [0 through 100]
