TYPE OF AGENT RELEASE HEIGHT

 

 

 

 

1A

TYPE OF AGENT RELEASE HEIGHT

 

ENTER THE TYPE OF AGENT-RELEASE HEIGHT, FOR EXAMPLE: "AIR". See table 1735/3 which contains a list of data items and associated data codes

2

AGENT DEFINITION

 

IF USED, PROVIDES THE IDENTIFICATION OF AGENT IN ONE OF THE THREE FOLLOWING WAYS:

 

 

2A

TYPE OF AGENT

TA

ENTER "TA:" FOLLOWED BY THE CATEGORY OF THE GROUP TO WHICH PERTAINS THE AGENT, FOR EXAMPLE: "TA:NERV". See table 1531/5 which contains a list of data items and associated data codes

 

 

2B

AGENT NAME

AN

ENTER "AN:" FOLLOWED BY THE NAME OF THE INDIVIDUAL AGENT, FOR EXAMPLE: "AN:IO". See table 1022/9 which contains a list of data items and associated data codes

 

 

2C

UN NA IDENTIFICATION NUMBER

 

ENTER THE FOUR DIGITS GIVING THE AGENT IDENTIFICATION AS DEFINED BY STANAG 2909, FOR EXAMPLE: "4639". See table 1012/240 a range [0000 through 9999]

3

TYPE OF PERSISTENCY

 

 

 

 

3A

TYPE OF PERSISTENCY

 

IF USED, ENTER THE TYPE OF PERSISTENCY, FOR EXAMPLE: "NP". See table 1531/2 which contains a list of data items and associated data codes

4

TYPE OF DETECTION

 

 

 

 

4A

TYPE OF DETECTION

 

IF USED, ENTER THE TYPE OF DETECTOR, DETECTOR SYSTEM OR AGENT DETECTOR,  REPEATING ONCE IF REQUIRED, FOR EXAMPLE: "FOX". See table 1736/1 which contains a list of data items and associated data codes
