CONTACT CATEGORY

 

 

 

 

1A

CONTACT IDENTIFIER

 

ENTER THE CATEGORY OF CONTACT THAT IS BEING INCLUDED IN THE ASSOCIATED REPORT, FOR EXAMPLE: "NAV". See table 1139/2 which contains a list of data items and associated data codes

2

CONTACT THREAT CATEGORY

 

 

 

 

2A

CONTACT THREAT CATEGORY

 

ENTER THE IDENTIFICATION OF THE CONTACT THREAT CATEGORY, FOR EXAMPLE: "AFD". See table 1103/1 which contains a list of data items and associated data codes

3

UNIT NAME

 

 

 

 

3A

UNIT NAME

 

ENTER THE NAME OF THE CONTACT, FOR EXAMPLE: "QUEEN ELIZABETH 2". See table 1022/48 which is an instructive entry

4

COUNTRY CODE

 

PROVIDES THE COUNTRY CODE OF THE CONTACT.

 

 

4A

GEOGRAPHICAL ENTITY (2A)

 

ENTER THE COUNTRY CODE OF THE CONTACT, FOR EXAMPLE: "UK". See table 1262/6 which contains a list of data items and associated data codes

 

 

4B

EXERCISE GEOGRAPHICAL ENTITY

EXEN

ENTER "EXEN:" FOLLOWED BY THE TWO-LETTER EXERCISE COUNTRY CODE OF THE CONTACT, FOR EXAMPLE: "EXEN:XB". See table 1262/4 which contains a list of data items and associated data codes

SHIP CLASS

NAME OF SHIP CLASS

 

ENTER THE NAME OF THE SHIP CLASS OF CONTACT, FOR EXAMPLE: "NIMITZ". See table 1022/35 which is an instructive entry
