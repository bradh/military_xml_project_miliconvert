Concept/Explanation/Examples

1

SORTIE IDENTIFICATION NUMBER

 

 

 

 

1A

AIR SORTIE IDENTIFICATION NUMBER

 

ENTER THE SEQUENTIAL NUMBER AND WHERE POSSIBLE LISTED IN DTG ORDER. AIRCRAFT REQUESTED SHOULD EACH BE GIVEN AS SEQUENTIAL NUMBER. THE AIRCRAFT TASKING AUTHORITY WILL BE RESPONSIBLE FOR PROVIDING THE AIRCRAFT AND GIVING IT A MISSION NUMBER, FOR EXAMPLE "1". See table 1012/205 a range [1 through 999]

2

ASW HELICOPTER TYPE

 

 

 

 

2A

ASW HELICOPTER ROLE

 

ENTER THE TYPE OF ASW HELICOPTER, FOR EXAMPLE "DIPPER". See table 1010/8 which contains a list of data items and associated data codes

3

NUMBER OF HELICOPTERS REQUIRED

 

 

 

 

3A

QUANTITY OF HELICOPTERS REQUIRED

 

ENTER THE NUMBER OF HELICOPTERS REQUIRED, FOR EXAMPLE "2". See table 1044/370 a range [0 through 9]

4

DTG OF START OF REQUIREMENT

 

 

 

 

4A

DTG OF START

 

ENTER THE DTG AT WHICH THE PROVISION OF AIRCRAFT IS TO START, FOR EXAMPLE: "101010ZJUN2005". See table 2033 which is a composite

5

DTG OF END OF REQUIREMENT

 

 

 

 

5A

DTG OF STOP

 

ENTER THE DTG AT WHICH THE PROVISION OF AIRCRAFT IS TO STOP, FOR EXAMPLE: "121800ZJUN2005". See table 2033 which is a composite

6

DAILY START TIME

 

 

 

 

6A

DAILY START TIME

 

IF USED, ENTER THE TIME  WHICH AIRCRAFT ARE REQUIRED FROM DAILY. FOR EXAMPLE: "0600Z". IF 24 HOUR COVERAGE IS REQUIRED THIS FIELD SHOULD BE OMITTED. See table 2485 which is a composite

7

DAILY STOP TIME

 

 

 

 

7A

DAILY STOP TIME

 

IF USED, ENTER THE TIME WHICH AIRCRAFT ARE NOT REQUIRED AFTER DAILY, FOR EXAMPLE, "1200Z". IF 24 HOUR COVERAGE IS REQUIRED THIS FIELD SHOULD BE OMITTED. See table 2485 which is a composite
