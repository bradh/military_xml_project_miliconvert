AREA WEAPON CONTROL INFO

 

PROVIDES THE DETAILS OF THE WEAPONS CONTROL REQUIRED WITHIN AN INDIVUAL AREA FOR BOTH AIRCRAFT AND SAW ASSETS.

 

 

4A

AREA WEAPON CONTROL SAW

SAW

ENTER "SAW:" FOLLOWED BY THE DETAILS OF THE TACTICAL CONTROL REQUIRED FOR ANY SAW UNIT OPERATING WITHIN THE AREA, FOR EXAMPLE: "SAW:PROVIDE TACTICAL CONTROL FOR SAW MISSION NUMBER ABT A 1101". See table 1407/33 which is an instructive entry

 

 

4B

AREA WEAPON CONTROL AIRCRAFT

AC

ENTER "AC:" FOLLOWED BY THE TYPE OF AIRCRAFT CONTROL REQUIRED WITHIN THE AREA, FOR EXAMPLE: "AC:THREAT BROADCAST". See table 1022/55 which is an instructive entry
