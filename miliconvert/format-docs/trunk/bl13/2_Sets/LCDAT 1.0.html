No

Designator

Field Desc

Concept/Explanation/Examples

1

NUMBER OF LANDING CRAFT EMBARKED

 

 

 

 

1A

NUMBER OF LANDING CRAFT EMBARKED

 

ENTER THE NUMBER OF EMBARKED LANDING CRAFT OF THE SPECIFIED TYPE, FOR EXAMPLE "4". See table 1044/85 a range [1 through 99]

2

TYPE OF LANDING CRAFT EMBARKED

 

 

 

 

2A

LANDING CRAFT TYPE

 

ENTER THE TYPE OF EMBARKED LANDING CRAFT, FOR EXAMPLE "LCVP". See table 1166/6 which contains a list of data items and associated data codes

3

STATION WHEN SECURED FOR SEA

 

 

 

 

3A

LANDING CRAFT STATION

 

ENTER THE LOCATION WHERE THE LANDING CRAFT IS/ARE NORMALLY STOWED AT SEA, FOR EXAMPLE "DAVIT". See table 1641/2 which contains a list of data items and associated data codes

4

RESPONSE TIME, MINUTES

 

 

 

 

4A

RESPONSE TIME IN MINUTES

 

ENTER THE NUMBER OF MINUTES REQUIRED FOR THE UNIT TO LAUNCH THE LANDING CRAFT, FOR EXAMPLE: "45". See table 1072/20 a range [1 through 99]

5

ABILITY TO LAUNCH UNDERWAY

 

 

 

 

5A

ABILITY TO LAUNCH UNDERWAY

 

ENTER THE INDICATION OF WHETHER OR NOT THE UNIT CAN LAUNCH THE LANDING CRAFT WHILE UNDERWAY, FOR EXAMPLE "YES". See table 1027/75 which contains a list of data items and associated data codes

6

MAXIMUM OPERATING SEA STATE

 

 

 

 

6A

MAX OPERATING SEA STATE

 

ENTER THE MAXIMUM SEA STATE IN WHICH THE SPECIFIED LANDING CRAFT MAY BE OPERATED, FOR EXAMPLE "4". See table 1392/5 which contains a list of data items and associated data codes

7

LANDING CRAFT PLANNING SPEED IN KNOTS

 

 

 

 

7A

LANDING CRAFT PLANNING SPEED IN KNOTS

 

ENTER THE MAXIMUM SPEED IN KNOTS TO BE USED FOR PLANNING PURPOSES FOR THE LANDING CRAFT, FOR EXAMPLE: "14". See table 1061/7 a range [1 through 99]

8

ENDURANCE IN NAUTICAL MILES AT PLANNING SPEED

 

 

 

 

8A

ENDURANCE IN NAUTICAL MILES AT PLANNING SPEED

 

ENTER THE ENDURANCE IN NAUTICAL MILES OF THE LANDING CRAFT AT THE SPECIFIED PLANNING SPEED, FOR EXAMPLE: "200". See table 1089/13 a range [1 through 9999]

9

LOAD CAPACITY, TROOPS

 

 

 

 

9A

TROOP LOAD CAPACITY ON LANDING CRAFT

 

ENTER THE MAXIMUM NUMBER OF TROOPS THAT MAY BE CARRIED BY THE LANDING CRAFT, FOR EXAMPLE: "20". See table 1044/86 a range [1 through 999]

10

LOAD CAPACITY, LINEAR METRES

 

 

 

 

10A

MAXIMUM LOAD LENGTH IN METRES

 

ENTER THE MAXIMUM LINEAR METERAGE THAT MAY BE CARRIED BY THE LANDING CRAFT, FOR EXAMPLE: "10". See table 1657/8 a range [0 through 999]

11

NUMBER OF ITEMS OF RADIO EQUIPMENT

 

 

 

 

11A

NUMBER OF ITEMS OF RADIO EQUIPMENT

 

ENTER THE NUMBER OF RADIOS OF A SPECIFIED TRANSMISSION BAND CARRIED BY THE LANDING CRAFT, FOR EXAMPLE: "2". See table 1044/231 a range [1 through 99]

12

TRANSMISSION BAND

 

 

 

 

12A

TRANSMISSION BAND

 

ENTER THE TRANSMISSION BAND OF THE RADIO(S) CARRIED BY THE LANDING CRAFT, FOR EXAMPLE: "VHF". See table 1175/3 which contains a list of data items and associated data codes
