PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PRI

ENTER "PRI:" FOLLOWED BY THE PRIMARY RADIO FREQUENCY FOR THE HELICOPTER CONTROL SHIP (HCS) IN MEGAHERTZ, FOR EXAMPLE: "PRI:123.4". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

SECONDARY FREQUENCY IN MEGAHERTZ

SECONDARY FREQUENCY IN MEGAHERTZ

 

ENTER "ALT:" FOLLOWED BY THE SECONDARY/ALTERNATE RADIO FREQUENCY FOR THE HELICOPTER SHIP (HCS) IN MEGAHERTZ, FOR EXAMPLE: "ALT:234.5". See table 1068/4 which is a regular expression

HCS LANDING ZONE NAME

HCS LANDING ZONE NAME
