1

SUBMARINE NAME

 

 

 

 

1A

UNIT

 

ENTER THE NAME OF THE SUBMARINE OR A CODE TO COMPLY WITH SECURITY CONSTRAINTS.  DO NOT USE PREFIXES SUCH AS "USS" OR "HMS", FOR EXAMPLE: "BATON ROUGE". See table 1022/57 which is an instructive entry

2

NAVAL TASK DESIGNATOR

 

 

 

 

2A

NAVAL TASK DESIGNATOR

 

ENTER THE NAVAL TASK DESIGNATOR IF THE SUBMARINE BELONGS TO A TASK ORGANISATION, FOR EXAMPLE: "TU411.1.0". See table 2143 which is a composite

3

SUBMARINE ASSET TYPE

 

 

 

 

3A

SUBMARINE ASSET TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE TYPE OF SUBMARINE ASSET, FOR EXAMPLE: "SSN". See table 1166/4 which contains a list of data items and associated data codes

4

HULL NUMBER

 

 

 

 

4A

HULL NUMBER

 

ENTER THE HULL NUMBER OF THE UNIT IDENTIFIED IN FIELD 1, FOR EXAMPLE: "689". See table 1012/30 which is an instructive entry

5

NATIONALITY

 

 

 

 

5A

NATIONALITY

 

ENTER THE COUNTRY CODE REPRESENTING THE NATIONALITY OF THE SUBMARINE, FOR EXAMPLE: "US". See table 1026/2 which contains a list of data items and associated data codes

6

NEXT OF KIN BOOK HOLDER

 

 

 

 

6A

NEXT OF KIN BOOK HOLDER

 

IDENTIFY THE COMMAND WHICH HOLDS THE SUBMARINE NEXT OF KIN LIST, FOR EXAMPLE: "SM TWO". See table 1028/3 which is an instructive entry

7

ESCAPE BUOY NUMBER

 

 

 

 

7A

ESCAPE BUOY NUMBER

 

ENTER THE IDENTIFICATION NUMBER OF THE ESCAPE BUOY, FOR EXAMPLE: "F6400". See table 2213 which is a composite

SLOT BUOY FREQUENCY

 

 

 

 

8A

RADIO FREQUENCY

 

ENTER THE RADIO FREQUENCY OF THE SLOT BUOY, FOR EXAMPLE: "123.5MHZ". See table 2064 which is a composite
