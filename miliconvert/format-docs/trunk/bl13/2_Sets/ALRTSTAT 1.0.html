Concept/Explanation/Examples

1

UNIT IDENTIFICATION

 

THE UNIT IDENTIFICATION EITHER BY UIC OR BY LITERAL IDENTIFIER.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE IDENTIFICATION CODE FOR THE UNIT, FOR EXAMPLE: "unit:ALPHAZULU". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC2)

 

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE, (THIS ALTERNATIVE IS FOR THE 2 LETTER COUNTRY CODE), OF THE UNIT TO WHICH THE AIRCRAFT BELONGS, FOR EXAMPLE: "UKF213121". See table 2243 which is a composite

 

 

1C

UNIT IDENTIFICATION CODE (UIC3)

 

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE, (THIS ALTERNATIVE IS FOR THE3 LETTER COUNTRY CODE), OF THE UNIT TO WHICH THE AIRCRAFT BELONG, FOR EXAMPLE: "GBRF213121". See table 2241 which is a composite

2

NUMBER OF AIRCRAFT AIRBORNE

 

 

 

 

2A

NUMBER OF AIRCRAFT AIRBORNE (1-99)

AB

ENTER "AB:" FOLLOWED BY THE NUMBER OF AIRCRAFT THAT ARE AIRBORNE, FOR EXAMPLE: "AB:2". See table 1044/40 a range [1 through 99]

3

AIRCRAFT TYPE AIRBORNE

 

THE CODES OF THE AIRCRAFT AIRBORNE EITHER USING THE MASTER AIRCRAFT DATABASE OR THE LITERAL CODES FOR THE ONES NOT YET INCLUDED.

 

 

3A

AIRCRAFT BY TYPE, MODEL OR CODE

AC

ENTER "AC:" FOLLOWED BY THE TYPE AND MODEL OF AIRCRAFT BEING REPORTED, FOR EXAMPLE: "AC:F5F". See table 1015/20 which contains a list of data items and associated data codes

 

 

3B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND MODEL OF AIRCRAFT BEING REPORTED NOT LISTED IN THE MASTER LIST, FOR EXAMPLE: "OTHAC:TYPHOON". See table 1015/18 which is an instructive entry

4

PRIMARY AERIAL MISSION TYPE

 

 

 

 

4A

PRIMARY AERIAL MISSION TYPE

MSN

ENTER "MSN:" FOLLOWED BY THE MISSION TYPE OF THE AIRCRAFT REPORTED IN THE PREVIOUS FIELD, FOR EXAMPLE: "MSN:AD". See table 1285/31 which contains a list of data items and associated data codes

5

AIRCRAFT ALERT READINESS

 

 

 

 

5A

AIRCRAFT NUMBER AND ALERT STATUS

 

ENTER THE NUMBER OF AIRCRAFT READY FOR LAUNCH AND THEIR READINESS IN MINUTES OR HOURS, SEPARATED BY HYPHEN, FOR EXAMPLE: "5-12H". See table 2462 which is a composite

6

AIRCRAFT AVAILABILITY

 

THE NUMBER OF AIRCRAFT AVAILABLE FOR OTHER SITUATIONS LIKE SCRAMBLE, OPERATIONAL TURN AROUND OR SPECIAL MISSION.

 

 

6A

NUMBER OF A C SCRAMBLE BUT NOT AIRBORNE

SCRAMBLE

ENTER "SCRAMBLE:" FOLLOWED BY THE NUMBER OF AIRCRAFT ORDERED TO SCRAMBLE, BUT NOT AIRBORNE, FOR EXAMPLE: "SCRAMBLE:5". See table 1044/41 a range [1 through 99]

 

 

6B

NUMBER OF A C IN OPERATIONAL TURN AROUND

OTA

ENTER "OTA:" FOLLOWED BY THE NUMBER OF THE AIRCRAFT IN OPERATIONAL TURN AROUND, FOR EXAMPLE: "OTA:8". See table 1044/42 a range [1 through 99]

 

 

6C

NUMBER OF A C ASSIGNED FOR SPECIAL MISSION

SPEC

ENTER "SPEC:" FOLLOWED BY THE NUMBER OF AIRCRAFT ASSIGNED FOR SPECIAL MISSION, FOR EXAMPLE: "SPEC:2". See table 1044/43 a range [1 through 99]
