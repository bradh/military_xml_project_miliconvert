Field Desc

Concept/Explanation/Examples

1

SHIP NAME

 

 

 

 

1A

SHIP NAME

 

ENTER THE NAME OF THE SHIP BEING REPORTED, FOR EXAMPLE: "ARK ROYAL". See table 1022/49 which is an instructive entry

2

COUNTRY CODE

 

THE COUNTRY CODE OF THE SHIP.

 

 

2A

GEOGRAPHICAL ENTITY (2A)

 

ENTER THE TWO-LETTER COUNTRY CODE OF THE SHIP, FOR EXAMPLE: "UK". See table 1262/6 which contains a list of data items and associated data codes

 

 

2B

EXERCISE GEOGRAPHICAL ENTITY

EXEN

ENTER "EXEN:" FOLLOWED BY THE TWO-LETTER EXERCISE COUNTRY CODE OF THE SHIP, FOR EXAMPLE: "EXEN:XB". See table 1262/4 which contains a list of data items and associated data codes

3

VESSEL CONTACT TYPE

 

 

 

 

3A

CONTACT VESSEL TYPE

 

ENTER THE VESSEL DESIGNATION OF THE CONTACT, FOR EXAMPLE: "CVSG". See table 1166/12 which is an instructive entry

4

DTG OF LOCATION

 

 

 

 

4A

DTG OF LOCATION

 

ENTER THE DATE-TIME GROUP OF THE ASSOCIATED UNIT'S POSITION, FOR EXAMPLE: "121345ZJUN2001". See table 2033 which is a composite

5

SHIP LOCATION

 

THE GEOGRAPHIC LOCATION OF THE ASSOCIATED SHIP.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

5C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

5D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

5E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

5F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

5G

PORT LOCATION

PORT

ENTER "PORT:" FOLLOWED BY THE NAME OF THE PORT AND THE COUNTRY CODE IN WHICH PORT IS LOCATED, FOR EXAMPLE: "PORT:ROTTERDAM-NL". See table 2505 which is a composite

 

 

5H

PLACE NAME, EXTENDED

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE ASSOCIATED PLACE, FOR EXAMPLE: "NAME:BUOY 10". See table 1022/261 which is an instructive entry

6

SHIP'S MOVEMENT STATUS

 

THE COURSE, SPEED AND SHIP'S MOVEMENT STATUS.

 

 

6A

COURSE IN DEGREES TRUE (000.0=359.9) AND SPEED IN KTS

 

ENTER THE SHIP'S COURSE IN DEGREES TRUE AND SHIP'S SPEED IN KNOTS, FOR EXAMPLE: "009.5-22". See table 2156 which is a composite

 

 

6B

NAVAL FORCE MOVEMENT STATUS

 

ENTER THE SHIP'S MOVEMENT STATUS, FOR EXAMPLE: "MODLOC". See table 1341/2 which contains a list of data items and associated data codes
