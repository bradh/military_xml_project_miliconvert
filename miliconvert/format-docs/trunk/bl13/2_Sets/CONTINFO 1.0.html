No

Designator

Field Desc

Concept/Explanation/Examples

1

TYPE OF CONTACT

 

 

 

 

1A

TYPE OF CONTACT

 

ENTER THE TYPE OF CONTACT, FOR EXAMPLE: "ONLOAD". See table 1351/1 which contains a list of data items and associated data codes

2

CONTACT LOCATION

 

THE GEOGRAPHICAL LOCATION OF THE CONTACT.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE GEOGRAPHIC POSITION, CENTIMINUTE, FOR EXAMPLE: "GEO:KQAA24151224". See table 2018 which is a composite

 

 

2H

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR, FOR EXAMPLE: "ICAO:KADW". See table 1125/1 which is an instructive entry

 

 

2I

LOCATION OR POSITION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE EMBARKATION POSITION NAME, FOR EXAMPLE: "LOC:SEMBACH".  IF THE POINT OF EMBARKATION IS AN UNLISTED AIRFIELD, FIELD DATA DESCRIBED IN STANAG 3570 AND STANAG 3601 ARE TO BE PROVIDED IN A SUBSEQUENT AMPN SET. See table 1022/6 which is an instructive entry

3

NAME/RANK OF CONTACT

 

 

 

 

3A

NAME RANK OF CONTACT

 

ENTER THE NAME AND RANK OF THE PERSON TO BE CONTACTED, FOR EXAMPLE: "MAJOR S.T SCHMIDT". See table 1022/72 which is an instructive entry

4

UNIT IDENTIFICATION WITH OFFICE SYMBOL

 

 

 

 

4A

UNIT IDENTIFICATION WITH OFFICE SYMBOL

 

ENTER THE IDENTIFICATION, WITH OFFICE SYMBOL, OF THE UNIT, ORGANIZATION OR COMPANY TO WHICH THE PERSON TO BE CONTACTED IS ASSIGNED, FOR EXAMPLE: "316TAW-DOO". See table 1028/31 which is an instructive entry

5

TELEPHONE NUMBER

 

 

 

 

5A

TELEPHONE NUMBER

 

ENTER THE TELEPHONE NUMBER OF THE PERSON TO BE CONTACTED, FOR EXAMPLE: "49-6371-47-7501". THIS FIELD MAY BE REPEATED AS NECESSARY. See table 1361/2 which is an instructive entry
