DATA ENTRY

 

 

 

 

1A

DATA ENTRY

 

ENTER THE TWO DIGIT NUMBER (01 THROUGH 99) THAT TIES THE INFORMATION IN THIS LINE TO RELATED DATA LINES IN SETS 2CARG AND 2DANGER, IN ASCENDING ORDER, FOR EXAMPLE: "01". See table 1348/1 a range [01 through 99]

2

PREFERRED AIR CARGO DELIVERY METHOD

 

 

 

 

2A

PREFERRED AIR CARGO DELIVERY METHOD

 

ENTER THE DATA CODE FOR THE PREFERRED CARGO DELIVERY METHOD.  ENTER THE DELIVERY METHOD FOLLOWED BY THE HEIGHT BAND, FOR EXAMPLE: "LANDNQR".  IF CARGO IS TO BE DELIVERED BY CHUTE, THE TYPE OF CHUTE IS TO BE STATED IN A SUBSEQUENT AMPN SET. See table 2183 which is a composite

3

REASON FOR PREFERRED DELIVERY METHOD

 

 

 

 

3A

REASON FOR PREFERRED DELIVERY METHOD

 

ENTER THE REASON FOR THE PREFERRED DELIVERY METHOD ENTERED IN FIELD 2, FOR EXAMPLE: "CGO". See table 1350/1 which contains a list of data items and associated data codes

4

AIRLIFT PRIORITY INDICATOR

 

 

 

 

4A

AIRLIFT PRIORITY INDICATOR

 

ENTER THE PRIORITY FOR THE AIR TRANSPORT REQUEST, FOR EXAMPLE: "2". See table 1355/1 which contains a list of data items and associated data codes
