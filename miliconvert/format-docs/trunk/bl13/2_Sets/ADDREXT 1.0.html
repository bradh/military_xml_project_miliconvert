Concept/Explanation/Examples

1

COUNTRY

 

 

 

 

1A

COUNTRY

 

ENTER A TWO LETTER CODE FROM STANAG 1059 THAT INDICATES THE COUNTRY IN WHICH THE ADDRESS IS LOCATED, FOR EXAMPLE: "BU". See table 1026/18 which contains a list of data items and associated data codes

2

POST CODE

 

 

 

 

2A

POST CODE

 

ENTER A LITERAL UP TO 30 CHARACTERS TO IDENTIFY THE POSTAL CODE USED IN AN ADDRESS TO IDENTIFY THE POSTAL DISTRICT,  FOR EXAMPLE: "53359". See table 1610/2 which is an instructive entry

3

ADDRESS PLACE NAME

 

 

 

 

3A

ADDRESS PLACE NAME

 

ENTER THE NAME OF A CITY, TOWN, VILLAGE; OR OTHER LOCATION WHERE A UNIT IS LOCATED, FOR EXAMPLE: "RHEINBACH". See table 1022/244 which is an instructive entry

4

STREET NAME

 

 

 

 

4A

STREET NAME

 

ENTER THE NAME OF THE STREET ON WHICH THE ENTITY  IS LOCATED, FOR EXAMPLE: "WEILERSHEIMER STRASSE". See table 1022/246 which is an instructive entry

5

STREET NUMBER

 

 

 

 

5A

STREET NUMBER

 

ENTER THE NUMBER ON THE STREET WHERE  THE HOUSE, APARTMENT OR OTHER BUILDING USED IN THE ADDRESS IS LOCATED, FOR EXAMPLE: "234B". See table 1012/263 which is an instructive entry

6

POST OFFICE BOX NUMBER

 

 

 

 

6A

POST OFFICE BOX NUMBER

 

ENTER A LITERAL UP TO 30 CHARACTERS GIVING THE POST OFFICE BOX NUMBER  USED IN AN ADDRESS, FOR EXAMPLE: "GU 15 3YL" See table 1012/310 which is an instructive entry

7

TELEPHONE NUMBER

 

GIVE THE TELEFON NUMBER EITHER FOR AN INDIVIDUAL OR AN ENTITY:

 

 

7A

DESIGNATION OF TELEPHONE NUMBER

TELORG

ENTER "TELORG:" FOLLOWED BY THE TELEPHONE NUMBER ALLOCATED TO AN ENTITY SUCH AS AN OFFICE OR A SWITCHBOARD, FOR EXAMPLE: "TELORG:0032-2-779.03964-0". See table 1361/1 which is an instructive entry

 

 

7B

TELEPHONE NUMBER, INDIVIDUAL

TELIND

ENTER "TELIND:" FOLLOWED BY THE TELEPHONE NUMBER ALLOCATED TO AN INDIVIDUAL NAMED IN REMARKS, FOR EXAMPLE: "TELIND:0049-2226-776644". See table 1361/9 which is an instructive entry

8

FAX NUMBER

 

 

 

 

8A

FAX NUMBER

FAX

ENTER "FAX:" FOLLOWED BY THE FAX NUMBER OF THE ENTITY FOR EXAMPLE: "FAX:0032-2-779.77886655". DEPENDING ON THE ENTRY IN FIELD 7 THE NUMBER GIVEN EITHER REFERS TO AN ENTITY (FACTORY ETC.) OR AN INDIVIDUAL. See table 1361/10 which is an instructive entry

9

E-MAIL ADDRESS, (30 CHARACTERS )

 

 

 

 

9A

E-MAIL ADDRESS, (30 CHARACTERS )

 

ENTER AN ELECTRONIC MAIL ADDRESS. DEPENDING ON THE ENTRY IN FIELD 7 THE EMAIL ADDRESS  GIVEN EITHER REFERS TO AN ENTITY (FACTORY ETC.), FOR EXAMPLE: "ROBERTS.MACHINES(AT)INDUSTRY.COM" OR AN INDIVIDUAL, FOR EXAMPLE: "PAUL.SMITH(AT)MAILNET.UK". See table 1183/8 which is an instructive entry

10

START DATE-TIME (Z), YYYYMMDDHHMMSS OF A DURATION PERIOD

 

 

 

 

10A

START DATE-TIME (Z), YYYYMMDDHHMMSS OF A DURATION PERIOD

START

ENTER "START:" FOLLOWED BY THE START DAY-TIME ( FORMAT "YYYYMMDDHHMMSS) WHEN THE ADDRESS WAS FIRST ESTABLISHED, FOR EXAMPLE: "START:20011121153621". See table 2417 which is a composite

11

END DATE-TIME (Z), YYYYMMDDHHMMSS OF A DURATION PERIOD

 

 

 

 

11A

END DATE-TIME (Z), YYYYMMDDHHMMSS OF A DURATION PERIOD

END

ENTER "END:" FOLLOWED BY THE END DAY-TIME ( FORMAT "YYYYMMDDHHMMSS") WHEN THE ADDRESS WAS GIVEN UP, FOR EXAMPLE: "END:20011121153621". See table 2417 which is a composite
