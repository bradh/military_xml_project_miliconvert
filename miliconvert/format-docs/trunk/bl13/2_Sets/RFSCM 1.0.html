Structured Language:      

SIGNIFICANT FIELD 8 IN EVERY OCCURRENCE OF FIELD GROUP IN SET ^ (RFSCM) IS REQUIRED IF FIELD 7 IN SET ^ (RFSCM) DOES NOT OCCUR, OTHERWISE IT IS PROHIBITED.
