UNIT DESIGNATOR

 

 

 

 

1A

UNIT DESIGNATOR

 

ENTER THE IDENTIFICATION OF A MILITARY, PARA-MILITARY OR GOVERNMENT AGENCY UNIT, AS USED IN OFFICIAL COMMUNICATIONS WITH MILITARY ESTABLISHMENTS. FOR EXAMPLE: "AATC F-16C". See table 1028/43 which is an instructive entry

2

POC PHONE NUMBER

 

PROVIDES THE PHONE NUMBER FOR THE SADL GATEWAY POC IN ONE OF THE FOLLOWING WAYS:

 

 

2A

NON-SECURE TELEPHONE

TEL

ENTER  "TEL:" FOLLOWED BY THE PHONE NUMBER OF THE POINT OF CONTACT AT UNIT. FOR EXAMPLE: "TEL:DSN 228-4567". See table 1361/3 which is an instructive entry

 

 

2B

SECURE TELEPHONE NUMBER

SECTEL

ENTER "SECTEL:" FOLLOWED BY THE SECURE PHONE NUMBER OF THE POINT OF CONTACT AT UNIT, FOR EXAMPLE: "SECTEL:DSN 228-4567". See table 1361/4 which is an instructive entry

3

PRIMARY JTIDS UNIT (JU) ADDRESS

 

 

 

 

3A

JTIDS UNIT (JU) SOURCE TRACK NUMBER

 

ENTER THE PRIMARY JTIDS UNIT (JU) ADDRESS FOR SADL USE. FOR EXAMPLE: "00025" See table 1625/40 which is an alphanumeric range

4

SADL NETWORK TYPE

 

 

 

 

4A

SADL NETWORK TYPE

 

ENTER THE TYPE OF SADL NETWORK TO BE FORMED. FOR EXAMPLE: "GATEWAY". See table 1378/3 which contains a list of data items and associated data codes

5

SADL DATA LINK MODE

SADL DATA LINK MODE

 

ENTER THE DATA LINK MODE USED IN A SADL NETWORK. FOR EXAMPLE: "ACTIVE". See table 1378/4 which contains a list of data items and associated data codes

SADL AIR KEY

SADL AIR KEY

 

THE KEY FOR ESTABLISHING THE SADL NET PLAYERS THAT WILL BE DISPLAYED. FOR EXAMPLE: "1". See table 1012/604 which is an alphanumeric range

SADL GATEWAY KEY

 

ENTER THE KEY TO SELECT THE SPECIFIC GATEWAY TO BE USED, FOR, EXAMPLE: "2". See table 1012/608 a range [1 through 8]

FREQUENCY CHANNEL

FREQUENCY CHANNEL

SADL FREQUENCY MAP

GATEWAY RANGE SETTING

GATEWAY RANGE SETTING

SADL POWER LEVEL IN WATTS

TCP IP PORT NUMBER

17

UNIT REFERENCE NUMBER

 

 

 

 

17A

UNIT REFERENCE NUMBER_1
