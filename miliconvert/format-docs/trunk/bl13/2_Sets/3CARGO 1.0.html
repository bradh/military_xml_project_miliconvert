CONTEXT QUANTITY, 1-999

2A

NATURE OF CARGO LOAD

 

MAKE AN ABBREVIATED LITERAL ENTRY OF UP TO EIGHT CHARACTERS TO IDENTIFY THE TYPE OF CARGO, FOR EXAMPLE: "TRACKVEH". IF APPLICABLE MAKE THE ENTRIES  IN THE FOLLOWING SEQUENCE:  TRACKED VEHICLES, WHEELED VEHICLES, CONTAINERS, BREAK BULK CARGO - THEN OTHERS, IN SUCCESSIVE ROWS. See table 1867/2 which is an instructive entry
