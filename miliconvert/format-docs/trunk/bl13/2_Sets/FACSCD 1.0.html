No

Designator

Field Desc

Concept/Explanation/Examples

1

NUMBER OF AIRCRAFT

 

 

 

 

1A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT PARTICIPATING IN THE PARTICULAR MISSION, FOR EXAMPLE: "4". See table 1044/322 a range [1 through 99]

2

AIRCRAFT TYPE

 

PROVIDES THE TYPE OF ATTACK AIRCRAFT BY AIRCRAFT BY TYPE, MODEL, CODE OR CATEGORY (ABBREVIATED).

 

 

2A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE AND MODEL NUMBER OF THE AIRCRAFT, FOR EXAMPLE: "ACTYP:FA18". See table 1015/20 which contains a list of data items and associated data codes

 

 

2B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF THE AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST, FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

 

 

2C

AIRCRAFT CATEGORY, ABBREVIATED

CT

ENTER "CT:" FOLLOWED BY THE IDENTIFIER OF THE GENERIC CATEGORY OF THE AIRCRAFT, FOR EXAMPLE: "CT:F". See table 1549/2 which contains a list of data items and associated data codes

3

DTG ON STATION

 

PROVIDES THE DATE-TIME OR THE DAY-TIME ON STATION, OR ALTERNATIVELY THE AIRCRAFT ALERT STATUS AS A CODE OR AS A TIME IN HOURS OR MINUTES.

 

 

3A

DTG ON STATION

TIME

ENTER "TIME:" FOLLOWED BY THE DTG ON STATION, FOR EXAMPLE "TIME:231415ZDEC2003". See table 2033 which is a composite

 

 

3B

DAY-TIME ON STATION

DTOS

ENTER "DTOS:" FOLLOWED BY THE DAY-TIME TO BE ON STATION, FOR EXAMPLE: "DTOS:201235Z". See table 2000 which is a composite

 

 

3C

AIRCRAFT ALERT CODE

AS

ENTER "AS:" FOLLOWED BY THE AIRCRAFT ALERT CODE, FOR EXAMPLE: "AS:RED". See table 1798/2 which contains a list of data items and associated data codes

 

 

3D

AIRCRAFT ALERT STATUS, HOURS

ASH

ENTER "ASH:" FOLLOWED BY THE AIRCRAFT ALERT STATUS IN HOURS, FOR EXAMPLE "ASH:12H". See table 2139 which is a composite

 

 

3E

AIRCRAFT ALERT STATUS, MINUTES

ASM

ENTER "ASM:" FOLLOWED BY THE AIRCRAFT ALERT STATUS IN MINUTES, FOR EXAMPLE "ASM:30M". See table 2142 which is a composite

4

PRIMARY ORDNANCE

 

IF USED, PROVIDES THE PRIMARY ORDNANCE LOAD ON THE ATTACK AIRCRAFT.

 

 

4A

PRIMARY CONFIGURATION LOAD

 

ENTER THE PRIMARY CONFIGURATION FOR THE AIRCRAFT, FOR EXAMPLE: "AC01". See table 1012/258 which is an instructive entry

5

SECONDARY CONFIGURATION LOAD

 

IF USED, PROVIDES THE SECONDARY ORDNANCE LOAD ON THE ATTACK AIRCRAFT.

 

 

5A

SECONDARY CONFIGURATION LOAD

 

ENTER THE SECONDARY CONFIGURATION FOR THE AIRCRAFT, FOR EXAMPLE: "AC02". See table 1012/259 which is an instructive entry

6

AIRCRAFT CALL SIGN

 

 

 

 

6A

ATTACK AIRCRAFT CALL SIGN

 

ENTER THE CALL SIGN OF THE ATTACK AIRCRAFT SCHEDULED TO RENDEZVOUS WITH A FORWARD AIR CONTROLLER, FOR EXAMPLE: "HAMMER 51". See table 1129/37 which is an instructive entry

7

MISSION NUMBER

 

 

 

 

7A

MISSION NUMBER

 

ENTER THE MISSION NUMBER ASSIGNED TO THE SPECIFIC MISSION TO BE CONTROLLED, FOR EXAMPLE: "4010A". See table 1012/69 which is an instructive entry
