1

DEFECTIVE PART NAME

 

 

 

 

1A

EQUIPMENT COMMON NAME

 

ENTER THE NAME OF THE DEFECTIVE PART, FOR EXAMPLE: "LOADING ARM". See table 1022/249 which is an instructive entry

2

DEFECTIVE PART STATUS

 

 

 

 

2A

DEFECTIVE EQUIPMENT STATUS

 

ENTER THE STATUS OF THE DEFECTIVE PART, FOR EXAMPLE: "INOP". See table 1108/2 which contains a list of data items and associated data codes

3

DEFECTIVE PART CODE

 

 

 

 

3A

EQUIPMENT IDENTIFIER CODE

 

ENTER AS A LITERAL THE NATIONAL EQUIPMENT IDENTIFIER CODE OF THE DEFECTIVE PART AT THE LOWEST CODE LEVEL AVAILABLE OR APPROPRIATE, FOR EXAMPLE: "ABC123DEF". See table 1539/3 which is an instructive entry

4

NUMBER OF DEFECTIVE ITEMS

 

 

 

 

4A

NUMBER OF DEFECTIVE ITEMS

 

ENTER THE TOTAL NUMBER OF THE SPECIFIED PARTS THAT ARE DEFECTIVE, FOR EXAMPLE: "1". See table 1044/69 a range [1 through 99]

5

SERIAL NUMBER OR UNIQUE IDENTIFIER

 

 

 

 

5A

EQUIPMENT SERIAL NUMBER

 

ENTER THE SERIAL NUMBER OR OTHER UNIQUE IDENTIFIER OF THE DEFECTIVE PART, REPEATING AS NECESSARY, FOR EXAMPLE: "12345678". See table 1012/10 which is an instructive entry
