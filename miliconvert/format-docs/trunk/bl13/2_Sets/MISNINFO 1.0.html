No

Designator

Field Desc

Concept/Explanation/Examples

1

MISSION NUMBER

 

PROVIDES A NUMBER/CODE IDENTIFYING THE AIR MISSION REPORTED.

 

 

1A

MISSION NUMBER

 

ENTER THE MISSION NUMBER OR CODE, FOR EXAMPLE "HL1033T". See table 1012/69 which is an instructive entry

2

MISSION FLEW INDICATOR

 

PROVIDES INDICATION AS TO WHETHER THE MISSION REPORTED WAS FLOWN, YES OR NO.

 

 

2A

AIR MISSION FLEW INDICATOR

 

ENTER AN INDICATION IF THE MISSION REPORTED WAS FLOWN, YES OR NO, FOR EXAMPLE "YES". See table 1027/85 which contains a list of data items and associated data codes

3

REASON MISSION ABORTED

 

PROVIDES THE REASON THAT CAUSED THE SCHEDULED MISSION NOT TO BE EXECUTED.

 

 

3A

REASON MISSION ABORTED

 

ENTER THE REASON WHY THE MISSION WAS ABORTED, FOR EXAMPLE "TGTORNG". See table 1212/11 which is an instructive entry

4

NUMBER OF AIRCRAFT, 1-99

 

PROVIDES THE NUMBER OF AIRCRAFT INVOLVED IN THE MISSION.

 

 

4A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT INVOLVED IN THE MISSION, FOR EXAMPLE "6". See table 1044/322 a range [1 through 99]

5

PACKAGE IDENTIFICATION

 

 

 

 

5A

PACKAGE IDENTIFICATION

 

ENTER THE IDENTIFICATION OF THE MISSION PACKAGE, FOR EXAMPLE "KA221". See table 1028/38 which is an instructive entry

6

PRIMARY AERIAL MISSION TYPE

 

PROVIDES THE TYPE OF THE AERIAL MISSION REPORTED.

 

 

6A

PRIMARY AERIAL MISSION TYPE

 

ENTER THE TYPE OF AERIAL MISSION REPORTED, FOR EXAMPLE: "ASARS". See table 1285/31 which contains a list of data items and associated data codes
