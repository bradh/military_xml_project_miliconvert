AGENCY ASSUMING CONTROL

 

PROVIDES THE IDENTIFICATION OF THE AGENCY WHO WILL ASSUME THE CONTROL, USING EITHER IDENTIFIER OR NAME OR UIC.

 

 

3A

UNIT IDENTIFIER OF AGENCY ASSUMING CONTROL

CONTUNIT

ENTER "CONTUNIT:" FOLLOWED BY THE LITERAL UNIT ID, FOR EXAMPLE: "CONTUNIT:507TACP". See table 1028/2 which is an instructive entry

ENTER "CONTSHIP:" FOLLOWED BYTHE LITERAL SHIP NAME, FOR EXAMPLE: "CONTSHIP:RANGER". See table 1022/23 which is an instructive entry

UIC OF AGENCY ASSUMING CONTROL
