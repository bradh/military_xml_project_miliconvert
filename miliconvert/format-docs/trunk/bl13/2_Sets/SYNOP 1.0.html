1

ON-STATION CONTACT

 

 

 

 

1A

ON-STATION CONTACT

 

REPORT "SRCH" (SEARCH) IF NO TURNOVER IS RECEIVED WHEN ARRIVING ON STATION OR IF TURNOVER INFORMATION IS MORE THAN 1 HOUR OLD.  USE OTHER ENTRIES IF THE AIRCRAFT ARRIVING RECEIVES A TURNOVER FROM THE AIRCRAFT DEPARTING, FOR EXAMPLE: "VISUAL". See table 1470/2 which contains a list of data items and associated data codes

2

INITIAL CONTACT

 

 

 

 

2A

INITIAL CONTACT

 

ENTER THE INITIAL CONTACT CODE, FOR EXAMPLE: "DP". See table 1470/3 which contains a list of data items and associated data codes

3

BEST CONTACT

 

 

 

 

3A

BEST CONTACT

 

ENTER THE BEST CONTACT CODE, FOR EXAMPLE: "CPA". See table 1470/4 which contains a list of data items and associated data codes

4

OFF-STATION CONTACT

 

 

 

 

4A

OFF-STATION CONTACT

 

ENTER THE OFF STATION CONTACT CODE, FOR EXAMPLE: "CPA". See table 1470/5 which contains a list of data items and associated data codes

5

INITIAL SEARCH AREA

 

 

 

 

5A

INITIAL SEARCH AREA

 

INDICATE THE SOURCE FOR THE INITIAL SEARCH AREA IF THE ENTRY FOR FIELD 1 IS "SRCH" OR "NC", FOR EXAMPLE: "E". IF REPORTED AS "P" (NON-ISSS PROBABILITY AREA) OR "N" (RANDOM SEARCH OR BARRIER), AN AMPN SET MUST FOLLOW THE SYNOP SET TO INDICATE THE SOURCE FOR THE INITIAL SEARCH AREA OR THE BASIS FOR THE RANDOM SEARCH. See table 1398/1 which contains a list of data items and associated data codes

6

SWAP TIME

 

THE STATION SWAP TIME.

 

 

6A

DAY-TIME OF SWAP

 

ENTER THE DAY-TIME, FOR EXAMPLE: "221345Z". See table 2000 which is a composite

7

SEARCH AREA ORIGIN

 

THE GEOGRAPHIC LOCATION OF THE ORIGIN OR BEARING ORIGIN OF THE SEARCH AREA.

 

 

7A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

7B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

7C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

7D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

7E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

7F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

8

BEARING OR SEMI-MAJOR AXIS ORIENTATION

 

THE LINE BEARING OR THE BEARING OF THE SEMI-MAJOR AXIS OF AN ELLIPSE, DEPENDING ON THE ENTRY IN FIELD 5, "E" OR "B".

 

 

8A

BEARING, DEGREES TRUE

 

ENTER THE BEARING IN DEGREES TRUE (T) FROM THE BEARING ORIGIN TO THE TARGET OF INTEREST BY USING THREE NUMERALS PLUS "T", FOR EXAMPLE: "235T". See table 2253 which is a composite

 

 

8B

SEMI-MAJOR AXIS ORIENTATION, DEGREES TRUE

AXIS

ENTER "AXIS:" FOLLOWED BY THE BEARING OF THE SEMI-MAJOR AXIS EXPRESSED AS THREE NUMERALS PLUS "T", FOR EXAMPLE: "AXIS:125T". See table 2256 which is a composite

9

RADIUS OR SEMI-MAJOR AXIS LENGTH

 

THE LENGTH OF THE SEMI-MAJOR AXIS, OR RADIUS OF THE PROBABILITY AREA, DEPENDING ON THE ENTRY IN FIELD 5, "E", "B", OR "P".

 

 

9A

SEMI-MAJOR AXIS LENGTH, NAUTICAL MILES

AXIS

ENTER "AXIS:" FOLLOWED BY THE LENGTH OF THE SEMI-MAJOR AXIS EXPRESSED AS THREE OR FEWER SIGNIFICANT DIGITS PLUS AN OPTIONAL DECIMAL POINT PLUS "NM" (NAUTICAL MILES), FOR EXAMPLE: "AXIS:133NM". See table 2280 which is a composite

 

 

9B

RADIUS, NAUTICAL MILES

 

ENTER THE ZONE RADIUS FROM THE ORIGIN AND UNIT OF MEASUREMENT, FOR EXAMPLE: "25NM". See table 2276 which is a composite

10

BEARING ERROR OR SEMI-MINOR AXIS LENGTH

 

THE LENGTH OF SEMI-MINOR AXIS OR VALUE OF THE BEARING ERROR OF A LINE OF BEARING (LOB) OR SECTOR HALF-WIDTH DISTANCE IN NAUTICAL MILES DEPENDING ON THE ENTRY IN FIELD 5.

 

 

10A

SEMI-MINOR AXIS LENGTH, NAUTICAL MILES

AXIS

ENTER "AXIS:" FOLLOWED BY THE LENGTH OF SEMI-MINOR AXIS EXPRESSED AS THREE OR FEWER SIGNIFICANT DIGITS PLUS AN ADDITIONAL OPTIONAL DECIMAL POINT PLUS "NM" (NAUTICAL MILES), FOR EXAMPLE: "AXIS:123NM". See table 2280 which is a composite

 

 

10B

BEARING ERROR

 

ENTER THE ERROR IN MEASUREMENT ASSOCIATED WITH READING OF BEARINGS, EXPRESSED AS DEGREES (DEG), FOR EXAMPLE: "45". See table 1391/1 which is an alphanumeric range

 

 

10C

SECTOR HALF-WIDTH DISTANCE, NAUTICAL MILES

 

ENTER THE SECTOR HALF WIDTH DISTANCE EXPRESSED AS THREE OR FEWER SIGNIFICANT DIGITS PLUS AN OPTIONAL DECIMAL POINT PLUS "NM" (NAUTICAL MILES), FOR EXAMPLE: "122.4NM" See table 2280 which is a composite
