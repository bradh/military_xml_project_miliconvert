1

MISSION PRIORITY

 

 

 

 

1A

MISSION PRIORITY

PRI

ENTER "PRI:" FOLLOWED BY THE RELATIVE IMPORTANCE OF THE MISSION, FOR EXAMPLE: "PRI:1". See table 1076/2 which contains a list of data items and associated data codes

2

DAY-TIME AND MONTH OVER TARGET

 

 

 

 

2A

DAY-TIME AND MONTH OVER TARGET

 

ENTER THE DAY-TIME AND MONTH OVER THE TARGET, FOR EXAMPLE: "141325ZFEB". See table 2030 which is a composite

3

NET TIME

 

PROVIDES THE DAY-TIME BEFORE WHICH INFORMATION CONCERNING THE MISSION WOULD NOT BE USEFUL.  MUST BE USED IF TIME OVER TARGET IS NOT PROVIDED.

 

 

3A

NOT EARLIER THAN DAY-TIME AND MONTH

NET

ENTER " NET:" FOLLOWED BY THE NOT EARLIER THAN DAY-TIME AND MONTH,  FOR EXAMPLE: "NET:141325ZFEB". See table 2030 which is a composite

 

 

3B

NOT EARLIER THAN DAY-TIME TO THE SECOND, ZULU

NETS

ENTER "NETS:" FOLLOWED BY THE DAY-TIME MEASURED TO THE SECOND, FOR EXAMPLE: "NETS:14132535Z". See table 2483 which is a composite

4

NLT TIME

 

PROVIDES THE DAY-TIME AFTER WHICH INFORMATION CONCERNING THE MISSION WOULD NOT BE USEFUL.  MUST BE USED IF TIME OVER TARGET IS NOT PROVIDED.

 

 

4A

NOT LATER THAN DAY-TIME

NLT

ENTER "NLT:" FOLLOWED BY NOT LATER THAN DAY-TIME, FOR EXAMPLE: "NLT:141325Z". See table 2000 which is a composite

 

 

4B

NOT LATER THAN DAY-TIME TO THE SECOND, ZULU

NLTS

ENTER "NLTS:" FOLLOWED BY THE DAY-TIME MEASURED TO THE SECOND, FOR EXAMPLE: "NLTS:14132535Z". See table 2483 which is a composite

5

DTG OF LATEST TIME OF INTELLIGENCE VALUE

 

 

 

 

5A

DTG OF LATEST TIME OF INTELLIGENCE VALUE

LTIOV

ENTER "LTIOV:" FOLLOWED BY THE DATE-TIME AND YEAR AFTER WHICH THE INFORMATION IS NO LONGER OF VALUE, FOR EXAMPLE: "LTIOV:141325ZFEB2006". See table 2033 which is a composite

6

RECONNAISSANCE MISSION TYPE

 

 

 

 

6A

RECONNAISSANCE MISSION TYPE

 

ENTER THE CODE FOR THE TYPE OF RECONNAISSANCE/SURVEILLANCE MISSION, FOR EXAMPLE: "SLAR". See table 1285/30 which contains a list of data items and associated data codes

7

RECCE DAYS PRIOR OK

 

 

 

 

7A

RECCE DAYS PRIOR OK

 

ENTER THE NUMBER OF DAYS PRIOR TO THE CURRENT DAY THAT RECONNAISSANCE DATA IS ACCEPTABLE, FOR EXAMPLE:"17". See table 1055/17 a range [1 through 99]

8

TYPE OF RECONNAISSANCE/SURVEILLANCE COVERAGE

 

 

 

 

8A

TYPE OF RECONNAISSANCE SURVEILLANCE COVERAGE

 

ENTER THE CODE FOR THE TYPE OF COVERAGE OF THE AREA BY THE RECONNAISSANCE/SURVEILLANCE PLATFORM, FOR EXAMPLE: "AREACOV". See table 1795/1 which contains a list of data items and associated data codes

9

SENSOR TYPE

 

ENTER THE SENSOR TYPE IN ONE OF THE FOLLOWING WAYS:

 

 

9A

SENSOR TYPE, ALPHABETICAL INDICATOR

 

ENTER THE ALPHABETIC CODE FOR THE SENSOR TYPE TO BE USED, FOR EXAMPLE: "SL". See table 1677/7 which contains a list of data items and associated data codes

 

 

9B

SENSOR TYPE, NUMERIC INDICATOR

 

ENTER THE NUMERIC CODE FOR THE SENSOR TYPE TO BE USED, FOR EXAMPLE: "32". See table 1677/10 a range [01 through 99]

10

IMAGE QUALIFIER

 

CAN ONLY BE USED IN CONJUNCTION WITH SPECIFIED SENSOR TYPE WITHIN THIS SET.

 

 

10A

IMAGE QUALIFIER

IMQ

ENTER "IMQ:" FOLLOWED BY THE CODE FOR THE FILM BASED ON TYPE OF IMAGERY THE INTERPRETER IS VIEWING VICE THE ORIGINAL FILM NEGATIVE TYPE, FOR EXAMPLE: "IMQ:MI". See table 1677/6 which contains a list of data items and associated data codes

11

IMAGERY COVERAGE

 

 

 

 

11A

IMAGERY COVERAGE EXTENT AND MODE

CM

ENTER "CM:" FOLLOWED BY THE CODE FOR THE VISIBILITY OF TARGET AND MODE OF COVERAGE, FOR EXAMPLE: "CM:B". See table 1677/4 which contains a list of data items and associated data codes

12

RECONNAISSANCE MISSION PURPOSE

 

 

 

 

12A

RECONNAISSANCE MISSION PURPOSE

 

ENTER THE CODE FOR THE PURPOSE OF THE RECONNAISSANCE MISSION, FOR EXAMPLE: "C". See table 1435/1 which contains a list of data items and associated data codes

13

TYPE OF INTELLIGENCE TASK

 

 

 

 

13A

TYPE OF INTELLIGENCE TASK

 

ENTER THE CODE FOR THE INTELLIGENCE TASK TO BE PERFORMED ON THE RECONNAISSANCE MISSION, FOR EXAMPLE: "DET". See table 1285/28 which contains a list of data items and associated data codes

14

COMPONENT TARGET IDENTIFIER

 

IT PROVIDES THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE SERVICE COMPONENT.

 

 

14A

COMPONENT TARGET IDENTIFIER

CTGT

ENTER "CTGT:" FOLLOWED BY THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE SERVICE COMPONENT, FOR EXAMPLE: "CTGT:A194762". See table 1416/5 which is an instructive entry

 

 

14B

TARGET ENTITY ID REFERENCE NUMBER

EIRN

ENTER "EIRN:" FOLLOWED BY THE EIRN NUMBER TO IDENTIFY A TARGET ON A VARIABLE MESSAGE INTERFACE (VMF) INTERFACE, FOR EXAMPLE: "EIRN:167772104294967290". See table 2085 which is a composite

15

TARGET CATEGORY CODE

 

 

 

 

15A

TARGET CATEGORY CODE

 

ENTER THE NUMERIC CODE THAT REPRESENTS THE GENERAL CATEGORY OF THE TARGET, FOR EXAMPLE: "01". See table 1309/2 which contains a list of data items and associated data codes

16

ESSENTIAL ELEMENTS OF INFORMATION (EEI) OF TARGET CATEGORY

 

 

 

 

16A

ESSENTIAL ELEMENTS OF INFORMATION (EEI) OF TARGET CATEGORY

 

ENTER THE CODE FOR THE TARGET CATEGORY ESSENTIAL ELEMENTS OF INFORMATION (EEI) THAT REPRESENTS THE RECONNAISSANCE OBJECTIVES ASSOCIATED WITH INDIVIDUAL TARGET CATEGORIES CONSIDERED TO BE OF CONTINUING TACTICAL VALUE, FOR EXAMPLE: "01-1". See table 1802/1 which contains a list of data items and associated data codes
