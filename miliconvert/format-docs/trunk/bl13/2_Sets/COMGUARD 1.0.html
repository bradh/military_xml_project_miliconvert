2A

GUARD REQUIREMENTS

 

ENTER 1 TO 3 LETTERS FROM ACP 176 COMMUNICATIONS GUARD TABLE, FOR EXAMPLE: "AS". See table 1246/23 which is an instructive entry

3

KEY MATERIAL

 

THE KEY MATERIAL IN USE ON THE CIRCUIT IDENTIFIED IN FIELD 1, INCLUDING EFFECTIVE EDITION (ALTERNATIVE A).  USE "NA" IF FIELD 1 IS NOT COVERED (ALTERNATIVE B).  OTHERWISE ALTERNATIVE B MAY BE USED AS DEEMED NECESSARY.

 

 

3A

KEYMAT AND CODES

KEY

ENTER "KEY:" FOLLOWED BY THE KEYMAT AND CODES, FOR EXAMPLE: "KEY:AKAT2031EZ". See table 1078/1 which is an instructive entry

 

 

3B

IDENTIFIER, NON-DISCRETE INDICATOR

ID

ENTER "ID:" FOLLOWED BY AN APPROPRIATE IDENTIFIER, FOR EXAMPLE: "ID:ALL". See table 1084/4 which contains a list of data items and associated data codes

4

START TIME

 

THE TIME THE GUARDSHIP DUTIES ARE TO START EXPRESSED IN ONE OF THE FOLLOWING WAYS:

 

 

4A

DTG

 

ENTER THE DTG OF COMMENCEMENT OF GUARDSHIP DUTIES, FOR EXAMPLE: "261200ZJUL2001". See table 2033 which is a composite

 

 

4B

DAY-TIME

 

ENTER THE DAY-TIME OF COMMENCEMENT OF GUARDSHIP DUTIES, FOR EXAMPLE "221200Z". See table 2000 which is a composite

 

 

4C

DAY-TIME AND MONTH

 

ENTER THE DAY-TIME AND MONTH OF COMMENCEMENT OF GUARDSHIP DUTIES, FOR EXAMPLE: "291200ZDEC". See table 2030 which is a composite

5

END TIME

 

THE TIME THE GUARDSHIP DUTIES ARE TO END EXPRESSED IN ONE OF THE FOLLOWING WAYS:

 

 

5A

DTG

 

ENTER THE DTG GUARDSHIP DUTIES ARE TO CEASE, FOR EXAMPLE: "271200ZJUL2001". See table 2033 which is a composite

 

 

5B

DAY-TIME

 

ENTER THE DAY-TIME GUARDSHIP DUTIES ARE TO CEASE, FOR EXAMPLE "230800Z". See table 2000 which is a composite

 

 

5C

DAY-TIME AND MONTH

 

ENTER THE DAY-TIME AND MONTH GUARDSHIP DUTIES ARE TO CEASE, FOR EXAMPLE: "291800ZDEC". See table 2030 which is a composite

6
