Concept/Explanation/Examples

1

WEAPON SYSTEM, CATEGORY

 

 

 

 

1A

WEAPON SYSTEM, CATEGORY

 

ENTER THE WEAPON SYSTEM CATEGORY, FOR EXAMPLE: "TORP". See table 1407/20 which contains a list of data items and associated data codes

2

MARITIME WEAPONS EQUIPMENT

 

ENTER THE WEAPON TYPE AS ONE OF THE FOLLOWING:

 

 

2A

MARITIME WEAPON SYSTEMS

 

ENTER THE MARITIME WEAPON EQUIPMENT TYPE , FOR EXAMPLE: "MK46". See table 1407/54 which contains a list of data items and associated data codes

 

 

2B

WEAPON TYPE, GENERIC

GEN

ONLY TO BE USED IF THE WEAPON TYPE IS NOT AVAILABLE IN ALT A.  ENTER "GEN:" FOLLOWED BY THE WEAPON TYPE, FOR EXAMPLE: "GEN:HARPOON". See table 1407/1 which is an instructive entry

3

NUMBER/PERCENTAGE OF WEAPONS

 

THE NUMBER OR PERCENTAGE OF WEAPONS CARRIED:

 

 

3A

PERCENTAGE OF WEAPONS

PCT

ENTER THE FIELD DESCRIPTOR "PCT:" FOLLOWED BY THE PERCENTAGE OF WEAPONS CARRIED COMPARED TO OPERATIONAL CAPACITY, FOR EXAMPLE: "PCT:25". See table 1064/18 a range [0 through 100]

 

 

3B

NUMBER OF WEAPONS

 

ENTER THE NUMBER OF WEAPONS AS 1 OR 2 DIGITS, FOR EXAMPLE: "2". See table 1044/67 a range [1 through 99]

 

 

Structured Notation:

[^]F3 = FF1064-18 ([^]F1 EQ /SRASW/ | /GUNAMMO/)

 

 

Structured Language:      

FIELD 3 IN SET ^ (WPNDAT) MUST USE ALTERNATIVE FF1064-018 IF FIELD 1 IN SET ^ (WPNDAT) EQUALS /SRASW/ OR /GUNAMMO/.
