RADIUS OF ZONE 1

 

 

 

 

1A

RADIUS OF ZONE 1

 

IF FIELDS 2, 3, AND 4 ARE NOT USED, THEN FIELD 1 MUST BE COMPLETED.  IF USED, ENTER RADIUS OF ZONE 1 (000-999) BASED ON THE LENGTH MEASUREMENT SPECIFIED IN SET UNITM, FOR EXAMPLE "007".  IF FIELD 1 IS NOT USED, THEN ENTER A HYPHEN "-". See table 1523/2 a range [000 through 999]

2

DOWNWIND DIRECTION

 

IF FIELD 1 IS NOT USED, THEN FIELD 2 MUST BE COMPLETED.  ENTER THE DOWNWIND DIRECTION BASED ON THE UNIT OF ANGULAR MEASUREMENT SPECIFIED IN SET UNITM.

 

 

2A

DOWNWIND DIRECTION IN DEGREES

 

ENTER THE DOWNWIND DIRECTION IN DEGREES: (000-360), FOR EXAMPLE: "180". (3N). See table 1043/2 a range [000 through 360]

 

 

2B

DOWNWIND DIRECTION IN MILS

 

ENTER THE WIND DIRECTION IN MILS: (0000-6400), FOR EXAMPLE: "3200". (4N). See table 1048/4 a range [0000 through 6400]

3

WIND SPEED

 

 

 

 

3A

WIND SPEED

 

IF FIELD 1 IS NOT USED, THEN FIELD 3 MUST BE COMPLETED.  ENTER THE WIND SPEED (000-999) BASED ON THE UNIT OF SPEED MEASUREMENT SPECIFIED IN SET UNITM, FOR EXAMPLE: "015". See table 1525/2 a range [000 through 999]

4

ANGLE EXPANSION INDICATOR

 

 

 

 

4A

ANGLE EXPANSION INDICATOR

 

IF FIELD 1 IS NOT USED, THEN FIELD 4 MUST BE COMPLETED.  ENTER THE ANGLE EXPANSION INDICATOR BETWEEN THE RADIAL LINES OF A NUCLEAR PREDICTION FALLOUT PATTERN, FOR EXAMPLE: "4". See table 1330/1 which contains a list of data items and associated data codes
