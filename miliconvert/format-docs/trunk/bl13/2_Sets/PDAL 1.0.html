1

DEFENDED ASSET NAME

 

 

 

 

1A

DEFENDED ASSET NAME

 

ENTER THE NAME OF THE DEFENDED ASSET, FOR EXAMPLE: "3RD WING COMMAND BUNKER". See table 1022/36 which is an instructive entry

2

DEFENDED ASSET SPECIFICATION

 

 

 

 

2A

DEFENDED ASSET SPECIFICATION

 

ENTER THE APPROPRIATE THREE LETTER CODE TO SPECIFY WHETHER THE ASSET IS TO BE DEFENDED AGAINST CONVENTIONAL THREAT OR BALLISTIC AND CRUISE MISSILES, FOR EXAMPLE "CON". See table 1111/1 which contains a list of data items and associated data codes

3

DEFENDED ASSET PRIORITY

 

 

 

 

3A

DEFENDED ASSET PRIORITY

 

ENTER THE PRIORITY OF THE DEFENDED ASSET, FOR EXAMPLE: "13". See table 1076/21 a range [1 through 999]

4

PLACE NAME

 

 

 

 

4A

PLACE NAME

 

ENTER THE NAME OF THE PLACE OF THE DEFENDED TARGET, FOR EXAMPLE: "CHARLEROI AFB". See table 1022/170 which is an instructive entry

5

GEGRAPHIC POSITION

 

THE GEOGRAPHIC LOCATION OF PRIORITISED DEFENDED ASSETS.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

6

LEVEL OF PROTECTION

 

 

 

 

6A

DEFENDED ASSET LEVEL OF PROTECTION

 

ENTER THE LEVEL OF PROTECTION TO BE PROVIDED TO THE DEFENDED ASSET AS AGREED BY THE JDAWG, RATING FROM LEVEL ONE TO LEVEL FOUR, FOR EXAMPLE: "2". See table 1424/4 which contains a list of data items and associated data codes

7

ENFORCEMENT START TIME

 

THE TIME FROM WHICH DEFENSE NEEDS TO BE ACHIEVED, USING EITHER DTG OR DAY TIME AND MONTH.

 

 

7A

EFFECTIVE DTG VALID FROM, ZULU

DTGFM

ENTER "DTGFM:" FOLLOWED BY THE EFFECTIVE DTG FROM WHICH DEFENSE NEEDS TO BE ACHIEVED, FOR EXAMPLE: "DTGFM:052340ZNOV2004". See table 2323 which is a composite

 

 

7B

EFFECTIVE DAY-TIME AND MONTH

DTMFM

ENTER "DTMFM:" FOLLOWED BY THE EFFECTIVE DAY TIME AND MONTH FROM WHICH DEFENSE NEEDS TO BE ACHIEVED, FOR EXAMPLE:"DTMFM:052340ZNOV". See table 2030 which is a composite

8

ENFORCEMENT CEASE TIME

 

THE TIME AT WHICH DEFENSE NEEDS TO BE CEASED, USING EITHER DTG OR DAY TIME AND MONTH.

 

 

8A

EFFECTIVE DTG VALID TO, ZULU

DTGTO

ENTER "DTGTO:" FOLLOWED BY THE DTG  AT WHICH DEFENSE NEEDS TO BE CEASED, FOR EXAMPLE:"DTGTO:061215ZNOV2004". See table 2323 which is a composite

 

 

8B

EFFECTIVE DAY-TIME AND MONTH

DTMTO

ENTER "DTMTO:" FOLLOWED BY THE TIME AT WHICH DEFENSE NEEDS TO BE CEASED, FOR EXAMPLE:"DTMTO:061215ZNOV". See table 2030 which is a composite

9

DEFENDED ASSET LIMITATION

 

 

 

 

9A

LIMITATIONS

 

ENTER THE LIMITATIONS FOR THE DEFENDED ASSET, FOR EXAMPLE: "NO FLIGHT OVER THE VERTICAL". See table 1357/15 which is an instructive entry

10

DEFENDED ASSET NOMINATING AUTHORITY

 

 

 

 

10A

DEFENDED ASSET NOMINATING AUTHORITY

 

ENTER THE COMMAND AND CONTROL AUTHORITY THAT PROVIDES THE DEFENDED ASSET NOMINATION, FOR EXAMPLE :"NAC". See table 1874/12 which is an instructive entry

11

COMMENTS

 

 

 

 

11A

COMMENTS

 

ENTER ADDITIONAL ESSENTIAL INFORMATION RELATED SPECIFICALLY TO THE DEFENDED ASSET, FOR EXAMPLE: "ASSET IS LOCATED NEAR INTERNATIONAL AIRPORT". See table 1357/1 which is an instructive entry
