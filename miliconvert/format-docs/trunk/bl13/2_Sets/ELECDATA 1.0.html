ELINT NOTATION

 

 

 

 

1A

ELINT NOTATION

 

ENTER THE ELINT NOTATION (ELNOT) THAT IS SPECIFIED IN THE NATO EMITTER DATA BASE (NEDB) STANDARD AND PROCEDURES, FOR EXAMPLE: "50959". See table 1012/34 which is an instructive entry

2

ELNOT CONFIDENCE

 

 

 

 

2A

ELNOT CONFIDENCE

 

ENTER THE  CONFIDENCE LEVEL (SUBJECTIVE ASSESSMENT BETWEEN 1 [LOW] AND 10 [HIGH]) THE REPORT DRAFTER ASSIGNS TO THE FACT THAT AN OBSERVED SIGNAL AND AN ELINT NOTATION HAVE BEEN CORRECTLY ASSOCIATED, FOR EXAMPLE: "4". See table 1123/7 a range [1 through 10]

3

EMITTER IDENTIFIER

 

 

 

 

3A

EMITTER IDENTIFIER

 

ENTER A NUMERIC CODE TO IDENTIFY A SPECIAL TRANSMITTER, FOR EXAMPLE: "1,001,897". See table 1246/41 which is an instructive entry

4

REMARKS

 

ENTER ANY OTHER DATA (LITERAL UP TO 68 CHARACTERS) WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS. See table 1357/28 which is an instructive entry
