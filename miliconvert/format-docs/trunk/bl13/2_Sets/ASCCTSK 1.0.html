4

PRIMARY COMMUNICATION PARAMETERS

 

PROVIDES THE PRIMARY COMMUNICATION PARAMETERS FOR CONTACTING THE CONTROL AGENCY, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.

 

 

4A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY TO BE USED BY THE CONTROL AGENCY, FOR EXAMPLE: "PFREQ:342.5". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

4B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR TO BE USED BY THE CONTROL AGENCY, FOR EXAMPLE: "PDESIG:TAD001". See table 1396/2 which is an instructive entry

 

 

4C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQKM:AB12-CD34-12082000-3EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

4D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQK:AB12-CD34-12082000". See table 2535 which is a composite

5

SECONDARY COMMUNICATION PARAMETERS

 

PROVIDES THE SECONDARY COMMUNICATION PARAMETERS FOR CONTACTING THE SUPPORTING MISSION, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.  CAN ONLY BE USED IN CONJUNCTION WITH PRIMARY COMMUNICATION PARAMETERS SPECIFIED IN THIS SET.

 

 

5A

SECONDARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

SFREQ

ENTER "SFREQ:" FOLLOWED BY THE SECONDARY FREQUENCY TO BE USED BY THE CONTROL AGENCY, FOR EXAMPLE: "SFREQ:350.5". See table 1068/14 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)
