EXERCISE NICKNAME

EXERCISE IDENTIFIER

 

THIS FIELD PROVIDES ADDITIONAL INFORMATION FOR THE CONDUCT OF THE EXERCISE.

EXERCISE ADDITIONAL IDENTIFIER

 

ENTER THE EXERCISE ADDITIONAL IDENTIFIER, FOR EXAMPLE: "DISTAFF". See table 1018/11 which contains a list of data items and associated data codes

EXERCISE ADDITIONAL NICKNAME

NICK
