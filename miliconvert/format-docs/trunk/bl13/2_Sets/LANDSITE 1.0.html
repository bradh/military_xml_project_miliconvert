LANDING SITE LOCATION

 

PROVIDES THE LANDING SITE LOCATION.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

1B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

1C

BEARING AND DISTANCE FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

1D

LOCATION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME OF THE LOCATION FOR EXAMPLE: "LOC:FIRE BASE ALFA". See table 1022/4 which is an instructive entry

 

 

1E

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE INTERNATIONAL CIVIL AVIATION ORGANISATION (ICAO) INDICATOR, FOR EXAMPLE: "ICAO:KPOB". See table 1125/1 which is an instructive entry

RUN IN HEADING IN DEGREES MAGNETIC

 

 

 

 

2A

RUN IN HEADING IN DEGREES MAGNETIC

 

ENTER THE RUN IN HEADING IN DEGREES MAGNETIC FOR THE APPROACH, FOR EXAMPLE: "320". See table 1804/4 a range [000 through 359]

3

LANDING SITE LENGTH IN METRES

 

 

 

 

3A

LANDING SITE LENGTH IN METRES

 

ENTER THE LANDING SITE LENGTH IN METRES, FOR EXAMPLE: "300". See table 1657/16 a range [1 through 9999]

4

LANDING SITE WIDTH IN METRES

 

 

 

 

4A

LANDING SITE WIDTH IN METRES

 

ENTER THE LANDING SITE WIDTH IN METRES, FOR EXAMPLE: "90". See table 1658/7 a range [1 through 9999]

ENEMY ACTION STATUS

 

 

 

 

5A

ENEMY ACTION STATUS

 

ENTER THE CODE TO INDICATE THE GENERAL STATUS OF ENEMY ACTION AT THE LANDING SITE, FOR EXAMPLE: "COLD". See table 1797/1 which contains a list of data items and associated data codes

6

DAY-TIME AS OF

 

 

 

 

6A

DAY-TIME AS OF

 

ENTER THE DAY-TIME WHEN THE INFORMATION REPORTED IN FIELD 5 WAS OR WILL BE APPLICABLE, FOR EXAMPLE: "141325Z". See table 2000 which is a composite

7

FRIENDLY POSITIONS MARKING METHOD

 

 

 

 

7A

METHOD OF MARKING FRIENDLY POSITIONS

 

ENTER THE METHOD OF MARKING THE LANDING SITE, FOR EXAMPLE: "PANELS". See table 1613/2 which contains a list of data items and associated data codes

8

COLOUR

 

CAN ONLY BE USED IN CONJUNCTION WITH FRIENDLY POSITIONS MARKING METHOD SPECIFIED IN THIS SET.

 

 

8A

COLOUR

 

ENTER THE COLOUR OF THE MARKINGS, FOR EXAMPLE: "RED". See table 1138/1 which contains a list of data items and associated data codes

9

CONTACT CALL SIGN OR NAME

 

THE NAME OR CALL SIGN OF THE LANDING SITE CONTACT.

 

 

9A

CONTACT CALL SIGN

CONTACT

ENTER "CONTACT:" FOLLOWED BY THE CALL SIGN OF THE UNIT TO BE CONTACTED, FOR EXAMPLE: "CONTACT:RAYMOND 25". See table 1129/31 which is an instructive entry

 

 

9B

CONTACT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE PERSON TO BE CONTACTED, FOR EXAMPLE: "NAME:LT YO". See table 1022/7 which is an instructive entry
