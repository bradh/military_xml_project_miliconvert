GEOGRAPHIC ELEVATION

 

ALTERNATE FIELDS CONTAINING A FIGURE FOR THE GEOGRAPHIC ELEVATION OR A CODE INDICATING A LACK OF DATA:

 

 

1A

ELEVATION

ELEV

ENTER "ELEV:" FOLLOWED BY A FIGURE GIVING THE GROUND ELEVATION OF THE GEOGRAPHIC GROUND COORDINATES ABOVE OR BELOW THE VERTICAL DATUM SPECIFIED IN FIELD 5, FOR EXAMPLE: "ELEV:-250". See table 1720/2 a range [-9999 through 99999]

2

ELEVATION MEASUREMENT ACCURACY

 

 

 

 

2A

ELEVATION MEASUREMENT ACCURACY

ELACC

ENTER "ELACC:" FOLLOWED BY A NUMERIC VALUE GIVING THE PLUS OR MINUS ACCURACY (ABSOLUTE VALUE OF THE ERROR) TO WHICH THE ELEVATION MEASUREMENT IS EXPRESSED, FOR EXAMPLE: "ELACC:10". See table 1811/2 a range [0 through 9999]

3

UNIT OF LINEAR MEASUREMENT, GEOGRAPHIC

 

 

 

 

3A

UNIT OF LINEAR MEASUREMENT

 

ENTER THE UNIT OF MEASUREMENT FOR THE FIGURES GIVEN IN FIELDS 1 AND 2, FOR EXAMPLE: "M". See table 1008/4 which contains a list of data items and associated data codes

4

ELEVATION CONFIDENCE LEVEL

 

 

 

 

4A

ELEVATION CONFIDENCE LEVEL

ELEVCONF

ENTER "ELEVCONF:" FOLLOWED BY A PERCENTAGE FIGURE BETWEEN 0 AND 100 EXPRESSING THE CONFIDENCE THAT THE DRAFTER HAS IN THE MEASUREMENT GIVEN, FOR EXAMPLE: "ELEVCONF:88". See table 1123/8 a range [00 through 99]

5

MAP DATUM

 

 

 

 

5A

MAP DATUM

 

ENTER A THREE DIGIT CODE INDICATING THE MAP DATUM ON WHICH THE ELEVATION VALUE IS BASED, FOR EXAMPLE: "APN". See table 1195/3 which contains a list of data items and associated data codes

6

METHOD OF ELEVATION DERIVATION

 

 

 

 

6A

METHOD OF ELEVATION DERIVATION

 

ENTER A CODE TO DESCRIBE THE METHOD THAT WAS USED TO DETERMINE OR TO DERIVE THE ELEVATION, FOR EXAMPLE: "K". See table 1866/2 which contains a list of data items and associated data codes

7

ELEVATION DERIVATION ACCURACY
