Concept/Explanation/Examples

1

SEQUENTIAL NUMBER ALLOCATED TO  AIRCRAFT.

 

 

 

 

1A

SERIAL NUMBER OF REQUEST

 

ENTER THE SEQUENTIAL NUMBER AND WHERE POSSIBLE LISTED IN DTG ORDER. AIRCRAFT REQUESTED SHOULD EACH BE GIVEN AS SEQUENTIAL NUMBER. THE AIRCRAFT TASKING AUTHORITY WILL BE RESPONSIBLE FOR PROVIDING THE AIRCRAFT AND GIVING IT A MISSION NUMBER. THE AIRCRAFT ALLOCATION WILL BE PUBLISHED IN THE RELEVANT AIRCRAFT TASKING SIGNAL. NOTE - SUFFICIENT TIME SHOULD BE GIVEN TO ALLOW THE TASKING AUTHORITY TO PRIORITIZE THE REQUIREMENTS AND ALLOCATE THE AIRCRAFT, FOR EXAMPLE "1". (1-3N) See table 1012/68 a range [1 through 999]

2

ASW HELICOPTER ROLE

 

 

 

 

2A

ASW HELICOPTER ROLE

 

ENTER THE TYPE OF HELICOPTER REQUESTED, FOR EXAMPLE: "DIPPER". See table 1010/8 which contains a list of data items and associated data codes

3

QUANTITY OF HELICOPTERS REQUIRED

 

 

 

 

3A

QUANTITY OF HELICOPTERS REQUIRED

 

ENTER THE QUANTITY OF HELICOPTERS REQUIRED, FOR EXAMPLE: "2". See table 1044/370 a range [0 through 9]

4

AIRCRAFT ALERT TIME IN MINUTES

 

 

 

 

4A

AIRCRAFT ALERT TIME IN MINUTES

ALERT

ENTER "ALERT:" FOLLOWED BY THE REQUIRED AIRCRAFT ALERT STATE IN MINUTES, FOR EXAMPLE: "ALERT:15" See table 1072/32 which contains a list of data items and associated data codes

5

DTG OF START

 

 

 

 

5A

DTG OF START

 

ENTER THE DTG FOR THE TIME THAT THE HEC IS TO COMMENCE PROVIDING AIRCRAFT ON ALERT, FOR EXAMPLE: "120600ZMAR2005". See table 2033 which is a composite

6

DTG OF STOP

 

 

 

 

6A

DTG OF STOP

 

ENTER THE DTG FOR THE TIME THAT THE HEC IS TO STOP PROVIDING AIRCRAFT ON ALERT, FOR EXAMPLE: "141800ZMAR2005". See table 2033 which is a composite

7

DAILY START TIME

 

 

 

 

7A

DAILY START TIME

 

IF REQUIRED, ENTER THE TIME THE AIRCRAFT ARE REQUIRED TO BE ON ALERT FROM DAILY,  FOR EXAMPLE, "0600Z".  IF 24 HOUR COVERAGE IS REQUIRED THEN THIS FIELD SHOULD BE OMITTED . See table 2485 which is a composite

8

DAILY STOP TIME

 

 

 

 

8A

DAILY STOP TIME

 

IF REQUIRED, ENTER THE TIME THE AIRCRAFT ARE REQUIRED TO STOP BEING ON ALERT DAILY, FOR EXAMPLE, "1800Z".  IF 24 HOUR COVERAGE IS REQUIRED THEN THIS FIELD SHOULD BE OMITTED. See table 2485 which is a composite
