Designator

Field Desc

Concept/Explanation/Examples

1

THE FORCE DISPOSITION TYPE.

 

 

 

 

1A

DISPOSITION TYPE

 

ENTER THE FORCE DISPOSITION, FOR EXAMPLE: "1RR". See table 1541/1 which contains a list of data items and associated data codes

2

NAME OF DISPOSITION

 

 

 

 

2A

DISPOSITION NAME

 

ENTER THE NAME OF THE FORCE DISPOSITION, FOR EXAMPLE: "SHARK". See table 1022/83 which is an instructive entry

3

GUIDE OR REFERENCE

 

THE GEOGRAPHIC POSITION OF THE UNIT OR THE REFERENCE POINT THAT UNITS WILL FORM ON.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

3C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

3D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

3E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

3F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

3G

FORMATION GUIDE NAME

GUIDE

ENTER "GUIDE:" FOLLOWED BY THE NAME OF THE FORMATION GUIDE, FOR EXAMPLE: "GUIDE:HURON". See table 1022/145 which is an instructive entry

 

 

3H

BEARING AND DISTANCE IN NM FROM REFERENCE POINT

BARN

ENTER "BARN:" FOLLOWED BY THE BEARING AND DISTANCE FROM THE REF POINT, FOR EXAMPLE: "BARN:270T-ZZ-2.5NM". See table 2307 which is a composite

 

 

3I

REFERENCE POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE REFERENCE POINT NAME, FOR EXAMPLE: "NAME:ALFA ALFA". See table 1022/2 which is an instructive entry

4

COURSE AND SPEED OF FORCE.

 

THE FORMATION COURSE AND SPEED OF THE FORCE.

 

 

4A

COURSE AND SPEED

CAS

ENTER "CAS:" FOLLOWED BY THE COURSE AND SPEED, FOR EXAMPLE: "CAS:180T-15KTS". See table 2039 which is a composite

 

 

4B

IDENTIFIER, NON-DISCRETE INDICATOR

 

ENTER AN IDENTIFIER TO INDICATE THAT THE COURSE AND SPEED INFORMATION IS TO BE ESTABLISHED BY OTHER MEANS, FOR EXAMPLE: "AS SIGNALLED". See table 1084/4 which contains a list of data items and associated data codes

5

THE GENERAL AXIS ALONG WHICH A FORMATION IS TO FORM.

 

 

 

 

5A

AXIS, QUALIFIED

 

ENTER THE GENERAL AXIS ALONG WHICH A FORMATION IS TO FORM, FOR EXAMPLE: "045T". See table 2159 which is a composite

6

INTERVAL

 

THE INTERVAL BETWEEN VESSELS OR GROUPS (WITHIN THE DISPOSITION SPECIFIED IN FIELD 1).

 

 

6A

INTERVAL/SPACING DISTANCE

 

ENTER THE SPACING OF FORMATION LINES OR CIRCLES INCLUDING UNITS, FOR EXAMPLE: "2.5KYD". See table 2158 which is a composite

 

 

6B

IDENTIFIER, NON-DISCRETE INDICATOR

 

ENTER THE DESIGNATOR INDICATING THE INTERVAL OR SPACING IS TO BE ESTABLISHED BY OTHER MEANS, FOR EXAMPLE: "AS SIGNALLED". WHEN "OTHER" IS USED, AN AMPN SET IS REQUIRED. See table 1084/4 which contains a list of data items and associated data codes

7

ASSIGNED SHIP OR FORCE

 

THE SHIP OR FORCE WHICH WILL BE ASSIGNED A STATION.

 

 

7A

UNIT

 

ENTER THE NAME OF THE SHIP TO BE ASSIGNED A STATION, FOR EXAMPLE: "ARK ROYAL". See table 1022/57 which is an instructive entry

 

 

7B

NAVAL TASK DESIGNATOR

NTD

ENTER "NTD:" FOLLOWED BY THE NAVAL TASK DESIGNATOR OF THE FORCE TO BE ASSIGNED A STATION, FOR EXAMPLE: "NTD:TU401.2.1". See table 2143 which is a composite

8

THE STATION ASSIGNMENT FOR THE SHIP OR FORCE.

 

 

 

 

8A

STATION ASSIGNMENT NAME

 

ENTER THE STATION ASSIGNMENT FOR THE SHIP OR FORCE IDENTIFIED IN FIELD 7, FOR EXAMPLE: "3". See table 1022/124 which is an instructive entry
