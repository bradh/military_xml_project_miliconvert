AIRCRAFT CODE

AIRCRAFT CODE

 

ENTER CODE OF THE DEPARTED AIRCRAFT, FOR EXAMPLE: "FLO". See table 1015/2 which contains a list of data items and associated data codes

2

AIRCRAFT MODEL AND MARK DESIGNATOR

 

 

 

 

2A

AIRCRAFT MODEL AND MARK DESIGNATOR

 

ENTER DESIGNATOR [LITERAL] FOR MODEL AND MARK OR VARIANT, OF THE AIRCRAFT DEPARTED FROM THE AIRFIELD, FOR EXAMPLE: "H". See table 1015/11 which is an instructive entry

3

THREAT AIRCRAFT PRIMARY ROLE

 

 

 

 

3A

THREAT AIRCRAFT PRIMARY ROLE

 

ENTER THREAT AIRCRAFT PRIMARY ROLE OF THE AIRCRAFT BEING REPORTED, FOR EXAMPLE: "FTG". See table 1010/2 which contains a list of data items and associated data codes

4

NUMBER OF AIRCRAFT INVOLVED

 

 

 

 

4A

NUMBER OF AIRCRAFT INVOLVED

 

ENTER THE NUMBER OF AIRCRAFT [1-999] DEPARTING FROM THE AIRFIELD SPECIFIED IN THE ASSOCIATED AIRFIELD SET, FOR EXAMPLE: "5". See table 1044/2 a range [0 through 999]
