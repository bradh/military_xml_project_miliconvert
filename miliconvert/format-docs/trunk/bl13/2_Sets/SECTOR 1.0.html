Field Desc

Concept/Explanation/Examples

1

AREA NAME

 

 

 

 

1A

AREA NAME

 

ENTER THE NAME OF THE AREA DEFINED BY THE SECTOR, FOR EXAMPLE: "GREENFIELD 11". IF THE AREA IS UNDESIGNATED, ENTER "UNNAMED AREA". See table 1022/69 which is an instructive entry

2

GEOGRAPHIC LOCATION

 

THE GEOGRAPHIC LOCATION OR OTHER POINT UPON WHICH THE SECTOR IS CENTRED.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

GEOGRAPHIC POSITION, GEOREF, CENTIMINUTE

GEO

ENTER "GEO:" FOLLOWED BY THE SECTOR CENTRE LOCATION AS GEOREF, CENTIMINUTE, FOR EXAMPLE: "GEO:ABAB11223344". See table 2018 which is a composite

 

 

2H

POSITION OR POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE PRE-DEFINED NAME OR DESIGNATOR FOR THE SECTOR CENTRE, FOR EXAMPLE: "NAME:AA". See table 1022/18 which is an instructive entry

 

 

2I

TRUE BEARING AND DISTANCE IN NM FROM REFERENCE POINT

TBRN

ENTER "TBRN:" FOLLOWED BY A LOCATION EXPRESSED AS A TRUE BEARING AND DISTANCE IN NAUTICAL MILES FROM A SPECIFIED REFERENCE POINT, FOR EXAMPLE: "TBRN:205-PORTLAND BILL LT-34.5". See table 2016 which is a composite

 

 

2J

UNIT NAME

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OR IDENTIFIER OF THE UNIT UPON WHICH THE SECTOR IS CENTRED, FOR EXAMPLE: "UNIT:JOHAN DE WITT". See table 1022/48 which is an instructive entry

 

 

2K

ATP-1 POSITION DESIGNATOR

RP

ENTER "RP:" FOLLOWED BY THE ATP-1 POSITION DESIGNATOR, FOR EXAMPLE: "RP:ZZ". See table 1612/1 which contains a list of data items and associated data codes

 

 

2L

PLACE NAME, EXTENDED

PLACE

ENTER "PLACE:" FOLLOWED BY THE EXTENDED  NAME OF THE ASSOCIATED PLACE, FOR EXAMPLE: "PLACE:BUOY 10A". See table 1022/261 which is an instructive entry

 

 

2M

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES VERIFIED

SUB

ENTER "SUB:" FOLLOWED BY THE LATITUDE AND LONGITUDE UP TO 1/10,000 OF A MINUTE, VERIFIED, FOR EXAMPLE: "SUB:2010.30N6-01040.20W7". See table 2577 which is a composite

3

INNER RADIUS

 

 

 

 

3A

INNER RADIUS, NAUTICAL MILES

 

ENTER THE SECTOR INNER RADIUS FOLLOWED BY "NM" TO INDICATE MEASUREMENT IN NAUTICAL MILES, FOR EXAMPLE: "12NM". See table 2276 which is a composite

4

OUTER RADIUS

 

 

 

 

4A

OUTER RADIUS, NAUTICAL MILES

 

ENTER THE SECTOR OUTER RADIUS FOLLOWED BY "NM" TO INDICATE MEASUREMENT IN NAUTICAL MILES, FOR EXAMPLE: "30NM". See table 2276 which is a composite

5

LEFT AREA BOUNDARY

 

 

 

 

5A

LEFT AREA BOUNDARY IN DEGREES TRUE

 

ENTER THE LEFT-MOST SECTOR BEARING IN DEGREES TRUE, FOR EXAMPLE: "235". See table 1791/7 a range [000 through 360]

6

RIGHT AREA BOUNDARY

 

 

 

 

6A

RIGHT AREA BOUNDARY IN DEGREES TRUE

 

ENTER THE RIGHT-MOST SECTOR BEARING IN DEGREES TRUE, FOR EXAMPLE: "300". See table 1791/8 a range [000 through 360]
