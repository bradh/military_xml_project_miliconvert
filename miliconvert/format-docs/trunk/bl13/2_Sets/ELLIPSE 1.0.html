Designator

Field Desc

Concept/Explanation/Examples

1

AREA NAME

 

 

 

 

1A

AREA NAME

 

ENTER THE NAME OR IDENTIFIER OF THE ASSOCIATED AREA DEFINED BY THE ELLIPSE, FOR EXAMPLE: "SEARCH 03". IF THE AREA IS UNDESIGNATED, ENTER "UNNAMED AREA". See table 1022/69 which is an instructive entry

2

LOCATION

 

THE GEOGRAPHIC LOCATION OF THE ELLIPTIC AREA OVERLAY.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

2D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

2E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

2F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

2G

PLACE NAME, EXTENDED

NAME

ENTER "NAME:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "NAME:DELTA3". See table 1022/261 which is an instructive entry

 

 

2H

POSITION, TRUE BEARING AND RANGE IN NM

BRGR

ENTER "BRGR:" FOLLOWED BY THE POSITION OF THE CENTRE OF THE ELLIPSE EXPRESSED AS A TRUE BEARING AND RANGE IN NM FROM A NAMED REFERENCE POINT, FOR EXAMPLE: "BRGR:181-SALAM-123". See table 2281 which is a composite

 

 

2I

UNIT

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT NAME, FOR EXAMPLE: "UNIT:INVINCIBLE". See table 1022/57 which is an instructive entry

 

 

2J

Q-ROUTE POSITION

QPOS

ENTER "QPOS:" FOLLOWED BY THE Q-ROUTE POSITION, FOR EXAMPLE: "QPOS:121R1234PS121". See table 2203 which is a composite

3

SEMI-MAJOR AXIS ORIENTATION, DEGREES TRUE

 

 

 

 

3A

SEMI-MAJOR AXIS ORIENTATION, DEGREES TRUE

 

ENTER THE BEARING OF SEMI-MAJOR AXIS (THE LONGER AXIS OF THE ELLIPSE) TO THE NEAREST DEGREE, TRUE, FOR EXAMPLE: "147T". See table 2256 which is a composite

4

SEMI-MAJOR AXIS LENGTH, NAUTICAL MILES

 

 

 

 

4A

SEMI-MAJOR AXIS LENGTH, NAUTICAL MILES

 

ENTER THE LENGTH OF THE SEMI-MAJOR AXIS (THE LONGER AXIS OF THE ELLIPSE) IN NAUTICAL MILES, FOR EXAMPLE: "29NM". See table 2280 which is a composite

5

SEMI-MINOR AXIS LENGTH, NAUTICAL MILES

 

 

 

 

5A

SEMI-MINOR AXIS LENGTH, NAUTICAL MILES

 

ENTER THE LENGTH OF THE SEMI-MINOR AXIS (THE SHORTER AXIS OF THE ELLIPSE) IN NAUTICAL MILES, FOR EXAMPLE: "19NM". See table 2280 which is a composite
