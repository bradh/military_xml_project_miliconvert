Designator

Field Desc

Concept/Explanation/Examples

1

DEPARTURE LOCATION IDENTIFIER

 

PROVIDES THE SHIP IDENTIFICATION OR THE ICAO LOCATION INDICATOR IN ONE OF THE TWO FOLLOWING WAYS:

 

 

1A

SHIP IDENTIFICATION

 

ENTER THE  DEPARTURE LOCATION IDENTIFIER OF THE AIRCRAFT BY ENTERING SHIP IDENTIFICATION OR ICAO LOCATION IDENTIFIER, FOR EXAMPLE: "KYAD". See table 2021 which is a composite

 

 

1B

ICAO LOCATION INDICATOR

 

ENTER THE ICAO LOCATION INDICATOR, FOR EXAMPLE: "BIKF". See table 1125/1 which is an instructive entry

2

DEPARTURE TIME

 

THE DEPARTURE TIME EXPRESSED IN EITHER DAY-TIME OR DAY TIME AND MONTH

 

 

2A

DAY-TIME OF DEPARTURE

 

ENTER THE DAY-TIME OF DEPARTURE, FOR EXAMPLE: "230200Z". See table 2000 which is a composite

 

 

2B

DAY-TIME AND MONTH OF DEPARTURE

 

ENTER THE DAY-TIME MONTH OF DEPARTURE, FOR EXAMPLE: "020345ZMAR". See table 2030 which is a composite

3

ON STATION TIME

 

PROVIDES THE ON STATION TIME IN ONE OF THE TWO FOLLOWING WAYS:

 

 

3A

DAY-TIME ON STATION

 

ENTER THE ON STATION DAY-TIME, FOR EXAMPLE: "251230Z". See table 2000 which is a composite

 

 

3B

DAY-TIME AND MONTH ON STATION

 

ENTER THE DAY-TIME MONTH ON STATION, FOR EXAMPLE: "121030ZFEB". See table 2030 which is a composite

4

OFF STATION TIME

 

PROVIDES THE OFF STATION TIME IN ONE OF THE TWO FOLLOWING WAYS:

 

 

4A

DAY-TIME OFF STATION

 

ENTER THE OFF STATION DAY-TIME, PLUS TIME ZONE, FOR EXAMPLE: "120900Z". See table 2000 which is a composite

 

 

4B

DAY-TIME AND MONTH OFF STATION

 

ENTER THE DAY-TIME AND MONTH OFF STATION, FOR EXAMPLE "121340ZMAR". See table 2030 which is a composite

5

ARRIVAL TIME

 

PROVIDES THE ARRIVAL TIME IN ONE OF THE TWO FOLLOWING WAYS:

 

 

5A

DAY-TIME OF ARRIVAL

 

ENTER THE DAY-TIME OF ARRIVAL, PLUS TIME ZONE, FOR EXAMPLE: "251315Z". See table 2000 which is a composite

 

 

5B

DAY-TIME AND MONTH OF ARRIVAL

 

ENTER THE DATE-TIME AND MONTH OF ARRIVAL, FOR EXAMPLE: "121031ZFEB". See table 2030 which is a composite

6

ARRIVAL LOCATION IDENTIFIER

 

PROVIDES THE SHIP IDENTIFICATION OR ICAO LOCATION INDICATOR IN ONE OF THE TWO FOLLOWING WAYS:

 

 

6A

SHIP IDENTIFICATION

 

ENTER THE ARRIVAL LOCATION IDENTIFIER FOR THE AIRCRAFT BY IDENTIFYING THE SHIP OR BY ICAO LOCATION INDICATOR, FOR EXAMPLE: "CV-59". See table 2021 which is a composite

 

 

6B

ICAO LOCATION INDICATOR

 

ENTER THE ICAO LOCATION INDICATOR, FOR EXAMPLE: "BIKF". See table 1125/1 which is an instructive entry

7

MISSION NUMBER

 

SPECIFIES THE NUMBER ASSIGNED BY APPROPRIATE TASKING AUTHORITY, IN ONE OF THE TWO FOLLOWING WAYS:

 

 

7A

MISSION NUMBER

MSNNO

ENTER THE FIELD DESCRIPTOR CODE "MSNNO:" FOLLOWED BY THE MISSION NUMBER, FOR EXAMPLE: "MSNNO:KK102". See table 1012/69 which is an instructive entry

 

 

7B

SORTIE NUMBER

 

ENTER THE SORTIE NUMBER, FOR EXAMPLE: "FL10A". See table 1012/65 which is an instructive entry
