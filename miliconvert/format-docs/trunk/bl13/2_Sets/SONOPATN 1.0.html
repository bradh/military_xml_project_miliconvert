SONOBUOY PATTERN TYPE

ROW ORIENTATION, DEGREES TRUE, MAGNETIC, GRID OR RELATIVE

 

 

 

 

2A

ROW ORIENTATION, DEGREES TRUE, MAGNETIC, GRID OR RELATIVE

 

ENTER THE BEARING (THREE TO FIVE NUMBERS) IN DEGREES FOLLOWED BY "T" (TRUE), "M" (MAGNETIC), "G" (GRID), OR "R" (RELATIVE), FOR EXAMPLE: "231M". See table 2257 which is a composite

NUMBER OF SONOBUOYS PER ROW

 

 

 

 

3A

NUMBER OF SONOBUOYS PER ROW

 

IF ROWS CONTAIN DIFFERENT NUMBERS OF BUOYS, THIS FIELD CONTAINS THE NUMBER OF BUOYS IN THE LONGEST ROW, FOR EXAMPLE: "16". See table 1044/194 a range [1 through 99]

4

SPACING BETWEEN SONOBUOYS, YD, KYD OR NM

 

 

 

 

4A

SPACING BETWEEN SONOBUOYS, YD, KYD OR NM

 

ENTER THE SPACING BETWEEN SONOBUOYS GIVEN AS ONE TO FOUR NUMERALS FOLLOWED BY UNIT OF LINEAR MEASUREMENT, FOR EXAMPLE: "200YD". See table 2434 which is a composite

5

SPACING BETWEEN SONOBUOY ROWS, YD, KYD OR NM

 

 

 

 

5A

SPACING BETWEEN SONOBUOY ROWS, YD, KYD OR NM

 

ENTER THE SPACING BETWEEN SONOBUOY ROWS GIVEN AS ONE TO FOUR NUMERALS FOLLOWED BY UNIT OF LINEAR MEASUREMENT, FOR EXAMPLE: "10YD". See table 2434 which is a composite

6

NUMBER OF SONOBUOY ROWS

 

 

 

 

6A

NUMBER OF SONOBUOY ROWS

 

ENTER THE NUMBER OF SONOBUOY ROWS, FOR EXAMPLE: "1". See table 1044/195 a range [1 through 99]

7

NUMBER OF SONOBUOYS IN PATTERN

 

 

 

 

7A

NUMBER OF SONOBUOYS IN PATTERN

 

ENTER THE TOTAL NUMBER OF SONOBUOYS IN THE PATTERN, FOR EXAMPLE: "16". See table 1044/196 a range [0 through 99]

8

SPACING, MDR

 

 

 

 

8A

SPACING, MDR

 

MEDIAN DETECTION RANGE (MDR) SPACING IS THE BUOY SPACING EXPRESSED AS A MULTIPLE OF THE COMPUTED MDR (IF THE MDR IS 10 NM, THEN 15 NM BUOY SPACING IS REPORTED AS "1.5MDR"), GIVEN AS ONE TO FOUR NUMBERS (DECIMAL PERMITTED) PLUS "MDR". See table 2267 which is a composite

9

PROBABILITY OF DETECTION

 

 

 

 

9A

PROBABILITY OF DETECTION

 

ENTER THE PROBABILITY OF DETECTION GIVEN AS ONE TO THREE NUMERALS EXPRESSED AS PERCENTAGE, FOR EXAMPLE: "80". See table 1064/11 a range [1 through 100]

10

FIRST CZ MINIMUM-MAXIMUM RANGE

 

 

 

 

10A

FIRST CZ MINIMUM-MAXIMUM RANGE

 

ENTER THE RANGE IN NAUTICAL MILES TO THE INNER AND OUTER RADIUS OF THE FIRST CONVERGENCE ZONE (CZ) SEPARATED BY A HYPHEN, AND FOLLOWED BY "NM", FOR EXAMPLE: "30-33NM". See table 2289 which is a composite

11

WEDGE ANGLE OR DIRECTION OF WALK

 

ENTER ONE OF THE FOLLOWING:

 

 

11A

WEDGE ANGLE, DEGREES, (1-180)

 

ENTER THE WEDGE ANGLE GIVEN IN DEGREES, EXPRESSED AS ONE TO THREE NUMERALS FOLLOWED BY "DEG", FOR EXAMPLE: "15DEG". See table 2293 which is a composite

 

 

11B

DIRECTION OF WALK, DEGREES TRUE

 

ENTER THE DIRECTION OF WALK IS GIVEN AS BEARING IN DEGREES TRUE, EXPRESSED AS THREE NUMERALS FOLLOWED BY "T", FOR EXAMPLE: "190T". See table 2253 which is a composite
