1

MUNITIONS TYPE, GENERIC

 

 

 

 

1A

MUNITIONS TYPE, GENERIC

 

PROVIDES A VERY GENERAL INDICATION OF THE TYPE OF MUNITION INVOLVED IN A BOMBARDMENT, FOR EXAMPLE: "SHL". See table 1407/29 which contains a list of data items and associated data codes

2

MUNITIONS CALIBRE IN MILLIMETRES

 

 

 

 

2A

MUNITIONS CALIBRE IN MILLIMETRES

 

IDENTIFIES THE CALIBRE (IN MILLIMETRES) OF THE MUNITION IN FIELD 1, FOR EXAMPLE: "155". See table 1697/1 a range [1 through 999]

3

LENGTH IN METRES, CENTIMETRES, OR MILLIMETRES

 

 

 

 

3A

LENGTH IN METRES, CENTIMETRES, OR MILLIMETRES

 

USED TO SPECIFY THE LENGTH OF A MUNITION, FOR EXAMPLE: "133-CM". See table 2115 which is a composite

4

DIAMETER IN METRES, CENTIMETRES, OR MILLIMETRES

 

 

 

 

4A

DIAMETER IN METRES, CENTIMETRES, OR MILLIMETRES

 

USED TO SPECIFY THE DIAMETER OF A MUNITION, FOR EXAMPLE: "25-CM". See table 2115 which is a composite

5

AIRCRAFT GENERIC CATEGORY

 

 

 

 

5A

AIRCRAFT GENERIC CATEGORY

 

INDICATE THE GENERAL TYPE OF AIRCRAFT OR HELICOPTER THAT WAS INVOLVED IN THE BOMBARDMENT, FOR EXAMPLE: "AAH". See table 1301/1 which contains a list of data items and associated data codes

6

QUANTITY OF ORDNANCE

 

 

 

 

6A

QUANTITY OF ORDNANCE

 

INDICATE THE NUMBER OF MUNITIONS (1-999) COUNTED IN THE BOMBARDMENT, FOR EXAMPLE: "48". See table 1044/131 a range [0 through 999]

7

ATTACK INTENSITY

 

 

 

 

7A

ATTACK INTENSITY

 

ENTER THE INTENSITY OF THE BOMBARDMENT, FOR EXAMPLE: "MOD". See table 1293/1 which contains a list of data items and associated data codes
