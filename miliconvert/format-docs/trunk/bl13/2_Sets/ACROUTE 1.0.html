No

Designator

Field Desc

Concept/Explanation/Examples

1

POINT NUMBER

 

 

 

 

1A

POINT NUMBER

 

ENTER THE NUMBER ASSIGNED THE ROUTE POINT, FOR EXAMPLE: "1". See table 1012/257 a range [1 through 99]

2

ROUTE POINT LOCATION

 

PROVIDES THE GEOGRAPHICAL POSITION OF THE ROUTE POINT.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

2B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

2C

BEARING AND DISTANCE FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

2D

GEOGRAPHIC POSITION, GEOREF, MINUTE

GEOREFM

ENTER "GEOREFM:" FOLLOWED BY THE LOCATION OF THE ROUTE POINT USING GEOREF MINUTE, FOR EXAMPLE: "GEOREFM:AABB1535". See table 2017 which is a composite

 

 

2E

POSITION OR POINT NAME

NAME

ENTER "NAME:" FOLLOWED BY THE LOCATION OF THE ROUTE POINT USING A NAME, FOR EXAMPLE: "NAME:ALFA". See table 1022/18 which is an instructive entry

 

 

2F

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE LOCATION OF THE ROUTE POINT USING AN ICAO BASE CODE, FOR EXAMPLE: "ICAO:ETME". See table 1125/2 which is an instructive entry

3

ROUTE POINT TYPE

 

 

 

 

3A

ROUTE POINT TYPE

 

ENTER THE TYPE OF ROUTE POINT, CHARACTERIZED BY THE ACTION TO TAKE PLACE AT THAT POINT, FOR EXAMPLE: "STP". See table 1284/13 which contains a list of data items and associated data codes

4

ARRIVAL TIME

 

PROVIDES THE ARRIVAL TIME OF THE AIRCRAFT AT THE ROUTE POINT IDENTIFIED IN FIELD TWO.

 

 

4A

RELATIVE AIRCRAFT ARRIVAL TIME

RTIME

ENTER "RTIME:" FOLLOWED BY THE TIME OF ARRIVAL OF AN AIRCRAFT AT A SPECIFIC LOCATION, FOR EXAMPLE: "RTIME:HP0130". See table 2482 which is a composite

 

 

4B

DAY-TIME OF ARRIVAL

TIME

ENTER "TIME:" FOLLOWED BY THE DAY-TIME OF ARRIVAL OF AN AIRCRAFT AT A SPECIFIC LOCATION, FOR EXAMPLE: "TIME:141325Z". See table 2000 which is a composite

5

SPEED OF MOVEMENT

 

 

 

 

5A

SPEED OF MOVEMENT IN MPH, KTS, KPH OR MPS

 

ENTER THE SPEED OF MOVEMENT TO BE MAINTAINED BETWEEN THE ROUTE POINTS, FOR EXAMPLE: "450KTS". See table 2481 which is a composite

6

ALTITUDE IN HUNDREDS OF FEET

 

 

 

 

6A

ALTITUDE, HUNDREDS OF FEET

 

ENTER THE ALTITUDE IN HUNDREDS OF FEET TO BE FLOWN IN THE ROUTE, FOR EXAMPLE: "180". See table 1255/14 a range [000 through 999]
