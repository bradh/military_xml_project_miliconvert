CORRIDOR WIDTH OR AREA RADIUS IN METRES

 

SPECIFIES THE WIDTH OF A RESTRICTIVE FIRE CORRIDOR OR RADIUS OF A RESTRICTIVE FIRE AREA IN METRES.

 

 

9A

WIDTH IN METRES, 1-99999

 

ENTER THE WIDTH IN METRES OF THE RESTRICTIVE FIRE CORRIDOR, FOR EXAMPLE: "500". See table 1658/6 a range [1 through 99999]

 

 

9B

RADIUS IN METRES, 1-99999

RAD
