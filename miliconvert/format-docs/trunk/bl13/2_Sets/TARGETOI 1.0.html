1

PRIORITY IDENTIFIER

 

 

 

 

1A

PRIORITY IDENTIFIER

 

ENTER THE PRIORITY FOR DETECTION OF AN EMITTER AS AN INTEGER 01-99 WHERE THE GREATEST PRIORITY IS "01" FOR EXAMPLE:"03". See table 1076/20 a range [01 through 99]

2

EMITTER CATEGORY

 

 

 

 

2A

EMITTER CATEGORY

 

IDENTIFY THE EMITTER ASSESSED AS EITHER THE THREAT, TARGET,  CONTACT OF INTEREST(COI) OR CRITICAL CONTACT OF INTEREST(CCOI)  FOR EXAMPLE: "TARGET". See table 1121/5 which contains a list of data items and associated data codes

3

RADAR FREQUENCY IN MHZ OR GHZ

 

 

 

 

3A

RADAR FREQUENCY, MHZ OR GHZ

 

ENTER THE FREQUENCY OF THE RADAR TRANSMISSION IN EITHER MHZ OR GHZ, FOR EXAMPLE:"16.666MHZ" See table 2108 which is a composite

4

PULSE REPETITION FREQUENCY MEASURED IN HZ

 

 

 

 

4A

PULSE REPETITION FREQUENCY MEASURED IN HZ

PRF

ENTER "PRF:" FOLLOWED BY THE PULSE REPETITION FREQUENCY IN PULSES PER SECOND, FOR EXAMPLE: "PRF:4.6". See table 1154/1 a range [0.0 through 9999.9] (1 to 1 decimal places)

5

PULSE REPETITION INTERVAL MEASURED IN MILLISECONDS

 

 

 

 

5A

PULSE REPETITION INTERVAL MEASURED IN MILLISECONDS

PRI

ENTER "PRI:" FOLLOWED BY THE PULSE REPETITION INTERVAL IN MILLISECONDS, FOR EXAMPLE: "PRI:217.4". See table 1154/2 a range [0.0 through 999.9] (1 to 1 decimal places)

6

PULSE WIDTH

 

 

 

 

6A

RADAR PULSE WIDTH MEASURED IN MICROSECONDS

PW

ENTER "PW:" FOLLOWED BY THE PULSE OF THE EMITTING RADAR WIDTH IN MICROSECONDS, FOR EXAMPLE: "PW:125". See table 1154/4 a range [0 through 999]

7

THE SCAN PERIOD OF A RADAR AERIAL.

 

 

 

 

7A

SCAN PERIOD MEASURED IN SECONDS

SPR

ENTER "SPR:" FOLLOWED BY THE PERIOD FOR A COMPLETE RADAR ROTATION EXPRESSED IN SECONDS PER ROTATION (SPR) ACCURATE TO 1 DECIMAL PLACE, FOR EXAMPLE: "SPR:3.3". See table 1154/3 a range [0.0 through 300.0] (1 to 1 decimal places)

8

THE NATION TO WHICH THE DESIGNATED EMITTER BELONGS.

 

 

 

 

8A

COUNTRY OF EMITTER

 

ENTER THE 2-LETTER COUNTRY CODE FOR THE NATION TO WHICH THE DESIGNATED EMITTER BELONGS, FOR EXAMPLE: "UK". See table 1026/34 which contains a list of data items and associated data codes
