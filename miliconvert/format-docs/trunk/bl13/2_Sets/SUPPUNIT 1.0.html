DUTY UNIT

 

PROVIDES THE UNIT IDENTIFIER PERFORMING CONTROL OVER THE REPORTING UNIT.

 

 

1A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UIC OF THE UNIT TO WHICH THE DUTY IS ASSIGNED, FOR EXAMPLE: "UIC:GEF2131". See table 2141 which is a composite

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE THE IDENTIFIER OF THE UNIT TO WHICH THE DUTY IS ASSIGNED, FOR EXAMPLE: "UNIT:CAOC 1". See table 1028/43 which is an instructive entry
