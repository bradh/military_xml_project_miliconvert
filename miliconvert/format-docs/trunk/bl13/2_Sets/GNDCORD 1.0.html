Concept/Explanation/Examples

1

DAY-TIME AVAILABLE FROM

 

IT PROVIDES THE DAY-TIME FROM WHEN THE GROUND COORDINATION IS AVAILABLE.

 

 

1A

DAY-TIME AVAILABLE FROM

 

ENTER THE DAY-TIME FROM WHEN THE GROUND COORDINATION IS AVAILABLE, FOR EXAMPLE: "121500Z". See table 2000 which is a composite

2

DAY-TIME AVAILABLE TO

 

 

 

 

2A

DAY-TIME AVAILABLE TO

 

ENTER THE DAY-TIME GROUND COORDINATION IS AVAILABLE TO, FOR EXAMPLE: "141045Z". See table 2000 which is a composite

3

UNIT ID

 

PROVIDES THE GROUND COORDINATION  UNIT IDENTIFICATION OR DESIGNATION.

 

 

3A

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE IDENTIFICATION CODE OF THE GROUND COORDINATION UNIT, FOR EXAMPLE: "UIC:GEF1234". See table 2141 which is a composite

 

 

3B

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE DESIGNATOR OF THE GROUND COORDINATION UNIT, FOR EXAMPLE: "UNIT:ARS BROCKZETEL". See table 1028/43 which is an instructive entry

 

 

3C

BASE IDENTIFICATION CODE

BASE

ENTER "BASE:" FOLLOWED BY THE BASE IDENTIFICATION CODE FOR THE UNIT, FOR EXAMPLE: "BASE:GEF123401". See table 2141 which is a composite

 

 

3D

ICAO BASE CODE

ICAO

ENTER "ICAO:" FOLLOWED BY THE LOCATION OF THE GROUND COORDINATION UNIT USING AN ICAO BASE CODE, FOR EXAMPLE: "ICAO:ETME". See table 1125/2 which is an instructive entry

4

GROUND AGENCY TYPE

 

 

 

 

4A

GROUND AGENCY TYPE

 

ENTER THE TYPE OF AIR OPERATIONS COORDINATION ELEMENTS (FIXED OR MOBILE) LOCATED ON THE GROUND OR ON THE SURFACE, FOR EXAMPLE: "AIRSOC". See table 1874/5 which contains a list of data items and associated data codes

5

CALL SIGN

 

 

 

 

5A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN TO CONTACT THE GROUND COORDINATION, FOR EXAMPLE: "BLUE BIRD 1". See table 1129/31 which is an instructive entry

6

LINK 16 CALL SIGN

 

 

 

 

6A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE ABBREVIATED LINK 16 CALL SIGN (NORMALLY THE FIRST AND LAST CHARACTERS AND THE LAST TWO NUMBERS, ASSIGNED TO THE LINK 16 PARTICIPANT, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry

7

PRIMARY COMMMUNICATION PARAMETERS

 

PROVIDES THE PRIMARY COMMUNICATION PARAMETERS FOR CONTACTING THE GROUND COORDINATION, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.

 

 

7A

PRIMARY FREQUENCY IN MEGAHERTZ, .0000001 - 99999999

PFREQ

ENTER "PFREQ:" FOLLOWED BY THE PRIMARY FREQUENCY IN MEGAHERTZ, FOR EXAMPLE: "PFREQ:243.5". See table 1068/13 a range [0.0000001 through 99999999.0000000] (0 to 7 decimal places)

 

 

7B

PRIMARY FREQUENCY DESIGNATOR

PDESIG

ENTER "PDESIG:" FOLLOWED BY THE PRIMARY FREQUENCY DESIGNATOR, FOR EXAMPLE: "PDESIG:TAD003". See table 1396/2 which is an instructive entry

 

 

7C

HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER

PHVQKM

ENTER "PHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite

 

 

7D

HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER

PHVQK

ENTER "PHVQK:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITHOUT MICKEY NET CONTROLLER, FOR EXAMPLE: "PHVQK:AB12-CD34-12082000". See table 2535 which is a composite

8

SECONDARY COMMUNICATION PARAMETERS

 

PROVIDES THE SECONDARY COMMUNICATION PARAMETERS, AS A FREQUENCY IN MEGAHERTZ, A FREQUENCY DESIGNATOR OR A SET OF HAVE QUICK PARAMETERS.

 

 

8A

8B

SECONDARY FREQUENCY DESIGNATOR

SDESIG

ENTER "SDESIG:" FOLLOWED BY THE SECONDARY FREQUENCY DESIGNATOR, FOR EXAMPLE: "SDESIG:TAD005". See table 1396/3 which is an instructive entry

SHVQKM

ENTER "SHVQKM:" FOLLOWED BY THE HAVE QUICK PARAMETERS WITH MICKEY NET CONTROLLER, FOR EXAMPLE: "SHVQKM:AB12-CD34-12082000-EFG 345-452.5MHZ". See table 2534 which is a composite
