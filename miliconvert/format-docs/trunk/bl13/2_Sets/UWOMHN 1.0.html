MOVING HAVEN DIMENSIONS

 

ENTER THE MOVING HAVEN DIMENSIONS IN ONE OF THE FOLLOWING WAYS:

 

 

5A

RECTANGULAR MOVING HAVEN DIMENSIONS, NM, VERIFIED

RECTMVH

ENTER "RECTMVH:" FOLLOWED BY THE LENGTH AND WIDTH OF THE RECTANGLE IN RELATION TO THE ROUTE SET: FIRST NUMBER LENGTH AHEAD, SECOND NUMBER ASTERN OF THE ROUTE, THIRD NUMBER WIDTH EITHER SIDE OF TRACK; NUMBERS ARE SEPARATED BY AN "X" AND THE LAST NUMBER TO BE FOLLOWED BY "NM", FOR EXAMPLE: "RECTMVH:25X15X10NM4". See table 2199 which is a composite

 

 

5B

CIRCULAR MOVING HAVEN, RADIUS, NM, VERIFIED

CIRCMVH

ENTER "CIRCMVH:" FOLLOWED BY THE CIRCULAR MOVING HAVEN, WHICH IS AUTOMATICALLY DEFINED BY USING THE KNOWN FRIENDLY NAVAL UNIT DESIGNATOR, WITH A RADIUS IN NAUTICAL MILES, VERIFIED, FOR EXAMPLE: "CIRCMVH:15NM6". See table 2362 which is a composite

 

 

5C

NON-DISCRETE INDICATOR, AS REQUIRED

 

ENTER THE WORDS "AS REQD" WHEN THE MOVEMENT IS SPECIFIED ELSEWHERE, FOR EXAMPLE: "OPORDER", "WPP" OR "NOTICE OF INTENT".  "AS REQD" MAY ALSO BE USED FOR RELATIVELY SHORT TRANSITS IN COASTAL WATERS AND A GRIDDED TRAINING AREA DEFINED IN LOCAL INSTRUCTIONS. See table 1084/2 which contains a list of data items and associated data codes
