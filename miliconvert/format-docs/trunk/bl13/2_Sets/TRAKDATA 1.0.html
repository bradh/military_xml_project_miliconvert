Concept/Explanation/Examples

1

NUMBER OF RAILROAD TRACKS

 

 

 

 

1A

NUMBER OF RAILROAD TRACKS

 

ENTER THE TOTAL NUMBER OF RAILROAD TRACKS BEING REPORTED ON, FOR EXAMPLE: "2". See table 1044/846 a range [1 through 99]

2

RAILROAD BED WIDTH

 

 

 

 

2A

RAILROAD BED WIDTH

 

ENTER THE WIDTH OF THE RAILROAD BED, THAT IS THE AVERAGE DISTANCE BETWEEN THE OUTER EDGES OF THE GRADED TRACK BED MATERIAL, IN EITHER FEET, YARDS OR METRES, ONE DECIMAL PLACE PERMITTED, FOR EXAMPLE: "15M". See table 2451 which is a composite

3

RAIL BED COMPOSITION

 

 

 

 

3A

RAIL BED COMPOSITION

 

ENTER THE COMPOSITION OF THE RAILROAD BED, FOR EXAMPLE: "GRAVEL". See table 1845/1 which contains a list of data items and associated data codes

4

GRADIENT

 

 

 

 

4A

GRADIENT

GRD

ENTER "GRD:" AND THE GRADIENT OF THE RAILROAD BED. EXPRESSED AS A RATIO OF DROP OR RISE IN THE TERRAIN. THE NUMBER OF UNITS FORWARD TO ACHIEVE ONE UNIT OF DROP OR RISE, FOR EXAMPLE: "GRD:15". See table 1789/1 a range [0 through 99]

5

INSIDE WIDTH OF RAILROAD TRACKS

 

 

 

 

5A

INSIDE WIDTH OF RAILROAD TRACKS

 

ENTER THE DISTANCE BETWEEN THE INNER MOST RAILS OF THE INNER MOST TRACKS IN EITHER FEET, YARDS OR METRES, ONE DECIMAL PLACE PERMITTED, FOR EXAMPLE: "8.5M". See table 2451 which is a composite

6

OUTSIDE WIDTH OF RAILROAD TRACKS

 

 

 

 

6A

OUTSIDE WIDTH OF RAILROAD TRACKS

 

ENTER THE DISTANCE BETWEEN THE TWO OUTER MOST RAILS OF THE OUTER MOST TRACKS IN EITHER FEET, YARDS, OR METRES, ONE DECIMAL PLACE PERMITTED, FOR EXAMPLE: "12M". See table 2451 which is a composite
