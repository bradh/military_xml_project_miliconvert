1

DTG ON TARGET

 

PROVIDES THE DATE-TIME GROUP OF THE TIME OVER THE TARGET FOR THE ATTACK.

 

 

1A

DTG ON TARGET

 

ENTER THE DATE-TIME GROUP OF THE TIME OVER THE TARGET FOR THE ATTACK., FOR EXAMPLE "100850AUG2006". See table 2033 which is a composite

2

RELEASE ACHIEVED INDICATOR

 

SPECIFIES WHETHER THE WEAPONS RELEASE WAS ACHIEVED OR NOT DURING THE TASKED MISSION OPERATIONS.

 

 

2A

RELEASE ACHIEVED INDICATOR

 

ENTER THE RELEASE ACHIEVED INDICATOR, YES OR NO, FOR EXAMPLE: "YES". See table 1027/88 which contains a list of data items and associated data codes

3

NON RELEASE REASON

 

PROVIDES THE REASON ORDNANCE WAS NOT RELEASED ON TASKED TARGET.

 

 

3A

NON RELEASE REASON

 

ENTER THE REASON ORDNANCE WAS NOT RELEASED ON TASKED TARGET, FOR EXAMPLE: "BF". See table 1212/12 which contains a list of data items and associated data codes

4

ORDNANCE DELIVERY METHOD

 

PROVIDES THE PRESCRIBED METHOD OF DELIVERY OF ALLOCATED ORDNANCE ON TASKED TARGET.

 

 

4A

ORDNANCE DELIVERY METHOD

 

ENTER THE PRESCRIBED METHOD OF DELIVERY OF ALLOCATED ORDNANCE ON TASKED TARGET, FOR EXAMPLE: "DIVE". See table 1349/7 which contains a list of data items and associated data codes

5

MISSION SUCCESS

 

PROVIDES THE OPERATOR'S INITIAL ASSESSMENT OF MISSION SUCCESS AGAINST TASKED TARGETS.

 

 

5A

MISSION SUCCESS

 

ENTER THE INITIAL ASSESSMENT OF MISSION SUCCESS AGAINST TASKED TARGETS, FOR EXAMPLE: "HIT". See table 1150/8 which contains a list of data items and associated data codes

6

MISSION RESULTS

 

PROVIDES THE RESULTS ON THE MISSION TASKED.

 

 

6A

MISSION RESULTS

 

ENTER THE RESULT OF THE TASKED MISSION, FOR EXAMPLE: "DEA". See table 1088/9 which contains a list of data items and associated data codes

7

MISSED TARGET REASON

 

PROVIDES THE REASON TASKED TARGET WAS MISSED AFTER ORDNANCE RELEASE.

 

 

7A

MISSED TARGET REASON

 

ENTER THE REASON TASKED TARGET WAS MISSED AFTER ORDNANCE RELEASE, FOR EXAMPLE: "UNEXPECTED SYSTEM FAILURE". See table 1212/13 which is an instructive entry

8

WEAPON TYPE EXPENDED

 

PROVIDES THE TYPE OF ORDNANACE OR WEAPON EXPENDED ON TASKED TARGET.

 

 

8A

WEAPON TYPE EXPENDED

 

ENTER THE TYPE OF ORDNANACE OR WEAPON EXPENDED ON TASKED TARGET, FOR EXAMPLE: "BRF250". See table 1407/48 which is an instructive entry

9

QUANTITY OF ORDNANCE EXPENDED

 

PROVIDES THE QUANTITY OF THE ORDNANCE OR WEAPON EXPENDED IN THE MISSION.

 

 

9A

QUANTITY OF ORDNANCE EXPENDED

 

ENTER THE QUANTITY OF THE ORDNANCE EXPENDED IN THE MISSION, FOR EXAMPLE: "1000". See table 1044/225 a range [0 through 99999]

10

FUZE TYPE

 

PROVIDES THE NAME OF THE FUZE USED BY THE WEAPON DELIVERY SYSTEM.

 

 

10A

FUZE TYPE

 

ENTER THE NAME OF THE FUZE USED BY THE WEAPON DELIVERY SYSTEM, FOR EXAMPLE: "LAIAM MRD". See table 1192/1 which is an instructive entry

11

FUZE POSITION

 

PROVIDES THE POSITION WHERE THE FUZE IS SITUATED ON THE WEAPON.

 

 

11A

FUZE POSITION

 

ENTER THE POSITION OF THE FUZE ON THE DELIVERED WEAPON, FOR EXAMPLE: "NOSE". See table 1223/2 which contains a list of data items and associated data codes

12

WEAPON FUZE TYPE

 

PROVIDES THE TYPE OF FUZE USED BY THE WEAPON DELIVERY SYSTEM.

 

 

12A

WEAPON FUZE TYPE

 

ENTER THE TYPE OF FUZE USED BY THE WEAPON DELIVERY SYSTEM, FOR EXAMPLE: "IMPACT". See table 1192/8 which contains a list of data items and associated data codes

13

FUZE DELAY SETTING

 

PROVIDES THE FUZE DELAY SETTING USED BY THE WEAPON DELIVERY SYSTEM.

 

 

13A

FUZE DELAY SETTING

 

ENTER THE FUZE DELAY SETTING USED BY THE WEAPON DELIVERY SYSTEM, FOR EXAMPLE: "120MS". See table 1192/9 which contains a list of data items and associated data codes

14

AIRCRAFT HEADING IN DEGREES AT RELEASE

 

PROVIDES THE DIRECTION OF AIRCRAFT MOVEMENT TOWARDS THE TARGET AT TIME OF RELEASE MEASURED IN DEGREES, TRUE, GRID, MAGNETIC, OR RELATIVE.

 

 

14A

AIRCRAFT HEADING IN DEGREES AT RELEASE

 

ENTER THE DIRECTION OF AIRCRAFT MOVEMENT TOWARDS THE TARGET AT TIME OF RELEASE MEASURED IN DEGREES, TRUE, GRID, MAGNETIC, OR RELATIVE, FOR EXAMPLE: "260M". See table 2065 which is a composite

15

AIRCRAFT ALTITUDE IN HUNDREDS OF FEET AT RELEASE

 

PROVIDES THE THE ALTITUDE OF THE AIRCRAFT AT THE TIME ORDNANCE IS RELEASED IN HUNDREDS OF FEET.

 

 

15A

AIRCRAFT ALTITUDE IN HUNDREDS OF FEET AT RELEASE

 

ENTER THE ALTITUDE OF THE AIRCRAFT AT THE TIME ORDNANCE IS RELEASED IN HUNDREDS OF FEET, FOR EXAMPLE: "250". See table 1255/17 a range [000 through 999]

16

DIVE ANGLE OF ATTACK

 

PROVIDES THE DIVE ANGLE OF ATTACK AT TIME OF RELEASE MEASURED IN DEGREES (0 DEGREES IS LEVEL FLIGHT AND 90 DEGREES IS STRAIGHT DOWN).

 

 

16A

DIVE ANGLE OF ATTACK

 

ENTER THE DIVE ANGLE OF ATTACK AT TIME OF RELEASE MEASURED IN DEGREES (0 DEGREES IS LEVEL FLIGHT AND 90 DEGREES IS STRAIGHT DOWN), FOR EXAMPLE "45". See table 1043/26 a range [0 through 90]

17

AIRCRAFT AIRSPEED AT RELEASE IN KPH, KTS, MPH

 

PROVIDES THE SPEED OF AN AIRCRAFT WITH RELATION TO THE AIR MASS SURROUNDING THAT AIRCRAFT AT TIME OF WEAPONS DELIVERY.

 

 

17A

AIRCRAFT AIRSPEED AT RELEASE IN KPH, KTS, MPH

 

ENTER PROVIDES THE SPEED OF THE AIRCRAFT PERFORMING THR MISSION, WITH RELATION TO THE AIR MASS SURROUNDING THAT AIRCRAFT AT TIME OF WEAPONS DELIVERY, IN KNOTS, KILOMETRES PER HOUR, OR STATUTE MILES PER HOUR, FOR EXAMPLE: "300KTS". See table 2461 which is a composite

18

RELEASE POSITION TO TARGET, BEARING AND DISTANCE

 

PROVIDES THE POSITION THE TASKED AIRCRAFT AT ORDNANCE RELEASE EXPRESSED AS A TRUE BEARING AND DISTANCE FROM A REFERENCE POINT.

 

 

18A

RELEASE POSITION TO TARGET, BEARING AND DISTANCE

 

ENTER THE POSITION THE TASKED AIRCRAFT AT ORDNANCE RELEASE EXPRESSED AS A TRUE BEARING AND DISTANCE FROM A REFERENCE POINT, FOR EXAMPLE: "261.5M-CUMBIA14-12.75NM". See table 2157 which is a composite

19

TARGET DISTANCE

 

PROVIDES THE DISTANCE BETWEEN THE AIRCRAFT AND THE TARGET EXPRESSED EITHER USING THE LINE OF SIGHT DISTANCE BETWEEN THE AIRCRAFT AND TARGET MEASURED IN FEET, OR THE HORIZONTAL DISTANCE MEASURED IN FEET.

 

 

19A

SLANT RANGE IN FEET

SR

ENTER "SR:" FOLLOWED BY THE SLANT RANGE IN FEET BETWEEN THE AIRCRAFT AND THE TARGET, FOR EXAMPLE: "SR:03650FT". See table 2439 which is a composite

 

 

19B

HORIZONTAL RANGE IN FEET

HR

ENTER "HR:" FOLLOWED BY THE HORIZONTAL RANGE IN FEET BETWEEN THE AIRCRAFT AND THE TARGET, FOR EXAMPLE: "HR:13650FT". See table 2439 which is a composite
