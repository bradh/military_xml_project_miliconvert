LOW CLOUD COVER

 

 

 

 

1A

LOW CLOUD COVER

 

ENTER THE RELATIVE AMOUNT OF CLOUD AT LOW LEVEL USING A CODE NUMBER (0 THROUGH 9), FOR EXAMPLE: "4". See table 1387/2 which contains a list of data items and associated data codes

2

LOW CLOUD BASE HEIGHT

 

 

 

 

2A

LOW CLOUD BASE HEIGHT

 

ENTER THE CODE TO INDICATE THE HEIGHT OF THE BASE OF THE LOWEST CLOUD ABOVE THE MET STATION, FOR EXAMPLE: "6". See table 1162/2 which contains a list of data items and associated data codes

3

SURFACE VISIBILITY

 

 

 

 

3A

VISIBILITY CODE

 

ENTER THE VISIBILITY NUMERIC CODE (0 THROUGH 9), FOR EXAMPLE: "7". See table 1414/2 which contains a list of data items and associated data codes

4

WEATHER CONDITION

 

 

 

 

4A

WEATHER CONDITION NUMERIC CODE

 

ENTER THE WEATHER CONDITION NUMERIC (0 THROUGH 9) CODE INDICATOR, FOR EXAMPLE: "1". See table 1415/1 which contains a list of data items and associated data codes

5

LUNAR ILLUMINATION

 

PROVIDES THE LEVEL OF LUNAR ILLUMINATION DURING NIGHT TIME OPERATIONS MEASURED IN HIGH, MEDIUM, OR LOW.

 

 

5A

LUNAR ILLUMINATION

 

ENTER THE LEVEL OF LUNAR ILLUMINATION DURING NIGHT TIME OPERATIONS MEASURED IN HIGH, MEDIUM, OR LOW, FOR EXAMPLE: "MEDIUM". See table 1386/10 which contains a list of data items and associated data codes

6

WEATHER REMARKS

 

 

 

 

6A

WEATHER REMARKS CODE

 

ENTER THE WEATHER REMARKS CODE, FOR EXAMPLE: "N",  REPEATING AS NECESSARY. See table 1386/5 which contains a list of data items and associated data codes
