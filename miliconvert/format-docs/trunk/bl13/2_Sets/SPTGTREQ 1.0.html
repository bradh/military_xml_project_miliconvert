1

DAY-TIME ON TARGET

 

THE DAY-TIME THE MISSION IS REQUESTED ON TARGET

 

 

1A

DAY-TIME AND MONTH TASKED ON TARGET

TOT

ENTER "TOT:", FOLLOWED BY THE DAY-TIME WHEN THE REQUESTED MISSION IS SUPPOSED TO BE ON TARGET, FOR EXAMPLE "TOT:231550ZAUG". See table 2030 which is a composite

 

 

1B

DAY-TIME ON TARGET TO THE SECOND, ZULU

TOTS

ENTER "TOTS:" FOLLOWED BY THE DAY , HOUR, MINUTE, SECOND, TIME ZONE, WHEN THE REQUESTED MISSION IS SUPPOSED TO BE ON TARGET, FOR EXAMPLE "TOTS:23155040Z". See table 2483 which is a composite

2

NET TIME

 

REQUIRED ONLY IF TIME DAY-TIME ON TARGET HAS NOT BEEN SPECIFIED WITHIN THIS SET.

 

 

2A

NOT EARLIER THAN DAY-TIME AND MONTH

NET

ENTER "NET:" FOLLOWED BY NOT EARLIER THAN DAY-TIME AND MONTH ON TARGET, FOR EXAMPLE: "NET:141320ZFEB". See table 2030 which is a composite

3

NLT TIME

 

THE NOT LATER THAN DAY-TIME OFF THE TARGET.  REQUIRED ONLY IF TIME DAY-TIME ON TARGET HAS NOT BEEN SPECIFIED WITHIN THIS SET.

 

 

3A

NOT LATER THAN DAY-TIME

NLT

ENTER "NLT:" FOLLOWED BY THE LATEST DAY-TIME ON TARGET, FOR EXAMPLE "NLT:131408Z". See table 2000 which is a composite

 

 

3B

DAY-TIME AND MONTH OFF TARGET

TOF

ENTER "TOF:" FOLLOWED BY THE DAY-TIME AND MONTH WHEN THE MISSION IS DUE TO BE OFF TARGET, FOR EXAMPLE: "TOF:251559ZMAY". See table 2030 which is a composite

4

SHIP TARGET IDENTITY

 

 

 

 

4A

SHIP TARGET IDENTITY

 

ENTER THE CODE FOR THE IDENTITY OF THE SURFACE SHIP, FOR EXAMPLE: "HOS". See table 1416/6 which contains a list of data items and associated data codes

5

SHIP TYPE

 

 

 

 

5A

VESSEL TYPE

 

ENTER THE BASIC DESIGN, CONFIGURATION OR PRIMARY FUNCTION FOR WHICH A SHIP WAS DESIGNED AND CONSTRUCTED OR CONVERTED, FOR EXAMPLE: "CG". See table 1166/1 which contains a list of data items and associated data codes

6

TARGET PRIORITY IDENTIFIER

 

 

 

 

6A

TARGET PRIORITY IDENTIFIER

 

ENTER A DESIGNATOR THAT CLASSIFIES TARGETS AS PRIMARY, SECONDARY OR ALTERNATE, FOR EXAMPLE: "P". See table 1076/8 which contains a list of data items and associated data codes

7

POSITION DAY-TIME

 

 

 

 

7A

DAY-TIME OF POSITION

 

ENTER THE DAY-TIME OF THE LASTEST POSITION OF THE SHIP, FOR EXAMPLE: "141325Z". See table 2000 which is a composite

8

SHIP LOCATION

 

THE GEOGRAPHIC LOCATION OF THE SHIP.

 

 

8A

BEARING AND DISTANCE FROM REFERENCE POINT

TBR

ENTER "TBR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "TBR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

8B

GEOGRAPHIC LOCATION, LAT/LONG, CENTISECONDS

LATCS

ENTER "LATCS:" FOLLOWED BY THE LAST KNOWN POSITION OF THE SHIP IN LAT/LONG CENTISECONDS, FOR EXAMPLE: "LATCS:35102530N079012530W". See table 2488 which is a composite

 

 

8C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

9

COUNT OF SHIPS

 

 

 

 

9A

COUNT OF SHIPS

 

ENTER THE NUMBER FOR THE TOTAL COUNT OF SHIPS, FOR EXAMPLE "21". See table 1044/851 a range [0 through 9999]

10

HULL NUMBER

 

 

 

 

10A

HULL NUMBER

 

ENTER THE ONE TO FIVE CHARACTER HULL/PENNANT NUMBER OF THE VESSEL THAT UNIQUELY IDENTIFIES THE VESSEL, FOR EXAMPLE: "F-123". See table 1012/30 which is an instructive entry

11

SHIP NAME

 

 

 

 

11A

SHIP NAME

 

ENTER THE NAME OF THE TARGET SHIP/MARITIME VESSEL, (OR "UNKNOWN" IF THE SHIP NAME IS UNKNOWN) FOR EXAMPLE: "VIRGINIA". See table 1022/49 which is an instructive entry

12

TARGET PRIORITY

 

 

 

 

12A

TARGET PRIORITY, LITERAL

 

ENTER THE CODE FOR THE PRIORITY ASSIGNED TO THE TARGET, FOR EXAMPLE: "1A". See table 1076/11 which is an instructive entry

13

TARGET OBJECTIVE

 

 

 

 

13A

TARGET OBJECTIVE

 

ENTER THE TARGET OBJECTIVE, FOR EXAMPLE: "DESTROY". See table 1456/3 which is an instructive entry

14

TARGET IDENTIFIER

 

THE TARGET IDENTIFIER

 

 

14A

COMPONENT TARGET IDENTIFIER

CTGT

ENTER "CTGT:" FOLLOWED BY THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE SERVICE COMPONENT, FOR EXAMPLE: "CTGT:N4792618". See table 1416/5 which is an instructive entry

 

 

14B

TARGET ENTITY ID REFERENCE NUMBER

EIRN

ENTER "EIRN:" FOLLOWED BY THE EIRN NUMBER TO IDENTIFY A TARGET ON A VARIABLE MESSAGE FORMAT (VMF) INTERFACE, FOR EXAMPLE: "EIRN:167772104294967290". See table 2085 which is a composite

15

COURSE

 

 

 

 

15A

COURSE IN DEGREES, TRUE

 

ENTER THE DIRECTION OF TRAVEL OF THE SHIP RELATIVE TO TRUE NORTH, FOR EXAMPLE: "235". See table 1439/4 a range [001 through 360]

16

SPEED

 

 

 

 

16A

SPEED IN KNOTS, DECIMAL PROHIBITED

 

ENTER SPEED MEASURED IN KNOTS (NAUTICAL MILES PER HOUR) OF THE SHIP, FOR EXAMPLE: "25". See table 1061/16 a range [0 through 999]
