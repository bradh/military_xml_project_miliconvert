Concept/Explanation/Examples

1

SERIAL NUMBER

 

 

 

 

1A

AIR SORTIE IDENTIFICATION NUMBER

 

ENTER THE SEQUENTIAL NUMBER AND WHERE POSSIBLE LISTED IN DTG ORDER. AIRCRAFT REQUESTED SHOULD EACH BE GIVEN AS SEQUENTIAL NUMBER. THE AIRCRAFT TASKING AUTHORITY WILL BE RESPONSIBLE FOR PROVIDING THE AIRCRAFT AND GIVING IT A MISSION NUMBER, FOR EXAMPLE "1". See table 1012/205 a range [1 through 999]

2

MPA INDICATOR

 

 

 

 

2A

MPA INDICATOR

 

ENTER IF AN MPA OR NON MPA AIRCRAFT IS REQUIRED, FOR EXAMPLE "MPA". See table 1549/3 which contains a list of data items and associated data codes

3

NUMBER OF AIRCRAFT REQUIRED

 

 

 

 

3A

NUMBER OF AIRCRAFT REQUIRED

 

ENTER THE NUMBER OF AIRCRAFT REQUIRED TO BE ON TASK DURING THE DAILY TASK PERIOD, FOR EXAMPLE "2". See table 1044/475 a range [1 through 9]

4

DTG OF START

 

 

 

 

4A

DTG OF START

 

ENTER THE DTG AT WHICH THE PROVISION OF AIRCRAFT IS TO START, FOR EXAMPLE "101010ZJUN2005". See table 2033 which is a composite

5

DTG OF STOP

 

 

 

 

5A

DTG OF STOP

 

ENTER THE DTG AT WHICH THE PROVISION OF AIRCRAFT IS TO STOP, FOR EXAMPLE "121010ZJUN2005". See table 2033 which is a composite

6

DAILY START TIME

 

 

 

 

6A

DAILY START TIME

 

IF USED, ENTER THE  REPEATED TIME WHEN AIRCRAFT ARE REQUIRED ON TASK DAILY, FOR EXAMPLE "0900Z". IF 24 HOUR COVERAGE IS REQUIRED THEN THIS FIELD SHOULD BE OMITTED. See table 2485 which is a composite

7

DAILY STOP TIME

 

 

 

 

7A

DAILY STOP TIME

 

IF USED, ENTER THE REPEATED TIME WHEN AIRCRAFT MAY PROCEED OFF TASK DAILY, FOR EXAMPLE "2100Z". IF 24 HOUR COVERAGE IS REQUIRED THEN THIS FIELD SHOULD BE OMITTED. See table 2485 which is a composite
