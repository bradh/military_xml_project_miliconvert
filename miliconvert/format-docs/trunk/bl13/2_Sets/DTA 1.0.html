TARGET ACQUISITION METEOROLOGICAL ZONE CODE

 

ENTER THE MET ZONE CODE (IN ASCENDING ORDER), FOR EXAMPLE: "05". See table 1685/3 a range [00 through 17]

WIND DIRECTION AND SPEED

 

ENTER THREE DIGITS (000-639) TO INDICATE THE WIND DIRECTION FROM TRUE NORTH FOLLOWED BY THREE DIGITS (000-300) INDICATING THE WIND SPEED FOR THE MET ZONE INDICATED IN FIELD 1, FOR EXAMPLE: "310004". See table 2418 which is a composite

MET AIR TEMPERATURE AND RELATIVE HUMIDITY

 

 

 

 

3A

MET AIR TEMPERATURE AND RELATIVE HUMIDITY

 

ENTER FOUR DIGITS (0000-5000) TO INDICATE THE AIR TEMPERATURE TO THE NEAREST TENTH OF A DEGREE KELVIN FOLLOWED BY TWO DIGITS (00-99) INDICATING THE RELATIVE HUMIDITY TO THE NEAREST PERCENT FOR THE MET ZONE INDICATED IN FIELD 1, FOR EXAMPLE: "297734".  A RELATIVE HUMIDITY OF 100 PERCENT IS REPRESENTED AS "00".
