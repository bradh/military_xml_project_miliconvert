ICAO BASE CODE

ICAO BASE CODE

NAEW AIRCRAFT ROLE

 

 

 

 

2A

NAEW AIRCRAFT ROLE

 

ENTER THE ROLE OF THE CONTROLLED AIRCRAFT, FOR EXAMPLE: "OFF". See table 1010/3 which contains a list of data items and associated data codes

AIRBORNE EARLY WARNING CONTROL TYPE

AIRBORNE EARLY WARNING CONTROL TYPE

4

CONTROLLED AIRCRAFT CALL SIGN

 

 

 

 

4A

CONTROLLED AIRCRAFT CALL SIGN

 

ENTER THE CALL SIGN OF THE LEAD AIRCRAFT OF THE CONTROLLED MISSION, FOR EXAMPLE: "B106". See table 1129/19 which is an instructive entry

NUMBER OF AIRCRAFT UNDER AIRBORNE EARLY WARNING CONTROL

 

 

 

 

5A

NUMBER OF AIRCRAFT UNDER AIRBORNE EARLY WARNING CONTROL

 

SPECIFY THE NUMBER OF AIRCRAFT CONTROLLED, FOR EXAMPLE: "4"  (ENTER A HYPHEN IF UNKNOWN). See table 1044/228 a range [0 through 999]

6

DESIGNATED AREA NAME

 

 

 

 

6A

DESIGNATED AREA NAME

 

SPECIFY THE NAME OF AREA IN WHICH THE CONTROL EVENT TOOK PLACE, FOR EXAMPLE: "R77". See table 1022/102 which is an instructive entry

NUMBER OF CONTROL EVENTS

 

 

 

 

7A

NUMBER OF CONTROL EVENTS
