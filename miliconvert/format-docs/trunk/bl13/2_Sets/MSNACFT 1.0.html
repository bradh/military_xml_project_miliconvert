No

Designator

Field Desc

Concept/Explanation/Examples

1

AIRCRAFT NUMBER

 

 

 

 

1A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT PARTICIPATING IN THE ASSIGNED MISSION, FOR EXAMPLE: "1". See table 1044/322 a range [1 through 99]

2

AIRCRAFT TYPE

 

PROVIDES THE TYPE OF AIRCRAFT TASKED.

 

 

2A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE AND MODEL NUMBER OF THE AIRCRAFT, FOR EXAMPLE: "ACTYP:FA18". See table 1015/20 which contains a list of data items and associated data codes

 

 

2B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF AN AIRCRAFT NOT CONTAINED IN THE AIRCRAFT CODES LIST,  FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

3

CALL SIGN

 

PROVIDES THE CALL SIGN ASSIGNED TO THE MISSION AIRCRAFT.

 

 

3A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN ASSIGNED TO THE MISSION AIRCRAFT, FOR EXAMPLE: "HAMMER 53". See table 1129/31 which is an instructive entry

4

PRIMARY CONFIGURATION LOAD

 

PROVIDES THE PRIMARY CONFIGURATION LOAD FOR THE TASKED AIRCRAFT.

 

 

4A

PRIMARY CONFIGURATION LOAD

 

ENTER THE PRIMARY CONFIGURATION LOAD FOR THE TASKED AIRCRAFT, FOR EXAMPLE: "2HARM". See table 1012/258 which is an instructive entry

5

SECONDARY CONFIGURATION LOAD

 

PROVIDES THE SECONDARY CONFIGURATION LOAD FOR THE TASKED AIRCRAFT.

 

 

5A

SECONDARY CONFIGURATION LOAD

 

ENTER THE SECONDARY CONFIGURATION LOAD FOR THE TASKED AIRCRAFT, FOR EXAMPLE: "2S2WG". See table 1012/259 which is an instructive entry

6

LINK 16 CALL SIGN

 

 

 

 

6A

LINK 16 ABBREVIATED CALL SIGN

 

ENTER THE LINK 16 ABBREVIATED CALL SIGN IF THE TASKED AIRCRAFT IS LINK 16 EQUIPPED, FOR EXAMPLE: "DA24". See table 1129/41 which is an instructive entry

7

TACAN CHANNEL

 

 

 

 

7A

TACAN CHANNEL WITH MODES

 

ENTER THE TACAN CHANNEL OF THE TASKED AIRCRAFT, FOR EXAMPLE: "25X". See table 2103 which is a composite

8

JTIDS UNIT ADDRESS

 

 

 

 

8A

PRIMARY JTIDS UNIT (JU) ADDRESS

 

ENTER THE JTIDS UNIT ADDRESS OF THE TASKED AIRCRAFT, FOR EXAMPLE: "34568". See table 1625/32 which is an alphanumeric range

9

IFF/SIF MODE AND CODE

 

 

 

 

9A

IFF/SIF MODE AND CODE, NON HYPHENATED

 

ENTER THE MODE AND CODE OF A SPECIFIC IDENTIFICATION FRIEND OR FOE/SELECTIVE IDENTIFICATION FEATURE (IFF/SIF), FOR THE TASKED AIRCRAFT, FOR EXAMPLE: "20153". See table 2500 which is a composite
