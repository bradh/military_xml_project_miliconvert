EFFECTIVE WIND SPEED AND UNIT OF MEASUREMENT

 

 

 

 

1A

EFFECTIVE WIND SPEED AND UNIT OF MEASUREMENT

 

ENTER THE EFFECTIVE WIND SPEED (000-999) FOLLOWED BY MEASUREMENT, FOR EXAMPLE: "009KPH". IF THE WIND SPEED IS GREATER THAN 999KPH, ENTER A HYPEN "-". See table 2318 which is a composite

DOWNWIND DISTANCE OF ZONE 1 AND UNIT OF MEASURE

 

 

 

 

2A

DOWNWIND DISTANCE OF ZONE 1 AND UNIT OF MEASURE

 

ENTER THE DOWNWIND DISTANCE OF ZONE 1 (001-999), STARTING FROM POINT ZERO, FOLLOWED BY UNIT OF MEASUREMENT, FOR EXAMPLE: "003KM". See table 2320 which is a composite

CLOUD RADIUS AND UNIT OF MEASUREMENT

 

 

 

 

3A

CLOUD RADIUS AND UNIT OF MEASUREMENT

 

ENTER THE CLOUD RADIUS (01-99) FOLLOWED BY UNIT OF MEASUREMENT, FOR EXAMPLE: "05KM". See table 2316 which is a composite

LEFT RADIAL LINE DIRECTION

 

PROVIDES THE LEFT RADIAL LINE DIRECTION AND UNIT OF MEASUREMENT IN ONE OF THE FOLLOWING WAYS:

 

 

4A

LEFT RADIAL LINE IN DEGREES

LEFT

ENTER THE FIELD DESCRIPTOR "LEFT:" FOLLOWED BY THE LEFT RADIAL LINE DIRECTION IN DEGREES, FOR EXAMPLE: "LEFT:250DGT". See table 2321 which is a composite

 

 

4B

LEFT RADIAL LINE IN MILS

LEFTM

ENTER THE FIELD DESCRIPTOR "LEFTM:" FOLLOWED BY THE LEFT RADIAL LINE DIRECTION IN MILS, FOR EXAMPLE: "LEFTM:1600MLT". See table 2322 which is a composite
