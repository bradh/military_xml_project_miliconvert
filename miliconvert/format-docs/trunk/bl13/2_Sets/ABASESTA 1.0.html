No

Designator

Field Desc

Concept/Explanation/Examples

1

AIR BASE STATUS

 

 

 

 

1A

AIR BASE STATUS

 

ENTER THE LATEST AVAILABLE STATUS OF THE AIR BASE, FOR EXAMPLE: "3". See table 1236/27 which contains a list of data items and associated data codes

2

REASON OPERATIONAL STATUS DEGRADED

 

MUST BE SPECIFIED IF THE AIR BASE STATUS IS NOT "FULLY OPERATIONAL".

 

 

2A

REASON AIR BASE OPERATIONAL STATUS DEGRADED

 

ENTER THE REASON THE OPERATIONAL STATUS OF THE BASE HAS BEEN DOWNGRADED, FOR EXAMPLE: "4" See table 1235/4 which contains a list of data items and associated data codes

3

TIME OPERATIONAL

 

MUST BE SPECIFIED IF THE AIR BASE STATUS IS NOT "FULLY OPERATIONAL".

 

 

3A

ESTIMATED DTG IN CHARGE OF NEXT OPERATIONAL STATUS (ETIC)

ETIC

ENTER "ETIC:" FOLLOWED BY THE DTG FOR THE NEXT CHANGE IN AIRBASE OPERATIONAL STATUS, FOR EXAMPLE "ETIC:261200ZMAY2005". See table 2033 which is a composite

4

NEXT AIRBASE STATUS

 

MUST BE SPECIFIED IF THE AIR BASE STATUS IS NOT "FULLY OPERATIONAL".

 

 

4A

NEXT AIR BASE STATUS

 

ENTER THE ANTICIPATED STATUS OF THE AIRBASE AT NEXT REPORTING PERIOD, FOR EXAMPLE: "2". See table 1236/29 which contains a list of data items and associated data codes

5

AIRBASE STATUS LIMITING FACTORS

 

 

 

 

5A

AIR BASE STATUS LIMITING FACTORS

 

ENTER ANY FACTOR THAT LIMITS THE OPERATIONAL CAPABILITY OF THE AIRBASE, FOR EXAMPLE: "41".IF THE CONTENT OF THIS FIELD EQUALS "OTHER", AN AMPN SET SHOULD BE INCLUDED TO EXPLAIN THE LIMITING FACTORS. See table 1347/20 which contains a list of data items and associated data codes
