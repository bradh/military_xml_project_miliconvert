NUMBER OF ADDITIONAL VESSELS IN AREA

SHIP PENNANT PREFIX

PENNANT NUMBER

SHIP CONTROL NUMBER

 

ENTER THE SHIP CONTROL NUMBER, FOR EXAMPLE: "M12345". See table 2282 which is a composite

10

NATO DESIGNATOR OR DISCRETE IDENTIFIER

 

SPECIFIES THE NATO DESIGNATOR OR IDENTIFIER OF THE CONTACT.

 

 

10A

NATO SURFACE CONTACT DESIGNATOR

DESIG

ENTER FIELD DESCRIPTOR "DESIG:" FOLLOWED BY THE NATO SURFACE CONTACT DESIGNATOR, FOR EXAMPLE: "DESIG:34ZA". See table 2154 which is a composite

 

 

10B

DISCRETE IDENTIFIER

 

ENTER THE DISCRETE IDENTIFIER (A MARITIME REPORTING CODE FOR TRACKS THAT ARE NOT IDENTIFIED AS "FRIEND", "ASSUMED FRIEND" OR "UNKNOWN") FOR THE CONTACT, FOR EXAMPLE: "1134". See table 1625/25 which is an alphanumeric range

11

VESSEL SUSPICION CODE

 

 

 

 

11A

VESSEL SUSPICION CODE

 

ENTER THE NUMERIC CODE INDICATING THE LEVEL OF SUSPICION CONCERNING THE VESSEL, FOR EXAMPLE: "01". See table 1698/2 which contains a list of data items and associated data codes
