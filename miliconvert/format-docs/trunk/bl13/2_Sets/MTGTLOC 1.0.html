No

Designator

Field Desc

Concept/Explanation/Examples

1

PRIMARY/ALTERNATE TARGET DESIGNATOR

 

PROVIDES INFORMATION WHETHER THE TARGET IS PRIMARY OR ALTERNATE.

 

 

1A

PRIMARY ALTERNATE TARGET DESIGNATOR

 

ENTER A DESIGNATOR THAT CLASSIFIES THE TARGET AS PRIMARY  OR ALTERNATE, FOR EXAMPLE: "P". See table 1076/13 which contains a list of data items and associated data codes

2

TIME ON TARGET

 

PROVIDES THE TIME ON TARGET.

 

 

2A

DAY-TIME AND MONTH TASKED ON TARGET

TOT

ENTER "TOT:"FOLLOWED BY THE DAY-TIME AND MONTH TASKED ON THE TARGET, FOR EXAMPLE: "TOT:141325ZFEB". See table 2030 which is a composite

 

 

2B

DAY-TIME ON TARGET TO THE SECOND, ZULU

TOTS

ENTER "TOTS:" FOLLOWED BY THE DAY-TIME MEASURED TO THE SECOND, FOR EXAMPLE: "TOTS:14132535Z". See table 2483 which is a composite

3

NET TIME ON TARGET

 

MUST BE USED IF TIME ON TARGET IS NOT PROVIDED, PROVIDES THE NOT EARLIER THAN TIME ON TARGET.

 

 

3A

NOT EARLIER THAN DAY-TIME

NET

ENTER "NET:" FOLLOWED BY THE DAY-TIME, FOR EXAMPLE: "NET:141325Z". See table 2000 which is a composite

 

 

3B

NOT EARLIER THAN DAY-TIME TO THE SECOND, ZULU

NETS

ENTER "NETS:" FOLLOWED BY THE DAY-TIME MEASURED TO THE SECOND, FOR EXAMPLE: "NETS:14132535Z". See table 2483 which is a composite

4

NLT TIME ON TARGET

 

MUST BE USED IF TIME ON TARGET IS NOT PROVIDED, PROVIDES THE NOT LATER THAN DAY-TIME ON TARGET.

 

 

4A

NOT LATER THAN DAY-TIME

NLT

ENTER "NLT:" FOLLOWED BY THE LATEST DAY-TIME WHICH THE TARGET CAN BE ATTACKED, FOR EXAMPLE: "NLT:141335Z". See table 2000 which is a composite

5

TARGET IDENTIFIER

 

PROVIDES THE TARGET IDENTIFICATION, EITHER BY ITS NAME OR BY AN IDENTIFIER, WHICH IS NORMALLY A BE NUMBER, A THEATRE TARGET NUMBER, OR A UNIQUE IDENTIFICATION SCHEME DESIGNATED BY THE OPERATIONAL COMMANDER.

 

 

5A

TARGET POSITION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE TARGET POSITION NAME, FOR EXAMPLE: "NAME:SAM SITE". See table 1022/81 which is an instructive entry

 

 

5B

TARGET POSITION IDENTIFIER

ID

ENTER "ID:" FOLLOWED BY THE UNIQUE IDENTIFIER OF A TARGET SUCH AS THE BE NUMBER OR THEATRE TARGET NUMBER, FOR EXAMPLE: "ID:1234FA1234ZZ". See table 1416/2 which is an instructive entry

6

TARGET TYPE

 

PROVIDES THE TARGET TYPE, EITHER SHIP OR NON-SHIP.

 

 

6A

NON-SHIP TARGET TYPE

NON-SHIP

ENTER "NON-SHIP:" FOLLOWED BY THE CODE FOR THE TYPE OF NON-SHIP TARGET, FOR EXAMPLE: "NON-SHIP:FTAFLD". See table 1650/6 which contains a list of data items and associated data codes

 

 

6B

VESSEL TYPE

SHIP

ENTER "SHIP:" FOLLOWED BY THE CODE FOR THE TYPE OF SHIP TARGET, FOR EXAMPLE: "SHIP:CG". See table 1166/1 which contains a list of data items and associated data codes

7

MISSILE TYPE

 

PROVIDES THE ONE ALPHABETIC CHARACTER CODE CORRESPONDING TO THE TYPE OF MISSILE ON TARGET.

 

 

7A

MISSILE TYPE, ABBREVIATED

 

ENTER THE ABBREVIATED CODE TO INDICATE THE TYPE OF MISSILE TO BE USED, FOR EXAMPLE: "C". See table 1407/27 which contains a list of data items and associated data codes

8

MISSILES NUMBER

 

PROVIDES THE NUMBER OF MISSILES ON TARGET.

 

 

8A

NUMBER OF MISSILES

 

ENTER THE NUMBER OF MISSILES TO BE USED, FOR EXAMPLE: "3". See table 1044/323 a range [0 through 999]

9

TARGET PRIORITY

 

IF USED, PROVIDES AS A LITERAL THE PRIORITY ASSIGNED TO THE TARGET.

 

 

9A

TARGET PRIORITY, LITERAL

 

ENTER THE PRIORITY ASSIGNED TO THE TARGET, FOR EXAMPLE: "1A". See table 1076/11 which is an instructive entry

10

TARGET OBJECTIVE

 

IF USED, PROVIDES THE GOAL OF THE MISSION ON TARGET IN PLAIN LANGUAGE.

 

 

10A

TARGET OBJECTIVE

 

ENTER THE GOAL OF THE MISSION ON THE TARGET IN PLAIN LANGUAGE, FOR EXAMPLE: "DESTROY". See table 1456/3 which is an instructive entry

11

COMPONENT TARGET IDENTIFIER

 

IF USED, PROVIDES EITHER THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE SERVICE COMPONENT, THE COMPONENT TARGET IDENTIFIER OR THE TARGET EIRN NUMBER.

 

 

11A

COMPONENT TARGET IDENTIFIER

CTGT

ENTER "CTGT:" FOLLOWED BY THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE SERVICE COMPONENT, FOR EXAMPLE: "CTGT:N4792601". See table 1416/5 which is an instructive entry

 

 

11B

TARGET ENTITY ID REFERENCE NUMBER

EIRN

ENTER "EIRN:" FOLLOWED BY THE EIRN NUMBER TO IDENTIFY A TARGET ON A VARIABLE MESSAGE INTERFACE (VMF) INTERFACE, FOR EXAMPLE: "EIRN:167772104294967290" See table 2085 which is a composite

12

DMPI IDENTIFIER

 

IF USED, PROVIDES THE IDENTIFIER OF THE DESIRED MEAN POINT OF IMPACT (DMPI) OF WEAPONS ON TARGET.

 

 

12A

TARGET DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER

 

ENTER THE DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER, FOR EXAMPLE: "QRS 1236-RTY". See table 1416/7 which is an instructive entry

13

DESIRED MEAN POINT OF IMPACT

 

IF USED, PROVIDES THE GEOGRAPHICAL COORDINATES OF THE DESIRED MEAN POINT OF IMPACT (DMPI) OF WEAPONS ON TARGET.

 

 

13A

GEOGRAPHIC LOCATION, LAT/LONG, CENTISECONDS

LATCS

ENTER "LATCS:", FOLLOWED BY THE LOCATION COORDINATES IN LAT/LONG, CENTISECONDS, FOR EXAMPLE: "LATCS:35102530N079012545W". See table 2488 which is a composite

 

 

13B

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

14

GEODETIC DATUM

 

IF USED, PROVIDES THE GEOGRAPHIC DATUM CONCERNING THE TARGET.

 

 

14A

GEODETIC DATUM

 

IF THE DEFAULT GEODETIC DATUM SPECIFIED IN THE BEGINNING OF THE MESSAGE DOES NOT APPLY TO THE LOCATION OF THE DMPI, ENTER THE CORRECT GEODATUM, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

15

PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR

 

USED ONLY IF THE DESIRED MEAN POINT OF IMPACT GEOGRAPHIC COORDINATES ARE PROVIDED AT SEQ.13, PROVIDES THE PROBABLE DMPI COORDINATE HORIZONTAL CIRCULAR ERROR.

 

 

15A

PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR

 

ENTER THE PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR, FOR EXAMPLE: "10.55FT". See table 2532 which is a composite

16

DMPI ELEVATION

 

USED ONLY IF THE DESIRED MEAN POINT OF IMPACT GEOGRAPHIC COORDINATES ARE PROVIDED AT SEQ.13, PROVIDES THE DMPI ELEVATION.

 

 

16A

DESIRED MEAN POINT OF IMPACT ELEVATION IN FEET OR METRES

 

ENTER THE DESIRED MEAN POINT OF IMPACT ELEVATION IN FEET OR METRES MEASURED ABOVE OR BELOW THE MEAN SEA LEVEL OR ABOVE THE REFERNCE ELLIPSOID, FOR EXAMPLE: "-200 FTMSL". See table 2492 which is a composite

17

PROBABLE COORDINATE ELEVATION LINEAR ERROR

 

USED ONLY IF THE DESIRED MEAN POINT OF IMPACT ELEVATION IS PROVIDED AT SEQ.16, PROVIDES THE PROBABLE DMPI COORDINATE ELEVATION LINEAR ERROR.

 

 

17A

PROBABLE COORDINATE ELEVATION LINEAR ERROR

 

ENTER THE PROBABLE COORDINATE ELEVATION LINEAR ERROR, FOR EXAMPLE: "20.55FT". See table 2532 which is a composite

18

TERMINAL AREA MODEL IDENTIFICATION

 

IF USED, PROVIDES THE TARGET TERMINAL AREA MODEL IDENTIFICATION.

 

 

18A

TERMINAL AREA MODEL IDENTIFICATION

TAMID

ENTER "TAMID:" FOLLOWED BY THE IDENTIFICATION OF THE TERMINAL AREA MODEL FOR THE JOINT AIR TO SURFACE STANDOFF MISSILE, FOR EXAMPLE: "TAMID:34678912452AM9". See table 1012/317 which is an instructive entry

19

AIMPOINT SUFFIX

 

IF USED, PROVIDES A TWO CHARACTER SUFFIX TO BE USED IN THE BASIC ENCYCLOPEDIA (BE) NUMBER OF THE TARGET.

 

 

19A

AIMPOINT SUFFIX

 

ENTER A TWO CHARACTER SUFFIX FOR THE TARGET IDENTIFIER, REPEATING AS NECESSARY, FOR EXAMPLE: "EB". See table 1012/244 which is an instructive entry
