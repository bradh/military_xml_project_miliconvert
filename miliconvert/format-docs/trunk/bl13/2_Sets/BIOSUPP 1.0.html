Designator

Field Desc

Concept/Explanation/Examples

1

ALLEGIANCE

 

 

 

 

1A

ALLEGIANCE

 

INDICATES THE COUNTRY (STANAG 1059) TO WHICH THE INDIVIDUAL GIVES HIS LOYALTY, FOR EXAMPLE: "UK" See table 1026/17 which contains a list of data items and associated data codes

2

MARITAL STATUS

 

 

 

 

2A

MARITAL STATUS

 

DEFINES THE MARITAL STATUS OF THE INDIVIDUAL, FOR EXAMPLE: "M". See table 1727/1 which contains a list of data items and associated data codes

3

NUMBER OF CHILDREN

 

 

 

 

3A

NUMBER OF CHILDREN

 

CAPTURES THE NUMBER OF KNOWN CHILDREN AS A NUMERIC VALUE, FOR EXAMPLE: "12". See table 1044/337 a range [0 through 99]

4

BELIEF OR RELIGION

 

 

 

 

4A

BELIEF OR RELIGION

 

THE NAME IDENTIFYING THE INDIVIDUAL'S FAITH, BELIEF OR RELIGION, FOR EXAMPLE: "MARONITE". See table 1016/2 which is an instructive entry

5

ETHNIC GROUP

 

 

 

 

5A

ETHNIC GROUP

 

THE GROUP WITH WHICH THE INDIVIDUAL HAS A STRONG AFFILIATION, FOR EXAMPLE: "EQUATORIAL BANTU". THIS AFFILIATION CAN BE THE RESULT OF ONE OR MORE OF THE FOLLOWING FACTORS: RACIAL, CULTURAL, LINGUISTIC OR RELIGIOUS. See table 1646/1 which is an instructive entry

6

DATE OF DEATH, YYYYMMDD

 

 

 

 

6A

DATE OF DEATH, YYYYMMDD

 

DATE OF DEATH IN THE FORM YYYYMMDD, FOR EXAMPLE: "19960331". See table 2053 which is a composite

7

PLACE NAME

 

 

 

 

7A

PLACE NAME, EXTENDED

 

IDENTIFIES THE CITY, TOWN, VILLAGE OR OTHER ADMINISTRATIVE DISTRICT WHERE THE INDIVIDUAL DIED, FOR EXAMPLE: "LONDON". See table 1022/261 which is an instructive entry

8

COUNTRY

 

 

 

 

8A

COUNTRY

 

IDENTIFIES THE COUNTRY (STANAG 1059) WHERE THE INDIVIDUAL DIED, FOR EXAMPLE: "UK". See table 1026/18 which contains a list of data items and associated data codes

9

COMMENTS

 

 

 

 

9A

COMMENTS

 

THIS FIELD ALLOWS THE RECORDING OF ANY OTHER DATA [LITERAL] WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS IN THIS SET, FOR EXAMPLE: "(REMARKS IN FREE TEXT UP TO 68 CHARACTERS)". See table 1357/1 which is an instructive entry
