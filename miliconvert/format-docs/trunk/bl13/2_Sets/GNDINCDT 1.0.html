Designator

Field Desc

Concept/Explanation/Examples

1

UNIT IDENTIFIER

 

PROVIDES THE UNIT IDENTIFICATION OF THE UNIT HAVING THE INCIDENT.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE IDENTITY OF THE REPORTING UNIT, FOR EXAMPLE: "UNIT:12.PANZERDIVISION". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE IDENTIFICATION CODE OF THE UNIT HAVING THE INCIDENT, FOR EXAMPLE: "UIC:GEA1234". See table 2141 which is a composite

2

DAY-TIME OF INCIDENT

 

 

 

 

2A

DAY-TIME OF INCIDENT

 

ENTER THE DAY-TIME OF THE GROUND INCIDENT, FOR EXAMPLE: "112301Z". See table 2000 which is a composite

3

INCIDENT POSITION

 

THE LAST KNOWN POSITION OF THE INCIDENT UNIT.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

3C

LOCATION OR POSITION NAME

LOC

ENTER "LOC:" FOLLOWED BY THE NAME ASSIGNED TO THE LOCATION, FOR EXAMPLE: "LOC:GARGANTA DEL DIABLO". See table 1022/6 which is an instructive entry

 

 

3D

BEARING AND DISTANCE FROM REFERENCE POINT

TBR

ENTER "TBR:" FOLLOWED BY THE BEARING DISTANCE FROM REF POINT, FOR EXAMPLE: "TBR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

4

LOCATION QUALIFIER

 

 

 

 

4A

LOCATION QUALIFIER

 

ENTER THE LOCATION QUALIFIER, FOR EXAMPLE: "AA". See table 1118/1 which contains a list of data items and associated data codes

5

CALL SIGN

 

 

 

 

5A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN OF THE UNIT INVOLVED IN THE INCIDENT, FOR EXAMPLE: "SHADOW 11". See table 1129/31 which is an instructive entry

6

PERSONNEL COUNT

 

 

 

 

6A

PERSONNEL COUNT

 

ENTER THE NUMBER OF PERSONNEL INVOLVED IN THE INCIDENT, FOR EXAMPLE: "315". See table 1044/133 a range [0 through 999]
