No

Designator

Field Desc

Concept/Explanation/Examples

1

NATIONALITY

 

THE NATIONALITY OF THE ALLOCATED ASACS UNIT, EITHER AS A 2 OR AS A 3 LETTER COUNTRY CODE.

 

 

1A

NATIONALITY (2A)

 

ENTER THE 2 LETTER NATO COUNTRY CODE CORRESPONDING TO THE NATIONALITY OF THE ALLOCATED ASACS UNIT, FOR EXAMPLE: "BE". See table 1262/5 which contains a list of data items and associated data codes

 

 

1B

NATIONALITY (3A)

 

ENTER THE NATO 3 LETTER COUNTRY CODE CORRESPONDING TO THE NATIONALITY OF THE ALLOCATED ASACS UNIT, IN ACCORDANCE WITH STANAG 1059, ED. 8, FOR EXAMPLE: "BEL". See table 1262/2 which contains a list of data items and associated data codes

 

 

1C

EXERCISE GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE CORRESPONDING TO THE FICTITOUS NATIONALITY OF THE ASACS UNIT ALLOCATED FOR THE EXERCISE, IN ACCORDANCE WITH STANAG 1059, ED. 8, FOR EXAMPLE: "XB". See table 1262/4 which contains a list of data items and associated data codes

2

ARMED SERVICE CODE

 

 

 

 

2A

ARMED SERVICE CODE

 

ENTER THE ALPHABETIC CHARACTER CODE CORRESPONDING TO THE ARMED SERVICE TO WHICH THE ALLOCATED ASACS UNITS BELONG, FOR EXAMPLE: "F". See table 1107/2 which contains a list of data items and associated data codes

3

UNIT IDENTIFIER

 

THE NAME OR UNIT IDENTIFICATION CODE CORRESPONDING TO THE ALLOCATED ASACS UNIT.

 

 

3A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME CORRESPONDING TO THE ALLOCATED ASACS UNITS, FOR EXAMPLE: "UNIT:ARS GLONS". See table 1028/43 which is an instructive entry

 

 

3B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (USING THE NATO 2 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED ASACS UNITS, FOR EXAMPLE: "UIC2:BEJ313131". See table 2243 which is a composite

 

 

3C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (USING THE NATO 3 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED ASACS UNITS, FOR EXAMPLE: "UIC3:BELJ313131". See table 2241 which is a composite

4

BASE IDENTIFIER

 

THE GEOGRAPHICAL LOCATION OF THE BASE WHERE THE ALLOCATED ASACS UNIT IS SITUATED.

 

 

4A

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR OF THE PLACE WHERE THE ALLOCATED ASACS UNIT IS SITUATED, FOR EXAMPLE: "ICAO:EBGL". See table 1125/1 which is an instructive entry

 

 

4B

PLACE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE BASE WHERE THE ALLOCATED ASACS UNIT IS SITUATED, FOR EXAMPLE: "NAME:GLONS". See table 1022/170 which is an instructive entry

 

 

4C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

5

LOCATION IDENTIFIER

 

 

 

 

5A

LOCATION OR POSITION NAME

 

ENTER AS A LITERAL THE NAME OF THE LOCATION WHERE THE ALLOCATED ASACS UNIT IS SITUATED, FOR EXAMPLE:"GLONS". See table 1022/6 which is an instructive entry

6

LINK 16 JTIDS UNIT (JU) ADDRESS

 

 

 

 

6A

UNIT PRIMARY JU ADDRESS NUMBERS

 

ENTER FIRST THE LOWER LIMIT LINK 16 ADDRESS AND THEN THE UPPER LIMIT LINK 16 ADDRESS SEPARATED BY A HYPHEN DEFINING THE BLOCK OF LINK 16 ADDRESSES ASSIGNED TO THE ALLOCATED ASACS UNIT, FOR EXAMPLE: "34521-65341". See table 2606 which is a composite

7

ASACS UNIT TYPE

 

 

 

 

7A

GROUND AGENCY TYPE

 

ENTER THE TYPE OF ALLOCATED ASACS UNIT, FOR EXAMPLE: "CRP". See table 1874/5 which contains a list of data items and associated data codes

8

START OF ALLOCATION PERIOD

 

 

 

 

8A

DTG OF START

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF THE START OF THE ALLOCATION TIME PERIOD, FOR EXAMPLE: "141325ZFEB2003". See table 2033 which is a composite

9

STOP TIME OF ALLOCATION PERIOD

 

CAN ONLY BE USED IN CONJUNCTION WITH "START OF ALLOCATION PERIOD" SPECIFIED IN THIS SET.

 

 

9A

DTG OF STOP

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF THE STOP OF THE ALLOCATION TIME PERIOD, FOR EXAMPLE: "141925ZFEB2003". See table 2033 which is a composite

10

UNIT REMARKS

 

 

 

 

10A

REMARKS

 

ENTER, AS LITERAL, ADDITIONAL ESSENTIAL INFORMATION CONCERNING THE ALLOCATED ASACS UNIT, FOR EXAMPLE: "GRAND 1 OUT". See table 1357/28 which is an instructive entry
