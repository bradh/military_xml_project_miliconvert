1

RECORD ORIGINATOR

 

 

 

 

1A

RECORD ORIGINATOR

 

THIS FIELD IDENTIFIES THE ORIGINATOR OF THE DATA CONTAINED IN THE RELATED SET(S). ENTER A FOUR LETTER CODE, FOR EXAMPLE: "DA99". See table 1029/5 which is an instructive entry

2

INTELLIGENCE SECURITY CLASSIFICATION, ABBREVIATED

 

 

 

 

2A

INTELLIGENCE SECURITY CLASSIFICATION, ABBREVIATED

 

ENTER THE HIGHEST CLASSIFICATION LEVEL OF DATA CONTAINED IN THE SEGMENT, SET OR NUMBER OF SETS THAT THE ADMINDAT SET REFERS TO. USE DATA CODES AS LAID DOWN IN AINTP-3,  FOR EXAMPLE: "T". See table 1288/3 which contains a list of data items and associated data codes

3

RECORD STATUS

 

 

 

 

3A

RECORD STATUS

 

ENTER A CODE INDICATING THE FUNCTIONAL STATUS OF THE DATA IN THE RECORD, FOR EXAMPLE: "A". See table 1584/6 which contains a list of data items and associated data codes

4

DATE-TIME (Z) CREATED, YYYYMMDDHHMMSS

 

 

 

 

4A

DATE-TIME (Z) CREATED, YYYYMMDDHHMMSS

CREATED

ENTER "CREATED:" FOLLOWED BY A 14 DIGIT DATE OF THE STRUCTURE YYYYMMDDHHMMSS  INDICATING WHEN THE DATA TO WHICH THIS ADMINISTRATION SET REFERS WAS CREATED, FOR EXAMPLE: "CREATED:19990710011523". See table 2417 which is a composite

5

DATE-TIME (Z) OF FIRST INFORMATION, YYYYMMDDHHMMSS

 

 

 

 

5A

DATE-TIME (Z) OF FIRST INFORMATION, YYYYMMDDHHMMSS

FIRST

ENTER "FIRST:" FOLLOWED BY A 14 DIGIT DATE OF THE STRUCTURE YYYYMMDDHHMMSS INDICATING WHEN THE INTELLIGENCE DATA WAS FIRST MADE AVAILABLE FOR THE RECORD, FOR EXAMPLE: "FIRST:19990710233422". See table 2417 which is a composite

6

DATE-TIME (Z)  OF LAST INFORMATION, YYYYMMDDHHMMSS

 

 

 

 

6A

DATE-TIME (Z)  OF LAST INFORMATION, YYYYMMDDHHMMSS

LAST

ENTER "LAST:" FOLLOWED BY A 14 DIGIT DATE OF THE STRUCTURE YYYYMMDDHHMMSS INDICATING THE DATE-TIME OF THE LATEST INFORMATION ON WHICH THE DATA IS BASED, FOR EXAMPLE: "LAST:19990710163435". See table 2417 which is a composite

7

REVIEW DATE (Z), YYYYMMDD

 

 

 

 

7A

REVIEW DATE (Z), YYYYMMDD

REVIEW

ENTER "REVIEW:" FOLLOWED BY AN 8 DIGIT DATE OF THE STRUCTURE YYYYMMDD INDICATING WHEN THE DATA IN THE RECORD WAS LAST REVIEWED BY THE RESPONSIBLE ANALYST FOR ACCURACY AND CURRENCY, FOR EXAMPLE: "REVIEW:19990711". See table 2053 which is a composite

8

DATE-TIME (Z) OF LAST CHANGE, YYYYMMDDHHMMSS

 

 

 

 

8A

DATE-TIME (Z) OF LAST CHANGE, YYYYMMDDHHMMSS

LASTCHA

ENTER "LASTCHA:" FOLLOWED BY A 14 DIGIT DATE OF THE STRUCTURE YYYYMMDDHHMMSS INDICATING WHEN THE RECORD WAS LAST CHANGED, FOR EXAMPLE: "LASTCHA:19990711194537". See table 2417 which is a composite

9

VALIDITY

 

 

 

 

9A

VALIDITY

 

ENTER A CODE TAKEN FROM THE AINTP-3 DATA DICTIONARY  EXPRESSING THE VALIDITY OF THE DATA TO WHICH THIS ADMINISTRATION SET REFERS, FOR EXAMPLE: "1". See table 1266/2 which contains a list of data items and associated data codes
