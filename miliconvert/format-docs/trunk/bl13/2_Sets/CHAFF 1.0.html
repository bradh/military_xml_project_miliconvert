Remarks:

THIS SET HAS A DIRECT RELATIONSHIP TO THE 8CHAFF SET.  CHAFF AND 8CHAFF PRESENT TWO DIFFERENT VIEWS OF THE SAME INFORMATION; THEREFORE, ANY CHANGES TO ONE OF THEM SHOULD ALSO BE MADE TO THE OTHER.  THE CHAFF SET IS USED IN CONTEXTS WHERE ALL THE DETAILS FOR EACH MISSION ARE LISTED CONTIGUOUSLY AND DO NOT REQUIRE AREFERENCE NUMBER TO TIE THEM TO OTHER PERTINENT MISSION DETAILS.  THE 8CHAFF SET, ON THE OTHER HAND, IS USED IN CONTEXTS WHERE A SUMMARY OF SEVERAL DIFFERENT MISSIONSIS DESIRED.  IN THIS CASE, THE REFERENCE NUMBER IS REQUIRED TO LINK THIS SUMMARY TOOTHER SUMMARIES OF RELATED MISSION DETAILS.
