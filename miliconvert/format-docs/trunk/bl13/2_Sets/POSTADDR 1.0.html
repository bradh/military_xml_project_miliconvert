1

STATE/PROVINCE/DISTRICT

 

 

 

 

1A

STATE PROVINCE DISTRICT

 

DATA CODE IS TAKEN FROM STANAG 1059 AND CONSISTS OF FOUR CHARACTERS REPRESENTING A REGIONAL ENTITY WHERE THE ADDRESS IS LOCATED, FOR EXAMPLE: "BU12". THE FIRST TWO REPRESENTS THE COUNTRY CODE, WHILE THE SECOND ARE NUMBERS THAT REFER TO SPECIFIC POLITICAL OR ADMINISTRATIVE SUB-DIVISIONS WITHIN THE COUNTRY. See table 1711/2 which contains a list of data items and associated data codes

2

COUNTRY

 

THIS FIELD MUST OCCUR, IF FIELD 1 DOES NOT OCCUR.

 

 

2A

COUNTRY

 

TWO LETTER CODE FROM STANAG 1059 THAT INDICATES THE COUNTRY IN WHICH THE ADDRESS IS LOCATED, FOR EXAMPLE: "BU". See table 1026/18 which contains a list of data items and associated data codes

3

POSTAL CODE

 

 

 

 

3A

POSTAL CODE

 

A POSTAL CODE USED IN AN ADDRESS TO FACILITATE DELIVERY OF MAIL, FOR EXAMPLE: "2240". See table 1610/1 which is an instructive entry

4

ADDRESS PLACE NAME

 

 

 

 

4A

ADDRESS PLACE NAME

 

A CITY, TOWN, VILLAGE WHERE A UNIT, FACILITY, OR INSTALLATION IS LOCATED, FOR EXAMPLE: "PLOVDIV AIRPORT". See table 1022/244 which is an instructive entry

5

STREET NAME

 

 

 

 

5A

STREET NAME

 

THE NAME OF A STREET ON WHICH A FACILITY OR INSTALLATION IS LOCATED USED IN THE ADDRESS, FOR EXAMPLE: "ULITZA ASENOVGRADSKY". See table 1022/246 which is an instructive entry

6

STREET NUMBER

 

 

 

 

6A

STREET NUMBER

 

NUMBER OF THE HOUSE, APARTMENT OR OTHER BUILDING USED IN THE ADDRESS, FOR EXAMPLE: "24". See table 1012/263 which is an instructive entry

7

TELEPHONE NUMBER

 

 

 

 

7A

TELEPHONE NUMBER

 

TELEPHONE NUMBER ALLOCATED TO A FACILITY OR INSTALLATION, FOR EXAMPLE: "+359.03964", REPEATING AS NECESSARY. See table 1361/2 which is an instructive entry

8

COMMENTS

 

 

 

 

8A

COMMENTS

 

THIS FIELD ALLOWS THE RECORDING OF ANY OTHER DATA [LITERAL] WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS IN THIS SET, FOR EXAMPLE: "(REMARKS IN FREE TEXT UP TO 68 CHARACTERS)". See table 1357/1 which is an instructive entry
