Concept/Explanation/Examples

1

TARGET CATEGORY PURPOSE

 

 

 

 

1A

TARGET CATEGORY PURPOSE

 

ENTER THE TARGET CATEGORY PURPOSE CODE, FOR EXAMPLE: "C". See table 1435/2 which contains a list of data items and associated data codes

2

TARGET CATEGORY DESIGNATOR

 

 

 

 

2A

TARGET CATEGORY DESIGNATOR

 

ENTER TARGET CATEGORY DESIGNATOR (STANAG 3596), FOR EXAMPLE: "AIRFIELD". See table 1309/4 which contains a list of data items and associated data codes

3

NAME OF TARGET

 

ENTER NAME OF ITEM IN ONE OF THE FOLLOWING WAYS:

 

 

3A

INSTALLATION NAME

INST

ENTER "INST:" FOLLOWED BY THE INSTALLATION NAME, FOR EXAMPLE: "INST:VITEBESK". See table 1022/217 which is an instructive entry

 

 

3B

SITE NAME

SITE

ENTER "SITE:" FOLLOWED BY THE SITE NAME, FOR EXAMPLE: "SITE:RAMBO-2". See table 1022/68 which is an instructive entry

 

 

3C

FIELD LOCATION NAME OF ENEMY

FLOC

ENTER "FLOC:" FOLLOWED BY THE FIELD LOCATION NAME, FOR EXAMPLE: "FLOC:REGENSBURG". See table 1022/218 which is an instructive entry

 

 

3D

AREA NAME

AREA

ENTER "AREA:" FOLLOWED BY THE AREA NAME, FOR EXAMPLE: "AREA:SOLTAU". See table 1022/69 which is an instructive entry

 

 

3E

REFERENCE POINT NAME

REF

ENTER "REF:" FOLLOWED BY THE NAME OF REFERENCE POINT,  FOR EXAMPLE: "REF:HEIDELBERG". See table 1022/2 which is an instructive entry

4

BASIC ENCYCLOPEDIA NUMBER

 

 

 

 

4A

BASIC ENCYCLOPEDIA NUMBER

 

ENTER THE BASIC ENCYCLOPEDIA NUMBER WHICH CONSISTS OF THE WORLD AREA NUMBER [4N],  THE PROGRAMME INDICATOR [1AS], AND THE INSTALLATION IDENTIFICATION SERIAL NUMBER [5AN], FOR EXAMPLE: "0123-45678". See table 2002 which is a composite

5

BASIC ENCYCLOPEDIA NUMBER SUFFIX

 

CAN ONLY BE USED IN CONJUNCTION WITH BASIC ENCYCLOPEDIA NUMBER.

 

 

5A

BASIC ENCYCLOPEDIA NUMBER SUFFIX

 

ENTER THE BASIC ENCYCLOPEDIA NUMBER SUFFIX, FOR EXAMPLE: "02". See table 1012/102 a range [01 through 99]

6

RECONNAISSANCE TARGET TYPE

 

 

 

 

6A

RECONNAISSANCE TARGET TYPE

 

ENTER A CODE GIVING THE RECONNAISSANCE  TARGET CATEGORY WHICH IS THE NUMERICAL REPRESENTATION OF THE FUNCTION OR ONE OF THE FUNCTIONS OF THE INSTALLATION, FOR EXAMPLE: "10". See table 1309/5 which contains a list of data items and associated data codes

7

GEOGRAPHICAL ENTITY

 

 

 

 

7A

GEOGRAPHICAL ENTITY

 

ENTER A CODE THAT IDENTIFIES THE COUNTRY IN WHICH THE TARGET IS LOCATED, FOR EXAMPLE: "GE". See table 1026/1 which contains a list of data items and associated data codes

8

INSTALLATION USER CATEGORY

 

 

 

 

8A

INSTALLATION USER CATEGORY

 

ENTER THE CATEGORY OF USER OPERATING THE INSTALLATION, FOR EXAMPLE: "MIL". See table 1644/1 which contains a list of data items and associated data codes

9

ORDER OF BATTLE INDICATOR

 

 

 

 

9A

ORDER OF BATTLE INDICATOR

 

ENTER THE ORDER OF BATTLE TYPE, FOR EXAMPLE: "A". See table 1670/5 which contains a list of data items and associated data codes

10

ORDER OF BATTLE ITEM

 

 

 

 

10A

ORDER OF BATTLE ITEM

 

ENTER THE ORDER OF BATTLE ITEM WHICH IDENTIFIES THE TARGET, FOR EXAMPLE: "PMA". See table 1650/5 which contains a list of data items and associated data codes

11

EVALUATION OF ACCURACY

 

 

 

 

11A

EVALUATION OF ACCURACY

 

ENTER THE KNOWN EVALUATION OF ACCURACY OF THE ORDER OF BATTLE ITEM IN THE PREVIOUS FIELD, FOR EXAMPLE: "2". See table 1251/1 which contains a list of data items and associated data codes
