Concept/Explanation/Examples

1

AIRCRAFT LOCATION

 

THE GEOGRAPHICAL POSITION OF THE REPORTING AIRCRAFT.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

2

AIRCRAFT ALTITUDE

 

PROVIDES THE ALTITUDE OF THE REPORTING AIRCRAFT.

 

 

2A

REPORTING AIRCRAFT ALTITUDE IN HUNDREDS OF FEET

 

ENTER THE ALTITUDE OF THE REPORTING AIRCRAFT IN HUNDREDS OF FEET, FOR EXAMPLE: "185". See table 1255/18 a range [000 through 999]

3

AIRCRAFT HEADING

 

PROVIDES THE REPORTING AIRSCRAFT HEADING.

 

 

3A

REPORTING AIRCRAFT HEADING IN DEGREES

 

ENTER THE HEADING OF THE REPORTING AIRCRAFT IN DEGRRES, WITH THE INDICATION IF TRUE, MAGNETIC, GRID OR RELATIVE, FOR EXAMPLE: "120T". See table 2065 which is a composite

4

TARGET POSITION

 

THE GEOGRAPHICAL POSITION OF THE MISSION TARGET.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

5

TARGET HEADING

 

PROVIDES THE APPROXIMATE DIRECTION OF MOVEMENT OF THE MOBILE TARGET.

 

 

5A

MOBILE TARGET HEADING IN DEGREES

 

ENTER THE HEADING OF THE MOBILE TARGET IN DEGREES, WITH THE INDICATION IF TRUE, MAGNETIC, GRID OR RELATIVE, FOR EXAMPLE: "300T". See table 2065 which is a composite

6

JAM ON DTG

 

PROVIDES THE DATE-TIME GROUP JAMMING OPERATIONS BEGAN.

 

 

6A

JAM ON DTG

ON

ENTER "ON:" FOLLOWED BY THE DATE-TIME GROUP JAMMING OPERATIONS BEGAN, FOR EXAMPLE: "ON:122155ZSEP2001". See table 2033 which is a composite

7

JAM OFF DTG

 

PROVIDES THE DATE-TIME GROUP JAMMING OPERATIONS ENDED.

 

 

7A

JAM OFF DTG

OFF

ENTER "OFF:", FOLLOWED BY THE DATE-TIME GROUP JAMMING OPERATIONS ENDED, FOR EXAMPLE: "OFF:130625ZSEP2001". See table 2033 which is a composite

8

DESCRIPTION OF SIGNALS JAMMED

 

IF USED, PROVIDES A PLAIN LANGUAGE DESCRIPTION OF THE JAMMED SIGNALS.

 

 

8A

DESCRIPTION OF SIGNALS JAMMED

 

ENTER A PLAIN LANGUAGE DESCRIPTION OF THE JAMMED SIGNALS, FOR EXAMPLE: "CW BROADCAST". See table 1363/27 which is an instructive entry

9

FREQENCY BAND

 

IF USED, PROVIDES THE ALPHABETIC IDENTIFIER FOR THE FREQUENCY BAND OF THE JAMMED SIGNALS.

 

 

9A

ALPHABETIC IDENTIFIER, FREQENCY BAND

 

ENTER THE ALPHABETIC IDENTIFIER FOR THE FREQUENCY BAND OF THE JAMMED SIGNALS, FOR EXAMPLE: "F". See table 1264/2 which is an alphanumeric range

10

FREQUENCY JAMMED DESIGNATOR

 

IF USED, PROVIDES THE DESIGNATOR FOR THE FREQUENCY BAND OF THE JAMMED SIGNALS.

 

 

10A

FREQUENCY JAMMED DESIGNATOR

 

ENTER THE FREQUENCY DESIGNATOR OF THE JAMMED SIGNAL, FOR EXAMPLE: "TA121". See table 1396/5 which is an instructive entry

11

JAMMING TECHNIQUE

 

IF USED, PROVIDES THE SPECIFIC TECHNIQUE OF JAMMING USED AGAINST THE JAMMED SIGNALS.

 

 

11A

JAMMING TECHNIQUE

 

ENTER THE SPECIFIC TECHNIQUE OF JAMMING USED AGAINST THE JAMMED SIGNALS, FOR EXAMPLE: "INTERFER". See table 1643/5 which contains a list of data items and associated data codes

12

ELECTRONIC ATTACK CONFIDENCE LEVEL

 

IF USED, PROVIDES THE LEVEL OF CONFIDENCE OF THE INFORMATION CONCERNING THE ELECTRONIC ATTACK.

 

 

12A

ELECTRONIC ATTACK CONFIDENCE LEVEL

 

ENTER THE LEVEL OF CONFIDENCE OF THE INFORMATION CONCERNING THE ELECTRONIC ATTACK., FOR EXAMPLE: "2". See table 1123/11 which contains a list of data items and associated data codes
