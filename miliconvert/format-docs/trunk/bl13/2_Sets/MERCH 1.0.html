Concept/Explanation/Examples

1

CONTACT SERIAL NUMBER

 

 

 

 

1A

CONTACT SERIAL NUMBER

 

ENTER THE SERIAL NUMBER OF THE CONTACT, FOR EXAMPLE: "3". See table 1012/67 a range [1 through 999]

2

VESSEL NAME

 

 

 

 

2A

VESSEL NAME

 

ENTER THE NAME OF THE VESSEL, FOR EXAMPLE: "DOBRYNA NIKITICH". See table 1022/264 which is an instructive entry

3

MERCHANT VESSEL TYPE

 

 

 

 

3A

MERCHANT VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE MERCHANT VESSEL TYPE, FOR EXAMPLE: "TMB". See table 1166/10 which contains a list of data items and associated data codes

4

FLAG

 

 

 

 

4A

FLAG

 

ENTER THE COUNTRY CODE OF THE FLAG UNDER WHICH THE SHIP SAILS, FOR EXAMPLE: "UR". See table 1026/3 which contains a list of data items and associated data codes

5

CONTACT PHOTO STATUS

 

 

 

 

5A

CONTACT PHOTO STATUS

 

ENTER THE CODE INDICATING WHETHER OR NOT PHOTOS WERE TAKEN OF THE CONTACT, FOR EXAMPLE: "NPH". See table 1450/1 which contains a list of data items and associated data codes

6

SHIP CONTROL NUMBER

 

 

 

 

6A

SHIP CONTROL NUMBER

 

ENTER THE SHIP CONTROL NUMBER, FOR EXAMPLE: "M12345". See table 2282 which is a composite

7

NATO DESIGNATOR OR DISCRETE IDENTIFIER

 

SPECIFIES THE NATO DESIGNATOR OR IDENTIFIER ASSIGNED TO THE CONTACT.

 

 

7A

NATO SURFACE CONTACT DESIGNATOR

DESIG

ENTER FIELD DESCRIPTOR "DESIG:" FOLLOWED BY THE NATO SURFACE CONTACT DESIGNATOR, FOR EXAMPLE: "DESIG:34ZA". See table 2154 which is a composite

 

 

7B

DISCRETE IDENTIFIER

 

ENTER THE DISCRETE IDENTIFIER (A MARITIME REPORTING CODE FOR TRACKS THAT ARE NOT IDENTIFIED AS "FRIEND", "ASSUMED FRIEND" OR "UNKNOWN") FOR THE CONTACT, FOR EXAMPLE: "1134". See table 1625/25 which is an alphanumeric range

8

SUSPICION CODE

 

 

 

 

8A

SUSPICION CODE

 

ENTER THE NUMERIC CODE THAT INDICATES THE LEVEL OF SUSPICION CONCERNING THE CONTACT, FOR EXAMPLE: "02". See table 1698/1 which contains a list of data items and associated data codes

9

CARGO INDICATOR

 

SPECIFIES THE TYPE OF CARGO BEING CARRIED BY THE MERCHANT VESSEL.

 

 

9A

TYPE OF DRUG CARGO

DRUG

ENTER FIELD DESCRIPTOR "DRUG:" FOLLOWED BY THE SPECIFIC DRUG CARGO BEING CARRIED, FOR EXAMPLE: "DRUG:AMP". See table 1173/12 which contains a list of data items and associated data codes

 

 

9B

CARGO INDICATOR

 

ENTER THE TYPE OF CARGO BEING CARRIED BY THE VESSEL, FOR EXAMPLE: "COL". See table 1173/4 which contains a list of data items and associated data codes
