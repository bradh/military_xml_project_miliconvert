Designator

Field Desc

Concept/Explanation/Examples

1

MISSION NUMBER

 

PROVIDES THE NUMBER/CODE IDENTIFYING THE MISSION WHOSE SORTIE(S) WERE NOT EXECUTED.

 

 

1A

MISSION NUMBER

 

ENTER THE NUMBER/CODE IDENTIFYING THE MISSION, FOR EXAMPLE "C3550". See table 1012/69 which is an instructive entry

2

AIRCRAFT CALL SIGN

 

PROVIDES THE CALL SIGN OF THE  AIR MISSION WHOSE SORTIE(S) WERE NOT EXECUTED.

 

 

2A

CONTACT CALL SIGN

 

ENTER THE CALL SIGN OF THE REPORTED AIR MISSION, FOR EXAMPLE: "RICKY ONE". See table 1129/31 which is an instructive entry

3

REASON SORTIE(S) ABORTED

 

PROVIDES THE REASON(S) WHY THE SORTIES ORIGINALLY FORESEEN IN THE MISSION WERE NOT EXECUTED.

 

 

3A

REASON SORTIE(S) ABORTED

 

ENTER THE REASON(S) WHY THE SORTIES ORIGINALLY FORESEEN IN THE MISSION WERE NOT EXECUTED, FOR EXAMPLE "WX". See table 1212/8 which contains a list of data items and associated data codes
