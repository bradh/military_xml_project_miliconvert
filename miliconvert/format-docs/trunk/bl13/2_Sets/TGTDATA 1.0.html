Concept/Explanation/Examples

1

PRIMARY/ALTERNATE TARGET DESIGNATOR

 

 

 

 

1A

PRIMARY ALTERNATE TARGET DESIGNATOR

 

ENTER A DESIGNATOR THAT CLASSIFIES THE TARGET AS PRIMARY  OR ALTERNATE, FOR EXAMPLE: "P". See table 1076/13 which contains a list of data items and associated data codes

2

TARGET/FACILITY NAME.

 

THE NAME OF EITHER THE TARGET FACILITY OR THE TARGET SHIP.

 

 

2A

TARGET FACILITY NAME

TGT

ENTER "TGT:" FOLLOWED BY THE NAME OF THE TARGET FACILITY, FOR EXAMPLE "TGT:CASE BRIDGE". See table 1022/234 which is an instructive entry

 

 

2B

SHIP NAME

SHIP

ENTER "SHIP:" FOLLOWED BY THE NAME OF THE TARGET SHIP, FOR EXAMPLE "SHIP:SHENZEN". See table 1022/49 which is an instructive entry

3

HULL NUMBER

 

THE HULL NUMBER ASSIGNED TO A VESSEL TYPE TARGET.

 

 

3A

HULL NUMBER

 

ENTER THE ONE TO FIVE CHARACTERS HULL/PENNANT NUMBER OF A VESSEL TYPE TARGET, FOR EXAMPLE: "D-167". See table 1012/30 which is an instructive entry

 

 

3B

SPECIAL HULL NUMBER

 

ENTER THE ONE TO FOUR DIGIT SPECIAL HULL/PENNANT NUMBER OF A VESSEL TYPE TARGET, FOR EXAMPLE: "167". See table 1012/245 a range [1 through 9999]

4

TARGET POSITION

 

THE POSITION WHERE THE TARGET IS SITUATED, USING EITHER ITS IDENTIFICATION CODE, ITS NAME OR ITS GEOGRAPHICAL COORDINATES.

 

 

4A

TARGET POSITION IDENTIFIER

POSNID

ENTER "POSNID:" FOLLOWED BY THE TARGET POSITION IDENTIFIER, FOR EXAMPLE: "POSNID:AR680". See table 1416/2 which is an instructive entry

 

 

4B

TARGET POSITION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE TARGET POSITION NAME, FOR EXAMPLE: "NAME:VARSANE". See table 1022/81 which is an instructive entry

 

 

4C

GEOGRAPHIC LOCATION, LAT/LONG, CENTISECONDS

LATCS

ENTER "LATCS:" FOLLOWED BY THE GEOGRAPHIC COORDINATES IN LAT/LONG, CENTISECONDS, OF THE POSITION WHERE THE TARGET IS SITUATED, FOR EXAMPLE: "LATCS:12201550N141323025E". See table 2488 which is a composite

 

 

4D

GEOGRAPHIC POSITION, GEOREF, MINUTE

GEOREFM

ENTER "GEOREFM:" FOLLOWED BY THE GEOGRAPHIC COORDINATES OF THE TARGET IN GEOREF APPROXIMATE TO THE MINUTE, FOR EXAMPLE: "GEOREFM:ASGB1029". See table 2017 which is a composite

 

 

4E

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4F

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

5

TARGET TYPE

 

THE TARGET TYPE, EITHER NON-SHIP OR VESSEL TYPE.

 

 

5A

NON-SHIP TARGET TYPE

TYPE

ENTER "TYPE:" FOLLOWED BY THE CODE FOR THE TYPE OF NON-SHIP TARGET, FOR EXAMPLE "TYPE:BRIDGE". See table 1650/6 which contains a list of data items and associated data codes

 

 

5B

VESSEL TYPE

SHIPTYPE

ENTER "SHIPTYPE:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE TYPE OF SHIP OR VESSEL TARGET, FOR EXAMPLE "SHIPTYPE:CG". See table 1166/1 which contains a list of data items and associated data codes

6

GEODETIC DATUM

 

THE GEODETIC DATUM FOR THE GROUND TARGET LOCATION.

 

 

6A

GEODETIC DATUM

 

ENTER THE GEODETIC DATUM USED FOR IDENTIFYING THE TARGET LOCATION, FOR EXAMPLE "WGE". See table 1195/1 which contains a list of data items and associated data codes

7

DESIRED MEAN POINT OF IMPACT IDENTIFIER

 

 

 

 

7A

TARGET DESIRED MEAN POINT OF IMPACT (DMPI) IDENTIFIER

 

ENTER THE IDENTIFIER OF THE DESIRED MEAN POINT OF IMPACT (DMPI) OF WEAPONS ON TARGET, FOR EXAMPLE: "QRS 1236-RTY". See table 1416/7 which is an instructive entry

8

DESIRED MEAN POINT OF IMPACT DESCRIPTION

 

 

 

 

8A

DESIRED MEAN POINT OF IMPACT DESCRIPTION

 

ENTER THE PLAIN LANGUAGE DESCRIPTION OF THE DESIRED MEAN POINT OF IMPACT (DMPI), FOR EXAMPLE "CENTER SPAN". See table 1363/44 which is an instructive entry

9

DESIRED MEAN POINT OF IMPACT LOCATION

 

THE COORDINATES OF THE DESIRED MEAN POINT OF IMPACT (DMPI) FOR THE WEAPONS BEING USED ON THE TARGET.

 

 

9A

GEOGRAPHIC LOCATION, LAT/LONG, CENTISECONDS

DMPIC

ENTER "DMPIC:" FOLLOWED BY THE LOCATION OF THE DESIRED MEAN POINT OF IMPACT IN LAT/LONG CENTISECONDS, FOR EXAMPLE: "DMPIC:35103045N079013045W". See table 2488 which is a composite

 

 

9B

GEOGRAPHIC POSITION, GEOREF, MINUTE

GEOREFM

ENTER"GEOREFM:" FOLLOWED BY THE  DESIRED MEAN IMPACT LOCATION IN GEOREF APPROXIMATE TO THE MINUTE, FOR EXAMPLE: "GEOREFM:ASGB1029". See table 2017 which is a composite

 

 

9C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

10

PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR

 

CAN ONLY BE USED IF THE DESIRED MEAN POINT OF IMPACT LOCATION IS SPECIFIED IN THIS SET.

 

 

10A

PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR

 

ENTER THE PROBABLE COORDINATE HORIZONTAL CIRCULAR ERROR, FOR EXAMPLE: "10.55FT". See table 2532 which is a composite

11

DMPI ELEVATION

 

CAN ONLY BE USED IF THE DESIRED MEAN POINT OF IMPACT LOCATION IS SPECIFIED IN THIS SET.

 

 

11A

DESIRED MEAN POINT OF IMPACT ELEVATION IN FEET OR METRES

 

ENTER THE DESIRED MEAN POINT OF IMPACT ELEVATION IN FEET OR METRES MEASURED ABOVE OR BELOW THE MEAN SEA LEVEL OR ABOVE THE REFERENCE ELLIPSOID, FOR EXAMPLE: "-200FTMSL". See table 2492 which is a composite

12

PROBABLE COORDINATE ELEVATION LINEAR ERROR

 

CAN ONLY BE USED IF THE DESIRED MEAN POINT OF IMPACT LOCATION IS SPECIFIED IN THIS SET.

 

 

12A

PROBABLE COORDINATE ELEVATION LINEAR ERROR

 

ENTER THE PROBABLE COORDINATE ELEVATION LINEAR ERROR, FOR EXAMPLE: "20.55FT". See table 2532 which is a composite

13

COMPONENT TARGET IDENTIFIER

 

THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE REQUESTING SERVICE.

 

 

13A

COMPONENT TARGET IDENTIFIER

CTGT

ENTER "CTGT:" FOLLOWED BY THE IDENTIFIER OR NUMBER ASSIGNED TO THE TARGET BY THE REQUESTING SERVICE, FOR EXAMPLE "CTGT:N194167". See table 1416/5 which is an instructive entry

 

 

13B

TARGET ENTITY ID REFERENCE NUMBER

EIRN

ENTER "EIRN:" FOLLOWED BY THE TARGET IDENTITY ID REFERENCE NUMBER, FOR EXAMPLE: "EIRN:150000114294960011". See table 2085 which is a composite

14

TARGET PRIORITY, LITERAL

 

 

 

 

14A

TARGET PRIORITY, LITERAL

 

ENTER THE CODE FOR THE PRIORITY ASSIGNED TO THE TARGET, FOR EXAMPLE "1A". See table 1076/11 which is an instructive entry

15

TARGET OBJECTIVE

 

 

 

 

15A

TARGET OBJECTIVE

 

ENTER THE TARGET OBJECTIVE, I.E. A PLAIN DESCRIPTION OF THE GOAL TO BE ACHIEVED BY THE MISSION HITTING THE TARGET, FOR EXAMPLE:"SINK". See table 1456/3 which is an instructive entry

16

LINK 16 TRACK NUMBER REFERENCE

 

 

 

 

16A

LINK 16 TRACK NUMBER REFERENCE FOR PRE-PLANNED TARGET

 

ENTER THE UNIQUE LINK16 TARGET IDENTIFIER, FOR EXAMPLE: "AA234". See table 1625/27 which is an alphanumeric range

17

TERMINAL AREA MODEL IDENTIFICATION

 

 

 

 

17A

TERMINAL AREA MODEL IDENTIFICATION

TAMID

ENTER "TAMID:" FOLLOWED BY THE IDENTIFICATION OF THE TERMINAL AREA MODEL FOR THE JOINT AIR TO SURFACE STANDOFF  MISSILE, FOR EXAMPLE:"TAMID:34678912452AM9". See table 1012/317 which is an instructive entry

18

AIMPOINT SUFFIX

 

 

 

 

18A

AIMPOINT SUFFIX

 

ENTER A TWO CHARACTER SUFFIX FOR THE TARGET IDENDIFIER, REPEATING AS NECESSARY, FOR EXAMPLE: "EB". See table 1012/244 which is an instructive entry

19

ADDITIONAL TARGET IDENTIFICATION FEATURES

 

 

 

 

19A

ADDITIONAL TARGET IDENTIFICATION FEATURES

 

ENTER ADDITIONAL IDENTIFYING INFORMATION ABOUT THE GROUND TARGET, FOR EXAMPLE:"THIRD ARCH". See table 1357/23 which is an instructive entry

20

MISSION TYPE

 

 

 

 

20A

TASK GROUPING CATEGORY

 

ENTER THE RECOMMENDED MISSION TYPE FOR TASKING, FOR EXAMPLE: "OCA". See table 1810/1 which contains a list of data items and associated data codes
