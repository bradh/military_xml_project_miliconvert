1

PRISONER OF WAR INTERNMENT NUMBER

 

 

 

 

1A

PRISONER OF WAR INTERNMENT NUMBER

 

THE DATA IN THIS COMPOSITE FIELD (CONCATENATED FIELD) IS THE UNIQUE IDENTIFICATION FOR EACH SINGLE PRISONER OF WAR, FOR EXAMPLE: "US0001AAA999". IT CONSISTS OF TWO SINGLE DATA ELEMENTS: THE COUNTRY CODE OF THE CAPTURING COUNTRY FOLLOWED BY A 10 DIGIT IDENTIFICATION NUMBER [LITERAL] WHICH IS ASSIGED TO THE PRISONER OF WAR BY THE HOLDING ORGANIZATION. See table 2448 which is a composite

2

PW INTERROGATION SERIAL NUMBER

 

THIS FIELD MUST OCCUR, IF FIELD 1 DOES NOT OCCUR.

 

 

2A

PW INTERROGATION SERIAL NUMBER

 

THIS NUMBER ALLOCATED TO THE PRISONER OF WAR BY THE INFORMATION UNIT FIRST DEALING WITH HIM IS TO IDENTIFY THE THE SOURCE OF INFORMATION, FOR EXAMPLE: "RS-NV-B-0012-2105-USNV-0159". THE NUMBER CONSISTS OF THE FOLLOWING ELEMENTS SEPERATED FROM EACH OTHER BY HYPHEN: NATIONALITY OF THE PW (2A), SERVICE OF THE PW (2A), ARM OF SERVICE OF THE PW (1A), INTERROGATION SEQUENCE NUMBER OF THE PW (4N), DAY AND MONTH OF CAPTURE (4N), NATIONALITY AND SERVICE OF THE INTERROGATION UNIT (4A), INTERROGATION UNIT INDICATOR (4N). See table 1012/231 which is an instructive entry

3

LAST NAME

 

 

 

 

3A

LAST NAME

 

SURNAME, OR GIVEN NAME CONFERRED UPON AN INDIVIDUAL AT BIRTH (FOR A WOMAN THE MAIDEN NAME), THROUGH MARRIAGE, OR THROUGH A LEGAL PROCESS. IT CAN ALSO BE THE FAMILY NAME, FOR EXAMPLE: "LEFEVER". See table 1022/255 which is an instructive entry

4

FIRST NAME(S)

 

 

 

 

4A

FIRST NAME(S)

 

CHOSEN NAME(S) CONFERRED UPON AN INDIVIDUAL OTHER THAT HIS LAST NAME(S), BY WHICH HE IS FAMILIARLY KNOWN, FOR EXAMPLE: "PAUL". See table 1022/256 which is an instructive entry

5

MIDDLE NAME(S)/INITIAL(S)

 

 

 

 

5A

MIDDLE NAME(S) INITIAL(S)

 

CHOSEN NAME(S) CONFERRED UPON AN INDIVIDUAL, APART FROM HIS FIRST AND LAST NAMES. THESE NAMES MAY BE EXPRESSED THROUGH THE USE OF THE INITIAL LETTERS, FOR EXAMPLE: "P S H". See table 1022/257 which is an instructive entry

6

BIRTH NAME

 

 

 

 

6A

BIRTH NAME

 

INDICATES THE FAMILY NAME CONFERRED UPON AN INDIVIDUAL AT BIRTH (E. G. MAIDEN NAME), FOR EXAMPLE: "(BIRTH NAME)". See table 1022/258 which is an instructive entry

7

PERSONAL TITLE

 

 

 

 

7A

PERSONAL TITLE

 

DESCRIPTIVE OR DISTINCTIVE METHOD OF ADDRESS GIVEN TO AN INDIVIDUAL BY RIGHT OF BIRTH, RANK, OFFICE ASSIGNMENT, OR OTHER ATTAINMENT, FOR EXAMPLE: "COLONEL". See table 1022/259 which is an instructive entry

8

GENDER, ABBREVIATED

 

 

 

 

8A

GENDER, ABBREVIATED

 

FIELD INDICATES THE SEX OF AN INDIVIDUAL AT THE TIME OF THE RECORD WITH THE ALLOWABLE VALUES OF "M" FOR MALE AND "F" FOR FEMALE. See table 1286/13 which contains a list of data items and associated data codes

9

COUNTRY

 

 

 

 

9A

COUNTRY

 

SPECIFIES THE COUNTRY (STANAG 1059) OF WHICH THE INDIVIDUAL IS A CITIZEN, FOR EXAMPLE: "BS". See table 1026/18 which contains a list of data items and associated data codes

10

PERSONAL IDENTIFICATION NUMBER

 

 

 

 

10A

PERSONAL IDENTIFICATION NUMBER

 

NATIONAL SERVICE IDENTIFICATION NUMBER OF THE INDIVIDUAL. See table 1012/230 which is an instructive entry

11

DATE OF BIRTH OR AGE

 

IF USED, ONE OF THE FOLLOWING ALTERNATIVES IS TO BE ENTERED:

 

 

11A

DATE OF BIRTH, YYYYMMDD

 

ENTER THE DATE OF BIRTH IN THE FORM YYYYMMDD, FOR EXAMPLE: "19401002". See table 2053 which is a composite

 

 

11B

AGE

 

IF DATE OF BIRTH CAN'T BE SPECIFIED THE AGE IS TO BE ESTIMATED See table 1044/357 a range [0 through 150]

12

PLACE NAME

 

 

 

 

12A

PLACE NAME, EXTENDED

 

NAME OF THE CITY/VILLAGE OR ADMINISTRATIVE DISTRICT WHERE THE INDIVIDUAL WAS BORN, FOR EXAMPLE" "UPAVON". See table 1022/261 which is an instructive entry

13

COUNTRY OF BIRTH

 

 

 

 

13A

COUNTRY OF BIRTH

 

IDENTIFIES THE COUNTRY (STANAG 1059) WHERE THE INDIVIDUAL WAS BORN, FOR EXAMPLE: "UK". See table 1026/13 which contains a list of data items and associated data codes

14

COMMENTS

 

 

 

 

14A

COMMENTS

 

THIS FIELD ALLOWS THE RECORDING OF ANY OTHER DATA [LITERAL] WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS IN THIS SET, FOR EXAMPLE: "(REMARKS IN FREE TEXT UP TO 68 CHARACTERS)". See table 1357/1 which is an instructive entry
