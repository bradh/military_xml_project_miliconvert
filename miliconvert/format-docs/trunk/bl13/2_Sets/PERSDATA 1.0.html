Concept/Explanation/Examples

1

CASUALTY NAME

 

 

 

 

1A

CASUALTY NAME

 

ENTER THE NAME OF THE CASUALTY, FOR EXAMPLE "FITZGERALD". See table 1022/311 which is an instructive entry

2

RANK OR POSITION

 

 

 

 

2A

RANK OR POSITION

 

ENTER THE RANK OR POSITION OF THE CASUALTY, FOR EXAMPLE "SGT". See table 1046/2 which is an instructive entry

3

CASUALTY NUMBER

 

ENTER ONE OF THE FOLLOWING: FIELD DESCRIPTOR "OCN:" FOLLOWED BY THE NUMBER ASSIGNED BY THE MEDICAL TEAM TO THE CASUALTY OR FIELD DESCRIPTOR " SERVNO:" FOLLOWED BY THE CASUALTY'S NATIONAL ASSIGNED SERVICE NUMBER.

 

 

3A

OFFICIAL CASUALTY NUMBER

OCN

ENTER THE NUMBER ASSIGNED BY THE MEDICAL REPORTING TEAM TO A PARTICULAR CASUALTY. THIS NUMBER STAYS WITH THE CASUALTY UNTIL HE IS POSITIVELY IDENTIFIED AND/OR HE CAN REPORT HIS NATIONAL SERVICE NUMBER. See table 1012/322 which is an instructive entry

 

 

3B

SERVICE NUMBER

SERVNO

ENTER THE SERVICE NUMBER OF THE CASUALTY, FOR EXAMPLE "SERVNO:26335078". See table 1012/325 which is an instructive entry

4

PERSONNEL GENDER

 

 

 

 

4A

PERSONNEL GENDER

 

ENTER THE APPROPRIATE GENDER CODE OF THE CASUALTY, FOR EXAMPLE "M". See table 1286/2 which contains a list of data items and associated data codes

5

NATIONALITY

 

 

 

 

5A

NATIONALITY

 

ENTER THE NATIONALITY OF THE CASUALTY, FOR EXAMPLE "UK". See table 1026/2 which contains a list of data items and associated data codes

NAME OF ORIGINATING UNIT OF CASUALTY

 

 

 

 

6A

NAME OF ORIGINATING UNIT OF CASUALTY

PERSONNEL CASUALTY STATUS

 

 

 

 

7A

PERSONNEL CASUALTY STATUS

 

ENTER THE APPROPRIATE CODE FOR DESCRIBING THE MEDICAL STATUS OF THE CASUALTY, FOR EXAMPLE "WIA". See table 1282/6 which contains a list of data items and associated data codes

8

CASUALTY CONDITION

 

 

 

 

8A

CASUALTY CONDITION

 

ENTER THE APPROPRIATE CODE FOR THE MEDICAL CONDITION OF THE CASUALTY, FOR EXAMPLE "VSI". See table 1690/9 which contains a list of data items and associated data codes
