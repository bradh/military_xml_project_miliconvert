Concept/Explanation/Examples

1

PLATFORM CARRYING EQUIPMENT.

 

THE UNIT CARRYING THE JAMMING EQUIPMENT ESPRESSED AS ONE OF THE FOLLOWING:

 

 

1A

UNIT

 

ENTER THE UNIT NAME, FOR EXAMPLE: "RANDOLPH". See table 1022/57 which is an instructive entry

 

 

1B

AIRCRAFT CALL SIGN

ACFT

ENTER "ACFT:" FOLLOWED BY THE AIRCRAFT CALLSIGN, FOR EXAMPLE: "ACFT:HAWKEYE". See table 1129/9 which is an instructive entry

2

JAMMING CONTROL AUTHORITY FOR UNIT

 

 

 

 

2A

JAMMING CONTROL AUTHORITY

 

ENTER THE POSITIVE CONTROL AUTHORITY FOR THE UNIT, FOR EXAMPLE:"OTC". See table 1169/1 which contains a list of data items and associated data codes

3

THE JAMMER TYPE TO BE USED BY THE TASKED UNIT

 

 

 

 

3A

JAMMER EQUIPMENT TYPE

 

ENTER THE JAMMER EQUIPMENT TYPE TO BE USED BY THE TASKED UNIT, SHIP OR AIRCRAFT IDENTIFIED IN FIELD 1, FOR EXAMPLE: "670". See table 1135/12 which is an instructive entry

4

FREQUENCY LIMITS OR SPOT NUMBER

 

THE FREQUENCY LIMITS OR SPOT NUMBER:

 

 

4A

FREQUENCY LIMITS HYPHENATED

RANGE

ENTER "RANGE:" FOLLOWED BY THE FREQUENCY RANGE THE MISSION IS TO BE CONDUCTED IN (SEPARATED BY A HYPHEN) AND FOLLOWED BY A UNIT OF MEASUREMENT, FOR EXAMPLE: "RANGE:1-3GHZ". See table 2335 which is a composite

 

 

4B

SPOT NUMBER

SPOT

ENTER "SPOT:" FOLLOWED BY THE 5 DIGIT SPOT NUMBER ASSIGNED TO THE THREAT/TARGET EMITTER, FOR EXAMPLE: "SPOT:06217". See table 1012/125 a range [00000 through 99999]

5

JAMMER ALERT STATE

 

THE JAMMER ALERT STATE:

 

 

5A

JAMMER ALERT STATE

ALERT

ENTER "ALERT:" FOLLOWED BY THE ALERT STATUS AS A PERIOD OF TIME (0-9999, 0 TO 3 DECIMAL PLACES PERMITTED AND A UNIT OF MEASUREMENT), FOR EXAMPLE: "ALERT:30MIN". See table 2161 which is a composite

 

 

5B

NON-DISCRETE INDICATOR

 

ENTER AN IDENTIFIER TO SPECIFY AN ALERT STATE OTHER THAN BY A TIME, FOR EXAMPLE: "READY". See table 1084/1 which contains a list of data items and associated data codes
