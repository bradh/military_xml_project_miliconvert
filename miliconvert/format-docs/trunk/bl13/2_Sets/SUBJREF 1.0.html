No

Designator

Field Desc

Concept/Explanation/Examples

1

TITLE

 

 

 

 

1A

TITLE

 

DESCRIPTION (TITLE) OF THE SUBJECT WHICH IS REPORTED IN THE MESSAGE, FOR EXAMPLE: "(FREE TEXT, 1-54X)". See table 1356/1 which is an instructive entry

2

COUNTRY, STATE, PROVINCE OR DISTRICT

 

THE COUNTRY, STATE, PROVINCE OR DISTRICT TO WHICH THE SUBJECT REFERS.

 

 

2A

GEOGRAPHICAL ENTITY

 

ENTER THE GEOGRAPHIC ENTITY (COUNTRY) TO WHICH THE SUBJECT REFERS, FOR EXAMPLE: "UK". See table 1026/1 which contains a list of data items and associated data codes

 

 

2B

STATE PROVINCE DISTRICT

 

ENTER THE REGIONAL ENTITY (STATE/PROVINCE/DISTRICT) TO WHICH THE SUBJECT REFERS, FOR EXAMPLE: "IT05". See table 1711/2 which contains a list of data items and associated data codes

3

PLACE NAME

 

 

 

 

3A

PLACE NAME, EXTENDED

 

NAME OF THE PLACE, INSTALLATION OR FACILITY TO WHICH THE SUBJECT REFERS, FOR EXAMPLE: "SALAPARUTA-A45". See table 1022/261 which is an instructive entry

4

GEOGRAPHIC REFERENCE

 

THE GEOGRAPHIC LOCATION, WHICH IS REFERENCE FOR THE SUBJECT.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

BEARING AND DISTANCE FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

4H

TRUE BEARING AND DISTANCE IN NM FROM REFERENCE POINT

TBRN

ENTER "TBRN:" FOLLOWED BY A LOCATION EXPRESSED AS A TRUE BEARING AND DISTANCE IN NAUTICAL MILES FROM A SPECIFIED REFERENCE POINT, FOR EXAMPLE: "TBRN:205-PORTLAND BILL LT-34.5". See table 2016 which is a composite

5

GEODETIC DATUM

 

 

 

 

5A

GEODETIC DATUM

 

IF REQUIRED, GEODETIC DATUM, FOR EXAMPLE: "WGE". See table 1195/1 which contains a list of data items and associated data codes

6

STARTING DATE

 

THE STARTING DATE TO ENABLE THE PROCESSING OF THE MESSAGE.

 

 

6A

START DATE, YYYYMMDD

 

START DATE [YYYYMMDD] OF A TIME PERIOD,  TO WHICH THE SUBJECT IN THE REPORT IS RELATED, FOR EXAMPLE: "19980223". See table 2053 which is a composite

 

 

6B

START DATE-TIME, YYYYMMDDHHMM

 

START DATE-TIME [YYYYMMDDHHMM] OF A TIME PERIOD,  TO WHICH THE SUBJECT IN THE REPORT IS RELATED, FOR EXAMPLE: "199802230900". See table 2416 which is a composite

 

 

6C

DTG OF START

 

START DATE-TIME GROUP [DATE, HOUR, MINUTES, TIME ZONE, MONTH NAME, 4-DIGIT YEAR] OF A TIME PERIOD,  TO WHICH THE SUBJECT IN THE REPORT IS RELATED, FOR EXAMPLE: "230900ZFEB1998". See table 2033 which is a composite

7

ENDING DATE

 

THE ENDING DATE TO ENABLE THE PROCESSING OF THE SUBJECT.

 

 

7A

END DATE, YYYYMMDD

 

END DATE [YYYYMMDD] OF A TIME PERIOD,  TO WHICH THE SUBJECT IN THE REPORT IS RELATED, FOR EXAMPLE: "19980223". See table 2053 which is a composite

 

 

7B

END DATE-TIME, YYYYMMDDHHMM

 

END DATE-TIME [YYYYMMDDHHMM] OF A TIME PERIOD,  TO WHICH THE SUBJECT IN THE REPORT IS RELATED, FOR EXAMPLE: "199802271200". See table 2416 which is a composite

 

 

7C

DTG OF STOP

 

END DATE-TIME GROUP [DATE, HOUR, MINUTES, TIME ZONE,  MONTH NAME, 4-DIGIT YEAR] OF A TIME PERIOD,  TO WHICH THE SUBJECT IN THE REPORT IS RELATED, FOR EXAMPLE: "271200ZFEB1998". See table 2033 which is a composite
