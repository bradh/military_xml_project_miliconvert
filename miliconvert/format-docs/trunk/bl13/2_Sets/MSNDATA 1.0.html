1A

ETA DAY-TIME AND MONTH OF FIRST AIRCRAFT AT ONLOAD BASE

 

ENTER THE ESTIMATE TIME THE FIRST AIRCRAFT WILL ARRIVE AT THE ONLOAD BASE.  ENTER THE DATE (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE AND MONTH, FOR EXAMPLE: "041405ZAPR". See table 2030 which is a composite

2

NUMBER AND TYPE OF TRANSPORT AIRCRAFT

 

 

 

 

2A

NUMBER AND TYPE OF TRANSPORT AIRCRAFT

 

ENTER THE NUMBER AND TYPE OF AIRCRAFT PARTICIPATING IN THE AIRLIFT MISSION.  ENTER THE QUANTITY OF AIRCRAFT (1 THROUGH 999) FOLLOWED BY THE TRANSPORT AIRCRAFT TYPE, FOR EXAMPLE: "10C130H". See table 2188 which is a composite

TOTAL NUMBER OF SORTIES TO BE FLOWN

 

ENTER THE TOTAL NUMBER (1 THROUGH 999) OF SORTIES TO BE FLOWN, FOR EXAMPLE: "22". See table 1044/178 a range [1 through 999]
