Concept/Explanation/Examples

1

MARITIME WEAPONS EQUIPMENT

 

THE MARITIME WEAPONS TYPE DESCRIBED AS ONE OF THE FOLLOWING:

 

 

1A

MARITIME WEAPON SYSTEMS

 

ENTER THE MARITIME WEAPON, FOR EXAMPLE:"PHALANX". See table 1407/54 which contains a list of data items and associated data codes

 

 

1B

WEAPON TYPE, GENERIC

GEN

ONLY TO BE USED IF THE WEAPON IS NOT AVAILABLE IN ALT A.  ENTER "GEN:" FOLLOWED BY THE NAME OF A WEAPON, FOR EXAMPLE "GEN:TORPEDO MK XX". See table 1407/1 which is an instructive entry

2

NUMBER/PERCENTAGE OF WEAPONS

 

THE NUMBER OR PERCENTAGE OF WEAPONS/AMMUNITION HELD DESCRIBED AS ONE OF THE FOLLOWING:

 

 

2A

NUMBER OF WEAPONS

 

ENTER THE NUMBER OF WEAPONS AS 1 OR 2 DIGITS, FOR EXAMPLE: "2". See table 1044/67 a range [1 through 99]

 

 

2B

PERCENTAGE OF WEAPONS

PCT

ENTER THE FIELD DESCRIPTOR "PCT:" FOLLOWED BY THE PERCENTAGE OF WEAPONS OR AMMUNITION HELD, FOR EXAMPLE: "PCT:25". See table 1064/18 a range [0 through 100]
