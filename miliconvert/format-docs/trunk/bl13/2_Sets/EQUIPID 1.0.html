Concept/Explanation/Examples

1

ORDER OF BATTLE INDICATOR

 

 

 

 

1A

ORDER OF BATTLE INDICATOR

 

ENTER A CODE IDENTIFYING THE ORDER OF BATTLE TYPE, FOR EXAMPLE: "G". THE CODE "A" INCLUDES NAVAL AIRCRAFT. See table 1670/5 which contains a list of data items and associated data codes

2

EQUIPMENT KEY

 

 

 

 

2A

EQUIPMENT KEY

 

ENTER THE EQUIPMENT KEY AS A COMPOSITE FIELD, FOR EXAMPLE: "Q-NM-UKXY-12345". See table 2442 which is a composite

3

EQUIPMENT NOMENCLATURE

 

 

 

 

3A

EQUIPMENT NOMENCLATURE

 

CONTAINS A MAN READABLE DESCRIPTION OF THE EQUIPMENT, USUALLY A CONCATENATION OF SOME OTHER DESCRIPTIVE FIELDS, FOR EXAMPLE: "(FREE TEXT)". See table 1714/2 which is an instructive entry

4

COUNTRY OF ORIGIN

 

 

 

 

4A

COUNTRY OF ORIGIN

 

THE COUNTRY IN WHICH THE EQUIPMENT WAS ORIGINALLY DESIGNED AND BUILT, FOR EXAMPLE: "US". See table 1026/19 which contains a list of data items and associated data codes

5

EQUIPMENT TYPE, EXPANDED

 

 

 

 

5A

EQUIPMENT TYPE, EXPANDED

 

MAKE A LITERAL ENTRY INDICATING THE TYPE OR TYPE-GROUPING TO WHICH THE SPECIFIC EQUIPMENT BELONGS, FOR EXAMPLE: "COUNTERBATTERY SUPRESSION RADAR". See table 1135/36 which is an instructive entry

6

EQUIPMENT COMMON NAME

 

 

 

 

6A

EQUIPMENT COMMON NAME

 

COMMONLY USED EQUIPMENT NAME OR NOMENCLATURE FOR AN EQUIPMENT, FOR EXAMPLE: "(EQUIPMENT NAME)". See table 1022/249 which is an instructive entry

7

EQUIPMENT CODE

 

 

 

 

7A

EQUIPMENT CODE

 

ENTER A 7 DIGIT EQUIPMENT CODE, FOR EXAMPLE: "123WXYZ". See table 1539/2 which is an instructive entry

8

REMARKS

 

 

 

 

8A

REMARKS

 

ENTER ANY OTHER DATA (LITERAL UP TO 68 CHARACTERS) WHICH IS NOT ALREADY COVERED IN THE SPECIFIC FIELDS. See table 1357/28 which is an instructive entry
