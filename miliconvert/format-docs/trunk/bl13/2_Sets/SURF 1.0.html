Designator

Field Desc

Concept/Explanation/Examples

1

SURFACE CONTACT DESIGNATOR

 

THE SURFACE CONTACT DESIGNATOR.

 

 

1A

NATO SURFACE CONTACT DESIGNATOR

 

ENTER THE NATO SURFACE CONTACT DESIGNATOR, FOR EXAMPLE: "20AE". See table 2154 which is a composite

 

 

1B

NATO EXERCISE SURFACE CONTACT DESIGNATOR

 

ENTER THE NATO EXERCISE SURFACE CONTACT DESIGNATOR, FOR EXAMPLE: "X20AE". See table 2155 which is a composite

 

 

1C

NATO UNDESIGNATED UNIT

 

ENTER THE CODE "UNDES" TO INDICATE A NATO UNDESIGNATED UNIT. See table 1028/25 which contains a list of data items and associated data codes

2

VESSEL TYPE

 

 

 

 

2A

VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE VESSEL TYPE, FOR EXAMPLE: "FF". See table 1166/1 which contains a list of data items and associated data codes

3

FLAG

 

 

 

 

3A

GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE REPRESENTING THE FLAG UNDER WHICH THE VESSEL SAILS, FOR EXAMPLE: "RS". See table 1026/1 which contains a list of data items and associated data codes

4

UNIT IDENTIFICATION

 

THE IDENTITY OF THE UNIT BEING REPORTED. IF NAME, CLASS AND PENNANT NUMBER ARE KNOWN AND NOT PREVIOUSLY REPORTED, ENTER NAME IN THIS FIELD AND IDENTIFY CLASS AND PENNANT NUMBER IN A FOLLOW-ON AMPN SET.

 

 

4A

SHIP NAME

NAME

ENTER FIELD DESCRIPTOR "NAME:" FOLLOWED BY THE VESSEL NAME, FOR EXAMPLE: "NAME:OTCHYANNY". See table 1022/49 which is an instructive entry

 

 

4B

CLASS AND PENNANT NUMBER

CP

IF VESSEL NAME UNKNOWN, ENTER FIELD DESCRIPTOR "CP:" FOLLOWED BY THE VESSEL CLASS AND PENNANT NUMBER, FOR EXAMPLE: "CP:KRIVAK 724". See table 1062/1 which is an instructive entry

 

 

4C

NAME OF SHIP CLASS

CLASS

ENTER FIELD DESCRIPTOR "CLASS:" FOLLOWED BY THE NAME OF THE VESSEL CLASS, FOR EXAMPLE: "CLASS:SOVREMENNYY". See table 1022/35 which is an instructive entry

 

 

4D

UNKNOWN SURFACE CONTACT

 

IF VESSEL IDENTITY UNKNOWN, ENTER DATA CODE "UNKNOWN". See table 1139/4 which contains a list of data items and associated data codes

5

CONTACT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE SURFACE CONTACT.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

5C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

5D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

5E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

5F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

5G

PLACE NAME

NAME

ENTER "PLACE:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "PLACE:SHETLANDS". See table 1022/170 which is an instructive entry

6

TIME OF CONTACT

 

THE TIME WHEN THE CONTACT OCCURRED.

 

 

6A

DAY-TIME OF CONTACT

 

ENTER THE DAY-TIME OF CONTACT, FOR EXAMPLE: "162215Z". See table 2000 which is a composite

 

 

6B

DAY-MONTH NAME

 

IF TIME OF CONTACT IS UNKNOWN, ENTER THE DAY-MONTH NAME, FOR EXAMPLE "16SEP". See table 2067 which is a composite

7

CONTACT MOVEMENT

 

THE COURSE AND SPEED OR MOVEMENT ACTIVITY OF THE CONTACT.

 

 

7A

COURSE IN DEGREES TRUE AND SPEED IN KNOTS

 

ENTER THE CONTACT'S COURSE IN DEGREES TRUE, A HYPHEN AND ITS SPEED IN KNOTS, FOR EXAMPLE: "265-13". See table 2025 which is a composite

 

 

7B

MOVEMENT ACTIVITY

 

ENTER THE MOVEMENT ACTIVITY OF THE CONTACT, FOR EXAMPLE: "DIW". See table 1095/4 which contains a list of data items and associated data codes

8

CONTACT VALIDITY

 

 

 

 

8A

CONTACT VALIDITY

 

ENTER THE ANALYST'S JUDGEMENT AS TO THE VALIDITY OF THE CONTACT REPORT,  FOR EXAMPLE: "BA". See table 2049 which is a composite
