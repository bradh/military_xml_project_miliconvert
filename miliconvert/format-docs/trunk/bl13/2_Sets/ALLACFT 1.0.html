No

Designator

Field Desc

Concept/Explanation/Examples

1

NATIONALITY

 

THE NATIONALITY OF THE ALLOCATED AIRCRAFT, EITHER AS A 2 OR AS A 3 LETTER COUNTRY CODE.

 

 

1A

NATIONALITY (2A)

 

ENTER THE 2 LETTER NATO CODE CORRESPONDING TO THE NATIONALITY OF THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "SP". See table 1262/5 which contains a list of data items and associated data codes

 

 

1B

NATIONALITY (3A)

 

ENTER THE 3 LETTER ALPHABETIC CHARACTER CODE CORRESPONDING TO THE NATIONALITY OF THE ALLOCATED AIRCRAFT, IN ACCORDANCE WITH STANAG 1059, ED. 8, FOR EXAMPLE: "ESP". See table 1262/2 which contains a list of data items and associated data codes

 

 

1C

EXERCISE GEOGRAPHICAL ENTITY

 

ENTER THE ALPHABETIC CHARACTER CODE CORRESPONDING TO ENTER THE FICTITOUS NATIONALITY OF THE AIRCRAFT ALLOCATED FOR THE EXERCISE, IN ACCORDANCE WITH STANAG 1059, ED. 8, FOR EXAMPLE: "XB". See table 1262/4 which contains a list of data items and associated data codes

2

ARMED SERVICE CODE

 

 

 

 

2A

ARMED SERVICE CODE

 

ENTER THE ALPHABETIC CHARACTER CODE CORRESPONDING TO THE ARMED SERVICE TO WHICH THE ALLOCATED AIRCRAFT BELONG, FOR EXAMPLE: "F". See table 1107/2 which contains a list of data items and associated data codes

3

UNIT IDENTIFIER

 

THE NAME OR UNIT IDENTIFICATION CODE CORRESPONDING TO THE ALLOCATED AIRCRAFT.

 

 

3A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OF THE UNIT CORRESPONDING TO THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "UNIT:GRUPO 12". See table 1028/43 which is an instructive entry

 

 

3B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 2 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "UIC2:FRJ123456". See table 2243 which is a composite

 

 

3C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (THIS ALTERNATIVE IS FOR THE 3 LETTER COUNTRY CODE) CORRESPONDING TO THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "UIC3:FRAJ123456". See table 2241 which is a composite

4

BASE IDENTIFIER

 

THE GEOGRAPHICAL LOCATION OF THE BASE WHERE THE ALLOCATED AIRCRAFT ARE SITUATED.

 

 

4A

ICAO LOCATION INDICATOR

ICAO

ENTER "ICAO:" FOLLOWED BY THE ICAO LOCATION INDICATOR OF THE PLACE WHERE THE ALLOCATED AIRCRAFT ARE SITUATED, FOR EXAMPLE: "ICAO:KX4Q". See table 1125/1 which is an instructive entry

 

 

4B

PLACE NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE BASE WHERE THE ALLOCATED AIRCRAFT ARE SITUATED, FOR EXAMPLE: "NAME:TORREJON". See table 1022/170 which is an instructive entry

 

 

4C

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4D

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

5

LOCATION IDENTIFIER

 

 

 

 

5A

LOCATION OR POSITION NAME

 

ENTER AS A LITERAL THE NAME OF THE LOCATION WHERE THE ALLOCATED AIRCRAFT ARE SITUATED, FOR EXAMPLE:"CHARLEROI". See table 1022/6 which is an instructive entry

6

LINK 16 JTIDS UNIT (JU) ADDRESS

 

IF USED, IT PROVIDES A BLOCK OF LINK 16 ADDRESSES ASSIGNED TO THE ALLOCATED AIRCRAFT AS PRIMARY ADDRESSES, DEFINED BY A LOWER LIMIT ADDRESS AND AN UPPER LIMIT ADDRESS.

 

 

6A

UNIT PRIMARY JU ADDRESS NUMBERS

 

ENTER FIRST THE LOWER LIMIT LINK 16 ADDRESS AND THEN THE UPPER LIMIT LINK 16 ADDRESS SEPARATED BY A HYPHEN DEFINING THE BLOCK OF LINK 16 ADDRESSES ASSIGNED TO THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "34521-65341". See table 2606 which is a composite

7

NUMBER OF AIRCRAFT

 

 

 

 

7A

NUMBER OF AIRCRAFT, 1-99

 

ENTER THE NUMBER OF AIRCRAFT ASSIGNED/ATTACHED TO THE ALLOCATED UNIT, FOR EXAMPLE: "6". See table 1044/322 a range [1 through 99]

8

AIRCRAFT TYPE

 

THE TYPE AND MODEL NUMBER OF ALLOCATED AIRCRAFT.

 

 

8A

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE TYPE AND MODEL NUMBER OF THE AIRCRAFT ASSIGNED/ATTACHED TO THE ALLOCATED UNIT, FOR EXAMPLE: "ACTYP:FA18". See table 1015/20 which contains a list of data items and associated data codes

 

 

8B

AIRCRAFT TYPE AND MODEL, OTHER

OTHAC

ENTER "OTHAC:" FOLLOWED BY THE TYPE AND/OR MODEL NUMBER OF THE AIRCRAFT, NOT CONTAINED IN THE AIRCRAFT CODES LIST, ASSIGNED/ATTACHED TO THE ALLOCATED UNIT,  FOR EXAMPLE: "OTHAC:F119A". See table 1015/18 which is an instructive entry

9

AIRCRAFT ROLE

 

 

 

 

9A

AIRCRAFT ROLE

 

ENTER THE THREE ALPHABETIC CHARACTERS CODE CORRESPONDING TO THE CAPABILITY IN ROLE OF THE ALLOCATED AIRCRAFT, FOR EXAMPLE: "FGD". See table 1010/6 which contains a list of data items and associated data codes

10

START OF ALLOCATION TIME PERIOD

 

 

 

 

10A

DTG OF START

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF START OF THE ALLOCATION TIME PERIOD, FOR EXAMPLE: "141325ZFEB2003". See table 2033 which is a composite

11

STOP TIME OF ALLOCATION PERIOD

 

CAN ONLY BE USED IN CONJUNCTION WITH "START OF ALLOCATION TIME PERIOD" SPECIFIED IN THIS SET.

 

 

11A

DTG OF STOP

 

ENTER THE DAY-TIME, MONTH, AND YEAR OF STOP OF THE ALLOCATION TIME PERIOD, FOR EXAMPLE: "141925ZFEB2003". See table 2033 which is a composite

12

UNIT REMARKS

 

 

 

 

12A

REMARKS

 

ENTER, AS LITERAL, ADDITIONAL ESSENTIAL INFORMATION CONCERNING THE UNIT TO WHICH THE ALLOCATED AIRCRAFT BELONG, FOR EXAMPLE: "DET 1 FOL". See table 1357/28 which is an instructive entry
