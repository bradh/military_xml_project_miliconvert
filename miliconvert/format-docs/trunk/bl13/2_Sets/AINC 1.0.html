Designator

Field Desc

Concept/Explanation/Examples

1

AIRCRAFT EVENT NUMBER

 

THE AIRCRAFT EVENT NUMBER.

 

 

1A

AIRCRAFT EVENT NUMBER

 

ENTER THE AIRCRAFT EVENT NUMBER, FOR EXAMPLE: "007". See table 1012/23 a range [001 through 999]

 

 

1B

EXERCISE AIRCRAFT EVENT NUMBER

 

ENTER THEEXERCISE AIRCRAFT EVENT NUMBER, FOR EXAMPLE: "X007". See table 1012/24 which is an instructive entry

2

NUMBER OF AIRCRAFT, 00-99

 

 

 

 

2A

NUMBER OF AIRCRAFT, 00-99

 

ENTER THE NUMBER OF AIRCRAFT [01-99] INVOLVED IN THE INCIDENT, FOR EXAMPLE: "02". See table 1044/3 a range [00 through 99]

3

AIRCRAFT TYPE

 

THE AIRCRAFT CODE OR AIRCRAFT TYPE.

 

 

3A

AIRCRAFT CODE

 

ENTER THE AIRCRAFT CODE, FOR EXAMPLE: "BKF". See table 1015/2 which contains a list of data items and associated data codes

 

 

3B

AIRCRAFT TYPE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE AIRCRAFT TYPE [LITERAL], FOR EXAMPLE: "ACTYP:MIRAGE 5". See table 1015/1 which is an instructive entry

4

AIRCRAFT VARIANT

 

 

 

 

4A

AIRCRAFT VARIANT

 

ENTER THE AIRCRAFT VARIANT EXPRESSED AS ONE OR TWO ALPHABETIC CHARACTERS [A-Z, AA-ZZ], FOR EXAMPLE: "C". See table 1219/2 which is an alphanumeric range

5

GEOGRAPHICAL ENTITY

 

 

 

 

5A

GEOGRAPHICAL ENTITY

 

ENTER THE TWO LETTER COUNTRY CODE, FOR EXAMPLE: "RS". See table 1026/1 which contains a list of data items and associated data codes

6

INCIDENT POSITION

 

THE GEOGRAPHICAL LOCATION OF THE INCIDENT.

 

 

6A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

6B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

6C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

6D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

6E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

6F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

7

TIME OF INCIDENT

 

THE TIME WHEN THE INCIDENT OCCURRED.

 

 

7A

DAY-TIME OF INCIDENT

 

ENTER THE DAY-TIME OF INCIDENT [DDHHMMZ], FOR EXAMPLE: "162215Z". See table 2000 which is a composite

 

 

7B

DAY-MONTH NAME OF INCIDENT

 

IF TIME IS UNKNOWN, ENTER DAY-MONTH OF INCIDENT, FOR EXAMPLE "16SEP". See table 2067 which is a composite

8

STATUS OF ENEMY AIRCRAFT

 

 

 

 

8A

STATUS OF ENEMY AIRCRAFT

 

ENTER THE SPECIFIED STATUS OF ENEMY AIRCRAFT INVOLVED IN THE INCIDENT, FOR EXAMPLE: "DAMAG". See table 1236/13 which contains a list of data items and associated data codes

9

ESTIMATED DAMAGE

 

 

 

 

9A

ESTIMATED DAMAGE

 

IF INCIDENT AIRCRAFT IS STILL OPERABLE, ENTER THE SPECIFIED DEGREE OF DEGRADATION TO THE WEAPON THREAT AND DEGREE OF STRUCTURAL DAMAGE, FOR EXAMPLE: "T1S2". See table 2094 which is a composite

10

STATUS VALIDITY

 

 

 

 

10A

STATUS VALIDITY

 

ENTER THE ANALYST'S JUDGEMENT AS TO REPORTED STATUS OF AIRCRAFT, FOR EXAMPLE: "EST". See table 1067/1 which contains a list of data items and associated data codes
