Concept/Explanation/Examples

1

AEW REQUEST IDENTIFIER

 

 

 

 

1A

AEW REQUEST IDENTIFIER

 

ENTER THE UNIQUE AEW REQUEST IDENTIFIER WHICH IS SATISFIED BY THIS AEW SORTIE, FOR EXAMPLE: "07060077". See table 1012/17 a range [0 through 9999999999999]

2

AEW MISSION ENVIRONMENT

 

 

 

 

2A

AEW MISSION ENVIRONMENT

 

ENTER THE CODE FOR THE ENVIRONMENT IN WHICH THE MISSION TAKES PLACE, FOR EXAMPLE: "A". See table 1285/4 which contains a list of data items and associated data codes

3

PRIORITY IDENTIFIER

 

 

 

 

3A

PRIORITY IDENTIFIER

 

ENTER THE SEQUENTIAL NUMBER WHICH OUTLINES THE IMPORTANCE OF THE MISSION USING A TWO-DIGIT CODE WHERE 01 IS THE HIGHEST PRIORITY, FOR EXAMPLE: "01". See table 1076/20 a range [01 through 99]

4

AEW MISSION TYPE

 

 

 

 

4A

AEW MISSION TYPE

 

ENTER THE TYPE OF AIRBORNE EARLY WARNING MISSION TO BE PERFORMED, FOR EXAMPLE: "FC". See table 1285/7 which contains a list of data items and associated data codes

5

LOCATION NAME

 

 

 

 

5A

LOCATION NAME

 

ENTER THE NAME OF THE AEW ORBIT AREA, FOR EXAMPLE: "FOXTROT". See table 1022/4 which is an instructive entry

6

ON STATION TIME

 

DEFINES WHEN THE RESPECTIVE AEW TASK IS TO COMMENCE, IN ONE OF THE TWO FOLLOWING WAYS:

 

 

6A

DTG ON STATION

 

ENTER THE DATE-TIME GROUP THAT THE AIRCRAFT IS TO PERFORM ITS MISSION, FOR EXAMPLE: "011212ZAUG2001". See table 2033 which is a composite

 

 

6B

DATE ON STATION

 

ENTER THE DAY-MONTH-YEAR THAT THE AIRCRAFT IS TO PERFORM ITS MISSION, FOR EXAMPLE: "01AUG2001". See table 2001 which is a composite

7

OFF STATION TIME

 

DEFINES THE TIME WHEN THE RESPECTIVE AEW TASK IS TO BE TERMINATED.

 

 

7A

DTG OFF STATION

 

ENTER THE DATE-TIME GROUP THAT THE AIRCRAFT IS TO TERMINATE ITS TASKED MISSION, FOR EXAMPLE: "011912ZAUG2001". See table 2033 which is a composite

 

 

7B

DATE OFF STATION

 

ENTER THE DAY-MONTH-YEAR THAT THE AIRCRAFT IS TO TERMINATE ITS TASKED MISSION, FOR EXAMPLE: "02AUG2001". See table 2001 which is a composite

8

TACTICAL CONTROL AUTHORITY

 

 

 

 

8A

TACTICAL CONTROL AUTHORITY

 

ENTER THE AGENCY HAVING TACTICAL CONTROL OF THE AEW MISSION, FOR EXAMPLE: "RED ONE". See table 1028/12 which is an instructive entry
