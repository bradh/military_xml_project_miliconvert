1A

UNIT

 

ENTER THE UNIT NAME OR IDENTIFIER, FOR EXAMPLE: "USS VINCENNES". See table 1022/57 which is an instructive entry

NUMBER ORDERED

 

 

 

 

2A

NUMBER ORDERED

 

ENTER THE NUMBER OF AIRCRAFT OR SHIPS ORDERED, FOR EXAMPLE: "7". See table 1044/99 a range [1 through 999]

3

PLATFORM TYPE

 

SPECIFIES THE TYPE OF PLATFORM ORDERED.

 

 

3A

NAVAL VESSEL TYPE

SHIP

ENTER FIELD DESCRIPTOR "SHIP:" FOLLOWED BY THE STANDARD DESIGNATOR FOR THE NAVAL VESSEL TYPE, FOR EXAMPLE: "SHIP:CG". See table 1166/3 which contains a list of data items and associated data codes

 

 

3B

AIRCRAFT TYPE MODEL

 

ENTER THE AIRCRAFT TYPE/MODEL, FOR EXAMPLE: "F16". See table 1015/4 which contains a list of data items and associated data codes

4

PLATFORM IDENTIFICATION

 

SPECIFIES THE IDENTITY OF THE PLATFORM ORDERED.

 

 

4A

AIRCRAFT TAIL LETTERS AND SIDE NUMBER

TAIL

ENTER FIELD DESCRIPTOR "TAIL:" FOLLOWED BY THE AIRCRAFT TAIL AND/OR SIDE NUMBER, FOR EXAMPLE: "TAIL:CG32". See table 1012/97 which is an instructive entry

 

 

4B

HULL NUMBER

 

ENTER THE HULL/PENNANT NUMBER OF THE ORDERED SHIP, FOR EXAMPLE: "F737". See table 1012/30 which is an instructive entry

 

 

4C

CALL SIGN

CALLSIGN

ENTER FIELD DESCRIPTOR "CALLSIGN:" FOLLOWED BY THE CALL SIGN OF THE PLATFORM, FOR EXAMPLE: "CALLSIGN:FOXTROT4". See table 1129/1 which is an instructive entry

5

MISSION OR SORTIE NUMBER

 

SPECIFIES THE IDENTIFYING NUMBER ASSIGNED TO THE MISSION OR SORTIE.

 

 

5A

MISSION NUMBER

MSNNO

ENTER FIELD DESCRIPTOR "MSNNO:" FOLLOWED BY THE MISSION NUMBER, FOR EXAMPLE: "MSNNO:2T280361" See table 1012/69 which is an instructive entry

 

 

5B

SORTIE NUMBER

 

ENTER THE SORTIE NUMBER ASSIGNED BY AUTHORIZED TASKING AUTHORITY, FOR EXAMPLE: "FL10TL". See table 1012/65 which is an instructive entry
