1

SOUNDING LINE DESIGNATOR

 

 

 

 

1A

SOUNDING LINE DESIGNATOR

 

ENTER THE THREE CHARACTER DESIGNATION RELATING A SOUNDING LINE TO A FIXED POSITION.  THE FIRST CHARACTER IS THE LETTER DESIGNATION OF THE FIXED POSITION TO WHICH A SOUNDING LINE IS RELATED.  THE SECOND AND THIRD CHARACTERS ARE TWO DIGIT NUMBERS THAT DESIGNATE THE SEQUENTIAL NUMBER OF THE INDIVIDUAL SOUNDING LINES RELATED TO THAT FIXED POSITION, FOR EXAMPLE: "A01". See table 1515/6 which is an alphanumeric range

2

BEARING AND RANGE FROM FIXED POSITION TO WATERLINE

 

 

 

 

2A

BEARING AND RANGE FROM FIXED POSITION TO WATERLINE

 

ENTER THE MAGNETIC BEARING AND RANGE FROM THE FIXED POSITION (FP) TO THE WATERLINE AT THE TIME OF THE SOUNDING(WLTS), FOR EXAMPLE: "120A50". See table 2471 which is a composite

3

SOUNDING DATE-TIME GROUP

 

 

 

 

3A

DTG

 

ENTER THE DTG WHEN THE SOUNDINGS WERE TAKEN, FOR EXAMPLE: "091200ZJUN1997". See table 2033 which is a composite

WATER LINE AT TIME OF SOUNDING TO BACK OF BEACH DISTANCE

 

 

 

 

4A

WATER LINE AT TIME OF SOUNDING TO BACK OF BEACH DISTANCE

 

ENTER THE DISTANCE FROM THE WLTS TO THE BACK OF THE BEACH ALONG THE SOUNDING LINE, MEASURED IN FEET OR METRES, FOR EXAMPLE: "120M". See table 2473 which is a composite

5

VERTICAL RISE

 

ENTER THE VERTICAL RISE OVER THE DISTANCE FROM THE WLTS TO THE BACK OF THE BEACH IN ONE OF THE FOLLOWING WAYS:

 

 

5A

VERTICAL RISE (FEET, YARDS OR METRES)

 

ENTER THE VERTICAL RISE IN FEET OR METRES, FOR EXAMPLE: "3M". See table 2452 which is a composite

 

 

5B

VERTICAL RISE CODE

 

ENTER THE ESTIMATED BEACH GRADIENT, FOR EXAMPLE: "MILD". See table 1773/1 which contains a list of data items and associated data codes

6

DISTANCE BETWEEN SOUNDING LINES

 

 

 

 

6A

DISTANCE BETWEEN SOUNDING LINES

 

ENTER THE DISTANCE BETWEEN EACH SOUNDING ON A SOUNDING LINE, MEASURED IN FEET OR METRES, FOR EXAMPLE: "15M". See table 2473 which is a composite

7

SOUNDING DEPTH (FEET, YARDS OR METRES)

 

 

 

 

7A

SOUNDING DEPTH (FEET, YARDS OR METRES)

 

ENTER THE DEPTH OF WATER AT EACH SOUNDING INTERVAL ALONG THE SOUNDING LINE MEASURED TO THE NEAREST 1/10 METRE OR ONE HALF FOOT.  REPEATED IN SEQUENCE FROM THE WLTS TO SEAWARD AS A REPEATABLE ENTRY, FOR EXAMPLE: "4.1M". See table 2453 which is a composite
