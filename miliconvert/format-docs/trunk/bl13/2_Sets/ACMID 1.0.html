Field Desc

Concept/Explanation/Examples

1

AIRSPACE CONTROL MEANS

 

ENTER THE TYPE OF AIRSPACE CONTROL MEANS USING ONE OF THE FOLLOWING:

 

 

1A

TYPE OF AIRSPACE CONTROL MEANS

ACM

ENTER "ACM:" FOLLOWED BY THE TYPE OF AIRSPACE CONTROL MEANS, FOR EXAMPLE: "ACM:REFPT". See table 1389/2 which contains a list of data items and associated data codes

 

 

1B

TYPE OF ACM, OTHER

OTR

ENTER "OTR:" FOLLOWED BY A TYPE OF AIRSPACE CONTROL MEANS NOT LISTED IN THE LOOK-UP TABLE FOR ALTERNATIVE A.  INCLUDE AN AMPN SET TO EXPLAIN THE ACM.  FOR EXAMPLE: "OTR:REFORG". See table 1389/4 which is an instructive entry

2

AIRSPACE CONTROL MEANS IDENTIFIER

 

ENTER THE AIRSPACE CONTROL MEANS IDENTIFIER USING ONE OF THE FOLLOWING:

 

 

2A

AIRSPACE CONTROL MEANS IDENTIFYING DESIGNATOR

DESIG

ENTER "DESIG:" FOLLOWED BY THE  AIRSPACE CONTROL MEANS IDENTIFYING DESIGNATOR, FOR EXAMPLE: "DESIG:C34". See table 1515/12 which is an instructive entry

 

 

2B

AIRSPACE CONTROL MEANS NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE AIRSPACE CONTROL MEANS, FOR EXAMPLE: "NAME:TEXACO" See table 1022/27 which is an instructive entry

3

TYPE OF AIRSPACE SHAPE

 

 

 

 

3A

TYPE OF AIRSPACE SHAPE

 

ENTER THE SHAPE OF THE AIRSPACE, FOR EXAMPLE: "CORRIDOR". See table 1237/2 which contains a list of data items and associated data codes

4

AIRSPACE USAGE

 

ENTER THE USE OF THE AIRSPACE USING ONE OF THE FOLLOWING:

 

 

4A

AIRSPACE USAGE

USE

ENTER "USE:" FOLLOWED BY THE SPECIFIC USE OF THE DEFINED AIRSPACE CONTROL MEANS, FOR EXAMPLE: "USE:TC". See table 1389/3 which contains a list of data items and associated data codes

 

 

4B

AIRSPACE USAGE, OTHER

OTR

ENTER "OTR:" FOLLOWED BY A USAGE NOT LISTED IN THE LOOK-UP TABLE FOR ALTERNATIVE A. INCLUDE AN EXPLANATION OF THE USAGE IN AN AMPN SET, FOR EXAMPLE: "OTR:SINGLE". See table 1389/5 which is an instructive entry

5

ACM LINK 16 TRACK NUMBER REFERENCE

 

 

 

 

5A

ACM LINK 16 TRACK NUMBER REFERENCE

 

IF USED, ENTER THE UNIQUE LINK 16 IDENTIFIER THAT IS ASSIGNED TO A SPECIFIC AIRSPACE CONTROL MEANS, FOR EXAMPLE: "AA232". See table 1625/28 which is an alphanumeric range

6

TRANSITION ALTITUDE

 

 

 

 

6A

TRANSITION ALTITUDE

 

IF USED, ENTER THE ALTITUDE AT OR BELOW WHICH THE VERTICAL POSITION OF AN AIRCRAFT IS CONTROLLED BY REFERENCE TO TRUE ALTITUDE, FOR EXAMPLE: "12000FT". See table 2279 which is a composite

Structured Language:      

THE AIRSPACE CONTROL MEANS MUST EQUAL "ADAREA" IF THE AIRSPACE USAGE IS SPECIFIED AS "ADIZ", "BZ", "BDZ", "WFZ", "HIDACZ", "LMEZ", "LFEZ", "JEZ", "JOA", "KILLZ", "HIMEZ", "LOMEZ" OR "SHORAD".
THE AIRSPACE CONTROL MEANS MUST EQUAL "ADOA" IF THE AIRSPACE USAGE IS SPECIFIED AS "APPCOR", "CCZONE", "CADA", "COZ", "MFEZ", "MMEZ", "SAFES", "SCZ", "MISARC", "ISR", "AOA", "ADAA", "ADZ", "FRAD", "FIRUB", "PIRAZ" OR "RTF".
THE AIRSPACE CONTROL MEANS MUST EQUAL "ATCA" IF THE AIRSPACE USAGE IS SPECIFIED AS "ARWY", "CLSA", "CLSB", "CLSC", "CLSD", "CLSE", "CLSF", "CLSG", "ATSRTE", "FIR", "TCA", "DA", "PROHIB", "RA", "TRSA", "WARN", "ADVRTE", "CONTZN", "CTA", "NAVRTE", "CDR", "CBA", "RCA" OR "TSA".
THE AIRSPACE CONTROL MEANS MUST EQUAL "CORRTE" IF THE AIRSPACE USAGE IS SPECIFIED AS "AIRCOR", "AIRRTE", "SAAFR", "TMRR", "TC", "SL", "TR", "SC" OR "MRR".
THE AIRSPACE CONTROL MEANS MUST EQUAL "PROC" IF THE AIRSPACE USAGE IS SPECIFIED AS "ALTREV", "CL", "FSCL", "GNDAOR", "FLOT", "TL", "FEBA", "IFFON", "IFFOFF", "CFL", "RFA", "RFL", "BNDRY", "FFA", "DBSL", "ACA", "SAFE" OR "MRL".
THE AIRSPACE CONTROL MEANS MUST EQUAL "REPT" IF THE AIRSPACE USAGE IS SPECIFIED AS "ACP", "CP", "EG", "HG", "MG", "MP", "BULL", "SARDOT" OR "ISP".
THE AIRSPACE CONTROL MEANS MUST EQUAL "ROZ" IF THE AIRSPACE USAGE IS SPECIFIED AS "AAR", "AEW", "EC", "LZ", "ABC", "DZ", "PZ", "RECCE", "UAV", "TRNG", "SEMA", "SOF", "CAS", "CAP" OR "ROA".
THE AIRSPACE CONTROL MEANS MUST EQUAL "SUA" IF THE AIRSPACE USAGE IS SPECIFIED AS "ASCA", "ACSS", "FACA", "NOFLY", "FARP", "FOL", "SSMS", "ALERTA", "MOA", "NFA" OR "KILLBX".
