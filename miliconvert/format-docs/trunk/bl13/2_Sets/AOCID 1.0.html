Concept/Explanation/Examples

1

UNIT IDENTIFIER

 

THE IDENTIFICATION FOR THE RECIPIENT UNIT OF THE ANNEX.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OF THE UNIT FOR WHICH THE ANNEX IS INTENDED, FOR EXAMPLE: "UNIT:CAOC5". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC2)

UIC2

ENTER "UIC2:" FOLLOWED BY THE UNIT IDENTIFICATION CODE, (USING THE NATO 2 LETTER COUNTRY CODE OF THE UNIT FOR WHICH THE ANNEX IS INTENDED), FOR EXAMPLE: "UIC2:FRJ12345". See table 2243 which is a composite

 

 

1C

UNIT IDENTIFICATION CODE (UIC3)

UIC3

ENTER "UIC3" FOLLOWED BY THE UNIT IDENTIFICATION CODE, (USING THE NATO 3 LETTER COUNTRY CODE OF THE UNIT FOR WHICH THE ANNEX IS INTENDED), FOR EXAMPLE: "UIC3:FRAJ12345". See table 2241 which is a composite

2

DOCUMENT IDENTIFIER, ORIGINAL OR CHANGE

 

 

 

 

2A

DOCUMENT IDENTIFIER, ORIGINAL OR CHANGE

 

ENTER THE DOCUMENT IDENTIFIER, FOR EXAMPLE: "CHANGE". See table 1589/4 which contains a list of data items and associated data codes

3

CHANGE SERIAL NUMBER

 

CAN ONLY BE USED IF THE DOCUMENT IDENTIFIER EQUALS "CHANGE" IN THIS SET.

 

 

3A

AMENDMENT MESSAGE SERIAL NUMBER

 

ENTER THE SEQUENTIAL NUMBER INDICATING THE INSTANCE OF THE CHANGE OCCURRED TO THE AOD, FOR EXAMPLE: "2". See table 1012/232 a range [1 through 99]
