1A

DATE-TIME MONTH, VERIFIED

 

SPECIFY THE SURFACING ZERO TIME, WHICH IS NORMALLY 6 HOURS AFTER THE CONCLUSION OF A MOVEMENT, I.E. IF THE MOVESUM SET SHOWS ARRIVAL AT CARTAGENA AT 0712200Z2APR, THE ASSOCIATED SZER TIME WOULD BE, FOR EXAMPLE: "071800Z6APR", WHICH MEANS THAT THE SURFACING SIGNAL MUST BE RECEIVED BY THE OPERATIONAL CONTROL AUTHORITY BEFORE THAT TIME, FAILURE TO MEET THIS REQUIREMENT WILL RESULT IN ACTIONS IN ACCORDANCE WITH ATP 10. See table 2032 which is a composite

2

ZERO START LOCATION

 

THE GEOGRAPHIC LOCATION AT WHICH THE SUBMARINE PLANS TO TRANSMIT A SURFACING SIGNAL.

 

 

2A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES
