1

CONTAMINATION, DOSE AND DOSE RATE/DOSAGE

 

PROVIDES THE LEVEL OF CONTAMINATION, LEVEL OF DOSE OR LEVEL OF DOSE RATE/DOSAGE IN ONE OF THE THREE FOLLOWING WAYS:

 

 

1A

LEVEL OF CONTAMINATION AND UNIT OF MEASUREMENT

CON

ENTER THE FIELD DESCRIPTOR "CON:" FOLLOWED BY THE LEVEL OF CONTAMINATION, FOR EXAMPLE: "CON:0.0005BQM2". See table 2464 which is a composite

 

 

1B

LEVEL OF DOSE AND UNIT OF MEASUREMENT

DOS

ENTER THE FIELD DESCRIPTOR "DOS:" FOLLOWED BY THE LEVEL OF DOSE, FOR EXAMPLE: "DOS:0.0002CSV'. See table 2457 which is a composite

 

 

1C

LEVEL OF DOSE RATE/DOSAGE AND UNIT OF MEASUREMENT

RAT

ENTER THE FIELD DESCRIPTOR "RAT:" FOLLOWED BY THE  LEVEL OF DOSE RATE/ DOSAGE, FOR EXAMPLE: "RAT:0.0060MSH". See table 2456 which is a composite

2

DOSE RATE TREND

 

 

 

 

2A

DOSE RATE TREND

 

ENTER THE DOSE RATE TREND, FOR EXAMPLE: "INIT". See table 1324/2 which contains a list of data items and associated data codes

3

DECAY RATE

 

IF USED, PROVIDES THE ACTUAL OR RELATIVE DECAY RATE IN ONE OF THE TWO FOLLOWING WAYS:

 

 

3A

ACTUAL RADIATION DECAY RATE

 

ENTER THE ACTUAL DECAY RATE, FOR EXAMPLE: "1.89". See table 1326/1 a range [0.20 through 2.00] (2 to 2 decimal places)

 

 

3B

RELATIVE RADIATION DECAY RATE

 

ENTER THE RELATIVE DECAY RATE, FOR EXAMPLE: "DF". See table 1325/1 which contains a list of data items and associated data codes
