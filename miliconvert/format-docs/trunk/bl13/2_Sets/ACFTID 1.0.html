1

AIRCRAFT TYPE/MODEL

 

 

 

 

1A

AIRCRAFT TYPE MODEL

 

ENTER THE AIRCRAFT TYPE MODEL, FOR EXAMPLE: "P3C". See table 1015/4 which contains a list of data items and associated data codes

2

AIRCRAFT IDENTIFICATION NUMBER

 

ENTER ONE OF THE FOLLOWING:

 

 

2A

AIRCRAFT IDENTIFICATION NUMBER

 

ENTER THE AIRCRAFT IDENTIFICATION NUMBER, FOR EXAMPLE: "MC28361". See table 1012/84 which is an instructive entry

 

 

2B

AIRCRAFT SIDE NUMBER

SIDENO

ENTER THE FIELD DESCRIPTOR "SIDENO:" FOLLOWED BY THE NUMBER, FOR EXAMPLE: "SIDENO:123". See table 1012/92 which is an instructive entry

3

AIRCRAFT BUREAU NUMBER

 

 

 

 

3A

AIRCRAFT BUREAU NUMBER

 

ENTER THE SERIAL NUMBER TO AN AIRCRAFT BY THE MANUFACTURER, FOR EXAMPLE: "H123". See table 1012/48 which is an instructive entry

4

NAVIGATION ACCURACY, YARDS, METRES OR NAUTICAL MILES

 

 

 

 

4A

NAVIGATION ACCURACY, YARDS, METRES OR NAUTICAL MILES

 

ENTER THE DIFFERENCE, GIVEN AS A LINEAR MEASUREMENT, BETWEEN AN ACTUAL POSITION AND  INTENDED POSITION, EXPRESSED AS ONE TO FOUR NUMERALS, FOLLOWED BY THE UNITS OF MEASUREMENT EXPRESSED AS YARDS, MILES OR NAUTICAL MILES,  FOR EXAMPLE: "8NM". See table 2284 which is a composite

5

SPECIAL CONFIGURATION EQUIPMENT NAME

 

 

 

 

5A

SPECIAL CONFIGURATION EQUIPMENT NAME

 

ENTER ANY EQUIPMENT INSTALLED ON AN AIRCRAFT, SHIP, OR WEAPON WHICH IS NOT NORMALLY ASSOCIATED WITH THAT PARTICULAR MODEL, FOR EXAMPLE: "BEARTRAP". See table 1022/76 which is an instructive entry
