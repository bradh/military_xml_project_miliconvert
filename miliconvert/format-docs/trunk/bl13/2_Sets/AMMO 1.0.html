1

AIR-TO-AIR WEAPON SYSTEM

 

THE TYPE OF AIR-TO AIR WEAPON SYSTEM ORDNANCE AVAILABLE AT THE BASE BEING REPORTED.

 

 

1A

ORDNANCE WEAPON TYPE, AIR-TO-AIR

ATOA

ENTER "ATOA:" FOLLOWED BY THE CODE FOR THE AIR-TO-AIR WEAPON TYPE AVAILABLE AT THE BASE, FOR EXAMPLE:"ATOA:AIM46". See table 1407/31 which contains a list of data items and associated data codes

 

 

1B

ORDNANCE WEAPON AMMUNITION TYPE NAME

ANAME

ENTER "ANAME:" FOLLOWED BY THE NAME OF THE AIR-TO-AIR WEAPON TYPE AVAILABLE AT THE BASE, FOR EXAMPLE:"ANAME:SIDEWINDER". See table 1022/24 which is an instructive entry

2

AIR-TO- SURFACE WEAPON SYSTEM

 

THE TYPE OF AIR-TO- SURFACE WEAPON SYSTEM ORDNANCE AVAILABLE AT THE BASE BEING REPORTED.

 

 

2A

ORDNANCE WEAPON TYPE AIR-TO-SURFACE

ATOS

ENTER "ATOS:" FOLLOWED BY THE CODE FOR THE AIR-TO-GROUND WEAPON TYPE AVAILABLE AT THE BASE, FOR EXAMPLE: "ATOS:MK84". See table 1407/32 which contains a list of data items and associated data codes

 

 

2B

ORDNANCE WEAPON AMMUNITION TYPE NAME

SNAME

ENTER "SNAME:" FOLLOWED BY THE NAME OF THE AIR-TO-GROUND WEAPON TYPE AVAILABLE AT THE BASE, FOR EXAMPLE: "SNAME:500 LB GP BOMB". See table 1022/24 which is an instructive entry

3

ANTI-SUBSURFACE WEAPON SYSTEM

 

THE TYPE OF ANTI-SUBSURAFCE WEAPON SYSTEM ORDNANCE AVAILABLE AT THE BASE BEING REPORTED.

 

 

3A

ORDNANCE WEAPON TYPE, ANTI-SUBSURFACE

ASUB

ENTER "ASUB:" FOLLOWED BY THE CODE FOR THE ANTI-SUBSURFACE WEAPON TYPE AVAILABLE AT THE BASE, FOR EXAMPLE: "ASUB:MK54". See table 1407/25 which contains a list of data items and associated data codes

 

 

3B

ORDNANCE WEAPON AMMUNITION TYPE NAME

SSNAME

ENTER "SSNAME:" FOLLOWED BY THE NAME OF THE ANTI-SUBSURFACE WEAPON TYPE AVAILABLE AT THE BASE, FOR EXAMPLE: "SSNAME:MARK 52 MINE". See table 1022/24 which is an instructive entry

4

CHAFF/FLARE TYPE

 

THE CHAFF/FLARE TYPE AVAILABLE AT THE BASE BEING REPORTED.

 

 

4A

CHAFF FLARE TYPE

CHFL

ENTER "CHFL:" FOLLOWED BY THE CODE FOR THE TYPE OF CHAFF OR FLARE AVAILABLE, FOR EXAMPLE: "CHFL:CCCAA". See table 1407/28 which contains a list of data items and associated data codes

 

 

4B

ORDNANCE WEAPON AMMUNITION TYPE NAME

CNAME

ENTER "CNAME:" FOLLOWED BY THE NAME OF THE TYPE OF CHAFF OR FLARE AVAILABLE , FOR EXAMPLE: "CNAME:HEAT FLARE". See table 1022/24 which is an instructive entry
