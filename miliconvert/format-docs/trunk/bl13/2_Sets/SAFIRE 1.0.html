REPORTING AIRCRAFT POSITION

 

THE GEOGRAPHICAL LOCATION OF THE REPORTING AIRCRAFT.

 

 

1A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

2

DTG OF EVENT

 

PROVIDES THE DATE-TIME OF THE SURFACE-TO-AIR REPORTED EVENT.

 

 

2A

DTG OF EVENT

 

ENTER THE DATE-TIME GROUP OF THE SURFACE-TO-AIR REPORTED EVENT, FOR EXAMPLE "302339ZJUN2005" See table 2033 which is a composite

3

REPORTING AIRCRAFT ALTITUDE IN HUNDREDS OF FEET

 

PROVIDES THE ALTITUDE OF THE REPORTING AIRCRAFT IN HUNDREDS OF FEET.

 

 

3A

REPORTING AIRCRAFT ALTITUDE IN HUNDREDS OF FEET

 

ENTER THE ALTITUDE OF THE RE[PORTING AITRCRAFT IN HUNDREDS OF FEET, FOR EXAMPLE: "155". See table 1255/18 a range [000 through 999]

4

REPORTING AIRCRAFT HEADING IN DEGREES

 

PROVIDES THE REPORTING AIRCRAFT HEADING IN DEGREES AND INDICATION IF TRUE, MAGNETIC, GRID OR RELATIVE.

 

 

4A

REPORTING AIRCRAFT HEADING IN DEGREES

 

ENTER THE REPORTING AIRCRAFT HEADING IN DEGREES AND INDICATION IF TRUE, MAGNETIC, GRID OR RELATIVE, FOR EXAMPLE "180T". See table 2065 which is a composite

5

AIRCRAFT AIRSPEED

 

IF USED, PROVIDES THE SPEED OF THE REPORTING AIRCRAFT IN KPH, KTS OR MPH.

 

 

5A

AIRCRAFT AIRSPEED IN KPH, KTS, MPH

 

ENTER THE SPEED OF THE REPORTING AIRCRAFT IN KPH, KTS OR MPH, FOR EXAMPLE "410KTS". See table 2461 which is a composite

6

BEARING FROM AIRCRAFT TO EVENT

 

PROVIDES THE BEARING BETWEEN THE REPORTING AIRCRAFT ANTHE EVENT.

 

 

6A

BEARING FROM AIRCRAFT TO EVENT, DEGREES TRUE

 

ENTER THE BEARING BETWEEN THE REPORTING AIRCRAFT AND THE EVENT, FOR EXEMPLE: "020T". See table 2253 which is a composite

7

ESTIMATED LAUNCH POSITION

 

THE ESTIMATED POSITION OF MISSILE LAUNCH.

 

 

7A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

8

SURFACE-TO-AIR FIRE TYPE

 

PROVIDES THE TYPE OF ENEMY SURFACE-TO-AIR FIRE FRIENDLY AIRCRAFT ENCOUNTERED DURING FLIGHT OPERATIONS.

 

 

8A

SURFACE-TO-AIR FIRE TYPE

 

ENTER THE TYPE OF ENEMY SURFACE-TO-AIR FIRE FRIENDLY AIRCRAFT ENCOUNTERED DURING FLIGHT OPERATIONS, FOR EXAMPLE "SAM". See table 1862/14 which contains a list of data items and associated data codes

9

ORDNANCE/WEAPON TYPE, SURFACE-TO-AIR

 

PROVIDES THE CODE FOR THE ORDNANCE/WEAPON TYPE FIRED FROM THE SURFACE.

 

 

9A

ORDNANCE WEAPON TYPE, SURFACE-TO-AIR

 

ENTER THE CODE FOR THE ORDNANCE/WEAPON TYPE FIRED FROM THE SURFACE, FOR EXAMPLE "SA8". See table 1407/5 which contains a list of data items and associated data codes

10

METHOD OF ACQUISITION

 

PROVIDES THE METHOD BY WHICH THE MISSION AIRCRAFT ACQUIRED ENEMY SURFACE-TO-AIR FIRE.

 

 

10A

METHOD OF ACQUISITION

 

ENTER THE METHOD BY WHICH THE MISSION AIRCRAFT ACQUIRED ENEMY SURFACE-TO-AIR FIRE, FOR EXAMPLE: "ELEC". See table 1866/7 which contains a list of data items and associated data codes

11

RADAR WARNING RECEIVER INDICATIONS RECEIVED

 

PROVIDES AN INDICATOR AS TO WHETHER RADAR WARNING RECEIVER INDICATIONS WERE RECEIVED DURING FLIGHT OPERATIONS. CAN ONLY BE USED IF METHOD OF ACQUISITION IS "ELECTRONIC" OR "BOTH" IN THIS SET.

 

 

11A

RADAR WARNING RECEIVER INDICATIONS RECEIVED INDICATOR

 

ENTER AN INDICATOR AS TO WHETHER RADAR WARNING RECEIVER INDICATIONS WERE RECEIVED DURING FLIGHT OPERATIONS, FOR EXAMPLE "YES". See table 1027/89 which contains a list of data items and associated data codes

12

PLUME COLOUR AT LAUNCH

 

IF USED, PROVIDES THE COLOR OF EITHER THE PLUME OF THE LAUNCHED SURFACE-TO-AIR MISSILE, OR OF THE ANTI-AIRCRAFT ARTILLERY'S MUZZLE FLASH.

 

 

12A

PLUME COLOUR AT LAUNCH

PL

ENTER THE "PL:" FOLLOWED BY THE COLOUR OF THE PLUME OF THE LAUNCHED SURFACE-TO-AIR MISSILE, FOR EXAMPLE "PL:WHITE". See table 1138/3 which contains a list of data items and associated data codes

 

 

12B

MUZZLE FLASH COLOUR

MZ

ENTER "MZ:", FOLLOWED BY THE COLOUR OF THE ANTI-AIRCRAFT ARTILLERY'S MUZZLE FLASH, FOR EXAMPLE "MZ:ORANGE". See table 1138/4 which contains a list of data items and associated data codes

13

PLUME COLOUR IN FLIGHT

 

IF USED, PROVIDES THE COLOUR OF EITHER THE PLUME OF THE LAUNCHED SURFACE-TO-AIR MISSILE DURING FLIGHT, OR OF THE ANTI-AIRCRAFT ARTILLERY'S TRACERS DURING ENGAGEMENT.

 

 

13A

PLUME COLOUR IN FLIGHT

PC

ENTER "PC:", FOLLOWED BY THE COLOR OF THE PLUME OF THE LAUNCHED SURFACE-TO-AIR MISSILE DURING FLIGHT, FOR EXAMPLE "PC:WHITE". See table 1138/13 which contains a list of data items and associated data codes

 

 

13B

TRACER COLOUR

TC

ENTER "TC:" FOLLOWED BY THE COLOUR OF ANTI-AIRCRAFT ARTILLERY TRACERS DURING ENGAGEMENT, FOR EXAMPLE "TC:ORANGE". See table 1138/14 which contains a list of data items and associated data codes

14

NUMBER OF ENEMY MISSILES LAUNCHED

 

IF USED, PROVIDES THE NUMBER OF EITHER THE ENEMY MISSILES LAUNCHED DURING ENGAGEMENT, OR THE NUMBER OF ROUNDS

 

 

14A

NUMBER OF ENEMY MISSILES LAUNCHED

SAM

ENTER "SAM:", FOLLOWED BY THE NUMBER OF ENEMY SURFACE-TO-AIR MISSILES LAUNCHED, FOR EXAMPLE: "SAM:8". See table 1012/123 a range [0 through 99]

 

 

14B

NUMBER OF ENEMY GUNS FIRED

AAA

ENTER "AAA:" FOLLOWED BY THE NUMBER OF ENEMY ANTI-AIRCRAFT ARTILLERY GUNS FIRED, FOR EXAMPLE: "AAA:4". See table 1012/179 a range [0 through 99]

15

ATTACK INTENSITY

 

IF USED, PROVIDES THE SUBJECTIVE EVALUATION OF THE LEVEL OF ATTACK SUSTAINED BY THE MISSION AIRCRAFT.

 

 

15A

ATTACK INTENSITY

 

ENTER A SUBJECTIVE EVALUATION OF THE LEVEL OF ATTACK SUSTAINED BY THE MISSION AIRCRAFT, FOR EXAMPLE: "HEAVY". See table 1293/1 which contains a list of data items and associated data codes

16

DURATION OF FIRE IN MINUTES

 

IF USED, PROVIDES THE NUMBER OF MINUTES THE MISSION AIRCRAFT WAS FIRED UPON.

 

 

16A

DURATION OF FIRE IN MINUTES

 

ENTER THE NUMBER OF MINUTES THE MISSION AIRCRAFT WAS FIRED UPON, FOR EXAMPLE: "15". See table 1072/21 a range [1 through 999]

17

GUIDANCE TYPE

 

IF USED, PROVIDES INDICATION AS TO WHETHER THE WEAPONS SYSTEMS EMPLOYED AGAINST THE MISSION AIRCRAFT WERE GUIDED OR UNGUIDED.

 

 

17A

GUIDANCE TYPE

 

ENTER AN INDICATION AS TO WHETHER THE WEAPONS SYSTEMS EMPLOYED AGAINST THE MISSION AIRCRAFT WERE GUIDED OR UNGUIDED, FOR EXAMPLE: "GUIDED". See table 1862/15 which contains a list of data items and associated data codes

18

SAM APEX ALTITUDE

 

IF USED, PROVIDES THE MAXIMUM ALTITUDE REACHED BY THE SURFACE-TO-AIR MISSILES LAUNCHED AGAINST THE MISSION AIRCRAFT IN FEET OR METRES ABOVE THE MEAN SEA LEVEL.

 

 

18A

SAM APEX ALTITUDE, FEET OR METRES

 

ENTER THE MAXIMUM ALTITUDE REACHED BY THE SURFACE-TO-AIR MISSILES LAUNCHED AGAINST THE MISSION AIRCRAFT IN FEET OR METRES ABOVE THE MEAN SEA LEVEL, FOR EXAMPLE: "38000FT". See table 2531 which is a composite

19

AIRBURST ALTITUDE, FEET OR METRES

 

IF USED, PROVIDES THE AIRBURST ALTITUDE OF SURFACE-TO-AIR MISSILE OR ANTI-AIRCRAFT ARTILLERY ROUNDS IN FEET OR METRES FROM MEAN SEA LEVEL.

 

 

19A

AIRBURST ALTITUDE, FEET OR METRES

 

ENTER THE AIRBURST ALTITUDE OF SURFACE-TO-AIR MISSILE OR ANTI-AIRCRAFT ARTILLERY ROUNDS IN FEET OR METRES FROM MEAN SEA LEVEL, FOR EXAMPLE "26000FT". See table 2531 which is a composite

20

AIRBURST COLOUR

 

IF USED, PROVIDES THE COLOUR OF THE AIRBURST PROVOKED BY THE SURFACE-TO-AIR WEAPON AT DETONATION.

 

 

20A

AIRBURST COLOUR

 

ENTER THE COLOUR OF THE AIRBURST PROVOKED BY THE SURFACE-TO-AIR WEAPON AT DETONATION, FOR EXAMPLE "WHITE". See table 1138/15 which contains a list of data items and associated data codes

21

AIRBURST SIZE

 

IF USED, PROVIDES THE ESTIMATE SIZE OF THE AIRBURST.

 

 

21A

AIRBURST SIZE

 

ENTER THE ESIMATED DIMENSION OF AIRBURST, FOR EXAMPLE: "LRG". See table 1170/2 which contains a list of data items and associated data codes

22

CLOSEST MISSED POSITION

 

IF USED, PROVIDES THE SPECIFICATION OF THE CLOSEST POSITION BETWEEN THE MISSION AIRCRAFT AND THE THE ENEMY SURFACE-TO-AIR MISSILE OR ANTI-AIR ARTILLERY WHICH MISSED IT, SPECIFIED BY CLOCK POSITION, LEVEL AND DISTANCE IN METRES.

 

 

 

22A

CLOSEST MISSED POSITION

 

ENTER THE SPECIFICATION OF THE CLOSEST POSITION BETWEEN THE MISSION AIRCRAFT AND THE THE ENEMY SURFACE-TO-AIR MISSILE OR ANTI-AIR ARTILLERY WHICH MISSED IT, FOR EXAMPLE: "10X500". See table 2140 which is a composite

23

PILOT THREAT PERCEPTION

 

IF USED, PROVIDES THE PILOT'S PERCEPTION OF THE THREAT TO AIRCRAFT DURING ENEMY ATTACK.

 

 

23A

PILOT THREAT PERCEPTION

 

ENTER THE PILOT'S PERCEPTION OF THE THREAT TO AIRCRAFT DURING ENEMY ATTACK, FOR EXAMPLE "HIGH". See table 1174/2 which contains a list of data items and associated data codes
