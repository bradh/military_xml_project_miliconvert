No

Designator

Field Desc

Concept/Explanation/Examples

1

MCM TASK ORDER NUMBER

 

 

 

 

1A

MCM TASK ORDER NUMBER

 

ENTER THE MCM AREA DESIGNATOR [000-9999] PLUS MCM OPERATION SUFFIX PLUS TASK SEQUENCE NUMGER [1-9], FOR EXAMPLE: "9000CH5". See table 2232 which is a composite

2

TASKED SHIP/UNIT

 

ENTER THE TASKED SHIP OR UNIT CHOOSING THE APPROPRIATE ALTERNATE FIELD:

 

 

2A

SHIP NAME

 

ENTER SHIP NAME [LITERAL], FOR EXAMPLE: "BRONNINGTON". See table 1022/49 which is an instructive entry

 

 

2B

TASKED UNIT

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT IDENTIFIER [LITERAL], FOR EXAMPLE: "UNIT:MAKKUM". See table 1028/10 which is an instructive entry

3

Q-ROUTE/AREA NAME

 

IF USED, ENTER THE Q-ROUTE OR AREA NAME SELECTING ONE OF THE FOLLOWING ALTERNATIVES:

 

 

3A

Q-ROUTE NUMBER

 

ENTER THE Q-ROUTE NUMBER [01-999], FOR EXAMPLE: "99". See table 1012/184 a range [1 through 999]

 

 

3B

Q-ROUTE SEGMENT

 

ENTER THE Q-ROUTE SEGMENT COMPRISING Q-ROUTE NUMBER [01-999], Q-ROUTE POINT DESIGNATOR [A-Z], DISTANCE IN NM FROM START POINT [0-9999 OR 99.9], HYPHEN, QROUTE POINT DESIGNATOR [A-Z] AND DISTANCE IN NM FROM FINISH POINT [0-9999 OR 99.9], FOR EXAMPLE: "01A99.9-B50". See table 2233 which is a composite

 

 

3C

MCM AREA NAME

AREA

ENTER "AREA:" FOLLOWED BY THE MCM AREA NAME [LITERAL], FOR EXAMPLE: "AREA:MORRAY". See table 1022/194 which is an instructive entry

4

TIME MODIFIER

 

 

 

 

4A

TIME MODIFIER

 

ENTER TIME MODIFIER, FOR EXAMPLE: "ON", REPEATING AS NECESSARY. See table 1220/11 which contains a list of data items and associated data codes

5

TASK TIME

 

IF USED, ENTER THE TASK TIME IN ONE OF THE FOLLOWING WAYS:

 

 

5A

DAY-TIME

 

ENTER THE DAY-TIME  [DATE, HOUR, MINUTE, TIME ZONE], FOR EXAMPLE: "061230Z". See table 2000 which is a composite

 

 

5B

DAY-TIME AND MONTH

 

ENTER THE DATE-TIME GROUP WHICH CONSISTS OF DATE [01-31], HOUR [00-24], MINUTE [00-59], TIME ZONE,  AND MONTH NAME, FOR EXAMPLE: "061230ZMAR". See table 2030 which is a composite
