1

NATO DESIGNATOR

 

THE NATO DESIGNATOR ASSIGNED TO THE SUBMARINE OR SURFACE CONTACT.

 

 

1A

NATO SURFACE CONTACT DESIGNATOR

 

ENTER THE NATO SURFACE CONTACT DESIGNATOR (SURF), FOR EXAMPLE: "16AA". See table 2154 which is a composite

 

 

1B

NATO SUBMARINE CONTACT DESIGNATOR

 

ENTER THE NATO SUBMARINE CONTACT DESIGNATOR (POD), FOR EXAMPLE: "1603". See table 2152 which is a composite

 

 

1C

NATO SUBMARINE DEPLOYMENT DESIGNATOR

 

ENTER THE NATO SUBMARINE DEPLOYMENT DESIGNATOR (DSUB), FOR EXAMPLE: "012". See table 1214/1 a range [001 through 599]

 

 

1D

NATO SUBMARINE DEPLOYMENT DESIGNATOR AND 4-DIGIT YEAR

 

ENTER THE NATO SUBMARINE DEPLOYMENT DESIGNATOR (DSUB) AND YEAR, FOR EXAMPLE: "012-96". See table 2040 which is a composite

 

 

1E

NATO EXERCISE SURFACE CONTACT DESIGNATOR

 

ENTER THE NATO EXERCISE SURFACE CONTACT DESIGNATOR (SURF), FOR EXAMPLE: "X16AA". See table 2155 which is a composite

 

 

1F

NATO EXERCISE SUBMARINE CONTACT DESIGNATOR

 

ENTER THE NATO EXERCISE SUBMARINE CONTACT DESIGNATOR (POD), FOR EXAMPLE: "X1603". See table 2153 which is a composite

 

 

1G

NATO EXERCISE SUBMARINE DEPLOYMENT DESIGNATOR

 

ENTER THE NATO EXERCISE SUBMARINE DEPLOYMENT DESIGNATOR (DSUB), FOR EXAMPLE: "X012". See table 2081 which is a composite

 

 

1H

NATO EXERCISE SUB DEPLOYMENT DESIGNATOR AND 4-DIGIT YEAR

 

ENTER THE NATO EXERCISE SUBMARINE DEPLOYMENT DESIGNATOR (DSUB) AND YEAR, FOR EXAMPLE: "X012-96". See table 2035 which is a composite

 

 

1I

NATO UNDESIGNATED UNIT

 

ENTER THE CODE "UNDES" FOR A NATO UNDESIGNATED UNIT. See table 1028/25 which contains a list of data items and associated data codes

2

NAVAL VESSEL TYPE

 

 

 

 

2A

NAVAL VESSEL TYPE

 

ENTER THE STANDARD DESIGNATOR FOR THE NAVAL VESSEL TYPE, FOR EXAMPLE: "DD". See table 1166/3 which contains a list of data items and associated data codes

3

NATIONALITY

 

 

 

 

3A

GEOGRAPHICAL ENTITY

 

ENTER THE COUNTRY CODE REPRESENTING THE NATIONALITY OF THE VESSEL, FOR EXAMPLE: "RS". See table 1026/1 which contains a list of data items and associated data codes

4

UNIT IDENTIFIER

 

THE IDENTIFIER OF THE SUBJECT UNIT.

 

 

4A

SHIP NAME

NAME

ENTER FIELD DESCRIPTOR "NAME:" FOLLOWED BY THE SHIP NAME, FOR EXAMPLE: "NAME:BKU". See table 1022/49 which is an instructive entry

 

 

4B

NAME OF SHIP CLASS

CLASS

ENTER FIELD DESCRIPTOR "CLASS:" FOLLOWED BY THE NAME OF THE VESSEL CLASS, FOR EXAMPLE: "CLASS:SOVREMENNYY". See table 1022/35 which is an instructive entry

 

 

4C

SUBMARINE CLASS NAME

SUBCL

ENTER FIELD DESCRIPTOR "SUBCL:" FOLLOWED BY THE SUBMARINE CLASS NAME, FOR EXAMPLE: "SUBCL:ECHO". See table 1038/1 which contains a list of data items and associated data codes

 

 

4D

SUBMARINE CLASS DESIGNATOR

 

ENTER THE SUBMARINE CLASS DESIGNATOR, FOR EXAMPLE: "C". See table 1338/1 which is an instructive entry

 

 

4E

SUBMARINE PROPULSION TYPE

PROPUL

ENTER FIELD DESCRIPTOR "PROPUL:" FOLLOWED BY THE SUBMARINE PROPULSION TYPE, FOR EXAMPLE: "PROPUL:NUC2-3". See table 1339/1 which contains a list of data items and associated data codes

 

 

4F

CLASS AND PENNANT NUMBER

CP

ENTER FIELD DESCRIPTOR "CP:" FOLLOWED BY THE CLASS AND PENNANT NUMBER, FOR EXAMPLE: "CP:KARA 745". See table 1062/1 which is an instructive entry

5

INCIDENT LOCATION

 

THE GEOGRAPHIC LOCATION OF THE SHIP/SUBMARINE INCIDENT.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

5C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

5D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

5E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

5F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

6

TIME OF INCIDENT

 

THE TIME WHEN THE INCIDENT OCCURRED.

 

 

6A

DAY-TIME OF INCIDENT

 

ENTER THE DAY-TIME OF THE INCIDENT, FOR EXAMPLE: "162215Z". See table 2000 which is a composite

 

 

6B

DAY-MONTH NAME OF INCIDENT

 

ENTER THE DAY-MONTH OF INCIDENT, FOR EXAMPLE "16SEP". See table 2067 which is a composite

7

STATUS OF ENEMY UNIT

 

 

 

 

7A

STATUS OF ENEMY UNIT

 

ENTER THE STATUS OF ENEMY UNIT INVOLVED IN THE INCIDENT, FOR EXAMPLE: "DAMAG". See table 1236/14 which contains a list of data items and associated data codes

8

ESTIMATED DAMAGE

 

 

 

 

8A

ESTIMATED DAMAGE

 

ENTER THE SPECIFIED DEGREE OF DEGRADATION OF THE WEAPON THREAT AND THE DEGREE OF STRUCTURAL DAMAGE, FOR EXAMPLE: "T1S2". See table 2094 which is a composite

9

STATUS VALIDITY

 

 

 

 

9A

STATUS VALIDITY

 

ENTER THE ANALYST'S JUDGEMENT AS TO THE REPORTED STATUS OF THE UNIT, FOR EXAMPLE: "EST". See table 1067/1 which contains a list of data items and associated data codes
