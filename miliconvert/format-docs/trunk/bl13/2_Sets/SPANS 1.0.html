OVERALL BRIDGE LENGTH (FEET, YARDS OR METRES)

2A

ORIENTATION, INTERCARDINAL

 

ENTER THE BRIDGE ORIENTATION USING INTERCARDINAL POINTS, FOR EXAMPLE: "NE". See table 1161/9 which contains a list of data items and associated data codes

3

MILITARY LOAD CLASSIFICATION

 

 

 

 

3A

MILITARY LOAD CLASSIFICATION

 

ENTER THE LOAD CLASSIFICATION CODE OBTAINED FROM POSTED ENGINEERING PLATES OR SIGNS ASSOCIATED WITH WEIGHT BEARING CAPABILITY OF BRIDGES, FERRIES, RAFTS AND VEHICLES, FOR EXAMPLE: "110". See table 1253/1 a range [1 through 150]

4

NUMBER OF SPANS

 

 

 

 

4A

NUMBER OF SPANS

 

ENTER THE TOTAL NUMBER OF SPANS MAKING UP THE BRIDGE BEING REPORTED ON, FOR EXAMPLE: "4". See table 1044/349 a range [1 through 99]

5

NUMBER OF THE SPAN BEING REPORTED ON

 

 

 

 

5A

NUMBER OF THE SPAN BEING REPORTED ON

 

ENTER THE NUMBER OF THE SPAN BEING REPORTED. SPANS WILL BE REPORTED IN SEQUENCE STARTING FROM WEST OR NORTH, FOR EXAMPLE: "6". See table 1012/335 a range [1 through 99]

6

MATERIAL USED IN CONSTRUCTION

 

 

 

 

6A

MATERIAL USED IN CONSTRUCTION

 

ENTER THE CONSTRUCTION MATERIAL CODE DETAILING THE SPAN CONSTRUCTION, FOR EXAMPLE: "BRICK". See table 1592/15 which contains a list of data items and associated data codes

7

SPAN TYPE

 

 

 

 

7A

SPAN TYPE

 

ENTER THE TYPE OF BRIDGE SPAN CONSTRUCTION, FOR EXAMPLE: "BASC". See table 1790/1 which contains a list of data items and associated data codes

8

INDIVIDUAL SPAN LENGTH

 

 

 

 

8A

INDIVIDUAL SPAN LENGTH

 

ENTER THE LENGTH OF AN INDIVIDUAL SPAN IN M, FT OR YD, FOR EXAMPLE: "20M". See table 2454 which is a composite

9

UNDER BRIDGE CLEARANCE

 

 

 

 

9A

UNDER BRIDGE CLEARANCE

 

ENTER THE MAXIMUM CLEAR DISTANCE BETWEEN THE UNDERSIDE OF THE BRIDGE AND THE SURFACE OF THE GROUND OR WATER, FOR EXAMPLE: "25M". See table 2452 which is a composite

10

SPAN DAMAGE

 

 

 

 

10A

SPAN DAMAGE

 

ENTER THE DAMAGE STATUS CODE, FOR EXAMPLE: "NODAM". See table 1854/2 which contains a list of data items and associated data codes

Notes:   

REPEAT FIELDS 5-10 TO DESCRIBE EACH SPAN.  SPAN DESCRIPTION COMMENCES WITH THE SPAN AT THE MOST WESTERLY END OF THE BRIDGE OR, IF BRIDGE ORIENTATION IS NEARLY NORTH-SOUTH, WITH THE SPAN AT THE NORTHERN END.  INFORMATION ON ALL SPANS WILL BE REPORTED IN SEQUENCE.IF CODE "OTR" IS ENTERED IN FIELD 6 OR 7, A FOLLOWING AMPN SET IS TO BE USED TO PROVIDE DESCRIPTION.
