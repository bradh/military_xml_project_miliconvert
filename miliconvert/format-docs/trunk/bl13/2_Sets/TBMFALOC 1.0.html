Concept/Explanation/Examples

1

UNIT IDENTIFIER

 

PROVIDES THE UNIT IDENTIFIER FOR THE UNIT FOR WHICH THE TBMF ALLOCATION APPLIES.

 

 

1A

UNIT DESIGNATOR

UNIT

ENTER "UNIT:" FOLLOWED BY THE NAME OF THE  UNIT, FOR EXAMPLE: "UNIT:ARS GLONS". See table 1028/43 which is an instructive entry

 

 

1B

UNIT IDENTIFICATION CODE (UIC)

UIC

ENTER "UIC:" FOLLOWED BY THE UNIT IDENTIFICATION CODE (UIC) FOR THE UNIT, FOR EXAMPLE: "UIC:FRJ12345". See table 2141 which is a composite

2

DAY-TIME START

 

 

 

 

2A

DAY-TIME START

 

ENTER THE START TIME FOR THE TBMF ALLOCATION, FOR EXAMPLE: "251200Z". See table 2000 which is a composite

3

DAY-TIME STOP

 

PROVIDES THE TERMINATION FOR THE TBMF ALLOCATION USING DAY-TIME OR TIME QUALIFIER.

 

 

3A

DAY-TIME STOP

 

ENTER THE STOP TIME FOR THE TBMF ALLOCATION, FOR EXAMPLE: "251400Z". See table 2000 which is a composite

 

 

3B

TIME QUALIFIER UNTIL FURTHER NOTICE

 

ENTER THE TERMINATION TIME QUALIFIER FOR THE TBMF ALLOCATION, FOR EXAMPLE: "UFN". See table 1220/5 which contains a list of data items and associated data codes

4

TACTICAL BATTLE MANAGEMENT FUNCTION

 

 

 

 

4A

TACTICAL BATTLE MANAGEMENT FUNCTION (TBMF)

 

ENTER THE TACTICAL BATTLE MANAGEMENT FUNCTION ALLOCATION AS PROVIDED BY THE DOCUMENT SDP 10901 "ANGRY HASP" (NATINADS) (NC), USING A 1-3AN CHARACTER STRUCTURE, FOR EXAMPLE: "XXX". See table 1874/10 which is an instructive entry

5

TBMF AREA TYPE

 

 

 

 

5A

TBMF AREA TYPE

 

ENTER THE TBMF AREA TYPE FOR THE SPECIFIC TBMF AS SPECIFIED IN COMAIRNORTH SUPPLAN 24600D "CONSTANT EFFORT", FOR EXAMPLE: "BMA". See table 1246/27 which contains a list of data items and associated data codes

AREA DESIGNATOR

 

ENTER THE AREA DESIGNATOR ASSIGNED TO THE TBMF, FOR EXAMPLE: "BMA2 ARS GLONS". See table 1515/24 which is an instructive entry
