Field Desc

Concept/Explanation/Examples

1

DTG OF ACTIVATION

 

 

 

 

1A

DTG OF ACTIVATION

 

ENTER THE DTG AT WHICH THE AOA BECOMES ACTIVE, FOR EXAMPLE: "090800ZJUN2003". See table 2033 which is a composite

2

DEACTIVATION TIME

 

 

 

 

2A

DTG OF DEACTIVATION

 

ENTER THE DTG AT WHICH THE AOA IS TO BE DEACTIVATED, FOR EXAMPLE: "150800ZJUN2003". See table 2033 which is a composite

 

 

2B

STOP TIME QUALIFIER

 

ENTER THE CODED TIME QUALIFIER FOR WHICH THE AOA IS TO BE DEACTIVATED, FOR EXAMPLE: "UFN". See table 1220/8 which contains a list of data items and associated data codes

3

MAXIMUM ALTITUDE OR FLIGHT LEVEL

 

THE MAXIMUM VERTICAL EXTENT OF THE AIRSPACE WITHIN THE AOA.

 

 

3A

MARITIME ALTITUDE IN HUNDREDS OF FEET

 

ENTER THE MAXIMUM ALTITUDE IN HUNDREDS OF FEET, FOR EXAMPLE: "ALT300". See table 2164 which is a composite

 

 

3B

FLIGHT LEVEL

 

ENTER THE FLIGHT LEVEL DATA QUALIFIER FOLLOWED BY THE FLIGHT LEVEL NUMBER, FOR EXAMPLE: "FL300". See table 2129 which is a composite

4

MAXIMUM DEPTH, FEET OR METRES, DECIMAL PERMITTED

 

 

 

 

4A

MAXIMUM DEPTH, FEET OR METRES, DECIMAL PERMITTED

 

ENTER THE MAXIMUM DEPTH OF THE WATERSPACE WITHIN THE AOA FOLLOWED BY THE UNIT OF DEPTH MEASUREMENT, FOR EXAMPLE: "240M". See table 2264 which is a composite

5

GEOGRAPHIC DEFINITION

 

DEFINES THE GEOGRAPHIC BOUNDARIES OF THE AOA AS A SERIES OF POINTS OR 4W GRID SEGMENTS, REPEATING AS NECESSARY.

 

 

5A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

5B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:"  FOLLOWED BY THE POSITION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

5C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE POSITION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

5D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE POSITION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

5E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE POSITION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

5F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

5G

SEGMENT PARAMETER

SEG

ENTER "SEG:" FOLLOWED BY THE AREA DEFINITION AS A 4W GRID SEGMENT PARAMETER, FOR EXAMPLE: "SEG:JM0205". See table 1527/1 which is an instructive entry
