1

UNIT DESIGNATION

 

ENTER THE DESIGNATION OF THE UNIT PARTICIPATING IN THE DATA LINK USING ONE OF THE FOLLOWING:

 

 

1A

SHIP NAME

SHIP

ENTER "SHIP:" FOLLOWED BY THE SHIP NAME, OMITTING ANY GOVERNMENT DESIGNATION, FOR EXAMPLE: "SHIP:STENNIS JC". See table 1022/49 which is an instructive entry

 

 

1B

BASE NAME

BASE

ENTER "BASE:" FOLLOWED BY THE BASE NAME, FOR EXAMPLE: "BASE:AZORES". See table 1022/104 which is an instructive entry

 

 

1C

PLACE NAME

PLACE

ENTER "PLACE:" FOLLOWED BY THE PLACE NAME, FOR EXAMPLE: "PLACE:GULF OF TONKIN". See table 1022/170 which is an instructive entry

 

 

1D

UNIT

UNIT

ENTER "UNIT:" FOLLOWED BY THE UNIT NAME AS A LITERAL, FOR EXAMPLE: "UNIT:COLBERT". See table 1022/57 which is an instructive entry

 

 

1E

AIRCRAFT BY TYPE, MODEL OR CODE

AC

ENTER "AC:" FOLLOWED BY THE AIRCRAFT TYPE OR MODEL, FOR EXAMPLE: "AC:E3". See table 1015/20 which contains a list of data items and associated data codes

2

CALL SIGN

 

ENTER THE SPECIFIC CALL SIGN OR THE CALL SIGN LINE NUMBER USING ONE OF THE FOLLOWING:

 

 

2A

CALL SIGN

CS

ENTER "CS:" FOLLOWED BY THE SPECIFIC CALL SIGN, FOR EXAMPLE: "CS:CHARLIE ONE". See table 1129/1 which is an instructive entry

 

 

2B

CALL SIGN LINE NUMBER

CSLN

ENTER "CSLN:" FOLLOWED BY THE CALL SIGN LINE NUMBER, FOR EXAMPLE:  "CSLN:9899". See table 1012/679 a range [0001 through 9999]

3

PARTICIPATING/JTIDS UNIT IDENTIFIER

 

ENTER THE PARTICIPATING/JTIDS UNIT IDENTIFIER USING ONE OF THE FOLLOWING:

 

 

3A

PARTICIPATING UNIT (PU) ADDRESS NUMBER

PU

ENTER "PU:" FOLLOWED BY THE PARTICIPATING UNIT NUMBER, FOR EXAMPLE:  "PU:27". See table 1625/23 which is an alphanumeric range

 

 

3B

CRYPTO PUBLICATION LINE NUMBER

LULN

ENTER "LULN:" FOLLOWED BY THE CRYPTO PUBLICATION LINE NUMBER, FOR EXAMPLE:  "LULN:3456". See table 1012/181 a range [0001 through 9999]

 

 

3C

PRIMARY JTIDS UNIT (JU) ADDRESS

JU

ENTER "JU:" FOLLOWED BY THE PRIMARY JTIDS UNIT ADDRESS, FOR EXAMPLE:  "JU:01345". See table 1625/32 which is an alphanumeric range

4

TRACK NUMBER BLOCK OR POOL ALLOCATION

 

ENTER THE TRACK NUMBER BLOCK OR TRACK NUMBER POOL ASSIGNED TO THE UNIT USING ONE OF THE FOLLOWING:

 

 

4A

LINK 11/11B TRACK NUMBER BLOCK

BLOCK

ENTER "BLOCK:" FOLLOWED BY THE TRACK NUMBER BLOCK EXPRESSED AS THE BLOCK LOWER LIMIT, A HYPHEN AND THEN THE BLOCK UPPER LIMIT, FOR EXAMPLE:  "BLOCK:2300-3500". See table 2611 which is a composite

 

 

4B

LINK 11 TRACK NUMBER POOL

POOL

ENTER "POOL:" FOLLOWED BY THE TRACK NUMBER POOL EXPRESSED AS THE POOL LOWER LIMIT, A HYPHEN AND THEN THE POOL UPPER LIMIT, FOR EXAMPLE:  "POOL:2345-4500". See table 2611 which is a composite

SATELLITE LINK 16 EQUIPMENT TYPE

 

ENTER THE DESIGNATION OF THE TYPE OF LINK 16 EQUIPMENT, FOR EXAMPLE: "ADSI". See table 1135/52 which contains a list of data items and associated data codes
