1

UNIT PROVIDING HELICOPTERS

 

 

 

 

1A

UNIT PROVIDING HELICOPTERS

 

ENTER THE NAME OF THE UNIT PROVIDING HELICOPTERS FOR A VERTICAL REPLENISHMNENT OPERATION, FOR EXAMPLE: "INVINCIBLE". See table 1022/121 which is an instructive entry

2

NUMBER OF HELICOPTERS

 

 

 

 

2A

QUANTITY OF HELICOPTERS PROVIDED

 

ENTER THE NUMBER OF HELICOPTERS TO BE PROVIDED BY THE SPECIFIED UNIT, FOR EXAMPLE: "2". See table 1044/46 a range [1 through 9]

3

VERTREP HELICOPTER TYPE

 

 

 

 

3A

VERTICAL REPLENISHMENT HELICOPTER, OTHERS

 

ENTER THE TYPE OF HELICOPTER TO BE PROVIDED BY THE SPECIFIED UNIT, FOR EXAMPLE: "SEAKING". See table 1015/7 which is an instructive entry

4

TYPE/INDICATOR AND QUANTITY OF CARGO OR PERSONNEL

 

PROVIDES THE TYPE/INDICATOR  AND QUANTITY OF CARGO OR PERSONNEL

 

 

4A

TYPE AND QUANTITY OF CARGO

 

ENTER THE TYPE AND QUANTITY OF CARGO TO BE TRANSFERRED, FOR EXAMPLE: "SOLID-SPARE PARTS-2PAL". See table 2450 which is a composite

 

 

4B

PERSONNEL INDICATOR AND QUANTITY

 

ENTER THE INDICATOR AND QUANTITY OF PERSONNEL TO BE TRANSFERRED, FOR EXAMPLE: "PERSONNEL-5". See table 2502 which is a composite

5

UNIT PROVIDING CARGO OR PERSONNEL

 

 

 

 

5A

UNIT PROVIDING CARGO OR PERSONNEL

 

ENTER THE NAME OF THE UNIT PROVIDING CARGO OR PERSONNEL FOR VERTICAL REPLENISHMENT, FOR EXAMPLE: "FRANKFURT". See table 1022/153 which is an instructive entry

6

RECEIVING UNIT NAME

 

 

 

 

6A

RECEIVING UNIT NAME

 

ENTER THE NAME OF THE UNIT TO RECEIVE CARGO OR PERSONNEL VIA A VERTICAL REPLENISHMENT OPERATION, FOR EXAMPLE: "AMSTERDAM". See table 1022/304 which is an instructive entry

7

HELO CONTROL UNIT

 

 

 

 

7A

HELICOPTER CONTROL UNIT

 

ENTER THE NAME OF THE UNIT CONTROLLING THE SPECIFIED HELICOPTER(S), FOR EXAMPLE: "INVINCIBLE". See table 1022/179 which is an instructive entry

8

EMERGENCY LANDING PROVISION

 

 

 

 

8A

EMERGENCY LANDING PROVISION

 

ENTER THE NAME OF THE UNIT OR AREA DESIGNATED AS THE EMERGENCY LANDING PROVISION FOR THE SPECIFIED HELICOPTER(S), FOR EXAMPLE: "BERLIN". See table 1022/182 which is an instructive entry

9

UNIT PROVIDING FUEL

 

 

 

 

9A

UNIT PROVIDING FUEL

 

ENTER THE NAME OF THE UNIT, OTHER THAN THE PARENT UNIT, REQUIRED TO PROVIDE FUELLING FACILITIES FOR THE SPECIFIED HELICOPTER(S), FOR EXAMPLE: "BERLIN". See table 1022/188 which is an instructive entry

10

EMPTY CONTAINER RECOVERY

 

 

 

 

10A

EMPTY CONTAINER RECOVERY

 

ENTER THE REQUIREMENTS FOR RECOVERY OF EMPTY CONTAINERS, FOR EXAMPLE: "RETURN EMPTY PALLETS TO FRANKFURT AFTER LAST LOAD". See table 1941/1 which is an instructive entry
