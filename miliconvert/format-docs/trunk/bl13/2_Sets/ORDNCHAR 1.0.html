ORDNANCE NOMENCLATURE

 

 

 

 

1A

ORDNANCE NOMENCLATURE

 

ENTER THE NAME OF THE  ORDNANCE  INCLUDING POSSIBLE VARIANTS OF THE ORDNANCE, FOR EXAMPLE: "ACTIVE IR, ATG-MISSILE, CONVENTIONAL WARHEAD-AS12." See table 1714/5 which is an instructive entry

2

CONTEXT QUANTITY, 1-999

 

 

 

 

2A

CONTEXT QUANTITY, 1-999

 

ENTER THE NUMBER OF ORDNANCE FOUND, FOR EXAMPLE: "36". See table 1023/3 a range [1 through 999]

3

UNIT OF PACKING MEASUREMENT

 

 

 

 

3A

UNIT OF PACKING MEASUREMENT

 

ENTER THE UNIT OF MEASUREMENT, FOR EXAMPLE: "CSE". See table 1008/59 which contains a list of data items and associated data codes

4

PRODUCTION COUNTRY

 

 

 

 

4A

PRODUCTION COUNTRY

 

ENTER THE PRODUCTION COUNTRY USING DATA CODES IN ACCORDANCE WITH STANAG 1059, FOR EXAMPLE: "KN". See table 1026/21 which contains a list of data items and associated data codes

5

ORDNANCE MANUFACTURER

 

 

 

 

5A

ORDNANCE MANUFACTURER

 

MAKE A LITERAL ENTRY NAMING THE MANUFACTURER OF THE ORDNANCE. See table 1022/239 which is an instructive entry

6

FACTORY OF ORIGIN

 

 

 

 

6A

FACTORY OF ORIGIN

 

ENTER THE NAME OF THE FACTORY IN WHICH THE ORDNANCE WAS PRODUCED. See table 1022/237 which is an instructive entry
