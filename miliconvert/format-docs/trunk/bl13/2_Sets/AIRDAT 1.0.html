Designator

Concept/Explanation/Examples

1

NUMBER OF AIRCRAFT EMBARKED

 

 

 

 

1A

NUMBER OF AIRCRAFT EMBARKED

 

ENTER THE NUMBER OF EMBARKED AIRCRAFT OF THE SPECIFIED TYPE, FOR EXAMPLE: "20". See table 1044/232 a range [0 through 99]

2

AIRCRAFT TYPE/MODEL

 

THE AIRCRAFT TYPE DESCRIBED USING ONE OF THE FOLLOWING:

 

 

2A

AIRCRAFT BY TYPE, MODEL OR CODE

 

ENTER THE AIRCRAFT TYPE OR MODEL, FOR EXAMPLE: "HASMK8". See table 1015/20 which contains a list of data items and associated data codes

 

 

2B

AIRCRAFT TYPE

OTHAC

ENTER "OTHAC" FOLLOWED BY THE AIRCRAFT TYPE OR MODEL, FOR EXAMPLE: "OTHAC:SH14C". See table 1015/1 which is an instructive entry

3

NUMBER OF PASSENGERS

 

 

 

 

3A

NUMBER OF PASSENGERS

 

ENTER THE NUMBER OF PASSENGERS THAT MAY BE CARRIED, FOR EXAMPLE: "400". See table 1044/88 a range [1 through 9999]

4

NUMBER OF STRETCHER CASES

 

 

 

 

4A

NUMBER OF STRETCHER PATIENTS

 

ENTER THE NUMBER OF STRETCHER CASES THAT MAY BE CARRIED, FOR EXAMPLE: "20". See table 1044/172 a range [1 through 9999]

5

WEIGHT OF INTERNAL STORES (KG)

 

 

 

 

5A

WEIGHT OF INTERNAL STORES IN KILOGRAMS

 

ENTER THE WEIGHT OF INTERNAL STORES IN KILOGRAMS, FOR EXAMPLE: "250". See table 1194/12 a range [0 through 99999]

6

WEIGHT OF EXTERNAL STORES (KG)

 

 

 

 

6A

WEIGHT OF EXTERNAL STORES IN KILOGRAMS

 

ENTER THE WEIGHT OF EXTERNAL STORES IN KILOGRAMS, FOR EXAMPLE: "500". See table 1194/13 a range [0 through 99999]

7

PERSONNEL RADHAZ DESIGNATOR

 

 

 

 

7A

PERSONNEL RADHAZ DESIGNATOR

 

ENTER THE PERSONNEL RADHAZ DESIGNATOR FOR THE EMBARKED AIRCRAFT, FOR EXAMPLE: "10-10-10-10-5-5". See table 2609 which is a composite

8

AIRCRAFT SRAD CODE

 

 

 

 

8A

PLATFORM SRAD CODE

 

ENTER THE AIRCRAFT SRAD CODE, FOR EXAMPLE: "1213054". See table 2604 which is a composite

9

AIRCRAFT LIMITATIONS

 

 

 

 

9A

AIRCRAFT LIMITATIONS

 

ENTER ANY LIMITATIONS ON AIRCRAFT, AIRCREW RESTRICTIONS, FLIGHT DECK LIMITATIONS ETC., FOR EXAMPLE: "AIRCRAFT DUE 100HR CHECK". See table 1357/16 which is an instructive entry
