No

Designator

Field Desc

Concept/Explanation/Examples

1

DATE-TIME

 

SPECIFIES THE EFFECTIVE DATE-TIME.

 

 

1A

DAY-TIME EFFECTIVE

 

ENTER THE DAY, HOUR, MINUTE AND TIME ZONE, FOR EXAMPLE: "271430Z". See table 2000 which is a composite

 

 

1B

DATE-TIME MONTH, VERIFIED

 

ENTER THE DAY, HOUR, MINUTES, TIME ZONE, CHECK DIGIT AND MONTH, FOR EXAMPLE: "151000Z7OCT". See table 2032 which is a composite

2

EFFECTIVE MONTH

 

CAN ONLY BE USED IN CONJUNCTION WITH DATE-TIME EFFECTIVE.

 

 

2A

MONTH NAME, ABBREVIATED

 

ENTER THE EFFECTIVE MONTH ABBREVIATED NAME, FOR EXAMPLE: "JAN". See table 1004/1 which contains a list of data items and associated data codes
