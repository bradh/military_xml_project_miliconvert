Concept/Explanation/Examples

1

AREA NAME

 

 

 

 

1A

AREA NAME

 

ENTER THE PHONETICISED SEQUENTIAL NUMBER WHICH IDENTIFIES THE "NOIAREA" THROUGHOUT THE "SUBNOI", FOR EXAMPLE: "TWO". See table 1022/69 which is an instructive entry

2

DATE-TIME MONTH OF START, VERIFIED

 

 

 

 

2A

DATE-TIME MONTH OF START, VERIFIED

 

PROVIDES THE DATE-TIME, VERIFIED, AT WHICH THE AREA BECOMES EFFECTIVE, FOR EXAMPLE: "151500Z2DEC". See table 2032 which is a composite

3

STOP DATE-TIME VERIFIED.

 

PROVIDES THE DATE-TIME, VERIFIED, AT WHICH THE AREA CEASES TO BE EFFECTIVE, SPECIFIED IN ONE OF THE TWO FOLLOWING WAYS:

 

 

3A

DATE-TIME MONTH OF STOP, VERIFIED

 

ENTER THE STOP DATE-TIME AND MONTH, VERIFIED, FOR EXAMPLE: "171800Z7DEC". See table 2032 which is a composite

 

 

3B

TIME QUALIFIER UNTIL FURTHER NOTICE

 

ENTER THE TIME QUALIFIER UNTIL FURTHER NOTICE, FOR EXAMPLE: "UFN". See table 1220/5 which contains a list of data items and associated data codes

4

AREA DEFINITION

 

PROVIDES THE AREA DEFINITION INFORMATION.

 

 

4A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

4B

UNIVERSAL TRANSVERSE MERCATOR (UTM)

UTM

ENTER "UTM:" FOLLOWED BY THE LOCATION USING THE UTM COORDINATE SYSTEM, FOR EXAMPLE: "UTM:32N2985945585243". See table 2572 which is a composite

 

 

4C

MILITARY GRID REFERENCE SYSTEM (UTM) (MGRS-UTM)

MGRS

ENTER "MGRS:" FOLLOWED BY THE LOCATION USING THE MGRS (UTM) COORDINATE SYSTEM, FOR EXAMPLE: "MGRS:32UKA9859485243". See table 2573 which is a composite

 

 

4D

UNIVERSAL POLAR STEREOGRAPHIC (UPS)

UPS

ENTER "UPS:" FOLLOWED BY THE LOCATION USING THE UPS COORDINATE SYSTEM, FOR EXAMPLE: "UPS:N20450002245522". See table 2574 which is a composite

 

 

4E

MILITARY GRID REFERENCE SYSTEM (UPS) (MGRS-UPS)

MUPS

ENTER "MUPS:" FOLLOWED BY THE LOCATION USING THE MGRS (UPS) COORDINATE SYSTEM, FOR EXAMPLE: "MUPS:ZAK4500045522". See table 2575 which is a composite

 

 

4F

NATIONAL GRID SYSTEM COORDINATES

GRID

ENTER "GRID:" FOLLOWED BY THE LOCATION USING A NATIONAL COORDINATE SYSTEM, FOR EXAMPLE: "GRID:SU654345". See table 1911/1 which is a regular expression

 

 

4G

SUBMARINE GRID SYSTEM AREA DESIGNATOR

SGSA

ENTER THE DESIGNATOR OF THE AREA, SPECIFIED IN THE SUBMARINE GRID SYSTEM, FOR EXAMPLE: "SGSA:ATLANT34NW". See table 1499/1 which is an instructive entry

 

 

4H

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES VERIFIED

SUB

ENTER "SUB:" FOLLOWED BY THE LATITUDE AND LONGITUDE OF THE POSITION UP TO 1/10,000 OF A MINUTE, VERIFIED, FOR EXAMPLE: "SUB:2010.3000N6-01040.2000W7". See table 2577 which is a composite
