No

Designator

Field Desc

Concept/Explanation/Examples

1

SHIP NAME

 

THE NAME OR IDENTIFICATION OF THE SHIP.

 

 

1A

SHIP NAME

NAME

ENTER "NAME:" FOLLOWED BY THE NAME OF THE SHIP, FOR EXAMPLE: "NAME:HR MS DE RUYTER". See table 1022/49 which is an instructive entry

 

 

1B

SHIP IDENTIFICATION

ID

ENTER "ID:" FOLLOWED BY THE SHIP IDENTIFICATION, FOR EXAMPLE: "ID:DD-963". See table 2021 which is a composite

2

DAY-TIME OF INCIDENT

 

THE TIME OF THE LOSS OF THE SHIP OR TIME OF DEPARTURE FROM THE LAST KNOWN LOCATION.

 

 

2A

DAY-TIME OF INCIDENT

 

THE TIME OF THE LOSS OR TIME OF DEPARTURE FROM THE LAST KNOWN LOCATION, FOR EXAMPLE: "221104Z". See table 2000 which is a composite

3

INCIDENT LOCATION

 

THE LAST KNOWN GEOGRAPHIC LOCATION OF THE SHIP.

 

 

3A

LATITUDE AND LONGITUDE, MINUTES, 0-4 DECIMAL PLACES

 

ENTER THE LOCATION LATITUDE AND LONGITUDE, FOR EXAMPLE: "4520.3500N-02126.1500E". See table 2571 which is a composite

 

 

3B

BEARING AND DISTANCE FROM REFERENCE POINT

BAR

ENTER "BAR:" FOLLOWED BY THE BEARING AND DISTANCE FROM REFERENCE POINT, FOR EXAMPLE: "BAR:087.5T-SEMACH-99.5FT". See table 2157 which is a composite

 

 

3C

LOCATION OR POSITION NAME

NAME

ENTER "NAME:" FOLLOWED BY THE INCIDENT LOCATION NAME, FOR EXAMPLE: "NAME:PT BARROW". See table 1022/6 which is an instructive entry

4

RADIUS, FT, YD, NM, M, KM, KYD

 

THE RADIUS OF THE CIRCLE WHERE THE SHIP IS LOCATED.

 

 

4A

RADIUS, IN, FT, YD, KF, SM, NM, MM, CM, M, KM, HM

 

ENTER THE RADIUS OF THE CIRCLE WHERE THE SHIP IS LOCATED, FOR EXAMPLE: "22NM". See table 2490 which is a composite

5

LOCATION QUALIFIER

 

THE ACTUAL OR THE ESTIMATED LOCATION OF THE SHIP.

 

 

5A

LOCATION QUALIFIER

 

THE ACTUAL OR THE ESTIMATED LOCATION, FOR EXAMPLE: "EE". See table 1118/1 which contains a list of data items and associated data codes

6

SHIP CALL SIGN

 

THE CALL SIGN OF THE LOST SHIP OR SHIP IN DISTRESS.

 

 

6A

SHIP CALL SIGN

 

THE CALL SIGN OF THE LOST SHIP OR SHIP IN DISTRESS, FOR EXAMPLE: "BANG 11". See table 1129/7 which is an instructive entry

7

COLOUR

 

THE COLOUR OF THE LOST SHIP OR SHIP IN DISTRESS.

 

 

7A

COLOUR

 

THE COLOUR OF THE LOST SHIP OR SHIP IN DISTRESS, FOR EXAMPLE: "BLUE". See table 1138/1 which contains a list of data items and associated data codes

8

COUNT OF PERSONNEL ON BOARD

 

THE NUMBER OF PERSONNEL ON BOARD OF THE SHIP.

 

 

8A

COUNT OF PERSONNEL ON BOARD

 

ENTER THE TOTAL NUMBER OF PERSONNEL ON BOARD OF THE SHIP, FOR EXAMPLE: "550". See table 1044/132 a range [0 through 999]

9

DATA QUALIFIER

 

PROVIDES THE DATA QUALIFICATION OF THE REPORTED DATA CONCERNING THE SHIP INVOLVED IN THE LOSS.  MUST BE USED IF COUNT OF PERSONNEL ON BOARD IS SPECIFIED WITHIN THIS SET.

 

 

9A

DATA QUALIFIER

 

ENTER THE DATA QUALIFIER CONCERNING THE REPORTED NUMBER OF PERSONNEL ON BOARD OF THE SHIP, FOR EXAMPLE: "ACT". See table 1130/2 which contains a list of data items and associated data codes
