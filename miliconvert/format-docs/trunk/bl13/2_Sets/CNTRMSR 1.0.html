No

Designator

Field Desc

Concept/Explanation/Examples

1

FRIENDLY COUNTERMEASURES EMPLOYED

 

PROVIDES THE TYPE OF COUNTERMEASURES USED BY THE MISSION AIRCRAFT DURING ENEMY ENGAGEMENT.

 

 

1A

FRIENDLY COUNTERMEASURES EMPLOYED

 

ENTER THE TYPE OF COUNTERMEASURES USED BY THE MISSION AIRCRAFT DURING ENEMY ENGAGEMENT, FOR EXAMPLE "BEAM". See table 1643/7 which contains a list of data items and associated data codes

2

COUNTERMEASURES EMPLOYED, CHAFF

 

PROVIDES AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR CHAFF EMPLOYED.

 

 

2A

COUNTERMEASURES EMPLOYED, CHAFF

CHAFF

ENTER "CHAFF:" FOLLOWED BY AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR CHAFF EMPLOYED, FOR EXAMPLE "CHAFF:YES". See table 1027/116 which contains a list of data items and associated data codes

3

COUNTERMEASURES EMPLOYED, ELECTRONIC ATTACK

 

PROVIDES AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR ELECTRONIC ATTACK EMPLOYED.

 

 

3A

COUNTERMEASURES EMPLOYED, ELECTRONIC ATTACK

EA

ENTER "EA:" FOLLOWED BY AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR ELECTRONIC ATTACK EMPLOYED, FOR EXAMPLE: "EA:YES". See table 1027/117 which contains a list of data items and associated data codes

4

COUNTERMEASURES EMPLOYED, FLARES

 

PROVIDES AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR FLARES EMPLOYED.

 

 

4A

COUNTERMEASURES EMPLOYED, FLARES

FLARE

ENTER "FLARE:", FOLLOWED BY AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR FLARES EMPLOYED, FOR EXAMPLE "FLARE:YES". See table 1027/118 which contains a list of data items and associated data codes

5

FLARE PROGRAM EMPLOYED

 

PROVIDES THE TYPE OF FLARE PROGRAM SETTING USED DURING ENEMY ENGAGEMENT.  CAN ONLY BE USED IF FLARES WERE EMPLOYED.

 

 

5A

FLARE PROGRAM EMPLOYED

FP

ENTER "FP:", FOLLOWED BY THE TYPE OF FLARE PROGRAM SETTING USED DURING ENEMY ENGAGEMENT, FOR EXAMPLE "FP:POPUP AG SEQ. CHAFF-FLARE". See table 1643/8 which is an instructive entry

6

COUNTERMEASURES EMPLOYED, SELF PROTECTION JAMMING

 

PROVIDES AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR SELF PROTECTION JAMMING EMPLOYED.

 

 

6A

COUNTERMEASURES EMPLOYED, SELF PROTECTION JAMMING

SPJ

ENTER "SPJ:" FOLLOWED BY AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION FOR SELF PROTECTION JAMMING EMPLOYED, FOR EXAMPLE "SPJ:YES". See table 1027/119 which contains a list of data items and associated data codes

7

COUNTERMEASURES EMPLOYED, OTHER

 

PROVIDES THE TYPE OF OTHER COUNTERMEASURES USED DURING ENEMY ENGAGEMENT.

 

 

7A

COUNTERMEASURES EMPLOYED, OTHER

OTH

ENTER "OTH:", FOLLOWED BY THE TYPE OF OTHER COUNTERMEASURES USED DURING ENEMY ENGAGEMENT, FOR EXAMPLE: "OTH:RANGE GATE PULL-OFF". See table 1643/9 which is an instructive entry

8

DESCRIPTION OF FRIENDLY COUNTERTACTICS

 

IF USED, PROVIDES A DESCRIPTION OF THE COUNTERTACTICS OR COUNTERMEASURES USED DURING ENGAGEMENT.

 

 

8A

DESCRIPTION OF FRIENDLY COUNTERTACTICS

CNTRTAC

ENTER "CNTRTAC:", FOLLWED BY A DESCRIPTION OF THE COUNTERTACTICS USED DURING ENGAGEMENT, FOR EXAMPLE "CNTRTAC:HIGH G OUT OF PLANE MANOEUVRE". See table 1363/29 which is an instructive entry

 

 

8B

DESCRIPTION OF COUNTERMEASURES TAKEN

CNTRMES

ENTER "CNTRMES:", FOLLOWED BY A DESCRIPTION OF THE COUNTERMEASURES USED DURING THE ENGAGEMENT, FOR EXAMPLE "CNTRMES:IR DFCM". See table 1363/30 which is an instructive entry

9

COUNTERMEASURES EFFECTIVE

 

PROVIDES AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION WHETHER COUNTERMEASURES TAKEN WERE EFFECTIVE.

 

 

9A

COUNTERMEASURES EFFECTIVE

 

ENTER AN INDICATOR OF AN AFFIRMATIVE OR A NEGATIVE CONDITION WHETHER COUNTERMEASURES TAKEN WERE EFFECTIVE, FOR EXAMPLE "NO". See table 1027/120 which contains a list of data items and associated data codes

10

BATTLE DAMAGE RECEIVED INDICATOR

 

PROVIDES AN INDICATOR AS TO WHETHER FRIENDLY AIRCRAFT RECEIVED BATTLE DAMAGE DURING ENEMY ENGAGEMENT.

 

 

10A

BATTLE DAMAGE RECEIVED INDICATOR

DAMAGE

ENTER "DAMAGE:", FOLLOWED BY AN INDICATOR AS TO WHETHER FRIENDLY AIRCRAFT RECEIVED BATTLE DAMAGE DURING ENEMY ENGAGEMENT, FOR EXAMPLE "DAMAGE:YES". See table 1027/53 which contains a list of data items and associated data codes
