Concept/Explanation/Examples

1

PLANNED OPERATIONAL COMMAND/UIC

 

ENTER THE COMMAND PLANNED TO ASSUME OPERATIONAL COMMAND OF THE REPORTED UNIT IN ONE OF TWO WAYS:

 

 

1A

PLANNED OPERATIONAL COMMAND

 

ENTER AS A LITERAL NAME, FOR EXAMPLE: "LANDCENT" OR "2 FLEET". See table 1028/4 which is an instructive entry

 

 

1B

PLANNED OPERATIONAL COMMAND UIC

UIC

ENTER  "UIC:" FOLLOWED BY THE NATIONALITY CODE, ARMED SERVICE CODE AND ASSIGNED SEQUENTIAL LOCATION NUMBER, FOR EXAMPLE: "UIC:CAA3957". See table 2141 which is a composite

2

PLANNED OPERATIONAL CONTROL/UIC

 

ENTER THE COMMAND PLANNED TO ASSUME OPERATIONAL CONTROL OF THE REPORTED UNIT IN ONE OF TWO WAYS:

 

 

2A

PLANNED OPERATIONAL CONTROL

 

ENTER AS A LITERAL NAME, FOR EXAMPLE: "5CORPS" OF "TF405.1". See table 1028/5 which is an instructive entry

 

 

2B

PLANNED OPERATIONAL CONTROL UIC

UIC

ENTER  "UIC:" FOLLOWED BY THE NATIONALITY CODE, ARMED SERVICE CODE AND ASSIGNED SEQUENTIAL LOCATION NUMBER, FOR EXAMPLE: "UIC:BEF4527". See table 2141 which is a composite

3

PLANNED TASK DESIGNATOR

 

 

 

 

3A

PLANNED TASK DESIGNATOR

 

IF REPORTING UNDER THE ACEREP SYSTEM, THIS FIELD MUST BE COMPLETED.  IF USED, ENTER THE LITERAL NUMBER OF THE OPERATIONAL/CONTINGENCY PLAN IN WHICH THE REPORTED UNIT IS INVOLVED, FOR EXAMPLE: "10405". See table 1037/2 which is an instructive entry

4

NATIONAL PARENT IDENTIFIER/UIC

 

IF REPORTING UNDER THE ACEREP SYSTEM, THIS FIELD MUST BE COMPLETED.  IF USED, ENTER THE COMMAND OF IMMEDIATE SUPERIOR ORGANIZATION IN THE NATIONAL CHAIN OF COMMAND OF THE REPORTED UNIT BEFORE TRANSFER OF OPERATION COMMAND OR CONTROL, IN ONE OF TWO WAYS:

 

 

4A

NATIONAL PARENT IDENTIFIER

 

ENTER AS A LITERAL NAME, FOR EXAMPLE: "3GE" OR "6(US) FLEET". See table 1028/6 which is an instructive entry

 

 

4B

NATIONAL PARENT UIC

UIC

ENTER "UIC:" FOLLOWED BY THE NATIONALITY CODE, ARMED SERVICE CODE AND ASSIGNED SEQUENTIAL LOCATION NUMBER, FOR EXAMPLE: "UIC:GEA7000". See table 2141 which is a composite

5

NATO PARENT IDENTIFIER/UIC

 

IF REPORTING UNDER THE ACEREP SYSTEM, THIS FIELD MUST BE COMPLETED.  IF USED, ENTER THE COMMAND OF IMMEDIATE SUPERIOR ORGANIZATION IN THE NATO CHAIN OF COMMAND OF THE REPORTED UNIT AFTER TRANSFER OF OPERATION COMMAND OR CONTROL, IN ONE OF TWO WAYS:

 

 

5A

NATO PARENT IDENTIFIER

 

ENTER AS A LITERAL NAME, FOR EXAMPLE: "3GE" OR "COMUKNLLF". See table 1028/7 which is an instructive entry

 

 

5B

NATO PARENT UIC

UIC

ENTER "UIC:" FOLLOWED BY THE NATIONALITY CODE, ARMED SERVICE CODE AND ASSIGNED SEQUENTIAL LOCATION NUMBER, FOR EXAMPLE: "UIC:GEA7000". See table 2141 which is a composite

6

PLANNED TOA COMMAND

 

 

 

 

6A

PLANNED TOA COMMAND

 

IF REPORTING UNDER THE ACEREP SYSTEM, THIS FIELD MUST BE COMPLETED.  IF USED, ENTER THE CODE FOR REPORTING TIME OF TRANSFER OF AUTHORITY TO SACEUR, FOR EXAMPLE: "RE". See table 1071/2 which is an instructive entry
