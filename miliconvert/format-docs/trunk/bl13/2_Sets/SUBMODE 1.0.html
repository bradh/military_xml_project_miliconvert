Concept/Explanation/Examples

1

TRACK IDENTIFIER

 

ENTER ONE OF THE FOLLOWING:

 

 

1A

AREA PHONETIC NUMERIC IDENTIFIER

 

THE PHONETIC NUMERIC IDENTIFIER CODE FOR AREA DESCRIPTION, FOR EXAMPLE: "SIX". See table 1128/9 which contains a list of data items and associated data codes

 

 

1B

AREA PHONETIC ALPHABETIC IDENTIFIER

 

THE PHONETIC ALPHABETIC IDENTIFIER CODE FOR AREA DESCRIPTION, FOR EXAMPLE: "BRAVO". See table 1128/10 which contains a list of data items and associated data codes

 

 

1C

DUAL SUBMARINE TRACK IDENTIFIERS

 

THAT PART OF THE SUBMARINE'S TRACK FOR WHICH THE MODE OF OPERATION IS APPLICABLE.  POINT TO POINT ROUTING IS TO BE USED.  THE START AND END POSITION FOR A SPECIFIC MODE OF OPERATION ARE TO BE LISTED SEPARATELY BY A DASH, FOR EXAMPLE: "AB-CD". See table 2198 which is a composite

2

SUBMARINE OPERATING MODE

 

 

 

 

2A

SUBMARINE OPERATING MODE

 

ENTER THE CODE TO INDICATE THE MODE OF OPERATION, FOR EXAMPLE: "SUBMERGED". See table 1056/1 which contains a list of data items and associated data codes

3

UPPER DEPTH LIMIT

 

ENTER ONE OF THE FOLLOWING:

 

 

3A

UPPER DEPTH LIMIT, VERIFIED

 

ENTER THE VERIFIED UPPER LIMIT (KEEL DEPTH) IN FEET OR METRES, ENTER THE LETTER"U" FOLLOWED BY THE DEPTH DESCRIPTION, FOR EXAMPLE: "U23FT5". See table 2192 which is a composite

 

 

3B

NO CEILING UNIT IDENTIFIER

 

IF NO LOWER DEPTH LIMIT IS REQUIRED, ENTER THE NO CEILING IDENTIFIER: "NONE". See table 1246/12 which contains a list of data items and associated data codes

4

LOWER DEPTH LIMIT, VERIFIED

 

 

 

 

4A

LOWER DEPTH LIMIT, VERIFIED

 

ENTER THE LOWER LIMIT (KEEL DEPTH) IN FEET OR METRES.  ENTER THE LETTER "L" FOLLOWED BY THE DEPTH LIMIT, FOR EXAMPLE: "L500FT5". See table 2191 which is a composite
