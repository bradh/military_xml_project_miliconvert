NAME OF PROVIDING UNIT

 

ENTER THE NAME OF THE UNIT THAT IS PROVIDING THE HELICOPTER, FOR EXAMPLE "USS JOHN C STENNIS". See table 1022/308 which is an instructive entry

HELICOPTER CALL SIGN

 

 

 

 

2A

HELICOPTER CALL SIGN

 

ENTER THE CALL SIGN OF THE HELICOPTER THAT WILL CARRY OUT THE TASK, FOR EXAMPLE "DE612". See table 1129/29 which is an instructive entry

PRIMARY FREQUENCY IN MEGAHERTZ

 

ENTER THE PRIMARY RADIO FREQUENCY ON WHICH THE HELICOPTER CAN BE CONTACTED, FOR EXAMPLE 225.4". See table 1068/3 which is a regular expression
