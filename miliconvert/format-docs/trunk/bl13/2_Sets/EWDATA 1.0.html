No

Designator

Field Desc

Concept/Explanation/Examples

1

EMITTER IDENTIFICATION

 

PROVIDES THE IDENTIFICATION OF THE EMITTER, USING EITHER ITS ASSIGNED NAME OR THE TYPE OF EQUIPMENT THAT IS THE SOURCE OF THE EMITTED SIGNAL.

 

 

1A

EMITTER NAME

NAME

ENTER "NAME:" FOLLOWED BY THE ASSIGNED NAME OF A PARTICULAR EMITTER, OPERATION OR STATION FOR EXAMPLE: "NAME:BIG BIRD". See table 1022/225 which is an instructive entry

 

 

1B

EMITTER TYPE

TYPE

ENTER "TYPE:" FOLLOWED BY THE TYPE OF EQUIPMENT THAT IS THE SOURCE OF A PARTICULAR EMITTED SIGNAL, FOR EXAMPLE: "TYPE:GCIRDR". See table 1135/26 which contains a list of data items and associated data codes

2

ELINT NOTATION OR SORTING CODE

 

PROVIDES THE IDENTIFIER OF A SPECIFIC EMITTER THAT IS TO BE THE TARGET OF ELECTRONIC WARFARE.

 

 

2A

ELINT NOTATION

E

ENTER "E:" FOLLOWED BY A 5-CHARACTER ELINT NOTATION FOR NON-COMMUNICATIONS ELECTRONICS EMISSIONS RELATED TO SUCH EQUIPMENT. THE ELINT NOTATION IS EXTRACTED FROM THE LATEST EDITION OF THE ELINT PARAMETER LIST, FOR EXAMPLE: "E:XXXXX". See table 1012/34 which is an instructive entry

 

 

2B

SORTING CODE

S

ENTER "S:" FOLLOWED BY A 4-CHARACTER CODE WHICH FACILITATES REPORTING EQUIPMENT TYPE THAT HAS NO ASSOCIATED ELINT NOTATION, FOR EXAMPLE: "S:1234". See table 1012/76 which is an instructive entry

3

RADIO/RADAR FUNCTION

 

 

 

 

3A

RADIO RADAR FUNCTION

 

ENTER THE CODE FOR THE PRIMARY FUNCTION OF A RADIO/RADAR SITE, FOR EXAMPLE: "GI". See table 1471/1 which contains a list of data items and associated data codes

4

LOWER FREQUENCY

 

PROVIDES THE LOWER FREQUENCY OF THE RANGE OF FREQUENCIES OR THE SPECIFIC FREQUENCY OF THE EMITTER.

 

 

4A

LOWER RADIO FREQUENCY LIMIT

F

ENTER "F:" FOLLOWED BY THE LOWER LIMITS OF THE ELECTRONIC ATTACK FREQUENCY, FOR EXAMPLE: "F:365.235MHZ". See table 2383 which is a composite

 

 

4B

LOWER FREQUENCY IN PULSES PER SECOND

P

ENTER "P:" FOLLOWED BY THE LOWEST FREQUENCY IN A RANGE,  IN PULSES PER SECOND, FOR EXAMPLE: "P:365". See table 1176/1 a range [0 through 99999999.0000000] (0 to 7 decimal places)

5

UPPER FREQUENCY

 

PROVIDES THE UPPER FREQUENCY OF THE RANGE OF FREQUENCIES OR THE SPECIFIC FREQUENCY OF THE EMITTER BY USING THE CORRESPONDING ALTERNATIVE FROM THE LOWER FREQUENCY.

 

 

5A

UPPER RADIO FREQUENCY LIMIT

F

ENTER "F:" FOLLOWED BY THE UPPER LIMIT OF THE ELECTRONIC ATTACK FREQUENCY, FOR EXAMPLE: "F:1525.25MHZ". See table 2383 which is a composite

 

 

5B

UPPER FREQUENCY IN PULSES PER SECOND

P

ENTER "P:" FOLLOWED BY THE HIGHEST FREQUENCY IN A RANGE,  IN PULSES PER SECOND, FOR EXAMPLE: "P:1580". See table 1176/2 a range [0 through 99999999.0000000] (0 to 7 decimal places)

6

EA TECHNIQUE

 

 

 

 

6A

ELECTRONIC ATTACK (EA) TECHNIQUE

 

ENTER THE ELECTRONIC ATTACK (EA) TECHNIQUE EMPLOYED AGAINST COMMUNICATION AND NON-COMMUNICATION ELECTRONICS, FOR EXAMPLE: "JAMBAR". See table 1643/3 which contains a list of data items and associated data codes
