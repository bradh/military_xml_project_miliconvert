Designator

Field Desc

Concept/Explanation/Examples

1

START DAY-TIME

 

ENTER THE START DAY-TIME USING ONE OF THE FOLLOWING:

 

 

1A

DAY-TIME START

 

ENTER THE DAY-TIME, FOR EXAMPLE: "221545Z". See table 2000 which is a composite

 

 

1B

DAY-TIME AND MONTH OF START

 

ENTER THE DAY-TIME AND MONTH OF START, FOR EXAMPLE: "221545ZJUN". See table 2030 which is a composite

 

 

1C

DTG OF START

 

ENTER THE DATE (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE, MONTH AND YEAR (0001-9999), FOR EXAMPLE: "041405ZAPR1995". See table 2033 which is a composite

2

STOP DAY-TIME

 

ENTER THE STOP DAY-TIME USING ONE OF THE FOLLOWING:

 

 

2A

DAY-TIME STOP

 

ENTER THE DAY-TIME, FOR EXAMPLE: "130800Z". See table 2000 which is a composite

 

 

2B

DAY-TIME AND MONTH OF STOP

 

ENTER THE DAY-TIME AND MONTH OF STOP, FOR EXAMPLE: "130800ZJUL". See table 2030 which is a composite

 

 

2C

STOP TIME QUALIFIER

 

ENTER THE STOP TIME QUALIFIER CODE, FOR EXAMPLE: "UFN". See table 1220/8 which contains a list of data items and associated data codes

 

 

2D

DTG OF STOP

 

ENTER THE DATE (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE, MONTH AND YEAR (0001-9999), FOR EXAMPLE: "041405ZAPR1995". See table 2033 which is a composite

3

QUALIFIED STOP TIME

 

IF REQUIRED BY CONDITIONAL LANGUAGE, ENTER THE QUALIFIED STOP TIME IN ONE OF THE FOLLOWING WAYS:  MUST BE USED ONLY IF FIELD 2 EQUALS "AFTER" OR "ASOF" OR "ASAPAFT" OR "ASAPNLT" OR "BEFORE" OR "NET" OR "NLT".

 

 

3A

DAY-TIME

 

ENTER THE DAY-TIME, FOR EXAMPLE: "201050Z". See table 2000 which is a composite

 

 

3B

DAY-TIME AND MONTH

 

ENTER THE DAY-TIME AND MONTH, FOR EXAMPLE: "201050ZDEC". See table 2030 which is a composite

 

 

3C

DTG

 

ENTER THE DATE (01-31), HOUR (00-23), MINUTE (00-59), TIME ZONE, MONTH AND YEAR (0001-9999), FOR EXAMPLE: "041405ZAPR1995". See table 2033 which is a composite

 

 

Structured Notation:

[^]F3 RP ([^]F2 EQ /AFTER/ | /ASOF/ | /ASAPAFT/ | /ASAPNLT/ | /BEFORE/ | /NET/ | /NLT/)

 

 

Structured Language:      

A QUALIFIED STOP TIME IS REQUIRED IF THE STOP TIME QUALIFIER EQUALS EITHER "AFTER" OR "ASOF" OR "ASAPAFT" OR "ASAPNLT" OR "BEFORE" OR "NET" OR "NLT".
