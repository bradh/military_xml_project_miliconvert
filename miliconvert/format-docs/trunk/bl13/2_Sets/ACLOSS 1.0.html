CALL SIGN

AIRCRAFT BY TYPE, MODEL OR CODE

 

PPROVIDES THE TYPE, MODEL OR CODE OF THE LOST AIRCRAFT.

AIRCRAFT BY TYPE, MODEL OR CODE

ACTYP

ENTER "ACTYP:" FOLLOWED BY THE AIRCRAFT CODE  FOR THE AIRCRAFT LOST,  FOR EXAMPLE: "ACTYP:MRC". See table 1015/20 which contains a list of data items and associated data codes

OTHAC

ENTER "OTHAC:", FOLLOWED BY THE AIRCRAFT CODE  FOR THE AIRCRAFT LOST,  FOR EXAMPLE: "OTHAC:SU47". See table 1015/18 which is an instructive entry
