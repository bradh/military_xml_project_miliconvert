ACTION ORGANISATION

 

SPECIFIES THE ORGANISATION THAT IS AUTHORISED TO IMPLEMENT ASSOCIATED ROE(s) IN ONE OF THE THREE FOLLOWING WAYS, REPEATING AS NECESSARY:

 

 

1A

TASK ORGANIZATION COMMANDER

TOG

ENTER "TOG:" FOLLOWED BY THE  CODE SPECIFYING THE TASK ORGANISATION COMMANDER, FOR EXAMPLE: "TOG:CJTF402". See table 1037/3 which is an instructive entry

 

 

1B

REACTION FORCE DESIGNATOR

MSC

ENTER "MSC:"FOLLOWED BY A CODE SPECIFYING A NATO MAJOR SUBORDINATE COMMANDER, FOR EXAMPLE: "CINCENT". See table 1515/15 which contains a list of data items and associated data codes

 

 

1C

COMMAND IDENTIFIER

CMD

ENTER "CMD:", FOLLOWED BY A LITERAL COMMAND IDENTIFIER. See table 1380/6 which is an instructive entry
