LANGUAGE

 

 

 

 

1A

LANGUAGE

 

IDENTIFIES THE NAME OF A LANGUAGE THAT AN INDIVIDUAL IS KNOWN OR ASSUMED TO SPEAK, FOR EXAMPLE: "BULGARIAN". See table 1647/1 which is an instructive entry

2

LANGUAGE  COMPETENCE LEVEL

 

SPECIFIED IN ONE OF THE FOLLOWING WAYS:

 

 

2A

STANDARDIZED LANGUAGE PROFILE

 

AN INDIVIDUAL'S LEVEL OF PROFICIENCY AT LISTENING, SPEAKING, READING, AND WRITING A LANGUAGE.. THIS WILL BE RECORDED WITH A PROFILE OF 4 DIGITS INDICATING THE SPECIFIC SKILLS IN THE ORDER MENTIONED ABOVE., FOR EXAMPLE: "4444". THIS NUMBER REFLECTS THE STANDARDIZED LANGUAGE PROFILE (SLP) AND THE ASSESSMENT IS TO BE DONE IN ACCORDANCE WITH STANAG 6001. See table 2447 which is a composite

 

 

2B

LANGUAGE PROFICIENCY

 

FIELD CONTAINS A CODE WHICH QUALIFIES AN INDIVIDUAL'S LANGUAGE PROFICIENCY IN GENERAL, FOR EXAMPLE: "GOOD". See table 1648/2 which contains a list of data items and associated data codes
