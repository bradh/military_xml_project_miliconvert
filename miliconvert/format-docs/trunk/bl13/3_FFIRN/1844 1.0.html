Field identifier (Name):

1844 (SUSCEPTIBILITY RADHAZ DESIGNATOR)

 

 

Definition:

A CODE WHICH DESCRIBES THE RADHAZ SUSCEPTIBILITY OF A MATERIEL IN TERMS OF ITS MAXIMUM PERMISSIBLE EXPOSURE LEVEL FOR A PARTICULAR RADHAZ FREQUENCY RANGE.
