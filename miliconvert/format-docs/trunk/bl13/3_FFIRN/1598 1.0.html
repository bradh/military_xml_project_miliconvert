Field identifier (Name):

1598 (MINE COUNTERMEASURES OPERATION STANDARD LETTER SUFFIXES)

 

 

Definition:

THE STANDARD LETTER SUFFIXES (SLS) ARE USED IN ASSEMBLING THE TASK ORDER NUMBER. THE SLS INDICATE THE STATE OR COMBINATION OF STAGES, WHICH MUST BE EXECUTED IN THAT PARTICULAR TASK, WITHIN THE FRAMEWORK OF A MINE COUNTERMEASURES OPERATION. THE ASSOCIATED STAGES MAY BE FOUND IN ATP-24(B), VOL 1, PAGE 1-4, TABLE 1-1.
