Field identifier (Name):

1048 (ANGLE IN MILS)

 

 

Definition:

AN ANGLE, IN MILS, MEASURED CLOCKWISE BETWEEN A REFERENCE DIRECTIONAND ANY OTHER LINE. THIS MEASUREMENT MAY INCLUDE NEGATIVE VALUES WHENBEING MEASURED IN OPPOSITE DIRECTIONS (COUNTERCLOCKWISE) FROM AREFERENCE LINE SUCH AS THE HORIZON.
