Table 1088/10 - RECUPERATION LOSS IMPACT (1-1)

 

Related Documents:

Unit of Measure:

 

 

RECUPERATION LOSS IMPACT (Data Item)

Data Code

Explanation

HINDERED WITH DELAY

D

ENTITY'S LOSS WOULD HINDER THE COUNTRY'S ABILITY TO PERFORM THIS FUNCTION OR SERVICE, HOWEVER, THE EFFECT WOULD NOT BE IMMEDIATE DUE TO OTHER FACTORS SUCH AS PRODUCTION CYCLE OR AVAILABLE STOCKS.

HINDERED

H

ENTITY'S LOSS WOULD HINDER THE COUNTRY'S ABILITY TO PERFORM THIS FUNCTION OR SERVICE.  LOSS WOULD CAUSE CONSIDERABLE DISRUPTION TO THE ENTITY'S EQUIPMENT INDUSTRY ALTHOUGH OTHER EQUIPMENT COULD BE PARTIALLY REPLACED FROM ALTERNATE SOURCES.

IRREPLACEABLE

I

LOSS WOULD EFFECTIVELY ELIMINATE THE COUNTRY'S CAPABILITY TO PERFORM THIS FUNCTION OR SERVICE.  THE ENTITY IS UNIQUE AND IRREPLACEABLE BY OTHER'S WITHIN COUNTRY.   THE ENTITY IS OF PRIMARY IMPORTANCE TO SIGNIFICANT CUSTOMERS.

HINDERED BUT MINOR

M

ENTITY'S LOSS WOULD HINDER COUNTRY'S ABILITY TO PERFORM FUNCTION OR SERVICE, BUT MINOR MODIFICATIONS TO OTHER ENTITIES COULD SERVE TO REPLACE FUNCTIONS OR SERVICES.  ADEQUATE REPLACEMENT OF ENTITY EQUIPMENT IS POSSIBLE WITH DIFFICULTY

HINDERED BUT SURGE

S

ENTITY'S LOSS WOULD HINDER THE COUNTRY'S ABILITY TO PERFORM THIS FUNCTION OR SERVICE, BUT SURGES AT OTHER ENTITIES COULD REPLACE AT LEAST HALF OF THE FUNCTION OR SERVICE IN THE NEAR TERM.

TEMPORARY OR INSIGNIFICANT

T

ENTITY'S LOSS WOULD NOT HAVE A SIGNIFICANT EFFECT ON THE FUNCTION OR SERVICE.  LOSS OF ENTITY WOULD BE PRACTICALLY INSIGNIFICANT SINCE ALTERNATE SOURCES EXIST SO AS TO CAUSE NO SIGNIFICANT DETERIORATION OF FUNCTION OR SERVICE.

UNKNOWN

U

None

NO EFFECT

X

ENTITY'S LOSS WOULD HAVE NO EFFECT ON THE COUNTRY'S CAPABILITY TO PERFORM THIS FUNCTION OR SERVICE.
