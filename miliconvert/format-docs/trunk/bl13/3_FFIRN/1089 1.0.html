Table 1089/1 - DISTANCE IN NAUTICAL MILES (1-5)

Unit of Measure:

 

 

DISTANCE IN NAUTICAL MILES

Instructive Regular Expression (1-5)

Explanation

[0-9]{1,5}|[0-9]{0,3}\.[0-9]

 

THE RANGE FOR THIS FUD IS: [0 THROUGH 99999, 0 TO 1 DECIMAL PLACES].

 

THE VALUES CONSIST OF THE INTEGERS 0 THROUGH 99999, AND DECIMAL POINT VALUES. DECIMAL POINT VALUES RANGE FROM .1 TO 999.9, CONSISTING OF ALL POSSIBLE COMBINATIONS OF FOUR OR FEWER SIGNIFICANT DIGITS AND A SIGNIFICANT DECIMAL POINT. THERE IS A MAXIMUM OF FIVE CHARACTERS.
