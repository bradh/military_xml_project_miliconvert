Field identifier (Name):

1846 (TRANSMITTER RADHAZ DESIGNATOR)

 

 

Definition:

A CODE WHICH DESCRIBES THE LEVEL OF EMISSION OF A TRANSMITTER IN TERMS OF THE MAXIMUM ELECTROMAGNETIC RADIATION (EMR) ENVIRONMENT IT IS CAPABLE OF GENERATING.
