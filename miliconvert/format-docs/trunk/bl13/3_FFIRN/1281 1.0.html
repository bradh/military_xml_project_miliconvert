Field identifier (Name):

1281 (REPORTABLE ITEM CODE (RIC))

 

 

Definition:

A CODE FOR ITEMS THAT ARE MAJOR BATTLE DECISIVE EQUIPMENT OR AMMUNITION OR ARE MILITARILY UNIQUE AND FOR WHICH THERE EXISTS EITHER STOCKPILE GUIDANCE OR OTHER MEASURABLE REQUIREMENTS.

 

 

Sponsor:

None

 

 

Remarks:

GENERIC TO THE ENTIRE REPORTABLE ITEM LIST, WHICH CAN BE SUBDIVIDED INTO SMALLER LISTS WHICH ARE: AIR/NAVAL AIR FORCES RIL (ANAFRIL), LAND FORCES RIL (LFRIL), AIR DEFENSE RIL (ADRIL), AND NAVAL FORCES RIL (NFRIL).

 

 

Related Documents:         

REPORTABLE ITEM LIST (RIL), ISSUED ANNUALLY BY SHAPE LOGMAN, LOG AND READINESS CELL.

 

 

Status:  

In Service

 

 

Field Version:

1.0
