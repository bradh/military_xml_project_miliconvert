1036

 

Field identifier (Name):

1036 (VESSEL TYPE, INTELLIGENCE)

 

 

Definition:

A DESIGNATOR USED BY THE NATO INTELLIGENCE COMMUNITY TO DESCRIBE SHIPS AND OTHER WATER-BORNE VESSELS FOR PLANNING AND REPORTING PURPOSES.

 

 

Sponsor:

None

 

 

Remarks:

DATA ITEM AND CODE SOURCE FOR THIS FFIRN IS AINTP-3.  FFIRN 1036 SHOULD ONLY BE MAINTAINED AS LONG AS THERE ARE SIGNIFICANT DIFFERENCES BETWEEN AINTP-3 AND THE LATEST EDITION OF STANAG 1166.  STANDARD SHIP DESIGNATORS COMPLYING WITH STANAG 1166 ARE REPRESENTED UNDER FFIRN 1166.

AINTP-3

In Service
