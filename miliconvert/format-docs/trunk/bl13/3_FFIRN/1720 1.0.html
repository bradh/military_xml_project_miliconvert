Field identifier (Name):

1720 (ELEVATION, DATUM-MSL)

 

 

Definition:

THE VERTICAL DISTANCE OF A SET OF COORDINATES EITHER ABOVE OR BELOW A DATUM OR ABOVE OR BELOW MEAN SEA LEVEL (MSL).

 

 

Sponsor:

None

 

 

Remarks:

A POSITIVE VALUE INDICATES ABOVE THE DATUM/MSL AND A NEGATIVE VALUE INDICATES BELOW THE DATUM/MSL.

 

 

Related Documents:         

NONE

 

 

Status:  

In Service
