Field identifier (Name):

1003 (TIME ZONE)

 

 

Definition:

ONE OF THE TWENTY-FOUR TIME ZONES INTO WHICH THE WORLD HAS BEEN DIVIDED FOR ESTABLISHING TIME RELATIONSHIPS BETWEEN GEOGRAPHICAL AREAS, BASED UPON THE LOCAL STANDARD TIME, GREENWICH, ENGLAND (UNIVERSAL TIME COORDINATED (UTC)).
