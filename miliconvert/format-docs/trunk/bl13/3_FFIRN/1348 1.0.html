Field identifier (Name):

1348 (DATA ENTRY)

 

 

Definition:

A NUMBER ASSOCIATING RELATED LINES OF DATA. [SEE REMARKS.]

 

 

Sponsor:

None

 

 

Remarks:

DATA ENTRY WILL BE USED WITHIN A GROUP OF RELATED DATA SETS TO ASSOCIATE DATA LINES THAT PERTAIN TO A COMMON ENTITY OR EVENT. THE DATA LINE WHICH FIRST IDENTIFIES THE ENTITY OR EVENT WILL HAVE AS ITS FIRST FIELD THE FIELD NAME "DE" WITH A TWO CHARACTER ENTRY ENTERED, FOR EXAMPLE 01. SUBSEQUENTLY, OTHER DATA LINES OF THE MESSAGE HAVING THE ENTRY 01 UNDER THE "DE" FIELD NAME WILL BE UNDERSTOOD TO PERTAIN TO THE SAME ENTITY OR EVENT.
