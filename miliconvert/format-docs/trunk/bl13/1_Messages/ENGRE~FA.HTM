ENGRECCEORD

 

Message identifier (Name):

ENGRECCEORD (ENGINEER RECCE ORDER)

 

 

Related Documents:

STANAG 2430
STANAG 2434

 

 

Purpose:

THE ENGRECCEORD IS USED TO ORDER COMBAT ENGINEER RECONNAISSANCE OF MOBILITY, COUNTERMOBILITY, SURVIVABILITY AND GENERAL ENGINEER SUPPORT TASKS.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:
