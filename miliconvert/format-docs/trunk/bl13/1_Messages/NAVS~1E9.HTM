NAVSITREP

 

Message identifier (Name):

NAVSITREP (NAVAL SITUATION REPORT)

 

 

Related Documents:

BI-SC REPORTING DIRECTIVE

 

 

Purpose:

THE NAVSITREP IS USED TO REPORT SPECIFIC EVENTS, CHANGES IN LOCATION AND MOVEMENT, CHANGES IN READINESS, OR CHANGES IN MAJOR EQUIPMENT STATUS OF MARITIME FORCES.

 

 

Sponsor:

BI-SC OPERATIONS REPORTS WORKING GROUP (ORWG).

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
