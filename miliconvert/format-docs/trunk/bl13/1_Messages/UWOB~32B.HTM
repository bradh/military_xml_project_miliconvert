UW OBJECT NOTE

 

Message identifier (Name):

UW OBJECT NOTE (UNDERWATER OBJECT NOTIFICATION)

 

 

Related Documents:

AHP-6
ATP-1 VOL I
ATP-28

 

 

Purpose:

THE UW OBJECT NOTE IS USED TO PROVIDE INFORMATION TO APPROPRIATE AUTHORITIES ON A UNIT'S INTENTION FOR STREAMING OR DEPLOYING A SONIC OR OTHER UNDERWATER DEVICE.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

ADDRESSES FOR THIS MESSAGE ARE DESCRIBED IN AHP-6.

 

 

Remarks:

None

 

 

Status:

In Service
