CCISSTATREP

 

Message identifier (Name):

CCISSTATREP (CCIS STATUS REPORT)

 

 

Related Documents:

BI-SC REPORTING DIRECTIVE
STANAG 2434

 

 

Purpose:

THE CCISSTATREP IS USED TO PROVIDE INFORMATION CONCERNING THE STATUS OF COMMAND, CONTROL AND INFORMATION SYSTEMS.

 

 

Sponsor:

SHAPE

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:
