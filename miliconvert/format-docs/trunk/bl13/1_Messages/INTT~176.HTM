INTTOLAY

 

Message identifier (Name):

INTTOLAY (INTENT TO LAY MINEFIELD)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE INTTOLAY IS USED TO DISSEMINATE INFORMATION RELATING TO A TACTICAL COMMANDER'S INTENT TO LAY A MINEFIELD.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
