OPTASK RAS

 

Message identifier (Name):

OPTASK RAS (OPERATIONAL TASKING REPLENISHMENT AT SEA)

 

 

Related Documents:

ATP-1
ATP-16

 

 

Purpose:

THE OPTASK RAS IS USED BY THE OFFICER IN TACTICAL COMMAND OR DELEGATED AUTHORITY TO PROMULGATE THE REPLENISHMENT PROGRAMME.

 

 

Sponsor:

NSA (NAVY) MARITIME LOGISTICS WORKING GROUP (MLWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
