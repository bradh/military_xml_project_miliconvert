Message identifier (Name):

SARREQ (SEARCH AND RESCUE REQUEST)

 

 

Related Documents:

ATP-10

 

 

Purpose:

THE SARREQ IS USED TO REQUEST FORCES TO PARTICIPATE IN A SEARCH AND RESCUE MISSION.

 

 

Sponsor:

NSA (AIR) SEARCH AND RESCUE WORKING GROUP (SARWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
