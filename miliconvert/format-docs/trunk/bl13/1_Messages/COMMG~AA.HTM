COMMGATEWAYESTA

 

Message identifier (Name):

COMMGATEWAYESTA (COMMUNICATIONS GATEWAY ESTABLISHMENT)

 

 

Related Documents:

STANAG 4211

 

 

Purpose:

THE COMMGATEWAYESTA IS USED TO ORDER THE ESTABLISHMENT OF A COMMUNICATIONS GATEWAY AND TO ANNOUNCE ITS OPERATIONAL CHARACTERISTICS.

 

 

Sponsor:

SC 6-WG1. TACTICAL COMMUNICATIONS WORKING GROUP.

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service

C

SET 14 (RADLNK) IS REQUIRED IF FIELD 1 IN SET 9 (GATETYPE) EQUALS /RR/, OTHERWISE IT IS PROHIBITED.

FIELD 4 IN SET 14 (RADLNK) IS REQUIRED IF FIELD 3 IN SET 14 (RADLNK) DOES NOT EQUAL /1/ AND /2/ AND /3/ AND /4/, OTHERWISE IT IS PROHIBITED.

SET 15 (SATLNK) IS REQUIRED IF FIELD 1 IN SET 9 (GATETYPE) EQUALS /SATCOM/, OTHERWISE IT IS PROHIBITED.
