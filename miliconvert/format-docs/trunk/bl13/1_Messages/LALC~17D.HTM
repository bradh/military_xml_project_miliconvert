LALC REPORT

 

Message identifier (Name):

LALC REPORT (LOCAL AIR LOGISTIC COORDINATOR REPORT)

 

 

Related Documents:

ALP-4.1

 

 

Purpose:

THE LALC REPORT ALLOWS THE LOCAL AIR LOGISTIC COORDINATOR (LALC) TO REPORT ON LOGISTIC AIRLIFT REQUIREMENTS FOR THE MOVEMENT OF SPARE PARTS AND TECHNICAL ASSISTANCE WITHIN THE TASK FORCE OR TASK GROUP. THE MESSAGE IS INTENDED TO COVER A TIME PERIOD OF 24HRS.

 

 

Sponsor:

NSA (NAVY) MARITIME LOGISTICS WORKING GROUP (MLWG)

 

 

Notes:

THE MESSAGE IS DESIGNED SO THAT HELICOPTERS CAN BE MULTI-TASKED, HENCE THE REPETITION THAT IS AVAILABLE AT BOTH THE HELOTASK AND TRNSUNIT SEGMENTS.

 

 

Remarks:

None

 

 

Status:

In Service
