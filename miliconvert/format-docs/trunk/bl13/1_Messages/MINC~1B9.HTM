MINCLRRECCEORD

 

Message identifier (Name):

MINCLRRECCEORD (MINEFIELD CLEARING RECONNAISSANCE ORDER)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE MINCLRRECCEORD IS USED TO ORDER FRIENDLY FORCES TO CONDUCT A MINEFIELD CLEARING RECONNAISSANCE.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:
