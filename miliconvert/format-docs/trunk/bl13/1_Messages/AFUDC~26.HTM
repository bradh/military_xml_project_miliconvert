AFU.DCMD

 

Message identifier (Name):

AFU.DCMD (AMMUNITION FIRE UNIT-DEPLOYMENT COMMAND)

 

 

Related Documents:

ASCA-012 02 CTIDP
STANAG 2432

 

 

Purpose:

THE AFU.DCMD IS USED TO TRANSMIT ASSIGNMENTS AND CHANGES TO PREPLANNED FIRE POSITIONS AND MOVEMENT ORDERS FOR BATTALION, REGIMENT, OR FIRE UNITS.

 

 

Sponsor:

NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

1.  WHEN A ROUTE CROSSES INTO ANOTHER GRID ZONE, SEPARATE MESSAGES ARE TO BE SENT EACH TIME THE ROUTE CHANGES ZONES.

 

 

Remarks:

None

 

 

Status:

In Service
