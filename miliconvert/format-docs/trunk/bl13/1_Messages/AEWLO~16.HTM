AEW LOGSITREP

 

Message identifier (Name):

AEW LOGSITREP (AIRBORNE EARLY WARNING LOGISTICS SITUATION REPORT)

 

 

Related Documents:

NONE

 

 

Purpose:

THE AEW LOGSITREP IS USED TO REPORT THE LOGISTICS STATUS OF EACH AEW E-3 DEPLOYED ELEMENT.

 

 

Sponsor:

SHAPE

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
