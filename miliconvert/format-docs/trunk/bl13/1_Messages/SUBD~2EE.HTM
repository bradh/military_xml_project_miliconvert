SUBDANGER

 

Message identifier (Name):

SUBDANGER (SUBMARINE DANGER AREA NOTICE)

 

 

Related Documents:

AHP-6
ATP-1 VOL I, (CHAPTER 6)

 

 

Purpose:

THE SUBDANGER IS USED TO PROMULGATE INTENDED OPERATIONS OR ACTIONS HAZARDOUS TO SUBMARINES.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

1.  ADDRESSES FOR THIS MESSAGE ARE DESCRIBED IN AHP-6.2.  The PASSAGE and MOVAREA sets should be used in combination to describe a passage plan with holding areas. For example, a passage through points AA, BB, CC, operate in an area called HOLDING AREA ALPHA and then return through points CC, BB, AA.
