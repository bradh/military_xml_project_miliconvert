MINLAYRECCEORD

 

Message identifier (Name):

MINLAYRECCEORD (MINEFIELD LAYING RECONNAISSANCE ORDER)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE MINLAYRECCEORD IS USED TO ORDER FRIENDLY FORCES TO CONDUCT A RECONNAISSANCE FOR THE LAYING OF MINEFIELDS BY ENGINEERS.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
