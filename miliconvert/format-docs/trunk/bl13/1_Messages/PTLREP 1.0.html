Message identifier (Name):

PTLREP (PATROL REPORT)

 

 

Related Documents:

STANAG 2003
STANAG 2434

 

 

Purpose:

THE PTLREP IS USED TO PROVIDE LAND FORCES INFORMATION FOLLOWING PATROL ACTIVITY.

 

 

Sponsor:

NSA (ARMY) LAND OPERATIONS WORKING GROUP (LOWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
