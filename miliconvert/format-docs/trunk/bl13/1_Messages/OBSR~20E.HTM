OBSRECCEORD

 

Message identifier (Name):

OBSRECCEORD (OBSTACLE RECONNAISSANCE ORDER)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE OBSRECCEORD IS USED TO ORDER FRIENDLY FORCES TO CONDUCT A RECONNAISSANCE OF FRIENDLY OR ENEMY OBSTACLES, EITHER EXISTING OR PLANNED.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
