Message identifier (Name):

GAPREP (GAP CROSSING COMPLETION REPORT)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE GAPREP IS USED TO REPORT THE COMPLETION OF A GAP CROSSING SITE TASK.

 

 

Sponsor:

Notes:

None

None
