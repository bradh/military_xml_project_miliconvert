ENGRDATAREP

 

Message identifier (Name):

ENGRDATAREP (ENGINEER DATA REPORT)

 

 

Related Documents:

STANAG 2430
STANAG 2434

 

 

Purpose:

THE ENGRDATAREP IS USED TO PASS EFFECTIVE COMBAT ENGINEER UNIT AND ASSET DETAILS FROM BRIGADE TO CORPS LEVEL. THIS MESSAGE PROVIDES DETAILED INFORMATION ABOUT THE NUMBER OF EFFECTIVE UNITS BY TYPE, GENERIC EQUIPMENT TYPES IN TERMS OF AVAILABILITY AND COMMITTED AND UNCOMMITTED MAJOR ITEMS OF MATERIEL.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
