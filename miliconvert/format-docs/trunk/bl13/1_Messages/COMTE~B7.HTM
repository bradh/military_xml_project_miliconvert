COMTECHREP - TYPE A

 

Message identifier (Name):

COMTECHREP - TYPE A (COMPLEMENTARY TECHNICAL REPORT - TYPE A)

 

 

Related Documents:

STANAG 2195 (AJP-2.5)

 

 

Purpose:

THE COMTECHREP - TYPE A IS USED TO REPORT INFORMATION ABOUT A SINGLE CAPTURED AIRCRAFT.

 

 

Sponsor:

NSA (JSB) JOINT INTELLIGENCE WORKING GROUP (JINTWG)

 

 

Notes:

None

 

 

Remarks:

TO BE SUBMITTED BY THE FASTEST AVAILABLE MEANS IMMEDIATELY FOLLOWING INITIAL EXAMINATION OF AN ENEMY AIRCRAFT.

 

 

Status:

In Service
