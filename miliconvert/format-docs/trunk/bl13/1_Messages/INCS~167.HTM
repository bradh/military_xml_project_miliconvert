INCSPOTREP

 

Message identifier (Name):

INCSPOTREP (INCIDENT SPOT REPORT)

 

 

Related Documents:

NONE

 

 

Purpose:

THE INCSPOTREP IS USED TO PROVIDE TIME CRITICAL INFORMATION ON IMPORTANT EVENTS THAT HAVE AN IMMEDIATE IMPACT ON OPERATIONS.

 

 

Sponsor:

COUNCIL OPERATIONS AND EXERCISE COMMITTEE (COEC). SHAPE

 

 

Notes:

1.  SET MSGID PROVIDES THE ABILITY TO INDICATE THAT THE MESSAGE IS AN INITIAL, FOLLOW-UP OR FINAL REPORT.2.  TO SPECIFY ANY ADDITIONAL INFORMATION NOT COVERED BY ANY OF THE OTHER SETS IN THE MESAGE, ADD SET REMARKS (RMKS) AS THE LAST SET IN THE MESSAGE.

 

 

Remarks:

None

 

 

Status:

In Service
