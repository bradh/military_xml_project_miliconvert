AFU.FUS

 

Message identifier (Name):

AFU.FUS (AMMUNITION FIRE UNIT-FIRE UNIT STATUS)

 

 

Related Documents:

ASCA-012 02 CTIDP
STANAG 2432
STANAG 2434

 

 

Purpose:

THE AFU.FUS IS USED TO ESTABLISH OR DELETE THE STATUS FOR A FIRE UNIT IN CURRENT OR PLANNED OPERATIONS FOR THE PURPOSE OF MAKING TACTICAL FIRE CONTROL DECISIONS.

 

 

Sponsor:

NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
