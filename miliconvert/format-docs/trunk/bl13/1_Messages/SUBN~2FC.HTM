SUBNOTE CHANGE REQ

 

Message identifier (Name):

SUBNOTE CHANGE REQ (SUBMARINE NOTICE CHANGE REQUEST)

 

 

Related Documents:

AHP-6
ATP-1

 

 

Purpose:

THE SUBNOTE CHANGE REQ IS A REQUEST TO ORDER AND INFORM ABOUT A CHANGE IN THE MOVEMENT OF A SUBMARINE.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

1.  PROCEDURES IN ATP-18 AND ADDRESSES FOR THIS MESSAGE ARE DESCRIBED IN AHP-6.

 

 

Remarks:

None

 

 

Status:

In Service
