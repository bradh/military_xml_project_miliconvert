BRIDGEREP

 

Message identifier (Name):

BRIDGEREP (BRIDGE REPORT)

 

 

Related Documents:

ATP-38
ATP-8

 

 

Purpose:

THE BRIDGEREP IS USED TO REPORT TO TACTICAL COMMANDERS THE PHYSICAL DETAILS OF BRIDGES LOCATED DURING RECONNAISSANCE OPERATIONS.

 

 

Sponsor:

NSA (NAVY) AMPHIBIOUS OPERATIONS WORKING GROUP (AMPHIB OPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
