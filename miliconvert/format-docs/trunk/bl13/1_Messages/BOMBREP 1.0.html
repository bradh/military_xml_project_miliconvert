BOMBREP

 

Message identifier (Name):

BOMBREP (BOMB, SHELL, ROCKET OR MORTAR REPORT)

 

 

Related Documents:

STANAG 2434

 

 

Purpose:

THE BOMBREP IS USED TO REPORT HOSTILE BOMB, SHELL, ROCKET AND MORTAR ATTACKS.

 

 

Sponsor:

NSA (ARMY) LAND OPERATIONS WORKING GROUP (LOWG). NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
