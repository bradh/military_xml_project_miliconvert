Message identifier (Name):

SARIR (SEARCH AND RESCUE INCIDENT REPORT)

 

 

Related Documents:

ATP-10

 

 

Purpose:

THE SARIR IS USED TO REPORT ANY SITUATION WHICH MAY REQUIRE A SEARCH AND RESCUE EFFORT.

 

 

Sponsor:

NSA (AIR) SEARCH AND RESCUE WORKING GROUP (SARWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
