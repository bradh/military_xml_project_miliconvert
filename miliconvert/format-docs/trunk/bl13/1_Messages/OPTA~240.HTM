OPTASK LOG FLC

 

Message identifier (Name):

OPTASK LOG FLC (OPTASK LOGISTIC FORCE LOGISTIC COORDINATOR INSTRUCTIONS)

 

 

Related Documents:

ALP-4.1

 

 

Purpose:

THE OPTASK LOG FLC PROVIDES THE FORCE LOGISTIC COORDINATOR'S (FLC) OR GROUP LOGISTIC COORDINATOR'S (GLC) DIRECTIVES, DUTY ASSIGNMENTS AND OTHER ESSENTIAL INSTRUCTIONS TO EXECUTE AFLOAT LOGISTIC SUPPORT FOR FORCES ASSIGNED.

 

 

Sponsor:

NSA (NAVY) MARITIME LOGISTICS WORKING GROUP (MLWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
