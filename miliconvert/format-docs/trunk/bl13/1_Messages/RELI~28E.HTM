RELINQTAC

 

Message identifier (Name):

RELINQTAC (RELINQUISH SUBMARINE TACON)

 

 

Related Documents:

ATP-18

 

 

Purpose:

THE RELINQTAC MESSAGE IS USED TO REPORT DETAILS OUTSTANDING BEFORE RELINQUISHMENT OF TACOM/TACON BY THE OTC/SOCA.

 

 

Sponsor:

MARITIME OPERATIONS WORKING GROUP (MAROPSWG)

 

 

Notes:

THE AREA WILL GENERALLY BE EFFECTIVE EARLIER THAN SPECIFIED IN THE PRECEDING SUBTRK SET AND UNTIL A TIME LATER THAN SPECIFIED IN THE IMMEDIATELY FOLLOWING SUBTRK SET. WHEN ADJACENT AREAS ARE ALLOCATED, REASONABLE OVERLAP TIME MUST ALLOW THE SUBMARINE TO MOVE FROM ONE AREA TO ANOTHER.

 

 

Remarks:

None

 

 

Status:

In Service
