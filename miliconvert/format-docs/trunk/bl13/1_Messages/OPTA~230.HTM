OPTASK ASUW

 

Message identifier (Name):

OPTASK ASUW (OPERATIONAL TASKING OF ANTI-SURFACE WARFARE)

 

 

Related Documents:

ATP-1 (C) VOL I
AXP-5

 

 

Purpose:

THE OPTASK ASUW IS USED TO PROMULGATE DETAILED TASKING AND INSTRUCTIONS FOR THE CONDUCT OF ANTI-SURFACE WARFARE.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

IF "OTHER" IS USED IN FIELD 1 OF SET "TRU", USE AN "AMPN" SET TO PROVIDE TYPE OF TRU.

 

 

Remarks:

None

 

 

Status:

In Service
