OPGEN

 

Message identifier (Name):

OPGEN (OPERATIONAL GENERAL MATTERS)

 

 

Related Documents:

ATP-1

 

 

Purpose:

THE OPGEN IS USED TO PROMULGATE GENERAL MATTERS OF POLICY, INSTRUCTIONS AND ASPECTS COMMON TO ALL FORMS OF WARFARE AND DETAILED INSTRUCTIONS FOR WARFARE RESPONSIBILITIES RETAINED BY THE OTC.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

1.  THE PASSAGE AND MOVAREA SETS SHOULD BE USED IN COMBINATION TO DESCRIBE A PASSAGE PLAN WITH HOLDING AREAS. FOR EXAMPLE, A PASSAGE THROUGH POINTS AA, BB, CC, OPERATE IN AN AREA CALLED HOLDING AREA ALPHA AND THEN RETURN THROUGH POINTS CC, BB, AA. NOTE THE        PASSAGE/5309.012N-00332.016E/121200ZJUN2008/AA//       PASSAGE/5330.012N-00332.026E/121400ZJUN2008/BB//       PASSAGE/5345.000N-00312.000E/121400ZJUN2008/CC//      MOVAREA/121400ZJUN2008/121800ZJUN2008/HOLDING AREA ALPHA       /5345.000N-00300.000E/5445.000N-00300.000E/5445.000N-00400.000E       /5345.000N-00400.000E//       PASSAGE/5345.000N-00312.000E/121800ZJUN2008/CC//       PASSAGE/5330.012N-00332.026E/122000ZJUN2008/BB//       PASSAGE/5309.012N-00332.016E/122200ZJUN2008/AA//
