COMTECHREP - TYPE C

 

Message identifier (Name):

COMTECHREP - TYPE C (COMPLEMENTARY TECHNICAL REPORT - TYPE C)

 

 

Related Documents:

STANAG 2195 (AJP-2.5)

 

 

Purpose:

THE COMTECHREP - TYPE C IS USED TO REPORT ITEMS NOT REPORTED UNDER COMTECHREP - TYPES A (AIRCRAFT) OR B (EXPLOSIVE ORDNANCE).

 

 

Sponsor:

NSA (JSB) JOINT INTELLIGENCE WORKING GROUP (JINTWG)

 

 

Notes:

TO BE SUBMITTED WITHIN 72 HOURS FOLLOWING THE ACQUISITION OF AN ITEM OF CAPTURED EQUIPMENT NOT COVERED UNDER TYPES A AND B.

 

 

Remarks:

None

 

 

Status:

In Service
