OPTASK LOG MNLC

 

Message identifier (Name):

OPTASK LOG MNLC (OPTASK LOGISTIC MULTINATIONAL LOGISTIC COMMANDER DIRECTIVE)

 

 

Related Documents:

ALP-4.1

 

 

Purpose:

THE OPTASK LOG MNLC PROMULGATES THE MULTINATIONAL LOGISTIC COMMANDER'S (MNLC) DIRECTIVES, DUTY ASSIGNMENTS, AND OTHER ESSENTIAL INSTRUCTIONS AND INFORMATION TO EXECUTE LOGISTIC SUPPORT FOR THE MULTINATIONAL MARITIME FORCE. THE OPTASK LOG MNLC IS THE PRIMARY VEHICLE FOR COORDINATING LOGISTIC MATTERS ASHORE.

 

 

Sponsor:

NSA (NAVY) MARITIME LOGISTICS WORKING GROUP (MLWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
