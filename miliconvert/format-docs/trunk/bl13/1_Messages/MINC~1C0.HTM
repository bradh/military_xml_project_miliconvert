Message identifier (Name):

MINCLRREP (MINEFIELD CLEARING COMPLETION REPORT)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE MINCLRREP IS USED TO REPORT THE RESULTS OF MINEFIELD CLEARING OPERATIONS BY FRIENDLY FORCES.

 

 

Sponsor:

Notes:

None

None
