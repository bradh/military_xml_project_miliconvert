INTRELREC

 

Message identifier (Name):

INTRELREC (INTELLIGENCE RELATIONSHIP RECORD)

 

 

Related Documents:

AINTP-3 (A)
STANAG 2077
STANAG 2097
STANAG 2433

 

 

Purpose:

THE INTRELREC  IS USED TO CREATE OR UPDATE A RELATIONSHIP RECORD BASED ON THE DATA STRUCTURES CONTAINED IN AINTP-3. THE INTRELREC DEFINES LINKS BETWEEN A MAJOR INTELLIGENCE CATEGORY AND OTHER MAJOR INTELLIGENCE CATEGORIES..

 

 

Sponsor:

NSA (JSB) JOINT INTELLIGENCE WORKING GROUP (JINTWG)

 

 

Notes:

THE MESSAGE CREATES A NEW RECORD OR AN UPDATE OF AN EXISTING ONE DEPENDING ON THE ENTRY IN FIELD 7 OF SET 3 (MSGID) WHICH MUST EITHER READ "NEW" OR "UPD".

 

 

Remarks:

None

 

 

Status:

In Service

MSGID

3

MESSAGE IDENTIFIER

SPECIFIES THE MESSAGE IDENTIFIER, MESSAGE ORIGINATOR, AND OTHER MESSAGE IDENTITY DETAILS.  IF USED, FIELD 7 OF THIS SET IN THIS MESSAGE MUST ONLY EQUAL "NEW" OR "UPD".

FIELD 1 IN SET 3 (MSGID) IS ASSIGNED THE VALUE /INTRELREC/.

FIELD 7 IN SET 3 (MSGID) MUST EQUAL /NEW/ IF FIELD 7 IN SET 3 (MSGID) DOES NOT EQUAL /UPD/.

FIELD 2 IN SET 3 (MSGID) IS ASSIGNED THE VALUE /APP-11(C)/.

FIELD 3 IN SET 3 (MSGID) IS ASSIGNED THE VALUE /ORIGINAL/.

M

CATIDENT

5

CATEGORY IDENTIFICATION

IDENTIFIES THE MAJOR INTELLIGENCE CATEGORY FORMING THE FIRST PART OF THE RELATIONSHIP AND CONSISTS OF THE UNIQUE IDENTIFICATION (ROOT) OF THE RECORD IN THIS CATEGORY WITH WHICH A RELATIONSHIP IS BEING FORMED.  FIELD 3 MUST BE USED ONLY IF ALTERNATIVE "E" (BIOGRAPHIC KEY) IS USED IN FIELD 1.  FIELDS 4 AND 5 MUST BE USED ONLY, IF ALTERNATIVE "D" (EVENT KEY) IS USED IN FIELD 1.  FIELD 6 MUST BE USED, IF ALTERNATIVE "A" (PLACE KEY) IS USED IN FIELD 1.  FIELD 2 MUST USE ALTERNATIVE A, B, C, D, OR E IF FIELD 1 USES ALTERNATIVE A, B, C, D OR E RESPECTIVELY.

FIELD 3 IN SET 5 (CATIDENT) IS REQUIRED IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2445-001, OTHERWISE IT IS PROHIBITED.

FIELD 4 IN SET 5 (CATIDENT) IS REQUIRED IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2444-001, OTHERWISE IT IS PROHIBITED.

FIELD 5 IN SET 5 (CATIDENT) IS REQUIRED IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2444-001, OTHERWISE IT IS PROHIBITED.

FIELD 6 IN SET 5 (CATIDENT) IS REQUIRED IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2441-001, OTHERWISE IT IS PROHIBITED.

FIELD 2 IN SET 5 (CATIDENT) MUST USE ALTERNATIVE FF1022-261 IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2441-001.

FIELD 2 IN SET 5 (CATIDENT) MUST USE ALTERNATIVE FF1714-002 IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2442-001.

FIELD 2 IN SET 5 (CATIDENT) MUST USE ALTERNATIVE FF1022-253 IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2443-001.

FIELD 2 IN SET 5 (CATIDENT) MUST USE ALTERNATIVE FF1022-254 IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2444-001.

FIELD 2 IN SET 5 (CATIDENT) MUST USE ALTERNATIVE FF1022-255 IF FIELD 1 IN SET 5 (CATIDENT) USES ALTERNATIVE FF2445-001.
