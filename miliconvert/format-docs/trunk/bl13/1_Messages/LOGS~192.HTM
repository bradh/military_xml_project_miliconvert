LOGSITREPMAR

 

Message identifier (Name):

LOGSITREPMAR (LOGISTIC SITUATION REPORT MARITIME)

 

 

Related Documents:

ALP-4.1

 

 

Purpose:

THE LOGSITREPMAR IS USED TO PROVIDE THE TASK GROUP OR TASK FORCE COMMANDER WITH THE LOGISTICAL STATUS OF AN INDIVIDUAL UNIT WITHIN THE TASK GROUP OR TASK FORCE. INDIVIDUAL MULTI-NATIONAL MARITIME FORCE (MNMF) UNITS ARE TO SUBMIT THE LOGSITREPMAR DAILY.

 

 

Sponsor:

NSA (NAVY) MARITIME LOGISTICS WORKING GROUP (MLWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
