DEPSTARTREP

 

Message identifier (Name):

DEPSTARTREP (DEPLOYMENT START REPORT)

 

 

Related Documents:

BI-SC REPORTING DIRECTIVE
STANAG 2434

 

 

Purpose:

THE DEPSTARTREP IS USED TO INFORM THE ACE MOBILITY COORDINATION CENTER (AMCC) OF THE START OF EACH SPECIFIC DEPLOYMENT.

 

 

Sponsor:

SHAPE

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
