LOGSITLAND

 

Message identifier (Name):

LOGSITLAND (LOGISTIC SITUATION REPORT LAND FORCES)

 

 

Related Documents:

STANAG 2406
STANAG 2434

 

 

Purpose:

THE LOGSITLAND IS USED TO PROVIDE A SUPERIOR HEADQUARTERS WITH AN EVALUATION OF A UNIT'S OR FORMATION'S LOGISTICAL SITUATION, CAPABILITY, AND DEFICIENCIES/SURPLUSES.

 

 

Sponsor:

NSA (ARMY) LAND FORCES LOGISTICS DOCTRINE WORKING GROUP (LOGWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:
