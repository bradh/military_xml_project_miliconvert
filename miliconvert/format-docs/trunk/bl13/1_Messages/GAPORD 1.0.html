GAPORD

 

Message identifier (Name):

GAPORD (GAP CROSSING EXECUTION ORDER)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE GAPORD IS USED TO ORDER THE EXECUTION OF A GAP CROSSING SITE TASK.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
