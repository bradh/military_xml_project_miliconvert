Message identifier (Name):

DELTREP (RIVER ESTUARY REPORT)

 

 

Related Documents:

ATP-38

 

 

Purpose:

THE DELTREP IS USED TO REPORT INFORMATION RELATING TO RIVER OR ESTUARY RECONNAISSANCE OPERATIONS.

 

 

Sponsor:

NSA (NAVY) AMPHIBIOUS OPERATIONS WORKING GROUP (AMPHIB OPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
