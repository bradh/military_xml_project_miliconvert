OPTASK MINING

 

Message identifier (Name):

OPTASK MINING (MINE TASKING)

 

 

Related Documents:

ATP-24
CNBA SOP 15

 

 

Purpose:

THE OPTASK MINING IS USED TO ORDER THE EXECUTION OF MINELAYING OPERATIONS AND TO ADDRESS RELATED ADMINISTRATIVE AND LOGISTIC MATTERS.

 

 

Sponsor:

NSA (NAVY) NAVAL MINE WARFARE WORKING GROUP (NMWWG)

 

 

Notes:

1.  THE PASSAGE AND MOVAREA SETS SHOULD BE USED IN COMBINATION TO DESCRIBE A PASSAGE PLAN WITH HOLDING AREAS. FOR EXAMPLE, A PASSAGE THROUGH POINTS AA, BB, CC, OPERATE IN AN AREA CALLED HOLDING AREA ALPHA AND THEN RETURN THROUGH POINTS CC, BB, AA. FOR EXAMPLE:       PASSAGE/5309.012N-00332.016E/121200ZJUN2008/AA//       PASSAGE/5330.012N-00332.026E/121400ZJUN2008/BB//       PASSAGE/5345.000N-00312.000E/121400ZJUN2008/CC//       MOVAREA/121400ZJUN2008/121800ZJUN2008/HOLDING AREA ALPHA       /5345.000N-00300.000E/5445.000N-00300.000E/5445.000N-00400.000E       /5345.000N-00400.000E//       PASSAGE/5345.000N-00312.000E/121800ZJUN2008/CC//       PASSAGE/5330.012N-00332.026E/122000ZJUN2008/BB//       PASSAGE/5309.012N-00332.016E/122200ZJUN2008/AA//

 

 

Remarks:

None

 

 

Status:

In Service
