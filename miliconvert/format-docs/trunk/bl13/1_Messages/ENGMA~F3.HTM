ENGMATREQREL

 

Message identifier (Name):

ENGMATREQREL (ENGINEER MATERIEL REQUEST RELEASE ORDER)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE ENGMATREQREL IS USED TO DISSEMINATE INFORMATION RELATING TO THE REQUEST OR RELEASE OF ENGINEER MATERIEL.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

IF FIELD 2 OF SET MTREQREL OCCURS THIS MESSAGE IS A MATERIEL RELEASE ORDER.

 

 

Remarks:

IF FIELD 2 OF SET MTREQREL OCCURS THIS MESSAGE IS A MATERIEL RELEASE ORDER.

 

 

Status:

In Service
