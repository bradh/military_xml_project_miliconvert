EQUIPINTREC

 

Message identifier (Name):

EQUIPINTREC (EQUIPMENT INTELLIGENCE RECORD)

 

 

Related Documents:

AINTP-3 (A)
STANAG 2077
STANAG 2097
STANAG 2433

 

 

Purpose:

THE EQUIPINTREC IS USED TO CREATE OR UPDATE AN EQUIPMENT RECORD BASED ON THE DATA STRUCTURES CONTAINED IN AINTP-3. THIS ENABLES THE EXCHANGE OF INDIVIDUAL DATA RECORDS BETWEEN NATIONS AND FORMATIONS MAINTAINING DATA AND DATA BASES TO THE STANDARDS CONTAINED IN AINTP-3.

 

 

Sponsor:

NSA (JSB) JOINT INTELLIGENCE WORKING GROUP (JINTWG)

 

 

Notes:

THE MESSAGE CREATES A NEW RECORD OR PROVIDES AN UPDATE TO AN EXISTING ONE DEPENDING ON THE ENTRY IN FIELD 7 OF SET 3 (MSGID), WHICH MUST EITHER READ "NEW" OR "UPD" FOR THIS MESSAGE.  SINCE THE FIELD IS OPERATIONALLY DETERMINED, IT IS POSSIBLE TO LEAVE THE FIELD BLANK OR ENTER A "-" FOR NO DATA.  IT HAS TO BE UNDERSTOOD THAT, IN THIS CASE, THE MESSAGE WILL BE ASSUMED TO BE A NEW ONE, REQUIRING THE USE SPECIFIC SETS, THROUGH STRUCTURED LANGUAGE STATEMENTS.

 

 

Remarks:

None

 

 

Status:

In Service
