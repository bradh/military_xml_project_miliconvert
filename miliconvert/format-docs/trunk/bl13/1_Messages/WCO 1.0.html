Message identifier (Name):

WCO (WEAPON CONTROL ORDER)

 

 

Related Documents:

Purpose:

THE WCO IS USED TO ORDER NEW AIR DEFENSE (AD) WEAPON CONTROL STATUSES FOR PARTICULAR GROUND-BASED AD UNITS FOR A GIVEN PERIOD OF TIME.

 

 

Sponsor:

Remarks:
