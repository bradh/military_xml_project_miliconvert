SUBSITREP

 

Message identifier (Name):

SUBSITREP (SUBMARINE SITUATION REPORT)

 

 

Related Documents:

ATP-18

 

 

Purpose:

THE SUBSITREP PROVIDES THE PRIMARY REPORTING MEANS FOR THE SUBMARINE TO CONVEY INFORMATION BACK TO THE SUBMARINE TASKING AUTHORITY.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
