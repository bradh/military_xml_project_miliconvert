COMTECHREP - TYPE B

 

Message identifier (Name):

COMTECHREP - TYPE B (COMPLEMENTARY TECHNICAL REPORT - TYPE B)

 

 

Related Documents:

STANAG 2195 (AJP-2.5)

 

 

Purpose:

THE COMTECHREP - TYPE B IS USED TO REPORT INFORMATION ABOUT ONE SPECIFIC TYPE OF CAPTURED EXPLOSIVE ORDNANCE.

 

 

Sponsor:

NSA (JSB) JOINT INTELLIGENCE WORKING GROUP (JINTWG)

 

 

Notes:

TO BE SUBMITTED BY THE FASTEST AVAILABLE MEANS IMMEDIATELY FOLLOWING INITIAL EXAMINATION OF CAPTURED EXPLOSIVE ORDNANCE, WITH A FOLLOW-ON MESSAGE ONCE DETAILED ANALYSIS HAS BEEN COMPLETED.

 

 

Remarks:

None

 

 

Status:

In Service
