SPRT.ACA

 

Message identifier (Name):

SPRT.ACA (SUPPORT-AIRSPACE COORDINATION AREA)

 

 

Related Documents:

ASCA-012 02
STANAG 2432

 

 

Purpose:

THE SPRT.ACA IS USED TO ESTABLISH OR DELETE AIRSPACE COORDINATION AREAS (ACA).

 

 

Sponsor:

NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

A SPRT.ACA CAN ONLY BE USED TO IDENTIFY AN AIRSPACE COORDINATION AREA IN A SINGLE GRID ZONE.  USE ADDITIONAL SPRT.ACA MESSAGES WHEN AN AIRSPACE COORDINATION AREA COVERS MULTIPLE GRID ZONES.

 

 

Remarks:

None

 

 

Status:

In Service
