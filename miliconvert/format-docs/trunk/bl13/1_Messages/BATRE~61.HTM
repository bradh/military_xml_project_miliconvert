BATRECEVACREQ

 

Message identifier (Name):

BATRECEVACREQ (BATTLEFIELD RECOVERY EVACUATION REQUEST)

 

 

Related Documents:

STANAG 2375
STANAG 2399
STANAG 2400
STANAG 2434
STANAG 3150
STANAG 3151

 

 

Purpose:

THE BATRECEVACREQ IS USED BY A UNIT TO REQUEST RECOVERY ASSISTANCE FOR ONE OR MORE VEHICLES AT A SINGLE LOCATION.

 

 

Sponsor:

NSA (ARMY) LAND FORCES BATTLEFIELD MAINTENANCE WORKING GROUP (BMWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
