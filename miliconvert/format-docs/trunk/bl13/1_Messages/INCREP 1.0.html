INCREP

 

Message identifier (Name):

INCREP (INCIDENT REPORT)

 

 

Related Documents:

STANAG 2434

 

 

Purpose:

THE INCREP IS USED TO REPORT ANY SIGNIFICANT INCIDENT CAUSED BY TERRORISM, CIVIL UNREST, NATURAL DISASTER, OR MEDIA ACTIVITY.

 

 

Sponsor:

NSA (ARMY) LAND OPERATIONS WORKING GROUP (LOWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
