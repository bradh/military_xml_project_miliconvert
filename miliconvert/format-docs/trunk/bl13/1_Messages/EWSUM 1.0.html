Message identifier (Name):

EWSUM (ELECTRONIC WARFARE SUMMARY)

 

 

Related Documents:

BI-SC REPORTING DIRECTIVE

 

 

Purpose:

THE EWSUM IS USED TO INFORM THE SC OF THE ELECTRONIC WARFARE (EW) SITUATION.

 

 

Sponsor:

SHAPE

 

 

Notes:

None

None
