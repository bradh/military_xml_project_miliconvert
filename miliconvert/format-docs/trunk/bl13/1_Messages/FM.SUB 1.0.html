Message identifier (Name):

FM.SUB (FIRE MISSION-SUBSEQUENT ADJUSTMENT)

 

 

Related Documents:

ASCA-012 02 CTIDP
STANAG 2432
STANAG 2434

 

 

Purpose:

THE FM.SUB IS USED TO TRANSMIT UPDATED GRID LOCATIONS, TO REPEAT FIRE FOR EFFECT AND/OR TO TERMINATE MISSIONS.

 

 

Sponsor:

NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
