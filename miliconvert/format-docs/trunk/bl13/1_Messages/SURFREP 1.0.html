SURFREP

 

Message identifier (Name):

SURFREP (SURF REPORT)

 

 

Related Documents:

ATP-38

 

 

Purpose:

THE SURFREP IS USED TO REPORT TIME CRITICAL SURF INFORMATION RELATING TO SPECIFIC BEACH(ES) TO BE EMPLOYED DURING AMPHIBIOUS OPERATIONS.

 

 

Sponsor:

NSA (NAVY) AMPHIBIOUS OPERATIONS WORKING GROUP (AMPHIB OPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
