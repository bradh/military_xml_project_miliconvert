MINLAYREP

 

Message identifier (Name):

MINLAYREP (MINEFIELD LAYING REPORT)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE MINLAYREP IS USED TO REPORT THE RESULTS OF MINE LAYING OPERATIONS BY FRIENDLY ENGINEER TROOPS.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
