Message identifier (Name):

NNFP.FP (NON-NUCLEAR FIRE PLANNING-FIRE PLAN)

 

 

Related Documents:

ASCA-012 02 CTIDP
STANAG 2432
STANAG 2434

 

 

Purpose:

THE NNFP.FP IS USED TO TRANSMIT OR DELETE A TARGET AND/OR ORDERS CONCERNING A TARGET FOR A FIRE PLAN OR SPECIFIED TARGET LIST, OR TO DELETE AN ENTIRE FIRE PLAN.

 

 

Sponsor:

NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
