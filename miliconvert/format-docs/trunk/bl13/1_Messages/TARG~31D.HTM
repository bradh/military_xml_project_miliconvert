TARGETINTREC

 

Message identifier (Name):

TARGETINTREC (TARGETING RECORD)

 

 

Related Documents:

AINTP-3 (A)
STANAG 1059
STANAG 2077
STANAG 2433
STANAG 2440
STANAG 6001

 

 

Purpose:

THE TARGETINTREC IS USED TO CREATE OR UPDATE A TARGET RECORD BASED ON THE DATA STRUCTURES CONTAINED IN AINTP-3.  THIS ENABLES THE EXCHANGE OF INDIVIDUAL DATA RECORDS BETWEEN NATIONS AND MILITARY FORMATIONS MAINTAINING DATA AND DATA BASES TO THE STANDARDS CONTAINED IN AINTP-3. IT FACILITATES THE PLANNING AND CONDUCT OF OPERATIONS.

 

 

Sponsor:

NSA (JSB) JOINT INTELLIGENCE WORKING GROUP (JINTWG)

 

 

Notes:

THE MESSAGE CREATES A NEW RECORD OR AN UPDATE OF AN EXISTING ONE DEPENDING ON THE ENTRY IN FIELD 7 OF SET 3 (MSGID) WHICH MUST EITHER READ "NEW" OR "UPD".  SINCE THE FIELD IS OPERATIONALLY DETERMINED IT IS POSSIBLE TO ENTER NO DATA CODE.  IT HAS TO BE UNDERSTOOD THAT IN THIS CASE, THROUGH STRUCTURED LANGUAGE STATEMENTS, THE MESSAGE WILL BE ASSUMED TO BE A NEW ONE.

 

 

Remarks:

None

 

 

Status:

In Service
