MOVEREP

 

Message identifier (Name):

MOVEREP (MOVEMENT REPORT)

 

 

Related Documents:

ACP 176

 

 

Purpose:

THE MOVEREP IS USED TO PROVIDE AND DISSEMINATE INFORMATION RELATED TO THE MOVEMENT OF NAVAL UNITS.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service

THIS SEGMENT DETAILS INFORMATION ABOUT A GROUP OR INDIVIDUAL SHIP MOVEMENT.

THIS SEGMENT COMPRISES SETS 'MARACT' THROUGH 'METEO'.

IT CAN BE REPEATED IF MORE THAN ONE GROUP OR UNIT IS BEING REPORTED.
