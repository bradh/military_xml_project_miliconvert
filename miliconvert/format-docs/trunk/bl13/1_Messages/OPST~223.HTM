OPSTAT READINESS

 

Message identifier (Name):

OPSTAT READINESS (SUBMARINE OPERATION CAPABILITY STATUS)

 

 

Related Documents:

ATP-18

 

 

Purpose:

THE OPSTAT READINESS IS A MESSAGE EXCHANGED BETWEEN SUBOPAUTHS PRIOR TO TACON BEING TRANSFERRED.

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

THE OPSTAT READINESS SHOULD BE READ IN CONJUNCTION WITH THE OPSTAT UNIT MESSAGE SENT BY THE SUBMARINE IAW THE INSTRUCTIONS IN ATP-18. THE STRUCTURE PROVIDES THE SUBOPAUTH ASSUMING TACON WITH INFORMATION ABOUT THE CAPABILITY OF THE SUBMARINE IN THE WARFARE AREAS IDENTIFIED. A POSITIVE STATEMENT IS REQUIRED IF THE SUBMARINE DOES NOT HAVE A CAPABILITY IN ONE OF THE AREAS OF WARFARE IDENTIFIED. A SEPARATE MESSAGE IS REQUIRED FOR EACH SUBMARINE.

 

 

Remarks:

None

 

 

Status:

In Service
