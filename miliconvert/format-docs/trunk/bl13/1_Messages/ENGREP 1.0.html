ENGREP

 

Message identifier (Name):

ENGREP (ENGINEER REPORT)

 

 

Related Documents:

STANAG 2430
STANAG 2434

 

 

Purpose:

THE ENGREP IS USED TO REPORT COMBAT ENGINEER TASK PROGRESS AND UNIT COMBAT EFFECTIVENESS FROM BRIGADE LEVEL TO CORPS LEVEL.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
