BRDMLRECCEREP

 

Message identifier (Name):

BRDMLRECCEREP (BRIDGE DEMOLITION RECONNAISSANCE REPORT)

 

 

Related Documents:

STANAG 2430 (AENGRP-2 (A))

 

 

Purpose:

THE BRDMLRECCEREP IS USED TO REPORT THE RESULTS OF A BRIDGE DEMOLITION RECONNAISSANCE BY FRIENDLY FORCES.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
