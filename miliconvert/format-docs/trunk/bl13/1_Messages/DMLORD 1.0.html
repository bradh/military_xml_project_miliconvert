DMLORD

 

Message identifier (Name):

DMLORD (RESERVED DEMOLITION ORDER)

 

 

Related Documents:

STANAG 2430
STANAG 2434

 

 

Purpose:

THE DMLORD MESSAGE IS USED TO DISSEMINATE INFORMATION RELATING TO THE EXECUTION OF A RESERVED DEMOLITION.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
