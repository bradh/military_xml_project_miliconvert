COMMGATEWAYSTAT

 

Message identifier (Name):

COMMGATEWAYSTAT (COMMUNICATIONS GATEWAY STATUS REPORT)

 

 

Related Documents:

STANAG 4211

 

 

Purpose:

THE COMMGATEWAYSTAT IS USED TO REPORT THE STATUS OF A COMMUNICATIONS GATEWAY.

 

 

Sponsor:

SC 6-WG1. TACTICAL COMMUNICATIONS WORKING GROUP.

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service

CHANDIGI

PROVIDES THE NUMBER OF COMMON USER CHANNELS, NUMBERS OF CHANNELS DEDICATED FOR PERMANENT CONNECTIONS, NUMBERS OF PERMANENT CONNECTION CHANNELS FOR PACKET SWITCHING, AND NUMBERS OF CHANNELS OCCUPIED BY SOLE USER CONNECTIONS.
