COMSPOT

 

Message identifier (Name):

COMSPOT (COMMUNICATIONS SPOT REPORT)

 

 

Related Documents:

BI-SC REPORTING DIRECTIVE, VOL. III
STANAG 2434

 

 

Purpose:

THE COMSPOT IS USED TO REPORT ACTUAL OR FORECAST COMMUNICATIONS OUTAGES INCLUDING RELOCATION AND EMCON.

 

 

Sponsor:

SHAPE

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
