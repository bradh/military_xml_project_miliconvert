FM.CFF

 

Message identifier (Name):

FM.CFF (FIRE MISSION-CALL FOR FIRE)

 

 

Related Documents:

ASCA-012 02 CTIDP
STANAG 2432
STANAG 2434

 

 

Purpose:

THE FM.CFF IS USED TO TRANSMIT INITIAL FIRE FOR EFFECT REQUESTS AND/OR ORDERS TO FIRE.

 

 

Sponsor:

NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service

O

MUE

14

MUNITIONS IN EFFECT

USED TO SPECIFY THE INITIAL PROJECTILE/FUZE COMBINATION AND ONE ADDITIONAL PROJECTILE/FUZE COMBINATION. FIELDS 2 AND 3 OF THIS SET ARE OPTIONAL IN THIS MESSAGE.

SIGNIFICANT FIELD 1 IN EVERY OCCURRENCE OF FIELD GROUP IN SET 14 (MUE) IS PROHIBITED IF SET 15 (EFR) OCCURS.

FIELD GROUP IN SET 14 (MUE) MUST OCCUR FEWER THAN 2 TIMES IF SET 17 (FIRINT) OCCURS.

FIELD GROUP IN SET 14 (MUE) MUST OCCUR FEWER THAN 3 TIMES.

 

 

 

C

EFR

15

EFFECTS REQUIRED

INDICATES THE PERCENT EFFECTS TO BE ACHIEVED AGAINST THE TARGET.

SET 15 (EFR) IS PROHIBITED IF SET 17 (FIRINT) OCCURS OR SIGNIFICANT FIELD 1 IN EVERY OCCURRENCE OF FIELD GROUP IN SET 14 (MUE) OCCURS.

 

 

 

O

MOC

16

METHOD OF CONTROL

USED TO SPECIFY THE METHOD OF CONTROL TO BE USED IN ATTACKING THE TARGET.

FIELD 2 IN SET 16 (MOC) IS REQUIRED IF FIELD 1 IN SET 16 (MOC) EQUALS /T/*, OTHERWISE IT IS PROHIBITED.

 

 

 

O

FIRINT

17

FIRING INTERVAL

INDICATES THE FIRING INTERVAL BETWEEN VOLLEYS OR THE DURATION OF FIRE AGAINST THE TARGET.

FIELD 2 IN SET 17 (FIRINT) IS REQUIRED IF SIGNIFICANT FIELD 1 IN EVERY OCCURRENCE OF FIELD GROUP IN SET 14 (MUE) DOES NOT OCCUR.

FIELD 2 IN SET 17 (FIRINT) IS PROHIBITED IF SET 14 (MUE) OCCURS AND SIGNIFICANT FIELD 1 IN EVERY OCCURRENCE OF FIELD GROUP IN SET 14 (MUE) OCCURS.

 

 

 

C

EOM

18

END OF MISSION

USED TO SPECIFY END OF MISSION, OR END OF MISSION AND RECORD AS TARGET.

SET 18 (EOM) IS PROHIBITED IF FIELD 1 IN SET 16 (MOC) EQUALS /AMC/.

FIELD 1 IN SET 18 (EOM) MUST EQUAL /Y/ IF SET 17 (FIRINT) OCCURS.
