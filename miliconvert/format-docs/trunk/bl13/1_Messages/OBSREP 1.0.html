OBSREP

 

Message identifier (Name):

OBSREP (OBSTACLE REPORT)

 

 

Related Documents:

STANAG 2430
STANAG 2434

 

 

Purpose:

THE OBSREP IS USED TO REPORT OBSTACLES UP THE CHAIN OF COMMAND.

 

 

Sponsor:

NSA (ARMY) COMBAT ENGINEER WORKING GROUP (ENGRWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
