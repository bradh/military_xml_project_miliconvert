OPTASK AMPHIB

 

Message identifier (Name):

OPTASK AMPHIB (OPERATIONAL TASKING AMPHIBIOUS OPERATIONS)

 

 

Related Documents:

ATP-1
ATP-36
ATP-39
ATP-8

 

 

Purpose:

THE OPTASK AMPHIB IS USED BY THE COMMANDER AMPHIBIOUS TASK FORCE TO PROVIDE INITIAL TASKING INSTRUCTIONS TO UNITS REGARDING THE EXECUTION OF AN AMPHIBIOUS OPERATION. IT CAN ALSO BE USED TO PROVIDE UPDATES TO AN INITIAL OPTASK AMPHIB MESSAGE.

 

 

Sponsor:

NSA (NAVY) AMPHIBIOUS OPERATIONS WORKING GROUP (AMPHIB OPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
