MAPREQ

 

Message identifier (Name):

MAPREQ (MAP REQUEST)

 

 

Related Documents:

STANAG 2434

 

 

Purpose:

THE MAPREQ IS USED TO SUBMIT REQUESTS FOR MAP COVERAGE.

 

 

Sponsor:

NSA (ARMY) LAND FORCES LOGISTICS DOCTRINE WORKING GROUP (LOGWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service

MESSAGE IDENTIFIER

GEODATUM

5

GEODETIC DATUM

PROVIDES GEODETIC DATUM REFERENCE FOR GEOGRAPHIC LOCATIONS IN THE MESSAGE.

SET 5 (GEODATUM) IS REQUIRED IF SET 13 (LOCNF) OCCURS, OTHERWISE IT IS PROHIBITED.

DTG

6

DATE-TIME GROUP

IF USED, SPECIFIES THE DATE-TIME GROUP OF THE MESSAGE ORIGIN OR AUTHORISATION FOR RELEASE.

 

 

 

O

ORGIDDFT

7

ORGANIZATION DESIGNATOR OF DRAFTER/RELEASER

IF USED, PROVIDES THE UNIT DESIGNATOR, UNIT SIZE, NATIONALITY, UNIT ROLE INDICATOR(S), HIGHER FORMATION NAME AND ARMED SERVICE OR CIVILIAN AGENCY OF THE UNIT THAT DRAFTED THE MESSAGE OR AUTHORIZED ITS RELEASE.

FIELD 10 IN SET 7 (ORGIDDFT) IS REQUIRED IF FIELD 2 IN SET 7 (ORGIDDFT) EQUALS /CORPS/ OR /ARMY/ OR /AG/ OR /MOD/ OR /MD/.

 

 

ü

O

MAP DATA WITHOUT GEODETIC DATUM

M

MAPSREQ

9

MAPS REQUIRED

PROVIDES DATA TO IDENTIFY  MAPS OR CHARTS REQUIRED AND SPECIFIES THE QUANTITY OF EACH OF THESE REQUESTED. REQUIRED.

 

 

 

M

GENTEXT

10

REASON FOR THE REQUEST

SPECIFIES THE REASON FOR THE REQUEST IN FREE TEXT.

FIELD 1 IN SET 10 (GENTEXT) IS ASSIGNED THE VALUE /REASON FOR THE REQUEST/.

 

 

 

M

PICKDEL

11

PICKUP OR DELIVERY

SPECIFIES THE TRANSFER METHOD: PICKUP OR DELIVERY.

 

 

 

M

TIME

12

TIME

SPECIFIES THE LATEST TIME THE REQUIRED MAPS ARE TO BE AVAILABLE.

 

 

 

C

LOCNF

13

LOCATION AND NEAREST FEATURE

IF USED, PROVIDES THE NAME OF THE NEAREST FEATURE AND THE  LOCATION FOR  DELIVERY.

SET 13 (LOCNF) IS REQUIRED IF FIELD 1 IN SET 11 (PICKDEL) EQUALS /DELIVERY/ AND SET 14 (ADDRESS) DOES NOT OCCUR.

SET 13 (LOCNF) IS PROHIBITED IF FIELD 1 IN SET 11 (PICKDEL) DOES NOT EQUAL /DELIVERY/.

 

 

 

C

ADDRESS

14

ADDRESS

IF USED, PROVIDES THE ADDRESS FOR DELIVERY.

SET 14 (ADDRESS) IS REQUIRED IF FIELD 1 IN SET 11 (PICKDEL) EQUALS /DELIVERY/ AND SET 13 (LOCNF) DOES NOT OCCUR.

SET 14 (ADDRESS) IS PROHIBITED IF FIELD 1 IN SET 11 (PICKDEL) DOES NOT EQUAL /DELIVERY/.

 

 

ü

PROVIDES THE POINT OF CONTACT DATA BY RANK, APPOINTMENT TITLE AND TELEPHONE NUMBER.
