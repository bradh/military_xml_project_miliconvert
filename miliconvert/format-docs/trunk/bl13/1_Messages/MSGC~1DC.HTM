MSGCORRCANX

 

Message identifier (Name):

MSGCORRCANX (MESSAGE CORRECTION OR CANCELLATION)

 

 

Related Documents:

STANAG 2434

 

 

Purpose:

THE MSGCORRCANX IS USED TO CANCEL A MESSAGE(S) AND/OR TO CORRECT THE INFORMATION IN A PREVIOUSLY TRANSMITTED MESSAGE(S).

 

 

Sponsor:

NSA (NAVY) MARITIME OPERATIONS WORKING GROUP (MAROPS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
