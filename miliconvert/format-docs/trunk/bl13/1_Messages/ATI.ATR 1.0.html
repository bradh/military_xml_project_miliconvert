ATI.ATR

 

Message identifier (Name):

ATI.ATR (ARTILLERY TARGET INTELLIGENCE-ARTILLERY TARGET REPORT)

 

 

Related Documents:

ASCA-012 02
STANAG 2432

 

 

Purpose:

THE ATI.ATR IS USED TO TRANSMIT TARGETS BY TARGET TYPE BASED ON A STANDING REQUEST FOR INFORMATION OR A ONE TIME QUERY FOR TARGET INFORMATION RESULTING FROM AN ATI.TIR MESSAGE. IT WILL ALSO BE USED TO ESTABLISH OR DELETE TARGET INFORMATION.

 

 

Sponsor:

NSA (ARMY) ARTILLERY WORKING GROUP (ARTYWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
