AIRMOVE

 

Message identifier (Name):

AIRMOVE (AIRMOVE DETAILS)

 

 

Related Documents:

ATP-34 (A)

 

 

Purpose:

THE AIRMOVE IS USED TO DETERMINE AND CONFIRM WHICH AGENCY HAS TACTICAL CONTROL AND TO ENABLE OVERDUE ACTION TO BE INITIATED AT THE EARLIEST OPPORTUNITY.

 

 

Sponsor:

NSA (AIR) AIR OPERATIONS WORKING GROUP (AOWG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
