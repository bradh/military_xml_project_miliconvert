Message identifier (Name):

HELTASK (HELICOPTER TASK)

 

 

Related Documents:

ATP-49 (A)
STANAG 2434

 

 

Purpose:

THE HELTASK IS USED BY A TASKING AGENCY TO ALLOCATE TRANSPORT HELICOPTER OR UTILITY HELICOPTER SUPPORT.

 

 

Sponsor:

NSA (ARMY) HELICOPTER INTERSERVICE WORKING GROUP (HIS WG)

 

 

Notes:

None

 

 

Remarks:

None

 

 

Status:

In Service
