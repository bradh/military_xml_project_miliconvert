STRUCTURED LANGUAGE
:
(NU)
1.
SET 2 (OPER) IS PROHIBITED IF SET 1 (EXER) OCCURS.
2.
FIELD 1 IN SET 3 (MSGID) IS ASSIGNED THE VALUE "NAVSITREP".
3.
SEGMENT 15 IS REQUIRED IF FIELD 2 IN SET 4 (REF) EQUALS "ORBATSEA" OR "NAVSITSUM" OR "NAVSITREP".
4.
SEGMENT 16 IS REQUIRED IF FIELD 1 IN SET 15 (SMRY) EQUALS "SHIP ACTIVITY" OR "COMBATANTS" OR "NONCOMBATANTS" OR "AMPHIBIOUS".
5.
SEGMENT 17 IS REQUIRED IF FIELD 2 IN SET 4 (REF) EQUALS "NAVSITSUM" OR "NAVSITREP" AND FIELD 1 IN SET 15 (SMRY) EQUALS "SHIP ACTIVITY" OR "COMBATANTS" OR "NONCOMBATANTS" OR "AMPHIBIOUS".
6.
SET 19 (1SHIPS) IS PROHIBITED IF SET 18 (SHIPS) OCCURS.
7.
SEGMENT 21 IS REQUIRED IF FIELD 1 IN SET 15 (SMRY) EQUALS "INDEPENDENT".
8.
SEGMENT 24 IS PROHIBITED IF FIELD 1 IN SET 15 (SMRY) DOES NOT EQUAL "AMPHIBIOUS".
9.
SET 25 (1APHIB) IS REQUIRED IF SET 24 (APHIB) DOES NOT OCCUR, OTHERWISE IT IS PROHIBITED.
10.
FIELD 2 IN SET 5 (EFDT) IS REQUIRED IF FIELD 1 EQUALS FFIRN/FUDN 2000-013, OTHERWISE IT IS PROHIBITED.
